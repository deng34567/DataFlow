# 1 适用范围和应用领域

本标准适用于饮用水、地面水和核工业排放废水中锶-90的分析。测定范围： $10^{-2}\sim 10\mathrm{Bq / L}$ $(10^{-12}\sim 10^{-9}\mathrm{Ci / L})$ 。干扰测定：水样中钙的浓度超过 $1.5\mathrm{g / L}$ 时，会使锶的化学回收率偏高。

# 2 原理

用乙二胺四乙酸二钠（简称EDTA二钠）和柠檬酸两种络合剂将水样中钙、镁等络合，调节溶液 $\mathrm{pH}$ 至 $4.5 \sim 5.0$ ，使绝大部分钙通过阳离子交换柱，而锶和部分钙被树脂吸附。再用不同浓度和 $\mathrm{pH}$ 的EDTA-乙酸铵溶液先后淋洗钙和锶。向含锶的流出液中加入铜盐，将锶从EDTA和柠檬酸的络合物中置换出来，进行碳酸盐沉淀，放置14d后分离出钇，通过测定钇-90的β活度来确定水中锶-90的浓度。

# 3 试剂

所有试剂除特别申明者外，均为分析纯，其放射性必须保证空白样品测得的计数率不超过仪器本底的统计误差。操作中作为试剂加入的水均指蒸馏水。

3.1 锂载体溶液（约 $50\mathrm{mgSr / ml}$ ）：按GB6766—86《水中锶-90放射化学分析方法二一（2-乙基己基）磷酸萃取色层法》中3.4所述进行配制和标定。
3.2 钽载体溶液（约 $20\mathrm{mgY / ml}$ ）：按GB6766—86中3.5所述进行配制和标定。
3.3 钡载体溶液(约 $20\mathrm{mgBa / ml}$ )：称取 $35.57\mathrm{g}$ 氯化钡（ $\mathrm{BaCl}_2\cdot 2\mathrm{H}_2\mathrm{O}$ ）溶于 $0.1\mathrm{mol / L}$ 盐酸中并稀释至 $1\mathrm{L}$ 。
3.4 三氯化铁溶液：浓度约 $10\mathrm{mgFe / ml}$
3.5 氨水：无二氧化碳。
3.6 柠檬酸溶液：浓度 $5\% (m / m)$ 。
3.7. 氯化钠溶液：浓度 $20\%$ （ $m / m$ ）。
3.8 EDTA二钠溶液：浓度 $10\%$ （ $m / m$ ）。
3.9 氨缓冲溶液：称取 $20\mathrm{g}$ 氯化铵溶于 $50\mathrm{ml}$ 蒸馏水中，加入 $100\mathrm{ml}$ 浓氨水，用水稀释到 $1\mathrm{L}$ 。
3.10 络黑T溶液：称取 $100\mathrm{mg}$ 络黑T溶于 $10\mathrm{ml}$ 氨缓冲溶液（3.9）中，用无水乙醇稀释到 $20\mathrm{ml}$ 。有效期一个月。
3.11 732苯乙烯型强酸性阳离子交换树脂（强酸 $1 \times 2$ ）， $50 \sim 100$ 目。
3.11.1 树脂的处理：用水浸泡 $10\mathrm{h}$ 以上，再用 $6\mathrm{mol} / \mathrm{L}$ 盐酸浸泡两次，每次 $4\mathrm{h}$ 。用水洗至中性。
3.11.2 树脂的装柱：量取 $50\mathrm{ml}$ 湿树脂（3.11.1），用水装入交换柱中，柱的上下部均用玻璃毛填

充。用 $200\mathrm{ml}$ 氯化钠溶液（3.7）以 $3\mathrm{ml} / \mathrm{min}$ 的流速通过交换柱。用 $200\mathrm{ml}$ 水淋洗，备用。

3.11.3 树脂的再生：依次用 $100\mathrm{ml}$ 水、 $200\mathrm{ml}6\mathrm{mol} / \mathrm{L}$ 盐酸、 $200\mathrm{ml}$ 水、 $200\mathrm{ml}$ 氯化钠溶液（3.7）和 $200\mathrm{ml}$ 水以 $3\mathrm{ml} / \mathrm{min}$ 流速淋洗交换柱。
3.12 钙淋洗剂：称取 $38\mathrm{g}$ EDTA 二钠和 $25\mathrm{g}$ 乙酸铵溶于 $1\mathrm{L}$ 水中，用 $6\mathrm{mol} / \mathrm{L}$ 的氢氧化钠溶液和 $6\mathrm{mol} / \mathrm{L}$ 的盐酸调节溶液 $\mathrm{pH}$ 至 4.4（用 $\mathrm{pH}$ 计测量）。
3.13 草酸-草酸铵溶液：向饱和草酸铵溶液中滴加饱和溶液，调节 $\mathrm{pH}$ 至 $4.0\sim 4.5$
3.14 锂解吸剂：称取 $38\mathrm{g}$ EDTA二钠和 $55\mathrm{g}$ 乙酸铵溶于 $1\mathrm{L}$ 水中，用 $6\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液和 $6\mathrm{mol} / \mathrm{L}$ 盐酸溶液调节溶液 $\mathrm{pH}$ 至 $5.5\sim 6.0$ （用 $\mathrm{pH}$ 计测量）。
3.15 钡淋洗剂：称取 $38\mathrm{g}$ EDTA 二钠溶于 $1\mathrm{L}$ 水中，用 $6\mathrm{mol} / \mathrm{L}$ 的盐酸和 $6\mathrm{mol} / \mathrm{L}$ 的氢氧化钠溶液调节溶液 $\mathrm{pH}$ 至 $9.0$ 。

# 4 仪器设备

4.1 低本底 $\beta$ 射线测量仪。
4.2 分析天平，感量 $0.1\mathrm{mg}$ 。
4.3 离心机， $0\sim 4000\mathrm{rpm}$ ， $4\times 15\mathrm{ml}$ 。
4.4 交换柱，内径 $18\mathrm{mm}$ ，高 $300\mathrm{mm}$ 。
4.5 pH计。
4.6 烘箱。
4.7 马福炉。

# 5 仪器的刻度

见GB6766—86《水中锶-90放射化学分析方法二-（2-乙基己基）磷酸萃取色层法》的5.1和5.2。

# 6 操作步骤

6.1 取澄清后的水样 $1\sim 40\mathrm{L}$ ，加入 $2.00\mathrm{ml}$ 锂载体溶液（3.1）和 $1.00\mathrm{ml}$ 钡载体溶液（3.3），加入EDTA二钠溶液（3.8）和等体积的柠檬酸溶液（3.6），直到钙、镁络合完全。再加入 $20\mathrm{ml}2\mathrm{mol} / \mathrm{L}$ 乙酸和 $2\mathrm{mol} / \mathrm{L}$ 乙酸铵溶液，用 $6\mathrm{mol} / \mathrm{L}$ 盐酸和 $6\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液调节 $\mathrm{pH}$ 至 $4.5\sim 5.0$ 。

注：钙、镁是否络合完全可用络黑T检查，其方法是：取约 $1\mathrm{ml}$ 样品溶液，加入等体积的氨缓冲溶液（3.9）及1滴络黑T溶液（3.10），如颜色为蓝色而不出现紫红色，表示钙、镁已络合完全。同时可用无离子水做对照。

6.2 溶液以 $20\mathrm{mol / min}$ 的流速通过阳离子交换树脂柱。

6.3 用钙淋洗剂（3.12）以 $8\mathrm{ml} / \mathrm{min}$ 的流速通过交换柱，流出液不时用草酸-草酸铵溶液（3.13）检查，至无钙后继续通过 $150\mathrm{ml}$ 钙淋洗剂（3.12）。

注：流出液中是否存在钙离子的检查方法：取 $1\mathrm{ml}$ 流出液与等体积的草酸-草酸铵溶液（3.13）混合摇动 $1\mathrm{min}$ 与无离子水对照，若无浑浊现象，则表示无钙。

6.4 用 $200\mathrm{ml}$ 锂解吸剂（3.14）以 $4\sim 5\mathrm{ml / min}$ 流速解吸，解吸液收集于烧杯中。

6.5 用 $200\mathrm{ml}$ 锁淋洗剂（3.15）以 $5\mathrm{ml} / \mathrm{min}$ 流速淋洗钡。弃去流出液。

注：若已知样品中无钡-140存在，可省去本步骤，同时在6.1中也不必加入钡载体溶液。

6.6 在含锶的解吸液中加入 $4 \sim 8 \mathrm{~g}$ 固体氯化铜，搅拌使其溶解，用氨水调节溶液至碱性（用广范 $\mathrm{pH}$ 试纸检查）。再加入 $5 \mathrm{ml}$ 氨水、 $2 \mathrm{~g}$ 结晶碳酸铵，加热至将近沸腾，冷却至室温。过滤，弃去滤液，再用 $20 \mathrm{ml}$ 水洗涤沉淀。用 $10 \mathrm{ml} 2 \mathrm{~mol} / \mathrm{L}$ 硝酸溶解沉淀，并用水稀释至 $30 \mathrm{ml}$ ，加入 $1 \mathrm{ml}$ 氨水（3.5），调节溶液至碱性（用广范 $\mathrm{pH}$ 试纸检查）。趁热过滤，用约 $10 \mathrm{ml}$ 水洗涤一次，弃去沉淀。记下日期和时间，作为 $\mathrm{H}_{2} - 90$ 开始生长的时刻。

6.7 将滤液收集于烧杯中，加入 $5 \mathrm{ml}$ 饱和碳酸铵溶液，加热至将近沸腾，冷却至室温。沉淀过滤于可拆卸式漏斗内的已称重滤纸上，用水和无水乙醇各 $10 \mathrm{ml}$ 洗涤沉淀，在 $105^{\circ} \mathrm{C}$ 烘干 $15 \mathrm{~min}$ ，在干燥器内冷却 $20 \mathrm{~min}$ 后称重，计算锶的化学回收率。
6.8 将称重后的碳酸锶用约 $10\mathrm{ml}2\mathrm{mol} / \mathrm{L}$ 硝酸溶解于烧杯中，加入 $10\mathrm{ml}$ 水和 $1.00\mathrm{ml}$ 钽载体溶液（3.2），放置 $14\mathrm{d}$ 以上。
6.9 将溶液煮沸几分钟，赶去二氧化碳，用氨水（3.5）调节溶液 $\mathrm{pH}$ 至8，继续加热 $10\mathrm{min}$ ，使沉淀凝聚，冷却到室温，离心，记下、亿分离的时刻。弃去上层清液。
6.10 在离心管中加入 $2 \mathrm{~mol} / \mathrm{L}$ 硝酸至沉淀溶解, 用 $20 \mathrm{ml}$ 水稀释, 用氨水 (3.5) 调节溶液至碱性 (用广范 $\mathrm{pH}$ 试纸检查)。离心, 弃去上层清液。
6.11 在离心管中加入 $2 \mathrm{~mol} / \mathrm{L}$ 硝酸至沉淀溶解，溶液用水转移到 $50 \mathrm{ml}$ 烧杯中，用氨水调节 $\mathrm{pH}$ 至 $1.5 \sim 2.0$ ，加入 $5 \mathrm{ml}$ 饱和草酸溶液，加热使沉淀凝聚，冷却到室温。沉淀在铺有已称重滤纸的可拆卸式漏斗上抽滤，先后用 $10 \mathrm{ml}$ 水和 $10 \mathrm{ml}$ 无水乙醇洗涤，抽干，半小时后将沉淀和滤纸固定在测量盘上，在低本底 $\beta$ 测量仪上进行 $\beta$ 计数。记下测量进行到一半的时刻。
6.12 沉淀在空气中风干 $5\mathrm{h}$ 后称重，称至恒重。计算钇的化学回收率。

注：草酸钇沉淀的组成为： $\mathrm{Y}_2(\mathrm{C}_2\mathrm{O}_4)_3\cdot 9\mathrm{H}_2\mathrm{O}$

# 7 计算

见GB6764—86《水中锶-90放射化学分析方法发烟硝酸沉淀法》第7章。

# 8 分析误差

本方法分析锶-90浓度为 $1\mathrm{Bq} / \mathrm{L}$ （ $3\times 10^{-11}\mathrm{Ci} / \mathrm{L}$ ）的水样，最大误差小于 $10\%$ ，同一实验室 $\mathcal{V}$ 变异系数小于 $10\%$ 。

# 正确使用标准的说明

（参考件）

A.1 水样中的锶含量超过 $1\mathrm{mg}$ 时，必须进行样品自身锶含量的测定，并在计算锶的化学回收率时将其扣除。
A.2 按下式决定样品的计数时间 $t_{\mathrm{c}}$ (min)：

$$
t _ {\mathrm {c}} = \frac {N _ {\mathrm {c}} + \sqrt {N _ {\mathrm {c}} \cdot N _ {\mathrm {b}}}}{N ^ {2} \cdot E ^ {2}}
$$

式中： $N_{\mathrm{c}}$ ——样品源加本底的计数率， $\mathrm{cpm}$ ；

$N_{\mathrm{v}}$ 本底计数率，cpm；

$N$ ——样品源净计数率， $\mathrm{cpm}$ ；

$E$ ——预定的相对标准误差。
