# 中华人民共和国国家标准

# 低、中水平放射性废物固化体性能要求塑料固化体

Characteristic requirements for solidified waste of low

and intermediate level radioactive waste

—Plastic solidified waste

GB 14569.2-93

# 1 主题内容与适用范围

本标准规定了低、中水平放射性废物塑料固化体的最低性能要求和检验方法。

本标准适用于近地表处置的塑料固化体，岩洞处置的塑料固化体也可参照使用。

# 2 引用标准

GB1041 塑料压缩试验方法

GB7023 放射性废物固化体长期浸出试验

GB9132 低、中水平放射性固体废物的浅地层处置规定

GB9133 放射性废物分类标准

GB13600 低、中水平放射性固体废物岩洞处置规定

GB11806 放射性物质安全运输规定

# 3 塑料固化体放射性比活度限值

塑料固化体放射性比活度必须满足GB9132，GB9133，GB13600中的有关规定。

# 4 塑料固化体性能要求

塑料固化体的性能必须满足GB9132第5.1条、GB11806第8章和GB13600中有关规定和要求。

# 4.1 物理性能

# 4.1.1 均匀性和密实性

废物被均匀密实地包容在固化体中，在固化体外表和剖面，不存在肉眼看得见的空隙和裂缝。

# 4.1.2 游离液体

完全硬化后的塑料固化体必须不存在游离液体。

# 4.2 机械性能

# 4.2.1 抗压强度

固化体的抗压强度必须不小于 $7\mathrm{MPa}$ 。

# 4.2.2 抗冲击强度

固化体试样从 $9\mathrm{m}$ 高度垂直跌落至混凝土地面上没有明显碎裂。

# 4.3 抗水性

# 4.3.1 抗浸出性

塑料固化体试样在 $25^{\circ} \mathrm{C}$ 的去离子水中浸出，第 42d 不同核素的浸出率分别为下列限值：

$^{137}\mathrm{Cs}$ 应小于 $5 \times 10^{-4} \mathrm{~cm} / \mathrm{d}$ ;

$^{60}\mathrm{Co}$ 应小于 $1 \times 10^{-4} \mathrm{~cm} / \mathrm{d}$ ;

$^{90}\mathrm{Sr}$ 应小于 $1 \times 10^{-4} \mathrm{~cm} / \mathrm{d}$ ;

${ }^{239} \mathrm{Pu}$ 应小于 $1 \times 10^{-5} \mathrm{~cm} / \mathrm{d}$ 。

对于其他 $\beta, \gamma$ 射线的放射性核素(不包括氚)的浸出率应不大于 $1 \times 10^{-4} \mathrm{~cm} / \mathrm{d}$ ，对其他超铀核素的浸出率应不大于 $1 \times 10^{-5} \mathrm{~cm} / \mathrm{d}$ 。

# 4.3.2 抗浸泡性

浸泡试验后，固化体试样不得破碎。体积变化应不大于 $5\%$ ，抗压强度的变化不得大于浸泡前的抗压强度的 $15\%$ 。

# 4.4 抗冻-融性

冻-融试验后，固化体的体积变化应不大于 $2\%$ ，肉眼应看不到裂缝，抗压强度的降低应不大于 $15\%$ 。

# 4.5 抗 $\pmb{\gamma}$ 辐照性

固化体试样辐照试验后，体积膨胀应不大于固化体试样的 $5\%$ ，抗压强度的降低应不大于 $15\%$ 。

如果固化体包容的放射性在300a所产生的累计积分剂量小于 $10^{4}\mathrm{Gy}$ ，不要求做辐照试验。

# 4.6 抗着火性

热固性塑料固化体试样在抗着火试验后，在火源移开后 $30\mathrm{s}$ 内要自熄，在固化体表面仅留下碳化层，碳化层的厚度应不大于 $5\mathrm{mm}$ ，质量损失应小于 $20\%$ 。

# 5 检验方法

# 5.1 游离液体

按照规定的配方实验室制备的模拟废物固化体试样，其直径不小于 $80 \mathrm{~mm}$ ，高度不小于 $900 \mathrm{~mm}$ ，养护硬化 $7 \mathrm{~d}$ 后，观察固化体表面有无游离液体，并在盛装固化体的容器底部用钻孔或其他适当的方法开口，开口的面积不小于 $6.5 \mathrm{~cm}^2$ ，从开口处检查有无游离液体流出或滴落。

# 5.2 抗压强度

按照规定的配方实验室制备五个平行模拟废物试样 $\phi = 50 \mathrm{~mm}, H = 50 \mathrm{~mm}$ 的圆柱体, 养护硬化7d后, 按GB1041中的要求进行检验。

# 5.3 抗冲击强度

试样制备同5.2条。固化体试样以自由落体方式竖直跌落至混凝土地面，观察固化体有没有碎裂（棱角小碎块和裂纹不作破裂看待）。

# 5.4 抗水性

# 5.4.1 抗浸出性

实验室制备的放射性示踪样品(如有载体应指出载体浓度)养护硬化7d后,或从实际固化体钻取的试样 $\phi = 50 \mathrm{~mm}, H = 50 \mathrm{~mm}$ , 按照GB7023规定的实验步骤进行试验。

# 5.4.2 浸泡性

试样制备同5.2条。固化体试样在 $25 \pm 5^{\circ} \mathrm{C}$ 的温度条件下至少连续浸泡 $90 \mathrm{~d}$ , 浸泡液最好采用处置场的模拟地下水, 观察固化体有无破裂, 测量固化体的直径和高度变化及抗压强度的变化。

# 5.5 抗冻-融性

试样制备同5.2条。固化体试样在 $-20\sim -15^{\circ}C$ 放置 $24\mathrm{h}$ ，紧接着在 $20\sim 25^{\circ}C$ 大气中放置 $24\mathrm{h}$ ，至少如此循环5次，每次循环后测量直径和高度变化及抗压强度变化。

# 5.6 抗 $\gamma$ 辐照性

试样制备同5.2条。把试样封装在玻璃管中，留有 $5\% \sim 10\%$ 的自由空间体积，把封装好的样品放

入专门的 ${}^{60}\mathrm{Co}$ 辐射源照射孔道内照射，辐照剂量率不大于 $2 \times 10^{3}\mathrm{Gy/h}$ ，直至总的积分吸收剂量达到 $10^{6}\mathrm{Gy}$ 。测量辐照后的试样的抗压强度变化及体积变化。

# 5.7 抗着火性

按照规定配方制备的径高比为 $1:1$ 的模拟废物试样 $2\mathrm{kg}$ ，在 $800 \pm 50^{\circ}\mathrm{C}$ 的喷灯火焰上方放置 $10\mathrm{min}$ ，火源移开后，观察火苗是否自熄，量其碳化层的厚度，称其质量，计算质量变化。
