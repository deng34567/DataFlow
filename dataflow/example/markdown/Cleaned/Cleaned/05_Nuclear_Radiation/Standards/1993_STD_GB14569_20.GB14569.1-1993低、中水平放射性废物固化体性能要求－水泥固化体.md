# 中华人民共和国国家标准

# 低、中水平放射性废物固化体性能要求水泥固化体

Characteristic requirements for solidified waste of low

and intermediate level radioactive waste

—Cement solidified waste

GB 14569.1-93

# 1 主题内容与适用范围

本标准规定了低、中水平放射性废物水泥固化体(以下简称水泥固化体)的最低性能要求和检验方法。

本标准适用于近地表处置的水泥固化体，岩洞处置的水泥固化体也可参照执行。

# 2 引用标准

GB177 水泥胶砂强度检验方法

GB7023 放射性废物固化体长期浸出试验

GB9132 低、中水平放射性固体废物的浅地层处置规定

GB9133 放射性废物分类标准

GB11806 放射性物质安全运输规定

GB13600 低、中水平放射性固体废物的岩洞处置规定

# 3 水泥固化体放射性比活度限值

水泥固化体的放射性比活度必须满足GB9132和GB9133的有关规定和要求。

# 4 性能要求

水泥固化体的性能必须满足GB9132的第5.1条、GB11806的第8章及GB13600第5.1条中有关规定和要求。

# 4.1 游离液体

在室温、密闭条件下，经过养护后的水泥固化体必须不存在泌出的游离液体。

# 4.2 机械性能

在室温、密闭条件下，经过养护、完全硬化后的水泥固化体，应是密实、均匀、稳定的块体，并必须满足下列要求：

a. 抗压强度水泥固化体试样的抗压强度不应小于 $7\mathrm{MPa}$ ；
b. 抗冲击性能从 $9 \mathrm{~m}$ 高处竖直自由下落到混凝土地面上的水泥固化体试样不应有明显的破碎。

# 4.3 抗水性

# 4.3.1 抗浸出性

水泥固化体试样在 $25\mathrm{C}$ 的去离子水中浸出，第42d核素的浸出率应分别低于下列限值：

$$
\begin{array}{l} \mathrm {s o} \mathrm {C o} \leqslant 2 \times 1 0 ^ {- 3} \mathrm {c m} / \mathrm {d} \\ ^ {1 3 7} \mathrm {C s} \leqslant 4 \times 1 0 ^ {- 3} \mathrm {c m} / \mathrm {d} \\ { } ^ { 9 0 } \mathrm { S r } \leqslant 1 \times 1 0 ^ { - 3 } \mathrm { c m / d } \\ { } ^ { 2 3 9 } \mathrm { P u } \leqslant 1 \times 1 0 ^ { - 5 } \mathrm { c m / d } \\ \end{array}
$$

对水泥固化体中其他 $\beta, \gamma$ 放射性核素（不包括 ${}^{3}\mathrm{H}$ ），其浸出率应低于 $4 \times 10^{-3} \mathrm{~cm} / \mathrm{d}$ 。

水泥固化体中其他超铀核素的浸出率应低于 $1 \times 10^{-5} \mathrm{~cm} / \mathrm{d}$ 。

# 4.3.2 抗浸泡性

水泥固化体试样做抗浸泡试验后不得发生破碎，其抗压强度损失不超过 $25\%$ 。

# 4.4 抗冻融性

水泥固化体试样作抗冻融试验后，其抗压强度损失不超过 $25\%$ 。

# 4.5 耐 $\gamma$ 辐照性

水泥固化体试样进行 $\gamma$ 辐照试验后，其抗压强度损失不超过 $25\%$ 。

当水泥固化体在 $300\mathrm{a}$ 内累积吸收剂量小于 $1\times 10^{4}\mathrm{Gy}$ 时，可不进行本项试验。

# 5 性能检验方法

# 5.1 游离液体

用非放射性的模拟废物按照规定的配方制备水泥固化体，水泥固化体的高度应尽量接近工程上水泥固化体的实际高度（直径不小于 $8\mathrm{cm}$ ，高度不小于 $75\mathrm{cm}$ ），在密闭条件下养护 $7\mathrm{d}$ 后，直观检查水泥固化体的上表面有无游离液体，并在盛装水泥固化体的容器底部用钻孔或其他适当的方法开口，开口的面积应不小于 $6.5\mathrm{cm}^2$ ，从开口处检查有无游离液体流出或滴落。

# 5.2 机械性能

# 5.2.1 抗压强度

用非放射性的模拟废物按照规定的配方制备水泥浆，水泥浆直接倒入试模，用刮平刀抹平（不放在振动台上）。

试样应为直径 $5\mathrm{cm}$ 、高 $5\mathrm{cm}$ 的圆柱体，在密闭条件下、温度为 $25\pm 5\%$ 、不受阳光直接照射的室内环境中养护，测量第28d的抗压强度。

抗压强度的测定应参照GB177中有关要求进行。

# 5.2.2 抗冲击性

试样的制备和养护条件同5.2.1条。养护28d后进行抗冲击试验，试验时试样从 $9\mathrm{m}$ 高处竖直自由下落到混凝土地面上，观察试样有否破碎（出现棱角小碎块和裂纹不作为破碎看待）。

# 5.3 抗水性

# 5.3.1 抗浸出性

抗浸出性试验应采用真实物料的水泥固化体试样进行。在缺乏条件时，也可采用模拟放射性废物的水泥固化体试样。试样的制备和养护条件同5.2.1条。养护28d后进行水泥固化体浸出试验，浸出试验必须遵照GB7023中的有关规定进行。

# 5.3.2 抗浸泡性

试样的制备和养护条件同5.2.1条。养护28d后进行抗浸泡试验，应采用处置场的模拟地下水，在 $25\pm 5\%$ 条件下浸泡，浸泡时间不少于90d，观察水泥固化体是否破碎，并测定其抗压强度。

# 5.4 抗冻融性

试样的制备和养护条件同5.2.1条。养护28d后进行抗冻融试验，当冷冻箱内温度达到 $-20^{\circ}\mathrm{C}$ 时，将装在密闭塑料袋中的试样放入箱中，当箱内温度重新降至 $-20^{\circ}\mathrm{C}$ 时，起算冻结时间，每次冻结时间不少于 $3\mathrm{h}$ （冻结温度应保持在 $-20\sim -15^{\circ}\mathrm{C})3\mathrm{h}$ 后取出试样（连同塑料袋），立即放在 $15\sim 20^{\circ}\mathrm{C}$ 的水槽中

融解, 每次试样的融解时间应不小于 $4 \mathrm{~h}$ 。融解完毕即为该次冻融循环结束。每块水泥固化体试样至少进行 5 次冻融循环, 然后测定其抗压强度。

# 5.5 $\gamma$ 辐照试验

试样的制备和养护条件同5.2.1条。养护28d后封装在玻璃管中，并留有 $5\% \sim 10\%$ 的自由空间体积，把封装好的样品放入专门的 ${}^{60}\mathrm{Co}$ 辐射源辐照孔道内照射（辐照剂量率应低于 $2\times 10^{5}\mathrm{Gy / h}$ ），直至试样累积吸收剂量达到相应比活度水泥固化体所可能受到的累积吸收剂量时，取出玻璃管，测定辐照后试样的抗压强度。
