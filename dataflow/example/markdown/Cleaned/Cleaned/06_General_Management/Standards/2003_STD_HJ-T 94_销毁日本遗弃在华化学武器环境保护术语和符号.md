# 3.1 化学武器

3.1.1

化学武器 chemical weapons

合指或单指：（1）毒剂及其前体；（2）经专门设计通过使用后而释放出毒剂的毒性造成死亡或其他伤害的弹药和装置；（3）经专门设计其用途与本条目（2）所指弹药和装置的使用直接有关的任何设备。

3.1.2

毒剂 chemical agent

通过其对生命过程的化学作用而能够对人类或动物造成死亡、暂时失能或永久伤害的任何化学品，无论其来源或其生产方法如何，也无论其是否在设施中、弹药中或其他地方产生出来。

3.1.3

前体 precursor

在以无论何种方法生产某种毒剂的任何阶段参与此一生产过程的任何化学反应物。

# 3.1.4

遗弃化学武器 abandoned chemical weapons

1925年1月1日以后一国未经另一国同意而遗留在另一国领土上的化学武器。

# 3.1.5

领土国 territorial state party

领土上有遗弃化学武器的国家。

# 3.1.6

遗弃国 abandoning state party

在另一国领土上遗留化学武器的国家。

# 3.1.7

日本遗弃在华化学武器 abandoned chemical weapons in China by Japan

在日本侵略中国的战争过程中，日本遗留在中国领土上的化学武器，包括化学炮弹、炸弹、航空炸弹、毒烟筒、散装毒剂及其相关设备和装置等。

# 3.1.8

糜烂性毒剂 blister agent

起泡性毒剂 vesicant agent

破坏机体细胞，以皮肤或粘膜糜烂作用为主要毒害特征的毒剂。

# 3.1.9

以刺激鼻喉、眼睛、皮肤为主要毒害特征的毒剂

# 3.1.11

室息性毒剂 choking agent

# 3.1.12

混合毒剂 mixed chemical agent

两种或两种以上毒剂的混合物。目的在于改善毒剂的军事使用性能，给侦检、防护、消毒和救治增加困难。

# 3.1.13

2，2’-二氯二乙硫醚的俗称，一种糜烂性毒剂。因含杂质的芥子气有较浓的芥末味而得名。

# 3.1.14

3.1.26

泄漏 leakage

日本遗弃在华化学武器中的装填物从弹体、容器、销毁设备等密闭环境向周围环境非正常扩散的现象。

3.1.27

染毒 chemical contamination

降解产物 degradation product

毒剂发生降解反应后形成的化合物。

3.2 毒性毒理

3.2.1

毒性 toxicity

毒害包括人在内的各种生物的性能。包括产生生理、病理变化、丧失正常生命活动能力乃至死亡，通常以剂量表示。

3.2.2

急性毒性 acute toxicity

在24h内一次或多次染毒条件下，短期内发生的毒害包括人在内的各种生物的性能。

3.2.3

亚慢性毒性 subchronic toxicity

在三个月内反复染毒条件下，毒害包括人在内的各种生物的性能。

3.2.4

慢性毒性 chronic toxicity

在长期（六个月以上）反复染毒条件下，毒害包括人在内的各种生物的性能。

# 3.2.5

毒害效应 toxic effect

损害机体组织器官，导致生理功能紊乱或死亡的现象。

# 3.2.6

致死效应 lethal effect

造成人员和动物死亡的毒害效应。

# 3.2.7

失能效应 incapacitating effect

造成人员和动物丧失正常活动能力的毒害效应。

# 3.2.8

蓄积作用 cumulative effect

反复染毒引起机体内毒剂及其有毒相关物蓄积、毒害作用累积的现象。

# 3.2.9

迟发效应 delayed effect

人或动物经单次剂量或短暂染毒后，经过一段时间（数月或数年）出现的不可逆毒害效应。如致癌、致畸等。

# 3.2.10

毒害剂量 poisoning dose

引起人或动物某种程度毒害的毒剂量。通常用单位体重所受到的毒剂量（如 $\mathrm{mg/kg}$ ）表示；蒸气或气溶胶染毒时，用染毒浓度和时间的乘积（即 $C_t$ 积，如 $\mathrm{mg\cdot min/m^3}$ ）表示：液滴皮肤染毒时，用单位染毒面积所含毒剂量（如 $\mathrm{mg/cm^2}$ ）表示。

# 3.2.11

染毒浓度 contamination concentration

单位体积或质量的染毒空气、水、土壤或固体废物中内所含的毒剂量，常以 $\mathrm{mg / m^3}$ 、 $\mathrm{mg / L}$ 或 $\mathrm{mg / kg}$ 等单位表示。

# 3.2.12

阈剂量 threshold dose

引起染毒人员产生可观察到的特征性毒害的最低毒剂量。

# 3.2.13

阈浓度 threshold concentration

引起染毒人员产生可观察到的特征性毒害的最低染毒浓度。常用单位为 $\mathrm{mg / L}$ 或 $\mathrm{mg / m^3}$ 。也称“临界浓度”。

# 3.2.14

致死剂量 lethal dose

LD

引起染毒人员或动物死亡的毒剂量。

# 3.2.15

半数致死剂量 median lethal dose

$\mathrm{LD}_{50}$ , $\mathrm{LC}_{50}$

引起染毒人员或动物半数死亡的毒剂量。

# 3.2.16

失能剂量 incapacitating dose

HJ/T 94-2003

引起染毒人员或动物丧失正常活动能力的毒剂量。

3.2.17

半数失能剂量 median incapacitating dose

$\mathsf{ID}_{50}$ ， $\mathsf{ICT}_{50}$

引起染毒人员或动物半数丧失正常活动能力的毒剂量。

3.2.18

染毒密度 contamination density

单位面积的物质表面所受到的毒剂量，常以 $\mathrm{mg/cm^2}$ 表示。

3.2.19

暴露时间 exposure time

无防护人员或动物在毒氛中停留的时间。

# 3.3 销毁化学武器

3.3.1

销毁日本遗弃在华化学武器 destruction of abandoned chemical weapons in China by Japan

按照中国有关法律、法规和环境保护标准，将日本遗弃在华的全部化学武器以一种基本上不可逆转的方式转变为一种不适于生产化学武器的形式，以及不可逆转地使弹药和其他装置无法用作化学武器，并确保人员和环境安全的过程。

3.3.2

化学武器销毁设施 chemical weapons destruction facility

以销毁化学武器为目的而制造或使用的设备以及构筑物。

3.3.3

化学武器销毁作业 chemical weapons destruction operation

为销毁化学武器所采取的与销毁过程有关的活动，包括弹药解体、毒剂销毁以及有毒有害废物的处理过程。

3.3.4

销毁日本遗弃在华化学武器全过程 entiredestructionprocessofabandonedchemicalweaponsin ChinabyJapan

为销毁日本遗弃在华化学武器而进行的全部活动，包括：（1）现场调查、探测、挖掘、鉴别、包装、运输和储存等；（2）弹药解体；（3）销毁过程；（4）废气、废水、污染土壤和固体废物的处理处置；（5）销毁作业和处理处置场所关闭后的生态恢复及后续环境管理；（6）除上述外，与销毁日本遗弃在华化学武器有关的其他所有活动。

3.3.5

探测 exploration

对可能埋藏有日本遗弃在华化学武器的区域用仪器设备进行考察与测量的活动。

3.3.6

挖掘 excavation

取出埋藏在地下或水底的日本遗弃在华化学武器的作业。

3.3.7

鉴别 identification

对已发现的弹药、散装化学品进行鉴定，以确认：（1）是否为日本遗弃在华化学武器；（2）是何种类型的化学武器，包括弹种和剂种的确定。

3.3.8

包装 packaging

为确保人员与环境安全以及便利日本遗弃在华化学武器的临时储存、运输和销毁，对日本遗弃在华化学武器及其污染物品采取的密封作业过程。

# 3.3.9

# 储存 storage

为确保销毁日本遗弃在华化学武器的处理过程中人员与环境的安全以及便于销毁作业的进行，对已发现的、已挖掘的或在销毁设施待销毁的日本遗弃在华化学武器及其污染物品采取的临时的、安全的保管过程与措施。

# 3.3.10

# 运输 transportation

为销毁日本遗弃在华化学武器而采用专用车辆移动日本遗弃在华化学武器及其污染物品的过程。

# 3.3.11

# 染毒废物 contaminated wastes

在销毁日本遗弃在华化学武器全过程中产生的沾染了毒剂及其有毒相关物的固体废物。

# 3.3.12

# 工作区 working area

为进行销毁日本遗弃在华化学武器各项活动所划定的区域。

# 3.3.13

销毁去除率 destruction and removal efficiency

DRE

某种毒剂或有毒相关物销毁后所减少的百分比。用公式（1）计算：

$$
\mathrm{DRE}(\%) = \frac{W_{i} - W_{0}}{W_{i}}\times 100\% \tag{1}
$$

式中： $W_{i}$ ——被销毁物中某种毒剂或有毒相关物的重量；

$W_{0}$ ——销毁设施排放气和残余物中与 $W_{i}$ 相应的毒剂或有毒相关物的重量之和。

# 3.4 防护和洗消

# 3.4.1

个人防护 personal protection

个人用于免受毒剂伤害的防护措施。包括呼吸道防护、皮肤防护与个人消毒、急救及侦检措施。

# 3.4.2

个人防护器材 individual protective equipment

个人用于免受毒剂伤害的各种器材的总称。通常包括防毒面具、防毒服、防毒斗篷、防毒手套、防毒靴套、个人消毒急救包和侦毒纸等。

# 3.4.3

防毒面具 protective mask

保护呼吸器官、眼睛和面部免受毒剂伤害的个人防护器材。按原理分为过滤式与隔绝式两类。

# 3.4.4

防毒服 chemical protective clothing

防止毒剂通过皮肤引起伤害等的服装。按原理分为隔绝式与透气式两类。

# 3.4.5

滤毒通风 gas filtration and ventilation

净化染毒空气的通风方式。通常指把外界染毒的空气净化后再送入建筑物或车辆等内部，在销毁日本遗弃在华化学武器过程中，还包括将密闭处理设施内的染毒空气净化后再向外界排放的通风

方式。

3.4.6

滤毒通风装置 filtration and ventilation unit

安装于密闭式设施，用以净化空气的防护器材。一般由过滤材料、吸附材料、离心式风机、密闭式阀门、通风管道、排气筒等设备构成。

3.4.7

负压滤毒通风 filtration and ventilation under negative pressure

为防止毒剂及其有毒相关物扩散，在负压环境（即低于 $101.3\mathrm{kPa}$ ）下对染毒空气所采取的一种净化通风方式。

3.4.8

换气次数 frequency of air exchange

单位时间内（如每小时）进入某固定空间（如房间等）的清洁空气体积与该空间的容积之比。

3.4.9

洗消 decontamination

破坏及除去染毒对象上毒剂及其有毒相关物的措施。

3.4.10

洗消剂 chemical decontaminant

用于破坏毒剂及其有毒相关物毒性的化学物质。

3.4.11

洗消站 decontamination station

对染毒对象实施全部洗消的综合场所。

3.4.12

洗消点 decontamination point

对染毒对象实施洗消的地点。

3.5 环境保护

3.5.1

环境风险 environmental risk

由销毁日本遗弃在华化学武器全过程的各个环节引起的，通过环境介质传播能对人类社会及自然环境可能发生危害的概率及其后果。

3.5.2

环境风险评价 environmental risk assessment

对销毁日本遗弃在华化学武器全过程的各个环节所引发的或面临的危害（包括自然灾害）所造成的环境风险及其可能带来的损失进行评估，提出减小环境风险的方案和对策，并据此进行管理和决策的过程。环境风险评价一般包括健康风险评价和生态风险评价。

3.5.3

健康风险评价 health risk assessment

利用现有的毒理学、流行病学及实验研究等最新成果，按一定的准则，对销毁日本遗弃在华化学武器全过程中各种有害环境因素作用于特定人群的有害健康效应（伤、残、病、出生缺陷、死亡等）进行综合定性与定量评价。

3.5.4

生态风险评价 ecological risk assessment

对销毁日本遗弃在华化学武器全过程中各种有害物质作用于生态环境（如水、气、土壤）的负效应进行综合定性和定量评价的过程。

# 3.5.5

环境风险度 environmental risk degree

不期望环境污染事件发生的频率，以单位时间内污染事件发生的次数表示。

# 3.5.6

风险管理 risk management

根据风险评价的结果，结合各种技术、经济、社会及其他有关因素对销毁日本遗弃在华化学武器全过程中各种风险进行管理并采取相应控制措施的过程。

# 3.5.7

应急计划 emergency plan

依据销毁日本遗弃在华化学武器全过程中的环境风险和健康风险的评价结果，针对可能产生的突发事件而采取的快速处理或救援的一套完整方案。

# 3.5.8

公众参与 public participation

项目方或环评工作组同公众之间的一种双向交流，其目的是使项目能被公众充分认可并提高项目的环境、社会和经济效益。

# 3.5.9

公众告知 public information

通过一定的途径让项目所在地的公众或更大范围的公众知道、了解项目的过程，其目的是获得公众的监督和支持。

# 3.5.10

生态恢复 ecological recovery

通过人工或自然的方法，使在销毁日本遗弃在华化学武器全过程中遭到破坏的当地生态功能得以复原。

# 3.5.11

后期管理 later period management

日本遗弃在华化学武器的销毁工作完成后对与销毁日本遗弃在华化学武器相关的场所和固体废物处理处置的设施继续进行监控的过程。

# 3.5.12

毒剂监测 chemical agent monitoring

间断或连续地测定销毁日本遗弃在华化学武器设施、作业区及周围环境中可能的毒剂及其有毒相关物的浓度，观察、分析其变化和对作业区及周围环境影响的过程。

# 3.5.13

毒剂监测网 chemical agent monitoring network

为监测毒剂及其相关污染物并报知可能的污染信息而建立的监测信息传递体系。

# 3.5.14

毒剂报警器 chemical agent alarm

自动监测空气是否染毒，当染毒空气浓度达到规定的限值时，发出声、光报警信号的化学侦察器材。

# 3.5.15

（毒剂报警器）误报 false alarm

毒剂报警器处于工作状态，在没有染毒空气的情况下，出现报警信号的现象。

# 3.5.16

（毒剂报警器）误动 malfunction

毒剂报警器处于工作状态，机电结构件出现不按要求的动作。

# 3.5.17

（毒剂报警器）不报 inability to alarm

在规定的染毒空气浓度范围和报警响应时间内，毒剂报警器不能发出报警信号的现象。

# 3.5.18

毒剂侦检 chemical agent detection

为确定目标是否染毒，初步辨别毒剂种类，概略测定染毒程度而采取的手段。

# 3.5.19

侦毒器 chemical agent detector kit

以抽气装置、侦毒管、侦毒纸等显色为特征的化学侦察器材。

# 3.5.20

侦毒管 detector tube

与毒剂发生化学或生化反应，从显示颜色变化，判明毒剂种类和概略浓度的侦毒器。一般是装有载体和试剂安瓶、两端封口的玻璃管。
