# 《海淀分区规划（国土空间规划）（2017年-2035年）》成果予以公布

日期:2019年12月11日 11:25 来源:规自委海淀分局

字号：大中小分享:

为贯彻落实《北京城市总体规划（2016年-2035年）》，按照市委、市政府统一要求，海淀区会同市自然资源主管部门组织编制了《海分区规划（国土空间规划）（2017年-2035年）》（以下简称“海淀分区规划”）。

在我市空间规划体系中，分区规划属区级层面的国土空间总体规划，是对《北京城市总体规划（2016年-2035年）》确定的目标、指标、任务在区级层面进行的分解和落实，是下一层级规划编制的重要依据。具体项目建设和设施落实将结合海淀区发展建设实际情况，随下一层控制性详细规划、村庄规划等规划编制和规划实施进一步细化落实。

海淀分区规划已于2019年11月20日获市政府正式批复。根据政府信息公开相关规定，现将海淀分区规划成果予以公布。我们将自觉接受会各界的监督，广泛听取各方面的意见、建议，期盼与社会各界一道推动分区规划的后续实施，为建设国际一流和谐宜居之都贡献海淀力量。
