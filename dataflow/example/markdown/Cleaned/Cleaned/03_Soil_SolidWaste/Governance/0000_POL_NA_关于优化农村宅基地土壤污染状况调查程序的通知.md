# 福建省生态环境厅福建省自然资源厅福建省农业农村厅关于优化农村宅基地土壤污染状况调查程序的通知

各设区市生态环境局、自然资源主管部门、农业农村局，平潭综合实验区自然资源与生态环境局、农业农村局：

为贯彻落实《中华人民共和国土壤污染防治法》《福建省土壤污染防治条例》等相关要求，在有效保障人居环境安全的前提下，我们对相关调查程序进行调整和简化，进一步优化农村

宅基地土壤污染状况调查流程，分类施策，切实减轻基层负担，请各地结合实际认真贯彻落实。

# 一、适用对象

相关简化调查适用于省内农用地、农村未利用地转为农村宅基地（0703）的土壤污染状况调查。农用地、农村未利用地转为农村社区服务设施用地（0704）的可参照执行。

# 二、不适用情形

存在以下情形的，不得简化调查程序：

（一）历史上曾涉及工矿用途、规模化养殖、有毒有害物质储存与输送；
（二）周边有明显的污染源对其产生影响（如：1公里范围涉及有色金属矿采选、有色金属冶炼、石油开采、石油加工、化工、焦化、电镀、制革等土壤重点行业企业）；
（三）曾经使用工业、医疗废水等污水灌溉；
（四）历史上曾涉及环境污染事故、危险废物堆放和处置、固废堆放与倾倒、固废填埋，曾经开展过污泥堆肥活动等；
（五）已有监测数据表明土壤及地下水存在污染，或属于严格管控类耕地；

（六）法律、法规、规章和标准规定的应适用非简化调查程序的其他情形。

# 三、相关要求

（一）农用地、农村未利用地转为农村宅基地（0703），可由土地使用权人或村（居）民委员会填写《福建省农村宅基地土壤污染状况简易调查表》（参考附件）开展土壤污染状况调查，经乡镇人民政府报生态环境部门、自然资源和农业农村部门备案。生态环境部门、自然资源和农业农村部门可组织现场核实或抽查，不再专门组织评审。
（二）各地可采取区域统一调查的方式进一步简化优化农用地、农村未利用地转为农村宅基地（0703）、农村公共管理和公共服务用地（08）土壤污染状况调查。
（三）在建设施工过程，一旦发现土壤或地下水的异常，应立即停止相关作业，及时报告所在地人民政府及生态环境主管部门，生态环境主管部门应及时指导采取有效措施处理，确保生态环境安全。
（四）各地可结合本地实际情况出台农村宅基地土壤污染状况简易调查的方案或实施细则。
