# 一、动态更新名录

各地按照《土壤法》的要求，工企分布情应对辖区内矿业况企型、模大小、污染物排放、生情等业类规产况进行全面排查，根据布局化、土壤境量据以及日常管等情产业变环质监测数监况，结点业业调合重行企用地查成果，动态区更新本地重管点监单录对位名并外公布，确保做到全面覆、不缺不漏。盖

# 二、压实主体责任

各地要及告知名企依法履行的，督促名时录内业应当义务录内业实严质态环企落 “ 格控制有毒有害物排放，并按年度向生境主管部告排放情门报况;开隐证续展土壤污染患排查，保持有效防止有毒有害物漏、流失、散质渗扬 ;制定、施自行方案实监测，并据生境主管部 ”等防控措施并以上容将监测数报态环门将内纳许证入排污可。

各地要督促重管位在拆除施、或者建筑物、构点监单设设备筑物前，制定包括急措施在的土壤污染防治工作方案，并在应内拆除活前十五日所在地生境、工和信息化主管部动报态环业门

备时点监单应时将现储储案。同，重管位及有地下罐存有毒有害物的情送地生境部行案，并填容的质况报属态环门进备对报内真性、全面性、完整性，新、改、建目涉及有毒有害物实负责扩项质储应项产报地下罐的，在目投入生或使用之前送。

# 三、强化日常监管

各地要按照《土壤法》《工用地土壤境管理法（矿环办试行）》等法律法要求，加大规对重管位点监单的管力度，重监对点监单边环开监监测对发现管位周土壤境定期展督性，于有土壤与地下水污染象的，督促迹重管位点监单在患排查的基隐础上，采取性污染防治措施防止污染物一步散。同，在针对进扩时重管位生用地的用途更或者在其土地使用收点监单产经营变权回、前，要督促土地使用人按照定行土壤污染转让权规进状况调查，土壤污染查告作不登料送交地方状况调报应当为动产记资人民政府不登机构，同步送地方人民政府生境主动产记报态环管部案并上污染地管理系。门备传块统
