# 三、工作内容与方法

# (一)典型行企及周土壤污染查业业边状况调

1.调查目的

初步了解3个业业产经营动对业典型行企生活企用地土壤(包括地下水)及周用地土对边农壤生境的影态环响,为针对强关业态环撑性化相行土壤与地下水生境管理提供支。

2.调对查象确定

根据生境部提供的典型行建清态环业议单,结业合我省行分布特征,综许证合排污可管理、二污普等相据关数 ,考生年限、模大、污量大、涉及有毒有害污染物排放的企虑产长规产业,共选发电车定水泥制造、火力、汽零部件制造 3个业典型行 ,45家企入本次查业纳调 ,调查名单见附表1。

3.基信息查础调

针对业业块典型行企地 ,参点业业调术规照《重行企用地查信息采集技定(试行)》(环办土壤

〔2017〕67号),收集企地基本情、污染源、移途、敏感受体、已有查业块况迁径调监测结础果等基信息,填查表。报调

# 4.点设位布

调点调业块点查位包括查企地土壤和地下水位(含地界外块边 ),以及企周影范业边响区围内的用地表和深土壤位。农层层点

企地部：原上不少于业块内则选择 2个点区布域,每布域布不少于个点区设 2个点土壤位和不少于1个点地下水位;必要可在企地界外时业块边 500 m范、污染物移的下游方围内迁向(原上就近则 )布设1个个点或多地下水位。

企周影业边响区:依据具体企的污染排业产 (包括大宗固堆存、大宗化工原料存等废储 )特征、污染散律、用地分布等扩规农 ,确定企周影范业边响区围(原上优先考企周则虑业边3公里范的域围内区 ),并合考用地利用方式、地形地貌等因素综虑农 ,划分用地查元。在农调单调查元单内,按网格法或以企中心按放射布表土壤查位。原上业为状设层调点则 ,影范小于响围1公里时,调单内点数为查元位量 4-9个;影范大于响围 1公里时,调单内点数为查元位量 7-14 个层调点层调点数。深土壤查位按照表土壤查位量 1/4 的比例布设,每查元不个调单内少于1个。

具体按照《重行企用地查点业业调块点术规疑似污染地布技定(试行)》(环办土壤〔2017〕67 号)和《用地土壤污染查位布技定》农状况详点设术规 (环办土壤函〔2017〕1021 号)编点制布方案。

# 5.样品采集

企周用地表土壤品目包括《土壤境量业边农层样检测项应环质农风险标用地土壤污染管控准(试行)》(GB 15618-2018)(以下简称 GB 15618-2018)中的 8 项项基本目、pH,以及企地特征污染物业块 (依据基信息查果确定础调结 ,重点关检测过注和易通大气沉降、灌溉水等途移和径迁积环烃烃挥发累的多芳、石油、持久性有机污染物等半性有机污染物,视污染情况卤烃挥发增加代、苯系物等性有机物)。深土壤品目包括层样检测项应 GB 15618-2018中的8项项基本目和pH。

# (二)中材集中种植土壤污染查药区状况调

# 1.调查目的

初步了解我省天麻、艾、菊花叶 3 种典型中材集中种植土壤重药区属状况金污染。

# 2.调区查域确定

以食药属叶同源、使用量大且容易富集重金的天麻、艾、菊花 3种典型中材品种重药为点,针对区开状况调选择农详围内规较主要种植域展土壤污染查。优先不在用地查范、模大、周存在工污染源或于边业处区高背景的种植基地(原上种植面不少于则积 50亩),作查为调区域。

# 3.点设位布

调点点药协调点查位土壤位和中材同查位。

根据查大小调区 ,每查域布个调区设3-20个点土壤位,每50亩过不超 3个点则土壤位。原上按网格法布设,可根据查域土壤分布情调区内况调酌情整,宜选对在地形相平坦的地方。

在每土壤位个点 ,对药样进协调中材品行同查。

# 4.样品采集

一般药区样样为材种植的土壤品采深度 0-20 cm,果林材种植的土壤品采深类药区样样为度0-60 cm。土壤采方法考《用地土壤品采集样参农样转备术规流制和保存技定》(环办土壤〔2017〕59号),填采。写样记录单
