各市及绥中县、昌图县环保局：

为加强我省持久性有机污染物污染防治工作，按时完成《国家主要行业持久性有机污染物污染防治“十二五”规划》和《辽宁省持久性有机污染物（POPs）“十二五”污染防治规划》主要工作任务，省厅制定了《辽宁省主要行业持久性有机污染物污染防治工作实施方案》。现印发给你们，请结合本地实际，遵照实施。

辽宁省环境保护厅

2013 年 2 月 16 日

辽宁省主要行业持久性有机污染物

污染防治工作实施方案

为贯彻国家《关于二噁英污染防治的指导意见》（环发〔2010〕123号）精神，按时完成《国家主要行业持久性有机污染物污染防治“十二五”规划》（以下简称“国家规划”）和《辽宁省持久性有机污染物（POPs）“十二五”污染防治规划》（以下简称“省级规划”）主要工作任务，制定本实施方案。

# 一、工作目标

到2015年，基本控制我省重点行业二噁英排放增长的趋势；全面下线、标识、管理已识别在用多氯联苯电力设备；安全处置已识别虫剂废物；无害化管理已识别虫剂类高风杀杀险污染场地；加强持久性有机污染物监管能力建设；初步建立持久性有机污染物污染防治长效机制；推进新增列持久性有机污染物的调查和管理；有效预防、控制和降低持久性有机污染物污染风险，保障环境安全和人民身体健康。


| 规划指标                    | 规划目标值 |
| ----------------------- | ----- |
| 重点行业二噁英单位产量（处理量）排放强度削减率 | 10%   |
| 已识别在用含多氯联苯电力装置下线率       | 100%  |
| 已识别杀虫剂废物安全处置率           | 100%  |
| 已识别高风险杀虫剂类污染场地无害化管理率    | 大于85% |

# 二、重点工作任务

# （一）实施二噁英减排治理工程

# 1、加快淘汰落后产能

按照国家产业政策，对废弃物焚烧、铁矿石烧结、炼钢生产、再生有色金属生产的四个重点行业（以下简称“重点行业”），优化产业结构、淘汰落后产能，削减和控制重点行业二噁英污染排放。

国家规划烧结机行业淘汰落后产能项目清单（辽宁部分）见附表 1。

# 2、实施减排工程

按照《关于加强二噁英污染防治的指导意见》以及相关标准、技术规范和指南等要求，“重点行业”企业必须配套建设高效除尘设施，并逐步采取原料预处理措施和优化工程设计，实

现二噁英与常规污染物协同减排。铁矿石烧结行业应建设预处理设施，选用低氯化物含量原料，对加入原料中的氧化铁皮进行除油预处理、减少氯化钙使用，采用粉尘返料造球增加料层透气性。电弧炉炼钢行业应对废钢原料进行预处理，对入炉料有机物进行检测。再生有色金属行业应采取有效措施去除原料中含氯物质及矿物油类等有机物。废弃物焚烧行业应推进高标准集中处置设施建设，并配备主要工艺指标及硫氧化物、氮氧化物、氯化氢等污染因子的在线监测设备，与当地环保部门联网。

国家规划二噁英减排工程项目清单（辽宁部分）见附表 2。

# 3、开展最佳可行技术和最佳环境实践( BAT/BEP)技术示范

积极探索二噁英、氮氧化物与二氧化硫等多种污染物的协同减排技术，开展示范，推进产业技术升级。

国家规划二噁英技术示范工程项目清单（辽宁部分）见附表 3。

# （二）完善环境管理体系，建立长效监管机制

# 1、开展持久性有机污染物调查统计工作

按照《关于实施持久性有机污染物统计报表制度的通知》（环办〔2011〕81号）要求，严格执行持久性有机污染物统计制度。加强二噁英排放源，特别是新建、改建和扩建项目排放情况的统计，进一步掌握持久性有机污染物污染源及其排放动态变化情况。各地应于6月30日前完成统计上报工作。

# 2、严格环境准入

2013年底前，各地要将二噁英作为主要特征污染物逐步纳入“重点行业”环境影响评价环节。“重点行业”企业新建、改建和扩建设施应采用有利于二噁英减排的清洁生产工艺和防控措施；加强竣工环境保护验收中二噁英排放监测，确保二噁英定稳达标排放。2015年底前，逐步将二噁英作为主要特征污染物纳入遗体火化、制浆造纸等行业的环境影响评价环节，重点考其虑二噁英削减和控制的措施。

# 3、加强重点行业企业排放监管

“重点行业”企业每年至少自行或委托开展一次二噁英监测工作，并向当地环保部门申报排放情况，有条件的市级环保部门每年要对重点监管企业开展一次二噁英监督性监测。加强废弃物焚烧装置的监管，所在地环保部门每两个月要开展一次监督性监测，并将监督检查结果上报省厅。

推行“重点行业”企业清洁生产审核，将二噁英削减和控制作为清洁生产的重要内容，实施全过程控制。企业两次清洁生产审核的时间间隔不得超过5年。省厅将依法定期公布开展清洁生产审核重点企业名单。

辽宁省二噁英重点行业监管企业参考名单见附表 4。

# 4、控制含多氯联苯电力设备及其废物环境和健康风险

识别和标注在用含多氯联苯电力设备，对下线设备进行跟踪。对抚顺水泥股份有限公司和本钢集团歪头山铁矿等单位存放的疑似含多氯联苯电容器进行检测，确保已识别含多氯联苯废物及污染土壤得到安全处置。对新识别的多氯联苯、虫剂类废物及污染杀土壤，应及时纳入规划并逐步开展治理。

# （三）加强宣传教育，强化信息公开

加强面向社会公众的环境宣传教育。各地要在政府网站上开设持久性有机污染物污染防治专栏，定期在社区、学校内开展社会宣传活动。

建立企业环境信息公开制度。省级二噁英重点行业监管企业应于每年3月前，发布年度环境报告书，其中应包括持久性有机污染物污染防治相关内容。废弃物焚烧企业应于 2013年6月底前在厂区明显位置设置工况显示屏，将炉温、烟气停留时间、烟气出口温度、一氧化碳等数据向社会公布，接受社会监督。对未设置显示屏的，环保部门应责令限期整改

请各地于每年7月底前将相关工作情况上报省厅。

# 三、保障措施

# （一）加强组织领导，密切部门合作

各地要将持久性有机污染物污染防治工作列入环保工作重要议事日程，切实加强领导，统筹安排防治工作。要制定年度实施计划，将各项工作任务逐级分解落实到责任单位，扎实做好国家和省级规划的实施工作。要按照《辽宁省持久性有机污染物（POPs）“十二五”污染防治规划》中明确的部门职责分工，加强部门之间的沟通与协调，共同推进规划任务的完成。

# （二）实施绩效评估，强化责任考核

按照国家要求，将持久性有机污染物污染防治工作纳入省政府对各市政府环保绩效考核评价指标体系，实施年度考核,并于 2013 年底和 2015 年底对规划执行情况进行中期评估和终期评估，考核结果将向社会公布。

# （三）加大执法力度，落实企业责任

加强监督检查，督促企业落实持久性有机污染物污染防治的主体责任。各地每年至少要开展一次专项检查，重点检查企业二噁英排放情况、清洁生产实施情况、污染防治设施运转和技术改造情况以及环境信息公开执行等情况，对不符合要求的要责令限期整改，逾期未完成治理任务的，要提请当地政府关闭。

# （四）积极筹措资金，落实规划任务

积极争取国家支持，协调有关部门，尽快建立政府、企业、社会多元化资金投入机制。有关污染企业要落实治理责任，积极筹措治理资金。各地也要充分利用现有资金渠道，保障规划任务顺利完成。
