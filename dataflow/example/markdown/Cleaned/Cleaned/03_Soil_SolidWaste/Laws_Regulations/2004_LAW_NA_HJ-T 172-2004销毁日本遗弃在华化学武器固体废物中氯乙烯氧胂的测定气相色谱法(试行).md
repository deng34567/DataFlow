# 销毁日本遗弃在华化学武器固体废物中氯乙烯氧脾的测定气相色谱法（试行）

Determination of chlorovinylarsinoxide in solid waste for the destruction of chemical weapons abandoned by Japan in China Gas chromatography

警告：氯乙烯氧脾具有糜烂性，操作时应按规定佩带防护器材，保持良好通风。测试后的残渣、残液应进行安全处理。

# 1 范围

本标准规定了用气相色谱法测定固体废物中氯乙烯氧肿的方法。

本标准适用于销毁日本遗弃在华化学武器全过程固体废物中氯乙烯氧肿的监测。若固体废物中含有路易氏剂或氯乙烯亚肿酸，也将同时被测定。

本方法对固体废物的检出限（干基）为 $0.05\mathrm{mg / kg}$ ；对弹（筒）片表面的最低检出染毒密度为 $0.01\mathrm{mg / m^2}$ 。

# 3 原理

对含氯乙烯氧肿（CVAO）、路易氏剂（ $\mathrm{L}_1$ ）和氯乙烯亚肿酸（CVAA）的固体废物进行浸出或擦拭，经2，3-二巯基丙醇进行衍生化，以苯萃取后，用具脉冲火焰光度（硫）检测器的毛细管气相色谱进行测定，根据色谱峰的保留时间定性，外标法定量。反应式如下：

$$
\begin{array}{c} \mathrm {C l C H} = \mathrm {C H A s} = \mathrm {O} + \mathrm {H O C H} _ {2} \mathrm {C H} (\mathrm {S H}) \mathrm {C H} _ {2} \mathrm {S H} \longrightarrow \mathrm {C l C H} = \mathrm {C H A s} \\ \mathrm {C V A O} \end{array} \begin{array}{c} \mathrm {S} \\ \mathrm {S} \\ \mathrm {C H} _ {2} \mathrm {O H} \end{array} + \mathrm {H} _ {2} \mathrm {O}
$$

$$
\begin{array}{c} \mathrm {C l C H} = \mathrm {C H A s C l} _ {2} + \mathrm {H O C H} _ {2} \mathrm {C H} (\mathrm {S H}) \mathrm {C H} _ {2} \mathrm {S H} \longrightarrow \mathrm {C l C H} = \mathrm {C H A s} \\ \mathrm {L} _ {1} \end{array} \begin{array}{c} \mathrm {S} \\ \mathrm {S} \\ \mathrm {C H} _ {2} \mathrm {O H} \end{array} + 2 \mathrm {H C l}
$$

$$
\begin{array}{c} \mathrm {C I C H} = \mathrm {C H A s} (\mathrm {O H}) _ {2} + \mathrm {H O C H} _ {2} \mathrm {C H} (\mathrm {S H}) \mathrm {C H} _ {2} \mathrm {S H} \longrightarrow \mathrm {C I C H} = \mathrm {C H A s} \left. \begin{array}{c} \mathrm {S} \\ \mathrm {S} \end{array} \right. \mathrm {C H} _ {2} \mathrm {O H} + 2 \mathrm {H} _ {2} \mathrm {O} \\ \mathrm {C V A A} \end{array}
$$

# 4 试剂与材料

除非另有说明，在分析中仅使用确认为分析纯的试剂和蒸馏水或去离子水或相当纯度的水。

# 4.1 试剂

4.1.1 2，3-二巯基丙醇：含量不小于 $97.0\%$ ，避光冷藏。

4.1.2 氯乙烯氧肿：含量不少于 $98\%$ ，密封、避光、冷藏。

# 4.2 溶液

4.2.1 2，3-二巯基丙醇乙醇溶液： $1 + 9$ ，将1体积的2，3-二巯基丙醇与9体积的乙醇相混合。
4.2.2 磷酸二氢钾溶液： $5\mathrm{g} / \mathrm{L}$ ，将 $0.5\mathrm{g}$ 磷酸二氢钾溶于水并稀释至 $100\mathrm{ml}$ 。
4.2.3 明矾溶液： $20\mathrm{g/L}$ ，将 $2\mathrm{g}$ 明矾溶于水并稀释至 $100\mathrm{ml}$ 。
4.2.4 醇水溶液： $1 + 1$ ，取1体积乙醇与1体积水混匀。
4.2.5 标准贮备液的制备：称取 $10\mathrm{mg}$ 氯乙烯氧肿（4.1.2），准确到 $0.1\mathrm{mg}$ ，用水溶解，并移至 $100\mathrm{ml}$ 容量瓶中，用水稀释至标线，可在 $4\%$ 下保存 $15\mathrm{d}$ 。
4.2.6 标准工作液的制备：根据检测器的灵敏度及线性要求，用水稀释标准贮备液（4.2.5），配制成7个不同浓度的标准工作液，现配现用。

# 4.3 气体和材料

4.3.1 载气 $(\mathrm{N}_2)$ ：纯度不小于 $99.99\%$ 。
4.3.2 燃烧气 $(\mathrm{H}_2)$ ：纯度不小于 $99.99\%$ 。
4.3.3 助燃气：经过净化的空气。
4.3.4 脱脂棉球：每个重 $0.1\mathrm{g}$ 。

# 5 仪器

5.1 气相色谱仪：具脉冲火焰光度检测器（滤光片S， $394\mathrm{nm}$ ）。
5.2 色谱柱： $5\%$ 二苯基 $-95\%$ 二甲基聚硅氧烷石英毛细管柱： $30\mathrm{m} \times 0.32\mathrm{mm} \times 0.25\mu \mathrm{m}$ （长度 $\times$ 内径 $\times$ 膜厚）。
5.3 水平往返式振荡器。
5.4 分液漏斗： $100\mathrm{ml}$ 。
5.5 离心机： $3000\mathrm{r / min}$
5.6 微量注射器：1， $10\mu \mathrm{l}$
5.7 色谱工作站（或色谱数据处理机）。

# 6样品

# 6.1 样品采集

6.1.1 固体废物残渣的采集：采集样品布点可采用梅花形或直线形等方式均匀布点或根据需要采集。每点采集的销毁残渣，剔除杂质后装入样品瓶中，内外各有一张标签（写明编号、采样地点、销毁残渣名称、深度、筛孔以及采样日期和采样量等信息），做好标识和记录。
6.1.2 弹（筒）片的取样：量取弹（筒）片表面积 $30\mathrm{cm}^2$ （ $5\mathrm{cm} \times 6\mathrm{cm}$ ），用5个浸有少量乙醇（4.2.4）的脱脂棉球依次从外到里擦拭5遍，将棉球置于锥形瓶中，贴好标签（写明编号、采样地点、深度、固体废物名称以及采样日期和采样量等信息），做好标识和记录。

# 6.2 样品保存

采集的样品于 $4\%$ 保存，尽快分析。

# 6.3 试样制备

# 6.3.1 提取和浸出

6.3.1.1 固体废物残渣提取：称取已粉碎的样品 $2\mathrm{g}$ ，准确到 $0.1\mathrm{mg}$ ，置于 $150\mathrm{ml}$ 烧杯中，加入 $20\mathrm{ml}$ 磷酸二氢钾溶液（4.2.2），手摇约 $1\mathrm{min}$ ，加入 $1\mathrm{ml}$ 明矾溶液（4.2.3）轻摇几下，静置 $30\mathrm{min}$ ，小心取出上层清液，再重复提取一次，合并提取的清液供衍生化。
6.3.1.2 弹（筒）片浸出：以 $20\mathrm{ml}$ 醇水溶液（4.2.4）对棉球（6.1.2）进行浸出，重复三次，得到的浸出液供衍生化。
6.3.2 衍生化：将上述提取和浸出的清液分别移入具塞磨口锥形瓶中，加入约 $1\mathrm{ml}$ 浓盐酸，摇匀后，用微量注射器加入 $50\mu \mathrm{l}2$ ，3-二巯基丙醇乙醇液（4.2.1）和 $0.1\mathrm{g}$ 抗坏血酸，振荡 $1\mathrm{min}$ 。室温下静置 $30\mathrm{min}$ ，使反应充分。
6.3.3 萃取：吸取 $5.00\mathrm{ml}$ 苯于上述锥形瓶中，分别用分液漏斗萃取，分出苯层置于磨口试管中，用离心机离心 $10\mathrm{min}$ 。
6.3.4 干燥：在苯层中加入约 $0.3\mathrm{g}$ 无水硫酸钠，干燥 $15\mathrm{min}$ ，得苯层试液供色谱分析。

# 6.4 水分测定

按GB5086.1—1997中4.1.2的规定测定固体废物中的水分。

# 7 分析步骤

# 7.1 色谱条件

7.1.1 进样口温度： $250~\mathrm{^\circ C}$
7.1.2 检测器温度： $250~\mathrm{^\circ C}$ 。
7.1.3 柱温：初始温度 $100^{\circ}\mathrm{C}$ ，保持 $2\mathrm{min}$ ，以 $20^{\circ}\mathrm{C / min}$ 的速度升温到 $175^{\circ}\mathrm{C}$ ，然后以 $4^{\circ}\mathrm{C / min}$ 的速度升温到 $200^{\circ}\mathrm{C}$ ，再在 $1\mathrm{min}$ 内升温到 $250^{\circ}\mathrm{C}$ ，保持 $3\mathrm{min}$ 。
7.1.4 气体流速：

氮气 $1.5\mathrm{ml / min}$

空气1 $17\mathrm{ml / min}$

空气2 $10\mathrm{ml / min}$

氢气 $14\mathrm{ml / min}$

7.1.5 进样方式：不分流进样，不分流时间 $1\mathrm{min}$

7.1.6 进样量： $1\mu \mathrm{l}$

# 7.2 校准

7.2.1 定量方法：外标法。
7.2.2 标准曲线绘制

取 $40\mathrm{ml}$ 不同浓度的标准工作液（4.2.6)，分别按6.3.2、6.3.3、6.3.4制备试样，在7.1规定的条件下测定峰面积，取三次测定的平均值为结果，要求三次测定的峰面积的相对偏差不大于 $10\%$ ，绘制峰面积与浓度关系的标准曲线，并得回归方程式（1)：

$$
\log A _ {i} = a + b \log c _ {i} \tag {1}
$$

式中： $A_{i}$ ——氯乙烯氧肿衍生物的峰面积；

$b$ ——标准曲线斜率；

$c_{i}$ ——试液中氯乙烯氧脾的浓度， $\mathrm{mg / L}$

$a$ ——标准曲线截距。

# 7.2.3 标准曲线校正

每天用中间浓度的氯乙烯氧肿标准溶液校正标准曲线，若误差大于 $10\%$ ，应重新绘制标准曲线。

# 7.3.2 定量分析

# 7.3.2.1 分析

取 $1\mu \mathrm{l}$ 试液 (6.3.4)，在7.1条件下进行色谱分析，用外标法与标准组分比较定量。

# 7.3.2.2 结果计算

7.3.2.2.1 固体废物残渣中氯乙烯氧脾的含量 $X$ 按公式（2）计算：

$$
X = \frac {c _ {i} V _ {1}}{W} \tag {2}
$$

式中： $X$ ——固体废物残渣中氯乙烯氧肿的含量， $\mathrm{mg / kg}$

$c_{i}$ ——试液中氯乙烯氧肿浓度， $\mathrm{mg / L}$ ，由公式（1）计算；

$V_{1}$ ——苯萃取液的体积，L；

$W$ ——固体废物残渣的重量（干基）， $\mathrm{kg}$ 。

7.3.2.2.2 弹（筒）片固体废物的染毒密度 $D$ 按公式（3）计算：

$$
D = \frac {c _ {i} V _ {2}}{S} \tag {3}
$$

式中： $D$ ——弹（筒）片表面氯乙烯氧肿的染毒密度， $\mathrm{mg / m^2}$

$c_{i}$ ——试液中氯乙烯氧脾的浓度， $\mathrm{mg / L}$ ，由公式（1）计算；

$V_{2}$ ——苯萃取液的体积，L；

$S$ ——弹（筒）片的擦拭面积， $\mathrm{m}^2$ 。

# 8 结果表示

# 8.1 定性结果

与标准样品保留时间比较定性。

# 8.2 定量结果

# 8.2.1 含量的表示方法

固体废物残渣中氯乙烯氧肿的含量以 $\mathrm{mg / kg}$ 表示。

弹（筒）片表面氯乙烯氧肿的染毒密度以 $\mathrm{mg / m^2}$ 表示。

# 8.2.2 精密度

氯乙烯氧脾重复测定的相对标准偏差不大于 $10\%$ 。

# 8.2.3 准确度

氯乙烯氧脾测定的相对误差不大于 $20\%$ 。

# 8.2.4 检出限

在二倍信噪比时固体废物残渣中氯乙烯氧脾的检出限为 $0.05\mathrm{mg / kg}$

弹（筒）片的最低检出染毒密度为 $0.01\mathrm{mg / m^2}$
