# 1 范围

本标准规定了固体废物中氧联双二苯肿的测定方法。

本方法适用于销毁日本遗弃在华化学武器全过程固体废物中氧联双二苯肿的监测。

本方法对固体废物的检出限（干基）为 $0.02\mathrm{mg / kg}$ ，对弹（筒）片表面的最低检出染毒密度为 $0.1\mathrm{mg / m^2}$ 。

# 3 原理

固体废物样品用二氯甲烷进行超声波萃取，不适合用溶剂萃取的弹（筒）片用浸有二氯甲烷的脱脂棉球擦拭并经二氯甲烷萃取，经浓缩富集，用具有氢火焰离子化检测器的气相色谱仪进行测定，以保留时间定性，外标法定量。

# 4 试剂与材料

# 4.1 载气和辅助气体

4.1.1 载气：氮气，纯度 $99.999\%$

4.1.2 燃气和助燃气：氢气（纯度 $99.99\%$ ）和净化后的空气。

# 4.2 试剂

4.2.1 色谱标准样品：氧联双二苯肿（含量不小于 $97\%$ ）。

4.2.2 二氯甲烷：分析纯。

4.2.3 乙酸乙酯：分析纯。

# 4.3 溶液

4.3.1 标准贮备溶液的制备：称取 $0.1\mathrm{g}$ 氧联双二苯肿（4.2.1）标准样品，准确至 $0.1\mathrm{mg}$ ，溶于乙

酸乙酯（4.2.3）中，在 $100\mathrm{ml}$ 容量瓶中定容， $4\%$ 下储存，可保存 $15\mathrm{d}$ 。

4.3.2 标准工作溶液的制备：用移液管分取不同体积贮备液（4.3.1）置于 $10\mathrm{ml}$ 容量瓶中，用乙酸乙酯（4.2.3）定容，配制成浓度为 $5\sim 100\mu \mathrm{g / ml}$ 的标准工作溶液（至少5种浓度），标准工作溶液须现用现配。

# 4.4 脱脂棉球

每个干棉球重 $0.1\mathrm{g}$ 。

# 5 仪器

5.1 仪器：具氢火焰离子化检测器的气相色谱仪。
5.2 记录仪、积分仪或色谱工作站。
5.3 色谱柱：5%-苯基-95%-甲基聚硅氧烷石英毛细管柱： $30\mathrm{m}\times 0.32\mathrm{mm}\times 0.25\mu \mathrm{m}$ （长度 $\times$ 内径 $\times$ 液膜厚度）。
5.4 试样预处理时使用的主要仪器

5.4.1 超声波清洗仪： $100\mathrm{W}$ 。
5.4.2 离心机： $3000\mathrm{r / min}$
5.4.3 旋转蒸发仪。

# 6样品

# 6.1 样品的采集

6.1.1 固体废物残渣的采集：采集样品布点可采用梅花形或直线形等方式均匀布点或根据需要采集。每点采集的销毁残渣，剔除杂质后装入样品瓶中，内外各有一张标签（写明编号、采样地点、销毁残渣名称、采样深度、筛孔以及采样日期和采样量等信息），做好标识和记录。
6.1.2 弹（筒）片的取样：量取弹（筒）片表面积 $30\mathrm{cm}^2$ （ $5\mathrm{cm} \times 6\mathrm{cm}$ ），用5个浸有少量二氯甲烷的脱脂棉球依次从外到里擦拭5遍，将棉球置于锥形瓶中，贴好标签（写明编号、采样地点、固体废物名称、采样深度及采样日期和采样量等信息），做好标识和记录。

# 6.2 样品的保存

采集的样品于 $4\mathrm{C}$ 保存，尽快分析。

# 6.3 分析试样的制备

# 6.3.1 固体废物残渣处理

称取经风干粉碎的固体废物样品 $10\mathrm{g}$ （准确到 $1\mathrm{mg}$ ）于 $25\mathrm{ml}$ 玻璃试管中，加入 $10\mathrm{ml}$ 二氯甲烷(4.2.2)，超声波萃取 $10\mathrm{min}$ ，离心 $5\mathrm{min}$ ，取出上层清液，再加入 $10\mathrm{ml}$ 二氯甲烷（4.2.2)，超声萃取 $10\mathrm{min}$ ，抽滤，滤渣用 $10\mathrm{ml}$ 二氯甲烷（4.2.2）洗涤，合并上述提取液、滤液及洗液，室温下旋转蒸发至近干，用 $5\mathrm{ml}$ 乙酸乙酯（4.2.3）润洗烧瓶，洗液转移至 $5\mathrm{ml}$ 刻度离心试管中， $45\%$ 水浴中吹氮气蒸发浓缩至 $0.20\mathrm{ml}$ ，得试样供气相色谱分析。

# 6.3.2 弹（筒）片处理

将来自6.1.2的棉球用 $20\mathrm{ml}$ 二氯甲烷（4.2.2）浸泡，超声波提取 $10\mathrm{min}$ ，挤出二氯甲烷，其余旋转蒸发、浓缩等操作同6.3.1。

# 6.4 水分测定

按GB5086.1—1997中4.1.2的规定测定固体废物中的水分。

# 7 分析步骤

# 7.1 色谱条件

7.1.1 进样口温度： $250\%$ 。

7.1.2 柱温度： $140\%$ 保持 $2\mathrm{min}$ 后以 $10\% / \mathrm{min}$ 升温到 $250\%$ ，保持 $2\mathrm{min}$ 。
7.1.3 检测器温度： $260^{\circ}\mathrm{C}$
7.1.4 进样方式：不分流进样，不分流时间 $1\mathrm{min}$
7.1.5 进样量： $1.0\mu \mathrm{l}$

# 7.2 校准

# 7.2.1 标准曲线绘制

取5个不同浓度的标准工作溶液（4.3.2)，在7.1规定的条件下测定峰面积，取重复测定三次的平均值作为结果（三次测定峰面积的相对标准偏差不大于 $10\%$ )，绘制氧联双二苯肿峰面积与其浓度对应关系的标准曲线，并得回归方程式（1)：

$$
y ^ {\prime} = a + b x \tag {1}
$$

式中： $y$ ——氧联双二苯肿的峰面积；

$a$ ——标准曲线截距；

$x$ ——试液中氧联双二苯肼的浓度， $\mathrm{mg / L}$

$b$ ——标准曲线斜率。

# 7.2.2 标准曲线的校正

每天用中间浓度的标准溶液校正标准曲线，若相对误差大于 $10\%$ ，应重新绘制标准曲线。

# 7.3.2 定量分析

7.3.2.1 分析取 $1\mu \mathrm{l}$ 试液（6.3）在7.1条件下进行气相色谱分析，外标法定量。

# 7.3.2.2 结果计算

# 7.3.2.2.1 固体废物残渣

固体废物残渣中氧联双二苯胂的含量 $c$ 按公式（2）计算：

$$
c = \frac {c _ {1} V _ {1}}{m} \tag {2}
$$

式中： $c$ ——固体废物残渣样品中氧联双二苯胂的含量， $\mathrm{mg / kg}$

$c_{1}$ ——测试液中氧联双二苯脾的浓度， $\mathrm{mg / L}$ ，由公式（1）计算；

$V_{1}$ ——试样的体积，L；

$m$ ——固体废物残渣样品的质量（干基）， $\mathrm{kg}$ 。

# 7.3.2.2.2 弹（筒）片

弹（筒）片氧联双二苯脾的染毒密度 $D$ 按公式（3）计算：

$$
D = \frac {c _ {2} V _ {2}}{s} \tag {3}
$$

式中： $D$ ——弹（筒）片表面氧联双二苯脾的染毒密度， $\mathrm{mg / m^2}$

$c_{2}$ ——测试液中氧联双二苯肼的浓度， $\mathrm{mg / L}$ ，由公式（1）计算；

$V_{2}$ ——试样的体积，L；

$S$ ——弹（筒）片的擦拭面积， $\mathrm{m}^2$ 。

# 8 结果表示

# 8.1 定性结果

根据试样色谱峰的保留时间与标样色谱峰的保留时间作比较以确定氧联双二苯肿的存在。

# 8.2 定量结果

# 8.2.1 含量表示方法

固体废物残渣样品以 $\mathrm{mg / kg}$ 表示，弹（筒）片样品以 $\mathrm{mg / m^2}$ 表示。

# 8.2.2 精密度

相对标准偏差不大于 $15\%$ 。

# 8.2.3 准确度

相对误差不大于 $20\%$ 。

# 8.2.4 检出限

固体废物残渣中氧联双二苯胺的检出限为 $0.02\mathrm{mg / kg}$ ，弹（筒）片的最低检出染毒密度为 $0.1\mathrm{mg / m^2}$ 。
