# 销毁日本遗弃在华化学武器

# 土壤中氰溴甲苯的测定高效液相色谱法（试行）

Determination of bromobenzylcyanide in soil for the destruction of chemical weapons abandoned by Japan in China High performance liquid chromatography

警告：氰溴甲苯是强催泪性物质，对眼睛、皮肤、呼吸道等刺激强烈。操作时应按规定要求佩带防护器材，在通风良好的环境中进行操作；测试后的残渣残液应做妥善的安全处理。

# 1 范围

本标准规定了测定土壤中氰溴甲苯的高效液相色谱方法。

本标准适用于销毁日本遗弃在华化学武器全过程土壤中氰溴甲苯的测定。

本标准的最低检测含量（干基）为 $0.01\mathrm{mg / kg}$

# 3 原理

采用二氯甲烷为提取溶剂、索氏提取法提取土壤中氰溴甲苯。提取液经浓缩后，用高效液相色谱/紫外检测器于波长 $237~\mathrm{nm}$ 处进行测定。

# 4 试剂和材料

4.1 甲醇：HPLC级。
4.2 水：二次蒸馏水，用 $0.45\mu \mathrm{m}$ 的微孔滤膜过滤。
4.3 二氯甲烷：HPLC级。
4.4 无水硫酸钠：分析纯，置于浅盘中在 $400\%$ 加热 $4\mathrm{h}$ 。
4.5 氰溴甲苯：含量不低于 $99.0\%$ 。
4.6 氰溴甲苯标准贮备液 $(1.0\mathrm{mg / ml})$ ：称取 $10\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ ）的氰溴甲苯（4.5），用甲醇溶解，在容量瓶中定容至 $10.0\mathrm{ml}$ 。4℃避光保存。
4.7 标准工作液：先用甲醇将标准贮备液（4.6）稀释成 $100.0\mu \mathrm{g / ml}$ 、 $10.0\mu \mathrm{g / ml}$ 的标准溶液，然后用该溶液配制五个或五个以上浓度的标准工作液。标准工作液浓度的确定应参照样品浓度范围，以样品浓度在曲线中段为宜。 $4\%$ 避光保存。

# 5 仪器

5.1 高效液相色谱仪：具紫外检测器。

5.2 色谱柱：十八烷基硅烷键合硅胶（ODSC18）柱， $150\mathrm{mm}\times \phi 6\mathrm{mm}$
5.3 试样预处理时使用的仪器
5.3.1 索氏提取器：带 $250\mathrm{ml}$ 圆底烧瓶。
5.3.2 K-D浓缩器。
5.3.3 恒温水浴锅。
5.3.4 研磨仪器。
5.3.5 微量注射器： $25\mu \mathrm{l}$

# 6样品

# 6.1 样品采集

选择具有代表性的地段，采集样品布点应因地制宜，可采用梅花法或网格法等方式均匀布点，或根据需要采集。每点采集的土壤，剔除杂质后装入样品瓶。

# 6.2 样品的提取

称取 $10\mathrm{g}$ （精确至 $0.1\mathrm{g}$ ）土壤样品和 $10\mathrm{g}$ 无水硫酸钠（4.4）混合，以玻璃纤维滤筒包好后，再放于索氏提取器（5.3.1）中。在放有 $1\sim 2$ 粒干净沸石的 $250\mathrm{ml}$ 圆底烧瓶中加入 $150\mathrm{ml}$ 二氯甲烷(4.3)，将烧瓶连接在提取器上，在 $55\%$ 恒温水浴中提取 $6\mathrm{h}$ 。提取速度要求达到 $4\sim 6$ 个循环/h。提取结束后，提取液冷却至室温，并转移至K-D浓缩瓶。

# 6.3 样品的浓缩

将盛有提取液的K-D浓缩瓶置于 $50\sim 55\%$ 的水浴上浓缩至约 $1\mathrm{ml}$ 。用微量注射器吸取少量二氯甲烷（4.3）冲洗玻璃接口并定容至 $2.0\mathrm{ml}$ ，待色谱分析用。

# 6.4 水分测定

土壤中水分的测定按照GB7172—87的有关规定执行。

# 7 操作步骤

# 7.1 色谱条件

7.1.1 柱温：室温。
7.1.2 流速： $1.0\mathrm{ml / min}$
7.1.3 流动相（体积分数）： $70\%$ 甲醇 $+30\%$ 水。
7.1.4 检测器：紫外检测器，测定波长 $237\mathrm{nm}$

# 7.2 标准曲线的绘制

7.2.1 以氰溴甲苯的色谱峰面积 $A$ （或峰高 $H$ ）为纵坐标，相应浓度 $c(\mu \mathrm{g} / \mathrm{ml})$ 为横坐标，绘制标准曲线。
7.2.2 每天用浓度居中的标准工作液（其浓度必须大于10倍最低检测浓度）做常规校正，组分响应值变化超过 $15\%$ 时，需重新绘制标准曲线。

# 7.3 测定

# 7.3.1 进样

进样量： $25\mu \mathrm{l}$ 。

7.3.2 空白实验：每批样品或试剂有变动时，都应有相应的现场空白实验。现场空白样品应经历样品制备和测定的所有步骤。

# 7.3.4 定量分析

以外标法进行定量。

1—甲醇；2—氰溴甲苯

土壤中氰溴甲苯的含量以公式（1）计算：

$$
X = \frac {c _ {i} V _ {1}}{m} \tag {1}
$$

式中： $X$ ——土壤样品中氰溴甲苯含量（干基）， $\mathrm{mg / kg}$

$c_{i}$ ——浓缩液中氰溴甲苯的浓度， $\mu \mathrm{g / ml}$

$V_{1}$ ——浓缩液体积，ml；

$m$ ——扣除水分后的土壤样品质量， $\mathbf{g}$ 。

# 8 结果的表示

# 8.1 定性结果

根据标准溶液色谱图保留时间进行样品中氰溴甲苯的鉴定。

# 8.2 定量结果

# 8.2.1 含量的表示方法

按式（1）算出土壤中氰溴甲苯的含量（干基），以 $\mathrm{mg / kg}$ 表示。

# 8.2.2 精密度

结果相对标准偏差不大于 $15\%$ 。

# 8.2.3 准确度

样品加标回收率不小于 $70\%$ 。

# 8.3 检测限

按仪器基线噪声3倍所对应的氰溴甲苯的量作为仪器的检测限，本方法要求仪器的最低检测限为 $1 \times 10^{-9} \mathrm{~g}$ 。
