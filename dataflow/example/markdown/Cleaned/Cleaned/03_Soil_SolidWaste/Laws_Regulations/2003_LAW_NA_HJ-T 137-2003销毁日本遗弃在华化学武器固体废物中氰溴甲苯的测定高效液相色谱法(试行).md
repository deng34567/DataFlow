# 固体废物中氰溴甲苯的测定高效液相色谱法（试行）

Determination of bromobenzylcyanide in solid waste for the destruction of

chemical weapons abandoned by Japan in China

High performance liquid chromatography

警告：氰溴甲苯是强催泪性物质，对眼睛、皮肤、呼吸道等刺激强烈。操作时应按规定要求佩带防护器材，在通风良好的环境中进行操作；测试后的残渣残液应做妥善的安全处理。

# 1 范围

本标准规定了测定污染固体废物中氰溴甲苯的高效液相色谱方法。

本标准适用于销毁日本遗弃在华化学武器全过程产生的固体废物中氰溴甲苯的测定。

本标准用于固体废物残渣的最低检测含量（干基）为 $0.01\mathrm{mg / kg}$

本标准用于弹（筒）片固体废物表面的最低检测染毒密度为 $0.1\mathrm{mg / m^2}$ 。

# 3 原理

对于适合细分的固体废物样品中的氰溴甲苯采用二氯甲烷索氏提取；对于不适合细分的弹（筒）片等大块金属废物以乙醇浸润棉球擦拭采样，棉球样品以甲醇振荡法提取。提取液经浓缩后，用高效液相色谱/紫外检测器于波长 $237~\mathrm{nm}$ 处进行测定。

# 4 试剂和材料

4.1 甲醇：HPLC级。
4.2 水：二次蒸馏水，用 $0.45\mu \mathrm{m}$ 的微孔滤膜过滤。
4.3 二氯甲烷：HPLC级。
4.4 无水硫酸钠：分析纯，置于浅盘中在 $400^{\circ}\mathrm{C}$ 加热 $4\mathrm{h}$ 。
4.5 玻璃棉： $400^{\circ}\mathrm{C}$ 加热 $4\mathrm{h}$
4.6 氰溴甲苯：含量不低于 $99.0\%$
4.7 氰溴甲苯标准贮备液 $(1.0\mathrm{mg / ml})$ ：称取 $10\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ ）的氰溴甲苯（4.6），用甲醇溶解，在容量瓶中定容至 $10.0\mathrm{ml}$ 。 $4^{\circ}\mathrm{C}$ 避光保存。
4.8 标准工作液：先用甲醇将标准贮备液（4.7）稀释成 $100.0\mu \mathrm{g / ml}$ 、 $10.0\mu \mathrm{g / ml}$ 的标准溶液，然后用该溶液配制五个或五个以上浓度的标准工作液。标准工作液浓度的确定应参照样品浓度范围，

以样品浓度在曲线中段为宜。 $4\%$ 避光保存。

4.9 乙醇：分析纯。
4.10 脱脂棉球：每个质量为 $100\mathrm{mg}$ ，并浸渍乙醇 $200\mu \mathrm{l}$ 。
4.11 样品浓缩气：氮气，纯度不小于 $99.99\%$

# 5 仪器

5.1 高效液相色谱仪：具紫外检测器。
5.2 色谱柱：十八烷基硅烷键合硅胶（ODSC18）柱， $150\mathrm{mm}\times \phi 6\mathrm{mm}$
5.3 试样预处理时使用的仪器

5.3.1 索氏提取器：带 $250\mathrm{ml}$ 圆底烧瓶。
5.3.2 K-D浓缩器。
5.3.3 恒温水浴锅。
5.3.4 研磨仪器。
5.3.5 微量注射器： $25\mu \mathrm{l}$
5.3.6 振荡器。
5.3.7 离心机：最高转速 $20000\mathrm{r / min}$

# 6样品

# 6.1 样品采集

# 6.1.1 销毁残渣的采集

可参照HJ/T20中有关规定进行样品的采集。根据需要多点采集代表性固体残渣样品，尽量混匀，各点分取一定数量装入样品瓶中备用。

# 6.1.2 弹（筒）片等废物的采集

取乙醇润湿棉球（4.10）5个，在 $30~\mathrm{cm}^2$ 弹（筒）片等固体的表面由外到内分别擦拭一遍，将擦拭后的棉球置于 $25\mathrm{ml}$ 具塞锥形瓶中。

# 6.2 样品的贮存

样品采集后，应迅速送回实验室， $4\mathrm{C}$ 避光保存， $7\mathrm{d}$ 内完成分析。

# 6.3 样品的提取

# 6.3.1 销毁残渣的提取

称取 $10\mathrm{g}$ （精确至 $0.1\mathrm{g}$ ）残渣样品和 $10\mathrm{g}$ 无水硫酸钠（4.4）混合，以玻璃纤维滤筒包好后，再放入索氏提取器（5.3.1）中。在放有 $1\sim 2$ 粒干净沸石的 $250\mathrm{ml}$ 圆底烧瓶中加入 $150\mathrm{ml}$ 二氯甲烷(4.3)，将烧瓶连接在提取器上，于 $55\%$ 恒温水浴中提取 $6\mathrm{h}$ 。提取速度要求达到 $4\sim 6$ 个循环/h。提取结束后，提取液冷却至室温，并转移至K-D浓缩瓶。

# 6.3.2 擦拭棉球的提取

擦拭后的每份棉球样品（6.1.2）中加入 $8\mathrm{ml}$ 甲醇（4.1）提取剂，置于振荡器上，调节频率至 $(110\pm 10)$ 次 $/\mathrm{min}$ ，于室温下振荡 $10\mathrm{min}$ 后，尽量取出提取液后将棉球放入离心试管内以 $10000\mathrm{r}/$ $\mathrm{min}$ ，离心 $5\mathrm{min}$ 。合并萃取液和离心液，过滤，并以 $2\mathrm{ml}$ 甲醇（4.1）分三次进行洗涤。滤液收集于具刻度试管中，用高纯氮气轻吹浓缩至 $2\mathrm{ml}$ 。

# 6.4 样品的浓缩

将盛有提取液的K-D浓缩瓶（6.3.1）置于 $50\sim 55\%$ 的水浴上浓缩至约 $1\mathrm{ml}$ 。用微量注射器吸取少量提取液冲洗玻璃接口并定容至 $2.0\mathrm{ml}$ ，待色谱分析用。

# 6.5 水分的测定

固体废物中水分的测定按照GB5086.1—1997的有关规定执行。

# 7 操作步骤

# 7.1 色谱条件

7.1.1 柱温：室温。

7.1.2 流速： $1.0\mathrm{ml / min}$

7.1.3 流动相（体积分数）： $70\%$ 甲醇 $+30\%$ 水。

7.1.4 检测器：紫外检测器，测定波长 $237\mathrm{nm}$

# 7.2 标准曲线的绘制

7.2.1 以氰溴甲苯的色谱峰面积 $A$ （或峰高 $H$ ）为纵坐标，相应浓度 $c(\mu \mathrm{g} / \mathrm{ml})$ 为横坐标，绘制标准曲线。
7.2.2 每天用浓度居中的标准工作液（其浓度必须大于10倍最低检测浓度）做常规校正，组分响应值变化超过 $15\%$ 时，需重新绘制标准曲线。

# 7.3 测定

# 7.3.1 进样

进样量： $25\mu \mathrm{l}$ 。

7.3.2 空白实验：每批样品或试剂有变动时，都应有相应的现场空白实验。现场空白样品应经历样品制备和测定的所有步骤。

# 7.3.4 定量分析

以外标法进行定量。

# 7.3.4.1 销毁残渣中氰溴甲苯的含量

销毁残渣中氰溴甲苯的含量（干基）以公式（1）计算：

$$
X = \frac {c _ {i} V _ {1}}{m} \tag {1}
$$

式中： $X$ ——固体废物中氰溴甲苯含量（干基）， $\mathrm{mg / kg}$

$c_{i}$ ——浓缩液中氰溴甲苯的浓度， $\mu \mathrm{g / ml}$

$V_{1}$ ——浓缩液体积，ml；

$m$ ——扣除水分后的被提取的固体样品质量， $\mathrm{g}$ 。

# 7.3.4.2 弹（筒）片等废物表面上氰溴甲苯的染毒密度

弹（筒）片等废物表面上氰溴甲苯的染毒密度以公式（2）计算：

$$
X = \frac {c _ {i} V _ {1}}{S} \times 0. 0 0 1 \tag {2}
$$

式中： $X$ ——弹（筒）片等废物表面上氰溴甲苯的染毒密度， $\mathrm{mg / m^2}$

$c_{i}$ ——浓缩液中氰溴甲苯的浓度， $\mu \mathrm{g} / \mathrm{ml}$

$V_{1}$ ——浓缩液体积， $\mathrm{ml}$

$S$ ——擦拭弹（筒）片表面积， $\mathrm{m}^2$ 。

# 8 结果的表示

# 8.1 定性结果

根据标准溶液色谱图保留时间进行样品中氰溴甲苯的鉴定。

# 8.2 定量结果

# 8.2.1 含量的表示方法

按式（1）算出销毁残渣中氰溴甲苯的含量（干基），以 $\mathrm{mg / kg}$ 表示。

按式（2）算出弹（筒）片等废物表面氰溴甲苯的染毒密度，以 $\mathrm{mg / m^2}$ 表示。

# 8.2.2 精密度

结果相对标准偏差不大于 $15\%$ 。

# 8.2.3 准确度

样品加标回收率不小于 $70\%$ 。

# 8.3 检测限

按仪器基线噪声3倍所对应的氰溴甲苯的量作为仪器的检测限，本方法要求仪器的最低检测限为 $1 \times 10^{-9} \mathrm{~g}$ 。
