# 销毁日本遗弃在华化学武器

# 土壤中路易氏剂的测定气相色谱法（试行）

Determination of lewisite in soil for the destruction of chemical weapons abandoned by Japan in China Gas chromatography

警告：路易氏剂为剧毒化学品，操作时应按规定要求佩带防护器材，在通风良好的环境中进行操作，测试后的残渣残液应作妥善的安全处理。

# 1 范围

本标准规定了测定土壤中路易氏剂的气相色谱方法。

本标准适用于销毁日本遗弃在华化学武器全过程土壤中路易氏剂的测定，同时也适用于2-氯乙烯氧肿、2-氯乙烯肿酸的测定。

本标准的最低检出含量（干基）A方法（液液萃取-气相色谱法）为 $0.05\mathrm{mg / kg}$ ；B方法（固相微萃取-气相色谱法）为 $0.005\mathrm{mg / kg}$ 。

# 3 方法原理

用抗坏血酸的稀盐酸溶液萃取土壤中的路易氏剂，萃取液经过离心分离、过滤净化和1，3-二巯基丙烷衍生，衍生产物经萃取后，用毛细管气相色谱火焰光度（硫）检测器测定。

# 4 试剂和材料

# 4.1 载气和辅助气体

4.1.1 载气：氦气，纯度不小于 $99.99\%$ 。
4.1.2 燃烧气：氢气，纯度不小于 $99.99\%$ ，硅胶净化。
4.1.3 助燃气：经净化的空气。

# 4.2 试剂

4.2.1 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ，分析纯。

4.2.2 丙酮 $\mathrm{(C_3H_6O)}$ ，分析纯。
4.2.3 硫基化衍生试剂：1,3-二巯基丙烷（ $\mathrm{C_3H_8S_2}$ ，简称PDT)，纯度不低于 $98\%$ ，使用时用丙酮配制成体积分数为0.02的溶液，配制后用高纯氮气保护，于 $4\%$ 避光保存，超过2d后需重新配制。
4.2.4 硅醚化衍生试剂：N,O-双（三甲基硅基）三氟乙酰胺（ $\mathrm{C_8H_{18}F_3NOSi_2}$ ，简称BSTFA），纯度不低于 $98\%$ 。
4.2.5 标准样品
4.2.5.1 路易氏剂标准样品：纯度不低于 $98\%$
4.2.5.2 贮备溶液的配制

称取路易氏剂标准样品（4.2.5.1）约 $10\mathrm{mg}$ ，精确至 $0.1\mathrm{mg}$ ，置于 $10\mathrm{ml}$ 容量瓶中，用丙酮定容，于 $4\mathrm{C}$ 密封保存，保存期为 $30\mathrm{d}$ 。

4.2.5.3 中间溶液的配制

用移液管取 $0.10\mathrm{ml}$ 贮备溶液（4.2.5.2）于 $10\mathrm{ml}$ 容量瓶中，用丙酮定容，配制浓度约为 $10\mu \mathrm{g}/$ ml的中间溶液，于 $4\%$ 保存，保存期为 $14\mathrm{d}$ 。

4.2.5.4 标准溶液的配制（A方法）

分别用移液管移取不同体积的中间溶液（4.2.5.3）置于 $10\mathrm{ml}$ 的容量瓶中，用 $10\mathrm{mmol / L}$ 的稀盐酸定容，配制成浓度范围在 $30\sim 1920\mathrm{ng / ml}$ 的标准溶液（至少五种浓度)。此标准溶液需现用现配。

4.2.5.5 标准溶液的配制（B方法）

分别用移液管移取不同体积的中间溶液（4.2.5.3）置于 $10\mathrm{ml}$ 的容量瓶中，用 $10\mathrm{mmol / L}$ 的稀盐酸定容，配制成浓度范围在 $2.5\sim 100\mathrm{ng / ml}$ 的标准溶液（至少五种浓度)。此标准溶液需现用现配。

4.2.6 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：分析纯， $300\%$ 烘烤 $4\mathrm{h}$ ，备用。
4.2.7 浓盐酸：分析纯，浓度为 $11.9\mathrm{mol / L}$
4.2.8 稀盐酸：浓度为 $10\sim 100\mathrm{mmol / L}$
4.2.9 抗坏血酸 $\mathrm{(C_6H_8O_6)}$ ：分析纯。

# 4.3 材料

微孔滤膜过滤器：水系列，孔径为 $0.45\mu \mathrm{m}$

# 5 仪器

5.1 色谱仪：具火焰光度检测器的气相色谱仪（GC）。
5.2 检测器：火焰光度-硫型（FPD-S），装有 $394\mathrm{nm}$ 蓝色滤光片。

5.3 色谱柱为： $14\%$ 氰丙基苯基， $86\%$ 甲基聚硅氧烷固定液的石英毛细管柱， $30\mathrm{m}\times 0.32\mathrm{mm}\times$ $0.25\mu \mathrm{m}$ （长度 $\times$ 内径 $\times$ 膜厚）。

5.4 试样预处理使用的仪器有：

a）振荡器；
b）超声波清洗器：90W；
c）精密移液器：20、 $200\mu \mathrm{l}$
d）离心机：最高转速不小于 $5000\mathrm{r / min}$
e）离心试管： $10\mathrm{ml}$ 具塞离心试管；
f）固相微萃取装置：萃取纤维为聚丙烯酸酯，膜厚为 $85\mu \mathrm{m}$

# 6样品

# 6.1 样品采集

选择具有代表性的地段，采集样品布点应因地制宜，可采用梅花形或直线形等方式均匀布点或根据需要采集。每点采集的土壤，剔除杂质后装入样品瓶中，内外各有一张标签（写明编号、采样

地点、土壤名称、深度以及采样日期和采样量等信息)，做好标识和记录。

# 6.2 样品保存

样品采集后于 $4\%$ 以下保存，应尽快分析。

# 6.3 样品制备

# 6.3.1 A方法

6.3.1.1 土样的提取及净化：称取约 $10\mathrm{g}$ 土样，精确至 $0.1\mathrm{g}$ ，置于 $50\mathrm{ml}$ 具塞磨口锥形瓶中，加入含有 $0.5\%$ 抗坏血酸、 $100\mathrm{mmol/L}$ 的稀盐酸 $10.0\mathrm{ml}$ ，振荡或超声 $10\mathrm{min}$ 后，用广泛 $\mathrm{pH}$ 值试纸测定 $\mathrm{pH}$ 值，在 $1\mathrm{h}$ 内保持 $\mathrm{pH}$ 值在 $2\sim 3$ 不变，若不在此范围内，用浓盐酸（4.2.7）或稀盐酸（4.2.8）调节，提取液转移至离心试管中离心后，离心清液转移至另一个离心试管中备用。
6.3.1.2 路易氏剂的衍生化：用移液管移取 $4.0\mathrm{ml}$ 路易氏剂-盐酸水溶液或土样提取离心清液(6.3.1.1）于离心试管中，用精密移液器加入 $10\mu \mathrm{l}$ 新配制的体积分数为0.02的PDT-丙酮溶液，振荡 $30\mathrm{s}$ 使体系充分混匀，室温放置 $1\mathrm{min}$ 后，用移液管加入 $1.0\mathrm{ml}$ 正己烷，每隔 $5\mathrm{min}$ 振荡 $30\mathrm{s}$ ，共振荡六次。分层后，将大部分正己烷层转移到另一个新的离心试管中，加入适量的无水硫酸钠进行干燥，用移液管取出 $0.20\mathrm{ml}$ 干燥后的萃取液于 $2\mathrm{ml}$ 气相色谱试样瓶中，用精密移液器加入 $10\mu \mathrm{l}$ BSTFA，充分振荡，放置 $10\mathrm{min}$ 后供气相色谱分析。

# 6.3.2 B方法

6.3.2.1 土样的提取及净化：同6.3.1.1。
6.3.2.2 固相微萃取试样的处理：将离心清液（6.3.2.1）用 $0.45 \mu \mathrm{m}$ 的滤膜过滤后，用移液管取 $2.0 \mathrm{ml}$ 滤液于 $4 \mathrm{ml}$ 固相微萃取瓶中，用精密移液器加入 $10 \mu \mathrm{l}$ 新配制的体积分数为 0.02 的 PDT-丙酮溶液，放入搅拌磁子，用顶端带有聚四氟乙烯隔垫的瓶盖密封，振荡 $30 \mathrm{~s}$ ，置于工作台上，室温放置 $3 \mathrm{~min}$ 后，将 SPME 萃取纤维插入萃取瓶内的水液中，SPME 纤维应完全浸入水液中，萃取针套管的其他部分不应与水相接触，防止水进入萃取针管。萃取 $15 \mathrm{~min}$ 后，将萃取纤维置于 GC 气化室内解吸 $5 \mathrm{~min}$ 后，供色谱分析。

# 6.4 水分测定

水分测定按GB7172—87中规定执行。

# 7 分析步骤

# 7.1 A方法

# 7.1.1 色谱条件

7.1.1.1 进样口温度： $250\%$
7.1.1.2 出口温度： $270\%$
7.1.1.3 检测器温度： $150\%$
7.1.1.4 升温程序：初始温度 $100\%$ ，保持 $1\mathrm{min}$ ，以 $10\% / \mathrm{min}$ 的速度升温至 $270\%$ ，保持 $5\mathrm{min}$ 。
7.1.1.5 载气：柱前压为 $78\mathrm{kPa}$ 。
7.1.1.6 氢气：压力为 $125\mathrm{kPa}$ 。
7.1.1.7 空气：压力为 $60\mathrm{kPa}$ 。应根据所用仪器的特点调节氢气/空气的流量比，以获得最佳检测能力。
7.1.1.8 进样方式：不分流进样，不分流时间为 $60\mathrm{s}$ 。
7.1.1.9 进样量： $2.0\mu \mathrm{l}$

# 7.1.2 仪器检测限

以二倍信噪比时所对应的路易氏剂的质量作为仪器的检测限，本方法要求仪器的最低检测限小于 $2 \times 10^{-10} \mathrm{~g}$ 。

# 7.1.3 校准

# 7.1.3.1 定量方法

外标法。

# 7.1.3.2 标准曲线的绘制

取不同浓度的路易氏剂标准溶液（4.2.5.4）各 $4.0\mathrm{ml}$ ，按6.3.1.2衍生后，在7.1.1的色谱条件下进行分析，测定峰面积，绘制峰面积对数值与浓度对数值对应对数值关系的标准曲线，并进行线性回归，回归方程见公式（1）。

$$
\log A _ {i} = b \log c _ {i} + a \tag {1}
$$

式中： $A_{i}$ ——路易氏剂衍生产物的峰面积；

$b$ ——标准曲线斜率；

$c_{i}$ ——测试液中路易氏剂的浓度， $\mathrm{mg / L}$

$a$ ——标准曲线截距。

# 7.1.3.3 标准曲线的校正

仪器稳定时在每批样品测定前后应用路易氏剂标准溶液各校正一次。若相对误差大于 $5\%$ 时，应重新绘制标准曲线。

# 7.1.4.2 定量分析

# 7.1.4.2.1 分析

取样品提取液按6.3.1.2衍生后，在7.1.1的色谱条件进行分析，一个样品连续进样三次，计算峰面积的平均值 $A_{i}$ 。

# 7.1.4.2.2 计算

土壤中路易氏剂的含量 $c$ 按式（2）计算：

$$
c = c _ {i} V _ {0} / m \tag {2}
$$

式中： $c$ —土壤中路易氏剂的含量（干基）， $\mathrm{mg / kg}$

$c_{i}$ ——测试液中路易氏剂的浓度， $\mathrm{mg / L}$ ，由公式（1）计算；

$V_{0}$ ——提取液的体积，L；

$m$ ——土壤样品的质量（干基）， $\mathrm{kg}$ 。
