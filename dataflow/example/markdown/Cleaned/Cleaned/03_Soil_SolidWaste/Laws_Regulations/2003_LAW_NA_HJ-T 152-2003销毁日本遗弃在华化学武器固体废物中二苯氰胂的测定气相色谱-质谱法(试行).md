# 销毁日本遗弃在华化学武器

# 固体废物中二苯氰脾的测定气相色谱-质谱法（试行）

Determination of diphenylarsine cyanide in solid waste for the destruction of chemical weapons abandoned by Japan in China Gas chromatography-mass spectrometry

警告：二苯氰脾有刺激作用，是剧毒化学品，操作时应按规定要求佩戴防护器材，在通风良好的环境中进行操作，测试后的残渣残液应作妥善的安全处理。

# 1 范围

本标准规定了测定固体废物中二苯氰脾的气相色谱-质谱法。

本标准适用于销毁日本遗弃在华化学武器全过程固体废物中二苯氰肿的测定，同时也适用于二苯氯肿的测定。

本标准用于固体废物销毁残渣的最低检测含量（干基）为 $0.02\mathrm{mg / kg}$

本标准用于弹（筒）片固体废物表面的检测最低染毒密度为 $0.1\mu \mathrm{g} / \mathrm{cm}^2$ 。

# 3 方法原理

固体废物残渣用甲醇萃取、不便用溶剂萃取的弹片等大块金属废物用乙醇浸润棉球擦拭后用甲醇萃取，萃取液经离心、过滤后定容，用1-丁硫醇试剂衍生，衍生化后的溶液加入内标二苯并噻吩溶液后，用毛细管气相色谱-质谱联用选择离子检测方式进行分离和测定。

# 4 试剂和材料

# 4.1 载气和辅助气体

4.1.1 载气：氦气，纯度不小于 $99.999\%$
4.1.2 样品浓缩气：氮气，纯度不小于 $99.999\%$

# 4.2 试剂

4.2.1 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ，分析纯。
4.2.2 乙醇 $\mathrm{(CH_3CH_2OH)}$ ，分析纯。
4.2.3 萃取剂：甲醇（ $\mathrm{CH}_3\mathrm{OH}$ ），分析纯。
4.2.4 内标物：二苯并噻吩 $\mathrm{(C_{12}H_8S)}$ ，纯度不小于 $98\%$ 。

4.2.5 衍生化试剂：1-丁硫醇 $(\mathrm{C_4H_{10}S})$ ，纯度不小于 $98\%$ 。

# 4.3 标准样品

二苯氰肿：纯度不小于 $98\%$ 。

# 4.4 标准溶液配制

# 4.4.1 二苯氰肿贮备溶液配制

称取二苯氰肿标样（4.3）约 $20\mathrm{mg}$ ，精确至 $0.1\mathrm{mg}$ ，置于 $10\mathrm{ml}$ 容量瓶内，用正己烷溶解定容，于 $4\mathrm{C}$ 保存，保存期为 $90\mathrm{d}$ 。

# 4.4.2 二苯氰肿中间溶液配制

准确移取 $0.50\mathrm{ml}$ 二苯氰肿贮备溶液（4.4.1）于 $10\mathrm{ml}$ 容量瓶中，用甲醇定容。该中间溶液浓度约为 $100\mu \mathrm{g / ml}$ ，现用现配。

# 4.4.3 二苯氰肿标准溶液配制

准确移取一定量的二苯氰肿贮备溶液（4.4.1）或中间溶液（4.4.2）于 $10\mathrm{ml}$ 容量瓶中，用甲醇定容，配制成浓度为 $0.5\sim 10\mu \mathrm{g / ml}$ 的二苯氰肿标准溶液。

# 4.4.4 内标贮备溶液配制

称取二苯并噻吩（4.2.4）约 $20\mathrm{mg}$ ，精确至 $0.1\mathrm{mg}$ ，置于 $10\mathrm{ml}$ 容量瓶内，用正己烷溶解定容，于 $4\%$ 保存，保存期为 $90\mathrm{d}$ 。

# 4.4.5 内标标准溶液配制

准确移取一定量的内标贮备溶液（4.4.4）于 $50~\mathrm{ml}$ 容量瓶中，用正己烷定容，配制成浓度为 $80~\mu \mathrm{g / ml}$ 的内标标准溶液，于 $4\%$ 保存，保存期为 $30\mathrm{d}$

# 4.5 材料

4.5.1 微孔滤膜过滤器：有机系列，孔径为 $0.45\mu \mathrm{m}$ 。
4.5.2 一次性注射器： $10\mathrm{ml}$
4.5.3 脱脂棉球：每个质量为 $100\mathrm{mg}$ ，并浸渍乙醇 $100\mu \mathrm{l}$ 。
4.5.4 离心过滤试管： $2\mathrm{ml}$

# 5 仪器

# 5.1 气相色谱-质谱联用仪

质谱仪应具有EI源和选择离子检测功能。

# 5.2 色谱柱

5%二苯基 $95\%$ 甲基硅酮固定相的石英毛细管柱， $30\mathrm{m}\times 0.25\mathrm{mm}\times 0.25\mu \mathrm{m}$ （长 $\times$ 内径 $\times$ 膜厚）。

# 5.3 其他仪器

5.3.1 康氏振荡器。
5.3.2. 离心机：最高转速 $20000\mathrm{r / min}$
5.3.3 超声波清洗器：90W。

# 6样品

# 6.1 样品采集

6.1.1 销毁残渣的采样：采集样品布点可采用梅花形或直线形等方式均匀布点或根据需要采集。将每点采集的残渣，剔除杂质后装入样品瓶中，做好标识和记录，注明编号、采样日期、采样地点、销毁残渣名称及采样量等信息。
6.1.2 弹（筒）片等废物的采样：取乙醇浸润棉球（4.5.3）5个，在弹（筒）片等的表面上取 $30~\mathrm{cm}^2$ ，由外到内分别擦拭一遍，将擦拭后的棉球置于 $25~\mathrm{ml}$ 具塞锥形瓶中，做好标识和记录，注明编号、采样日期、采样地点以及弹（筒）片废物名称、编号和部位等信息。

# 6.2 样品保存

样品采集后于 $4\%$ 以下保存，尽快分析。

# 6.3 试样的制备

# 6.3.1 销毁残渣的萃取及净化

称取残渣 $10\mathrm{g}$ ，精确至 $0.1\mathrm{g}$ ，置于 $50\mathrm{ml}$ 具塞锥形瓶中，加入 $10.0\mathrm{ml}$ 甲醇，放入超声波清洗器（5.3.3）中在室温下振荡 $10\mathrm{min}$ 。静置 $5\mathrm{min}$ ，待微粒沉降。如固体微粒不易沉降，取上层液体于离心过滤试管中，以 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，使其分层。用一次性 $10\mathrm{ml}$ 注射器取上层清液约 $3\mathrm{ml}$ ，用微孔滤膜过滤器（4.5.1）过滤。

# 6.3.2 擦拭棉球的萃取及净化

在每份擦拭棉球样品（6.1.2）中加入 $9\mathrm{ml}$ 甲醇萃取剂，超声 $3\mathrm{min}$ 后，尽量取出萃取液后将棉球放入离心过滤试管（4.5.4）内以 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，合并萃取液和离心液，过滤并定容到 $10\mathrm{ml}$ 。

# 6.3.3 二苯氰脾的衍生

取 $2.0\mathrm{ml}$ 萃取液（6.3.1或6.3.2）或适当浓度的二苯氰肿标准溶液（4.4.3）至 $5\mathrm{ml}$ 具塞试管中，加入 $50\mu \mathrm{l}$ 1-丁硫醇，在 $50\%$ 的恒温水浴中衍生 $30\mathrm{min}$ ，在常温下静置 $10\mathrm{min}$ ，加入 $30\mu \mathrm{l}$ 内标标准溶液（4.4.5），摇匀后供气相色谱-质谱测定。进行固体废物最低含量检出时，不加内标，用微弱的氮气流将衍生液浓缩至 $0.5\mathrm{ml}$ ，供气相色谱-质谱测定。

# 6.4 水分测定

称取与测试样品相同质量的残渣样品，风干后，于 $105\sim 110\mathrm{^\circ C}$ 烘干 $2\mathrm{h}$ ，冷却至室温称重，计算干基样品中水分含量。

# 7 分析步骤

# 7.1 色谱条件

7.1.1 进样口温度： $280^{\circ}\mathrm{C}$
7.1.2 柱温：初始温度 $130\%$ ，保持 $2.5\mathrm{min}$ ，再以 $22\% / \mathrm{min}$ 的速度升温到 $280\%$ ，保持 $3\mathrm{min}$ 。
7.1.3 载气流速： $1\mathrm{ml / min}$
7.1.4 进样方式：不分流进样，不分流时间 $0.7\mathrm{min}$
7.1.5 进样量： $1.0\mu \mathrm{l}$

# 7.2 质谱条件

7.2.1 接口温度： $280~\mathrm{^\circ C}$
7.2.2 离化方式：EI。
7.2.3 离子源温度： $180~\mathrm{^\circ C}$
7.2.4 电子能量： $70\mathrm{eV}$
7.2.5 选择离子： $\mathrm{m / z184}$ ， $\mathrm{m / z261}$
7.2.6 每一选择离子扫描时间： $0.05\mathrm{s}$ 或 $0.1\mathrm{s}$ ，根据仪器设定。

# 7.3 仪器的检测限

以二苯氰脾产生信号响应强度为二倍的噪声计作为仪器的检测限，本方法要求仪器的检测限小于 $1 \times 10^{-10} \mathrm{~g}$ 。

# 7.4 校准

# 7.4.1 调谐仪器

在进行标准曲线测定前，对质谱仪进行调谐、校正。

# 7.4.2 标准曲线绘制

取不同浓度（至少6个点）的二苯氰肿系列标准溶液（4.4.3）按6.3.3进行衍生化并加内标标

准溶液（4.4.5）后，按7.1和7.2的条件测定它们的气相色谱-质谱响应值，每个浓度样品测定三次，取平均值，绘制响应值与浓度对应关系的曲线，并进行线性回归，回归方程见公式（1）：

$$
y = a + b \left(A _ {i} / A _ {s}\right) \tag {1}
$$

式中： $y$ ——试样中二苯氰肿的浓度， $\mu \mathrm{g / ml}$

$b$ ——标准曲线的斜率；

$A_{i}$ 二苯氰脾的峰面积；

$A_{s}$ ——内标物的峰面积；

$a$ ——标准曲线的截距。

# 7.4.3 曲线的校正

每批样品测定前后用单点中间浓度二苯氰肿标准溶液进行校正，若相对误差大于 $15\%$ ，应重新绘制标准曲线。

# 7.5.2 定量分析

# 7.5.2.1 分析

将6.3.3制备的溶液，按7.1和7.2条件测定它们的气相色谱-质谱响应值，每份溶液分析三次，取平均值。

# 7.5.2.2 计算

7.5.2.2.1 销毁残渣中二苯氰肿的含量按公式（2）计算：

$$
c = 1 0 ^ {- 3} \mathrm {y V} / \mathrm {m} \tag {2}
$$

式中： $c$ —销毁残渣中二苯氰肿的含量， $\mathrm{mg / kg}$

$y$ ——按公式（1）计算的试样溶液中二苯氰肿的浓度， $\mu \mathrm{g} / \mathrm{ml}$

$V$ —萃取液的体积， $\mathrm{ml}$

$m$ —销毁残渣的质量（干基）， $\mathrm{kg}$

7.5.2.2.2 弹（筒）片等废物表面的二苯氯肿的含量按公式（3）计算：

$$
c = y V / s \tag {3}
$$

式中： $c$ ——弹（筒）片等废物表面二苯氰肿的染毒密度， $\mu \mathrm{g} / \mathrm{cm}^2$

$y$ ——按公式（1）计算的试样溶液中二苯氰脾的浓度， $\mu \mathrm{g} / \mathrm{ml}$ ；

$V$ ——萃取液的体积，ml；

$s$ ——擦拭弹（筒）片表面积， $\mathrm{cm}^2$ 。

# 8.1 定性结果

按7.5.1表示。

# 8.2 定量结果

# 8.2.1 含量表示

销毁残渣中二苯氰脾的含量按7.5.2.2.1计算，单位为 $\mathrm{mg / kg}$

弹（筒）片等废物表面二苯氰肿的染毒密度按7.5.2.2.2计算，单位为 $\mu \mathrm{g} / \mathrm{cm}^2$ 。

相对标准偏差不大于 $20\%$ 。

# 8.2.3 准确度
