# 1 范围

本标准规定了测定固体废物中芥子气的气相色谱法。

本标准适用于销毁日本遗弃在华化学武器全过程固体废物中芥子气的测定。

本标准测定销毁固体残渣中芥子气的最低检出含量（干基）为 $0.01\mathrm{mg / kg}$ ，弹（筒）片中芥子气最低染毒密度为 $0.2\mu \mathrm{g} / \mathrm{cm}^2$ 。

# 3 方法原理

固体残渣中芥子气采用二氯甲烷萃取，将萃取溶液过滤、浓缩；不适合直接用溶剂萃取的染毒弹（筒）片等大块金属废物，用石油醚湿润棉球擦拭弹片表面，擦拭棉球用二氯甲烷萃取，将萃取溶液过滤、浓缩。用气相色谱火焰光度-硫（FPD-S）检测器测定，外标法定量。

# 4 试剂和材料

# 4.1 载气和辅助气体

4.1.1 载气：高纯氦气，纯度不小于 $99.999\%$
4.1.2 燃烧气：氢气，纯度不小于 $99.99\%$
4.1.3 助燃气：经过净化的空气。
4.1.4 样品浓缩气：氮气，纯度不小于 $99.999\%$

# 4.2 试剂

4.2.1 二氯甲烷（ $\mathrm{CH}_2\mathrm{Cl}_2$ ），分析纯。
4.2.2 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ，分析纯。
4.2.3 石油醚，沸程为 $90\sim 120\%$

# 4.3 样品

芥子气，纯度不小于 $98\%$ 。

# 4.4 标准溶液配制

# 4.4.1 贮备溶液配制

称取芥子气（4.3）约 $21.0\mathrm{mg}$ ，精确至 $0.1\mathrm{mg}$ ，置于 $100\mathrm{ml}$ 容量瓶中，用二氯甲烷定容，配制成芥子气浓度为 $210\mu \mathrm{g / ml}$ 的贮备溶液，于 $4^{\circ}\mathrm{C}$ 下密封保存，保存期为 $60\mathrm{d}$ 。

# 4.4.2 中间溶液配制

移取一定体积的芥子气贮备溶液（4.4.1）于 $10\mathrm{ml}$ 容量瓶中，用二氯甲烷定容，配制成芥子气浓度约为 $40\mu \mathrm{g / ml}$ 的中间溶液，于 $4\%$ 下密封保存，保存期为 $30\mathrm{d}$ 。

# 4.4.3 标准溶液配制

准确移取不同体积的芥子气中间溶液（4.4.2）于 $5\mathrm{ml}$ 容量瓶中，用二氯甲烷定容，配制成芥子气浓度为 $0.50,1.0,2.5,5.0,10.0\mu \mathrm{g / ml}$ 的标准系列溶液，现配现用。

# 4.5 材料

4.5.1 干燥脱脂棉。
4.5.2 石油醚浸渍棉球：每个干棉球 $0.1\mathrm{g}$ ，浸渍石油醚 $0.1\mathrm{ml}$ 。
4.5.3 微孔过滤膜：有机系列，孔径为 $0.45\mu \mathrm{m}$ 。
4.5.4 过滤针筒： $10\mathrm{ml}$
4.5.5 刻度试管：5、 $10\mathrm{ml}$
4.5.6 离心过滤试管： $2\mathrm{ml}$

# 5 仪器

# 5.1 气相色谱仪

带有火焰光度-硫（FPD-S）检测器。

# 5.2 色谱柱

$14\%$ 氰苯基， $86\%$ 甲基聚硅氧烷的石英毛细管色谱柱， $30\mathrm{m}\times 0.32\mathrm{mm}\times 0.25\mu \mathrm{m}$ （长度 $\times$ 内径 $\times$ 膜厚）。

# 5.3 辅助仪器

5.3.1 振荡器：康氏振荡器或旋涡振荡器。
5.3.2 离心机：最高转速不小于 $5000\mathrm{r / min}$

# 6样品

# 6.1 样品采集

# 6.1.1 销毁残渣的采集

采集样品布点可采用梅花形或直线形等方式均匀布点或根据需要采集。将每点采集的销毁残渣，剔除杂质后装入样品瓶中，做好标识和记录，注明编号、采样日期、采样地点、销毁残渣名称以及采样量等信息。

# 6.1.2 弹（筒）片样品采集

在染毒弹（筒）片上标记面积 $30~\mathrm{cm}^2$ ，依次用五个石油醚侵润棉球（4.5.2）擦拭弹片标记区，每个棉球由外向里擦拭一遍，擦拭棉球置于 $10\mathrm{ml}$ 样品瓶中，做好样品标记或记录。标明编号、样品名称、面积、数量和日期等信息，密封备用。

# 6.1.3 样品保存

采集样品后于 $4\%$ 下密封保存，应尽快分析。

# 6.2 样品制备

# 6.2.1 残渣样品制备

称取残渣（6.1.1） $10\mathrm{g}$ ，精确至 $0.1\mathrm{g}$ ，置于 $50\mathrm{ml}$ 具塞锥形瓶中，加入 $10.0\mathrm{ml}$ 二氯甲烷，在室温下振荡萃取 $5\mathrm{min}$ ，静置 $5\mathrm{min}$ 。如萃取液中有悬浮颗粒，以转速为 $1000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。萃取液用微孔过滤膜（4.5.3）针筒（4.5.4）过滤，移取萃取过滤液约 $6.0\mathrm{ml}$ 于刻度试管（4.5.5）中，加

入 $0.2\mathrm{g}$ 无水硫酸钠干燥。

# 6.2.2 萃取液浓缩

准确移取 $5.0\mathrm{ml}$ 萃取过滤液（6.2.1）于刻度试管中，用高纯氮气轻吹，浓缩定容至 $1.0\mathrm{ml}$ 。如果残渣萃取液中芥子气达不到最低检出量，继续浓缩定容至 $0.5\mathrm{ml}$ ，供气相色谱分析。

# 6.3 擦拭棉球样品制备

# 6.3.1 擦拭棉球样品萃取

在装有五个擦拭棉球的样品（6.1.2）瓶中，加入 $5.0\mathrm{ml}$ 二氯甲烷，用瓶盖密封，在室温下振荡萃取 $2\mathrm{min}$ ，静置 $2\mathrm{min}$ 。用吸管取出可取的萃取液后，将萃取棉球放入过滤离心试管中（4.5.6），离心 $2.0\mathrm{min}$ ，将离心液与萃取液合并定容至 $5.0\mathrm{ml}$ 。

# 6.3.2 萃取液浓缩

将 $5.0\mathrm{ml}$ 擦拭棉球萃取液（6.3.1）用高纯氮气轻吹，浓缩定容至 $1.0\mathrm{ml}$ 。如果弹（筒）片萃取液中芥子气达不到最低检出量，继续浓缩定容至 $0.5\mathrm{ml}$ ，供气相色谱分析。

# 6.4 水分测定

称取与样品相同质量的固体残渣，如果含水较多，风干后于 $105\sim 110\mathrm{^\circ C}$ 烘烤 $2\mathrm{h}$ ，冷却至室温，称重残渣（干基)，计算水分含量。

# 7 分析步骤

# 7.1 试验条件

7.1.1 进样口温度： $250~\mathrm{^\circ C}$

7.1.2 出口温度： $250~\mathrm{^\circ C}$

7.1.3 检测器温度： $150~\mathrm{^\circ C}$

7.1.4 柱温：初始温度 $50\%$ ，保持 $1\mathrm{min}$ ， $10\% / \mathrm{min}$ 的速度升温至 $180\%$ ，再以 $20\% / \mathrm{min}$ 的速度升温至 $240\%$ ，保持 $1\mathrm{min}$ 。

7.1.5 进样方式：不分流进样，不分流时间 $0.8\mathrm{min}$

7.1.6 载气压力： $70\mathrm{kPa}$ （恒压）。

7.1.7 氢气流量： $90~\mathrm{ml / min}$

7.1.8 空气流量： $115\mathrm{ml / min}$

7.1.9 进样量： $1.0\mu \mathrm{l}$

# 7.2 校准

# 7.2.1 定量方法

外标法。

# 7.2.2 标准曲线制作

按7.1的试验条件，移取不同浓度的芥子气标准系列溶液（4.4.3），每种浓度的标准溶液色谱连续分析三次，用芥子气平均峰面积的对数值与芥子气浓度的对数值进行一元线性回归，得回归公式(1)。

$$
\lg c = b \lg A - a \tag {1}
$$

式中： $c$ ——芥子气的浓度， $\mathrm{mg / L}$

$A$ ——芥子气的峰面积；

$a$ ——曲线的截距；

$b$ ——曲线的斜率。

# 7.2.3 曲线校正

每批样品检测前，用芥子气标准溶液进行校正，若校正检出值的相对误差大于 $15\%$ ，应重新制作标准曲线。

# 7.3 样品测定

# 7.3.1 定性分析

按试验步骤进行残渣和弹片样品制备分析，检测试样各色谱峰中的保留时间与标准样品色谱峰的保留时间比较，所测组分的保留时间窗口差为 $\pm 10\mathrm{~s~}$ ，可以确定样品中芥子气存在。残渣样品中芥子气检出色谱图见图1。

# 7.3.2 定量分析

# 7.3.2.1 残渣样品分析

7.3.2.1.1 按7.1的试验条件，移取残渣萃取浓缩样品(6.2.2)，每种萃取浓缩样品色谱连续分析三次，用检出芥子气平均峰面积的对数值，按公式（1）计算萃取液中芥子气浓度。

# 7.3.2.1.2 残渣含量计算

残渣样品中芥子气含量按公式（2）计算：

$$
c _ {1} = \frac {V _ {1} c}{n _ {1} m} \tag {2}
$$

式中： $c_{1}$ ——残渣样品中芥子气含量（干基）， $\mathrm{mg / kg}$

$V_{1}$ ——萃取溶剂的体积，L；

$c$ ——萃取溶液中芥子气浓度，由公式（1）得， $\mathrm{mg / L}$ ；

$n_1$ ——萃取溶液的浓缩倍数；

$m$ ——残渣质量（干基）， $\mathrm{kg}$ 。

# 7.3.2.2 弹（筒）片样品分析

7.3.2.2.1 按7.1的试验条件，移取擦拭棉球萃取浓缩样品（6.3.2)，每种萃取浓缩样品色谱连续分析三次，用检出芥子气平均峰面积的对数值，按公式（1）计算萃取液中芥子气浓度。

# 7.3.2.2.2 弹（筒）片染毒密度计算

弹（筒）片中芥子气染毒密度按公式（3）计算：

$$
c _ {2} = \frac {V _ {2} c}{n _ {2} S} \tag {3}
$$

式中： $c_{2}$ ——弹（筒）片染毒密度， $\mu \mathrm{g} / \mathrm{cm}^{2}$

$V_{2}$ ——萃取溶剂体积，L；

$c$ ——萃取溶液中芥子气浓度，由公式（1）得， $\mathrm{mg / L}$

$n_2$ ——萃取溶液的浓缩倍数；

$S$ ——弹（筒）片染毒面积， $\mathrm{cm}^2$ 。

# 8 结果表示

# 8.1 定性结果

检测试样各色谱峰中的保留时间与标准样品色谱峰的保留时间比较，保留时间窗口差为 $\pm 10\mathrm{~s~}$ 可以定性确定固体废物中芥子气存在。

# 8.2 定量结果

# 8.2.1 准确度

相对误差小于 $20\%$ 。

# 8.2.2 精密度

相对标准偏差不大于 $20\%$ 。

# 8.2.3 最低检出含量

残渣中芥子气的最低检出含量（干基）为 $0.01\mathrm{mg / kg}$ 。弹（筒）片表面芥子气的最低染毒密度检测为 $0.2\mu \mathrm{g} / \mathrm{cm}^2$ 。
