# 固体废物苯系物的测定顶空/气相色谱-质谱法

警告：实验中使用的有机溶剂和标准溶液为易挥发有毒物质，试剂配制和前处理过程应在通风柜内进行；操作时应按要求佩戴防护器具，避免接触皮肤和衣物。

# 1 适用范围

本标准规定了测定固体废物及其浸出液中苯系物的顶空/气相色谱-质谱法。

本标准适用于固体废物及其浸出液中苯、甲苯、乙苯、对-二甲苯、间-二甲苯、异丙苯、邻-二甲苯、正丙苯和苯乙烯等 9 种苯系物的测定。其他苯系物若通过验证，也可用本标准方法测定。

当固体废物取样量为 $2 \mathrm { g }$ 时，9 种目标物的方法检出限为 $3 { \sim } 5 ~ \mu \mathrm { g / k g }$ ，测定下限为 $1 2 \sim 2 0$ $\mu \mathrm { g / k g }$ 。当固体废物浸出液取样体积为 $1 0 ~ \mathrm { m l }$ 时，9 种目标物的方法检出限为 $0 . 6 \mathrm { \sim 1 ~ } \mu \mathrm { g / L }$ ，测定下限为 $2 . 4 { \sim } 4 ~ \mu \varrho / \mathrm { L }$ 。详见附录 A。

# 3 方法原理

在一定的温度下，顶空瓶内样品中苯系物向液上空间挥发，产生蒸汽压，达到热力学动态平衡。气相中的苯系物经气相色谱分离后，用质谱仪进行检测。通过与标准物质保留时间和标准质谱图相比较进行定性，内标法定量。

# 4 试剂和材料

除非另有说明，分析时均使用符合国家标准的分析纯试剂。

4.1 实验用水：二次蒸馏水或通过纯水设备制备的水。使用前需经过空白试验，目标物浓度低于方法检出限。
4.2 甲醇（ $\mathrm { C H } _ { 3 } \mathrm { O H }$ ）：农残级或相当级别。通过空白试验，确认在目标物的保留时间区间内无干扰色谱峰出现。
4.3 氯化钠（NaCl）。

在马弗炉中 $4 0 0 \mathrm { { ^ \circ C } }$ 烘烤 $4 \mathrm { { h } }$ ，冷却后转移至磨口玻璃瓶中，置于干燥器中保存。

4.4 磷酸（ $\mathrm { H } _ { 3 } \mathrm { P O } _ { 4 } ,$ ）： $\rho { = } 1 . 6 9 ~ \mathrm { g / m l }$ ，优级纯。
4.5 饱和氯化钠溶液。

量取 $5 0 0 \mathrm { m l }$ 水（4.1），滴加磷酸（4.4）调节 $\mathsf { p H } \leqslant 2$ ，加入 $1 8 0 ~ \mathrm { g }$ 氯化钠（4.3），溶解并混匀，于 $4 \%$ 以下保存。

4.6 标准贮备液： $\rho { = } 1 0 0 0 \ \mathrm { m g / L }$ ，溶剂为甲醇。可直接购买有证标准溶液，也可用标准物质配制。
4.7 标准使用液 1： $\rho { = } 1 0 \ \mathrm { m g / L }$ ，溶剂为甲醇。
4.8 标准使用液 2： $\rho { = } 1 0 0 \ \mathrm { m g / L }$ ，溶剂为甲醇。
4.9 内标标准溶液： $\rho { = } 2 5 0 \ \mathrm { m g / L }$ ，溶剂为甲醇。

选用氟苯作为内标。可直接购买有证标准溶液。

4.10 替代物标准溶液： $\rho { = } 2 0 0 0 \ \mathrm { m g / L }$ ，溶剂为甲醇。

选用甲苯-d8或 4-溴氟苯作为替代物。

4.11 替代物使用液 1： $\rho { = } 1 0 \ \mathrm { m g / L }$ ，溶剂为甲醇。
4.12 替代物使用液 2： $\rho { = } 1 0 0 \ \mathrm { m g / L }$ ，溶剂为甲醇。
4.13 4-溴氟苯（BFB）溶液： $\rho { = } 2 5 ~ \mathrm { m g / L }$ ，溶剂为甲醇。

可直接购买有证标准溶液，也可用高浓度标准溶液配制。

4.14 石英砂： $0 . 3 0 { \sim } 0 . 8 5 ~ \mathrm { m m }$ （50~20 目）。
在马弗炉中 $4 0 0 \mathrm { { ^ \circ C } }$ 灼烧 $4 \mathrm { { h } }$ ，冷却后转移至磨口玻璃瓶中，密封保存。
4.15 载气：高纯氦气（纯度 ${ \geqslant } 9 9 . 9 9 9 \%$ ）。

# 5 仪器和设备

5.1 气相色谱仪：具有毛细管分流/不分流进样口，可程序升温。
5.2 质谱仪：具 $7 0 \mathrm { e V }$ 的电子轰击（EI）电离源，具 NIST 质谱图库、手动/自动调谐、数据采集、定量分析及谱库检索等功能。
5.3 色谱柱：石英毛细管色谱柱， $3 0 \mathrm { m } { \times } 0 . 2 5 \mathrm { m m } { \times } 0 . 2 5 \mathrm { \mu } \mathrm { m }$ （聚乙二醇 20M），也可使用其他等效毛细管色谱柱。

5.5 往复式振荡器：振荡频次 50~250次/分，振荡幅度 $2 0 \mathrm { m m }$ 。
5.6 天平：感量为 $0 . 0 1 \ \mathrm { g }$
5.7 采样瓶：具聚四氟乙烯/硅胶衬垫螺旋盖的 $6 0 \mathrm { m l }$ 或 $2 0 0 ~ \mathrm { m l }$ 的螺纹棕色广口玻璃瓶。
5.8 采样器材：铁铲和不锈钢药勺。
5.9 便携式冷藏箱。
5.10 微量注射器： $\mathrm { : ~ \ 5 \mu l \cdot \ 1 0 \mu l \cdot \ 2 5 \mu l \cdot \ 1 0 0 \mu l \cdot \ 2 5 0 \mu l \cdot \ 5 0 0 \mu l \cdot \ 1 0 0 0 \mu l \circ }$
5.11 棕色密实瓶： $2 ~ \mathrm { m l }$ ，具聚四氟乙烯衬垫和实芯螺旋盖。
5.12 顶空瓶： $2 2 ~ \mathrm { m l }$ ，配有密封垫（聚四氟乙烯/硅氧烷）和瓶盖（螺旋盖或一次使用的压盖）。
5.13 一般实验室常用仪器和设备。

# 6 样品

# 6.1 样品采集和保存

6.1.1 按照 HJ/T 20 和 HJ/T 298的相关规定进行固体废物样品的采集和保存。用铁铲和不锈钢药勺（5.8）迅速将样品采集到采样瓶（5.7）中，并尽量装满。快速清除掉采样瓶螺纹及外表面上粘附的样品，立即密封采样瓶。置于便携式冷藏箱（5.9）内，于 $4 \mathrm { { } ^ { \circ } C }$ 以下保存，带回实验室。

6.1.2 样品送入实验室后应尽快分析。若不能立即分析，应在 $4 \mathrm { { } ^ { \circ } C }$ 以下密封保存，保存期限不超过 $1 4 \textup d$ 。样品存放区域应无目标物干扰。

注：样品采集时切勿过度搅动固体废物，以免造成样品中苯系物的挥发。

# 6.2 试样的制备

# 6.2.1 固体废物试样的制备

# 6.2.1.1 低含量试样

称取 $2 \mathrm { \ g }$ （精确至 $0 . 0 1 \ \mathrm { g }$ ）样品置于顶空瓶（5.12）中，迅速向顶空瓶中加入 $1 0 . 0 \mathrm { m l }$ 饱和氯化钠溶液（4.5）、 $2 \mu \mathrm { l }$ 内标标准溶液（4.9）和 $2 0 \mu \mathrm { l }$ 替代物使用液 1（4.11），立即密封，在往复式振荡器（5.5）上以 150 次/ min 的频率振荡 $1 0 \mathrm { m i n }$ ，待测。

# 6.2.1.2 高含量试样

如果目标物含量大于 $1 0 0 0 ~ \mu \varrho / \ k \mathrm { g }$ 时应视为高含量试样。高含量试样制备如下：取出采样瓶（5.7），称取 $2 \mathrm { g }$ （精确至 $0 . 0 1 { \bf \tau g }$ ）样品置于顶空瓶（5.12）中，迅速向顶空瓶中加入 10.0ml 甲醇（4.2），立即密封，在往复式振荡器（5.5）上以 150 次/ min 的频率振荡 $1 0 \mathrm { m i n }$ 。静置沉降后，移取约 $1 ~ \mathrm { m l }$ 甲醇提取液至 $2 ~ \mathrm { m l }$ 棕色密实瓶（5.11）中。该提取液可置于冷藏箱内 $4 \mathrm { { } ^ { \circ } C }$ 以下保存，保存期为 $1 4 \mathrm { d }$ 。

在分析之前将提取液恢复到室温后，向空的顶空瓶（5.12）中加入 $2 \mathrm { g }$ （精确至 $0 . 0 1 { \mathrm { ~ g } } { \cdot }$ ）石英砂（4.14）、 $1 0 . 0 \mathrm { m l }$ 饱和氯化钠溶液（4.5）、 $1 0 { \sim } 1 0 0 ~ \mu 1$ 甲醇提取液、 $2 \mu \mathrm { l }$ 内标标准溶液（4.9）和 $2 0 \mu \mathrm { l }$ 替代物使用液 1（4.11），立即密封，在往复式振荡器（5.5）上以 150次/ min的频率振荡 $1 0 \mathrm { { m i n } }$ ，待测。

注1：若甲醇提取液中苯系物浓度较高，可用甲醇适当稀释。
注2：若采用高含量方法测定含量值过低或未检出，应采用低含量方法重新分析样品。

# 6.2.2 固体废物浸出液试样的制备

按照 HJ/T 299方法制备固体废物水浸出液试样；按照 HJ/T 300 方法制备固体废物醋酸浸出液试样。移取 $1 0 . 0 \mathrm { m l }$ 浸出液至顶空瓶（5.12）中，加入 $2 \mu \mathrm { l }$ 内标标准溶液（4.9）和 20μl 替代物使用液 1（4.11），立即密封，待测。

# 6.3 空白试样的制备

# 6.3.1 全程序空白试样

采样前在实验室将 $1 0 . 0 \mathrm { m l }$ 饱和氯化钠溶液（4.5）和 $2 \mathrm { \ g }$ （精确至 $0 . 0 1 \ \mathrm { g } \cdot $ ）石英砂（4.14）放入顶空瓶（5.12）中，加入 $2 \mu \mathrm { l }$ 内标标准溶液（4.9）密封，将其带到采样现场后开封，再密封，之后随样品运回实验室，在往复式振荡器（5.5）上以 150次/ min 的频率振荡 $1 0 \mathrm { m i n }$ ，待测。

# 6.3.2 实验室空白试样

# 6.3.2.1 固体废物低含量空白试样

称取 $2 \mathrm { \ : g }$ （精确至 $0 . 0 1 \ \mathrm { g }$ ）石英砂（4.14）代替高含量样品，按照 6.2.1.2 步骤制备高含量空白试样。

# 7 分析步骤

# 7.1 顶空进样器参考条件

加热平衡温度 $8 5 \mathrm { { ^ \circ C } }$ ；加热平衡时间 $5 0 \mathrm { { m i n } }$ ；取样针温度 $1 0 0 ^ { \circ } \mathrm { C }$ ；传输线温度 $1 1 0 ^ { \circ } \mathrm { C }$ ；压力化平衡时间 $1 \mathrm { m i n }$ ；进样时间 $0 . 2 \mathrm { m i n }$ ；拨针时间 $0 . 4 \mathrm { m i n }$ 。

# 7.2 气相色谱仪参考条件

程序升温： $4 0 \mathrm { { ^ \circ C } }$ 保持 $6 \mathrm { { m i n } }$ ，以 $5 \mathrm { { ^ { \circ } C / m i n } }$ 的升温速率升至 $1 1 0 ^ { \circ } \mathrm { C }$ ，保持 $1 \mathrm { m i n }$ ，再以 $6 \mathrm { ^ { \circ } C }$ /min 的升温速率升至 $2 0 0 \mathrm { { ^ \circ C } }$ ，保持 $3 \mathrm { { m i n } }$ ；进样口温度： $2 5 0 \mathrm { { ^ \circ C } }$ ；接口温度： $2 3 0 \mathrm { { ^ \circ C } }$ ；载气：氦气；进样方式：分流进样，分流比：5：1；柱流量： $1 . 0 \mathrm { m l / m i n }$ 。

# 7.3 质谱仪参考条件

扫描范围： $3 5 \mathrm { \ a m u } { \sim } 3 0 0 \mathrm { \ a m u }$ ；每个色谱峰至少扫描 5 次；离子化能量： $7 0 \mathrm { e V }$ ；离子源温度： $2 3 0 \mathrm { { ^ \circ C } }$ ；四级杆温度： $1 5 0 ^ { \circ } \mathrm { C }$ ；扫描方式：选择离子扫描（SIM）。

# 7.4 选择离子扫描（SIM）程序

分别取 $5 \mu \mathrm { l }$ 替代物使用液 2（4.12）、5 μl 标准使用液 2（4.8）和 $2 \mu \mathrm { l }$ 内标标准溶液（4.9）置于装有 $1 0 . 0 \mathrm { m l }$ 饱和氯化钠溶液的顶空瓶（5.12），立即密封。按照仪器参考条件（7.1、7.2和 7.3）进行全扫描（SCAN）分析，根据保留时间确定选择离子程序，参见附录 B。

标准系列第 i 点中目标物（或替代物）的相对响应因子（RRF），按照公式（1）进行计算。

$$
R R F _ {i} = \frac {A _ {i}}{A _ {I S i}} \times \frac {m _ {I S i}}{m _ {i}} \tag {1}
$$

式中：

$R R F _ { i }$ ——标准系列中第 i 点目标物（或替代物）的相对响应因子；

$A _ { i }$ ——标准系列中第 i 点目标物（或替代物）定量离子的响应值；

$A _ { I S i }$ ——标准系列中第 i 点与目标物（或替代物）相对应内标定量离子的响应值；

mISi—标准系列中内标的含量，g；

$m _ { i }$ —标准系列中第 i 点目标物（或替代物）的含量，g。

目标物（或替代物）的平均相对响应因子 $\overline { { R R F } }$ ，按照公式（2）进行计算。

$$
\overline {{R R F _ {i}}} = \frac {\sum_ {i = 1} ^ {n} R R F _ {i}}{n} \tag {2}
$$

式中：

$\overline { { R R F _ { i } } }$ ——目标物（或替代物）的平均相对响应因子；

RRFi——标准系列中第 i点目标物（或替代物）的相对响应因子；

n—标准系列点数。

# 8 结果计算与表示

# 8.1 定性分析

以样品中目标化合物相对保留时间（RRT）、辅助离子和定量离子丰度比（Q）来定性。

样品中目标化合物的相对保留时间与工作曲线中间点该目标化合物相对保留时间的偏差应在 $\pm 0 . 0 6$ 以内。样品中目标化合物的特征离子相对丰度在工作曲线中间点该目标化合物的特征离子相对丰度的 $\pm 3 0 \%$ 以内。

按公式（3）计算相对保留时间（RRT）。

$$
R R T = \frac {R T _ {x}}{R T _ {I S}} \tag {3}
$$

式中：

RRT——相对保留时间；

$R T _ { x }$ ——目标物的保留时间，min；

RTIS—内标物的保留时间，min。

# 8.2 结果计算

8.2.1 根据目标物和内标定量离子的响应值进行计算。当样品中目标物的定量离子有干扰时，可以使用辅助离子定量，参见附录 B。

# 8.2.2 目标物（或替代物）含量的计算

# 8.2.2.1 用平均相对响应因子计算

当目标物（或替代物）采用平均相对响应因子进行校准时，目标物的含量按公式（4）进行计算。

$$
m _ {1} = \frac {A _ {x} \times m _ {I S}}{A _ {I S} \times \overline {{R R F}}} \tag {4}
$$

式中：

m1—目标物（或替代物）的含量，g；

$A _ { x }$ ——目标物（或替代物）定量离子的响应值；

$A _ { I S }$ —与目标物（或替代物）相对应内标定量离子的响应值；

mIS —内标物的含量，g；

RRF ——目标物（或替代物）的平均相对响应因子。

# 8.2.2.2 用工作曲线计算

当目标物采用工作曲线进行校准时，目标物含量通过相应的工作曲线计算。

# 8.2.2.3 低含量固体废物中目标物的结果计算

低含量固体废物中苯系物的含量（ $\mu \mathrm { g / k g ) }$ ，按照公式（5）进行计算。

$$
w = \frac {m _ {1} \times 1 0 ^ {3}}{m} \tag {5}
$$

式中：

w—目标化合物的含量， $\mu \mathrm { g / k g }$ ；

m1—根据平均相对响应因子或工作曲线计算出目标化合物的含量，μg；

m—样品量（湿重），g。

# 8.2.2.4 高含量固体废物中目标物的结果计算

高含量固体废物中苯系物的含量（ $\mu \mathrm { g / k g ) }$ ），按照公式（6）进行计算。

$$
w = \frac {m _ {1} \times V _ {c} \times f \times 1 0 ^ {3}}{m \times V _ {s}} \tag {6}
$$

式中：

w—目标化合物的含量， $\mu \mathrm { g / k g }$ ；

$m _ { I ^ { \cdot } }$ —根据平均相对响应因子或工作曲线计算出目标化合物的含量， $\mu \mathrm { g }$

$V _ { c }$ —甲醇提取液体积，ml；

$m$ —样品量（湿重），g；

$V _ { s }$ ——用于顶空测定的甲醇提取液体积，ml；

$f$ —提取液的稀释倍数。

# 8.2.2.5 固体废物浸出液的结果计算

测定固体废物浸出液时，苯系物的浓度直接从工作曲线查得，以 $\mu \mathrm { g / L }$ 表示。

# 10 质量保证和质量控制

# 10.1 空白试验

每 20 个样品或每批次（少于 20 个/批）应分析一个全程序空白样品和一个实验室空白样品，测定结果中目标化合物含量应低于方法检出限。

# 10.2 校准

# 10.2.1 初始校准

采用 7.5.3的平均相对响应因子进行校准时，标准系列各点相对响应因子的相对标准偏差应 $\text{‰}$ ；利用 7.5.2建立工作曲线的相关系数应 ${ \geqslant } 0 . 9 9 5$ ，否则应重新进行校准。

# 10.2.2 连续校准

每 20 个样品或每 $2 4 \mathrm { { h } }$ ，利用工作曲线中间点进行校准确认，目标化合物的测定值与标准值间的相对误差在 $\pm 2 0 \%$ ，否则，应重新绘制工作曲线。

# 10.3 平行样

每 20 个样品或每批次（少于 20 个/批）应分析一个平行样，平行双样测定结果的相对偏差 $\text{‰}$ 。

# 10.4 基体加标

每 20 个样品或每批次（少于 20 个/批）应分析一个基体加标样品，样品中目标物的加标回收率应在 $5 0 \% { \sim } 1 1 0 \%$ ，样品中替代物加标回收率应在 $7 0 \% { \sim } 1 1 0 \%$ ，否则应重新分析样品。若重复测定替代物回收率仍不合格，此时应分析一个空白加标样品，其中的目标物回收率应在 $7 0 \% { \sim } 1 2 0 \%$ 。
