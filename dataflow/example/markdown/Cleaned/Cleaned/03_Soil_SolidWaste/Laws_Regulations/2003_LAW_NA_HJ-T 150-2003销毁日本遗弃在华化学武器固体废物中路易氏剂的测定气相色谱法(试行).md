# 1 范围

本标准规定了测定固体废物中路易氏剂的气相色谱方法。

本标准适用于销毁日本遗弃在华化学武器全过程固体废物中路易氏剂的测定，同时，也适用于2-氯乙烯氧肿、2-氯乙烯肿酸的测定。

本标准用于测定固体废物残渣中路易氏剂的最低检出含量（干基）为 $0.05\mathrm{mg / kg}$ ；弹（筒）片表面检测的最低染毒密度为 $0.20\mu \mathrm{g} / \mathrm{cm}^2$ 。

# 3 方法原理

固体废物残渣用1，3-二巯基丙烷的二氯甲烷溶液进行衍生、萃取；不适于液-固萃取的弹片表面用1，3-二巯基丙烷丙酮溶液浸润的棉球（或棉垫）擦拭后，用1，3-二巯基丙烷的二氯甲烷溶液进行衍生、萃取，萃取液经过滤净化、浓缩（若有必要）后，用毛细管气相色谱火焰光度（硫）检测器测定。

# 4 试剂和材料

# 4.1 载气和辅助气体

4.1.1 载气：氦气，纯度不小于 $99.99\%$
4.1.2 燃烧气：氢气，纯度不小于 $99.99\%$ ，硅胶净化。
4.1.3 助燃气：经净化的空气。

# 4.2 试剂

4.2.1 二氯甲烷（ $\mathrm{CH}_2\mathrm{Cl}_2$ ），分析纯。

4.2.2 丙酮 $\mathrm{(C_3H_6O)}$ ，分析纯。
4.2.3 硫基化衍生试剂：1，3-二巯基丙烷（ $\mathrm{C_3H_8S_2}$ ，简称PDT)，纯度不低于 $98\%$ ，使用时用二氯甲烷配制成体积分数为0.02的溶液，配制后用高纯氮气保护，于 $4\%$ 避光保存，保存期 $2\mathrm{d}$ 。
4.2.4 硅醚化衍生试剂：N，O-双（三甲基硅基）三氟乙酰胺（ $\mathrm{C_8H_{18}F_3NOSi_2}$ ，简称BSTFA），纯度不低于 $98\%$ 。
4.2.5 标准样品
4.2.5.1 路易氏剂标准样品：纯度不低于 $98\%$
4.2.5.2 贮备溶液的配制

称取路易氏剂标准样品（4.2.5.1）约 $10\mathrm{mg}$ ，精确至 $0.1\mathrm{mg}$ ，置于 $10\mathrm{ml}$ 容量瓶中，用丙酮定容，于 $4\mathrm{C}$ 密封保存，保存期为 $30\mathrm{d}$ 。

4.2.5.3 中间溶液的配制

用移液管取 $0.50\mathrm{ml}$ 贮备溶液（4.2.5.2）于 $10\mathrm{ml}$ 容量瓶中，用二氯甲烷定容，配制浓度为 $50~\mu \mathrm{g / ml}$ 左右的中间溶液，于 $4\%$ 密封保存，保存期为 $7\mathrm{d}$ 。

4.2.5.4 标准溶液的配制

用移液管移取不同体积的中间溶液（4.2.5.3）于 $10\mathrm{ml}$ 的容量瓶中，用二氯甲烷定容，配制浓度为 $0.5\sim 40~\mu \mathrm{g / ml}$ 的标准溶液（至少五种浓度)。此标准溶液需现用现配。

# 4.3 材料

4.3.1 微孔滤膜过滤器：有机系列，孔径 $0.50\mu \mathrm{m}$
4.3.2 干燥脱脂棉球（或棉垫）。
4.3.3 离心过滤试管。

# 5 仪器

5.1 色谱仪：具火焰光度检测器的气相色谱仪（GC）。
5.2 检测器：火焰光度-硫型（FPD-S)，装有 $394\mathrm{nm}$ 蓝色滤光片。
5.3 色谱柱： $14\%$ 氰丙基苯基， $86\%$ 甲基聚硅氧烷固定液的石英毛细管柱， $30\mathrm{m} \times 0.32\mathrm{mm} \times 0.25\mu \mathrm{m}$ （长度 $\times$ 内径 $\times$ 膜厚）。
5.4 试样预处理使用的仪器有：

a）具塞刻度试管；
b）振荡器；
c）超声波清洗器：90W；
d）精密移液器：20、200 $\mu \mathrm{l}$
e）离心机：最高转速不小于 $5000\mathrm{r / min}$
f）离心试管： $10\mathrm{ml}$ 具塞离心试管。

# 6样品

# 6.1 样品采集

6.1.1 销毁残渣的采样：采集样品布点可采用梅花形或直线形等方式均匀布点或根据需要采集。将每点采集的销毁残渣，剔除杂质后装入样品瓶中，做好标识和记录，注明编号、采样日期、采样地点、销毁残渣名称以及采样量等信息。
6.1.2 弹（筒）片的取样：用 $2\mathrm{ml}$ 体积分数为0.0012的PDT丙酮溶液浸润后的脱脂棉球（或棉垫）在弹（筒）片表面取 $30~\mathrm{cm}^2$ 由外向里擦拭一遍，共用五个 $100\mathrm{mg}$ 棉球（或棉垫）擦拭，将采样后棉球（或棉垫）置于锥形瓶中，做好标识和记录。

# 6.2 样品保存

样品采集后于 $4\%$ 以下保存，应尽快分析。

# 6.3 样品制备

# 6.3.1 固体废物残渣

称取约 $10\mathrm{g}$ 左右固体废物残渣（6.1.1)，精确至 $0.1\mathrm{g}$ ，置于 $50\mathrm{ml}$ 具塞磨口锥形瓶中，加入 $10.0\mathrm{ml}$ 的二氯甲烷及 $120\mu \mathrm{l}$ 新配制的PDT二氯甲烷溶液（4.2.3)，于振荡器上振荡 $10\mathrm{min}$ 。静置1 $\mathrm{min}$ 后，取约 $6\mathrm{ml}$ 萃取液至试管中，用 $0.50\mu \mathrm{m}$ 微孔滤膜过滤器过滤，取 $5.0\mathrm{ml}$ 滤液于具塞刻度试管中，加入 $50\mu \mathrm{l}$ 的BSTFA（4.2.4）混匀，室温放置 $10\mathrm{min}$ 后，供气相色谱分析，若达不到最低检出含量时，则用氮气浓缩至 $0.5\mathrm{ml}$ ，再供气相色谱分析。

# 6.3.2 弹（筒）片

在按6.1.2处理后的锥形瓶样品中加入 $8\mathrm{ml}$ 二氯甲烷及 $120\mu \mathrm{l}$ 新配制的PDT二氯甲烷溶液(4.2.3)，超声 $3\mathrm{min}$ ，转移萃取液，用离心过滤试管将余下棉球离心过滤，滤液与原取出的萃取液合并定容至 $10.0\mathrm{ml}$ 。取 $5.0\mathrm{ml}$ 滤液于具塞刻度试管中，加入 $50\mu \mathrm{l}$ 的BSTFA（4.2.4）混匀，室温放置 $10\mathrm{min}$ 后，供气相色谱分析，若达不到最低检出含量时，则用氮气浓缩至 $0.5\mathrm{ml}$ ，再供气相色谱分析。

# 6.4 水分测定

称取与测试样品相同质量的固体废物样品，风干后，于 $105\sim 110\mathrm{^\circ C}$ 烘干 $2\mathrm{h}$ ，冷却至室温、称量，计算干基样品中水分含量。

# 7 分析步骤

# 7.1 色谱条件

7.1.1 进样口温度： $250\%$
7.1.2 出口温度： $260\%$ 。
7.1.3 检测器温度： $150\%$
7.1.4 升温程序：初始温度为 $100\%$ ，保持 $1\mathrm{min}$ ，以 $10\% / \mathrm{min}$ 的速度升温至 $250\%$ ，保持 $2\mathrm{min}$ 。
7.1.5 载气：柱前压为 $78\mathrm{kPa}$ 。
7.1.6 氢气：压力为 $140\mathrm{kPa}$ 。
7.1.7 空气：压力为 $70\mathrm{kPa}$ 。应根据所用仪器的特点调节氢气/空气的流量比以获得最佳检测能力。
7.1.8 进样方式：不分流进样，不分流时间为 $60\mathrm{s}$ 。
7.1.9 进样量： $1.0\mu \mathrm{l}_{\circ}$

# 7.2 仪器检测限

以二倍信噪比时所对应的路易氏剂的质量作为仪器的检测限，本方法要求仪器的最低检测限小于 $2 \times 10^{-10} \mathrm{~g}$ 。

# 7.3 校准

# 7.3.1 定量方法

外标法。

# 7.3.2 标准曲线的绘制

取不同浓度的路易氏剂标准溶液（4.2.5.4）各 $1.0\mathrm{ml}$ ，分别加入 $10\mu \mathrm{l}$ 新配制的PDT二氯甲烷溶液（4.2.3），室温放置 $3\mathrm{min}$ 后，加入 $10\mu \mathrm{l}$ BSTFA（4.2.4）充分混合， $10\mathrm{min}$ 后，按7.1进行色谱分析，测定峰面积，绘制峰面积对数值与浓度对数值对应关系的标准曲线，并进行线性回归，回归公式见公式（1）：

$$
\log A _ {i} = b \log c _ {i} + a \tag {1}
$$

式中： $A_{i}$ ——路易氏剂衍生产物的峰面积；

$b$ ——标准曲线斜率；

$c_{i}$ ——测试液中路易氏剂的浓度， $\mathrm{mg / L}$

$a$ ——标准曲线截距。

# 7.3.3 标准曲线的校正

仪器稳定时在每批样品测定前后应用路易氏剂标准溶液各校正一次。若校正点相对误差大于 $5\%$ 时，应重新绘制标准曲线。

# 7.4.1 定性分析

# 7.4.2 定量分析

# 7.4.2.1 固体废物残渣

# 7.4.2.1.1 分析

取样品过滤液（6.3.1）在7.1条件下进行色谱分析，一个样品连续进样三次，计算峰面积的平均值 $A_{i}$ 。

# 7.4.2.1.2 计算

路易氏剂的含量 $c$ 按公式（2）计算：

$$
c = \left(c _ {i} V _ {0}\right) / (m n) \tag {2}
$$

式中： $c$ ——固体废物残渣中路易氏剂的含量（干基）， $\mathrm{mg / kg}$

$c_{i}$ ——测试液中路易氏剂的浓度， $\mathrm{mg / L}$ ，由公式（1）计算；

$V_{0}$ ——提取液的体积，L；

$m$ ——所称固体废物残渣样品的质量（干基）， $\mathrm{kg}$ ；

$n$ ——样品体积浓缩倍数。

# 7.4.2.2 弹（筒）片

# 7.4.2.2.1 分析

取样品提取液（6.3.2）在7.1条件下进行色谱分析，一个样品连续进样三次，计算峰面积的平均值 $A_{i}$ 。

# 7.4.2.2.2 计算

路易氏剂的染毒密度 $c$ 按公式（3）计算：

$$
c = \left(c _ {i} V _ {0}\right) / (n s) \tag {3}
$$

式中： $c$ ——弹（筒）片表面路易氏剂的染毒密度， $\mathrm{mg/cm^2}$

$c_{i}$ ——测试液中路易氏剂的浓度， $\mathrm{mg / L}$ ，由公式（1）计算；

$V_{0}$ ——提取液的体积，L；

$n$ ——样品体积浓缩倍数；

$s$ ——擦拭弹（筒）片表面积， $\mathrm{cm}^2$ 。

# 8 结果表示

# 8.1 定性结果

按7.4.1表示。

# 8.2 最低检出含量

固体废物残渣中路易氏剂的最低检出含量（干基）为 $0.05\mathrm{mg / kg}$ ；（不适用液－固萃取以擦拭方式取样的）弹（筒）片表面检测的最低染毒密度为 $0.20\mu \mathrm{g} / \mathrm{cm}^2$ 。

# 8.3 精密度

相对标准偏差小于 $15\%$ 。

# 8.4 准确度

相对误差不大于 $20\%$ 。
