# 1 范围

本标准规定了用气相色谱法分析土壤中氧联双二苯脾的方法。

本方法适用于销毁日本遗弃在华化学武器全过程土壤中氧联双二苯胂的监测。

本方法检出限（干基）为 $0.1\mathrm{mg / kg}$

# 3 原理

用二氯甲烷对土样进行提取，用具氢火焰离子化检测器的毛细管气相色谱仪进行测定，外标法定量。

# 4 试剂与材料

4.1 载气： $\mathbf{N}_2$ ，纯度 $99.999\%$
4.2 辅助气：氢气（纯度 $99.99\%$ ）和净化过的空气。
4.3 色谱标准样品：氧联双二苯肿（含量不小于 $97\%$ ）。
4.4 二氯甲烷：分析纯，经重蒸无明显测定干扰峰。
4.5 无水硫酸钠：分析纯。
4.6 氧联双二苯肿标准贮备液：准确称取一定量的氧联双二苯肿（4.3）标准样品（准确到0.1mg)，溶于二氯甲烷中，在 $100\mathrm{ml}$ 容量瓶中定容，在 $4\%$ 下储存。储存液可以连续使用 $15\mathrm{d}$ 。
4.7 氧联双二苯脾标准工作液：根据检测器的灵敏度及线性要求，用二氯甲烷稀释贮备液（4.6），配制成7种不同浓度的标准工作液，在 $4\%$ 下保存并限当日使用。

# 5 仪器

5.1 仪器：具氢火焰离子化检测器的气相色谱仪。

5.2 微量进样器：1、2、5 $\mu \mathrm{l}$ 。
5.3 记录仪、积分仪或色谱工作站。
5.4 色谱柱： $5\%$ 二苯基- $95\%$ 二甲基聚硅氧烷毛细管柱， $30\mathrm{m}\times 0.25\mathrm{mm}\times 0.25\mu \mathrm{m}$ （长度 $\times$ 内径 $\times$ 膜厚）。
5.5 试样预处理时使用的主要仪器
5.5.1 超声波清洗机， $100\mathrm{W}$
5.5.2 微孔滤膜 $0.8\mu \mathrm{m}$
5.5.3 金属筛100目。
5.5.4 离心机： $2000\mathrm{r / min}$

# 6样品

# 6.1 样品采集

选择具有代表性的地段，采用梅花形或直线形等方式因地制宜均匀布点，或根据监测需要选点采集，剔除土壤中的杂质，装入样品袋中，内外各有一张标签（写明编号、采样地点、土壤名称、深度、筛孔以及采样日期和采样量等），做好标识和记录。

# 6.2 样品保存

采集的样品应在 $4\%$ 保存，尽快分析。

# 6.3 试样制备

称取土样 $20\mathrm{g}$ （准确到 $1\mathrm{mg}$ ），置于一个有螺旋帽的玻璃瓶中，加入 $20\mathrm{ml}$ 二氯甲烷（4.4），扭紧瓶塞，在超声波槽中振荡萃取 $5\mathrm{min}$ ，快速转移上层液体到一个具塞离心管中。再用 $10\mathrm{ml}$ 二氯甲烷重复萃取一次，合并两次萃取液，离心 $3\mathrm{min}$ ，以 $0.8\mu\mathrm{m}$ 滤膜过滤。

若有肉眼可见的水珠，可加入少量无水硫酸钠（4.5）进行干燥。

如萃取液浓度太低，可于 $35\%$ 水浴中用轻微的 $\mathrm{N}_2$ 气流吹扫浓缩，准确计算浓缩倍数。

萃取液应在 $4^{\circ}\mathrm{C}$ 下保存并立即用于色谱分析。

# 6.4 水分测定

按照GB7172—87《土壤水分测定法》测定样品水分。

# 7 分析步骤

# 7.1 色谱条件

7.1.1 进样口温度： $280\%$ 。
7.1.2 柱温度：初始温度 $50\%$ （保持 $1\mathrm{min}$ ），以 $10\% / \mathrm{min}$ 升温到 $280\%$ （保持 $5\mathrm{min}$ ）。
7.1.3 检测器温度： $280~\mathrm{^\circ C}$ 。
7.1.4 气体流量：载气流量 $5\mathrm{ml} / \mathrm{min}$ ；尾吹气量 $35\mathrm{ml} / \mathrm{min}$ ；氢气 $30\mathrm{ml} / \mathrm{min}$ ；空气 $300\mathrm{ml} / \mathrm{min}$ 。
7.1.5 进样方式：不分流进样，不分流时间 $1\mathrm{min}$
7.1.6 进样量： $1\mu \mathrm{l}_{\circ}$

# 7.2 校准

7.2.1 定量方法：外标法。

# 7.2.2 标准曲线的绘制

取7个不同浓度的标准工作液。在（7.1）规定的条件下，将上述标准溶液，用微量进样器进样 $1\mu \mathrm{l}$ 。各连续进样3次（要求3次测定的峰面积的相对偏差不大于 $10\%$ ），以氧联双二苯肿面积或峰高为纵坐标，氧联双二苯肿浓度为横坐标，绘制标准曲线。氧联双二苯肿的浓度按公式（1）计算：

$$
y = a + b x \tag {1}
$$

式中： $y$ ——氧联双二苯肿的峰面积或峰高；

$x$ ——氧联双二苯肿的浓度， $\mu \mathrm{g} / \mathrm{ml}$ ；

$a$ ——标准曲线的截距；

$b$ ——标准曲线的斜率。

# 7.2.3 标准曲线的校正

标准曲线应每天校正一次，选标准曲线中间浓度进行校正，当相对误差大于 $10\%$ 时，应重新绘制标准曲线。

# 7.2.4 仪器的检测限

在仪器稳定条件下，配制一定浓度的氧联双二苯肿溶液，以产生二倍噪声峰高所对应的氧联双二苯肿含量计算，本仪器的检测限为 $1 \times 10^{-12} \mathrm{~g}$ 。

# 7.3.2 定量分析

按（7.2）的操作，对试液（6.3）进行分析，外标法定量。

# 7.4 计算

土壤中氧联双二苯胂的含量按公式（2）计算：

$$
X = \frac {c V}{M K} \tag {2}
$$

式中： $X$ ——试样中氧联双二苯肿的含量， $\mathrm{mg / kg}$

$c$ ——从标准曲线上查得氧联双二苯脾的浓度， $\mu \mathrm{g} / \mathrm{ml}$

$V$ ——样品定容体积， $\mathrm{ml}$

$K$ ——体积浓缩倍数；

$M$ ——土样重量（干基），g。

# 8 结果表示

# 8.1 定性结果

根据标准色谱图中的氧联双二苯肿的保留时间来确定被测样品中氧联双二苯肿的存在。保留时间误差为 $\pm 10\mathrm{s}$ ，应在色谱图上标明属于氧联双二苯肿的峰。

# 8.2 定量结果

# 8.2.1 含量表示方法

土壤中氧联双二苯胂的含量以 $\mathrm{mg / kg}$ 表示。

# 8.2.2 精密度

HJ/T119-2003

相对标准偏差不大于 $15\%$ 。

# 8.2.3 准确度

相对误差不大于 $15\%$ 。

# 8.2.4 检出限

在二倍信噪比时土壤中氧联双二苯脾的检出限为 $0.1\mathrm{mg / kg}$ （干基）。
