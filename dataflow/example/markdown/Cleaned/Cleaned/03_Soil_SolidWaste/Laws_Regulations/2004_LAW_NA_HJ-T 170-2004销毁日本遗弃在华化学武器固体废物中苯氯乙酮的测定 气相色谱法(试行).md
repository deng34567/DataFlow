# 1 范围

本标准规定了用气相色谱法分析固体废物中苯氯乙酮的方法。

本方法适用于销毁日本遗弃在华化学武器全过程固体废物中苯氯乙酮的监测。

本方法对固体废物的检出限（干基）为 $0.01\mathrm{mg / kg}$ ；对弹（筒）片表面的最低检出染毒密度为 $0.1\mathrm{mg / m^2}$ 。

# 3 原理

用无水乙醇对固体废物样品进行提取，用具电子捕获（ECD）检测器的毛细管气相色谱仪对富集的苯氯乙酮进行测定，外标法定量。

# 4 试剂与材料

4.1 载气： $\mathbf{N}_2$ ，纯度 $99.999\%$
4.2 苯氯乙酮：含量不小于 $99.5\%$ 。
4.3 苯氯乙酮标准贮备溶液（ $10.0\mathrm{mg / L}$ ）：称取苯氯乙酮（4.2） $0.01\mathrm{g}$ （准确到 $0.1\mathrm{mg}$ ）于干燥的烧杯中，加入一定量的无水乙醇溶解，待全部溶解后，用无水乙醇定容至 $1000\mathrm{ml}$ ，在 $4\%$ 可保存 $15\mathrm{d}$ 。
4.4 无水乙醇：分析纯，经重蒸无明显测定干扰峰。
4.5 脱脂棉球：每个干棉球重 $0.1\mathrm{g}$ 。

# 5 仪器

5.1 仪器：具电子捕获检测器的气相色谱仪。
5.2 进样器： $1,2\mu \mathrm{l}$

5.3 记录仪、积分仪或色谱工作站。
5.4 色谱柱： $100\%$ 二甲基聚硅氧烷石英毛细管柱， $25\mathrm{m} \times 0.32\mathrm{mm} \times 0.25\mu \mathrm{m}$ （长度 $\times$ 内径 $\times$ 膜厚）。
5.5 试样预处理时使用的仪器
5.5.1 微孔滤膜： $0.8\mu \mathrm{m}$
5.5.2 超声波振荡器， $500\mathrm{W}$ 。

# 6样品

# 6.1 样品采集

6.1.1 固体废物残渣的采集：采集样品布点可采用梅花形或直线形等方式均匀布点或根据需要采集。每点采集的销毁残渣，剔除杂质后装入样品瓶中，内外各有一张标签（写明编号、采样地点、销毁残渣名称、深度、筛孔以及采样日期和采样量等信息），做好标识和记录。
6.1.2 弹（筒）片的取样：量取弹（筒）片表面积 $30\mathrm{cm}^2$ （ $5\mathrm{cm} \times 6\mathrm{cm}$ ），用5个浸有少量乙醇（4.4）的脱脂棉球依次从外到里擦拭5遍，将棉球置于锥形瓶中，贴好标签（写明编号、采样地点、深度、固体废物名称以及采样日期和采样量等信息），做好标识和记录。

# 6.2 样品的保存

采集的样品于 $4^{\circ} \mathrm{C}$ 保存，尽快分析。

# 6.3 试样的制备

# 6.3.1 固体废物残渣处理

称取 $10\mathrm{g}$ 样品（6.1.1)，准确到 $0.1\mathrm{mg}$ ，置于一个有螺旋帽的玻璃瓶中，加入一定量无水乙醇，在超声波清洗槽中萃取 $10\mathrm{min}$ ，抽滤、再萃取一次，抽滤洗涤、抽滤，抽滤液用无水乙醇定容为 $10\mathrm{ml}$ 或 $50\mathrm{ml}$ 。

# 6.3.2 弹（筒）片处理

在来自6.1.2的棉球中，加入一定量无水乙醇，超声萃取 $10\mathrm{min}$ ，倾出萃取溶液，再次加入无水乙醇重复上述操作，合并萃取液并定容为 $10\mathrm{ml}$ 或 $50\mathrm{ml}$ 。

若萃取液浓度太低，可置于通风柜中自然蒸发浓缩，但应准确计算浓缩倍数。

# 6.4 水分测定

按GB5086.1—1997中4.1.2的规定测定固体废物中的水分。

# 7 分析步骤

# 7.1 色谱条件

7.1.1 进样口温度： $220\%$
7.1.2 色谱柱温度：初始温度为 $80\%$ ，保留 $1\mathrm{min}$ ，升温速率 $20\% / \mathrm{min}$ ，终止温度为 $220\%$ ，保留 $1\mathrm{min}$ 。
7.1.3 检测器温度： $300\%$
7.1.4 载气流量： $1.0\mathrm{ml / min}$
7.1.5 尾吹气流量： $29\mathrm{ml / min}$
7.1.6 进样方式：不分流进样，不分流时间为 $1\mathrm{min}$

# 7.2 校准

7.2.1 定量方法：外标法。
7.2.2 标准曲线的绘制

取7支 $25\mathrm{ml}$ 容量瓶，分别加入一定量的苯氯乙酮标准溶液（4.3)，然后用无水乙醇稀释至标线，配制成浓度为0.01、0.10、0.20、0.40、0.50、0.80、 $1.00\mathrm{mg / L}$ 的工作液。待色谱基线稳定后，将上述标准溶液，用微量进样器进样 $1\mu \mathrm{l}$ 。各连续进样3次（要求三次测定的峰面积的相对偏差不大

于 $10\%$ ，以苯氯乙酮峰面积为纵坐标，苯氯乙酮浓度为横坐标，绘制标准曲线。

苯氯乙酮的浓度按公式（1）计算：

$$
y = a + b x \tag {1}
$$

式中： $y$ ——苯氯乙酮的峰面积；

$x$ ——苯氯乙酮的浓度， $\mu \mathrm{g} / \mathrm{ml}$

$a$ ——标准曲线的截距；

$b$ ——标准曲线的斜率。

# 7.2.3 标准曲线的校正

标准曲线应每天校正一次，选标准曲线中间浓度进行校正，当相对误差大于 $10\%$ 时，应重新绘制标准曲线。

# 7.2.4 仪器的检测限

在仪器稳定条件下，配制一定浓度的苯氯乙酮溶液，以产生二倍噪声峰高所对应的苯氯乙酮含量计算，本仪器的检测限为 $1 \times 10^{-12} \mathrm{~g}$ 。

# 7.3.2 定量分析

按（7.2.2）的操作，将制备的试液（6.3）进样分析，外标法定量。

# 7.4 计算

7.4.1 对固体残渣废物中苯氯乙酮的含量按公式（2）计算：

$$
X = \frac {c V}{M} \tag {2}
$$

式中： $X$ ——试样中苯氯乙酮的含量， $\mathrm{mg / kg}$

$c$ ——从标准曲线上查得苯氯乙酮的浓度， $\mu \mathrm{g} / \mathrm{ml}$

$V$ ——样品定容体积，ml；

$M$ ——样品重量（干基），g。

7.4.2 弹（筒）片样品的染毒密度按公式（3）计算：

$$
X _ {1} = \frac {c _ {1} V _ {1}}{M _ {1}} \tag {3}
$$

式中： $X_{1}$ ——弹（筒）片样品的染毒密度， $\mathrm{mg / m^2}$

$c_{1}$ ——从标准曲线上查得苯氯乙酮的浓度， $\mu \mathrm{g} / \mathrm{ml}$

$V_{1}$ ——样品定容体积，ml；

$M_{1}$ ——弹（筒）片样品擦试面积， $\mathrm{m}^2$ 。

# 8 结果表示

# 8.1 定性结果

根据标准色谱图中的苯氯乙酮的保留时间来确定被测样品中苯氯乙酮的存在。保留时间误差为 $\pm 10\mathrm{s}$ ，应在色谱图上标明属于苯氯乙酮的峰。

# 8.2 定量结果

# 8.2.1 含量表示方法

固体废物残渣样品中苯氯乙酮的含量以 $\mathrm{mg / kg}$ 表示，弹（筒）片样品的染毒密度以 $\mathrm{mg / m^2}$ 表示。

# 8.2.2 精密度

相对标准偏差不大于 $10\%$ 。

# 8.2.3 准确度

相对误差不大于 $15\%$ 。

# 8.2.4 检出限

固体废物的检出限（干基）为 $0.01\mathrm{mg / kg}$ ；对弹（筒）片表面的最低检出染毒密度为 $0.1\mathrm{mg / m^2}$ 。
