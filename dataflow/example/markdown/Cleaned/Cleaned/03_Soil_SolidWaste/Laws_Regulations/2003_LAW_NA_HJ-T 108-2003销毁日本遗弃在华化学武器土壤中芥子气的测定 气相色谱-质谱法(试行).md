# 1 范围

本标准规定了测定土壤中芥子气的气相色谱-质谱方法。

本标准适用于销毁日本遗弃在华化学武器全过程土壤中芥子气的测定。

本标准的最低检测含量为（干基） $0.02\mathrm{mg / kg}$

# 3 方法原理

用二氯甲烷萃取土壤中的芥子气，萃取液经过滤净化、浓缩，加入内标丁基硫醚后，用毛细管气相色谱-质谱选择离子检测方式进行分离测定。

# 4 试剂和材料

# 4.1 载气和辅助气体

4.1.1 载气：氦气，纯度不小于 $99.99\%$
4.1.2 样品浓缩气：氮气，纯度不小于 $99.999\%$

# 4.2 试剂

4.2.1 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ，分析纯。
4.2.2 二氯甲烷（ $\mathrm{CH}_2\mathrm{Cl}_2$ ），分析纯。
4.2.3 内标：丁基硫醚 $\mathrm{(C_8H_{18}S)}$ ，纯度不小于 $98\%$ 。

# 4.2.3.1 内标贮备溶液配制

称取丁基硫醚（4.2.3）约 $20\mathrm{mg}$ ，精确至 $0.1\mathrm{mg}$ ，置于 $10\mathrm{ml}$ 容量瓶内，用正己烷（4.2.1）定容，于 $4\%$ 下保存，保存期为 $90\mathrm{d}$ 。

# 4.2.3.2 内标标准溶液配制

用移液管取一定量的内标贮备溶液（4.2.3.1）于 $50~\mathrm{ml}$ 容量瓶中，用二氯甲烷（4.2.2）定容，配制成浓度约为 $2\mathrm{mg / L}$ 的内标标准溶液，于 $4^{\circ}\mathrm{C}$ 下保存，保存期为 $30\mathrm{d}$ 。

# 4.2.4 标准样品

4.2.4.1 芥子气：纯度不小于 $98\%$ 。

# 4.2.4.2 芥子气贮备溶液配制

称取芥子气标准样品（4.2.4.1）约 $10\mathrm{mg}$ ，精确至 $0.1\mathrm{mg}$ ，置于 $10\mathrm{ml}$ 容量瓶中，用正己烷（4.2.1）定容，于 $4^{\circ}\mathrm{C}$ 下密封保存，保存期为 $30\mathrm{d}$ 。

# 4.2.4.3 芥子气中间溶液配制

用移液管取 $0.10\mathrm{ml}$ 贮备溶液（4.2.4.2）于 $10\mathrm{ml}$ 容量瓶中，用二氯甲烷（4.2.2）定容，配制浓度约为 $10\mathrm{mg / L}$ 的中间溶液，于 $4^{\circ}\mathrm{C}$ 下保存，保存期为 $14\mathrm{d}$ 。

# 4.2.4.4 芥子气标准溶液配制

分别用移液管移取不同体积的中间溶液（4.2.4.3）置于 $10\mathrm{ml}$ 的容量瓶中，用二氯甲烷（4.2.2）定容，配制成浓度范围在 $20\times 10^{-3}\sim 400\times 10^{-3}\mathrm{mg / L}$ 的标准溶液（至少五种浓度)。此标准溶液需现用现配。

4.2.5 无水硫酸钠（ $\mathrm{Na_2SO_4}$ ），分析纯， $200^{\circ}\mathrm{C}$ 烘烤 $4\mathrm{h}$ ，备用。

# 4.3 材料

4.3.1 微孔滤膜过滤器：有机系列，孔径为 $0.45\mu \mathrm{m}$ 。
4.3.2 一次性医用注射器： $5\mathrm{ml}$

# 5 仪器

# 5.1 气相色谱-质谱联用仪

质谱仪具有EI源和选择离子检测功能。

# 5.2 色谱柱

$5\%$ -二苯基 $-95\%$ 甲基硅酮固定相的石英毛细管柱， $30\mathrm{m}\times 0.25\mathrm{mm}\times 0.25\mu \mathrm{m}$ （长 $\times$ 内径 $\times$ 膜厚)。

# 5.3 其他仪器

其它仪器有：

a）康氏振荡器；
b）离心机：转速不低于 $5000\mathrm{r / min}$
c）试管： $5\mathrm{ml}$ 具塞刻度试管。

# 6样品

# 6.1 样品采集

选择具有代表性的地段，采集样品布点应因地制宜，可采用梅花形或直线形等方式均匀布点或根据需要采集。每点采集的土壤，剔除杂质后装入样品瓶中，内外各有一张标签（写明编号、采样地点、土壤名称、深度以及采样日期和采样量等信息），做好标识和记录。

# 6.2 样品保存

样品采集后于 $4\%$ 下保存，应尽快分析。样品保存期为 $7\mathrm{d}$ 。

# 6.3 样品制备

称取样品约 $10\mathrm{g}$ ，精确至 $0.1\mathrm{g}$ ，置于 $50\mathrm{ml}$ 具塞锥形瓶中，加入 $15.0\mathrm{ml}$ 二氯甲烷（4.2.2），放入振荡器（5.3a）中在室温振荡 $5\mathrm{min}$ 。静置 $10\mathrm{min}$ ，等待微粒沉降。如固体微粒不易沉降，以 $1000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，使其分层。取上层清液约 $6\mathrm{ml}$ 到试管中，加入适量的无水硫酸钠（4.2.5）干燥后，用微孔滤膜过滤器（4.3.1）过滤。将 $5.0\mathrm{ml}$ 滤液转移到刻度试管（5.3c）中用高纯氮气（4.1.2）流轻吹浓缩并定容到 $1.0\mathrm{ml}$ ，然后加入 $0.1\mathrm{ml}$ 的内标溶液（4.2.3.2）混匀后，转移到 $2\mathrm{ml}$ 气相色谱试样瓶中，供气相色谱-质谱联用仪（5.1）测定。

# 6.4 水分测定

水分测定按GB7172—87中规定执行。

# 7 分析步骤

# 7.1 色谱条件

7.1.1 进样口温度： $250\%$

7.1.2 柱温：初始温度 $50\%$ ，保持 $1.0\mathrm{min}$ ，以 $10\% / \mathrm{min}$ 的速度升温至 $170\%$ ，再以 $20\% / \mathrm{min}$ 的速度升温至 $260\%$ ，保持 $2.0\mathrm{min}$ 。

7.1.3 载气流速： $1\mathrm{ml} / \mathrm{min}$

7.1.4 进样方式：不分流进样，不分流时间为 $1.0\mathrm{min}$

7.1.5 进样量： $1.0\mu \mathrm{l}$

7.2 质谱条件

7.2.1 接口温度： $280~\mathrm{^\circ C}$ 。

7.2.2 离化方式：EI。

7.2.3 离子源温度： $180~\mathrm{^\circ C}$

7.2.4 电子能量： $70\mathrm{eV}$

7.2.5 选择离子质量： $\mathrm{m / z~109}$ （芥子气）、 $\mathrm{m / z~146}$ （丁基硫醚）。

7.2.6 每一选择离子扫描时间：0.05s或0.1s，根据仪器设定。

# 7.3 仪器检测限

以强度为二倍的噪声所对应的芥子气的量作为仪器的检测限，本方法仪器的最低检测限应不大于 $2 \times 10^{-12} \mathrm{~g}$ 。

# 7.4 校准

# 7.4.1 调谐

在进行标准曲线测定前，对质谱仪进行调谐、校正。

# 7.4.2 标准曲线绘制

取不同浓度（至少五个点）的芥子气标准溶液（4.2.4.4）各 $1.0\mathrm{ml}$ ，分别加入 $0.1\mathrm{ml}$ 内标（4.2.3.2）后，在7.1和7.2的条件下用选择离子法测定他们的气相色谱-质谱响应值，绘制响应值与浓度对应关系的曲线，并进行线性回归，回归方程见公式（1）。

$$
y = a + b \frac {A _ {i}}{A _ {s}} \tag {1}
$$

式中： $y$ ——试样中芥子气的浓度， $\mathrm{mg / L}$

$a$ ——标准曲线的截距；

$b$ ——标准曲线的斜率；

$A_{i}$ ——芥子气的峰面积；

$A_{s}$ ——内标物的峰面积。

# 7.4.3 标准曲线校正

每批样品测定前后用中间浓度芥子气标准溶液进行单点校正，若相对误差大于 $15\%$ ，应重新绘制标准曲线。

# 7.5 样品测定

# 7.5.1 定性分析

根据标准样品的色谱保留时间及离子丰度来确定被测样品芥子气的存在，所测组分保留时间窗口为 $\pm 10\mathrm{s}$ 。应在色谱图上标明属于芥子气的峰。试样的选择离子色谱图见图1。

# 7.5.2.1 分析

将6.3制备的溶液，在7.1和7.2的条件下用选择离子法分别测定他们的气相色谱-质谱响应值，每份溶液进样三次，取响应值平均值。

# 7.5.2.2 计算

土样（干基）中芥子气的含量按公式（2）计算：

$$
c = \frac {y V _ {0} V _ {2}}{m V _ {1}} \tag {2}
$$

式中： $c$ 一土样中芥子气的含量， $\mathrm{mg / kg}$

$y$ ——按公式（1）计算的试样溶液中芥子气的浓度， $\mathrm{mg / L}$

$V_{0}$ ——加入试样中提取液的体积，ml；

$V_{2}$ ——浓缩后提取液的体积，ml；

$m$ ——土样的质量（干基），g；

$V_{1}$ ——用于浓缩提取液的体积， $\mathrm{ml}$ 。
