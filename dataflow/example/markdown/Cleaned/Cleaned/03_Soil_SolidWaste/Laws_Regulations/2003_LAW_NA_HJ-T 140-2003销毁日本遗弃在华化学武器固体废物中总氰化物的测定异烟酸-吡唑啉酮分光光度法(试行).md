本标准规定了应用异烟酸-吡唑啉酮分光光度法测定固体废物中总氰化物含量。

本标准适用于销毁日本遗弃在华化学武器全过程产生的固体废物中总氰化物的测定。

本标准用于固体废物残渣的最低检测含量（干基）为 $0.15\mathrm{mg / kg}$

本标准用于弹(筒)片固体废物表面的最低检测染毒密度为 $0.5 \mathrm{mg} / \mathrm{m}^{2}$ 。

# 3 术语和定义

# 总氰化物 total cyanide

总氰化物是指在EDTA存在下，在 $\mathrm{pH} < 2$ 的磷酸介质中，加热蒸馏时能形成氰化氢的含氰化合物的总称，包括全部自由状态的氰化氢和氢氰酸、简单含氰化合物（多为碱金属、碱土金属和铵的氰化合物）以及除钴氰络合物以外的绝大部分金属络合氰化物，如锌氰络合物、铁氰络合物、镍氰络合物和铜氰络合物等。

总氰化物的量，以在上述条件下蒸馏所得的氰化氢的量为表征。在分析测定中，以氰离子（ $\mathrm{CN}^{-}$ ）的量来表征。

# 4 原理

# 4.1 固体废物中总氰化物的释放和吸收原理

将待测固体废物试样置于适量水中，加入磷酸和 $\mathrm{Na_2 - EDTA}$ ，在 $\mathrm{pH} < 2$ 的条件下加热蒸馏，把自由状态的氰化氢和氢氰酸以及简单氰化物中的氰离子以氰化氢的形式蒸馏出来；同时，利用EDTA与金属离子的络合能力大于氰离子络合能力的性质，把金属络合氰化物中的氰离子解离，以氰化氢的形式蒸馏出来；用氢氧化钠溶液吸收氰化氢，测定吸收液中氰离子（ $\mathrm{CN^{-}}$ ）的量。

# 4.2 固体废物中总氰化物的测定异烟酸-吡唑啉酮分光光度法原理

在中性条件下，样品中的氰化物与氯胺T反应生成氯化氰（CNCl)，氯化氰与异烟酸作用，经水解后，生成戊烯二醛，后者与吡唑啉酮缩合，生成蓝色染料。在一定浓度范围内，染料颜色强度与氰化物含量成正比。

# 5.16 铬酸钾指示剂

称取铬酸钾（ $\mathrm{K_2CrO_4}$ ） $10\mathrm{g}$ 溶于少量水中，滴加硝酸银标准溶液（5.18）至产生橙红色沉淀为止，放置过夜，过滤，用水稀释至 $100\mathrm{ml}$ 。

# 5.17 氯化钠标准溶液 $c(\mathrm{NaCl}) = 0.01\mathrm{mol / L}$

将氯化钠（NaCl）置于瓷坩埚内，经 $500\sim 600\%$ 灼烧至恒重，在干燥器中冷却。称取 $0.5844\mathrm{g}$ 于

烧杯中，用水溶解，移入 $1000\mathrm{ml}$ 容量瓶中，稀释至标线，混合均匀。

# 5.18 硝酸银标准溶液 $c(\mathrm{AgNO}_3) = 0.01\mathrm{mol / L}$

# 5.18.1 配制

称取硝酸银（ $\mathrm{AgNO_3}$ ） $1.699\mathrm{g}$ 溶于水中，稀释至 $1000\mathrm{ml}$ ，储于棕色试剂瓶中，摇匀，待标定。

# 5.18.2 标定

（1）吸取氯化钠标准溶液（5.17） $10.0\mathrm{ml}$ 于 $150\mathrm{ml}$ 具柄瓷皿或 $250\mathrm{ml}$ 锥形瓶中，加水 $50\mathrm{ml}$ ；同时另取一同样规格具柄瓷皿或锥形瓶，加水 $60\mathrm{ml}$ ，做空白试验。
(2）向溶液中滴加 $3\sim 5$ 滴铬酸钾指示剂（5.16)，在不断搅拌下，从滴定管加入待标定的硝酸银溶液（5.18.1)，直至溶液由黄色变为浅砖红色为止，记下读数（ $V$ ）；同样滴定空白溶液，记下读数 $(V_0)$ 。
(3）硝酸银浓度 $c_{1}$ （mol/L）按式（1）计算：

$$
c _ {1} = \frac {1 0 . 0 c}{V - V _ {0}} \tag {1}
$$

式中： $c$ ——氯化钠标准溶液浓度， $\mathrm{mol / L}$

$V$ ——滴定氯化钠标准溶液时，硝酸银标准溶液用量， $\mathrm{ml}$ ；

$V_{0}$ ——滴定空白溶液时，硝酸银标准溶液用量，ml。

注：若能获得硝酸银基准试剂，可用直接法配制标准溶液，免去标定步骤。

# 5.19 氰化钾（KCN）标准溶液

# 5.19.1 氰化钾标准储备溶液的配制和标定

(1) 配制。称取氰化钾（KCN，注意剧毒！） $0.25\mathrm{g}$ 溶于 $1\%$ 氢氧化钠（5.5）中，稀释至 $100\mathrm{ml}$ ，混匀，储于棕色瓶中，避光保存。
(2）标定。吸取氰化钾标准储备溶液（5.19.1） $10.0\mathrm{ml}$ 于 $150~\mathrm{ml}$ 具柄瓷皿或 $250~\mathrm{ml}$ 锥形瓶中，加水 $50~\mathrm{ml}$ ，加 $2\%$ 氢氧化钠（5.4） $1\mathrm{ml}$ ，加试银灵指示剂（5.15） $0.2\mathrm{ml}$ ，用硝酸银标准溶液(5.18）滴定至溶液由黄色刚转变为橙红色为止。记录硝酸银标准溶液用量（ $V_{1}$ ）；同时另取实验用水 $60.0\mathrm{ml}$ 作空白试验，记录硝酸银标准溶液用量（ $V_{0}$ ）。

(3) 计算。氰化物含量 $c_{2}(\mathrm{mg} / \mathrm{ml})$ 以氰离子 $\left(\mathrm{CN}^{-}\right)$ 计, 按式 (2) 计算:

$$
c _ {2} = \frac {5 2 . 0 4 c \left(V _ {1} - V _ {0}\right)}{1 0 . 0} \tag {2}
$$

式中： $c$ —硝酸银标准溶液浓度， $\mathrm{mol / L}$

$V_{1}$ ——滴定氰化钾标准储备溶液时，硝酸银标准溶液用量，ml；

$V_{0}$ ——空白试验时, 硝酸银标准溶液用量, $\mathrm{ml}$ ;

52.04——相当于1L的 $1\mathrm{mol} / \mathrm{L}$ 硝酸银标准溶液的氰离子（ $\mathrm{CN}^{-}$ ）的质量，g；

10.0——氰化钾标准储备溶液体积，ml。

5.19.2 氰化钾标准中间溶液（每 $1.0\mathrm{ml}$ 含 $10.0\mu \mathrm{g}$ 氰离子，以下简写为 $10.0\mu \mathrm{gCN}^{-} / \mathrm{ml}$ ）。

先按式（3）计算出配制 $V$ 体积浓度为 $10.0\mu \mathrm{g}\mathrm{CN}^{-} / \mathrm{ml}$ 氰化钾标准中间溶液时，应吸取氰化钾标准储备溶液（5.19.1）的体积 $V_{1}$ (ml)：

$$
V _ {1} = \frac {1 0 . 0 V}{T \times 1 0 0 0} \tag {3}
$$

式中：10.0—1ml氰化钾标准中间溶液所含的氰离子质量， $\mu \mathrm{g}$

$V$ ——所欲配制的氰化钾中间标准溶液的体积，ml；

$T$ —氰化钾标准储备溶液浓度， $\mathrm{mg / ml}$

准确吸取体积 $V_{1}$ 氰化钾标准储备溶液（5.19.1）于体积 $V$ 棕色容量瓶中，用氢氧化钠溶液（5.6）稀释至标线，摇匀。

# 5.19.3 氰化钾标准使用溶液 $(2.0\mu \mathrm{g}\mathrm{CN}^{-} / \mathrm{ml})$

临用前，吸取氰化钾标准中间溶液（5.19.2） $10.0\mathrm{ml}$ 于 $50\mathrm{ml}$ 棕色容量瓶中，用氢氧化钠溶液（5.6）稀释至标线，摇匀。

# 8 分析步骤

# 8.1 氰化氢的释放和吸收

8.1.1 称取固体废物试样 $10\mathrm{g}$ （精确至 $0.1\mathrm{g}$ ；若氰化物含量过高时可少取）或取样棉球于蒸馏瓶中，加水 $100\mathrm{ml}$ 。

8.1.2 向吸收瓶中加入氢氧化钠溶液（5.5） $5\mathrm{ml}$ ，作为吸收液。当样品中存在亚硫酸钠和碳酸钠时，可用 $4\%$ 氢氧化钠溶液（5.3） $5\mathrm{ml}$ 作为吸收液。

8.1.3 将馏出液导管连接在冷凝器出口上，下端插入吸收瓶吸收液液面下。检查蒸馏系统的气密性。

8.1.4 将 $\mathrm{Na_2}$ -EDTA溶液（5.7） $5\mathrm{ml}$ 加入蒸馏瓶中。

8.1.5 通入冷却水后，在蒸馏瓶中加入磷酸（5.1） $5\mathrm{ml}$ ，使 $\mathrm{pH} < 2$ ，立即盖好瓶塞，打开加热装置加热蒸馏。馏出液流速以 $2\sim 4\mathrm{ml / min}$ 为宜。

8.1.6 当吸收瓶中溶液接近 $50 \mathrm{ml}$ 时停止蒸馏，用少量水洗涤馏出液导管，取出吸收瓶，用水定容至 $50 \mathrm{ml}$ ，此溶液为总氰化物待测溶液，记为 $A$ 。

# 8.2 干扰物的检验和消除

# 8.2.1 干扰物的检验

# 8.2.1.1 固体废物中干扰物检验溶液的制备方法

称取固体废物试样 $10\mathrm{g}$ （精确至 $0.1\mathrm{g}$ ，置于 $250\mathrm{ml}$ 具塞锥形瓶或碘量瓶中，加入蒸馏水 $100\mathrm{ml}$ ，盖紧瓶盖，垂直固定于往复式水平振荡机上，调节频率至（ $110\pm 10$ ）次/min，在室温下振荡1h，静置 $2\mathrm{h}$ 后取下，在预先装好 $0.45\mu \mathrm{m}$ 滤膜或滤纸的真空过滤装置上过滤，收集全部滤出液即为浸出液记为B。

# 8.2.1.2 干扰物质的检验

(1) 活性氯等氧化剂的检验。取滤液 B1 滴, 滴在一片碘化钾 - 淀粉试纸 (5.9) 上, 加硫酸 (5.10) 酸化, 若试纸变蓝, 表明有活性氯等氧化剂存在。
(2) 硫化物的检验。取滤液 B1 滴, 滴加在乙酸铅试纸 (5.8) 上, 若试纸变黑, 说明有硫化物存在。

# 8.2.2 干扰物的消除

(1) 若固体废物试样中含有活性氯等氧化剂，则移取浸出液 $\mathrm{B}20 \mathrm{ml}$ ，投入碘化钾-淀粉试纸(5.9) $1 \sim 3$ 片，加硫酸（5.10）酸化，用亚硫酸钠溶液（5.11）滴定至蓝色褪去，记录亚硫酸钠溶液的体积用量，然后在固体废物蒸馏前加入上述2.5倍量的亚硫酸钠溶液，进行蒸馏。
(2) 若固体废物中含有大量硫化物, 则将浸出液 B 中加入适量碳酸镉或碳酸铅固体粉末, 滤除沉淀。沉淀用 $1\%$ 氢氧化钠溶液 (5.5) 洗涤, 合并滤液和洗涤液, 定容至 $50 \mathrm{~ml}$ , 另加水 $50 \mathrm{ml}$ 后供蒸馏。
(3) 若固体废物中含有大量亚硝酸根离子，可在蒸馏前加入适量氨基磺酸 (5.2)，一般 $1 \mathrm{mg}$ 亚硝酸根离子需加 $2.5 \mathrm{mg}$ 氨基磺酸。然后蒸馏。
(4) 固体废物馏出液中酚的含量低于 $500 \mathrm{mg} / \mathrm{L}$ 时不影响氰化物的测定。

8.3.5 向各比色管（6.4）中加入磷酸盐缓冲溶液（5.12） $5\mathrm{ml}$ ，混匀，迅速加入氯胺T溶液（5.13） $0.2\mathrm{ml}$ ，立即加塞，轻轻混匀，放置 $3\sim 5\mathrm{min}$ 。
8.3.6 向各管中加入异烟酸-吡唑啉酮溶液（5.14） $5\mathrm{ml}$ ，加水稀释至标线，混匀，在 $25\sim 35^{\circ}\mathrm{C}$ 的水浴中放置 $40\mathrm{min}$ 。
8.3.7 用分光光度计在 $638~\mathrm{nm}$ 波长处，用 $10\mathrm{mm}$ 比色皿，以试剂空白（零浓度）做参比，测定吸光度，绘制工作曲线。

# 8.4 测定

8.4.1 称取固体废物 $10\mathrm{g}$ (精确至 $0.1\mathrm{g}$ ), 或取样棉球置于 $500\mathrm{ml}$ 蒸馏瓶 (2) 中, 加水 $100\mathrm{ml}$ ; 以下按 8.1.2 至 8.1.6 步骤操作。
8.4.2 移取固体废物蒸馏液 $10.0\mathrm{ml}$ 于比色管（6.4）中，按8.3.5至8.3.7步骤操作。
8.4.3 从工作曲线上查得相应的氰离子的含量。
8.4.4 在整个分析测定过程中，所有溶液（蒸馏吸收液、标准溶液、标准色列溶液、所有稀释溶液等）的氢氧化钠浓度必须相同，溶液 $\mathrm{pH}$ 值必须控制在 $6.8 \sim 7.5$ 。

# 9 结果计算

9.1 固体残渣上总氰化物的含量 $c_{3}$ （ $\mathrm{mg / kg}$ ，干基）以氰离子（ $\mathrm{CN^{-}}$ ）计，按式（4）计算：

$$
c _ {3} = \frac {m}{W - W _ {\mathrm {a}}} \cdot \frac {V _ {1}}{V _ {2}} \tag {4}
$$

式中： $m$ ——从工作曲线上查得的试份（比色时分取固体废物馏出液）中氰离子的含量， $\mu \mathrm{g}$

$W$ ——测定时称取的固体废物试样的质量，g；

$W_{\mathrm{a}}$ ——固体废物的含水量，g；

$V_{1}$ ——试样馏出液的总体积，ml；

$V_{2}$ ——试份（比色时分取馏出液）的体积， $\mathrm{ml}$ 。

9.2 弹（筒）片等废物表面上总氰化物的染毒密度 $c_{3}$ $(\mathrm{mg / m^2})$ 按公式（5）计算：

$$
c _ {3} = \frac {m}{S} \cdot \frac {V _ {1}}{V _ {2}} \tag {5}
$$

式中： $m$ ——从工作曲线上查得的试份（比色时分取固体废物馏出液）中氰离子的含量， $\mathrm{mg}$

$S$ ——擦拭弹（筒）片表面积， $\mathfrak{m}^2$

$V_{1}$ ——试样馏出液的总体积，ml；

$V_{2}$ ——试份（比色时分取馏出液）的体积， $\mathrm{ml}$ 。
