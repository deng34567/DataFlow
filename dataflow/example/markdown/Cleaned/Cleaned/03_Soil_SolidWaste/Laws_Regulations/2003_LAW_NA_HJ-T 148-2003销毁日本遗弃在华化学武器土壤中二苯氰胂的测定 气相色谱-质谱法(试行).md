Determination of diphenylarsine cyanide in soil for the destruction of

# 1 范围

本标准规定了测定土壤中二苯氰脾的气相色谱-质谱法。

本标准适用于销毁日本遗弃在华化学武器全过程污染土壤中二苯氰肿的测定，同时也适用于二苯氯肿的测定。

本标准的最低检测含量（干基）为 $0.02\mathrm{mg / kg}$

# 3 方法原理

用二氯甲烷萃取土壤样品，萃取液经过滤净化、1-丁硫醇试剂衍生，衍生后的溶液加入内标二苯并噻吩溶液后，采用毛细管气相色谱-质谱联用选择离子检测方式进行分离和测定。

# 4 试剂和材料

# 4.1 载气和辅助气体

4.1.1 载气：氦气，纯度不小于 $99.999\%$

4.1.2 样品浓缩气：氮气，纯度不小于 $99.999\%$

# 4.2 试剂

4.2.1 正已烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ，分析纯。
4.2.2 二氯甲烷（ $\mathrm{CH}_2\mathrm{Cl}_2$ ），分析纯。
4.2.3 甲醇 $\mathrm{(CH_3OH)}$ ，分析纯。
4.2.4 内标物：二苯并噻吩 $\mathrm{(C_{12}H_8S)}$ ，纯度不小于 $98\%$ 。
4.2.5 衍生化试剂：1-丁硫醇 $\left(\mathrm{C}_{4} \mathrm{H}_{10} \mathrm{~S}\right)$ ，纯度不小于 $98 \%$ 。

# 4.3 标准样品

二苯氰肿：纯度不小于 $98\%$ 。

# 4.4 标准溶液配制

# 4.4.1 二苯氰肿贮备溶液配制

称取二苯氰肿标样（4.3）约 $20\mathrm{mg}$ ，精确至 $0.1\mathrm{mg}$ ，置于 $10\mathrm{ml}$ 容量瓶内，用正己烷溶解定容，于 $4\mathrm{C}$ 保存，保存期为 $90\mathrm{d}$ 。

# 4.4.2 二苯氰肿中间溶液配制

用移液管取 $0.50\mathrm{ml}$ 二苯氰肿贮备溶液（4.4.1）于 $10\mathrm{ml}$ 容量瓶中，用二氯甲烷定容。该溶液浓度约为 $100\mu \mathrm{g / ml}$ 。

# 4.4.3 二苯氰肿标准溶液配制

用移液管取一定量的二苯氰肿中间溶液（4.4.2）于 $10\mathrm{ml}$ 容量瓶中，用二氯甲烷定容，配制成浓度为 $0.5\sim 10\mu \mathrm{g / ml}$ 的二苯氰肿系列标准溶液。

# 4.4.4 内标贮备溶液配制

称取二苯并噻吩（4.2.4）约 $20\mathrm{mg}$ ，精确至 $0.1\mathrm{mg}$ ，置于 $10\mathrm{ml}$ 容量瓶内，用二氯甲烷溶解定容，于 $4\mathrm{C}$ 保存，保存期为 $90\mathrm{d}$ 。

# 4.4.5 内标标准溶液配制

用移液管取一定量的内标贮备溶液（4.4.4）于 $50~\mathrm{ml}$ 容量瓶中，用二氯甲烷定容，配制成浓度约为 $80~\mu \mathrm{g / ml}$ 的内标标准溶液，于 $4^{\circ}\mathrm{C}$ 保存，保存期为 $30\mathrm{d}$ 。

# 4.5 材料

4.5.1 微孔滤膜过滤器：有机系列，孔径为 $0.45\mu \mathrm{m}$ 。
4.5.2 一次性注射器： $10\mathrm{ml}$

# 5 仪器

# 5.1 气相色谱-质谱联用仪

质谱仪具有EI源和选择离子检测功能。

# 5.2 色谱柱

$5\%$ 二苯基 $95\%$ 甲基硅酮固定相的石英毛细管柱， $30\mathrm{m}\times 0.25\mathrm{mm}\times 0.25\mu \mathrm{m}$ （长 $\times$ 内径 $\times$ 膜厚）。

# 5.3 其他仪器

5.3.1 康氏振荡器。
5.3.2 离心机：最高转速为 $20000\mathrm{r / min}$

# 6样品

# 6.1 样品采集

选择具有代表性的地段，采集样品布点应因地制宜，可采用梅花形或直线形等方式均匀布点或根据需要采集。每点采集的土壤，剔除杂质后装入样品瓶中，内外各有一张标签（写明编号、采样地点、土壤名称、深度以及采样日期和采样量等信息），做好标识和记录。

# 6.2 样品保存

样品采集后于 $4\%$ 以下保存，尽快分析。

# 6.3 试样制备

# 6.3.1 土样的提取及净化

称取土样 $10\mathrm{g}$ ，精确至 $0.1\mathrm{g}$ ，置于 $50\mathrm{ml}$ 具塞锥形瓶中，加入 $10.0\mathrm{ml}$ 二氯甲烷，放入康氏振荡器中在室温下振荡 $10\mathrm{min}$ 。静置 $5\mathrm{min}$ ，待微粒沉降。如固体微粒不易沉降，取上层液体于离心试管中，以 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，使其分层。用一次性 $10\mathrm{ml}$ 注射器取上层清液约 $3\mathrm{ml}$ ，用微孔滤膜过

滤器（4.5.1）过滤。

# 6.3.2 二苯氰肿的衍生

取 $2.0\mathrm{ml}$ 土样提取滤液（6.3.1）或适当浓度的二苯氰肿标准溶液（4.4.3）至 $5\mathrm{ml}$ 具塞试管中，加入 $70\mu \mathrm{l}$ 1-丁硫醇，再加入 $50\mu \mathrm{l}$ 甲醇，在 $37\%$ 的恒温水浴中衍生化 $40\mathrm{min}$ ，在常温下静置 $10\mathrm{min}$ ，加入 $30\mu \mathrm{l}$ 内标标准溶液（4.4.5），摇匀后供气相色谱-质谱测定。作土样最低检出含量时，不加内标，用微弱的氮气流将衍生液浓缩至 $0.5\mathrm{ml}$ ，供气相色谱-质谱测定。

# 6.4 水分测定

水分测定按GB7172—87中规定执行。

# 7 分析步骤

# 7.1 色谱条件

7.1.1 进样口温度： $280~\mathrm{^\circ C}$
7.1.2 柱温：初始温度 $150~\mathrm{^\circ C}$ ，保持 $2.5\mathrm{min}$ ，再以 $22\mathrm{^\circ C / min}$ 的速度升温到 $280\mathrm{^\circ C}$ ，保持 $3\mathrm{min}$ 。
7.1.3 载气流速： $1\mathrm{ml} / \mathrm{min}$
7.1.4 进样方式：不分流进样，不分流时间为 $0.7\mathrm{min}$
7.1.5 进样量： $1.0\mu \mathrm{l}$

# 7.2 质谱条件

7.2.1 接口温度： $280~\mathrm{^\circ C}$
7.2.2 离化方式：EI。
7.2.3 离子源温度： $180~\mathrm{^\circ C}$
7.2.4 电子能量： $70\mathrm{eV}$
7.2.5 选择离子质量： $\mathrm{m / z184}$ 、 $\mathrm{m / z227}$ 、 $\mathrm{m / z261}$
7.2.6 每一选择离子扫描时间： $0.05\mathrm{s}$ 或 $0.1\mathrm{s}$ ，根据仪器设定。

# 7.3 仪器的检测限

以强度为二倍的噪声所对应的二苯氰肿的量作为仪器的检测限，本方法要求仪器的最低检测限小于 $1 \times 10^{-10} \mathrm{~g}$ 。

# 7.4 校准

# 7.4.1 调谐仪器

在进行标准曲线测定前，对质谱仪进行调谐、校正。

# 7.4.2 标准曲线绘制

取不同浓度（至少6个点）的二苯氰脾系列标准溶液（4.4.3）按6.3.2进行衍生并加内标后，在7.1和7.2的条件下分别测定它们的气相色谱-质谱响应值，绘制响应值与浓度对应关系的曲线，并进行线性回归，回归方程见公式（1）。

$$
y = a + b \left(A _ {i} / A _ {s}\right) \tag {1}
$$

式中： $y$ ——试样中二苯氰脾的浓度， $\mu \mathrm{g / ml}$

$b$ ——标准曲线的斜率；

$A_{i}$ 二苯氰脾的峰面积；

$A_{s}$ ——内标物的峰面积；

$a$ ——标准曲线的截距。

# 7.4.3 曲线的校正

每批样品测定前后用单点中间浓度二苯氰肿标准溶液进行校正，若相对误差大于 $15\%$ ，应重新绘制标准曲线。

# 7.5 样品测定

# 7.5.1 定性分析

根据标准样品的色谱保留时间及离子丰度来确定被测样品二苯氰肿的存在，所测组分保留时间窗口为 $\pm 10\mathrm{s}$ 。应在色谱图上标明属于二苯氰肿的峰。土样中二苯氰肿经提取衍生后样品的选择离子色谱图见图1。

# 7.5.2 定量分析

# 7.5.2.1 分析

将6.3.2制备的溶液，在7.1和7.2的条件下分别测定它们的气相色谱-质谱响应值，每份溶液进样三次，取平均值。

# 7.5.2.2 计算

土样干基中二苯氰肿的含量按公式（2）计算：

$$
c = 1 0 ^ {- 3} \mathrm {y V} / \mathrm {m} \tag {2}
$$

式中： $c$ 一土样中二苯氰肿的含量， $\mathrm{mg / kg}$

$y$ ——按公式（1）计算的试样溶液中二苯氰脾的浓度， $\mu \mathrm{g} / \mathrm{ml}$ ；

$V$ ——加入试样中提取液的体积，ml；

$m$ ——土样的质量（干基）， $\mathrm{kg}$ 。
