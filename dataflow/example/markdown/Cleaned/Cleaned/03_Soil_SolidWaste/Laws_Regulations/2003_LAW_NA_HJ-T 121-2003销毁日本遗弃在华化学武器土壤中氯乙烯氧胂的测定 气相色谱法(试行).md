# 销毁日本遗弃在华化学武器

# 土壤中氯乙烯氧肿的测定气相色谱法（试行）

Determination of chlorovinylarsinoxide in soil for the destruction of

chemical weapons abandoned by Japan in China

Gas chromatography

警告：氯乙烯氧肿具有糜烂性，操作时应按规定佩戴防护器材，保持良好通风。测试后的残渣液应进行安全处理。

# 1. 范围

本标准规定了用气相色谱法测定土壤中氯乙烯氧肿的方法。

本标准适用于销毁日本遗弃在华化学武器全过程土壤中氯乙烯氧脾的监测，若土壤中含有路易氏剂或氯乙烯亚砷酸，也将同时被测定。

本方法的检出限为 $0.2\mathrm{mg / kg}$

# 3 原理

用2，3-二硫基丙醇对土壤中的氯乙烯氧肿（CVAO）、路易氏剂（ $\mathrm{L}_1$ ）和氯乙烯亚肿酸（CVAA）进行衍生化，以苯萃取后，用具火焰光度（硫）检测器的毛细管气相色谱进行测定，根据色谱峰的保留时间定性，外标法定量。反应式如下：

$$
\begin{array}{c} \mathrm {C l C H} = \mathrm {C H A s} = \mathrm {O} + \mathrm {H O C H} _ {2} \mathrm {C H} (\mathrm {S H}) \mathrm {C H} _ {2} \mathrm {S H} \longrightarrow \mathrm {C l C H} = \mathrm {C H A s} \\ \mathrm {C V A O} \end{array} \begin{array}{c} \mathrm {S} \\ \mathrm {S} \\ \mathrm {C H} _ {2} \mathrm {O H} \end{array} + \mathrm {H} _ {2} \mathrm {O}
$$

$$
\begin{array}{c} \mathrm {C l C H} = \mathrm {C H A s C l} _ {2} + \mathrm {H O C H} _ {2} \mathrm {C H} (\mathrm {S H}) \mathrm {C H} _ {2} \mathrm {S H} \longrightarrow \mathrm {C l C H} = \mathrm {C H A s} \\ \mathrm {L} _ {1} \end{array} \begin{array}{c} \mathrm {S} \\ \mathrm {S} \\ \mathrm {C H} _ {2} \mathrm {O H} \end{array} + 2 \mathrm {H C l}
$$

$$
\begin{array}{c} \mathrm {C l C H} = \mathrm {C H A s} (\mathrm {O H}) _ {2} + \mathrm {H O C H} _ {2} \mathrm {C H} (\mathrm {S H}) \mathrm {C H} _ {2} \mathrm {S H} \longrightarrow \mathrm {C l C H} = \mathrm {C H A s} \left. \begin{array}{c} \mathrm {S} \\ \mathrm {S} \end{array} \right| _ {\mathrm {C H} _ {2} \mathrm {O H}} + 2 \mathrm {H} _ {2} \mathrm {O} \\ \mathrm {C V A A} \end{array}
$$

# 4 试剂与材料

除非另有说明，在分析中仅使用确认为分析纯的试剂和蒸馏水或去离子水或相当纯度的水。

4.1 2,3-二硫基丙醇 $\left(\mathrm{C}_{3} \mathrm{H}_{8} \mathrm{OS}_{2}\right)$ ：含量不小于 $97.0 \%$ , 避光冷藏。

4.2 氯乙烯氧肿（ $\mathrm{C_2H_2AsClO}$ ）：含量不少于 $98\%$ ，密封、避光、冷藏。

4.3 2,3-二巯基丙醇乙醇溶液： $1 + 9$ ，将1体积的2,3-二巯基丙醇与9体积的乙醇相混合。

4.4 磷酸二氢钾溶液： $5\mathrm{g / L}$ ，将 $0.5\mathrm{g}$ 磷酸二氢钾溶于水并稀释至 $100\mathrm{ml}$ 。

4.5 明矾溶液： $20\mathrm{g/L}$ ，将 $2\mathrm{g}$ 明矾溶于水并稀释至 $100\mathrm{ml}$ 。

# 4.6 标准贮备液

准确称取 $10\mathrm{mg}$ 氯乙烯氧肿（4.2）（准确到 $0.1\mathrm{mg}$ ）溶于水后，移至 $100\mathrm{ml}$ 容量瓶中，用水稀释至标线，在 $4\%$ 可保存 $15\mathrm{d}$ 。

# 4.7 标准工作液

根据检测器的灵敏度及线性要求，用水稀释标准贮备液（4.6），配制成 $5\sim 7$ 个浓度点的标准工作液，现配现用。

4.8 载气 $(\mathrm{N}_2)$ ：纯度不小于 $99.99\%$ 。
4.9 燃烧气 $(\mathrm{H}_2)$ ：纯度不小于 $99.99\%$ 。
4.10 助燃气：经过净化的空气。

# 5 仪器

5.1 气相色谱仪：具火焰光度检测器（滤光片S， $394\mathrm{nm}$ ）。
5.2 色谱柱： $5\%$ 二苯基- $95\%$ 二甲基聚硅氧烷石英毛细管柱： $25\mathrm{m} \times 0.32\mathrm{mm} \times 0.5\mu \mathrm{m}$ （长度 $\times$ 内径 $\times$ 膜厚）。
5.3 往返式振荡器。
5.4 离心机： $3000\mathrm{r / min}$
5.5 微量注射器：1，10， $50\mu \mathrm{l}$
5.6 色谱工作站（或色谱数据处理机）。

# 6样品

# 6.1 采样步骤：

选择具有代表性的地段，采用梅花形或直线形等方式因地制宜均匀布点，或根据监测需要选点采集，剔除土壤中的杂质，装入样品袋中，内外各有一张标签（写明编号、采样地点、土壤名称、深度、筛孔以及采样日期和采样量等），做好标识和记录。采集的样品于 $4^{\circ}\mathrm{C}$ 保存，尽快分析。

# 6.2 试样的制备

# 6.2.1 样品的提取

称取土样 $4\mathrm{g}$ ，准确到 $0.1\mathrm{mg}$ ，加入 $20\mathrm{ml}$ 磷酸二氢钾溶液（4.4），手摇约 $1\mathrm{min}$ ，加入 $1\mathrm{ml}$ 明矾溶液（4.5）轻摇几下，静置 $30\mathrm{min}$ ，小心取出上层清液，再重复提取一次，合并提取的清液供衍生化。

# 6.2.2 样品的衍生化

将上述合并的清液全部移入具塞磨口锥形瓶中，加入约 $1\mathrm{ml}$ 浓盐酸，摇匀后，用微量注射器加入 $50\mu \mathrm{l}2,3$ -二巯基丙醇乙醇液（4.3）和 $0.1\mathrm{g}$ 抗坏血酸，振荡 $1\mathrm{min}$ 。室温下静置 $30\mathrm{min}$ ，使反应充分。

# 6.2.3 样品的萃取

吸取 $5.00\mathrm{ml}$ 苯于上述锥形瓶中，盖好塞子摇匀后，振荡 $10\mathrm{min}$ ，静置 $5\mathrm{min}$ 。取出苯层置于磨口试管中，用离心机离心 $10\mathrm{min}$ 。

# 6.2.4 样品的干燥

弃去上述试管底部的水层，在苯层中加入约 $0.3\mathrm{g}$ 无水硫酸钠，干燥，得苯层试液供色谱分析。

# 6.3 水分测定

按照GB7172—87《土壤水分测定法》对土样水分进行测定。

# 7 色谱测定操作步骤

# 7.1 色谱条件

7.1.1 汽化室温度： $250~\mathrm{^\circ C}$
7.1.2 检测室温度： $250\%$
7.1.3 柱温： $100\mathrm{^\circ C}$ （保持 $5\mathrm{min}$ ），以 $10\mathrm{^\circ C / min}$ 升温到 $220\mathrm{^\circ C}$ （保持 $5\mathrm{min}$ ）。
7.1.4 气体流速：氮气 $40\mathrm{ml / min}$ ；氢气 $55\mathrm{ml / min}$ ；空气 $50\mathrm{ml / min}$ 。
7.1.5 进样方式：不分流进样，不分流时间 $1\mathrm{min}$
7.1.6 进样量： $1\mu \mathrm{l}$

# 7.2 校准

# 7.2.1 定量方法：外标法。

# 7.2.2 绘制标准曲线

取 $40\mathrm{ml}$ 标准工作液（4.7），按6.2.2；6.2.3；6.2.4制成标样溶液。每个浓度点的标样溶液各取 $1\mu \mathrm{l}$ 进样，精确到 $0.1\mu \mathrm{l}$ ，得各个浓度的色谱峰和保留时间，每个浓度测3次，求峰面积平均值（要求3次测定的峰面积的相对偏差不大于 $10\%$ ）。氯乙烯氧脾的响应信号在 $0.5\sim 6.0\mathrm{mg / L}$ 之间呈良好线性关系。以峰面积平均值的对数值为纵坐标，氯乙烯氧脾浓度的对数值为横坐标，绘制标准曲线。

# 7.2.3 测定量校正因子

每天用单点校正法求校正因子，取与试样中含氯乙烯氧脾浓度相近的标样溶液，按7.2.2操作，测量色谱峰平均峰面积和保留时间，按式（1）计算校正因子 $f$

$$
f = \frac {\lg c}{\lg A} \tag {1}
$$

式中： $\lg c$ —标样浓度的对数；

$\lg A$ ——平均峰面积的对数。

# 7.3 测定

取 $1\mu \mathrm{l}$ 试样溶液，准确到 $0.1\mu \mathrm{l}$ ，注入色谱仪，按7.2.2操作，与标准样品保留时间比较定性，测量氯乙烯氧肿峰面积，每个试样测3次，求峰面积平均值。用外标法定量。标准气相色谱图见图1。

# 8 结果计算

8.1 从标准曲线上查出试样中氯乙烯氧肿的对数 $\lg c_{1}$ 或用单点校正法按式（2）计算：

$$
\lg c _ {1} = \lg A _ {1} \times f \tag {2}
$$

式中： $\lg A_{1}$ ——试样中氯乙烯氧肿峰面积的对数值；

$f$ ——由式（1）得到的校正因子。

8.2 土样中氯乙烯氧肿含量 $X(\mathrm{mg / kg})$ 按式（3）计算：

$$
X = \frac {V c _ {1}}{R W \left(1 - f ^ {\prime}\right)} \tag {3}
$$

式中： $V$ —土样前处理中萃取用苯的体积，L；

$c_{1}$ ——由8.1得出的试样溶液的浓度， $\mathrm{mg / L}$

$W$ ——土样的质量（干基）， $\mathrm{kg}$ ；

$f^{\prime}$ ——土样中水分含量 $(\%)$

$R$ ——回收率 $(\%)$ 。

# 9 精密度和准确度

# 9.1 精密度

氯乙烯氧肿重复测定的相对标准偏差不大于 $10\%$ 。

# 9.2 准确度

氯乙烯氧肿测定的相对误差不大于 $15\%$ 。
