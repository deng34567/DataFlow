# 1 范围

本标准规定了用气相色谱法分析土壤中苯氯乙酮的方法。

本方法适用于销毁日本遗弃在华化学武器全过程土壤中苯氯乙酮的监测。

本方法的检出限（干基）为 $0.1\mathrm{mg / kg}$

# 3 原理

用无水乙醇对土壤样品进行提取，用具氢火焰（FID）检测器的毛细管气相色谱仪对富集的苯氯乙酮进行测定，外标法定量。

# 4 试剂与材料

4.1 载气： $\mathbf{N}_2$ ，纯度 $99.999\%$
4.2 辅助气：氢气（纯度 $99.99\%$ ）和净化过的空气。
4.3 苯氯乙酮：含量不小于 $99.5\%$
4.4 苯氯乙酮标准贮备溶液（ $100\mathrm{mg / L}$ ）：称取苯氯乙酮（4.3） $0.05\mathrm{g}$ （准确到 $0.01\mathrm{mg}$ ）于干燥的烧杯中，加入一定量的无水乙醇溶解，待全部溶解后，用无水乙醇定容至 $500\mathrm{ml}$ ，在 $4^{\circ}\mathrm{C}$ 可保存 $15\mathrm{d}$ 。
4.5 无水乙醇：分析纯，经重蒸无明显测定干扰峰。

# 5 仪器

5.1 仪器：具氢火焰检测器的气相色谱仪。
5.2 微量进样器： $1\mu \mathrm{l}$ 、 $2\mu \mathrm{l}$
5.3 记录仪、积分仪或色谱工作站。
5.4 色谱柱： $100\%$ 二甲基聚硅氧烷石英毛细管柱， $25\mathrm{m}\times 0.32\mathrm{mm}\times 0.25\mu \mathrm{m}$ （长度 $\times$ 内径 $\times$ 膜

5.5 试样预处理时使用的仪器
5.5.1 微孔滤膜： $0.8\mu \mathrm{m}$
5.5.2 超声波振荡器， $500\mathrm{W}$

# 6样品

# 6.1 样品采集

选择具有代表性的地段，采用梅花形或直线形等方式因地制宜均匀布点，或根据监测需要选点采集，剔除土壤中的杂质，装入样品袋中，内外各有一张标签（写明编号、采样地点、土壤名称、深度、筛孔以及采样日期和采样量等)，做好标识和记录。

# 6.2 样品保存

采集的样品应在 $4\%$ 下保存，尽快分析。

# 6.3 样品制备

称取土样 $10\mathrm{g}$ （准确到 $1\mathrm{mg}$ ），置于一个有螺旋帽的玻璃瓶或具塞锥形瓶中，加入 $10\mathrm{ml}$ 无水乙醇，扭紧瓶塞，于振荡器上连续振荡 $10\mathrm{min}$ ，抽滤，并对土样进行3次洗涤，滤液定容为 $10\mathrm{ml}$ 或 $50\mathrm{ml}$ （根据含量高低确定），备分析用。

若萃取液浓度太低，可置于通风柜中自然蒸发浓缩，记录浓缩倍数。

# 6.4 水分测定

依照GB7172—87《土壤水分测定法》对土样水分进行测定。

# 7 分析步骤

# 7.1 色谱条件

7.1.1 进样口温度： $220\%$
7.1.2 色谱柱温度： $80\%$ 保留 $1\mathrm{min}$ ，升温速率 $20\% / \mathrm{min}$ ，终止温度为 $220\%$ ，保留 $1\mathrm{min}$ 。
7.1.3 检测器温度： $250\%$ 。
7.1.4 载气流量： $1.0\mathrm{ml / min}$
7.1.5 尾吹气流量： $29\mathrm{ml / min}$
7.1.6 进样方式：不分流进样，不分流时间为 $1\mathrm{min}$
7.1.7 氢气流量： $30\mathrm{ml / min}$
7.1.8 空气流量： $300\mathrm{ml / min}$
7.1.9 进样量： $1\mu \mathrm{l}$

# 7.2 校准

7.2.1 定量方法：外标法。
7.2.2 标准曲线的绘制

取7支 $25\mathrm{ml}$ 容量瓶，分别加入一定量的苯氯乙酮标准贮备溶液（4.4)，然后用无水乙醇稀释至标线，配制成浓度为1.00、10.0、20.0、40.0、60.0、80.0、 $100\mathrm{mg / L}$ 的工作液。待色谱基线稳定后，将上述标准溶液，用微量进样器进样 $1\mu \mathrm{l}$ 。各连续进样3次（要求3次测定的峰面积的相对偏差不大于 $10\%$ )，以苯氯乙酮峰面积或峰高为纵坐标，苯氯乙酮浓度为横坐标，绘制标准曲线。苯氯乙酮的浓度按公式（1）计算：

$$
y = a + b x \tag {1}
$$

式中： $y$ ——苯氯乙酮的峰面积或峰高；

$x$ ——苯氯乙酮的浓度， $\mu \mathrm{g / ml}$

$a$ ——标准曲线的截距；

$b$ ——标准曲线的斜率。

# 7.2.3 标准曲线的校正

标准曲线应每天校正一次，选标准曲线中间浓度进行校正，当相对误差大于 $10\%$ 时，应重新绘制标准曲线。

# 7.2.4 仪器的检测限

在仪器稳定条件下，配制一定浓度的苯氯乙酮溶液，以产生二倍噪声峰高所对应的苯氯乙酮含量计算，本仪器的检测限为 $1 \times 10^{-12} \mathrm{~g}$ 。

# 7.3.2 定量分析

按（7.2）的操作，对试液（6.3）进行分析，外标法定量。

# 7.4 计算

土壤中苯氯乙酮的含量按公式（2）计算：

$$
X = \frac {c V}{M K} \tag {2}
$$

式中： $X$ ——试样中苯氯乙酮的含量， $\mathrm{mg / kg}$

$c$ ——从标准曲线上查得苯氯乙酮的浓度， $\mu \mathrm{g} / \mathrm{ml}$

$V$ ——样品定容体积，ml；

$K$ ——体积浓缩倍数；

$M$ ——样品重量（干基），g。

# 8 结果表示

# 8.1 定性结果

根据标准色谱图中的苯氯乙酮的保留时间来确定被测样品中苯氯乙酮的存在。保留时间误差为 $\pm 10\mathrm{s}$ ，应在色谱图上标明属于苯氯乙酮的峰。

# 8.2 定量结果

# 8.2.1 含量表示方法

土壤中苯氯乙酮的含量以 $\mathrm{mg / kg}$ 表示。

# 8.2.2 精密度

相对标准偏差不大于 $10\%$ 。

# 8.2.3 准确度
