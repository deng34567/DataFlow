# 中华人民共和国国家标准

# 有色金属工业固体废物腐蚀性试验方法标准

UDC 628.191:699

.2/.8:543.06

GB 5087-85

Test method standard for corrosivity

of solid waste in nonferrous metal industry

为鉴别有色金属工业固体废物腐蚀性，特制订本标准。

# 1 试样的采集与制备

# 1.1 取样

试样的采集，按GB5086—85《有色金属工业固体废物浸出毒性试验方法标准》1.1执行。

# 1.2 试样制备

粉、粒和块状物料，在自然条件下干燥至水分小于 $5\%$ 。当样品粒度小于 $2\mathrm{mm}$ 时，取原样混匀，用四分法缩分，装瓶备用；当样品粒度部分大于 $2\mathrm{mm}$ 时，则以 $2\mathrm{mm}$ 筛筛分。筛上物经破碎筛分，直至全部通过，混匀后缩分，装瓶备用；当样品为大于 $2\mathrm{mm}$ 的块状时，经破碎筛分，直至全部通过 $2\mathrm{mm}$ 筛孔，混匀后缩分，装瓶备用。

泥状或浆状物料，要充分混匀后分装于玻璃或塑料容器中备用。

# 2 水分测定

2.1 取试样 $20\mathrm{g}$ ，在 $105^{\circ}\mathrm{C}$ 干燥，恒重至重量差小于 $1\mathrm{mg}$ ，计算含水量。

# 3 仪器与主要材料

# 3.1 仪器

pH计（最小指示值在0.1pH单位以下）及其配套电极。

# 3.2 混合容器

$\phi 130 \times 150$ ，2L，带盖聚乙烯瓶或其他类似形状的玻璃瓶。

# 3.3 振荡器

往复水平振荡式振荡器：振幅 $40\mathrm{mm}$ 。

# 3.4 蒸馏水

新鲜蒸馏水， $\mathrm{pH}$ 值为 $6.7\sim 7.0$

# 3.5 其他

标准溶液若干。

# 4 测定方法

4.1 用玻璃电极测定 $\mathrm{pH}$ 值, 每次测定前要用相应的标准溶液对 $\mathrm{pH}$ 计进行校正, 并应有温度补偿。
4.2 对含水分很高呈流态状的物料, 如稀泥和薄浆等, 可直接插入电极进行测量, 也可经离心分离或过滤后, 测其清液的 $\mathrm{pH}$ 值。
4.3 对颗粒状物料, 取上述1.2制备好的样品 $50 \mathrm{~g}$ (干基), 放入混合容器, 加新鲜蒸馏水 $250 \mathrm{ml}$ , 使其固液比为 $1:5$ , 加盖后固定在振荡器上, 以振荡频率为 $120 \pm 5$ 次 $/\mathrm{min}$ , 振幅 $40 \mathrm{~mm}$ , 振荡时间

30min。振荡结束后，放置30min，测其上清液的pH值。

4.4 粘稠状污泥或其他含水分较高, 但呈非流态状物料, 可采用离心或过滤方法, 分离出其中所含液体进行 $\mathrm{pH}$ 值测量。若分离不出足够量的测量液时, 则按4.3规定的程序制备测量液。
4.5 测定时，应先用水冲洗经校正后的 $\mathrm{pH}$ 计的电极 $3 \sim 5$ 次，再将电极插入测量液，摇动测量液 $1 \mathrm{~min}$ 后，读取 $\mathrm{pH}$ 值，读数保持稳定至少 $30 \mathrm{~s}$ 。
4.6 当 $\mathrm{pH}$ 值大于9时，为了消除“钠差”，可使用低钠差电极，或选用与被测溶液 $\mathrm{pH}$ 值相近的标准溶液对仪器进行反复校正。

# 5. 数据处理与报告

5.1 每个样品至少做三次平行试验；其标准差不超过 $\pm 0.15\mathrm{pH}$ 单位，取算术平均值报告试验结果。对高于 $\mathrm{pH}$ 值（9以上）的测量，允许标准差 $\pm 0.2\mathrm{pH}$ 单位。

当标准差超过规定范围时，必须分析并报告原因。
