# 镍氢电池材料废弃物回收利用的处理方法

# 1 范围

本标准规定了镍氢电池材料废弃物回收利用的术语和定义、方法提要、原辅料和设备、处理条件及工艺控制要求、环境保护和安全要求。

本标准适用于镍氢电池材料废弃物中镍、铜、稀土的湿法回收处理方法。

# 4 方法提要

镍氢电池材料废弃物中的金属铜利用机械物理法分离回收,其他金属及其化合物溶于酸,转化为易溶于水的离子形态。通过分离、富集实现稀土、镍的回收。

# 65 工艺流程

651 镍氢电池材料废弃物先进行粗破。
652 粗破后的物料经热处理去除隔膜、粘结剂等。
653 热处理后应直接采用机械法分离回收铜、泡沫镍。
654 分离后的剩余物料应先进行酸溶,得到的溶液进行镍、稀土元素的回收。
655 回收稀土复盐后的溶液经萃取、反萃回收得到镍盐纯化液。
656 得到镍盐纯化液用于生产化工镍盐、合成电池生产原料等。
657 镍氢电池材料废弃物经碱性硫酸盐沉淀后得到稀土复盐通过电冶金方法加工获得各稀土元素。

# 66 工艺控制要求

661 通过热处理去除镍氢电池材料废弃物中的隔膜粘结剂等去除率应不低于 $9 9 \%$ 计算方法见附录的。
662 通过机械分离获得铜,回收率应不低于 $90 \%$ 。计算方法见。
663 控制浸出工艺条件,镍氢电池材料废弃物中镍元素的浸出率应不低于 $9 9 \%$ ,稀土元素的浸出率应不低于 $9 5 \%$ 。计算方法见。
664 采用碱性硫酸盐法沉淀浸出液中的稀土元素,各稀土元素的回收率应不低于 $9 5 \%$ 。
665 采用调节 $\mathrm { p H }$ 和萃取除杂工艺,铜、铁、铝、钙、镁等的去除率应不低于 $9 9 \%$ 。计算方法见。
666 采用萃取法分离提纯镍元素,镍元素的损失率应不高于 $1 \%$ ,回收率应不低于 $98 \%$ 。计算方法见和。
667 镍氢电池材料废弃物中金属元素含量的检测方法参见附录。

# 计算方法

# A1 热处理隔膜、粘结剂等去除率的计算

热处理隔膜、粘结剂去除率以 $r _ { 1 }$ 计,按式( )计算:

$$
r_{1} = \frac{m_{1}}{m_{2}}\times 100\% \tag{A.1}
$$

式中:

$m _ { 1 }$ 热处理后镍氢电池材料废弃物经 $4 0 0 \ { } ^ { \circ } \mathrm { C } \sim 6 0 0 \ { } ^ { \circ } \mathrm { C }$ 灼烧后的质量的数值，单位为克(g)；

$m _ { 2 }$ -热处理后镍氢电池材料废弃物未经灼烧的质量的数值，单位为克（g)。

# A2 铜元素回收率的计算

铜元素回收率以 $R _ { 1 }$ 计,按式( )计算:

$$
R_{1} = \frac{m_{2}}{m_{1}}\times 100\% \tag{A.2}
$$

式中:

m2 -1 t镍氢电池材料废弃物经破碎、分选,回收的铜元素的质量的数值，单位为克(g);

$m _ { 1 }$ ——1t镍氢电池材料废弃物中铜元素的质量的数值,单位为克(g)。

# A 镍元素稀土浸出率的计算

镍元素、稀土浸出率以 $\boldsymbol { e } _ { i }$ 计,按式( )计算:

$$
e _ {i} = \frac {\rho_ {i 1} \times V}{m _ {i}} \times 100 \% \tag{A.3}
$$

式中:

$\rho _ { i 1 }$ —1t镍氢电池材料废弃物经酸溶,浸出液中金属元素i的浓度的数值,单位为克每升(g/L);

$V$ ———镍氢电池材料废弃物经酸溶后浸出液的体积的数值,单位为升();

$m _ { i }$ ——1t镍氢电池材料废弃物中金属元素 $i$ 的质量的数值，单位为克(g)。

注 i 代表镍元素稀土元素

# A4 浸出液中铜铁铝钙镁元素去除率的计算

浸出液中铜、铁、铝、钙、镁元素去除率以 $r _ { k }$ 计,按式( )计算:

$$
r _ {k} = \left(1 - \frac {\rho_ {k 1} \times V _ {1}}{\rho_ {k 2} \times V}\right) \times 100 \% \quad \dots \tag{A.4}
$$

式中:

$\rho _ { { } _ { k 1 } }$ —1t镍氢电池材料废弃物经酸溶的浸出液除杂后溶液中金属元素k的浓度的数值，单位为克每升 ${ ( \mathrm { g / L ) } }$ );

$V _ { 1 }$ ——1t镍氢电池材料废弃物经酸溶的浸出液除杂后溶液体积的数值,单位为升(L);

$\rho _ { { } _ { k 2 } }$ Pk2 -1t镍氢电池材料废弃物经酸溶后浸出液中金属元素 $k$ 的浓度的数值，单位为克每升 ${ ( \mathrm { g / L ) } }$ ；

———镍氢电池材料废弃物经酸溶后浸出液的体积的数值,单位为升()。

注:k 浸出液中铜、铁、铝、钙、镁金属元素。

# A5 镍元素损失率的计算

镍元素损失率以 l 计,按式( )计算:

$$
l = \left(1 - \frac {\rho_ {2} \times V _ {1}}{\rho_ {1} \times V}\right) \times 100 \% \quad \dots \tag{A.5}
$$

式中:

$\rho _ { 2 }$ -1 t镍氢电池材料废弃物经处理,得到镍纯化液中镍元素的浓度的数值,单位为克每升 ${ ( \mathrm { g / L ) } }$ ；

$V _ { \bar { 1 } }$ —1t镍氢电池材料废弃物经处理,得到镍纯化液的体积的数值,单位为升(L);

$\rho _ { 1 }$ —1t镍氢电池材料废弃物经酸溶,浸出液中镍元素的浓度的数值，单位为克每升 ${ ( \mathrm { g / L ) } }$ ；

$V$ ———镍氢电池材料废弃物经酸溶,浸出液的体积的数值,单位为升()。

# A6 镍元素、稀土回收率的计算

镍元素稀土元素回收率以 $R _ { i }$ 计按式计算

$$
R _ {i} = \frac {\rho_ {i 2} \times V _ {1}}{m _ {i}} \times 100 \% \tag{A.6}
$$

式中:

$\rho _ { { \scriptscriptstyle i } 2 }$ —1t镍氢电池材料废弃物经处理,得到纯化液中金属元素i的浓度的数值，单位为克每升 ${ \mathrm { ( g / L ) } }$ ；

$V _ { 1 }$ ——1t镍氢电池材料废弃物经净化、萃取处理,得到纯化液的体积的数值,单位为升(L);

$m _ { i }$ ——1t镍氢电池材料废弃物中元素i的质量的数值,单位为克(g)。

注: $i$ 代表镍元素、稀土元素。
