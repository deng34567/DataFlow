# 中华人民共和国国家标准

# 有色金属工业固体废物急性毒性初筛试验方法标准

Screening test method standard for acute toxicity of solid waste in nonferrous metal industry

UDC 628.191 :669.2/.8 :543.06 GB 5088-85

为初步鉴别有色金属工业固体废物的急性毒性，特制订本标准。

# 1 样品的采集与制备

1.1 采样：按GB5086—85《有色金属工业固体废物浸出毒性试验方法标准》1.1执行。
1.2 浸出液制备：将样品 $100\mathrm{g}$ ，置于三角瓶中，加入 $100\mathrm{ml}$ 蒸馏水（即固液比为 $1:1$ ），在室温条件下静止浸泡 $24\mathrm{h}$ ，用滤纸过滤，测定 $\mathrm{pH}$ 值。滤液留待灌胃实验用。

# 2 实验方法

2.1 实验动物：以体重 $18\sim 24\mathrm{g}$ 的小鼠（或体重为 $200\sim 300\mathrm{g}$ 的大鼠）作为实验动物。
2.2 灌胃：按照制订有害物质卫生标准的急性毒性试验的灌胃方法，对10只小鼠（或大鼠）进行一次灌胃。

# 3 结果判定

对灌胃后的小鼠（或大鼠）进行中毒症状观察，记录48h内实验动物的死亡数。根据实验结果，对固体废物的综合毒性作出初步评价，如出现半数以上小鼠（或大鼠）死亡者，判定为有害固体废物；死亡不足半数（或全部存活）者，判定为一般固体废物。

附加说明：

本标准由原国务院环境保护领导小组提出。

本标准由中国有色金属工业总公司《有色金属工业固体废物污染控制标准》编制组起草。

本标准主要起草人潘顺昌、范继仲、李少时、王韧、水志良。

本标准委托国家环境保护局负责解释。
