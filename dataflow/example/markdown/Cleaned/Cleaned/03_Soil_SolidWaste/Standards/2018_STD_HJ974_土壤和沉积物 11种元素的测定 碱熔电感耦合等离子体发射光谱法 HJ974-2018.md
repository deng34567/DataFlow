# 土壤和沉积物 11 种元素的测定碱熔-电感耦合等离子体发射光谱法

警告：实验中所用的硝酸和盐酸具有强氧化性、强刺激性和腐蚀性，溶液配制及样品前处理过程应在通风橱中进行，操作时应注意佩戴防护器具，避免吸入呼吸道或直接接触皮肤和衣物。

# 1 适用范围

本标准规定了测定土壤和沉积物中 11种元素的碱熔-电感耦合等离子体发射光谱法。

本标准适用于土壤和沉积物中锰（Mn）、钡（Ba）、钒（V）、锶（Sr）、钛（Ti）、钙（Ca）、镁（Mg）、铁（Fe）、铝（Al）、钾（K）和硅（Si）等 11种元素的测定。

当取样量为 $0 . 2 \ \mathrm { g }$ ，定容体积为 $5 0 0 \mathrm { m l }$ 时，锰（ $\mathbf { M n }$ ）、钡（Ba）、钒（V）、锶（Sr）和钛（Ti）5 种金属元素的方法检出限为 $0 . 0 1 \ { \mathrm { g / k g } } { \sim } 0 . 0 2 \ { \mathrm { g / k g } }$ ，测定下限为 $0 . 0 4 ~ \mathrm { g / k g } \sim 0 . 0 8 ~ \mathrm { g / k g } ;$ ；钙（以 CaO 计）、镁（以 $\mathrm { M g O }$ 计）、铁（以 $\mathrm { F e } _ { 2 } \mathrm { O } _ { 3 }$ 计）、铝（以 $\mathrm { A l } _ { 2 } \mathrm { O } _ { 3 }$ 计）、钾（以 $\mathrm { K } _ { 2 } \mathrm { O }$ 计）和硅（以 $\mathrm { S i O } _ { 2 }$ 计）6种元素氧化物的方法检出限为 $0 . 0 1 \% { \sim } 0 . 0 7 \%$ ，测定下限为 $0 . 0 4 \% \sim 0 . 2 8 \%$ 。详见附录 A。

# 3 方法原理

样品与碱性熔剂熔融，熔融物经酸溶解后注入电感耦合等离子体发射光谱仪，目标元素在等离子炬中被气化、电离、激发并辐射出特征谱线。在一定浓度范围内，其特征谱线的强度与元素的浓度呈正比。

# 4 干扰和消除

# 4.1 光谱干扰

光谱干扰主要包括连续背景干扰和谱线重叠干扰。校正光谱干扰常用的方法是背景扣除

法（根据单元素试验确定扣除背景的位置及方式）及干扰系数扣除法。当存在单元素干扰时，可通过配制一系列已知干扰元素含量的溶液，在分析元素测定波长下测定其 $\rho ^ { \prime }$ ，根据公式（1）求出干扰系数 $K _ { t }$ 。

$$
K _ {t} = \frac {\left(\rho^ {\prime} - \rho\right)}{\rho_ {t}} \tag {1}
$$

式中： $K _ { t }$ ——干扰系数；

p' $\rho ^ { \prime }$ -干扰元素加分析元素在分析元素测定波长下测定的浓度， $\mathrm { m g / L }$ ；

——分析元素的浓度，mg/L； $\rho$ $\mathrm { m g / L }$

Pt_干扰元素的浓度，mg/L。 $\rho _ { t }$ $\mathrm { m g / L }$

不同仪器测定的干扰系数会有区别，分析元素测定波长下共存元素的干扰见附录 B。

# 4.2 非光谱干扰

非光谱干扰主要包括化学干扰、物理干扰以及去溶剂干扰等，在实际分析过程中各类干扰可能共同存在，是否予以补偿和校正，与样品中干扰元素的浓度有关。当样品中含有大量可溶盐或样品酸度过高时，会对测定产生干扰。可采用稀释样品（但应保证待测元素的含量高于测定下限）、内标法、优化仪器条件和基体匹配法（配制与待测样品基体成份相似的标准溶液）等措施消除和降低上述干扰。

# 7.4 试样的制备

先在铂金坩埚（6.5）底部加入少量的碳酸钠（5.3）垫底，称取 $1 . 0 \ \mathrm { g }$ 碳酸钠（5.3）、0.1$\mathbf { g }$ 四硼酸锂（5.4）和 $0 . 4 \ \mathrm { g }$ 偏硼酸锂（5.5），适当混匀制成熔剂，再依次加入约 2/3 的熔剂和$0 . 2 \ \mathrm { g }$ （精确至 $0 . 1 \mathrm { m g }$ ）样品（7.2），最后放入剩余的熔剂，使其铺在混合物表面。将铂金坩

埚置于马弗炉（6.4）中，升温至 $1 0 0 0 ^ { \circ } \mathrm { C }$ ，保持 $3 0 \mathrm { m i n }$ ，停止加热。约 $5 \mathrm { { m i n } }$ 后用坩埚钳夹住铂金坩埚直立于已盛有 $1 0 0 \mathrm { m l }$ 水的 $5 0 0 \mathrm { m l }$ 烧杯中，待熔融物出现裂纹后，取出坩埚并向坩埚内加水直至没过熔融物，当熔融物与坩埚脱离后，将脱落的熔融物转移至 $2 5 0 ~ \mathrm { m l }$ 烧杯中。取 $4 0 ~ \mathrm { m l }$ 硝酸-盐酸混合溶液（5.10），先用少许硝酸-盐酸混合溶液多次淋洗坩埚壁上的沉淀，淋洗液移入烧杯中，再用水冲洗坩埚，最后将剩余的硝酸-盐酸混合溶液加入烧杯，使熔融物全部溶解，将烧杯中的溶液转移至 $5 0 0 \mathrm { m l }$ 容量瓶中，用水定容至标线，待测。

# 9.1.1 土壤样品

土壤样品中元素锰（Mn）、钡（Ba）、钒（V）、锶（Sr）和钛（Ti）的含量 $w _ { 1 }$ （g/kg），按照公式（2）进行计算；钙（以CaO计）、镁（以MgO计）、铁（以 $\mathrm { F e } _ { 2 } \mathrm { O } _ { 3 }$ 计）、铝（以 $\mathrm { A l } _ { 2 } \mathrm { O } _ { 3 }$ 计）、钾（以 ${ \bf K } _ { 2 } { \bf O }$ 计）和硅（以 $\mathrm { S i O } _ { 2 }$ 计）的含量 $w _ { 2 }$ （ $\%$ ），按照公式（3）进行计算。

$$
w _ {1} = \frac {\left(\rho_ {1} - \rho_ {0}\right) \times V _ {1}}{m _ {1} \times w _ {d m}} \times 1 0 ^ {- 3} \tag {2}
$$

式中： $w _ { 1 }$ ——样品中待测元素的含量， $\mathrm { g / k g }$ ；

$\rho _ { 1 }$ ——试样中待测元素的质量浓度， $\mathrm { m g / L }$ ；

$\rho _ { 0 }$ —空白试样中待测元素的质量浓度， $\mathrm { m g / L }$ ；

$V _ { 1 }$ —试样的定容体积，ml；

$m _ { 1 }$ —称取样品的质量，g；

w dm—土壤样品中干物质含量， $\%$ 。

$$
w _ {2} = \frac {\left(\rho_ {1} - \rho_ {0}\right) \times V _ {2} \times M _ {2}}{m _ {2} \times w _ {d m} \times M _ {1}} \times 1 0 ^ {- 4} \tag {3}
$$

式中： $w _ { 2 }$ ——样品中待测元素的含量， $\%$ ；

$\rho _ { 1 }$ —试样中待测元素的质量浓度， $\mathrm { m g / L }$ ；

$\rho _ { 0 }$ —空白试样中待测元素的质量浓度， $\mathrm { m g / L }$ ；

$V _ { 2 }$ —试样的定容体积，ml；

$M _ { 2 } .$ —待测元素氧化物摩尔质量；

$m _ { 2 }$ —称取样品的质量，g；

w dm—土壤样品中干物质含量， $\%$ ；

$M _ { 1 }$ ——待测元素摩尔质量。

# 9.1.2 沉积物样品

沉积物样品中元素锰（Mn）、钡（Ba）、钒（V）、锶（Sr）和钛（Ti）的含量 $w _ { 3 }$ （g/kg），按照公式（4）进行计算；钙（以CaO计）、镁（以MgO计）、铁（以 $\mathrm { F e } _ { 2 } \mathrm { O } _ { 3 }$ 计）、铝（以 $\mathrm { A l } _ { 2 } \mathrm { O } _ { 3 }$ 计）、钾（以 ${ \bf K } _ { 2 } { \bf O }$ 计）和硅（以 $\mathrm { S i O } _ { 2 }$ 计）的含量 $w _ { 4 }$ （ $\%$ ），按照公式（5）进行计算。

$$
w _ {3} = \frac {\left(\rho_ {1} - \rho_ {0}\right) \times V _ {3}}{m _ {3} \times \left(1 - w _ {H _ {2} O}\right)} \times 1 0 ^ {- 3} \tag {4}
$$

式中： $w _ { 3 }$ —样品中待测元素的含量，g/kg；

$\rho _ { 1 }$ —试样中待测元素的质量浓度， $\mathrm { m g / L }$ ；

$\rho _ { 0 }$ —空白试样中待测元素的质量浓度， $\mathrm { m g / L }$ ；

$V _ { 3 }$ —试样的定容体积，ml；

$m _ { 3 }$ —称取样品的质量，g；

w H2O—沉积物样品的含水率， $\%$ 。

$$
w _ {4} = \frac {\left(\rho_ {1} - \rho_ {0}\right) \times V _ {4} \times M _ {2}}{m _ {4} \times \left(1 - w _ {H _ {2} O}\right) \times M _ {1}} \times 1 0 ^ {- 4} \tag {5}
$$

式中： $w _ { 4 }$ —样品中待测元素的含量， $\%$ ；

$\rho _ { 1 } .$ —试样中待测元素的质量浓度， $\mathrm { m g / L }$ ；

$\rho _ { 0 }$ —空白试样中待测元素的质量浓度， $\mathrm { m g / L }$ ；

$V _ { 4 }$ —试样的定容体积，ml；

$M _ { 2 }$ —待测元素氧化物摩尔质量；

$m _ { 4 }$ —称取样品的质量，g；

w H2O—沉积物样品的含水率， $\%$ ；

$M _ { 1 } .$ ——待测元素摩尔质量。
