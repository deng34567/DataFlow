# 前言

为贯彻《中华人民共和国环境保护法》和《中华人民共和国土壤污染防治法》，保护生态环境，保障人体健康，规范土壤和沉积物中六价铬的测定方法，制定本标准。

本标准规定了测定土壤和沉积物中六价铬的碱溶液提取-火焰原子吸收分光光度法。

# 土壤和沉积物六价铬的测定碱溶液提取-火焰原子吸收

# 分光光度法

# 1 适用范围

本标准规定了测定土壤和沉积物中六价铬的碱溶液提取-火焰原子吸收分光光度法。

本标准适用于土壤和沉积物中六价铬的测定。

当土壤和沉积物取样量为 $5 . 0 \ \mathrm { g }$ ，定容体积为 $1 0 0 ~ \mathrm { m l }$ 时，本标准测定的六价铬的方法检出限为 $0 . 5 ~ \mathrm { { m g / k g } }$ ，测定下限为 $2 . 0 \mathrm { m g / k g }$ 。

# 2 规范性引用文件

本标准引用了下列文件或其中的条款。凡是不注日期的引用文件，其有效版本适用于本标准。

# 3 方法原理

用 pH 不小于 11.5 的碱性提取液，提取出样品中的六价铬，喷入空气-乙炔火焰，在高温火焰中形成的铬基态原子对铬的特征谱线产生吸收，在一定范围内，其吸光度值与六价铬的质量浓度成正比。

# 4 干扰和消除

在碱性环境（ $\mathrm { \ p H } \geqslant 1 1 . 5 $ ）中，经氯化镁和磷酸氢二钾-磷酸二氢钾缓冲溶液抑制，样品中三价铬的存在对六价铬的测定无干扰。

# 5 试剂和材料

除非另有说明，分析时均使用符合国家标准的分析纯试剂，实验用水为新制备的去离子水。

5.1 硝酸： $\rho$ （ $( \mathrm { H N O _ { 3 } } ) = 1 . 4 2 \ : \mathrm { g / m l }$ ，优级纯。
5.2 碳酸钠（ $\mathrm { N a } _ { 2 } \mathrm { C O } _ { 3 }$ ）。
5.3 氢氧化钠（NaOH）。
5.4 氯化镁（ $\mathrm { { \bf M g C l } } _ { 2 } \mathrm { { \bf . } }$ ）。
5.5 磷酸氢二钾（ ${ \mathrm { K } } _ { 2 } { \mathrm { H P O } } _ { 4 }$ ）。
5.6 磷酸二氢钾（ ${ \mathrm { K H } } _ { 2 } { \mathrm { P O } } _ { 4 } .$ ）。
5.7 磷酸氢二钾-磷酸二氢钾缓冲溶液： $\mathrm { p H } = 7$ 。

称取 $8 7 . 1 \ \mathrm { g }$ 磷酸氢二钾（5.5）和 $6 8 . 0 \ \mathrm { g }$ 磷酸二氢钾（5.6）溶于水中，稀释定容至 1 L。

5.8 碱性提取溶液。

称取 $3 0 \ \mathrm { g }$ 碳酸钠（5.2）与 $2 0 \ \mathrm { g }$ 氢氧化钠（5.3）溶于水中，稀释定容至 1 L，贮存在密封聚乙烯瓶中。使用前必须保证其 pH值大于 11.5。

5.9 重铬酸钾（ $\mathrm { K } _ { 2 } \mathrm { C r } _ { 2 } \mathrm { O } _ { 7 } .$ ）：基准试剂。

称取 $5 . 0 \ \mathrm { g }$ 重铬酸钾于瓷坩埚中，在 $1 0 5 \mathrm { { ‰} }$ 干燥箱中烘 $^ { 2 \mathrm { ~ h ~ } }$ ，冷却至室温，保存于干燥器内，备用。

5.10 六价铬标准贮备液： $\rho { = } 1 0 0 0 ~ \mathrm { { m g / L } }$

准确称取 $2 . 8 2 9 \ \mathrm { g }$ （精确至 $0 . 1 \mathrm { { \ m g } }$ ）重铬酸钾（5.9）溶于水中，稀释定容至 1 L。也可直接购买市售有证标准物质或者有证标准溶液。

5.11 六价铬标准使用液： $\rho { = } 1 0 0 \ \mathrm { m g / L }$

准确移取 $1 0 . 0 \mathrm { m l }$ 六价铬标准贮备液（5.10）加入 $1 0 0 ~ \mathrm { m l }$ 容量瓶中，用水定容至标线，摇匀。常温保存 6 个月。

5.12 滤膜： $0 . 4 5 \mu \mathrm { m }$ 。

5.13 聚乙烯薄膜。

6.2 铬空心阴极灯或其他光源。

# 7 样品

# 7.1 样品采集和保存

按照HJ/T 166或HJ 25.2的相关要求进行土壤样品的采集和保存，按照HJ/T 91、HJ 494或HJ 495的相关要求进行水体沉积物样品的采集和保存，按照GB 17378.3的相关要求进行海洋沉积物样品的采集和保存。样品的采集与保存应使用塑料或玻璃的装置和容器，不得使用金属制品贮存器。

# 7.2 样品的制备

按照 HJ/T 166，将采集的样品在实验室中风干、破碎、过尼龙筛（6.7）、保存。

注：样品的干燥也可采用冻干法。

# 7.3 水分的测定

按照 HJ 613测定土壤样品（7.2）的干物质含量，按照 GB 17378.5 测定沉积物样品（7.2）的含水率。

# 7.4 试样的制备

准确称取 $5 . 0 ~ \mathrm { g }$ （精确至 $0 . 0 1 \ \mathrm { g }$ ）样品（7.2）置于 $2 5 0 ~ \mathrm { m l }$ 烧杯中，加入 $5 0 . 0 ~ \mathrm { m l }$ 碱性提取溶液（5.8），再加入 $4 0 0 \mathrm { m g }$ 氯化镁（5.4）和 $0 . 5 { \mathrm { m l } }$ 磷酸氢二钾-磷酸二氢钾缓冲溶液（5.7）。放入搅拌子，用聚乙烯薄膜（5.13）封口，置于搅拌加热装置（6.3）上。常温下搅拌样品5min 后，开启加热装置，加热搅拌至 $9 0 ^ { \circ } \mathrm { C } \sim 9 5 ^ { \circ } \mathrm { C }$ ，保持 $6 0 \ \mathrm { m i n }$ 。取下烧杯，冷却至室温。用滤膜（5.12）抽滤，将滤液置于 $2 5 0 ~ \mathrm { m l }$ 的烧杯中，用硝酸（5.1）调节溶液的 pH 值至 $7 . 5 { \pm } 0 . 5 $ 。将此溶液转移至 $1 0 0 \mathrm { m l }$ 容量瓶中，用水定容至标线，摇匀，待测。

注1：调节试样溶液pH值时，如果有絮状沉淀产生，需再用滤膜（5.12）过滤。
注 2：制备好的试样，若不能立即分析，在 $0 ^ { \circ } C { \sim } 4 ^ { \circ } C$ 下密封保存，保存期为 $3 0 \mathrm { d } .$

# 7.5 空白试样的制备

不加样品，按照与试样制备（7.4）相同的步骤制备空白试样。

# 9 结果计算与表示

# 9.1 结果计算

# 9.1.1 土壤样品的结果计算

土壤样品中六价铬的含量 $w ( \mathrm { m g / k g } )$ ，按照公式（1）进行计算：

$$
w = \frac {\rho \times V \times D}{m \times W _ {d m}} \tag {1}
$$

式中：w—土壤样品中六价铬的含量, $\mathrm { m g / k g }$ ；

$\rho$ -试样中六价铬的浓度， $\mathrm { m g / L }$ ；

V——试样定容体积，ml；

$D$ 试样稀释倍数；

m—称取土壤样品的重量，g；

$W _ { d m } .$ —土壤样品干物质含量， $\%$ 。

# 9.1.2 沉积物样品的结果计算

沉积物样品中六价铬的含量 $w ( \mathrm { m g / k g } )$ ，按照公式（2）进行计算：

$$
w = \frac {\rho \times V \times D}{m \times \left(1 - W _ {H : O}\right)} \tag {2}
$$

式中： $w \mathrm { . }$ ——沉积物样品中六价铬的含量, $\mathrm { m g / k g }$

$\rho$ -试样中六价铬的浓度， $\mathrm { m g / L }$ ；

V——试样定容体积，ml；

D—试样稀释倍数；

m—称取沉积物样品的重量，g；

$W _ { H z O }$ —沉积物样品含水率， $\%$ 。

# 9.2 结果表示

测定结果小数点后位数与方法检出限一致，最多保留 3 位有效数字。

# 10 精密度和准确度

# 10.1 精密度

六家实验室分别对六价铬含量为 $8 . 4 ~ \mathrm { m g / k g }$ 、 $3 1 . 6 ~ \mathrm { m g / k g }$ 和 $6 6 . 5 ~ \mathrm { m g / k g }$ 的土壤样品进行了 6 次重复测定：实验室内相对标准偏差分别为： $5 . 0 \% \sim 1 4 \%$ 、 $3 . 8 \% { \sim } 5 . 8 \%$ 和 $2 . 4 \% \sim 1 4 \%$ ；实验室间相对标准偏差分别为： $7 . 8 \%$ 、 $4 . 0 \%$ 和 $9 . 1 \%$ ；重复性限分别为：2.6 mg/kg、4.1 mg/kg和 $1 2 \mathrm { m g / k g }$ ；再现性限分别为： $3 . 0 \mathrm { m g / k g }$ 、 $5 . 2 \mathrm { m g / k g }$ 和 $2 0 \mathrm { m g / k g }$ 。

六家实验室分别对六价铬含量为 $7 . 8 \mathrm { m g / k g }$ 和 $2 4 . 9 \mathrm { m g / k g }$ 的沉积物样品进行了6次重复测定：实验室内相对标准偏差分别为： $5 . 2 \% { \sim } 9 . 3 \%$ 和 $2 . 3 \% { \sim } 6 . 4 \%$ ；实验室间相对标准偏差分别为： $7 . 9 \%$ 和 $1 1 \%$ ；重复性限分别为： $1 . 6 \mathrm { m g / k g }$ 和 $3 . 4 ~ \mathrm { m g / k g }$ ；再现性限分别为： $2 . 3 ~ \mathrm { m g / k g }$ 和 $8 . 4 \mathrm { m g / k g }$ 。

六家实验室分别对六价铬含量为 $2 . 0 \mathrm { m g / k g }$ 的空白加标样品进行了 6 次重复测定：实验室内相对标准偏差为 $2 . 6 \% { \sim } 8 . 5 \%$ ；实验室间相对标准偏差为 $4 . 1 \%$ ；重复性限为 $0 . 3 4 \mathrm { m g / k g }$ ；再现性限为 $0 . 3 8 \mathrm { m g / k g }$ 。

# 11 质量保证和质量控制

# 11.1 工作曲线

每批样品测定前均应绘制工作曲线，相关系数应大于等于0.999。

# 11.2 空白试验

每 20 个样品或每批次（少于 20 个样品/批）至少分析 1 个空白试样，空白试样的测定值应低于方法检出限。

# 11.3 平行样

每 20 个样品或每批次（少于 20 个样品/批）至少分析 1 个平行样，平行样测定值的相对偏差 $\text{‰}$ 。

# 11.4 基体加标

每 20 个样品或每批次（少于 20 个样品/批）至少分析 1 个基体加标样，加标回收率应在 $7 0 \% \sim 1 3 0 \%$ 之间。

# 12 废物处理

实验中产生的废物应集中收集，并做好相应标识，委托有资质的单位进行处理。
