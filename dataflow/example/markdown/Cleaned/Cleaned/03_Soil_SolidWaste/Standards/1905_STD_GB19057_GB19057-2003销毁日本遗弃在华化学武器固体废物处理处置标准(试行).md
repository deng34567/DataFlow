# 销毁日本遗弃在华化学武器固体废物处理处置标准（试行）

Standard for solid waste treatment and disposal for the destruction of chemical weapons abandoned by Japan in China

# 1 范围

本标准规定了在日本遗弃在华化学武器销毁过程中产生的固体废物处理处置及其监测的环境保护要求。

本标准适用于日本遗弃在华化学武器销毁过程中产生的固体废物处理处置设施的建设、运行及监督管理。

# 3 术语

HJ/T94—2003中确立的以及下列术语和定义适用于本标准。

# 3.1 化学武器

合指或单指：（1）毒剂及其前体；（2）经专门设计，通过使用后释放出毒剂毒性从而造成死亡或其他伤害的弹药和装置；（3）经专门设计其用途与本条目（2）所指弹药和装置的使用直接有关的任何设备。

# 3.2 毒剂

通过其对生命过程的化学作用而能够对人类或动物造成死亡、暂时失能或永久伤害的任何化学品，无论其来源或其产生方法如何，也无论其是否在设施中、弹药中或其他地方产生出来。

# 3.3 毒剂有毒相关物

毒剂的降解产物、杂质、前体等与毒剂相关的有毒化合物。

# 3.4 固体废物

在销毁遗弃化学武器全过程中产生的所有固态、半固态废弃物质。

# 3.5 危险废物

根据有关标准规定认定的具有危险特性的废物。危险特性是指浸出毒性、急性毒性、腐蚀性、反应性、易燃性、传染性。

# 3.6 染毒废物

在销毁遗弃化学武器全过程中产生的沾染了毒剂及其毒剂有毒相关物的固体废物。

# 5 处理处置场所的一般规定

5.1 固体废物处理处置场所必须设立防护设施，防止无关人员擅自进入。
5.2 固体废物处理处置场所入口处必须设立醒目的标识牌，标识牌应写明处理处置场所性质、所属单位、运营单位、责任人姓名及联系方式和启用时间等。
5.3 处理处置场所周围应该设置绿化隔离带，其宽度不得小于 $10\mathrm{m}$ 。
5.4 处理处置设施建设施工前，应编制施工质量保证书并获得环境保护主管部门的批准。施工质量保证书应该包括施工质量检验程序、检验内容、检验频率以及人员培训、施工文件管理和其他施工质量保证措施和保证程序。施工中应该严格按照施工质量保证书中的质量保证程序进行。
5.5 在处理处置场所投入运行之前，应制定一个运行计划。这一计划不但要满足常规运行的要求，而且要提出应急措施和应急计划，保证处理处置场所的有效运营和环境安全。
5.6 处理处置场所的运行、管理人员必须经过有关环境保护和环境安全的岗位培训并经相应主管部门考核合格后上岗。

处理处置场所的运行、管理人员必须具备毒剂及其化学武器常识方可上岗工作。

5.7 固体废物处理处置场所管理单位应建立有关这一场所的全部档案。这其中应包括选址，勘察，征地，设计，施工，验收，运行管理，填埋场封场和封场管理，除填埋场外其他处理处置场所的关闭和清理、移交，环境和污染物排放监测，以及废物特性，处理工艺参数，在填埋场的填埋位置等内容的所有文件资料。这些档案必须按照国家档案管理条例进行整理与保管，保证完整无损。填埋场档案的最低保存年限为封场后 $100\mathrm{a}$ ，其他处理处置场所档案的最低保存年限为关闭后 $50\mathrm{a}$ 。

# 6 焚烧及其他化学处理

6.1 各种类型的固体废物焚烧设施和其他化学处理工厂不允许建设在GB3838中规定的地表水环境质量I、Ⅱ类功能区和GB3095中规定的环境空气质量一类功能区，即自然保护区、风景名胜区、生态功能保护区、森林公园、地质公园和其他需要特殊保护的地区；不允许建设在人口密集的居住区、商业区和文化区；不允许建设在居民区主导风向的上风向地区。
6.2 各种类型的固体废物焚烧工厂和其他化学处理工厂的建设必须考虑地震、洪水、台风以及其他破坏性自然灾害可能造成的后果及其必要的预防措施。
6.3 染毒废物必须采用与化学武器销毁方式相同或具有相同效果的处理方式进行处理，不得直接进行填埋处置。
6.4 染毒废物处理设施的化学毒剂及其毒剂有毒相关产物的销毁去除率必须达到 $99.9999\%$ 以上。
6.5 危险废物焚烧设施的建设与运行应满足GB18484中的各项技术要求。
6.6 一般废物焚烧设施的建设与运行应满足GB18485中的各项技术要求。
6.7 固体废物焚烧和其他化学处理设施的运行中，要求保证系统的密闭性或处于负压状态以防止有害气体逸出；禁止进行敞开式运行操作。
6.8 固体废物焚烧和其他化学处理设施应设置必要的废气处理装置，以控制可能产生的有害气体对环境的影响；
6.9 在固体废物焚烧和其他化学处理工厂内，必须设置专门的固体废物贮存设施。固体废物贮存设施要求具有良好的防雨、防渗性能和配备必要的污水收集、排放系统，以收集、排除可能产生的渗滤液和其他污水。危险废物贮存设施要考虑危险废物之间和危险废物与贮存设施材料的相容性。危险废物贮存设施应设置有害气体监测、排除和处理设施，以防止有害气体的逸出。危险废物贮存设施应远离焚烧设施和其他高温设施并符合消防要求。
6.10 固体废物焚烧烟气净化和其他化学处理过程中产生的废水禁止排放。废水净化产生的污泥作为危险废物进行处理处置。

6.11 各种类型的固体废物焚烧工厂和其他化学处理工厂厂区内环境空气质量应该满足GB19059—2003的要求。
6.12 各种类型的固体废物焚烧工厂和其他化学处理工厂运行中噪声应符合GB12348中的要求。
6.13 各种类型的固体废物焚烧工厂和其他化学处理工厂必须制定事故应急反应行动计划，确定事故应急反应行动计划的责任人，并且定期对有关人员进行相关知识培训和演习，提高事故反应能力。

# 7 填埋

7.1 一般固体废物填埋应该满足GB16889的各项要求。
7.2 危险废物填埋可以采用危险废物填埋场或危险废物填埋库的填埋形式。
7.3 7.2条所指危险废物填埋场是指满足GB18598中的各项要求的危险废物填埋处置场所。
7.4 7.2条所指危险废物填埋库是指满足下列条件的危险废物填埋处置场所：

a）主体由防渗钢筋混凝土构成，壁厚度不小于 $350~\mathrm{mm}$ ，单轴抗压强度不小于 $25\mathrm{N} / \mathrm{mm}^2$ ；
b）结构上具有耐受自重、土压、水压、波动、地震等因素的安全性；
c）在与危险废物接触的面上应该全部敷设HDPE防渗衬层，其厚度不得小于 $1.5\mathrm{mm}$ ；或者铺设其他具有同等防渗作用且与填埋废物相容的合成材料防渗衬层；
d）采取措施以便有效地防止地表水、地下水以及周围土壤相应的腐蚀；
e）设置渗滤液集排水系统用于渗漏监测和排除事故产生的渗滤液及渗入填埋层的地下水；

f）应该设置必要的设施防止雨水、地表径流、地下水进入库内。在设计时采用的暴雨强度重现期不得低于 $50\mathrm{a}$
g）当其面积超过 $25\mathrm{m}^2$ 或容积超过 $250\mathrm{m}^3$ 时，应该采用分区填埋。其分区面积不宜超过 $25\mathrm{m}^2$ 容积不宜超过 $250\mathrm{m}^3$ ，分区间的隔壁应该符合本条上述各款的要求；
h）对不相容性废物进行分区填埋。如果无法分区，对不相容性废物可分类用容器盛放后填埋，容器材料应与所有可能接触的物质相容，而且不被腐蚀；

7.5 危险废物进入危险废物填埋场和危险废物填埋库，应满足GB18598中所列各项入场条件，并按照销毁日本遗弃在华化学武器土壤与固体废物监测方法检验，不含有毒剂及其毒剂有毒相关物成分。

如果废物不符合上述要求，必须进行焚烧、固化、稳定化或其他处理并满足上述要求后方可进行填埋处置。

7.6 危险废物填埋库的场址选择应满足GB18598中的场址选择要求。
7.7 危险废物填埋场和危险废物填埋库应该设置必要的渗漏检测设施，以保证在衬层发生破损导致渗滤液渗漏或地下水进入填埋层时能及时发现。
7.8 当填埋场和填埋库处置的废物数量达到其设计容量时，应实行封场。
7.9 填埋场的最终覆盖层结构按照GB18598执行；填埋库的封场结构应该包括与底部结构相同的防渗结构，以及排水层、保护层、植被恢复层。
7.10 封场后应继续进行下列维护管理工作，并至少延续到封场后 $50\mathrm{a}$

a）维护防渗衬层和最终覆盖层的完整性和有效性；
b）维护和监测检漏系统；
c）继续进行渗滤液的收集和处理；
d) 继续监测地下水水质的变化。

7.11 在填埋场、填埋库运行和封场后维护过程中，如发现衬层破损，应立即采取补救措施防止渗滤液渗漏和地下水进入填埋层，同时将破损情况、污染现状及趋势、补救措施和效果以及下一步所要采取的措施及时向环境保护主管部门报告。

7.12 当填埋场和填埋库运行中出现下列情况之一时，应实行非正常封场：

a）发现场址或处置系统的设计有不可改正的错误；
b）发生严重事故及发生不可预见的自然灾害；
c）在例行的环境监测中发现地下水被污染；
d）其他使得填埋场不能继续运行的事件。

当填埋场和填埋库实行非正常封场时，首先应该停止将废物运入填埋区并停止填埋作业。同时应该根据预先作好的应急计划采取必要的补救措施，防止污染扩散；并对已经发生的污染进行清除治理。实施非正常封场必须得到环境保护主管部门的批准。

7.13 危险废物填埋场区环境质量应该满足GB19059—2003的要求。

# 8 污染物控制与监测

8.1 一般废物焚烧的污染物排放应该满足GB18485的要求。
8.2 危险废物焚烧和其他化学处理的大气污染物排放应该同时满足GB18484和GB19060—2003的要求。
8.3 一般废物填埋污染物排放应该满足GB16889的要求。
8.4 危险废物填埋的污染物排放应该同时满足GB18598和GB19060—2003的要求。危险废物填埋产生的渗滤液，按照销毁日本遗弃在华化学武器废水中污染物监测方法检验，不得含有芥子气、路易氏剂、二苯氯肿、二苯氰肿、氰溴甲苯、苯氯乙酮、氰化物、光气、三氯化砷、氯乙烯氧肿、氯乙烯肿酸、氧联双二苯肿、芥子砜、芥子亚砜等物质成分。
8.5 危险废物焚烧和其他化学处理工厂以及危险废物填埋场（库）的环境监测和污染物排放监测参照GB/T14848、GB18484、GB19060—2003、GB19061—2003和GB19059—2003执行。
8.6 危险废物焚烧和其他化学处理工厂的废气排放和废水循环处理设施出口以及工厂内重要的污染监测位置应该尽量设置在线监测系统，保证能在最短的时间内发现污染事故并发出警报。
8.7 固体废物填埋场（库）建设前应对场址所在地地下水进行监测，作为场址地下水水质的背景值。地下水背景值的监测指标包括GB18598中规定的指标以及电导率、盐离子浓度。
8.8 危险废物填埋场封场后和危险废物焚烧场或其他化学处理场拆解后 $50\mathrm{a}$ 内，应该对其场址继续进行地下水和土壤监测。

# 9 标准实施

本标准由国家环境保护总局监督实施。
