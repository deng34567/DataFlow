# 水和土壤总氰化物的测定气液分离收集分光光度法

# 1 适用范围

本标准规定了测定水和土壤中总氰化物的气液分离收集分光光度法。

本标准适用于地表水、地下水、生活污水、工业废水以及土壤中总氰化物的测定。

对水中总氰化物的检出限为 $0.003\mathrm{mg / L}$ ，测定下限为 $0.012\mathrm{mg / L}$ 。

当土壤样品量为 $2.50\mathrm{g}$ ，本方法的检出限为 $0.04\mathrm{mg / kg}$ ，测定下限为 $0.16\mathrm{mg / kg}$ 。

# 2 方法原理

# 2.1 前处理原理

# 2.1.1 水质总氰化物的前处理原理

向水样中加入磷酸和EDTA二钠，在 $\mathrm{pH} < 2$ 的条件下，加热蒸馏，利用EDTA与金属离子络合能力比氰离子与金属离子络合能力强的特点，使络合氰化物离解出氰离子，并以氰化氢形式被蒸馏出，用氢氧化钠溶液吸收。

# 2.1.2 土壤总氰化物的前处理原理

将土壤试样在碱性条件下用氯化亚锡和硫酸铜抑制硫化物干扰，并催化络合氰化物离解，通过磷酸调节酸度后加热蒸馏，使络合氰化物离解出氰离子，并以氰化氢形式被蒸馏出，用氢氧化钠溶液吸收。

# 2.2 测定原理

收集液中的氰离子在中性条件下与氯胺T反应生成氯化氰，其与异烟酸反应，经水解后生成戊烯二醛，再与吡唑啉酮缩合反应生成蓝色染料，该物质在波长 $638\mathrm{nm}$ 处有最大吸收，计算获得样品中总氰化物的浓度。

# 3 干扰及消除

# 3.1 水质样品测定过程中的干扰和消除

试样中存在的活性氯等氧化物会干扰测定，可在蒸馏前加亚硫酸钠溶液（ $\mathrm{Na_2SO_3}$ ）排除干扰。

试样中存在的亚硝酸离子会干扰测定，可在蒸馏前加氨基磺酸（ $\mathrm{NH_2SO_2OH}$ ）排除干扰。

试样中存在的硫化物会干扰测定，可在蒸馏前加碳酸镉（ $\mathrm{CdCO_3}$ ）或碳酸铅（ $\mathrm{PbCO_3}$ ）固体粉末排除干扰。

少量油类对测定无影响，但中性油或酸性油含量大于 $40\mathrm{mg / L}$ 时会干扰测定，对此，可加入水样体积的 $20\%$ 量的正己烷（ $\mathrm{C_6H_{14}}$ ），在中性条件下短时间萃取，分离出正己烷相后，再将水相用于蒸馏测定。

# 3.2 土壤样品测定过程中的干扰和消除

试样中微粒不能完全在水中均匀分散，而是积聚在试剂一空气表面或试剂一玻璃器壁界面，会导致

准确度和精密度降低，可在蒸馏前加 $5\mathrm{mL}$ 乙醇以消除干扰。

试样中存在的硫化物会干扰测定，蒸馏时加入硫酸铜可以抑制干扰。

试样中酚的含量低于 $500\mathrm{mg / L}$ 不会影响氰化物的测定。

油脂类的干扰可在显色前加入十二烷基硫酸钠予以消除。

# 4 试剂和材料

4.1 实验用水。

新制备的不含氰化物和活性氯的蒸馏水或去离子水。

4.2 乙二胺四乙酸二钠盐（EDTA-2Na）溶液： $\rho(\mathrm{C}_{10} \mathrm{H}_{14} \mathrm{~N}_{2} \mathrm{O}_{8} \mathrm{Na}_{2} \cdot 2 \mathrm{H}_{2} \mathrm{O}) = 100 \mathrm{~g} / \mathrm{L}$ 。

称取 $10.0\mathrm{g}$ 乙二胺四乙酸二钠盐（EDTA-2Na）溶于水中，稀释定容至 $100~\mathrm{mL}$ ，摇匀。

4.3 磷酸： $\rho (\mathrm{H}_3\mathrm{PO}_4) = 1.69\mathrm{g / mL}$

4.4 氢氧化钠溶液： $\rho (\mathrm{NaOH}) = 1\mathrm{g / L}$

称取 $1\mathrm{g}$ 氢氧化钠溶于水中，稀释至 $1000~\mathrm{mL}$ ，摇匀，贮于聚乙烯塑料容器中。

4.5 氢氧化钠溶液： $\rho(\mathrm{NaOH}) = 20 \mathrm{~g} / \mathrm{L}$ 。

称取 $20\mathrm{g}$ 氢氧化钠溶于水中，稀释至 $1000~\mathrm{mL}$ ，摇匀，贮于聚乙烯塑料容器中。

4.6 氢氧化钠溶液： $\rho (\mathrm{NaOH}) = 100\mathrm{g / L}$

称取 $100\mathrm{g}$ 氢氧化钠溶于水中，稀释至 $1000\mathrm{mL}$ ，摇匀，贮于聚乙烯容器中。

4.7 磷酸盐缓冲溶液（ $\mathrm{pH} = 7$ ）。

称取 $34.0\mathrm{g}$ 无水磷酸二氢钾（ $\mathrm{KH_2PO_4}$ ）和 $35.5\mathrm{g}$ 无水磷酸氢二钠（ $\mathrm{Na_2HPO_4}$ ）溶于水，稀释定容至 $1000\mathrm{mL}$ ，摇匀。

4.8 氯胺T溶液： $\rho (\mathrm{C_7H_7ClNNaO_2S\cdot 3H_2O}) = 10\mathrm{g / L}$

称取 $1.0\mathrm{g}$ 氯胺T溶于水，稀释定容至 $100~\mathrm{mL}$ ，摇匀，贮于棕色瓶中，用时现配。

注：氯胺T发生结块不易溶解，可致显色无法进行，氯胺T固体试剂应注意保管条件以免迅速分解失效，勿受潮，最好冷藏。

# 4.9 异烟酸-吡唑啉酮溶液

# 4.9.1 异烟酸溶液

称取 $1.5\mathrm{g}$ 异烟酸 $\mathrm{(C_6H_6NO_2)}$ ，溶于 $25~\mathrm{mL}$ 浓度为 $20~\mathrm{g / L}$ 的氢氧化钠溶液（4.5）中，加水稀释定容至 $100~\mathrm{mL}$ 。

# 4.9.2 吡唑啉酮溶液

称取 $0.25\mathrm{g}$ 吡唑啉酮（3-甲基-1-苯基-5-吡唑啉酮， $\mathrm{C_{10}H_{10}ON_2}$ ），溶于 $20~\mathrm{mL}$ N,N-二甲基甲酰胺 $\left[\mathrm{HCON}\left(\mathrm{CH}_3\right)_2\right]$ 中。

# 4.9.3 异烟酸-吡唑啉酮溶液

将吡唑啉酮溶液（4.9.2）和异烟酸溶液（4.9.1）按 $1:5$ 的比例混合，用时现配。

注：异烟酸配成溶液后如呈明显淡黄色，会使空白值增高，可过滤。为降低试剂空白值，实验中优先选用无色的N,N-二甲基甲酰胺。

4.10 盐酸： $\rho (\mathrm{HCl}) = 1.19\mathrm{g / mL}$

4.11 盐酸溶液： $c(\mathrm{HCl}) = 1\mathrm{mol} / \mathrm{L}$

量取 $83~\mathrm{mL}$ 盐酸（4.10）缓慢注入水中，冷却后稀释至 $1000~\mathrm{mL}$ 。

4.12 氯化亚锡溶液： $\rho (\mathrm{SnCl}_2\bullet 2\mathrm{H}_2\mathrm{O}) = 50\mathrm{g / L}$

称取 $5.0\mathrm{g}$ 二水合氯化亚锡溶于 $40~\mathrm{mL}$ 盐酸溶液（4.11）中，用水稀释定容至 $100~\mathrm{mL}$ ，用时现配。

4.13 硫酸铜溶液： $\rho (\mathrm{CuSO_4\cdot 5H_2O}) = 200\mathrm{g / L}$

称取 $200\mathrm{g}$ 五水合硫酸铜溶于水中，稀释至 $1000~\mathrm{mL}$ ，摇匀。

4.14 氰化钾标准贮备溶液： $\rho(\mathrm{KCN}) = 50 \mu \mathrm{g/mL}$ 。

购买市售有证标准物质。如自行配制，执行HJ484标准。

4.15 水质氰化钾标准使用溶液： $\rho(\mathrm{KCN}) = 1.00 \mathrm{mg} / \mathrm{L}$ 。

吸取 $10.00\mathrm{mL}$ 氰化钾标准贮备溶液（4.14），移入 $500\mathrm{mL}$ 棕色容量瓶中，用 $1\mathrm{g/L}$ 的氢氧化钠溶液（4.4）稀释至标线，摇匀，避光保存，用时现配。

4.16 土壤氰化钾标准使用溶液： $\rho(\mathrm{KCN}) = 0.500 \mu \mathrm{g/mL}$ 。

吸取 $10.00\mathrm{mL}$ 氰化钾标准贮备溶液（4.14)，移入 $1000\mathrm{mL}$ 棕色容量瓶中，用 $1\mathrm{g / L}$ 的氢氧化钠溶液（4.4）稀释至标线，摇匀，避光保存，用时现配。

# 4.17 乙酸铅试纸

称取 $5\mathrm{g}$ 乙酸铅 $\mathrm{[Pb(C_2H_3O_2)_2\cdot 3H_2O]}$ 溶于水中，并稀释至 $100~\mathrm{mL}$ 。将滤纸条浸入上述溶液中，1h后取出晾干，贮于广口瓶中，密塞保存。

# 6 样品

# 6.1 样品的采集和保存

# 6.1.1 水样的采集和保存

6.1.1.1 采集的水样需贮存于用无氰水清洗并干燥后的聚乙烯塑料瓶或硬质玻璃瓶中。现场采样时，先用所采水样淋洗3次采样容器，再采集水样 $500~\mathrm{mL}$ 供实验室分析使用。水样采集后必须立即加氢氧化钠固定，一般每升水样加约 $0.5\mathrm{g}$ 固体氢氧化钠。当水样酸度高时，应多加固体氢氧化钠，使样品的 $\mathrm{pH} > 12$ 。
6.1.1.2 采集的水样应及时进行测定。如果不能及时测定水样，应将水样在 $4^{\circ}\mathrm{C}$ 以下冷藏，24h 内必须完成分析。如果水样中有硫化物存在，保存不得超过 $12\mathrm{h}$ 。
6.1.1.3 当水样中含有大量硫化物时，应先加碳酸镉或碳酸铅固体粉末，除去硫化物后，再加氢氧化钠固定。否则，在碱性条件下，氰离子和硫离子作用形成硫氰酸离子而干扰测定。

注1：检验硫化物的方法，取1滴水样或样品，放在乙酸铅试纸（4.17）上，若变成黑色（硫化铅），表明有硫化物存在。

# 6.1.2 土壤样品的采集和保存

土壤采样点位的布设和采样方法执行HJ/T166—2004，对采集到的土壤样品，用可密封的聚乙烯或玻璃容器盛装，并使土壤样品充满容器，在 $4^{\circ}\mathrm{C}$ 左右冷藏保存，且应在 $48\mathrm{h}$ 内完成分析工作。

# 6.2 样品的预处理

# 6.2.1 水样的预处理

在收集瓶过滤嘴处装好疏水性滤膜，然后在收集瓶内定量加入 $25.00\mathrm{mL}$ （ $1\mathrm{g / L}$ ）NaOH溶液(4.4)。

连接收集瓶与抽真空装置，开启电源，打开真空泵，选择合适档位，检查气密性。

分取 $50.0 \mathrm{~mL}$ 水样移入玻璃蒸馏试管中（若氰化物浓度高，可少取样品，加水稀释至 $50.0 \mathrm{~mL}$ ）。将装有双口插盖、加液漏斗、加液长管、垫圈的试管帽与蒸馏试管拧紧。并与收集瓶连接，再次检查气密性。

设置炉温 $128^{\circ}\mathrm{C}$ ，时间 $80\mathrm{min}$ ，真空压力 $35\mathrm{kPa}$ 左右，开启风扇，调节微调阀至收集瓶内气泡速率为 $6\sim 8$ 个/秒，开始加热。

沿加液漏斗加入 $2.5\mathrm{mL}$ EDTA-2Na溶液(4.2)和 $2.5\mathrm{mL}$ 磷酸(4.3)。开始加热至收集完毕，收集瓶内收集液待测，此为试样“A”。

用实验用水代替水样，按以上步骤操作得到空白实验收集液待测，此为空白实验试样“B”。

# 6.2.2 土壤样预处理

另称取样品按照HJ613的规定进行干物质的测定。

在收集瓶过滤嘴处装好疏水性滤膜，然后在收集瓶内定量加入 $25.00\mathrm{mL}$ （1g/L）NaOH溶液(4.4)。连接收集瓶与抽真空装置，开启电源，打开真空泵，选择合适的档位，检查气密性。

称取约 $2.50\mathrm{g}$ 干重的土壤样品于称量纸上（精确到 $0.01\mathrm{g}$ ），略微裹紧后移入玻璃蒸馏试管（若样品中氰化物含量较高，可适量减少样品称量或对吸收液稀释后进行测定），加入 $50\mathrm{mL}$ 蒸馏水，加入

0.75 mL 氢氧化钠 (4.6)，将装有双口插盖、加液漏斗、加液长管、垫圈的试管帽拧紧，并与吸收瓶连接，再次检查气密性。

设置炉温 $128^{\circ}\mathrm{C}$ ，时间 $80\mathrm{min}$ ，真空压力 $35\mathrm{kPa}$ 左右，开启风扇，调节微调阀至收集瓶内气泡速率为 $6\sim 8$ 个/秒，开始加热。

从加液漏斗中加入 $1.0\mathrm{mL}$ 氯化亚锡（4.12）， $2.5\mathrm{mL}$ 硫酸铜（4.13），摇匀，然后加入 $5.0\mathrm{mL}$ 磷酸（4.3）。开始加热至蒸馏完毕。收集瓶内收集液待测，此为试样“A”。

在玻璃蒸馏试管中加 $50.0\mathrm{mL}$ 蒸馏水和 $0.75\mathrm{mL}$ 氢氧化钠溶液（4.6），按以上步骤操作，得到土壤空白试验收集液待测，此为空白实验试样“B”。

# 7 分析步骤

# 7.1 水质总氰化物校准曲线

取8支 $25\mathrm{mL}$ 具塞比色管，分别加入氰化钾标准使用溶液（4.15）0.00、0.20、0.50、1.00、2.00、3.00、4.00和 $5.00~\mathrm{mL}$ ，再加入氢氧化钠溶液（4.4）至 $10\mathrm{mL}$ ，氰化物含量依次为 $0.00\mu \mathrm{g}$ 、 $0.20\mu \mathrm{g}$ 、 $0.50\mu \mathrm{g}$ 、 $1.00\mu \mathrm{g}$ 、 $2.00\mu \mathrm{g}$ 、 $3.00\mu \mathrm{g}$ 、 $4.00\mu \mathrm{g}$ 和 $5.00\mu \mathrm{g}$ 。向各管中加入 $5.0\mathrm{mL}$ 磷酸盐缓冲溶液（4.7），混匀，迅速加入 $0.20\mathrm{mL}$ 氯胺T（4.8）溶液，立即盖塞子，混匀，放置 $3\sim 5\mathrm{min}$ 。

向各管中加入 $5.0 \mathrm{~mL}$ 异烟酸-吡唑啉酮溶液（4.9.3），混匀。加水稀释至标线，摇匀。在 $25 \sim 35^{\circ} \mathrm{C}$ 的水浴装置（5.2）中放置 $40 \mathrm{~min}$ ，立即比色。

在 $638~\mathrm{nm}$ 波长处，用 $10~\mathrm{mm}$ 比色皿，以水作参比，测定吸光度，以氰化物的含量为横坐标，扣除试剂空白的吸光度为纵坐标，以最小二乘法绘制校准曲线。

# 7.2 土壤总氰化物校准曲线

取6支 $25\mathrm{mL}$ 具塞比色管（5.4)，分别加入氰化钾标准使用溶液（4.16）0.00、0.10、0.50、1.50、4.00和 $10.00\mathrm{mL}$ ，再加入氢氧化钠溶液（4.4）至 $10\mathrm{mL}$ 。标准系列中氰离子的含量分别为 $0.00\mu \mathrm{g}$ 、 $0.05\mu \mathrm{g}$ 、 $0.25\mu \mathrm{g}$ 、 $0.75\mu \mathrm{g}$ 、 $2.00\mu \mathrm{g}$ 、 $5.00\mu \mathrm{g}$ 。向各管中加入 $5.0\mathrm{mL}$ 磷酸盐缓冲溶液（4.7)，混匀，迅速加入 $0.20\mathrm{mL}$ 氯胺T（4.8）溶液，立即盖塞，混匀，放置 $1\sim 2\mathrm{min}$ 。向各管中加入 $5.0\mathrm{mL}$ 异烟酸-吡唑啉酮溶液（4.9.3)，加水稀释至标线，摇匀，于 $25\sim 35^{\circ}\mathrm{C}$ 的水浴装置（5.2）中显色 $40\mathrm{min}$ ，立即比色，比色方法同前。

注2：当氰化物以HCN存在时易挥发，因此，加入缓冲溶液后，每一步骤操作都要迅速，并随时盖紧塞子。

# 7.3 试样的测定

从试样“A”吸取 $10.00\mathrm{mL}$ 水质试样“A”或土壤试样“A”，于 $25\mathrm{mL}$ 具塞比色管中，分别按水质或土壤校准曲线相同步骤测量吸光度。从校准曲线上计算出相应的氰化物的含量。

# 7.4 空白试验

从试样“B”中分别吸取 $10.00\mathrm{mL}$ 水质试样“B”或土壤试样“B”空白试验试样于 $25\mathrm{mL}$ 具塞比色管中，按与样品相同的步骤进行测定。

# 8 结果计算

# 8.1 水质总氰化物浓度

水质氰化物质量浓度 $\rho$ 以氰离子（CN-）计，按式（1）计算。

$$
\rho = \frac {A - A _ {0} - a}{b} \times \frac {V _ {1}}{V _ {2} \times V} \tag {1}
$$

式中： $\rho$ ——氰化物的质量浓度， $\mathrm{mg / L}$ ；

$A$ ——试样A的吸光度；

$A_0$ ——空白试样B的吸光度；

$a$ ——校准曲线截距；

$b$ 校准曲线斜率；

$V$ ——预处理的取样体积， $\mathrm{mL}$

$V_{I}$ ——馏出液的总体积， $\mathrm{mL}$

$V_{2}$ 测定时所取试样的体积， $\mathrm{mL}$

# 8.2 土壤总氰化物的含量

土壤总氰化物的含量 $\omega (\mathrm{mg / kg})$ ，以氰离子（CN-）计，按式（2）计算。

$$
\omega = \frac {A - A _ {0} - a}{b \times m} \times \frac {V _ {1}}{W _ {\mathrm {d m}} \times V _ {2}} \tag {2}
$$

式中： $\omega$ ——总氰化物（ $105^{\circ}\mathrm{C}$ 干重）的含量， $\mathrm{mg/kg}$ ；

$A$ ——试样A的吸光度；

$A_{o}$ ——空白试样B的吸光度；

$a$ ——校准曲线截距；

$b$ 校准曲线斜率；

$V_{I}$ ——馏出液的总体积， $\mathrm{mL}$

$V_{2}$ 测定时所取试料的体积， $\mathrm{mL}$

$m$ 称取的土壤样品质量，g；

$W_{dm}$ ——土壤样品中干物质含量， $\%$ 。

# 9 精密度和准确度

由5家实验室对高、低两个浓度水平的水质样品进行了总氰化物的测定，实验室间的相对标准偏差分别为 $4.72\%$ 和 $1.93\%$ ，重复性限r分别为 $10.6\mu \mathrm{g} / \mathrm{L}$ 和 $11.0\mu \mathrm{g} / \mathrm{L}$ ，再现性限R分别为 $13.2\mu \mathrm{g} / \mathrm{L}$ 和 $12.4\mu \mathrm{g} / \mathrm{L}$ 。
5 家实验室对高、低两个浓度水平水质中氰化物有证标准物质测定结果的相对误差范围分别为 $-7.35\% \sim 3.68\%$ 和 $-2.94\% \sim 0.74\%$ ，相对误差最终值分别为 $-0.35\% \pm 8.4\%$ 和 $-0.93\% \pm 3.6\%$ 。5 家实验室对土壤实际样品的加标回收率为 $72.0\% \sim 96.7\%$ ，加标回收率标准偏差为 $9.6\%$ 。

# 10 质量保证和质量控制

# 10.1 空白试验

每批样品应至少做两个空白试验。

# 10.2 精密度控制

每批样品应做 $10\%$ 的平行样分析，其相对偏差应符合实验室质量控制要求。如果样品不均匀，应在满足精密度的前提下做至少两个平行样的测定，取均值报出结果。

# 10.3 准确度控制

每批样品测定时，应分析一个有证标准样品或质控样品，或做 $10\%$ 的加标样分析，质控样品的测定值或加标回收率应符合规定的质量控制要求。加标物可使用铁氰化钾标准溶液（配制与标定见HJ745—2015附录A)，加标后的样品与待测样品同步处理。

# 11 废物处理

实验室产生的废液应统一收集，委托有资质单位集中处理。

# 12 注意事项

12.1 空白试验的总氰化物含量应小于方法检出限。
12.2 个别样品在加热过程中出现不冒气泡现象，应检查各连接部分是否漏气，如旋紧试管帽或者增大微调阀。
12.3 氢氰酸和氰化物属于剧毒物质，除非是在特定步骤下进行实验，否则不应酸化样品。整个实验过程在通风橱内进行，实验人员应按规定佩戴防护器具，避免接触皮肤和衣服。对检测后的残渣残液应做妥善的安全处理。
