# 中华人民共和国国家标准

# 固体废物腐蚀性测定玻璃电极法

GB/T15555.12—1995

Solid waste—Glass electrode test—Method of corrosivity

# 1 主题内容与适用范围

1.1 本标准规定了固体废物的腐蚀性，用 $\mathrm{pH}$ 玻璃电极的试验方法。
1.2本标准试验方法适用于固体、半固体的浸出液和高浓度液体的 $\mathrm{pH}$ 的测定。
1.2.1 固体废物腐蚀性 $\mathrm{pH}$ 值的测定，采用玻璃电极法。 $\mathrm{pH}$ 的测定范围 $0\sim 14$

# 2 定义

本标准所称的固体废物腐蚀性是指单位、个人在生产、经营、生活和其他活动中所产生的固体、半固体和浓度液体，具有下述性质者：

采用指定的标准鉴别方法，或者根据规定程序批准的等效方法，测定其溶液或固体、半固体浸出液的 $\mathrm{pH}$ 值小于、等于2，或者等于、大于12.5，则这种废物即具有腐蚀性。

# 3 浸出液的制备

# 3.1 仪器和材料

3.1.1 混合容器：容积为2L的带密封塞的高压聚乙烯瓶。
3.1.2 振荡器：往复式水平振荡器。
3.1.3 过滤装置：市售成套过滤器，纤维滤膜孔径为 $\phi 0.45\mu \mathrm{m}$
3.1.4 蒸馏水或去离子水。
3.2 浸出步骤
3.2.1 称取 $100\mathrm{g}$ 试样（以干基计），置于浸取用的混合容器中，加水 $1\mathrm{L}$ （包括试样的含水量）。
3.2.2 将浸取用的混合容器垂直固定在振荡器上，振荡频率调节为 $110 \pm 10$ 次 $/ \mathrm{min}$ ，振幅为 $40 \mathrm{~mm}$ ，在室温下振荡 $8 \mathrm{~h}$ ，静置 $16 \mathrm{~h}$ 。
3.2.3 通过过滤装置分离固液相，滤后立即测定滤液的 $\mathrm{pH}$ 值。如果固体废物中干固体的含量小于
$0.5\% (m / m)$ 时，则不经过浸出步骤，直接测定溶液的 $\mathrm{pH}$ 值。

注：固体试样风干、磨碎后应能通过 $\phi 5\mathrm{mm}$ 的筛孔。

# 4 测定方法

# 4.1 原理

用玻璃电极为指示电极，饱和甘汞电极为参比电极组成电池。在 $25^{\circ}\mathrm{C}$ 条件下，氢离子活度变化10倍，使电动势偏移 $59.16\mathrm{mV}$ 。仪器上直接以 $\mathrm{pH}$ 的读数表示。许多 $\mathrm{pH}$ 计上有温度补偿装置，可以校正湿度

# 4.2 干扰

4.2.1 当废物浸出液的 $\mathrm{pH}$ 值大于10，钠差效应对测定有干扰，宜用低（消除）钠差电极，或者用与浸出液的 $\mathrm{pH}$ 值相近的标准缓冲溶液对仪器进行校正。
4.2.2 当电极表面被油质或者粒状物质沾污会影响电极的测定，应用洗涤剂清洗。或用 $1 + 1$ 的盐酸溶液除尽残留物，然后用蒸馏水冲洗干净。
4.2.3 温度影响 $\mathrm{pH}$ 值的准确测定。因为，在不同的温度下电极的电势输出不同，温度变化也会影响到样品的 $\mathrm{pH}$ 值。所以，必须进行温度的补偿。温度计与电极应同时插入待测溶液中，在报告测定的 $\mathrm{pH}$ 值时同时报告测定时的温度。

# 5 仪器和材料

5.1 pH计：各种型号的 $\mathrm{pH}$ 计或离子活度计，精度 $\pm 0.02\mathrm{pH}$
5.2 玻璃电极，消除钠差电极。
5.3 参比电极：甘汞电极、银/氯化银电极或者其他具有固定电势的参比电极。
5.4 磁力搅拌棒、以及用聚四氟乙烯或者聚乙烯等塑料包裹的搅拌棒。
5.5 温度计或有自动补偿功能的温度敏感元件。

# 6 试剂

6.1 一级标准缓冲剂的盐，它在很高准确度的场合下使用。由这些盐制备的缓冲溶液需用低电导的、不含二氧化碳的水，而且这些溶液至少每月更换一次。
6.2 二级标准缓冲溶液，可用国家认可的标准 $\mathrm{pH}$ 缓冲盐，用低电导率（低于 $2\mu \mathrm{s} / \mathrm{cm}$ )并除去二氧化碳的水配制。

# 7 步骤

7.1 按仪器的使用说明书准备。
7.2 如果样品和标准缓冲液的温差大于 $2^{\circ}\mathrm{C}$ ，测量的 $\mathrm{pH}$ 值必须校正。可通过仪器带有的自动或手动补偿装置进行，也可预先将样品和标准在室温下平衡达到同一温度。记录测定的温度。
7.3宜选用与样品的 $\mathrm{pH}$ 值相差不超过2个 $\mathrm{pH}$ 单位的两个溶液(两者相差3个 $\mathrm{pH}$ 单位)校准仪器。用第一个标准溶液定位后，取出电极，彻底冲洗干净，并用滤纸吸去水分。再浸入第二个标准溶液进行校核，其值应在标准的允许差范围内。否则就该检查仪器、电极或标准溶液是否有问题。当校核无问题时方可测定样品。
7.4 如果在现场测定流体或半固体的流体(如稀泥、薄浆等)的 $\mathrm{pH}$ 值，电极可直接插入样品，其深度适当并可移动，保证有足够的样品通过电极的敏感元件。
7.5 对块状或颗粒状的物料，则取其浸出液（制备方法参见3)进行测定。

将样品或标准溶液倾倒入清洁烧杯中，其液面应高于电级的敏感元件，放入搅拌子，将清洁干净的电极插入烧杯中，以缓和、固定的速率搅拌或摇动使其均匀，待读数稳定后记录其 $\mathrm{pH}$ 值。应重复测定2～3次直到其 $\mathrm{pH}$ 值变化小于0.1pH单位。

# 8 数据处理与报告

8.1 每个样品至少做3个平行试验，其标准差不得超过 $\pm 0.15\mathrm{pH}$ 单位，取算术平均值报告试验结果。
8.2 当标准差超过规定范围时，必须分析并报告原因。
