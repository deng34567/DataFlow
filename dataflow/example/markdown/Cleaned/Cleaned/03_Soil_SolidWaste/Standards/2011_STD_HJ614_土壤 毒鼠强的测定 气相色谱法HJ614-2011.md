# 3 方法原理

用乙酸乙酯提取土壤中的毒鼠强，提取液经净化浓缩后，以气相色谱分离，氮磷检测器检测，以保留时间定性，外标法定量。

# 6.2 试样的制备

将样品放置在搪瓷盘或不锈钢盘上，去除沙砾、植物根系等杂物，充分混匀。称取 $5 \mathrm { \textrm { g } }$ （精确至$0 . 0 1 { \bf \tau g }$ ）土壤样品，加入同等重量的无水硫酸钠（4.2），充分混匀。用滤纸包好，放入索氏提取器中，加入 $1 0 0 ~ \mathrm { m l }$ 乙酸乙酯（4.1）。水浴温度在 $8 5 { \sim } 9 0 ^ { \circ } \mathrm { C }$ ，以回流 4 次/h 提取 $1 2 \sim 1 6 \mathrm { h }$ 。将提取液转移至$1 5 0 \mathrm { m l }$ 分液漏斗中，用 $2 0 ~ \mathrm { m l }$ 乙酸乙酯（4.1）分别清洗索氏提取器两次，与提取液合并。

安装净化装置（见图 1），控制流速 $4 { \sim } 6 ~ \mathrm { m l / m i n }$ ，用具塞磨口三角瓶收集洗脱液。用 $1 0 \mathrm { m l }$ 乙酸乙酯（4.1）清洗玻璃层析柱，将洗脱液合并。将上述洗脱液移入 $2 0 0 ~ \mathrm { m l }$ 氮吹管中，在 $6 0 ^ { \circ } \mathrm { C }$ 水浴温度，用高纯氮气（4.8）吹扫浓缩至 $0 . 5 { \mathrm { m l } }$ 左右，用少量乙酸乙酯（4.1）清洗氮吹管，再用乙酸乙酯（4.1）定容至 $1 . 0 \mathrm { m l }$ ，然后转移至 $2 \mathrm { m l }$ 螺口玻璃样品瓶中，密封，待测。

# 8 结果计算与表示

# 8.1 结果计算

土壤样品中的毒鼠强含量 $w$ （ $\mu \mathrm { g / k g ) }$ ），按照式（1）进行计算。

$$
w = \frac {\rho \times V}{m \times w _ {\mathrm {d m}}} \times 1 0 0 0 \tag {1}
$$

式中： $w$ ——土壤样品中毒鼠强的含量， $\mu \mathrm { g / k g }$ ；

$\rho$ -从标准曲线中查得的毒鼠强质量浓度， $\mathrm { m g / L }$

V——提取液净化浓缩后定容的体积，ml；

$m$ —样品量，g；

$w _ { \mathrm { d m } }$ —干物质含量（质量分数）， $\%$ 。

# 8.2 结果表示

测定结果小于 $1 0 0 ~ \mu \mathrm { g / k g }$ 时，保留小数点后一位，测定结果大于等于 $1 0 0 ~ \mu \mathrm { g / k g }$ 时，保留三位有效数字。

# 10 质量保证和质量控制

10.1 每批样品应至少做一个空白试验，测定结果应低于方法检出限。
10.2 每 10 个样品应分析一个校准曲线的中间点浓度标准溶液，其测定结果与最近一次校准曲线该点浓度的相对偏差应小于等于 $20 \%$ 。
10.3 每批样品应至少测定 $10 \%$ 的平行双样，样品数量少于 10 个时，应至少测定一个平行双样，两次平行测定结果的相对偏差应小于等于 $20 \%$ 。
10.4 每批样品应至少测定 $10 \%$ 的加标样品，样品数量少于 10 个时，应至少测定一个加标样品，加标回收率应在 $7 0 \% \sim 1 2 0 \%$ 。

# 11 废物处理

毒鼠强属于剧毒化学品，实验结束后，实验所用器具应用乙酸乙酯洗涤干净，实验过程中产生的所有废液应置于密闭容器中保存，委托相关单位进行处理。
