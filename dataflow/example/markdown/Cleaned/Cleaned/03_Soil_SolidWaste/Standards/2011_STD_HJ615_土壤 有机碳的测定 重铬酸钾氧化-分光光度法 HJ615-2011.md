# 土壤有机碳的测定重铬酸钾氧化-分光光度法

# 1 适用范围

本标准规定了测定土壤中有机碳的重铬酸钾氧化-分光光度法。

本标准适用于风干土壤中有机碳的测定。本标准不适用于氯离子（Cl− ）含量大于 $2 . 0 { \times } 1 0 ^ { 4 } \mathrm { m g / k g }$ 的盐渍化土壤或盐碱化土壤的测定。

当样品量为 $0 . 5 \ : \mathrm { g }$ 时，本方法的检出限为 $0 . 0 6 \%$ （以干重计），测定下限为 $0 . 2 4 \%$ （以干重计）。

# 2 规范性引用文件

本标准内容引用了下列文件或其中的条款。凡是不注明日期的引用文件，其有效版本适用于本标准。

HJ 613 土壤干物质和水分的测定重量法

HJ/T 166 土壤环境监测技术规范

# 3 方法原理

在加热条件下，土壤样品中的有机碳被过量重铬酸钾-硫酸溶液氧化，重铬酸钾中的六价铬（ $\mathrm { C r } ^ { 6 + }$ ）被还原为三价铬 $( \mathrm { C r } ^ { 3 + } )$ ），其含量与样品中有机碳的含量成正比，于 $5 8 5 \mathrm { n m }$ 波长处测定吸光度，根据三价铬（ ${ \mathrm { ~ C r } } ^ { 3 + }$ ）的含量计算有机碳含量。

# 4 干扰和消除

4.1 土壤中的亚铁离子（ ${ ( \mathrm { F e } ^ { 2 + } }$ ）会导致有机碳的测定结果偏高。可在试样制备过程中将土壤样品摊成 $2 \sim$ $3 \mathrm { c m }$ 厚的薄层，在空气中充分暴露使亚铁离子（ ${ \mathrm { F e } } ^ { 2 + }$ ）氧化成三价铁离子（ ${ \mathrm { F e } } ^ { 3 + }$ ）以消除干扰。
4.2 土壤中的氯离子（ $\mathrm { C l ^ { - } }$ ）会导致土壤有机碳的测定结果偏高，通过加入适量硫酸汞以消除干扰。

# 5 试剂和材料

除非另有说明，分析时均使用符合国家标准的分析纯化学试剂，实验用水为在 $2 5 \mathrm { { ^ \circ C } }$ 下电导率 ${ \leqslant } 0 . 2 \ : \mathrm { m S / m }$ 的去离子水或蒸馏水。

5.1 硫酸： $\rho ( \mathrm { H } _ { 2 } \mathrm { S O } _ { 4 } ) { = } 1 . 8 4 \ \mathrm { g } / \mathrm { m l } .$ 。
5.2 硫酸汞。
5.3 重铬酸钾溶液： $c ( \mathrm { K } _ { 2 } \mathrm { C r } _ { 2 } \mathrm { O } _ { 7 } ) { = } 0 . 2 7 ~ \mathrm { m o l / L }$ 。

称取 $8 0 . 0 0 \ \mathrm { g }$ 重铬酸钾溶于适量水中，溶解后移至 $1 \ 0 0 0 \ \mathrm { m l }$ 容量瓶，用水定容，摇匀。该溶液贮存于试剂瓶中， $4 \mathrm { { } ^ { \circ } C }$ 下保存。

5.4 葡萄糖标准使用液： $\rho ( \mathrm { C } _ { 6 } \mathrm { H } _ { 1 2 } \mathrm { O } _ { 6 } ) { = } 1 0 . 0 0 ~ \mathrm { g / L }$ 。

称取 $1 0 . 0 0 \ \mathrm { g }$ 葡萄糖溶于适量水中，溶解后移至 $1 \ 0 0 0 \ \mathrm { m l }$ 容量瓶，用水定容，摇匀。该溶液贮存于试剂瓶中，有效期为一个月。

# 6 仪器和设备

6.1 分光光度计：具 $5 8 5 \mathrm { n m }$ 波长，并配有 $1 0 \mathrm { m m }$ 比色皿。
6.2 天平：精度为 $0 . 1 \mathrm { m g }$ 。
6.3 恒温加热器：温控精度为 $1 3 5 \mathrm { ^ \circ C } \pm 2 \mathrm { ^ \circ C }$ 。恒温加热器带有加热孔，其孔深应高出具塞消解玻璃管内液面约 $1 0 \mathrm { m m }$ ，且具塞消解玻璃管露出加热孔部分约 $1 5 0 \mathrm { m m }$ 。
6.4 具塞消解玻璃管：具有 $1 0 0 ~ \mathrm { m l }$ 刻度线，管径为 $3 5 { \sim } 4 5 ~ \mathrm { m m }$ 。

注：具塞消解玻璃管外壁必须能够紧贴恒温加热器的加热孔内壁，否则不能保证消解完全。

6.5 离心机： $0 { \sim } 3 0 0 0 \mathrm { r / m i n }$ ，配有 $1 0 0 \mathrm { m l }$ 离心管。
6.6 土壤筛： $2 \mathrm { m m }$ （10 目）、 $0 . 2 5 \mathrm { m m }$ （60 目），不锈钢材质。
6.7 一般实验室常用仪器和设备。

# 7 样品

# 7.1 样品的采集和保存

土壤样品的采集和保存参照 HJ/T 166 的相关规定。

# 7.2 试样的制备

将土壤样品置于洁净白色搪瓷托盘中，平摊成 $2 { \sim } 3 ~ \mathrm { c m }$ 厚的薄层。先剔除植物、昆虫、石块等残体，用木棰压碎土块，自然风干，风干时每天翻动几次。充分混匀风干土壤，采用四分法，取其两份，一份留存，一份通过 $2 \mathrm { m m }$ 土壤筛用于干物质含量测定。在过 $2 \mathrm { m m }$ 筛的样品中取出 $1 0 { \sim } 2 0 ~ \mathrm { g }$ 进一步细磨，并通过 60 目（ $0 . 2 5 \mathrm { m m }$ ）土壤筛，装入棕色具塞玻璃瓶中，待测。

# 7.3 干物质含量的测定

准确称取适量风干土壤，参照 HJ 613 测定干物质的含量。

# 8 分析步骤

# 8.1 校准曲线的绘制

8.1.1 分别量取 0.00、0.50、1.00、2.00、4.00 和 $6 . 0 0 \mathrm { m l }$ 葡萄糖标准使用液（5.4）于 $1 0 0 \mathrm { m l }$ 具塞消解玻璃管中，其对应有机碳质量分别为 0.00、2.00、4.00、8.00、16.0 和 $2 4 . 0 \mathrm { m g }$ 。
8.1.2 分别加入 $0 . 1 \mathrm { \ g }$ 硫酸汞（5.2）和 $5 . 0 0 \mathrm { m l }$ 重铬酸钾溶液（5.3），摇匀。再缓慢加入 $7 . 5 \mathrm { m l }$ 硫酸（5.1），轻轻摇匀。
8.1.3 开启恒温加热器，设置温度为 $1 3 5 \mathrm { { \small C } }$ 。当温度升至接近 $1 0 0 ^ { \circ } \mathrm { C }$ 时，将上述具塞消解玻璃管开塞放入恒温加热器的加热孔中，以仪器温度显示 $1 3 5 \mathrm { { \small C } }$ 时开始计时，加热 $3 0 \mathrm { { m i n } }$ 。然后关掉恒温加热器开关，取出具塞消解玻璃管水浴冷却至室温。向每个具塞消解玻璃管中缓慢加入约 $5 0 ~ \mathrm { m l }$ 水，继续冷却至室温。再用水定容至 $1 0 0 ~ \mathrm { m l }$ 刻线，加塞摇匀。
8.1.4 于波长 $5 8 5 \mathrm { n m }$ 处，用 $1 0 \mathrm { m m }$ 比色皿，以水为参比，分别测量吸光度。
8.1.5 以零浓度校正吸光度为纵坐标，以对应的有机碳质量（mg）为横坐标，绘制校准曲线。

# 8.2 测定

准确称取适量试样（7.2），小心加入到 $1 0 0 \mathrm { m l }$ 具塞消解玻璃管中，避免沾壁。按照8.1.2加入试剂，按照 8.1.3 进行消解、冷却、定容。将定容后试液静置 1 h，取约 $8 0 \mathrm { m l }$ 上清液至离心管中以 $2 0 0 0 \mathrm { r / m i n }$ 离心分离 $1 0 \mathrm { m i n }$ ，再静置至澄清；或在具塞消解玻璃管内直接静置至澄清。最后取上清液按照 8.1.4 测量吸光度。土壤有机碳含量与试样取样量关系见表 1。

表 1 土壤有机碳含量与试样取样量关系


| 土壤有机碳含量/% | 0.00~4.00       | 4.00~8.00       | 8.00~16.0       |
| --------- | --------------- | --------------- | --------------- |
| 试样取样量/g   | 0.400 0~0.500 0 | 0.200 0~0.250 0 | 0.100 0~0.125 0 |


注1：当样品有机碳含量超过 $1 6 . 0 \%$ 时，应增大重铬酸钾溶液的加入量，重新绘制校准曲线。
注2：一般情况下，试液离心后静置至澄清约需5 h或直接静置至澄清约需8 h。

# 8.3 空白试验

在具塞消解玻璃管中不加入试样，按照 8.1.2、8.1.3 和 8.1.4 的步骤进行测定。

# 9 结果计算与表示

# 9.1 结果计算

土壤中的有机碳含量（以干重计，质量分数， $\%$ ），按照式（1）、式（2）进行计算。

$$
m _ {1} = m \times \frac {w _ {\mathrm {d m}}}{1 0 0} \tag {1}
$$

$$
w _ {\mathrm {o c}} = \frac {\left(A - A _ {0} - a\right)}{b \times m _ {1} \times 1 0 0 0} \times 1 0 0 \tag {2}
$$

式中： $m _ { 1 }$ —试样中干物质的质量， g；

$m$ 试样取样量，g；

${ w _ { \mathrm { d m } } }$ wdm 土壤的干物质含量（质量分数）， $\%$ ；

$\omega _ { \mathrm { o c } }$ @0C 土壤样品中有机碳的含量（以干重计，质量分数)， $\%$

$A$ —试样消解液的吸光度；

$A _ { 0 }$ ——空白试验的吸光度；

$a$ 校准曲线的截距；

$^ { b }$ 校准曲线的斜率。

# 9.2 结果表示

当测定结果 $< 1 . 0 0 \%$ 时，保留到小数点后两位；当测定结果 $\geq 1 . 0 0 \%$ 时，保留三位有效数字。

# 10 精密度和准确度

# 10.1 精密度

6 家实验室对有机碳含量为 $1 . 8 0 \%$ 的统一样品进行了测定：实验室内相对标准偏差为 $0 . 6 \% { \sim } 4 . 0 \%$ ，实验室间相对标准偏差为 $4 . 1 \%$ ，重复性限为 $0 . 1 2 \%$ ，再现性限为 $0 . 2 4 \%$ 。

# 10.2 准确度

6 家实验室对有机碳含量为（ $1 . 8 0 { \pm } 0 . 1 6$ ）%的有证标准样品进行了测定：相对误差为 $2 . 2 \% { \sim } 8 . 3 \%$ ，相对误差最终值为 $5 . 6 \% { \pm } 5 . 2 \%$ 。

# 11 质量保证和质量控制

11.1 每批样品应做两个空白试验，两个测定结果的相对偏差应 $\text{‰}$ 。式（2）中 $A _ { 0 }$ 为两个空白试验测定的平均值。
11.2 每 20个样品应至少测定 $10 \%$ 的平行双样，样品数量少于 10 个时，每批样品应至少测定一个平行双样。当样品的有机碳含量 $\leqslant 1 . 0 0 \%$ 时，两个测定结果之差应在 $\pm 0 . 1 0 \%$ 之内；当样品的有机碳含量$> 1 . 0 0 \%$ 时，两个测定结果的相对偏差 $\leqslant 1 0 . 0 \%$ 。
11.3 每批样品测定时，应分析一个有证标准物质，其测定值应在保证值范围内。
11.4 校准曲线的相关系数应大于等于 0.999。

# 12 注意事项

12.1 为保证恒温加热器加热温度的均匀性，样品进行消解时，在没有样品的加热孔内放入装有 $1 5 ~ \mathrm { m l }$ 硫酸（5.1）的具塞消解玻璃管，避免恒温加热器空槽加热。
12.2 硫酸具有较强的化学腐蚀性，操作时应按规定要求佩戴防护器具，避免接触皮肤和衣物。样品消解应在通风橱内进行操作。检测后的废液应妥善处理。
