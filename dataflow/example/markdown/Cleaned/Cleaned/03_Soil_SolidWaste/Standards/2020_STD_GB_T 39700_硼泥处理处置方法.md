# 硼泥处理处置方法

# 1 范围

本文件规定了硼泥的组成、处理处置方法与环境保护要求。

本文件适用于硼砂、硼酸生产产生的硼泥的处理处置，不适用于含有铀等放射性物质的硼泥的处理处置。

# 3 术语和定义

下列术语和定义适用于本文件。

3.1

硼泥 boron mud

以硼镁矿或盐湖硼矿为原料生产硼砂或硼酸产生的泥状物。

# 4 硼泥的组成

硼镁矿产生的硼泥主要组分：氧化镁 $25\% \sim 42\%$ 、二氧化硅 $15\% \sim 25\%$ 、氧化铁 $5\% \sim 15\%$ 、氧化钙 $3\% \sim 4\%$ 、氧化硼 $2\% \sim 5\%$ ，含水率 $8\% \sim 60\%$ 。

盐湖硼矿产生的硼泥主要组分：氧化镁 $1\% \sim 10\%$ 、二氧化硅 $0.5\% \sim 5\%$ 、氧化铁 $0.5\% \sim 5\%$ 、氧化钙 $2\% \sim 12\%$ 、氧化硼 $1\% \sim 6\%$ ，含水率 $8\% \sim 60\%$ 。

# 5 处理处置方法

# 5.1 生产硼镁硅钙颗粒（粉）

# 5.1.1 适用范围

适用于处理硼镁矿采用碳碱法工艺生产硼砂产生的硼泥。

# 5.1.2 工艺流程

在湿硼泥中加入工业硫酸酸化、活化，控制加入的量，搅拌一定时间后放置使其硬化，失水（或露天自然晾晒干燥）。将硬化干燥后的硼泥进行粉碎，添加调制剂调整成分后，以稀硫酸(或水等)为黏合剂进行喷酸(或喷水)造粒。冷却筛分后，大于粒径控制要求的颗粒返回粉碎工序；小于粒径控制要求的返回到混合、混料工序。筛分后包装得到成品。生产中粉尘收集后返回混合配料工序。硼泥制硼镁硅钙颗粒(粉)工艺流程简图见图1。

# 5.1.3 工艺参数

硼泥制硼镁硅钙颗粒（粉）工艺主要控制指标如下：

稀硫酸控制体积浓度为 $25\% \sim 35\%$
——造粒控制粒径 $(\varphi)$ 为 $2\mathrm{mm}\sim 4\mathrm{mm}$
——控制喷酸后颗粒 $\mathrm{pH}$ 值为 $6\sim 8$

# 5.1.4 生产设备

主要生产设备：料釜、混合机、粉碎机、造粒机、输送机、筛分设备、包装机及成套除尘设备等。

# 5.2 生产冶金烧结球团

# 5.2.1 适用范围

适用于硼化物含量不小于 $2\%$ ，主要含镁、钙、铁的硼泥。

# 5.2.2 原理

硼具有能控制烧结过程中硅酸二钙的产生并阻止其相变作用，减少粉化。氧化硼熔点低，可与许多氧化物形成固熔体并降低其熔点，烧结过程中可以促进液相核的产生，从而降低渣相中的熔点和黏度，促使在相同的温度下早成渣与多成渣，加强了胶结作用，提高了球团矿的强度。

# 5.2.3 工艺流程

将硼泥在晒场进行自然风干后运送至烟气干燥工序，经过干燥后根据球团成分需求添加部分添加剂调整硼泥中元素含量，然后进入粉碎机粉碎研磨至要求粒度，与精矿粉、其他主要球团成团添加剂进行混合配料、造球后入炉焙烧，焙烧后进行冷却、筛分后得到冶金烧结球团。生产中返尘及烟尘收集后返回配料混合工序。硼泥制冶金烧结球团工艺流程简图见图2。

# 5.2.4 工艺参数

硼泥制冶金烧结球团工艺主要控制指标如下：

——晒场自然干燥后控制水分小于 $40\%$
——烟气干燥温度不大于 $850^{\circ}\mathrm{C}$ ；
干燥后控制水分为 $8\% \sim 12\%$
——粉碎后粒度要求小于 $0.074\mathrm{mm}$ 的泥粉质量分数大于 $80\%$
——作为球团添加剂添加量控制为 $1.5\% \sim 3.0\%$
球团焙烧温度为 $1100^{\circ}\mathrm{C}\sim 1300^{\circ}\mathrm{C}$

# 5.2.5 生产设备

主要生产设备：成套烘干塔或烘干窑，运料器、粉碎研磨机、给料机，配料装置、强力混合机，造球机、筛分机、干燥预热器、焙烧窑、环冷机及成套收尘除尘设备等。

# 5.3 生产硼泥建材制品

# 5.3.1 适用范围

适用于符合GB6566要求，且氧化镁含量大于 $20\%$ ，氧化硼含量为 $1.5\% \sim 5\%$ ，二氧化硅含量为 $5\% \sim 30\%$ 的硼泥。

# 5.3.2 工艺流程

生产硼泥建材制品有多种配料方案，推荐配料如下：

a) 配料一: 原料干料包括硼泥、轻烧氧化镁、珍珠岩、聚丙烯纤维、稻壳粉、锯末、粉煤灰; 原料湿料包括氯化镁溶液、聚苯乳液、 $\gamma$ -氨丙基三乙氧基硅烷、水泥发泡剂及改性硅树脂聚醚微乳液 (MPS)等;
b）配料二：原料包括硼泥、水泥、砂子、生石灰、水、其他外加剂及发泡剂等。

将上述配料中的原料按照配比配料后，充分搅拌得到料浆泥，将料浆泥注入模具中，经养护、脱模后可选择进入蒸压养护设备进行蒸养或检验后得到成品。以上两种配料的硼泥制建材制品工艺流程简图分别见图3、图4：

# 5.3.3 工艺参数

硼泥制建材制品工艺主要控制指标如下：

——配料一中干料质量占比：硼泥 $20\% \sim 50\%$ ，轻烧氧化镁 $15\% \sim 35\%$ ，珍珠岩不大于 $5\%$ ，聚丙烯纤维不大于 $5\%$ ，稻壳粉 $1\% \sim 5\%$ ，锯末 $1\% \sim 5\%$ ，粉煤灰 $3\% \sim 5\%$ ；湿料质量占比：氯化镁溶液 $20\% \sim 40\%$ ，聚苯乳液不大于 $2\%$ ， $\gamma$ -氨丙基三乙氧基硅烷不大于 $1\%$ ，水泥发泡剂不大于 $5\%$ ，改性硅树脂聚醚微乳液(MPS)不大于 $2\%$ 。

配料二中组分质量占比：硼泥 $30\% \sim 40\%$ ，水泥 $15\% \sim 35\%$ ，砂子 $20\% \sim 35\%$ ，生石灰 $10\% \sim 15\%$ ，外加剂等约 $1\%$ ，控制水胶质量比 $0.5\sim 0.6$ ，砂子二氧化硅含量应大于 $90\%$ ，粒径要求 $0.08\mathrm{mm}$ 方孔筛筛余物不大于 $10\%$ ，外加剂为水泥减水增强剂。

——自然养护的时间为 $8\mathrm{h}\sim 24\mathrm{h}$ ，自然养护过程中控制含水率小于 $20\%$ 。
——蒸压养护前静停养护时间约为 $2\mathrm{h}$ 。
——蒸压养护控制温度为 $170^{\circ}\mathrm{C}\sim 200^{\circ}\mathrm{C}$ 。

# 5.3.4 生产设备

主要生产设备：成套配料设备，搅拌设备、灌浆机、蒸压养护设备及成套收尘除尘设备等。

# 6 环境保护要求

6.1 处理处置硼泥过程中产生的粉尘、废气应收集处理后排放，且应达到相关废气排放环保要求。
6.2 处理处置硼泥过程中产生的废水应处理后回用，不能回用的应达到相关废水排放环保要求。
6.3 处理处置硼泥过程中产生的粉尘、烟尘及其他废料回用生产系统不应产生废渣。
