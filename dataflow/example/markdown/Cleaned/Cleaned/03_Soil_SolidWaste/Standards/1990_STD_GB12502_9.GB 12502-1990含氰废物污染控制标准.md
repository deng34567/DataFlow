# 1 主题内容与适用范围

# 1.1 主题内容

本标准规定了含氰废物污染控制的分级方法和含氰废物污染控制标准值，以及含氰废物的处置与监测方法。

# 1.2 适用范围

本标准适用于任何过程中产生的含氰废物。

# 2 引用标准

GB5086有色金属工业固体废物浸出毒性试验方法标准

GB7486 氰化物的测定第一部分：总氰化物的测定

# 3 术语

# 3.1 含氰废物

指任何过程中产生的含氰固态或半固态废物，包括尾矿、废渣、污泥和尘渣等。

# 3.2 废物含氰量

指废物在浸出液中总氰化物的浓度。

# 4 标准分级

含氰废物污染控制标准分为两级。

# 4.1 第一级

指本标准实施之日起，新建、扩建、改建的企事业单位应执行的标准。

# 4.2 第二级

指本标准实施之前，已有企事业单位应执行的标准。

# 5 标准值

mg/L

含氰废物污染控制标准


| 项目          | 第一级  | 第二级  |
| ------------ | ---- | ---- |
| 废物含氰量(以CN-计) | ≤1.0 | ≤1.5 |


# 6 含氰废物的处置

# 6.1 一般含氰废物的处置

凡符合表中标准值的含氰废物，属于一般含氰废物，可在环境保护部门批准的场地堆放。

# 6.2 有害含氰废物的处置

凡大于表中标准值的含氰废物，属于有害含氰废物，应在具有防水、防渗、防扬散、防流失的专用处置场所堆放，并设立明显标记。运输此类废物要遵守公安交通部门的有关规定。

# 7 标准实施

7.1 本标准由各级政府环境保护行政主管部门负责监督与实施。
7.2 本标准不能满足地方环境保护要求时，省、自治区、直辖市人民政府可以制定严于此标准的地方含氰废物污染控制标准，并报国务院环境保护行政主管部门备案。

# 附录A

# 取样与制样

（补充件）

A1 取样

A1.1 生产线上取样。应在废物排出口，每天取样一次，取样量 $2\mathrm{kg}$ 。连续取样三天，混合均匀后，取 $2\mathrm{kg}$ 作为试样。
A1.2废物堆取样。在堆的顶、中、底三部位的断面，每个断面周边上设等距离的四个采样点，在每点离表层 $30~\mathrm{cm}$ 深处采样 $1\mathrm{kg}$ 。混合均匀后，取 $2\mathrm{kg}$ 作试样。
A1.3污泥池取样。在污泥堆放池的四周及中心设5个取样点，各取 $0.5\mathrm{kg}$ 。混合均匀后，作为试样。
A2 制样

大块试样经破碎后，使之全部通过 $5\mathrm{mm}$ 筛孔，混合均匀后，装入聚乙烯塑料瓶中备测定用。

# 附录B

# 浸出液的制备

（补充件）

B1 浸出用水
用 $0.1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液调节蒸馏水 $\mathrm{pH}$ 值为 $8\sim 9$
B2 浸出液的制备

称取试样 $100\mathrm{g}$ (以干渣重量计)于 $2\mathrm{L}$ 聚乙烯塑料瓶中，加入浸出用水 $1000\mathrm{mL}$ (包括试样的含水量)，密塞，在振荡器上振荡 $4\mathrm{h}$ (振荡频率 $100\sim 120$ 次/min)，放置，待澄清后，用 $0.45\mu \mathrm{m}$ 滤膜过滤，弃去初滤液 $30\mathrm{mL}$ 后，收集其余滤液于塑料瓶内。

浸出液中氰化物的测定应在当天进行，如果不能及时测定，应加入固体氢氧化钠，使滤液 $\mathsf{pH}$ 值至12，放入 $4^{\circ}C$ 冰箱内保存。
