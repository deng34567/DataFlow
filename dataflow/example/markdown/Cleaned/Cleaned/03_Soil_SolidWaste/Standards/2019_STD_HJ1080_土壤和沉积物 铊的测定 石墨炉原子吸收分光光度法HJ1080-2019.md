# 土壤和沉积物铊的测定石墨炉原子吸收分光光度法

警告：实验中所使用的铊标准溶液属于有毒化学品；实验用酸具有挥发性和腐蚀性，溶液配制及样品前处理过程应在通风橱中进行，操作时应按要求佩戴防护器具，避免直接接触皮肤和衣物。

# 1 适用范围

本标准规定了测定土壤和沉积物中铊的石墨炉原子吸收分光光度法。

本标准适用于土壤和沉积物中铊总量的测定。

当称样量为 $0 . 5 \mathrm { \textrm { g } }$ ，定容体积为 $5 0 \ \mathrm { m l }$ 时，本方法检出限为 $0 . 1 \mathrm { \ m g / k g }$ ，测定下限为0.4 mg/kg。

# 3 方法原理

土壤和沉积物样品经消解后，注入石墨炉原子化器中，铊及其化合物形成基态原子，对特征谱线（ $2 7 6 . 8 \ : \mathrm { n m }$ ）产生选择性吸收，其吸收强度在一定范围内与铊含量成正比。

# 4 干扰和消除

氯离子对铊的测定产生负干扰，对铊浓度为 $2 0 . 0 \mu \mathrm { g / L }$ 的试样，当氯离子浓度为 $5 0 \mathrm { m g / L }$ 时，测定吸光度下降 $50 \%$ 。当氯离子浓度低于 $1 0 0 0 \mathrm { m g / L }$ 时，加入硝酸钯和抗坏血酸基体改进剂可消除干扰。

# 7.4.1 电热板消解法

准确称取 $0 . 2 \ \mathrm { g } \{ \sim  0 . 5 \ \mathrm { g }$ （精确至 $0 . 1 \mathrm { m g }$ ）样品置于 $5 0 \mathrm { m l }$ 聚四氟乙烯坩埚中，加 $2 \sim 3$ 滴水湿润试样后加 $1 0 ~ \mathrm { m l }$ 硝酸（5.1）、 $3 ~ \mathrm { m l }$ 氢氟酸（5.2）， $1 6 0 ^ { \circ } \mathrm { C } \sim 1 8 0 ^ { \circ } \mathrm { C }$ 加盖消解至无明显黑色物质，若黑色物质较多，可补加 $2 \mathrm { m l }$ 硝酸（5.1）和 $2 \mathrm { m l } \sim 3 \mathrm { m l }$ 双氧水（5.3）继续加盖消解，重复上述消解过程。 $1 8 0 ^ { \circ } \mathrm { C }$ 开盖赶酸（为达到良好的赶酸效果，应经常摇动坩埚），蒸至近干（内容物呈不流动的粘稠状）。取下坩埚稍冷，加入 $0 . 5 \mathrm { \ m l }$ 硝酸（5.1），温热溶解可溶性残渣，冷却后转移至 $5 0 \mathrm { m l }$ 容量瓶中，用水定容至标线，摇匀，静置取上清液待测或经滤膜（5.14）过滤后待测。

# 9 结果计算与表示

# 9.1 结果计算

# 9.1.1 土壤样品的结果计算

土壤样品中铊元素的含量 $w _ { 1 }$ （ $\mathrm { m g / k g }$ ），按照式（1）进行计算。

$$
w _ {1} = \frac {\rho \times V \times D}{m \times w _ {d m} \times 1 0 0 0} \tag {1}
$$

式中： $w _ { 1 } .$ —土壤样品中铊元素的含量， $\mathrm { m g / k g }$ ；

$\rho$ 由标准曲线计算所得试样中铊元素的质量浓度， $\mu \mathrm { g / L }$ ；

V——消解后试样的定容体积，ml；

D—试样的稀释倍数；

m—称取土壤样品的质量，g；

wdm—土壤样品干物质的含量， $\%$ 。

# 9.1.2 沉积物样品的结果计算

沉积物样品中铊元素的含量 $w _ { 2 }$ （ $\mathrm { m g / k g }$ ），按照式（2）进行计算。

$$
w _ {2} = \frac {\rho \times V \times D}{m \times \left(1 - w _ {H _ {2} O}\right) \times 1 0 0 0} \tag {2}
$$

式中： $w _ { 2 } .$ —沉积物样品中铊元素的含量， $\mathrm { m g / k g }$ ；

$\rho$ 由标准曲线计算所得试样中铊元素的质量浓度， $\mu \mathrm { g / L }$ ；

V——消解后试样的定容体积，ml；

$D _ { - }$ 试样的稀释倍数；

m—称取沉积物样品的质量，g；

$w _ { H _ { 2 } O }$ wH2O 沉积物样品含水率， $\%$ 。

# 11 质量保证和质量控制

11.1 每批样品至少要做 2 个空白试样，其测定结果应不超过方法检出限。

11.2 每次分析应建立标准曲线，标准曲线相关系数 ${ \geqslant } 0 . 9 9 5$ 。每分析 20 个样品，应分析一次标准曲线中间浓度点，其测定值与标准值的相对误差应在 $\pm 1 0 \%$ 之内。
11.3 每 20 个样品或每批样品（少于 20 个样品/批）测定 1 个平行样，平行样测定结果的相对偏差应 ${ \leqslant } 2 5 \%$ 。
11.4 每 20 个样品或每批样品（少于 20 个样品/批）测定 1 个土壤/沉积物标准样品，其测定结果与标准值的相对误差应在 $\pm 2 5 \%$ 以内。或测定 1 个基体加标样品，加标回收率应在$7 0 \% \sim 1 4 0 \%$ 。

# 12 废物处理

实验中产生的废物应集中收集，并做好相应标识，委托有资质的单位进行处理。

# 13 注意事项

实验中试样的消解、保存和标准溶液的配制过程中应避免引入含氯的物质。
