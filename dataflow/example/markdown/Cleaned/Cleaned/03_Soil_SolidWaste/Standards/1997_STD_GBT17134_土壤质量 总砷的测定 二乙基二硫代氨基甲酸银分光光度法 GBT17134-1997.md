# 1 主题内容与适用范围

1.1 本标准规定了测定土壤中总砷的二乙基二硫代氨基甲酸银分光光度法。
1.2 本标准方法的检出限为 $0.5\mathrm{mg / kg}$ （按称取 $1\mathrm{g}$ 试样计算）。
1.3 锑和硫化物对测定有正干扰。锑在 $300\mu \mathrm{g}$ 以下，可用 $\mathrm{KI - SnCl_2}$ 掩蔽。在试样氧化分解时，硫已被硝酸氧化分解，不再有影响。试剂中可能存在的少量硫化物，可用乙酸铅脱脂棉吸收除去。

# 2 原理

通过化学氧化分解试样中以各种形式存在的砷，使之转化为可溶态砷离子进入溶液。锌与酸作用，产生新生态氢。在碘化钾和氯化亚锡存在下，使五价砷还原为三价砷，三价砷被新生态氢还原成气态砷化氢（脾）。用二乙基二硫代氨基甲酸银-三乙醇胺的三氯甲烷溶液吸收砷化氢，生成红色胶体银，在波长 $5^{10}\mathrm{nm}$ 处，测定吸收液的吸光度。

# 3 试剂

除非另有说明，分析中均使用符合国家标准或专业标准的分析纯试剂和蒸馏水或同等纯度的水。

3.1 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ ， $\rho = 1.84\mathrm{g / ml_o}$
3.2 硫酸溶液， $1 + 1$
3.3 硝酸 $(\mathrm{HNO}_3)$ ， $\rho = 1.42\mathrm{g / ml_o}$
3.4 高氯酸 $\mathrm{(HClO_4)}$ ， $\rho = 1.67\mathrm{g / ml_o}$
3.5 盐酸（HCl）， $\rho = \mathrm{~\sim~\sim~g~ / ~ml~}$
3.6碘化钾（KI）溶液：将 $15\mathrm{g}$ 碘化钾（KI）溶于蒸馏水中并稀释至 $\mathrm{100ml}$
3.7 氯化亚锡溶液：将 $40\mathrm{g}$ 氯化亚锡（ $\mathrm{SnCl_2\cdot 2H_2O}$ ）置于烧杯中，加入 $40\mathrm{ml}$ 盐酸（3.5），微微加热。待完全溶解后，冷却，再用蒸馏水稀释至 $100\mathrm{ml}$ 。加数粒金属锡保存。
3.8 硫酸铜溶液：将 $15\mathrm{g}$ 硫酸铜（ $\mathrm{CuSO_4\cdot 5H_2O}$ ）溶于蒸馏水中并稀释至 $100\mathrm{ml}$ 。
3.9 乙酸铅溶液：将 $\mathrm{g}$ 乙酸铅 $\left[\mathrm{Pb}\left(\mathrm{CH}_{2} \mathrm{COO}\right)_{2} \cdot \mathrm{H}_{2} \mathrm{O}\right]$ 溶于蒸馏水中并稀释至 $\sim 100 \mathrm{ml}$ 。
3.10 乙酸铅棉花：将 $10\mathrm{g}$ 脱脂棉浸于 $100\mathrm{ml}$ 乙酸铅溶液（3.9）中，浸透后取出风干。
3.11 无砷锌粒（ $10\sim 20$ 目）。
3.12 二乙基二硫代氨基甲酸银 $\mathrm{(C_2H_{16}NS_2Ag)}$
3.13 三乙醇胺 $(\mathrm{(HOCH_3CH_3)_3N)}$ 。
3.14 三氯甲烷（ $\mathrm{CHCl}_3$ ）。
3.15 吸收液：将 $0.25\mathrm{g}$ 二乙基二硫代氨基甲酸银（3.12）用少量三氯甲烷（3.14）溶成糊状，加入 $2\mathrm{ml}$ 三乙醇胺（3.13），再用氯仿（3.14）稀释到 $100\mathrm{ml}$ 。用力振荡使其尽量溶解。静置暗处 $24\mathrm{h}$ 后，倾出上清液或用定性滤纸过滤，贮于棕色玻璃瓶中。贮存在 $2\sim 5\%$ 冰箱中。

3.16 氢氧化钠溶液， $2\mathrm{mol} / \mathrm{L}$ ：贮存在聚乙烯瓶中。
3.17 砷标准贮备溶液， $1.00\mathrm{mg / ml}$ ：称取放置在硅胶干燥器中充分干燥过的 $0.1320\mathrm{g}$ 三氧化二砷 $(\mathrm{As}_2\mathrm{O}_3)$ ，溶于 $2\mathrm{ml}$ 氢氧化钠溶液（3.16）中，溶解后加入 $10\mathrm{ml}$ 硫酸溶液（3.2），转移到 $100\mathrm{ml}$ 容量瓶中，用蒸馏水稀释至标线，摇匀。

注：三氧化二砷剧毒，小心使用。

3.18 砷标准中间溶液， $100\mathrm{mg / L}$ ：取 $10.00\mathrm{ml}$ 砷标准贮备液（3.17）于 $100\mathrm{ml}$ 容量瓶中，用蒸馏水稀释至标线，摇匀。
3.19 砷标准使用溶液， $1.00\mathrm{mg / L}$ ；取 $1.00\mathrm{ml}$ 砷标准中间溶液（3.18）于 $100\mathrm{ml}$ 容量瓶中，用蒸馏水稀释至标线，摇匀。

# 4 仪器

一般实验室仪器和以下仪器：

4.1 分光光度计： $10\mathrm{mm}$ 比色皿。
4.2 砷化氢发生装置，此仪器由下述部件组成
4.2.1 砷化氢发生瓶：容量为 $150\mathrm{ml}$ 、带有磨口玻璃接头的锥形瓶。
4.2.2 导气管：一端带有磨口接头，并有一球形泡（内装乙酸铅棉花（3.10））；一端被拉成毛细管，管口直径不大于 $1\mathrm{mm}$ 。
4.2.3 吸收管：内径为 $8\mathrm{mm}$ 的试管，带有 $5.0\mathrm{ml}$ 刻度。

注：吸收液柱高保持 $8\sim 10\mathrm{cm}$

# 5样品

将采集的土壤样品（一般不少于 $500\mathrm{g}$ ）混匀后用四分法缩分至约 $100\mathrm{g}$ 。缩分后的土样经风干（自然风干或冷冻干燥）后，除去土样中石子和动植物残体等异物，用木棒（或玛瑙棒）研压，通过 $2\mathrm{mm}$ 尼龙筛（除去 $2\mathrm{mm}$ 以上的砂砾），混匀。用玛瑙研钵将通过 $2\mathrm{mm}$ 尼龙筛的土样研磨至全部通过100目（孔径 $0.149\mathrm{mm}$ ）尼龙筛，混匀后备用。

# 6 分析步骤

# 6.1 试液的制备

称取按步骤5制备的土壤样品 $0.5\sim 2\mathrm{g}$ （准确至 $0.0002\mathrm{g}$ ）于 $150\mathrm{ml}$ 锥形瓶（4.2.1）中，加 $7\mathrm{ml}$ 硫酸溶液（3.2）， $10\mathrm{ml}$ 硝酸（3.3）， $2\mathrm{ml}$ 高氯酸（3.4），置电热板上加热分解，破坏有机物（若试液颜色变深，应及时补加硝酸），蒸至冒白色高氯酸浓烟。取下放冷，用水冲洗瓶壁，再加热至冒浓白

烟，以驱尽硝酸。取下锥形瓶，瓶底仅剩下少量白色残渣（若有黑色颗粒物应补加硝酸继续分解），加蒸馏水至约 $50~\mathrm{ml}$ 。

# 6.2 测定

6.2.1 于盛有试液的砷化氢发生瓶（4.2.1）中，加 $4\mathrm{ml}$ 碘化钾溶液（3.6），摇匀，再加 $2\mathrm{ml}$ 氯化亚锡溶液（3.7），混匀，放置 $15\mathrm{min}$ 。
6.2.2 取 $5.00\mathrm{ml}$ 吸收液（3.15）至吸收管（4.2.3）中，插入导气管（4.2.2）。
6.2.3 加 $1\mathrm{ml}$ 硫酸铜溶液（3.8）和 $4\mathrm{g}$ 无砷锌粒（3.11）于砷化氢发生瓶（4.2.1）中，并立即将导气管（4.2.2）与砷化氢发生瓶（4.2.1）连接，保证反应器密闭。
6.2.4 在室温下，维持反应 $1\mathrm{h}$ ，使砷化氢完全释出。加三氯甲烷（3.14）将吸收液体积补充至5.0ml。

注：砷化氢剧毒，整个反应应在通风橱内进行。

2. 在完全释放砷化氢后，红色生成物在 $2.5\mathrm{h}$ 内是稳定的，应在此期间内进行分光光度测定。

6.2.5 用 $10\mathrm{mm}$ 比色皿，以吸收液（3.15）为参比液，在 $510\mathrm{nm}$ 波长下测量吸收液的吸光度，减去空白试验所测得的吸光度，从校准曲线（6.4）上查出试样中的含砷量。

# 6.3 空白试样

每分析一批试样，按步骤6.1制备至少两份空白试样，并按步骤6.2进行测定。

# 6.4 校准曲线

分别加入0.00、1.00、2.50、5.00、10.00、15.00、20.00及 $25.00\mathrm{ml}$ 砷标准使用溶液（3.19）于八个砷化氢发生瓶（4.2.1）中，并用蒸馏水稀释至 $50\mathrm{ml}$ 。加入 $7\mathrm{ml}$ 硫酸溶液（3.2），以下按6.2所述步骤进行测定。

将测得的吸光度为纵坐标，对应的砷含量（ $\mu \mathrm{g}$ ）为横坐标，绘制校准曲线。

# 7 结果的表示

土样中总砷的含量 $c(\mathrm{mg / kg})$ 按下式计算：

$$
c = \frac {m}{W (1 - f)}
$$

式中： $m$ —测得试液中砷量， $\mu \mathrm{g}$

$W$ —称取土样重量，g；

$f$ ——土样水份含量， $\%$ 。

# 土样水分含量测定

A1称取通过100目筛的风干土样 $5\sim 10\mathrm{g}$ （准确至 $\mathrm{0.01g}$ ，置于铝盒或称量瓶中，在 $105\%$ 烘箱中烘 $4\sim 5\mathrm{h}$ ，烘干至恒重。
A2 以百分数表示的风干土样水分含量 $f$ 按下式计算：

$$
f (\%) = \frac {W _ {1} - W _ {2}}{W _ {1}} \times 100
$$

式中： $f$ —土样水分含量， $\%$

$W_{\mathrm{r}}$ ——烘干前上样重量， $\mathrm{g}$

$W_{2}$ ——烘干后土样重量， $\mathrm{g}$ 。
