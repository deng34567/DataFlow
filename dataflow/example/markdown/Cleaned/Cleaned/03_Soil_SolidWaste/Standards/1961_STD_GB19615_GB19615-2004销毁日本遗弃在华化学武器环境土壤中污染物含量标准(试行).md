# 销毁日本遗弃在华化学武器

# 环境土壤中污染物含量标准（试行）

Pollutants content limitation in soil for the destruction of

chemical weapons abandoned by Japan in China

# 1 范围

本标准规定了销毁日本遗弃在华化学武器土壤中污染物的含量限值及监测方法。

本标准适用于销毁日本遗弃在华化学武器各项设施建设或作业的土壤环境影响评价，以及销毁作业过程和后续环境管理与评价。

# 3 技术内容

3.1 土壤中污染物的含量不得超过表1所列的限值，其他污染物的含量按GB15618的规定执行。

3.2 当土壤中污染物含量超过本标准规定的限值时，应采取措施或限制使用。

# 4 监测

# 4.1 采样原则

4.1.1 根据作业地区的地形、地貌，可采用放射线、同心圆或网格等方法布设采样点。
4.1.2 土壤监测需设置4个以上的采样点，为确定土壤受污染范围可根据实际需要增加采样点数；在主要大气污染源最大落地浓度出现的地区，要加大布点密度。
4.1.3 样品测定值超过本标准规定的含量限值时，须采集土壤剖面样品。
4.1.4 在每一次作业前、作业期间和作业后必须对土壤进行监测，可根据实际需要增加监测频次。
4.1.5 土壤本底样品的采集参照《土壤元素的近代分析方法》（1992，中国环境监测总站编，中国环境科学出版社）的规定执行。
