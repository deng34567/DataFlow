# 4 采样

# 4.1 采样计划

在销毁日本遗弃在华化学武器的过程中，采集土壤样品的目的，在于通过对样品的检测，判断其是否达到处理的目的，是否达到环境标准的要求。为此，采样前必须制定详细的采样计划。计划内容主要包括采样目的和要求，背景调查和现场踏勘，采样程序，安全措施，质量控制，采样记录和报告等。

# 4.1.1 采样目的

在销毁日本遗弃在华化学武器之前和以后的污染土壤中采集代表样，通过检测和分析，获取在允许误差范围内的数据，故采样应符合下述要求：

（1）特性鉴别和分类：鉴别污染物的类型和性质。

(2）环境污染监测：判别销毁处理前后污染物含量的变化，确定污染物的水平分布和纵向分布的状况及范围。

(3) 销毁方式、方法或其他处置方案。

（4）污染环境事故调查分析和应急监测。

(5）环境影响评价：确保销毁处理达到环境法规和环境标准要求，确保生态环境和人员安全。

(6）法律调查、法律责任和仲裁等。

# 4.1.2 背景调查和现场踏勘

根据采样目的和要求，调查下列影响采样计划制定的因素，并进行现场踏勘。

(1) 土壤中污染物的散发、堆存、埋藏或处置的现场踏勘，了解现场及周围环境。污染物散布的区域、散布区域的地形地貌特征；污染物的存在形式、贮存方式、包装物及其损坏程度；污染物载体或其包装物埋藏深度、范围、方式；对生态和人员安全的影响等。

(2）污染物的种类、形态、数量、物理化学性质等。

(3）污染物试验和分析的允许误差和要求。

(4）污染环境、监测分析的历史资料。

# 4.1.3 采样程序

——确定采样目标、目的和要求；

——选派采样人员；

——进行背景调查和现场踏勘；

——确定采样方法；

——确定样品包装；
——确定份样量；
——确定份样数；
——确定采样点；
——选择采样工具；
——制定安全措施；
——制定质量控制措施；
采样；
——采样记录。

# 4.1.4 采样工具

可采用木片、竹片、铁铲、采样钻等工具进行采样；对于有特殊要求的采样，可根据采样要求自选或自行设计合适工具。

# 4.1.5 采样记录和报告

采样时应记录样品的名称、编号、土壤类型、监测项目、采样方法、采样量、采样时间、采样地点、采样点位、采样深度、采样人员等。

# 4.2 化学武器污染情况调查及挖掘过程土壤样品采集

# 4.2.1 网格采样法

4.2.1.1 根据背景调查和现场踏勘的结果，初步确定化学武器埋藏区域面积大小、埋藏深度和埋藏区域污染源中心点。

# 4.2.1.2 采样区域的测量和标定

对采样区域进行实地测量，划定采样区域。根据地理位置的特点，选取适当参照物作为参照点，利用定位仪对采样点的位置进行标定，画出方状网格。

# 4.2.1.3 水平采样密度

在污染区域，每 $(10 \times 10) \mathrm{m}^2$ 建立一个网格，在网格中心附近设置一个取样点。

污染区域外，每 $(30 \times 30) \mathrm{m}^2$ 建立一个网格，在网格中心附近设置一个取样点。

也可以根据调查化学武器埋藏区域的面积及调查目的，设定适当的采样网格间距。

# 4.2.1.4 垂直采样密度

对水平方向建立的采样点，在垂直方向，从表层开始取第一个样，并每隔 $0.5\mathrm{m}$ 取一个样，直至能够达到环境标准的深度为止。

遇到岩石无法继续采样时，采样深度可以适当小些。

# 4.2.1.5 补充采样

(1) 当距离中心点水平距离最远或垂直深度最深的样品经分析后，污染物浓度仍超出环境标准限值时，需要进行补充采样。
(2）对某一采样点的分析结果有怀疑时，应补充采样。

# 4.2.1.6 污染范围的确定

(1) 水平污染范围：分别以直线连接超过环境标准的采样点和与其邻近的符合环境标准的采样点，并在其中间点引垂线，各垂线中点所构成的多边形既为水平污染范围。
(2) 垂直污染范围：超过环境标准的土层与达到环境标准限值的土层的中间，即为垂直污染范围。

# 4.2.1.7 实施挖掘后埋藏坑土壤的确认采样

4.2.1.7.1 埋藏坑污染土壤被挖掘后，为确认挖出后没有残留，需对挖掘后的埋藏区域底面和侧面的土壤进行采样监测。
4.2.1.7.2 土壤采样最少采集挖掘底面中心及4个方位5个点位的样品，再根据挖掘面积的大小，

适当地追加样品采集点位，确认其达到环境标准。

# 4.2.2 土壤剖面采样法

# 4.2.2.1 土壤剖面的建立

# 4.2.2.1.1 确定埋藏区污染源中心点

根据背景调查和现场踏勘的结果，初步确定化学武器埋藏区域面积大小、埋藏深度和埋藏区域污染源中心点，例如 $O$ 点（参见图1）。

# 4.2.2.1.2 建立土壤剖面

通过 $O$ 点，在化学武器埋藏区内沿任一方向开挖一个土壤剖面。土壤剖面的宽度和深度均为污染物浓度值小于环境标准限值的某一点，例如A1（A2）点。

# 4.2.2.1.3 采样密度

建立第一个剖面后，以 $O$ 点为圆点，顺时针或逆时针每隔 $45^{\circ}$ 角建立一个剖面。

# 4.2.2.3 确定污染物在垂直方向的分布和污染深度

以化学武器埋藏坑底部为起点，向下至污染区边缘（设距离为 $L$ ）为第一个取样点A2，取样测定 $c_{\mathrm{A2}}$ ，若大于环境标准限值，则继续向下取样测 $c_{\mathrm{B2}}$ ；依此类推，直至找到污染物含量小于/等于环境标准限值的部位。该位置与化学武器埋藏坑底部之间的距离即为污染物污染深度。

若 $c_{\mathrm{A2}}$ 小于环境标准限值，则将采样点向上延伸一定距离 $1 / 2L$ ，令此点为 $\mathrm{B2^{\prime}}$ 。取 $\mathrm{B2^{\prime}}$ 点土壤检测，若 $c_{\mathrm{B2^{\prime}}}^{\prime}$ 小于环境标准限值，则继续将采样点向上延伸 $1 / 4L$ 至 $\mathrm{C2^{\prime}}$ 点，依此类推；若 $c_{\mathrm{C2^{\prime}}}^{\prime}$ 超过环境标准限值，可认为 $\mathrm{B2^{\prime}}$ 点为恰好是污染物含量小于/等于环境标准限值的部位。该位置与化学武器埋藏坑底部之间的距离即为污染物污染深度。

# 4.2.2.4 污染范围的确定

参照“4.2.1.6污染范围的确定”方法确定。

4.2.2.5 此法原则上适用于圆形、方形、长方形或不规则形状的化学武器埋藏坑及其周边污染土壤的采样点设置。对于长形、蛇形或不规则形状的化学武器埋藏坑及周边和底部污染土壤的采样，可视现场情况现场设计，将埋藏坑划分为若干区段，每区段按土壤剖面法采样。

# 4.2.3 常规采样法

4.2.3.1 确定化学武器污染区中心点。参见4.2.2.1.1。

# 4.2.3.2 采样点的设置

采样点的设置应在按采样计划进行背景调查和现场踏勘的基础上，全面考虑土质类型，成土母质、地形地貌、植被和农作物情况等因素，现场设计、安排。

# 4.2.3.2.1 水平方向采样点的设置

通过污染区中心部位 $O$ 沿任一方向建立一条射线。设射线与化学武器埋藏坑边缘之交点为A，则从A点开始，沿着射线方向设置采样点。采样点按等距离设置，两点之间的距离视污染区面积大小和污染程度现场决定。原则上，采样间距的大小在污染区域可以小些（例如 $10\mathrm{m}$ ；污染区域外，可以大些（例如 $30\mathrm{m}$ ）。

建立第一条射线后，以 $O$ 点为圆点，顺时针或逆时针每隔 $45^{\circ}$ 角建立一条射线。以相同的方法设置采样点。

# 4.2.3.2.2 垂直方向采样点的设置

垂直方向的采样点，与水平方向采样点的位置对应。

在垂直方向，采样点等间隔设置。从表层开始取第一个样，并每隔 $0.5\mathrm{m}$ 取一个样，直至能够达到环境标准的深度为止。

# 4.2.3.3 确定污染区面积和深度

4.2.3.3.1 在水平方向，从A点开始，由里向外，逐点采样检测，直至某采样点之污染物含量小于/等于环境标准限值为止。
4.2.3.3.2 在垂直方向，对应于水平方向各采样点，由上而下，逐点采样检测，直至某采样点之污染物含量小于/等于环境标准限值为止。

# 4.2.3.4 污染范围的确定

参照“4.2.1.6污染范围的确定”方法确定。
4.2.3.5 此法原则上适用于圆形或近似圆形的化学武器埋藏坑及其周边污染土壤的采样。

# 4.3 化学武器销毁现场周边环境的土壤采样

4.3.1 化学武器销毁现场周边环境土壤，在销毁前、销毁过程中及销毁后均应进行采样监测。
4.3.2 以化学武器销毁设施为中心，网格采样法布点采样。在主导风向上采样点间距小些，其它方向上采样点间距可大些。
4.3.3 采样点一般可采表层土样。采样深度 $0 \sim 15 \mathrm{~cm}$ 。
4.3.4 必要时采用分层采样。从表层开始取第一个样，每隔一定间距取一个样，采样总深度由实际情况而定。

# 4.4 销毁处理过程污染事故监测的土壤采样

4.4.1 现场调查和观察，取证土壤被污染时间，根据污染物及其对土壤的影响确定监测项目，尤其是污染事故的特征污染物是监测的重点。根据污染物的颜色、气味并结合考虑地势、风向等因素初步界定污染事故对土壤的污染范围。

# 4.4.2 固体污染物抛洒污染的土壤采样

等打扫后采集表层 $5\mathrm{cm}$ 土样，采样点数不少于5个。

# 4.4.3 固体废物及污染土壤临时存放地点污染的土壤采样

采用每个点分层采样，采样总深度由实际情况而定。采样点数目根据现场的污染范围确定，采样点不少于5个。

# 4.4.4 液体倾翻污染的土壤采样

污染物向低洼处流动的同时向深度方向渗透并向两侧横向方向扩散，每个点分层采样。一般地，事故发生点采样点间距小些，采样深度较深，离事故发生点相对远处采样点间距大些，采样深度较

浅。采样点不少于5个。

# 4.4.5 爆炸污染的土壤采样

以放射性同心圆方式布点，采样点不少于5个，爆炸中心分层采样，周围采表层土（ $0\sim 15\mathrm{cm}$ ）。可参照“4.2.3常规采样法”进行采样。

# 4.5 其他情况的土壤采样

销毁化学武器全过程中其他情况的土壤采样可原则上按照网格法、梅花法、蛇形法或其他科学方法布设采样点，必要时对土壤进行分层采样。采样点数目及采样深度根据现场情况确定。

# 4.6 对照土壤样品的采集

# 4.6.1 对照土壤采样点设置的基本要求

4.6.1.1 土壤的母质、组成、结构、性能基本相同。

4.6.1.2 化学武器对对照土壤的污染影响忽略不计。

# 4.6.2 对照土壤采样点的设置

# 4.6.2.1 采样点的布设

对照土壤采样点应布设在污染区域边界以外的适当距离内，结合现场情况，选择3~5个对照区分别进行采样。

# 4.6.2.2 采样深度

（1）表层采样：耕层土壤 $15\mathrm{cm}$ 左右，耕层以下土壤 $15\sim 30\mathrm{cm}$
(2) 深层采样: 深层采样的深度应与污染土壤的采样深度基本一致。

# 4.7 采样量

样品采集后，去除碎石、植物根茎，混匀，用四分法进行缩分，留足小样。采样量根据待测组分所需要的样品量现场确定，样品的采集量必须保证足够用于样品分析。

# 5 制样

5.1 在销毁日本遗弃在华化学武器过程中，土壤样品中若被测组分具有挥发性、半挥发性、热不稳定性、光敏感性或其他特殊性质，经试样处理会造成被测组分损失的样品，一般不做预处理；样品经混合均匀后直接测定。
5.2 对于其他土壤的样品，需要制备时，按照HJ/T166的有关规定执行。

# 6 质量控制

6.1 为保证在允许范围内获得土壤的具有代表性的样品，应在采样、制样的全过程中进行质量控制。
6.2 采样制样前，应设计详细的采样制样方案（计划）；在采样制样过程中，严格按照采样制样方案进行操作。
6.3 对相关采样制样人员进行培训。采样制样人员应熟悉样品的性状、掌握采样技术、懂得安全操作的有关知识和处理方法。采样制样时，应由2人以上在场进行操作。
6.4 采样制样工具、设备所用材质不能和样品有任何反应、不破坏样品代表性、不改变样品组成。
6.5 采样制样过程中要防止待采土壤受到污染、发生变质，要防止土壤样品中待测组分的降解、挥发等造成的样品损失。
6.6 盛样容器应满足以下要求：

——盛样容器材质与样品物质不起作用，没有渗透性；
——具有符合要求的盛样容器的盖、塞，在使用前应洗净至不干扰测定；
——对光敏性土壤样品，盛样容器应是不透光的；
待测组分有特殊要求的，按分析方法中规定要求进行。

6.7 样品经采集后，盛人容器，在容器壁上应随即贴上标签。标签内容包括：

——样品的名称、编号；
——土壤类型；
——监测项目；
——采样时间；
——采样地点；
——采样点位；
——采样深度；
——采样人员。

6.8 样品移交

在采样现场样品必须逐件与样品登记表、样品标签和采样记录进行核对，核对无误后分箱；
——样品运输过程中应防止不同土壤样品之间的交叉污染；
——土壤样品由专人送到指定实验室；
——送样人和接样人双方同时清点核实样品，并在样品交接单上签字确认；
——样品移交单需记录样品名称、样品编号、数量、样品状态的相关描述、移交时间等信息；
——样品移交单由送样人和接样人双方各存一份备查。

6.9 样品经制样后，盛入容器，在容器壁上应随即贴上标签。标签内容包括：

——样品的名称、编号；
——土壤类型；
——监测项目；
——送样时间；
——送样人；
——制样日期；
——制样人员；
——样品保存期。

6.10 填写好、保存好采样制样记录和采样制样报告。

# 7 安全措施

由于销毁日本在华化学武器过程中的土壤为化学毒剂污染，采样制样之前，需制定严格的安全措施。操作时应按规定要求进行操作，确保采样制样的人员和环境安全。

7.1 为了保障采样制样操作的安全，需要对可能实施现场采样及制样所涉及的人员进行必要的安全防护知识培训和防护演练，保证采样制样人员了解样品的危险性及预防措施。
7.2 针对可能出现的紧急情况，制定相应的紧急救助方案，指定出现紧急情况的相关负责人。
7.3 每次作业前审核作业程序，确保作业过程所需防护器材完备。
7.4 现场采样需按规定要求佩戴防护器材，如防护眼镜、防护服等。
7.5 任何一次采样制样均应两人或两人以上进行操作，需有一名为陪伴者，陪伴者应处于能清楚地看到采样点的地方并观察整个采样操作过程。陪伴者应受过专门训练，懂得在紧急情况时该采取什么行动，这些训练要求他首先报告相关负责人，除非在极特殊的情况下，不要单独一人去进行营救。
7.6 采样前及采样后应向相关负责人汇报，尤其要汇报发生的异常事件和情况。
7.7 采样制样过程中，操作人员一旦感到不适时，应立即向相关负责人报告。
7.8 样品运输过程中，采用适当（如有密闭、减压措施等）的运输工具，确保运输沿途人员及环境安全。
7.9 实验室操作时应按规定要求佩戴防护器材，在通风良好的环境中进行操作。

7.10 样品制样及测试后的残渣残液应作妥善的安全处理。
7.11 测试后的残余样品除部分作验证结果所需外，其余样品均应做妥善的安全处理。

# 8 样品保存

# 8.1 样品的暂存和运输

8.1.1 样品采集后，容器采样瓶采用无顶空密闭。并于 $4^{\circ}\mathrm{C}$ 以下保存，尽快分析。
8.1.2 样品运输过程中应保持低温状态，防止因运输引起的温度升高，从而导致待测组分的变化。
8.1.3 待测组分有特殊要求的，按分析方法中规定要求进行暂存和运输。

# 8.2 样品的实验室保存

8.2.1 样品装入容器后应立即密封并贴上样品标签。
8.2.2 对易挥发待测组分，样品应采取无顶空存样，并采取低温保存。
8.2.3 对光敏感的待测组分，样品应装入深色容器中并置于避光处。
8.2.4 对温度敏感的待测组分，样品应保存在规定的温度之下。
8.2.5 样品保存应防止受潮或受灰尘等污染。
8.2.6 待测组分有特殊要求的，按分析方法中规定要求进行保存。
8.2.7 确定样品的保存日期。
8.2.8 样品应在特定场所由专人保管。
8.2.9 超过保存期的样品不允许随意丢弃，按危险废物要求进行处置。
