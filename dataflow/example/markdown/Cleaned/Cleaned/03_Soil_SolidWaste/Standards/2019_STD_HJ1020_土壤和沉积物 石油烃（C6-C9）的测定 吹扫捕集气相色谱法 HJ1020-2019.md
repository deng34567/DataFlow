# 前言

为贯彻《中华人民共和国环境保护法》和《中华人民共和国土壤污染防治法》，保护生态环境，保障人体健康，规范土壤和沉积物中石油烃（ $\mathrm { C } _ { 6 } { \cdot } \mathrm { C } _ { 9 }$ ）的测定方法，制定本标准。

本标准规定了测定土壤和沉积物中石油烃（ $\mathrm { C } _ { 6 } { \mathrm { - } } \mathrm { C } _ { 9 }$ ）的吹扫捕集/气相色谱法。

# 1 适用范围

本标准规定了测定土壤和沉积物中石油烃（ $\mathrm { C } _ { 6 } { \mathrm { - } } \mathrm { C } _ { 9 }$ ）的吹扫捕集/气相色谱法。

本标准适用于土壤和沉积物中石油烃（ $\mathrm { C } _ { 6 } { \cdot } \mathrm { C } _ { 9 }$ ）的测定。

当取样量为 $5 . 0 \ \mathrm { g }$ 时，本标准测定石油烃（ $\mathrm { C } _ { 6 } \mathrm { - C } _ { 9 }$ ）的方法检出限为 $0 . 0 4 \mathrm { m g / k g }$ ，测定下限为 $0 . 1 6 \mathrm { m g / k g }$ 。

# 4 方法原理

土壤和沉积物中的石油烃（ $\mathrm { C } _ { 6 } { \cdot } \mathrm { C } _ { 9 }$ ）经高纯氮气吹扫后吸附于捕集管中，将捕集管加热并以高纯氮气反吹，被热脱附出来的组分经气相色谱柱分离后，用氢火焰离子化检测器（FID）检测，根据保留时间窗定性，外标法定量。

# 7 样品

# 7.1 样品采集和保存

按照HJ/T 166中挥发性有机物的相关要求采集和保存土壤样品，按照HJ 494的相关要求采集水体沉积物样品，按照GB 17378.3的相关要求采集海洋沉积物样品。所有样品均应采集2份平行样品，装入采样瓶（6.8），装满压实并密封。样品采集后， $4 \mathrm { { } ^ { \circ } C }$ 以下避光保存，7 d内完成分析。样品存放区域应无有机物干扰。

# 7.2 全程序空白

在现场加入5 ml同批次的水到样品瓶（6.4）中，盖紧瓶盖，与样品一起运回实验室。

# 7.3 水分的测定

按照HJ 613 测定土壤样品干物质含量，按照GB 17378.5 测定沉积物样品含水率。

# 7.4 试样的制备

# 7.4.1 低浓度试样的制备

称取约 $5 \mathrm { g }$ （精确到 0.01 g）样品于样品瓶（6.4）中，迅速加入 $5 \mathrm { m l }$ 水，盖紧瓶盖，摇匀，待测。

注：当样品浓度大于 $6 . 0 \mathrm { m g / k g }$ 时，可适当减少取样量，但取样量不得低于 $1 . 0 { \mathrm { g } } ,$

# 7.4.2 高浓度试样的制备

当样品浓度大于 $3 0 . 0 \mathrm { m g / k g }$ 时，准确称取 ${ \boldsymbol { 5 } } \mathbf { g } { \sim } 1 0 \ \mathbf { g }$ （精确到 $_ { 0 . 0 1 \ \mathrm { g } } )$ ）样品到样品瓶（6.4）中，迅速加入适量甲醇（5.1）（1 g 样品加入 $1 \mathrm { m l } { \sim } 2 \mathrm { m l }$ 的甲醇），摇匀，静置 $1 \mathrm { h } { \sim } 2 \mathrm { h }$ 。

注：对有明显汽油味的样品可先按高浓度试样进行制备。

# 7.5 实验室空白试样的制备

# 7.5.1 低浓度空白试样的制备

在样品瓶（6.4）中加入 $5 \mathrm { m l }$ 同批次的水，盖紧瓶盖，待测。

# 7.5.2 高浓度空白试样的制备

在样品瓶（6.4）中加入 5 ml同批次的水，同时，加入 $1 0 \mu \mathrm { l } { \sim } 1 0 0 \mu \mathrm { l }$ 同批次的甲醇（5.1）（甲醇加入体积与高浓度试样测定时加入的甲醇提取液相同），盖紧瓶盖，待测。

# 8.2 校准

# 8.2.1 石油烃（ $\left( \mathsf { c } _ { 6 } { - } \mathsf { c } _ { 9 } \right)$ ）保留时间窗的确定

用微量注射器（6.5）分别移取 $1 . 0 \mu \mathrm { l } 2 \cdot$ -甲基戊烷标准溶液（5.5）和正癸烷标准溶液（5.6），加入到事先装有 $5 \mathrm { m l }$ 水的样品瓶（6.4）中，盖紧瓶盖，摇匀。

按照仪器参考条件（8.1）进行保留时间窗的确定。根据 2-甲基戊烷的出峰开始时间确定石油烃（ $\mathrm { C } _ { 6 } { \mathrm { - } } \mathrm { C } _ { 9 }$ ）的开始时间，正癸烷的出峰开始时间确定石油烃（ $\mathrm { C } _ { 6 } \mathrm { - C } _ { 9 }$ ）的结束时间。

# 8.2.2 工作曲线的建立

用微量注射器（6.5）分别移取适量的石油烃（ $\mathrm { C } _ { 6 } – \mathrm { C } _ { 9 }$ ）标准使用液（5.3）快速加入到对应装有 $5 ~ \mathrm { m l }$ 水的6个样品瓶（6.4）中，同时，在上述样品瓶中各加入 $1 . 0 ~ \mu \mathrm { l }$ 替代物标准溶液（5.4），盖紧瓶盖，摇匀。配制成石油烃（ $\mathrm { C } _ { 6 } – \mathrm { C } _ { 9 }$ ）质量分别为 $0 . 0 0 { \mu } \mathrm { g }$ 、 $0 . 5 0 \mu \mathrm { g }$ 、 $1 . 0 0 \mu \mathrm { g }$ 、$5 . 0 0 \mu \mathrm { g }$ 、 $1 0 . 0 \mu \mathrm { g }$ 、 $3 0 . 0 { \mu } \mathrm { g }$ ，替代物质量为 $0 . 5 0 ~ { \mu \mathrm { g } }$ 的标准系列。

按照仪器参考条件（8.1），从低浓度到高浓度依次测定。以浓度为横坐标，以确定的保留时间窗（8.2.1）内总峰面积为纵坐标，建立工作曲线。

注 1：实验用水配制的标准溶液不稳定，需现用现配。

注 2：也可用气密性注射器（6.6）配制标准溶液，分别用微量注射器（6.5）移取适量石油烃（ $\mathrm { C } 6 { - } \mathrm { C } 9$ ）标准使用液（5.3）和替代物标准溶液（5.4）直接加入装有 $5 \mathrm { m l }$ 水的气密性注射器（6.6）中。

# 11 质量保证和质量控制

# 11.1 空白试验

# 11.1.1 实验室空白

每 20 个样品或每批次（少于 20 个样品/批）至少分析一个实验室空白。实验室空白测定结果应低于方法检出限。

# 11.1.2 全程序空白

每批样品至少采集一个全程序空白。全程序空白测定结果应低于方法检出限。

# 11.2 校准

工作曲线的相关系数应 ${ \geqslant } 0 . 9 9 9$ 。每分析 20 个样品或每批次（少于 20 个样品/批）进行一次校准，校准点测定值的相对误差应在 $\pm 1 5 \%$ 以内。

当校准时石油烃（ $C _ { 6 ^ { - } } C _ { 9 }$ ）的保留时间窗与建立工作曲线时石油烃（ $\mathrm { C } _ { 6 } – \mathrm { C } _ { 9 }$ ）的保留时间窗不一致时，需重新按（8.2.1）确定保留时间窗。

# 11.3 平行样

每 20 个样品或每批次（少于 20 个样品/批）应至少分析一个平行样，平行样测定结果的相对偏差应 ${ \leqslant } 2 5 \%$ 。

# 11.4 样品加标

每 20 个样品或每批次（少于 20 个样品/批）应至少分析一个样品加标样，加标样中石油烃（ $\mathrm { C } _ { 6 } { \cdot } \mathrm { C } _ { 9 }$ ）和4-溴氟苯的加标回收率应在 $5 0 \% { \sim } 1 3 0 \%$ 。

# 12 废物处理

实验中产生的废液和废物应分类收集，并做好相应标识，委托有资质的单位进行处理。

# 13 注意事项

13.1 吹扫捕集系统中不得使用聚四氟乙烯以外的塑料或橡胶材料；应保证周边环境的清洁，防止外界污染干扰测定。
13.2 分析高浓度样品后，需分析实验室空白样品，如实验室空白样品的测定结果大于方法检出限，必须用蒸馏水清洗干净，必要时可用 $10 \%$ 的甲醇水溶液进行整个管路清洗。直至实验室空白样品的测定结果低于方法检出限。
13.3 所有玻璃器皿必须严格清洗，并在 $1 3 0 ^ { \circ } \mathrm { C }$ 的烘箱中烘干 $^ { 2 \mathrm { h } }$ ，存放在清洁的环境中。
