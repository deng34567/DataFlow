# 前言

为贯彻《中华人民共和国环境保护法》，保护环境，保障人体健康，规范土壤和沉积物中多环芳烃的测定方法，制定本标准。

本标准规定了土壤和沉积物中多环芳烃类化合物的高效液相色谱法。

本标准为首次发布。

本标准的附录 A 为规范性附录，附录 B 为资料性附录。

本标准由环境保护部科技标准司组织制订。

本标准主要起草单位：河南省环境监测中心。

本标准验证单位：中国地质科学院水文地质环境地质研究所、郑州市环境保护监测中心站、开封市环境监测站、洛阳市环境监测站、新乡市环境监测站。

# 土壤和沉积物多环芳烃的测定高效液相色谱法

警告：部分多环芳烃属于强致癌物，操作时应按规定要求佩戴防护器具，避免接触皮肤和衣服。溶液配制及样品预处理过程应在通风橱内操作。

# 1 适用范围

本标准规定了测定土壤和沉积物中多环芳烃的高效液相色谱法。

本标准适用于土壤和沉积物中 16 种多环芳烃的测定，包括萘、苊烯、苊、芴、菲、蒽、荧蒽、芘、苯并(a)蒽、䓛、苯并(b)荧蒽、苯并(k)荧蒽、苯并(a)芘、二苯并(a, h)蒽、苯并(g,h,i)苝、茚并(1,2,3-c,d)芘。

当取样量为 $1 0 . 0 \ \mathrm { g }$ ，定容体积为 $1 . 0 \mathrm { m l }$ 时，用紫外检测器测定 16 种多环芳烃的方法检出限为 $3 ~ \mu \mathrm { g } / \mathrm { k g } \sim 5 ~ \mu \mathrm { g } / \mathrm { k g }$ ，测定下限为 $1 2 ~ { \mu \mathrm { g } } / { \mu \mathrm { g } } { \sim } 2 0 ~ \mu \mathrm { g } / \mathrm { k g }$ ；用荧光检测器测定 16 种多环芳烃的方法检出限为 $0 . 3 ~ { \mu \mathrm { g } } / { \mu \mathrm { g } } \sim 0 . 5 ~ { \mu \mathrm { g } } / { \mu \mathrm { g } }$ ，测定下限为 $1 . 2 \mu \mathrm { g } / \mathrm { k g } \widetilde { - 2 . 0 \mu g / k g }$ 。详见附录 A。

# 3 方法原理

土壤和沉积物样品中的多环芳烃用合适的萃取方法（索氏提取、加压流体萃取等）提取，根据样品基体干扰情况采取合适的净化方法（硅胶层析柱、硅胶或硅酸镁固相萃取柱等）对萃取液进行净化、浓缩、定容，用配备紫外/荧光检测器的高效液相色谱仪分离检测，以保留时间定性，外标法定量。

# 4 试剂和材料

除非另有说明，分析时均使用符合国家标准的分析纯试剂和实验用水。

4.1 乙腈（ $\mathrm { C H } _ { 3 } \mathrm { C N } )$ ）：HPLC 级。
4.2 正己烷（ $\mathrm { C _ { 6 } H _ { 1 4 } } ,$ ）：HPLC 级。
4.3 二氯甲烷（ $\mathrm { C H } _ { 2 } \mathrm { C l } _ { 2 }$ ）：HPLC 级。
4.4 丙酮（ $\mathrm { C H } _ { 3 } \mathrm { C O C H } _ { 3 } .$ ）：HPLC 级。
4.5 丙酮-正己烷混合溶液： $1 { + } 1$ 。

用丙酮（4.4）和正己烷（4.2）按 1:1 的体积比混合。

4.6 二氯甲烷-正己烷混合溶液： $2 { + } 3$ 。

用二氯甲烷（4.3）和正己烷（4.2）按 2:3 的体积比混合。

4.7 二氯甲烷-正己烷混合溶液： $1 { + } 1$ 。

用二氯甲烷（4.3）和正己烷（4.2）按 1:1的体积比混合。

4.8 多环芳烃标准贮备液： $\rho { = } 1 0 0 \ \mathrm { m g / L } { \sim } 2 0 0 0 \ \mathrm { m g / L }$ 。

购买市售有证标准溶液，于 $4 \mathrm { { } ^ { \circ } C }$ 下冷藏、避光保存，或参照标准溶液证书进行保存。使用时应恢复至室温并摇匀。

4.9 多环芳烃标准使用液： $\rho { = } 1 0 . 0 ~ \mathrm { m g / L } { \sim } 2 0 0 ~ \mathrm { m g / L }$ 。

移取 $1 . 0 \mathrm { m l }$ 多环芳烃标准贮备液（4.8）于 $1 0 ~ \mathrm { m l }$ 棕色容量瓶，用乙腈（4.1）稀释并定容至刻度，摇匀，转移至密实瓶中于 $4 \%$ 下冷藏、避光保存。

4.10 十氟联苯（ $\mathrm { C _ { 1 2 } F _ { 1 0 } } )$ ）：纯度为 $9 9 \%$ 。

替代物，亦可采用其他类似物。

4.11 十氟联苯贮备溶液： $\rho { = } 1 0 0 0 \ \mathrm { m g / L }$ 。

称取十氟联苯（4.10） $_ { 0 . 0 2 5 \mathrm { \textrm { g } } }$ （精确到 $0 . 0 0 1 \ \mathrm { g } )$ ），用乙腈（4.1）溶解并定容至 $2 5 ~ \mathrm { m l }$ 棕色容量瓶，摇匀，转移至密实瓶中于 $4 \%$ 下冷藏、避光保存。或购买市售有证标准溶液。

4.12 十氟联苯使用液： $\rho { = } 4 0 ~ \mu \mathrm { g / m l }$ 。

移取 $1 . 0 \mathrm { m l }$ 十氟联苯贮备溶液（4.11）于 $2 5 ~ \mathrm { m l }$ 棕色容量瓶，用乙腈（4.1）稀释并定容至刻度，摇匀，转移至密实瓶中于 $4 \%$ 下冷藏、避光保存。

4.13 干燥剂：无水硫酸钠（ $\mathrm { N a } _ { 2 } \mathrm { S O } _ { 4 }$ ）或粒状硅藻土

置于马弗炉中 $4 0 0 \mathrm { { ^ \circ C } }$ 烘 $4 \mathrm { { h } }$ ，冷却后置于磨口玻璃瓶中密封保存。

4.14 硅胶：粒径 $7 5 \mu \mathrm { m } { \sim } 1 5 0 \mu \mathrm { m }$ （200 目 $\sim 1 0 0$ 目）。

使用前，应置于平底托盘中，以铝箔松覆， $1 3 0 \mathrm { { } ^ { \circ } C }$ 活化至少 $1 6 \mathrm { { h } }$ 。

4.15 玻璃层析柱：内径约 $2 0 \mathrm { m m }$ ，长 $1 0 \mathrm { c m } { \sim } 2 0 \mathrm { c m }$ ，带聚四氟乙烯活塞。
4.16 硅胶固相萃取柱： $1 0 0 0 \mathrm { m g / 6 m l }$ 。
4.17 硅酸镁固相萃取柱： $1 0 0 0 \mathrm { m g } / 6 \mathrm { m l }$ 。
4.18 石英砂：粒径 $1 5 0 \ \mu \mathrm { m } \sim 8 3 0 \ \mu \mathrm { m }$ （100 目 ${ \sim } 2 0$ 目），使用前须检验，确认无干扰。
4.19 玻璃棉或玻璃纤维滤膜：在马弗炉中 $4 0 0 \mathrm { { ^ \circ C } }$ 烘 $^ { 1 \mathrm { ~ h ~ } }$ ，冷却后置于磨口玻璃瓶中密封保存。
4.20氮气：纯度 ${ \geqslant } 9 9 . 9 9 9 \%$ 。

# 6 样品

# 6.1 样品的采集与保存

按照HJ/T 166的相关要求采集和保存土壤样品，按照 GB 17378.3 的相关要求采集和保

存沉积物样品。样品应于洁净的棕色磨口玻璃瓶中保存，运输过程中应避光、密封、冷藏。如不能及时分析，应于 $4 \%$ 以下冷藏、避光和密封保存，保存时间为 7d。

# 6.2 水分的测定

土壤样品干物质测定按照 HJ 613执行，沉积物样品含水率按照 GB 17378.5执行。

# 6.3 试样的制备

除去样品中的枝棒、叶片、石子等异物，称取样品 $1 0 \ \mathrm { g }$ （精确到 $_ { 0 . 0 1 \ \mathrm { g } } )$ ），加入适量无水硫酸钠（4.13），研磨均化成流沙状。如果使用加压流体提取，则用粒状硅藻土（4.13）脱水。

注1：也可采用冷冻干燥的方式对样品脱水，将冻干后的样品研磨、过筛，均化处理成约 $1 \mathrm { m m }$ 的颗粒。

# 6.3.1 提取

将制备好的试样放入玻璃套管或纸质套管内，加入 $5 0 . 0 \mu \mathrm { l }$ 十氟联苯使用液（4.12），将套管放入索氏提取器中。加入 $1 0 0 ~ \mathrm { m l }$ 丙酮-正己烷混合溶液（4.5），以每小时不小于 4 次的回流速度提取 $1 6 \mathrm { h } { \sim } 1 8 \mathrm { h }$ 。

注2：若通过验证并达到本标准质量控制要求，亦可采用其他提取方式。

注3：套管规格根据样品量而定。

# 6.3.2 过滤和脱水

在玻璃漏斗上垫一层玻璃棉或玻璃纤维滤膜（4.19），加入约 5 g 无水硫酸钠（4.13），将提取液过滤到浓缩器皿中。用适量丙酮-正己烷混合溶液（4.5）洗涤提取容器3 次，再用适量丙酮-正己烷混合溶液（4.5）冲洗漏斗，洗液并入浓缩器皿。

# 6.3.3 浓缩

氮吹浓缩法：开启氮气至溶剂表面有气流波动（避免形成气涡），用正己烷（4.2）多次洗涤氮吹过程中已经露出的浓缩器壁，将过滤和脱水后的提取液浓缩至约 $1 ~ \mathrm { m l }$ 。如不需净化，加入约 3 ml 乙腈（4.1），再浓缩至约 $1 ~ \mathrm { m l }$ ，将溶剂完全转化为乙腈。如需净化，加入约 $5 \mathrm { m l }$ 正己烷并浓缩至约 1 ml，重复此浓缩过程 3次，将溶剂完全转化为正己烷，再浓缩至约 $1 ~ \mathrm { m l }$ ，待净化。

注4：也可采用旋转蒸发浓缩或其他浓缩方式。

# 6.3.4 净化

# 6.3.4.1 硅胶层析柱净化

# （1）硅胶柱制备

在玻璃层析柱（4.15）的底部加入玻璃棉（4.19），加入 $1 0 \mathrm { m m }$ 厚的无水硫酸钠（4.13），用少量二氯甲烷（4.3）进行冲洗。玻璃层析柱上置一玻璃漏斗，加入二氯甲烷（4.3）直至充满层析柱，漏斗内存留部分二氯甲烷，称取约 $1 0 \ \mathrm { g }$ 硅胶（4.14）经漏斗加入层析柱，以玻璃棒轻敲层析柱，除去气泡，使硅胶填实。放出二氯甲烷，在层析柱上部加入 $1 0 \mathrm { m m }$ 厚的无水硫酸钠（4.13）。层析柱示意图见图 1。

# （2）净化

用 $4 0 ~ \mathrm { m l }$ 正己烷（4.2）预淋洗层析柱，淋洗速度控制在 $2 \mathrm { m l / m i n }$ ，在顶端无水硫酸钠暴露于空气之前，关闭层析柱底端聚四氟乙烯活塞，弃去流出液。将浓缩后的约 1 ml 提取液（6.3.3）移入层析柱，用 $2 \mathrm { m l }$ 正己烷（4.2）分 3 次洗涤浓缩器皿，洗液全部移入层析柱，在顶端无水硫酸钠曝露于空气之前，加入 $2 5 ~ \mathrm { m l }$ 正己烷（4.2）继续淋洗，弃去流出液。用$2 5 ~ \mathrm { m l }$ 二氯甲烷-正己烷混合溶液（4.6）洗脱，洗脱液收集于浓缩器皿中，用氮吹浓缩法（或其他浓缩方式）将洗脱液浓缩至约 $1 ~ \mathrm { m l }$ ，加入约 $3 ~ \mathrm { m l }$ 乙腈（4.1），再浓缩至 1 ml 以下，将溶剂完全转换为乙腈，并准确定容至 $1 . 0 \mathrm { m l }$ 待测。净化后的待测试样如不能及时分析，应于$4 \mathrm { { } ^ { \circ } C }$ 下冷藏、避光、密封保存，30d 内完成分析。

# 6.3.4.2 固相萃取柱净化（填料为硅胶或硅酸镁）

用固相萃取柱（4.16或 4.17）作为净化柱，将其固定在固相萃取装置（5.5）上。用 $4 \mathrm { m l }$ 二氯甲烷（4.3）冲洗净化柱，再用 $1 0 ~ \mathrm { m l }$ 正己烷（4.2）平衡净化柱，待柱充满后关闭流速控制阀浸润 $5 \mathrm { { m i n } }$ ，打开控制阀，弃去流出液。在溶剂流干之前，将浓缩后的约 1 ml 提取液（6.3.3）移入柱内，用 $3 ~ \mathrm { m l }$ 正己烷（4.2）分 3次洗涤浓缩器皿，洗液全部移入柱内，用 10ml 二氯甲烷-正己烷混合溶液（4.7）进行洗脱，待洗脱液浸满净化柱后关闭流速控制阀，浸润 $5 \mathrm { { m i n } }$ ，再打开控制阀，接收洗脱液至完全流出。用氮吹浓缩法（或其他浓缩方式）将洗脱液浓缩至约 $1 ~ \mathrm { m l }$ ，加入约 $3 ~ \mathrm { m l }$ 乙腈（4.1），再浓缩至 1 ml 以下，将溶剂完全转换为乙腈，并准确定容至 $1 . 0 \mathrm { m l }$ 待测。净化后的待测试样如不能及时分析，应于 $4 \mathrm { { } ^ { \circ } C }$ 下冷藏、避光、密封保存，30d 内完成分析。

# 7.2 校准

# 7.2.1 校准曲线的绘制

分别量取适量的多环芳烃标准使用液（4.9），用乙腈（4.1）稀释，制备至少 5个浓度点

的标准系列，多环芳烃的质量浓度分别为 $0 . 0 4 ~ \mu \mathrm { g / m l }$ 、 $0 . 1 0 ~ { \mu \mathrm { g / m l } }$ 、 $0 . 5 0 ~ \mu \mathrm { g / m l }$ 、 $1 . 0 0 ~ \mu \mathrm { g / m l }$ 和 $5 . 0 0 ~ \mu \mathrm { g / m l }$ （此为参考浓度），同时取 $5 0 . 0 \mu \mathrm { l }$ 十氟联苯使用液（4.12），加入至标准系列中任一浓度点，十氟联苯的质量浓度为 $2 . 0 0 \mu \mathrm { g / m l }$ ，贮存于棕色进样瓶中，待测。

由低浓度到高浓度依次对标准系列溶液进样，以标准系列溶液中目标组分浓度为横坐标，以其对应的峰面积（峰高）为纵坐标，建立校准曲线。校准曲线的相关系数 ${ \geqslant } 0 . 9 9 5$ ，否则重新绘制校准曲线。

# 8.1 目标化合物的定性分析

以目标化合物的保留时间定性，必要时可采用标准样品添加法、不同波长下的吸收比、紫外谱图扫描等方法辅助定性。

# 8.2 结果计算

8.2.1 土壤样品中多环芳烃的含量（ $\bf { \nabla } \cdot \lVert u g / k g \rVert$ ），按照公式（1）进行计算。

$$
\omega_ {i} = \frac {\rho_ {i} \times V}{m \times W _ {d m}} \tag {1}
$$

式中：

$\omega _ { i }$ @ 样品中组分i的含量， $\mu \mathrm { g / k g }$ ；

$\rho _ { i }$ 由标准曲线计算所得组分i的的浓度， $\mu \mathrm { g / m l }$ ；

$V$ 定容体积，ml；

m 样品量（湿重），kg；

Wdm $W _ { d m }$ 土壤样品干物质含量， $\%$ 。

8.2.2 沉积物样品中多环芳烃的含量（ $\bf { \nabla } \cdot \lVert u g / k g \rVert$ ），按照公式（2）进行计算。

$$
\omega_ {i} = \frac {\rho_ {i} \times V}{m \times (1 - W)} \tag {2}
$$

式中：

$\omega _ { i }$ 样品中组分i的含量， $\mu \mathrm { g / k g }$ ；

$\rho _ { i }$ 由校准曲线计算所得组分i的浓度， $\mu \mathrm { g / m l }$ ；

V ——定容体积，ml；

m 样品量（湿重），kg；

W —沉积物样品含水率， $\%$ 。

8.2.3 十氟联苯的回收率 $( \% )$ ），按照公式（3）进行计算。

$$
P = \frac {A _ {1} \times \rho_ {2} \times V _ {2}}{A _ {2} \times \rho_ {1} \times V _ {1} \times 10 ^ {- 3}} \times 100 \% \tag{3}
$$

式中：

$P$ 十氟联苯的回收率， $\%$ ；

$A _ { 1 }$ —试样中十氟联苯的峰面积；

$A _ { 2 }$ —标准系列中十氟联苯的峰面积；

$\rho _ { \mathrm { 1 } }$ 十氟联苯使用液的质量浓度， $4 0 ~ \mu \mathrm { g / m l }$ ；

$\rho _ { 2 }$ 标准系列中十氟联苯的质量浓度， $2 \mu \mathrm { g } / \mathrm { m l }$

$V _ { 1 }$ —试样中加入十氟联苯使用液的体积， $5 0 . 0 \mu \mathrm { l }$ ；

$V _ { 2 }$ 试样定容体积，ml。

# 8.3 结果表示

当测定结果大于或等于 $1 0 ~ \mu \mathrm { g / k g }$ 时，保留三位有效数字；当测定结果小于 $1 0 ~ \mu \mathrm { g / k g }$ 时，保留至小数点后 1 位。苊烯保留整数位，最多保留三位有效数字。

# 10 质量保证和质量控制

# 10.1 空白分析

每次分析至少做一个实验室空白实验和一个全程序空白，以检查可能存在的干扰，其目标化合物的测定值不得高于方法的检出限。

# 10.2 平行样测定

每 20个样品或每批次（少于 20 个样品/批）须分析一个平行样。平行双样测定结果的相对偏差应 $\text{‰}$ 。

# 10.3 基体加标

每20个样品或每批次（少于20个样品/批）须做1个基体加标样，各组分的回收率在 $50 \%$ ～$120 \%$ 之间。十氟联苯回收率在 $6 0 \% \sim 1 2 0 \%$ 之间。

# 10.4 校准

# 10.4.1 初始校准

初次使用仪器，或在仪器维修、更换色谱柱或连续校准不合格时，须重新绘制校准曲线，进行初始校准。

# 10.4.2 连续校准

每 20 个样品或每批次（少于 20 个样品/批）须用校准曲线的中间浓度点进行 1 次连续校准。连续校准的相对误差应 $\text{‰}$ ，否则应查找原因，或重新绘制校准曲线。按照公式（4）计算 $C _ { c }$ 与校准点 $C _ { i }$ 的相对误差（ $D$ ）：

$$
D = \frac {C _ {c} - C _ {i}}{C _ {i}} \times 100 \% \tag{4}
$$

式中：

D $C _ { c }$ 与校准点 $C _ { i }$ 的相对误差， $\%$ ；

$C _ { i }$ 校准点的质量浓度；

$C _ { c }$ 测定该校准点的质量浓度。

# 11 废物处理

实验中产生的所有废液和废物（包括检测后的残液）应分类收集，置于密闭容器中集中保管，粘贴明显标识，委托具有资质的单位处置。
