# 土壤 pH值的测定电位法

# 1 适用范围

本标准规定了测定土壤 pH 值的电位法。

本标准适用于土壤 pH 值的测定。

# 2 规范性引用文件

本标准引用了下列文件或其中的条款。凡是不注日期的引用文件，其有效版本适用于本标准。

HJ/T166 土壤环境监测技术规范

# 3 方法原理

以水为浸提剂，水土比为 2.5:1，将指示电极和参比电极（或 pH 复合电极）浸入土壤悬浊液时，构成一原电池，在一定的温度下，其电动势与悬浊液的 pH 值有关，通过测定原电池的电动势即可得到土壤的 pH 值。

# 5 仪器和设备

5.1 pH 计：精度为 0.01 个 $\mathsf { p H }$ 单位，具有温度补偿功能。
5.2 电极：玻璃电极和饱和甘汞电极，或 pH 复合电极。
5.3 磁力搅拌器或水平振荡器：具有温控功能。
5.4 土壤筛：孔径 $2 \mathrm { m m }$ （10 目）。
5.5 一般实验室常用仪器和设备。

# 6 样品

# 6.1 样品采集和保存

按照 HJ/T 166的相关规定进行土壤样品的采集和保存。

# 6.2 样品的制备

按照 HJ/T 166的相关规定进行土壤样品的制备，包括样品的风干、缩分、粉碎和过筛（5.4）。制备后的样品不立刻测定时，应密封保存，以免受大气中氨和酸性气体的影响，同时避免日晒、高温、潮湿的影响。

# 6.3 试样的制备

称取 $1 0 . 0 \ \mathrm { g }$ 土壤样品置于 $5 0 \mathrm { m l }$ 的高型烧杯或其他适宜的容器中，加入 $2 5 ~ \mathrm { m l }$ 水（4.1）。将容器用封口膜或保鲜膜密封后，用磁力搅拌器（5.3）剧烈搅拌 $2 \mathrm { m i n }$ 或用水平振荡器（5.3）剧烈振荡 $2 \mathrm { m i n }$ 。静置 $3 0 \mathrm { { m i n } }$ ，在 1 h内完成测定。

# 7 分析步骤

# 7.1 校准

至少使用两种 pH 标准缓冲溶液对 pH计进行校准。先用 $\mathsf { p H } 6 . 8 6$ （ $2 5 \mathrm { { ^ \circ C } }$ ）标准缓冲溶液（4.7），再用 $\mathrm { p H } 4 . 0 1$ （ $2 5 \mathrm { { ^ \circ C } }$ ）标准缓冲溶液（4.6）或 $\mathsf { p H 9 . l 8 }$ （ $2 5 \mathrm { { ^ \circ C } }$ ）标准缓冲溶液（4.8）校准。校准步骤如下：

a）将盛有标准缓冲溶液并内置搅拌子的烧杯置于磁力搅拌器（5.3）上，开启磁力搅拌器。
b）控制标准缓冲溶液的温度在（ $2 5 \pm 1$ ） $\mathrm { { } ^ { \circ } C }$ ，用温度计测量标准缓冲溶液的温度，并将2

pH 计的温度补偿旋钮调节到该温度上。有自动温度补偿功能的仪器，可省略此步骤。

c）将电极插入标准缓冲溶液中，待读数稳定后，调节仪器示值与标准缓冲溶液的 pH值一致。重复步骤 a 和 b，用另一种标准缓冲溶液校准 pH 计，仪器示值与该标准缓冲溶液的 pH 值之差应 ${ \leqslant } 0 . 0 2$ 个 pH单位。否则应重新校准。

注：用于校准 pH的两种标准缓冲溶液，其中一种标准缓冲溶液的 pH 值应与土壤 pH值相差不超过 2个pH单位。若超出范围，可选择其他 pH 标准缓冲溶液，参见附录A。

# 7.2 测定

控制试样（6.3）的温度为（ $2 5 \pm 1$ ）℃，与标准缓冲溶液的温度之差不应超过 $2 \mathrm { { } ^ { \circ } C }$ 。将电极插入试样的悬浊液，电极探头浸入液面下悬浊液垂直深度的 1/3~2/3 处，轻轻摇动试样。待读数稳定后，记录 pH 值。每个试样测完后，立刻用水冲洗电极，并用滤纸将电极外部水吸干，再测定下一个试样。

# 8 结果表示

测定结果保留至小数点后 2 位。当读数小于 2.00 或大于 12.00 时，结果分别表示为 pH$< 2 . 0 0$ 或 $\mathrm { p H } > 1 2 . 0 0$ 。

# 9 精密度

六家实验室分别对湖南黄壤和贵州紫色土统一样品进行 6 次重复测定：湖南黄壤 pH 值平均值为 4.62，实验室内相对标准偏差为 $0 . 1 2 \% { \sim } 1 . 5 \%$ ，实验室间相对标准偏差为 $2 . 7 \%$ ，重复性限为 0.10，再现性限为 0.37；贵州紫色土 $\mathsf { p H }$ 值平均值为 5.83，实验室内相对标准偏差为 $0 . 3 0 \% { \sim } 1 . 5 \%$ ，实验室间相对标准偏差为 $2 . 8 \%$ ，重复性限为 0.11，再现性限为 0.46。

# 10 质量保证和质量控制

每批样品应至少测定 $1 0 \%$ 的平行双样，每批少于 10个样品时，应至少测定 1 组平行双样。两次平行测定结果的允许差值为 0.3 个 $\mathsf { p H }$ 单位。

# 11 注意事项

11.1 pH 计应参照仪器说明书使用和维护。
11.2 电极应参照电极说明书使用和维护。
11.3 温度对土壤 pH 值的测定具有一定影响，在测定时，应按要求控制温度。
11.4 在测定时，将电极插入试样的悬浊液，应注意去除电极表面气泡。
