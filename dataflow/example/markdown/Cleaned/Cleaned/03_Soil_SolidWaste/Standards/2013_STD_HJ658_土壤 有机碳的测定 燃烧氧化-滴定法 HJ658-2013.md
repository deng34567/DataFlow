# 土壤有机碳的测定燃烧氧化-滴定法

# 1 适用范围

本标准规定了测定土壤中有机碳的燃烧氧化-滴定法。

本标准适用于土壤中有机碳的测定，不适用于油泥污染土壤中有机碳的测定。

当样品量为 $0 . 5 0 \mathrm { g }$ 时，本标准的方法检出限为 $0 . 0 0 4 \%$ ，测定下限为 $0 . 0 1 6 \%$ ，测定上限为 $4 . 0 0 \%$ 。样品中有机碳含量较高时，可减少取样量，但最低不能低于 $0 . 0 5 0 \ \mathrm { g }$ 。

# 2 规范性引用文件

本标准内容引用了下列文件或其中的条款。凡是不注明日期的引用文件，其有效版本适用于本标准。

HJ/T 166 土壤环境监测技术规范

HJ/T 613 土壤干物质和水分的测定重量法

# 3 方法原理

风干土壤样品在燃烧炉中加热至 $9 0 0 \mathrm { { } ^ { \circ } C }$ 以上，样品中有机碳被氧化为二氧化碳，产生的二氧化碳用过量的氢氧化钡溶液吸收生成碳酸钡沉淀，反应后剩余的氢氧化钡用草酸标准溶液滴定，由空白滴定和样品滴定消耗的草酸标准溶液的体积差计算二氧化碳产生，根据二氧化碳产生量计算土壤中的有机碳含量。

# 4 干扰和消除

当样品加热至 $2 0 0 ~ \mathrm { { } ^ { \circ } C }$ 以上时，所有碳酸盐均完全分解，产生二氧化碳，对本方法的测定产生正干扰，可通过加入适量盐酸去除。

空气中的二氧化碳会对测定产生相当于 $0 . 2 \%$ 有机碳的正干扰，通过扣除空白去除。

# 5 试剂和材料

除非另有说明，分析时均使用符合国家标准的分析纯试剂。

5.1 无二氧化碳水：临用现制，电导率 ${ \leqslant } 0 . 2 ~ \mathrm { m S / m }$ （ $2 5 \mathrm { { ^ \circ C } }$ ）。
5.2 浓盐酸： $\rho$ $( \mathrm { H C l } ) = 1 . 1 9 \ : \mathrm { g / m l } .$
5.3 正丁醇： $\varphi$ $( \mathrm { C _ { 4 } H _ { 9 } O H } ) ~ \geq 9 9 . 0 \%$ $\mathrm { C _ { 4 } H _ { 9 } O H }$ 。
5.4 乙醇： $\varphi$ （ $( \mathrm { C } _ { 2 } \mathrm { H } _ { 5 } \mathrm { O H } ) \ { = } 9 5 \ \% \ $ 。
5.5 草酸（ $\mathrm { \cdot H } _ { 2 } \mathrm { C } _ { 2 } \mathrm { O } _ { 4 } { \cdot } 2 \mathrm { H } _ { 2 } \mathrm { O } )$ ）：基准试剂。
5.6 盐酸溶液： $c$ $\mathrm { ( H C l ) } = 4 \ : \mathrm { m o l / L }$

量取 $3 4 0 ~ \mathrm { m l }$ 浓盐酸（5.2），边搅拌边缓慢倒入 $5 0 0 \mathrm { m l }$ 水（5.1）中，用水（5.1）稀释至$1 0 0 0 \mathrm { m l }$ ，混匀。

5.7 氢氧化钡（ $\mathrm { \cdot B a ( O H ) } _ { 2 } { \cdot } 8 \mathrm { H } _ { 2 } \mathrm { O } )$ 。
5.8 氯化钡（ ${ \mathrm { B a C l } } _ { 2 } { \cdot } 2 { \mathrm { H } } _ { 2 } \mathrm { O } $ ）。
5.9 氢氧化钡吸收液Ⅰ： $\rho \left[ \mathrm { B a ( O H ) } _ { 2 } \right] { = } 1 . 4 0 ~ \mathrm { g / L }$

称取 $1 . 4 0 \ \mathrm { g }$ 氢氧化钡（5.7）和 $0 . 0 8 \ \mathrm { g }$ 氯化钡（5.8）溶于 $8 0 0 \mathrm { m l }$ 水（5.1）中，加入 3 ml正丁醇（5.3），用水（5.1）稀释至 $1 0 0 0 \mathrm { m l }$ ，混匀。

5.10 氢氧化钡吸收液Ⅱ： $\rho [ \mathrm { B a ( O H ) _ { 2 } } ] { = } 2 . 8 0 \mathrm { g / L }$

注：上述两种吸收液配制后，密封保存，放置 1 d 使之沉淀。

5.11 草酸标准溶液： $\rho$ （ $\mathrm { \langle H _ { 2 } C _ { 2 } O _ { 4 } 2 H _ { 2 } O \rangle = 0 . 5 6 3 7 ~ g / L }$

称取 $0 . 5 6 3 7 \mathrm { ~ g ~ }$ 草酸（5.5）溶于适量水（5.1），移至 $1 0 0 0 \mathrm { ~ m l }$ 容量瓶中，用水（5.1）稀释至标线，混匀。 $1 ~ \mathrm { m l }$ 此溶液相当于标准状态下（ $1 0 1 . 3 2 5 \mathrm { k P a }$ ，273.15 K） $0 . 1 \mathrm { m l }$ 二氧化碳。临用现配。

5.12 酚酞指示剂

称取 $0 . 5 \ \mathrm { g }$ 酚酞溶于 $5 0 \mathrm { m l }$ 乙醇（5.4）中，再加入 $5 0 \mathrm { m l }$ 水（5.1），摇匀。

# 6 仪器和设备

6.1 管式炉：采用硅碳管作为加热体，能够加热样品至 $9 0 0 \mathrm { { ^ \circ C } }$ 以上，温度可调节，精度 $1 ~ \mathrm { { ^ \circ C } }$ ；高温区长度大于 $9 0 \mathrm { m m }$ 。
6.2 玻板吸收瓶：吸收瓶容积为 $4 5 0 ~ \mathrm { m l }$ ，玻板直径大于等于 $1 0 \mathrm { m m }$ 。
6.3 磁力搅拌器：搅拌速度约为 $5 0 0 ~ \mathrm { r / m i n }$ ，且连续可调。
6.4 陶瓷舟。
6.5 抽气泵。
6.6 气体流量计：浮子流量计，配有针型阀，流量范围为 $0 \sim 1 . 0 \mathrm { L / m i n }$ 。
6.7 天平：精度为 $0 . 1 \mathrm { m g }$
6.8 烘箱：温度调节范围为 $0 \sim 2 5 0 ~ \mathrm { ~ \textdegree ~ }$ 。
6.9 土壤筛： $2 \mathrm { m m }$ （10 目）、 $0 . 0 9 7 \mathrm { m m }$ （160 目），不锈钢材质。
6.10 酸式滴定管： $5 0 . 0 0 \mathrm { m l }$ 。
6.11 一般实验室常用仪器和设备。

# 7 样品

7.1 样品的采集和保存

土壤样品的采集和保存参照 HJ/T 166 的相关规定执行。

7.2 试样的制备

将土壤样品置于风干盘中，平摊成 $2 \sim 3 ~ \mathrm { c m }$ 厚的薄层，先剔除植物、昆虫、石块等残体，用铁棰或瓷质研磨棒压碎土块，每天翻动几次，自然风干。

充分混匀风干土壤，采用四分法，取其两份，一份留存，一份研磨至全部过 $2 \mathrm { m m }$ （10目）土壤筛。取 $1 0 \sim 2 0 ~ \mathrm { g }$ 过筛后的土壤样品，研磨至全部过 $0 . 0 9 7 \mathrm { m m }$ （160 目）土壤筛，装入棕色具塞玻璃瓶中，待测。

# 7.3 干物质含量的测定

准确称取过 $2 \mathrm { m m } ( 1 0$ 目）筛后的风干土壤样品，参照 HJ/T 613测定土壤的干物质含量。

# 8 分析步骤

# 8.1 试料的制备

称取适量试样（7.2），精确到 $_ { 0 . 0 0 1 \mathrm { ~ g ~ } }$ ，置于陶瓷舟（6.4）中，并缓慢滴加盐酸溶液（5.6）至试样无气泡冒出。充分混合，静置 $4 \mathrm { { h } }$ 后，于 $6 0 \sim 7 0 ~ \mathrm { ^ { \circ } C }$ 下烘干 $1 6 \mathrm { { h } }$ ，待测。

注1：可根据称取试样的质量和碳酸盐的含量确定盐酸溶液浓度及加入量，若土壤试样中仅含碳酸盐，可在每 1 g 试样中加 5 ml 盐酸溶液（5.6）。

# 8.2 气密性检查

连接管式炉燃烧和吸收装置，塞好玻板吸收瓶（6.2），打开抽气泵（6.5），关闭气体流量计（6.6）前阀门。若流量计流量归零，则设备气密性良好。

# 8.3 测定

向玻板吸收瓶（6.2）中准确加入 $2 0 0 \mathrm { m l }$ 氢氧化钡吸收液Ⅰ（5.9）的上清液，塞紧吸收瓶，将上述装有试料的陶瓷舟（6.4）放入管式炉中，调节管式炉（6.1）炉温至 $9 0 0 { \sim } 1 0 0 0 \ \mathrm { ^ { \circ } C }$ ，打开抽气泵（6.5），调节抽气流量为 $0 . 5 \mathrm { \ L } / \mathrm { m i n }$ ，调节磁力搅拌器（6.3）转速，使气泡分布均匀。反应时间为 $6 0 0 { \pm } 1 0 \mathrm { s }$ ，反应结束后，倾出所有吸收液于 $2 5 0 ~ \mathrm { m l }$ 具塞玻璃瓶中，加塞密闭静置 $3 { \sim } 4 \mathrm { ~ h ~ }$ ，使碳酸钡沉淀完全，准确量取 $5 0 ~ \mathrm { m l }$ 上清液于 $2 5 0 ~ \mathrm { m l }$ 锥形瓶中，加入 $4 \sim 5$ 滴酚酞指示剂（5.12），用草酸标准溶液（5.11）滴定至溶液由红色变为无色为终点，记录所消耗的草酸标准溶液体积 $\mathrm { V } _ { 1 }$ 。

注2：当样品中有机碳含量较高时，可用氢氧化钡溶液Ⅱ（5.10）作为样品吸收液。

# 8.4 空白试验

陶瓷舟（6.4）中不加入试料，按照步骤 8.3 测定，记录所消耗的草酸标准溶液体积 $\mathrm { V } _ { 0 }$ 。

# 9 结果计算与表示

# 9.1 结果计算

土壤中的有机碳含量 $\omega _ { o c }$ （以碳计，质量分数，%），按照公式（1）和（2）进行计算。

$$
m _ {1} = m \times \frac {w _ {d m}}{1 0 0} \tag {1}
$$

$$
\omega_ {O C} = \frac {\left(V _ {0} - V _ {1}\right) \times C \times 1 2 \times 2 0 0}{1 2 6 \times 5 0 \times 1 0 0 0 \times m _ {1}} \times 1 0 0 \tag {2}
$$

式中：

$m _ { 1 }$ —试样中干物质的质量，g；

m—试样取样量，g；

wdm $w _ { d m }$ 土壤样品的干物质含量（质量分数）， $\%$ ；

Q0C $\omega _ { o c }$ 土壤样品中有机碳的含量（以碳计，质量分数)， $\%$

$V _ { 0 }$ ——滴定空白消耗草酸标准溶液体积，ml；

$V _ { 1 }$ ——滴定试料消耗草酸标准溶液体积，ml；

$C$ —草酸标准溶液质量浓度， $\mathrm { g / L }$ ；

12——碳元素的摩尔质量， $\mathrm { { g } / \mathrm { { m o l } } }$

200——氢氧化钡吸收液体积，ml；

126——草酸的摩尔质量， $\mathrm { { g } / \mathrm { { m o l } } }$ ；

50——用于滴定的氢氧化钡吸收液体积，ml。

# 9.2 结果表示

当测定结果 $< 1 . 0 0 \%$ 时，保留到小数点后三位；当测定结果 $\geq 1 . 0 0 \%$ 时，保留三位有效数字。

# 10 精密度和准确度

# 10.1 精密度

6家实验室分别对有机碳含量为 $0 . 5 4 \%$ 的有证标准物质和 $0 . 5 1 \%$ 的实际样品进行了测定，实验室内相对标准偏差分别为 $3 . 8 \% { \sim } 1 0 . 6 \%$ ， $1 . 9 \% \sim 1 2 . 6 \%$ ；实验室间相对标准偏差分别为$2 . 7 \%$ ， $3 . 2 \%$ ；重复性限分别为 $0 . 1 1 \%$ ， $0 . 1 1 \%$ ；再现性限分别为 $0 . 1 1 \%$ ， $0 . 1 1 \%$ 。

# 10.2 准确度

6 家实验室分别对有机碳含量为 $0 . 5 4 \%$ 的有证标准样品进行了测定，相对误差为$0 \% { \sim } 3 . 7 \%$ ；相对误差最终值为 $2 . 2 \% \pm 3 . 6 \%$ 。

# 11 质量保证和质量控制

11.1 每批样品应至少做 $10 \%$ 的平行样品测定，样品数不足 10 个时，每批样品应至少做一个平行样品测定。当样品有机碳含量 $\leqslant 1 . 0 0 \%$ 时，平行样测定结果的差值应在 $\pm 0 . 1 0 \%$ 之内；当样品有机碳含量 $> 1 . 0 0 \%$ 时，平行样测定结果的相对偏差 $\leqslant 1 0 . 0 \%$ 。

11.2 每批样品测定时，应分析一个有证标准样品，其测定值应在保证值范围内。

# 12 废弃物的处理

实验室产生的废酸、废碱等废液及固体废物应定期收集，委托有资质的单位进行处置。

# 13 注意事项

13.1 二氧化碳的吸收效率受气泡的大小和分布情况影响较大，因此要求玻板吸收瓶的玻板孔隙较小，使用前应检查玻璃砂芯的质量。方法如下：以 $0 . 5 \mathrm { L } / \mathrm { m i n }$ 的流量抽气，气泡路径（泡沫高度）为 $5 0 { \pm } 5 \ \mathrm { m m }$ ，玻板阻力为 $4 . 7 { \pm } 0 . 7 \ \mathrm { k P a }$ ，且气泡均匀，无特大气泡。磁力搅拌器搅拌速度要合适，以使气泡在溶液中分布均匀。

13.2 陶瓷舟在初次使用前，应将其放入小烧杯中，向烧杯中加入盐酸溶液（5.6），使之浸没完全，片刻后取出，沥干，于 $6 0 { \sim } 7 0 \ \mathrm { ~ } ^ { \circ } \mathrm { C }$ 下烘干 $1 6 \mathrm { { h } }$ 后，再将陶瓷舟放入管式炉中，调节

炉温至 900~1000 ℃，灼烧 $1 0 \mathrm { m i n }$ ，以去除陶瓷舟材质对测定结果的影响。
