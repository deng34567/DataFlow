# 中华人民共和国国家标准

# 固体废物总铬的测定

# 二苯碳酰二肼分光光度法

GB/T15555.5-1995

Solid waste-Determination of total chromium

1,5-Diphenylcarbohydrazide spectrophotometric method

# 1 主题内容与适用范围

1.1 本标准规定了固体废物浸出液中总铬的测定，用二苯碳酰二肼分光光度法。
1.2 本标准适用于固体废物浸出液中总铬的测定。
1.2.1 测定范围

试液为 $50~\mathrm{mL}$ ，使用 $30~\mathrm{mm}$ 光程比色皿，方法的最小检出量为 $0.2\mu \mathrm{g}$ ，最低检出浓度为 $0.004\mathrm{mg / L}$ 。使用 $10\mathrm{mm}$ 光程比色皿，测定上限浓度为 $1.0\mathrm{mg / L}$ 。

1.2.2 干扰

试液颜色：混浊，或者有氧化性、还原性物质及有机物等均干扰测定。铁含量大于 $1.0\mathrm{mg / L}$ 也干扰测定。钼、汞于显色剂生成有色络合物有干扰，但是在方法的显色酸度下，反应不灵敏。钒浓度大于 $4.0\mathrm{mg / L}$ ，也干扰测定，但是，显色 $10\mathrm{min}$ 后，可自行退色。

# 2 原理

在酸性溶液中，试料中的三价铬被高锰酸钾氧化成六价铬，六价铬与二苯碳酰二肼反应生成紫红色络合物，于 $540\mathrm{nm}$ 处测吸光度。过量的高锰酸钾用亚硝酸钠分解，再用脲素分解过量的亚硝酸钠。

# 3 试剂

本标准所有试剂除另有说明外，均用符合国家标准或专业标准的分析纯试剂和蒸馏水或同等纯度的水。

3.1 硝酸 $\left(\mathrm{HNO}_{3}\right), \rho = 1.42 \mathrm{~g} / \mathrm{mL}$ 。
3.2 硫酸 $(\mathrm{H_2SO_4})$ ， $\rho = 1.84\mathrm{g / mL}$
3.3 氯仿 $\mathrm{(CHCl_3)}$
3.4 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O}),\rho = 0.90\mathrm{g / mL}$
3.5 铜铁试剂 $\left[\mathrm{C}_{6} \mathrm{H}_{5} \mathrm{~N}(\mathrm{NO}) \mathrm{ONH}_{4}\right]$ 。
3.6 氨水， $1 + 1$ 。将氨水(3.4)与等体积水混合。
3.7 铜铁试剂溶液， $5\%$ ：

称取铜铁试剂 $5\mathrm{g}$ ，溶于冰水中，用水稀至 $100\mathrm{mL}$ 。用时现配。

3.8 高锰酸钾溶液， $40\mathrm{g / L}$

称取高锰酸钾 $\mathrm{(KMnO_4)}4\mathrm{g}$ ，在加热下溶于少量水中。待溶解后，用水稀释至 $100~\mathrm{mL}$ 。

3.13 铬标准溶液， $5.00\mu \mathrm{g / mL}$

3.14 显色剂I：

称取二苯碳酰二肼 $\mathrm{(C_{13}H_{14}N_4O)}$ 0.2g，溶于 $50~\mathrm{mL}$ 丙酮 $\mathrm{(C_3H_6O)}$ 中，加水稀释至 $100~\mathrm{mL}$ ，摇匀于棕色瓶中，在低温下保存。

3.15 硫酸溶液， $1 + 1$

将硫酸 $\mathrm{(H_2SO_4,}\rho = 1.84\mathrm{g / mL})$ 缓慢加到同体积的水中，边加边搅，待冷却后使用。

3.16 磷酸溶液， $1 + 1$

将磷酸 $\mathrm{(H_3PO_4}$ $\rho = 1.69\mathrm{g / mL}$ 与等体积水混匀。

# 5 步骤

5.1 样品的保存

浸出液用硝酸(3.1)调 $\mathrm{pH}$ 小于2，贮于玻璃或聚乙烯瓶中并尽快分析。如放置，不要超过 $24\mathrm{h}$ 。

5.2 样品的处理

5.2.1 三价铬的氧化

取适量试样(含铬少于 $50~\mu \mathrm{g}$ )于 $150~\mathrm{mL}$ 三角瓶中作为试料，调至中性。加入几粒玻璃珠，加硫酸(3.15) $0.5\mathrm{mL}$ ，磷酸(3.16) $0.5\mathrm{mL}$ ，加水至 $50~\mathrm{mL}$ ，摇匀，加高锰酸钾(3.8)2滴，如红色消失，再加高锰酸钾，直至保持红色不退。加热煮沸至溶液剩 $20~\mathrm{mL}$ ，冷却后加脲素(3.9) $1.0~\mathrm{mL}$ ，摇匀，滴加亚硝酸钠(3.10)，每加一滴充分摇匀，至高锰酸钾溶液红色刚退，稍停片刻，待溶液内气泡完全逸出，转入 $50~\mathrm{mL}$ 比色管中，用水稀释至标线。若有大量有机物存在，需按附录A进行预处理后再测定。

5.3 空白试验

用 $50~\mathrm{mL}$ 水代替试液，按测定步骤(5.2)、(5.4)作空白试验。

5.4 测定

适量经5.2.1处理的试料于 $50~\mathrm{mL}$ 比色管中，用水稀释至刻线，加入显色剂(3.14) $2.0~\mathrm{mL}$ ，摇匀，放置 $10\mathrm{min}$ 后用10或 $30~\mathrm{mm}$ 光程比色皿，于 $540~\mathrm{nm}$ 处，以水作参比，测定吸光度，扣除空白试验(5.3)的吸光度，从校准曲线(5.5)上查得六价铬的量。

5.5 标准曲线的绘制

向9个 $150~\mathrm{mL}$ 三角瓶中，分别加入铬标准溶液(3.12)0.00、0.20、0.50、1.00、2.00、4.00、6.00、8.00、10.00mL，加水至 $50~\mathrm{mL}$ ，按(5.2)、(5.4)步骤测定，以减去空白吸光度为纵坐标，对应铬量为横坐标，作图。

# 6 结果的表示

浸出液中总铬的浓度 $c(\mathrm{mg / L})$ ，按下式计算：

$$
c = \frac {m}{V} \tag {1}
$$

式中： $m$ ——从校准曲线上查得试料中总铬的量， $\mu \mathrm{g}$

$V$ ——试料的体积， $\mathrm{mL}$ 。

# 7 精密度与准确度

7.1 可参考GB7466《水质总铬的测定》。
7.2室内对总铬浓度为 $196.8\mathrm{g / L}$ 的浸出液，9次平行测定相对偏差为 $0.54\%$ 。对总铬的含量为 $3.936\mu \mathrm{g}$ 的浸出液双样，各加入 $3.000\mu \mathrm{g}$ 的标样，其回收率为 $102.5\%$ 和 $103.9\%$ 。

# A1 有机物的消除

取适量试样于 $100\mathrm{mL}$ 烧杯中，加硝酸(3.1) $5\mathrm{mL}$ ，硫酸(3.2) $5\mathrm{mL}$ ，蒸发至冒白烟，冷却后用水稀释到 $10\mathrm{mL}$ ，用氨水溶液(3.6)调 $\mathrm{pH}$ 为 $1\sim 2$ ，转入 $50~\mathrm{mL}$ 容量瓶中，用水稀释至标线，摇匀，以下操作步骤同(5.2)。

# A2 试样中钼、钒、铜、铁的含量过高而干扰测定时可按下述方法消除：

取适量试样于分液漏斗中，加水至 $50~\mathrm{mL}$ ，用氨水(3.6)调至中性，加硫酸(3.15)3mL，用冰水冷却后，加铜铁试剂(3.7)5mL，振摇 $1\mathrm{min}$ ，于冰水中冷却 $2\mathrm{min}$ ，用氯仿(3.3)5mL，共萃取3次，弃去氯仿层，水层转入三角瓶中，用少量水洗分液漏斗于三角瓶中，加热煮沸。赶掉水中氯仿，氧化处理后测定。
