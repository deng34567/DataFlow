# 废电池冷却液处理处置技术规范

# 1 范围

本文件规定了废电池冷却液（以下简称冷却液）的一般要求、处理处置要求及运行管理要求。
本文件适用于各类动力电池的冷却液处理处置。

# 3 术语和定义

下列术语和定义适用于本文件。

# 3.1

电池冷却液 batterycoolant

以防冻剂、缓蚀剂等原料复配而成，主要含有乙二醇或丙二醇等成分，应用于动力电池冷却系统，与动力电池进行热量交换，保证动力电池处于正常使用温度的功能性液体。

# 3.2

乙二醇型冷却液 ethylene glycol base coolant

以乙二醇作为防冻剂的冷却液。

# 3.3

丙二醇型冷却液 propylene glycol base coolant

以1，2-丙二醇作为防冻剂的冷却液。

# 4 一般要求

4.1冷却液收集、处理处置过程应避免对环境造成污染，做好防泄漏、防遗洒等预防措施。
4.2冷却液收集、贮存和处理处置应在封闭或半封闭空间进行，并按照HJ610的相关要求做好作

业地面防渗处理。

4.3 企业应编制冷却液收集或处理处置作业指导书及事故应急预案。
4.4 操作过程中作业人员应按照GB/T11651的要求穿戴和使用防护装备。
4.5 根据废电池冷却液的回收量选择适合的冷却液处理处置方法。宜优先选择回收利用方法，无法回收利用的冷却液应进行无害化处理。

# 5.2 预处理

5.2.1 应对回收的动力电池进行信息采集，包括型号、种类、类别等，并进行追溯登记。

5.2.2 应通过外形检查、电压检测等方式对废弃动力电池的安全性进行评估，主要检查项目包括是否漏电或存在绝缘失效、外壳变形、破损、腐蚀和电池电压是否超出安全限制条件等，根据评估结果处理：

a）对无安全隐患的动力电池做好绝缘处理措施，先拆除动力电池外接导线及脱落的附属件，再进行冷却液的收集；
b）对存有安全隐患的动力电池，应排除安全隐患后再进行收集作业。

5.2.3 无法直接进行冷却液收集时，应按照GB/T33598中的要求对动力电池进行拆解，再进行冷却液的收集。

# 5.3 收集

# 5.3.1 分类收集

5.3.1.1 可确认冷却剂类型时宜单独收集、分类贮存和处理处置。
5.3.1.2 参照5.2.1采集的信息进行冷却液类型判定。

根据冷却液化学成分的不同，可分为以下3类：

a）乙二醇型冷却液；
b）丙二醇型冷却液；
c）其他类型冷却液。

# 5.3.2 混合收集

无法确认冷却剂类型时，可混合收集和处理。

# 5.3.3 收集要求

5.3.3.1 操作人员不应直接接触冷却液，宜采用机械化或自动化收集工艺，并配备冷却液收集设备。
5.3.3.2收集过程应配备至少2名作业人员，并做好安全防护，发生泄漏或紧急情况时快速启动应急措施。
5.3.3.3 冷却液收集设备应清洁、密封，配备相符的容量，具有防渗漏等功能。
5.3.3.4收集前应对收集设备进行气密性和破损性检查，存在漏气或破损时应及时进行检修或更换。
5.3.3.5 可采用负压抽空或正压吹出等方式收集冷却液，严格按照作业指导书进行操作。
5.3.3.6 收集完成后，宜通过收集设备继续操作 $15\mathrm{s}$ 以上，排出残余的冷却液。

# 5.4 贮存和运输

5.4.1 冷却液应根据NB/SH/T0164—2019中5.2.2的要求选择贮存和运输容器，且容器应具备闭合严密的盖子，并进行标注和区分。
5.4.2 冷却液应按照GB29743的要求存放，并定期对容器破损和泄漏情况进行检查。
5.4.3 贮存区域应按照GB2894的要求设置警告图形标识，配有符合国家相关规定的消防设施。
5.4.4 运输前应检查贮存容器气密性，不得对存在漏气或胀气风险的贮存容器进行运输，其运输宜符合NB/SH/T0164—2019第6章的有关要求。

# 5.5 处理处置方法

# 5.5.1 蒸馏法

5.5.1.1 分类收集的冷却液应按照GB29743的要求进行含量和成分检测，防冻剂（乙二醇或丙二醇）含量较高时可通过沉淀、过滤等手段去除冷却液中的杂质，采用蒸馏法对冷却液进行回收利用。简要流程框图如图2所示。

5.5.1.2 蒸馏得到的乙二醇或丙二醇浓缩液可作为溶剂收集利用，蒸馏残渣应按危险废物进行管理和处置。

# 5.5.2 焚化处理法

5.5.2.1 对无法回收利用或混合收集的冷却液可采用废液焚化工艺进行无害化处理，焚化温度应不低于 $1100\%$ 。
5.5.2.2 冷却液焚化过程产生的废气按照GB16297的排放限值执行。

# 6 运行管理要求

6.1 企业应建立废电池回收管理体系文件，冷却液的收集和处理处置文件应包括但不限于冷却液的收集时间、入库时间、出库时间、数量、类型、人员等。
6.2 贮存管理人员应对冷却液转移联单进行保管和记录，并做好月度和年度汇总。
6.3 应结合企业收集贮存规模，定期对废电池冷却液进行处理处置
