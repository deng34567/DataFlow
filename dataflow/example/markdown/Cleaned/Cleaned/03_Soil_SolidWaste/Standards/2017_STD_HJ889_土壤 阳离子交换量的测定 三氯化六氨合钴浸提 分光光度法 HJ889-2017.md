# 土壤阳离子交换量的测定三氯化六氨合钴浸提

# -分光光度法

# 1 适用范围

本标准规定了测定土壤中阳离子交换量的三氯化六氨合钴浸提-分光光度法。

本标准适用于土壤中阳离子交换量的测定。

当取样量为 $3 . 5 \ \mathrm { g }$ ，浸提液体积为 $5 0 . 0 \mathrm { m l }$ ，使用 $1 0 \mathrm { m m }$ 光程比色皿时，本标准测定的阳离子交换量的方法检出限为 $0 . 8 ~ \mathrm { c m o l ^ { + } / k g }$ ，测定下限为 $3 . 2 \mathrm { c m o l ^ { + } / k g }$ 。

# 2 规范性引用文件

本标准引用了下列文件或其中的条款。凡是不注日期的引用文件，其有效版本适用于本标准。

HJ613 土壤干物质和水分的测定重量法

HJ/T166 土壤环境监测技术规范

# 3 术语和定义

# 3.1

阳离子交换量 cation exchange capacity（CEC）

在本标准所规定的条件下，土壤胶体所能吸附的各种阳离子总量，称为阳离子交换量，以 $\mathrm { c m o l ^ { + } / k g }$ 表示。由于三氯化六氨合钴土壤悬浮液的 pH 值与水悬浮液的 pH 值接近，本方法测定的阳离子交换量为有效态阳离子交换量。

# 4 方法原理

在（20±2）℃条件下，用三氯化六氨合钴溶液作为浸提液浸提土壤，土壤中的阳离子被三氯化六氨合钴交换下来进入溶液。三氯化六氨合钴在 $4 7 5 \mathrm { n m }$ 处有特征吸收，吸光度与浓度成正比，根据浸提前后浸提液吸光度差值，计算土壤阳离子交换量。

# 5 干扰和消除

当试样中溶解的有机质较多时，有机质在 $4 7 5 \mathrm { n m }$ 处也有吸收，影响阳离子交换量的测定结果。可同时在 $3 8 0 \mathrm { n m }$ 处测量试样吸光度，用来校正可溶有机质的干扰。

假设 $\mathbf { A } _ { 1 }$ 和 ${ \bf A } _ { 2 }$ 分别为试样在 $4 7 5 \mathrm { n m }$ 和 $3 8 0 \mathrm { n m }$ 处测量所得的吸光度，则试样校正吸光度（A）为： $\mathrm { A } { = } 1 . 0 2 5 \mathrm { A } _ { 1 } { - } 0 . 2 0 5 \mathrm { A } _ { 2 }$ 。

# 6 试剂和材料

除非另有说明，分析时均使用符合国家标准的分析纯试剂（实验用水为电导率小于 0.5$\mu \mathrm { S } / \mathrm { c m }$ 的蒸馏水或去离子水）。

6.1 三氯化六氨合钴 $[ \mathrm { C o } ( \mathrm { N H } _ { 3 } ) _ { 6 } \mathrm { C l } _ { 3 } ]$ ：优级纯。
6.2 三氯化六氨合钴溶液： $c \mathrm { [ C o ( N H _ { 3 } ) _ { 6 } C l _ { 3 } ] = 1 . 6 6 ~ c m o l / L } .$ 。

准确称取 $4 . 4 5 8 \ \mathrm { g }$ 三氯化六氨合钴（6.1）溶于水中，定容至 $1 0 0 0 \mathrm { m l }$ ， $4 \mathrm { { } ^ { \circ } C }$ 低温保存。

# 7 仪器和设备

7.1 分光光度计：配备 $1 0 \mathrm { m m }$ 光程比色皿。
7.2 振荡器：振荡频率可控制在 $1 5 0 { \sim } 2 0 0$ 次 $/ \mathrm { m i n }$ 。
7.3 离心机：转速可达 $4 0 0 0 ~ \mathrm { r / m i n }$ ，配备 $1 0 0 \mathrm { m l }$ 圆底塑料离心管（具密封盖）。
7.4 分析天平：感量为 $0 . 0 0 1 \ \mathrm { g }$ 和 $0 . 0 1 \ \mathrm { g }$
7.5 尼龙筛：孔径 $1 . 7 \mathrm { m m }$ （10目）。
7.6 一般实验室常用仪器和设备。

# 8 样品

# 8.1 样品采集和保存

土壤样品采集和保存应按照 HJ/T 166 执行。土壤样品采集时，应使用木刀、木片或聚乙烯采样工具，土壤样品用布袋或塑料袋贮存。

# 8.2 试样的制备

将风干样品过尼龙筛（7.5），充分混匀。称取 $3 . 5 \ \mathrm { g }$ 混匀后的样品，置于 $1 0 0 \mathrm { m l }$ 离心管中，加入 $5 0 . 0 \mathrm { m l }$ 三氯化六氨合钴溶液（6.2），旋紧离心管密封盖，置于振荡器（7.2）上，在（20±2） $\mathrm { { } ^ { \circ } C }$ 条件下振荡（ $( 6 0 \pm 5 ) \mathrm { \ m i n }$ ，调节振荡频率，使土壤浸提液混合物在振荡过程中保持悬浮状态。以 $4 0 0 0 \mathrm { r / m i n }$ 离心 $1 0 \mathrm { m i n }$ ，收集上清液于比色管中， $^ { 2 4 \mathrm { h } }$ 内完成分析。

# 8.3 空白试样的制备

用实验用水代替土壤，按照与试样的制备（8.2）相同步骤进行实验室空白试样的制备。

# 10 结果计算与表示

# 10.1 结果计算

样品中，按照公式（1）进行计算：

$$
\mathrm {C E C} = \frac {\left(A _ {0} - A\right) \times V \times 3}{b \times m \times w _ {d m}} \tag {1}
$$

式中：CEC——土壤样品阳离子交换量，cmol +/kg；

$A \rho$ ——空白试样吸光度；

A——试样吸光度或校正吸光度；

V——浸提液体积，ml；

3—— $\mathrm { \cdot } \mathrm { [ C o ( N H _ { 3 } ) _ { 6 } ] ^ { 3 + } }$ 的电荷数；

$b _ { \mathbf { \alpha } }$ —标准曲线斜率；

$m$ —取样量，g；

wdm—土壤样品干物质含量， $\%$ 。

# 10.2 结果表示

当测定结果小于 10cmol $^ + / \mathrm { k g }$ 时，保留小数点后一位；当测定结果大于等于 10cmol +/kg时，保留三位有效数字。

# 12 质量保证和质量控制

12.1 每批样品应做标准曲线，标准曲线的相关系数不应小于 0.999。
12.2 每批样品应至少做 $1 0 \%$ 的平行样，当样品数量少于 10 个时，平行样不少于 1 个。

# 13 废物处理

实验过程中产生的废液和废物应分类收集和保管，并做好相应标识，委托有资质的单位进行处理。
