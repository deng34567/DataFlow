# 中华人民共和国国家标准

# 固体废物砷的测定

# 二乙基二硫代氨基甲酸银分光光度法

GB/T15555.3—1995

Solid waste-Determination of arsenic

Silver diethyldithiocarbamate spectrophotometric method

# 1. 主题内容与适用范围

1.1本标准规定了测定固体废物浸出液中砷用二乙基二硫代氨基甲酸银分光光度法。
1.2 本标准适用于固体废物浸出液中砷的测定。
1.2.1 试料量为 $50~\mathrm{mL}$ 时，用 $5\mathrm{mL}$ 吸收液， $10\mathrm{mm}$ 比色皿，检出限为 $0.007\mathrm{mg / L}$ 。测定上限浓度为 $0.5\mathrm{mg / L}$ 。
1.2.2 干扰

有锑、铋、硫离子共存时，有正干扰。

# 2 原理

在碘化钾与氯化亚锡存在下，使五价砷还原成三价砷。锌与酸作用产生新生态的氢，与三价砷作用生成砷化氢气体。此气体用二乙基二硫代氨基甲酸银-三乙醇胺氯仿溶液吸收，生成红色胶态银，在 $530\mathrm{nm}$ 波长处测量吸收液的吸光度。

# 3 试剂

本标准所用试剂除另有说明外，均使用符合国家标准或专业标准的分析纯试剂和蒸馏水或同等纯度的水。

3.1 二乙基二硫代氨基甲酸银 $(\mathrm{C}_5\mathrm{H}_{10}\mathrm{NS}_2\mathrm{Ag})$ 。
3.2 三乙醇胺 $\left[\left(\mathrm{HOCH}_{2} \mathrm{CH}_{3}\right)_{3} \mathrm{~N}\right]$ 。
3.3 氯仿 $\mathrm{(CHCl_3)}$
3.4 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ ， $\rho = 1.84\mathrm{g / mL}$

3.9 三氧化二砷 $\mathrm{(As_2O_3}$ ，优级纯）。

3.12 碘化钾溶液， $15\% (m / V)$

3.13 氢氧化钠(NaOH)溶液，5 mol/L。贮存于聚乙烯瓶中。

3.15 硫酸溶液， $2\mathrm{mol} / \mathrm{L}$
3.16 醋酸铅棉

称取 $10\mathrm{g}$ 脱脂棉浸泡在 $100\mathrm{mL}$ 醋酸铅溶液(3.14)中， $0.5\mathrm{h}$ 后取出，挤去水分，在室温下自然凉干。

3.17 砷化氢吸收溶液

称取 $0.25\mathrm{g}$ 二乙基二硫代氨基甲酸银(3.1)，用少量氯仿(3.3)溶成糊状，加入三乙醇胺(3.2) $2\mathrm{mL}$ ，再用氯仿(3.3)稀释到 $100\mathrm{mL}$ ，用力振动使之溶解后，于暗处放置24小时，用定性滤纸过滤至棕色瓶中，在冰箱中保存。

3.18 砷标准贮备溶液， $1000.0\mathrm{mg / L}$

将三氧化二砷(3.9)于 $110^{\circ}\mathrm{C}$ 下烘 $2\mathrm{h}$ ，冷却后准确称取 $0.1320\mathrm{g}$ ，用氢氧化钠溶液 $(3.13)2\mathrm{mL}$ 溶解后，加入硫酸溶液(3.15) $10\mathrm{mL}$ ，转移到 $100\mathrm{mL}$ 容量瓶中，用水稀释到刻度，充分摇匀。

3.19 砷标准溶液， $10.0\mathrm{mg / L}$

吸取 $1.0\mathrm{mL}$ 砷标准溶液(3.18)于 $100\mathrm{mL}$ 容量瓶中，用水稀释到刻度，充分摇匀。

3.20 砷标准使用溶液， $1.0\mathrm{mg / L}$

吸取 $10.0\mathrm{mL}$ 砷标准溶液(3.19)于 $100\mathrm{mL}$ 容量瓶中，用水稀释到刻度，充分摇匀。

注：三氧化二砷为剧毒化学药品(俗称砒霜),用时小心,切勿入口。

# 5 步骤

5.1 样品保存

制备好的浸出液置于玻璃瓶中，用硫酸(3.4)调节 $\mathrm{pH} < 2$ ，于 $4^{\circ}\mathrm{C}$ 下保存，不要超过 $7\mathrm{d}$ 。

# 5.2 试料

取适量浸出液（含砷不超过 $25\mu \mathrm{g}$ ），于砷化氢发生瓶（图中1）中，用水稀到 $50\mathrm{mL}$ 。

5.3 空白试验：在测定实际样品同时进行空白试验，所用试剂及用量均与测定(5.4)相同，包括预处理步骤。但用水取代试料。

# 5.4 测定

# 5.4.1 显色

5.4.1.1 向砷化氢发生瓶(图中1)中加入硫酸溶液(3.10)8 mL,碘化钾溶液(3.12)5.0 mL;氯化亚锡溶液(3.11)2.0 mL,摇匀放置10min。
5.4.1.2 吸取砷化氢吸收溶液(3.17) $5.0\mathrm{mL}$ ，置于吸收管(图中3)中，插入导气管，使其与发生瓶（图中1)连接好。
5.4.1.3 加入 $4\mathrm{g}$ 无砷锌粒(3.6)于砷化氢发生瓶(图中1)，立即将导气管与砷化氢发生瓶(图中1)连接好，保证反应装置密闭不漏气。
5.4.1.4 在室温下反应 $1\mathrm{h}$ ，使砷化氢气体完全释放出来，加氯仿(3.3)于吸收管(图中3)中，补充其吸收液体积到 $5.0\mathrm{mL}$ 并混匀。

注：① 砷化氢是剧毒气体，整个反应在通风橱内或通风良好处进行。

② 在砷化氢完全释放以后，吸收管内的红色胶体溶液在 $2.5\mathrm{h}$ 内是稳定的，应在这段时间内进行吸光度测定。

# 5.4.2 测量

用 $10\mathrm{mm}$ 比色皿，以氯仿(3.3)为参比，在 $530\mathrm{nm}$ 波长处测量吸收液(5.4.1.2)的吸光度，减去空白(5.3)试验的吸光度值，由所得的吸光度，从校准曲线(5.5.3)查得试样的含砷量 $(\mu \mathrm{g})$ 。

# 5.5 校准曲线的绘制

# 5.5.1 制备标准工作溶液

于8个砷化氢发生瓶(图中1)中，分别加入0.0、1.00、2.50、5.00、10.00、15.00、20.00、25.00 mL 砷标准溶液(3.20)，加水到 $50~\mathrm{mL}$

# 5.5.2 显色与测量

按测定（5.4）中所述步骤进行。

# 5.5.3 绘制校准曲线

减去空白试验(5.3)的吸光度，修正对应的每个标准溶液(5.5.1)的吸光度。以修正后的吸光度为纵坐标，与之对应的砷量为横坐标作图。

要经常绘制校准曲线，最好与被测试料同时绘制。

# 6 结果的表示

浸出液中砷的浓度 $c(\mathrm{mg / L})$ 按下式计算：

$$
c = \frac {m}{V} \tag {1}
$$

式中： $m$ ——于校准曲线上查得试料中的砷量， $\mu \mathrm{g}$

$V$ ——试料体积， $\mathrm{mL}$ 。

# 注意事项

（参考件）

A1 吸收液的高度应保持在 $8\sim 10\mathrm{cm}$ 为宜，且各管的高度应一致。
A2 各反应瓶的反应温度及酸度应保持一致，否则会影响精密度。
A3 试样的保存应用硫酸调至 $\mathrm{pH}$ 小于2，不可用硝酸。因硝酸浓度在 $0.01\mathrm{mol} / \mathrm{L}$ 时对砷的测定有负干扰。
A4 吸收管毛细管的口径必须小于 $1.0\mathrm{mm}$
A5 有时空白值偏高是因为 DDC-Ag 试剂变质。
A6 二乙基二硫代氨基甲酸银溶液颜色变深时，需要重配或用活性碳脱色后再用，否则会引起空白偏高。
A7当反应环境温度很高，还原反应速度激烈，可适当减少浓硫酸的用量或将砷化氢发生器放入冰水中，并不断补充氯仿于吸收管中，使吸收液高度一致。
A8 醋酸铅棉稍有变黑时，即应更换。

注：浸出液的制备方法，参见GB/T15555.1—1995《固体废物总汞的测定冷原子吸收分光光度法》中的附录B。
