# 引　言

塑料因其密度小、易加工、高性能、多功能等优异性能，被广泛应用于国民经济各领域。塑料工业不仅为各行各业提供了必要的材料，而且还是许多基础工业的重要组成部分。但随着塑料的推广应用，塑料废弃物也不断产生。通过物理或化学方法对其进行再生显得日益重要，这不仅能够减少塑料废弃物的产生，还能将废弃物转化为有价值的资源，实现资源的循环利用。然而，再生塑料中可能由于塑料废弃物中含有的限用物质，阻碍其再次利用，因此对其中的限用物质及其限量进行规范有利于塑料领域循环经济的健康发展。

为防止塑料废弃物中含有的限用物质对环境造成危害，需兼顾现阶段企业生产经营现状、监管机构监管效能和促进循环经济等多方面的情况。本文件的制定旨在以标准的形式为再生塑料的加工利用提供限用物质限量技术依据，完善塑料废弃物循环利用标准体系。因此本文件制定时确定了如下原则：一是尽可能提高塑料废弃物的循环利用率；二是鼓励塑料废弃物同级化、高附加值利用；三是在执行国家法律、法规、政策，统筹社会经济发展、确保经济发展的同时，使生态环境质量得到持续改善、国家环境安全得到切实维护。在上述原则基础上借鉴国际惯例，按照限制物质限制使用的时间、有害物质危害程度以及与再生塑料关联的紧密程度，本文件设置了三类不同监管要求的再生塑料中限用物质及其限量，即管控类限用物质及其限量、声明类限用物质及其限量和预警类限用物质及其限量。本文件给出了检验管控、声明采信、预警指引等分级管理模式。其中，管控类限用物质及其限量的设置旨在相关部门通过检验管控防止再生塑料中有害限用物质对环境、人身的危害，落实国家塑料污染防控有关规定的措施；声明类限用物质及其限量的设置遵循了国际惯例，在强化企业作为第一责任人同时并实现效能提高的管理手段。当制造商或进口商未提供相应的声明时，监管部门可升级转变监管模式；预警类的限用物质及其限量的设置旨在给出再生塑料在各个领域应用的信息，预防再生塑料制成产品后才发现限用物质超标，从而造成产生新的污染和浪费。

# 3　术语和定义

GB/T 2035、GB/T 30102、GB/T 38295 和 GB/T 40006.1 界定的以及下列术语和定义适用于本文件。

3.1

# 限用物质　restricted substances

再生塑料中限制含量的物质。

3.2

# 限量　limits

限用物质在再生塑料中允许的最大含量。

# 4　分类

# 4.1　总则

本文件将限用物质分为管控类、声明类、预警类，分别采用检验管控、声明采信、预警指引的分级管理模式。

# 4.2　管控类

管控类再生塑料限用物质指通过实施检验检测予以管控的一类限用物质，其限量应符合法律、法规和标准规定。

# 4.3　声明类

声明类再生塑料限用物质指按照国际惯例由制造商或进口商通过声明并由相关单位以采信形式保证符合法律、法规和标准规定再生塑料中的限用物质。

该声明应关注这些物质可能对人类健康和环境造成的潜在危害，制造商或进口商应及时更新其声明，以确保其合规性。

# 4.4　预警类

预警类再生塑料限用物质指由企业自愿检验是否符合法律、法规和标准限量要求的一类物质，仅起警示、预防、指引作用。

# 7　检验规则

# 7.1　组批规则

在生产领域，再生塑料由同一生产线上、相同原料、相同工艺所生产的同一牌号的产品组批，也可按一定生产周期或储存料仓为一批对产品进行组批。

流通、贸易、认证和监管等领域按相关规定进行。

产品以批为单位进行检验和验收。

注：批包括申报批、生产批和检验批等。

# 7.2　抽样方案

在生产领域，再生塑料可在料仓的取样口抽样，也可根据生产周期等实际情况确定具体的抽样方案。包装后产品的取样按 GB/T 2547 进行。

流通、贸易、认证和监管等领域，按相关规定进行。

# 7.3　判定和复验规则

管控类再生塑料限用物质按表 1 要求的项目进行检验和判定。

声明类再生塑料限用物质需要检验时，按表 2 要求的项目进行检验和判定。

预警类再生塑料限用物质需要检验时，检验和判定见表 A.1。

除另有规定外，按本文件要求得到的结果不符合本文件要求时，重新加倍取样作为复验样品，对该不符合项目进行复验。以复验结果作为该批产品的质量判定依据。
