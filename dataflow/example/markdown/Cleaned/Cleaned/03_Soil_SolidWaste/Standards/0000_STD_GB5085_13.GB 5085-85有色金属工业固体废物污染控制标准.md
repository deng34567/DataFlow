为贯彻执行《中华人民共和国环境保护法（试行)》，控制有色金属工业固体废物对环境的污染，保护人体健康，促进经济发展，特制订本标准。

本标准适用于全国有色金属工业的选矿、冶炼、加工生产过程及其环境保护设施中排出的固体废物。

# 1 定义

1.1 有色金属工业固体废物是指采矿、选矿、冶炼和加工过程及其环境保护设施中排出的固体或泥状的废弃物。其种类包括：采矿废石、选矿尾矿、冶炼弃渣、污泥和工业垃圾，无处理设施、长期堆存并对环境造成影响的生产过程排出的固体物，亦列为固体废物。
1.2 本标准所称的有色金属工业有害固体废物是指具有浸出毒性、腐蚀性、放射性和急性毒性四种中的一种或一种以上的固体废物及列入表2的固体废物。
1.3 凡不具有1.2规定特性的固体废物称为一般固体废物。

# 2 鉴别

2.1 有色金属工业固体废物浸出毒性的鉴别，应按GB5086—85《有色金属工业固体废物浸出毒性试验方法标准》执行。浸出液中任一种有害成分的浓度超过鉴别标准（见表1）的固体废物，定为有害固体废物。

# 有色金属工业固体废物污染控制标准

UDC 628.191:669. 2/.8

GB5085-85

Pollution control standard of solid waste in nonferrous metal industry

为贯彻执行《中华人民共和国环境保护法（试行)》，控制有色金属工业固体废物对环境的污染，保护人体健康，促进经济发展，特制订本标准。

本标准适用于全国有色金属工业的选矿、冶炼、加工生产过程及其环境保护设施中排出的固体废物。

# 1 定义

1.1 有色金属工业固体废物是指采矿、选矿、冶炼和加工过程及其环境保护设施中排出的固体或泥状的废弃物。其种类包括：采矿废石、选矿尾矿、冶炼弃渣、污泥和工业垃圾，无处理设施、长期堆存并对环境造成影响的生产过程排出的固体物，亦列为固体废物。
1.2 本标准所称的有色金属工业有害固体废物是指具有浸出毒性、腐蚀性、放射性和急性毒性四种中的一种或一种以上的固体废物及列入表2的固体废物。
1.3 凡不具有1.2规定特性的固体废物称为一般固体废物。

# 2 鉴别

2.1 有色金属工业固体废物浸出毒性的鉴别，应按GB5086—85《有色金属工业固体废物浸出毒性试验方法标准》执行。浸出液中任一种有害成分的浓度超过鉴别标准（见表1）的固体废物，定为有害固体废物。

表 1 有色金属工业固体废物浸出毒性鉴别标准


| 项目            | 浸出液的最高容许浓度，mg/L |
| -------------- | --------------- |
| 汞及其无机化合物（按Hg计） | 0.05            |
| 镉及其化合物（按Cd计）   | 0.3             |
| 砷及其无机化合物（按As计） | 1.5             |
| 六价铬化合物（按Cr+6计） | 1.5             |
| 铅及其无机化合物（按Pb计） | 3.0             |
| 铜及其化合物（按Cu计）   | 50              |
| 锌及其化合物（按Zn计）   | 50              |
| 镍及其化合物*（按Ni计）  | 25              |
| 铍及其化合物*（按Be计）  | 0.1             |
| 氟化物（按F计）       | 50              |


*为试行鉴别标准。

2.2 有色金属工业固体废物腐蚀性的鉴别，应按GB5087—85《有色金属工业固体废物腐蚀性试验方法标准》执行。pH值大于或等于12.5，或者小于或等于2.0的固体废物，定为有害固体废物。

2.3有色金属工业固体废物放射性的鉴别。应按现行的《放射防护规定》执行。

2.4 有色金属工业固体废物急性毒性的鉴别，应按GB5088—85《有色金属工业固体废物急性毒性初筛试验方法标准》执行。当引起小鼠（大鼠）在48h内死亡半数以上者，定为有害固体废物。

2.5 有色金属工业固体废物经调查、鉴别确认者，列入表2。

表 2 有害固体废物表


| 来源     | 有害固体废物名称          |
| ------ | ----------------- |
| 选矿     | 含高砷尾矿、含铀尾矿        |
| 铜冶炼    | 湿法炼铜浸出渣、砷铁渣       |
| 铅冶炼    | 含砷烟尘、砷钙渣          |
| 锌冶炼    | 湿法炼锌浸出渣、中和净化渣、砷铁渣 |
| 锡冶炼    | 含砷烟尘、砷铁渣、污泥       |
| 锑冶炼    | 湿法炼锑浸出渣、碱渣        |
| 稀有金属冶炼 | 铍渣                |
| 制酸     | 酸泥、废触媒            |


2.6 铜、铅、锌、镍、锡的选矿尾矿（含高砷尾矿和含铀尾矿除外）及火法熔炼渣，鉴别确认为一般固体废物。若原料成分和生产工艺改变时，需重新鉴别。

# 3 管理

3.1 有色金属工业固体废物在产生、处理和处置过程中应进行登记，按国家环境保护局的有关规定执行。
3.2有色金属工业固体废物处置场应设立标志，按附录A（补充件）规定执行。
3.3 有害固体废物的运输应遵照交通及有关部门的规定，采取相应的防护措施，不得流失。

# 4 综合利用

4.1 凡综合利用具有环境效益、经济效益和社会效益的有色金属工业固体废物，必须积极开发利用。
4.2 有色金属工业固体废物的综合利用，应采用新技术，并防止产生再次污染。其过程所产生的固体废物应执行本标准及现行有关标准。

# 5 有害固体废物的处理和处置

5.1 有害固体废物经无害化处理，鉴别合格者，可作为一般固体废物处置。
5.2 有害固体废物的最终处置应确保不污染环境。进行安全土地填筑处置时，应按附录B（参考件）规定执行。
5.3 有害固体废物暂无条件进行安全土地填筑处置或其他适用方法处置时，应专设防水、防渗的仓库暂存，严防毒物流失。
5.4 有害尾矿可在尾矿库中堆存，尾矿废水应加强监测，并按GB4913—85《重有色金属工业污染物排放标准》执行。尾矿库封场后应覆土植被。
5.5 放射性固体废物的处置应按现行的《放射防护规定》及其他有关法规执行。

# 6 一般固体废物处置

一般固体废物的处置应设固定处置场，集中堆放，分类管理，不得混入有害固体废物，防止扬散、流失和排入水体。处置场地表径流水的排放，应按现行的GB4913—85执行。

# 7 实施说明

当地方执行本标准不适用于当地环境特点时（如集中生活水源），可以按照国家有关规定制订地方固体废物污染控制标准。

# 安全土地填筑技术要求

（参考件）

B.1 安全土地填筑处置场（简称处置场）的选址原则
B.1.1 处置场应避开断层、滑坡、溶洞、洪泛区和地下水源，尽可能建在矿体边界线以外；离开城市居民区、历史文物古迹保护区、自然生物保护区、河流、湖泊、水库和风景游览区等。
B.1.2 选址时，应对该地区进行环境调查和环境影响评价工作、必要的工程地质勘探和土质类型分析工作以及技术经济论证工作等。
B.1.3 处置场应选在粘土层较厚的地区，利用天然峡谷、沟壑地形。处置场底部与地下水最高水位的相隔粘土层厚度需有 $1.5\mathrm{m}$ 以上。
B.2 处置场规模的确定原则
B.2.1 处置场的规模，应以生产厂（矿）的设计规模和服务年限为确定规模的依据，以年产生的有害固体废物和终期总量，规划并确定征用土地的范围和数量。
B.2.2 处置场的构筑规模，可采用把总规划的大处置场划分为几个单元小处置场，用满一个单元再建一个单元。每个单元服务年限四至五年为宜（见图B1）。
B.2.3 有害固体废物年处置量在1000吨以下的，化学毒性较强或有放射性的固体废物，可采用混凝土制的分格箱型处置场（见图B2）。
B.2.4 有害固体废物年处置量在1000吨以上，可采用地面周边筑坝式处置场（见图B3）。依处置场的规模和该地区土壤特性，确定处置场的防渗方案，选用防渗材料。

# B.3 处置场的构筑技术要求

B.3.1 有害固体废物安全土地填筑处置场，采用的天然和人造的防渗透材料，其渗透系数应小于或等于 $10^{-7} \mathrm{~cm} / \mathrm{s}$ 。渗透系数的测定应按水利电力部颁发的《土工试验规程》（SDS 01-79）中的《渗透试验（±-013-78）规程》进行。
B.3.2 对就地可取得丰富的粘土或炼铝厂的赤泥做处置场防渗材料的地区应积极加以开发并采用。
B.3.3 对缺少粘土的地区建造中小型处置场时，可采用塑料薄膜（聚氯乙烯、聚乙烯等）防渗材料。塑料薄膜厚度应在 $0.45\mathrm{mm}$ 以上，抗拉强度 $150\mathrm{kg/cm^2}$ 以上，拉断时伸长率在 $150\sim 200\%$ 以上为宜。
B.3.4 铺设塑料薄膜时，要清除处置场底部和坝坡周边的杂草、树根和腐植物；在薄膜的底层铺垫厚 $200\mathrm{mm}$ 以上的中细砂粒土保护层，整平压实；薄膜搭接宽度不小于 $100\mathrm{mm}$ ，粘接要保证严密牢固；薄膜上部铺压厚 $200\mathrm{mm}$ 以上的中细砂粒土保护层，整平压实，防止大块尖棱固体物刺破。
B.3.5 处置场周边坝的高度和坡度，按设计容量和处置场地的土质特性决定：如坝高 $2 \sim 3 \mathrm{~m}$ ，采用内挖（ $2 \sim 3 \mathrm{~m}$ ）外堆筑坝方式，余土做覆盖植被用；坡度可取 $1:1.5$ ， $1:2.0, 1:2.5, 1:3.0$ 等。
B.3.6 处置场周围设置浸出液收集井和监测（截渗）井。收集井按处置场规模设 $1\sim 2$ 眼或更多，敷设耐腐蚀管道通至处置场底部，将浸出液导出并排至污水处理系统；监测（截渗）井用于监测处置场周围的水质变化并能截断渗流，应沿处置场四周布井，井间距按处置场规模而定，最长不超过 $200\mathrm{m}$ 。
B.3.7 在处置场外围要构筑导流渠或导流坝，以保护处置场不受地面径流水冲涮和侵蚀。导流渠（坝）排水断面，应按处置场周围最大汇水量与纵向排水统一考虑，构筑要耐久。

在处置场周边设置沟渠，将处置场周边的积水导流至收集井内，以防外泄污染。

B.3.8 处置场随着有害固体废物处置进程最终实行土层覆盖，覆盖厚度要 $500\mathrm{mm}$ 以上，并植被。覆盖顶部要做排水梯级和散水坡度。坡度在 $0.5\sim 1.0\%$ 即可。
B.3.9 距处置场周边外 $5 \sim 10 \mathrm{~m}$ 处，安设处置场标志。大型处置场沿四周安设，每边长 $200 \mathrm{~m}$ 以内安设一个，超过 $200 \mathrm{~m}$ 设两个。小型处置场酌情安设，安设位置要面向公路、人行道或人活动频繁的方向为宜。
B.4 有色金属工业有害固体废物要分类处置，不要混杂，便于综合回收。
B.5 处置场要设专人管理、维护，保证运输通畅，场区整齐美观。
