# 3 术语和定义

固体废物 solid waste

固体废物是指在销毁日本遗弃在华化学武器的全过程中产生的所有固态、半固态废弃物质。主要包括下述几类：

3.1 一般固体废物，即除危险固体废物以外的所有其他固体废物。
3.2 危险固体废物，包括：
3.2.1 化学武器在销毁（焚烧）过程中产生的焚烧残渣和除尘器飞灰，以及其他销毁过程中产生的残余物。
3.2.2 化学武器在销毁过程中产生的废水/污水处理的污泥和残渣以及在必要时蒸发某些废水/污水的残余物。
3.2.3 在化学武器销毁过程中产生的所有染毒固体废物如化学武器及固体废物的包装物、工作服及防护用具、处理用工具和材料等。
3.2.4 其他危险废物焚烧过程中或其他处理过程中产生的残渣、飞灰、污泥等残余物。
3.2.5 一般固体废物在焚烧和其他处理过程中产生的残渣和飞灰等残余物。

# 5 采样

# 5.1 采样计划

在销毁日本遗弃在华化学武器的过程中，采集固体废物样品的目的，在于通过对样品的检测，判断其是否达到处理的目的，是否达到环境标准的要求。为此，在采样前应根据检测要求制定采样计划。主要内容包括采样目的和要求，背景调查和现场踏勘，采样程序，安全措施，质量控制，采样记录和报告等。

# 5.1.1 采样目的

从日本遗弃在华化学武器销毁处理前后的固体废物中，采集具有代表性的样品，通过检测和分析，获取在允许误差范围内的数据，故采样应尽可能达到下述要求：

（1）特性鉴别和分类：鉴别毒剂的类型和特性。
(2）环境污染监测：判别销毁处理前后污染物含量的变化。
(3) 销毁方式、方法或其他处置方案。

(4) 污染环境事故调查分析和应急监测。
(5）环境影响评价：确保销毁处理达到环境法规和环境标准要求，确保生态环境和人员安全。
(6）法律调查、法律责任和仲裁等。

# 5.1.2 背景调查和现场踏勘

根据采样目的和要求，调查以下影响采样计划制定的因素，并进行现场踏勘。

(1) 固体废物产生、堆存、埋藏或处置的现场踏勘，了解现场及周围环境。固体废物散布的区域、散布区域的地形地貌特征；固体废物的存在形式、贮存方式、包装物损坏程度；固体废物的埋藏深度、范围、方式；对生态和人员安全的影响等。
(2) 固体废物的种类、形态、数量、物理化学性质等。
(3) 固体废物试验和分析的允许误差和要求。
(4) 固体废物污染环境、监测分析的历史资料。

# 5.1.3 采样程序

——确定采样目标、目的和要求；
——选派采样人员；
——进行背景调查和现场踏勘；
——确定采样方法；
——确定样品包装；
——确定份样量；
——确定份样数；
——确定采样点；
——选择采样工具；
——制定安全措施；
——制定质量控制措施；
采样；
——采样记录。

# 5.1.4 采样工具

5.1.4.1 采样工具，原则上参照采用HJ/T20之4.3.1.1之所列。
5.1.4.2 对于有特殊要求的采样，可根据采样要求自选或自行设计合适工具。

# 5.1.5 采样记录和报告

采样时应记录样品的名称、编号、固体废物类型、监测项目、采样方法、采样量、采样时间、采样地点、采样点位及采样人员。

# 5.2 样品采集

# 5.2.1 概述

5.2.1.1 日本遗弃在华化学武器在销毁前存在的固体废物和在销毁全过程中产生的固体废物种类繁多，性质各异。这些固体废物经销毁处理后残余物的形式，大致有如下几种：

(1) 残渣。包括焚烧或焚毁残渣，其他销毁处理方式产生的残渣；污水处理产生的残渣以及某些污水或废水蒸发后遗留的残渣等。
(2）污泥。污水废水处理或其他销毁处理产生的污泥。
(3) 飞灰。主要是烟道气处理的除尘器飞灰。
(4) 金属残渣。如化学武器的弹体、弹片、金属包装物、销毁处理过程中使用的染毒金属设备、工具等。
(5) 半固态废物。

5.2.1.2 由于销毁日本遗弃在华化学武器是一次性销毁处理，作业现场的污染监测没有历史记录可

供参考，无法估计采样方法和采样点设置等因素可能带来的监测误差，故只能采用随机采样法采样。5.2.1.3 采样方法的设计和实施，必须保证固体废物中待检测组分，尤其是那些易挥发、易分解，对热不稳定、对光敏感、毒性剧烈、形态多变的待测组分在采样和制样过程中的损失最少，以切实保证检测结果的准确度。

# 5.2.2 残渣采样

# 5.2.2.1 焚烧炉残渣的采样

# 5.2.2.1.1 间歇操作炉残渣的采样

用简单随机采样法采样。

(1) 若残渣量较少, 可按炉逐炉采样, 每炉一个。采样时, 待炉温降至室温后采样。
(2) 若残渣量较大，可将残渣全部清出，平铺，按梅花形、对角线、棋盘形或其他科学方法采样；或先设定取样个数，用简单随机数表法或抽签法（参见HJ/T20之4.2.1.1）设定采样点，逐点采取份样；再取各采样点的等量份样混匀，缩分，留足试样。

# 5.2.2.1.2 连续出渣炉残渣的采样

用系统随机采样法采样。

（1）根据焚烧炉出渣速度和数量，设定采样时间间隔，按时采样。通常情况下，设备运行初期或维修等不正常运行时，采样频率为1个/h；正常运行时，采样频率为1个/d。
(2) 不论是在运送带上还是在落口处采样, 均应采集废物流的全截面作为一个份样。
(3) 采集第一个份样时, 不可在第一间隔的起点开始, 可在第一间隔内随机确定。

# 5.2.2.2 废水/污水蒸发残渣的采样

将蒸发残渣全部清出，平铺，参照本标准之“5.2.2.1.1间歇操作炉残渣的采样”法采样。

# 5.2.3 污泥池污泥的采样

# 5.2.3.1 简单随机采样法

(1) 根据污泥池面积的大小, 设定采样个数 (份样数) $n$ ;
(2) 将污泥池划分为约 $5n$ 个面积相等的网格，顺序编号，用抽签法或随机数表法确定 $n$ 个网格作为采样单元；在每个采样单元的中心位置用适当的工具垂直采取样品，即为一个份样；
(3) 若污泥有一定厚度，且非连续沉积，可分层采样；分两层时按污泥厚度的 $1/3$ 、 $2/3$ 处设采样点；分三层时，按污泥厚度的 $1/6$ 、 $1/2$ 、 $5/6$ 处设采样点，或用其他科学方法逐层设置采样点。按采样点分别采样，再把各采样点的样品取等量混匀、缩分、留取份样。

# 5.2.3.2 亦可采用梅花形、对角线、蛇形、棋盘形或其他合适方法确定采样点，逐点采取份样。

# 5.2.4 除尘器飞灰的采样

用系统随机采样法采样。

(1) 根据除尘器的出灰速度和数量，设定采样时间间隔，按时采样。通常情况下，设备运行初期或维修等不正常运行时，采样频率为1个/h；正常运行时，采样频率为1个/d。
(2）采样位置：除尘器出灰口。
(3) 采样时，必须采集除尘器出灰口整个断面上的飞灰样品，混匀，缩分，留足小样。

# 5.2.5 弹（筒）片等大块残渣的采样

（1）适合提取的，采用溶剂提取法采样。
(2) 不适合提取的，采用擦拭法采样。

# 5.2.6 待固化固体废物的采样

(1) 以每只固化体为一批, 根据批量大小, 参照 “HJ/T 20 之 4.2.1.2 系统采样法” 中 “表 1 批量大小与最少份样数” 确定最少采集份样数目。
(2）按固化体的体积（或容积）和采集份样数目，用梅花形、对角线、棋盘形或其他科学方法确定采样点。

(3) 若有必要，可将每个采样点分为若干层，层的设置参照本标准之5.2.3.1（3）进行。每点每层采取一个份样；取等量份样混匀，缩分，采集本采样点的小样。

# 5.2.7 半固态废物的采样

5.2.7.1 对半固态废物的采样，可根据其物理、化学性质和存在状态，参照本标准之 $5.2.2\sim 5.2.6$ 进行；或者参照HJ/T20之4.3.2的规定进行采样。
5.2.7.2 在常温下为固体，受热时易变成流动的液体而不改变其化学性质的半固态废物，参照HJ/T20之4.3.3.2的规定进行采样。
5.2.7.3 对粘稠的液体、有流动而又不易流动的废物，参照HJ/T20之4.3.3.3的规定进行采样。

# 5.3 采样量

根据待测组分所需要的样品量现场确定，样品的采集量必须保证足够用于样品分析。

# 6 制样

6.1 在销毁日本遗弃在华化学武器过程中，用于检测销毁处理前固体废物中化学毒剂的样品时，若被测物具有挥发性、半挥发性、热不稳定性、光敏感性或其他特殊性质，经试样处理会造成被测组分损失的样品，一般不做预处理；样品经混合均匀后直接检测。

# 6.2 弹（筒）片等大块残渣的制样

# 6.2.1 擦拭法

6.2.2.1 擦拭用材料及溶剂。擦拭用材料及溶剂根据被测残渣的性质和被检测物的性质而定。擦拭用材料可用滤纸或脱脂棉或其他满足制样要求的材料制样。擦拭用溶剂采用分析方法中规定要求的溶剂。
6.2.2.2 擦拭面积。擦拭面积根据被制样的大块固体废物、残片面积和制样要求、目的，选择合适的比例。
6.2.2.3 制样单元和制样点的设置。可根据弹片面积的大小和几何形状，根据染毒物的分布特点，在弹片总面积的合适的比例，随机划分为若干制样单元，再在各制样单元内随机设置或均匀设置若干制样点。
6.2.2.4 擦拭效率。擦拭材料的选择和预处理，应根据被检组分的性质和制样要求而定，以保证制样效率和制样的代表性、精密度。尤其是在挥发性组分制样时，应特别注意采取必要的措施，避免挥发性组分的损失。

# 6.2.2 溶剂提取法

6.2.2.1 对于体积较小、具有空腔的固体废物，需要采集内外表面污染物总量时可用此法。
6.2.2.2 提取溶剂的选择，采用分析方法中规定要求的溶剂。
6.2.2.3 为提高提取效率，可采用机械振荡、超声波搅动或用对被测组分为惰性的气体通气搅动等方法加速之。
6.3 对于其他销毁化学武器固体废物，需要预处理时，参照HJ/T20有关规定执行。

# 7 质量控制

7.1 为保证在允许范围内获得固体废物的具有代表性的样品，应在采样、制样的全过程中进行质量控制。
7.2 采样制样前，应设计详细的采样制样方案（计划）；在采样制样过程中，严格按照采样制样方案进行操作。
7.3 对相关采样制样人员进行培训。采样制样人员应熟悉样品的性状、掌握采样技术、懂得安全操作的有关知识和处理方法。采样制样时，应由2人以上在场进行操作。
7.4 采样制样工具、设备所用材质不能和样品有任何反应、不破坏样品代表性、不改变样品组成。

7.5 采样制样过程中要防止待采固体废物受到污染、发生变质，要防止固体废物样品中待测组分的降解、挥发等造成的样品损失。

7.6 盛样容器应满足以下要求：

——盛样容器材质与样品物质不起作用，没有渗透性；
——具有符合要求的盛样容器的盖、塞，在使用前应洗净、干燥；
待测组分有特殊要求的，按分析方法中规定要求进行。

7.7 样品经采集后，盛入容器，在容器壁上应随即贴上标签。标签内容包括：

——样品的名称、编号；
固体废物类型；
——监测项目；
——采样方法；
采样量；
——采样时间；
采样地点；
——采样点位；
采样人员。

7.8 样品移交

在采样现场样品必须逐件与样品登记表、样品标签和采样记录进行核对，核对无误后分箱；
——样品运输过程中应防止不同样品之间的交叉污染；
——样品由专人送到指定实验室；
——送样人和接样人双方同时清点核实样品，并在样品交接单上签字确认；
——样品移交单需记录样品名称、样品编号、数量、样品状态的相关描述、移交时间等信息；
——样品移交单由送样人和接样人双方各存一份备查。

7.9 样品经制样后，盛入容器，在容器壁上应随即贴上标签。标签内容包括：

——样品的名称、编号；
——固体废物类型；
——监测项目；
——送样时间；
——送样人；
——制样日期；
——制样人员；
——样品保存期。

7.10 填写、保存好采样制样记录和采样制样报告。

# 8 安全措施

由于销毁日本在华化学武器过程中的固体废物为化学毒剂污染，采样制样之前，需制定严格的安全措施。操作时应按规定要求进行操作，确保采样制样的人员和环境安全。

8.1 为了保障采样制样操作的安全，需要对可能实施现场采样及制样所涉及的人员进行必要的安全防护知识培训和防护演练，保证采样制样人员了解样品的危险性及预防措施。
8.2 针对可能出现的紧急情况，制定相应的紧急救助方案，指定出现紧急情况的相关负责人。
8.3 每次作业前审核作业程序，确保作业过程所需防护器材完备。
8.4 现场采样需按规定要求佩戴防护器材，如防护眼镜、防护服等。
8.5 任何一次采样制样均应两人或两人以上进行操作，需有一名为陪伴者，陪伴者应处于能清楚地

看到采样点的地方并观察整个采样操作过程。陪伴者应受过专门训练，懂得在紧急情况时该采取什么行动，这些训练要求他首先报告相关负责人，除非在极特殊的情况下，不要单独一人去进行营救。

8.6 采样前及采样后应向相关负责人汇报，尤其要汇报发生的异常事件和情况。
8.7 采样制样过程中，操作人员一旦感到不适时，应立即向相关负责人报告。
8.8 样品运输过程中，采用适当（如有密闭、减压措施等）的运输工具，确保运输沿途人员及环境安全。
8.9 实验室操作时应按规定要求佩戴防护器材，在通风良好的环境中进行操作。
8.10 样品制样及测试后的残渣残液应作妥善的安全处理。
8.11 测试后的残余样品除部分作验证结果所需外，其余样品均应做妥善的安全处理。

# 9 样品保存

# 9.1 样品的暂存和运输

9.1.1 样品采集后，容器采样瓶采用无顶空密闭。并于 $4\%$ 以下保存，尽快分析。
9.1.2 样品运输过程中应保持低温状态，防止因运输引起的温度升高，从而导致待测组分的变化。
9.1.3 待测组分有特殊要求的，按分析方法中规定要求进行暂存和运输。

# 9.2 样品的实验室保存

9.2.1 样品装入容器后应立即密封并贴上样品标签。
9.2.2 对易挥发待测组分，样品应采取无顶空存样，并采取低温保存。
9.2.3 对光敏感的待测组分，样品应装入深色容器中并置于避光处。
9.2.4 对温度敏感的待测组分，样品应保存在规定的温度之下。
9.2.5 样品保存应防止受潮或受灰尘等污染。
9.2.6 待测组分有特殊要求的，按分析方法中规定要求进行保存。
9.2.7 确定样品的保存日期。
9.2.8 样品应在特定场所由专人保管。
9.2.9 超过保存期的样品不允许随意丢弃，按危险废物要求进行处置。
