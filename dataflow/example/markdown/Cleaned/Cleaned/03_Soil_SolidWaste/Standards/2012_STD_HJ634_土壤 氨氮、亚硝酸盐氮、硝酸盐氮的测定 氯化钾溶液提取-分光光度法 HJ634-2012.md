# 前言

为贯彻《中华人民共和国环境保护法》，保护环境，保障人体健康，规范土壤中氨氮、亚硝酸盐氮、硝酸盐氮的测定方法，制订本标准。

本标准规定了测定土壤中氨氮、亚硝酸盐氮、硝酸盐氮的氯化钾溶液提取－分光光度法。

本标准的技术内容为修改采用《土壤质量硝酸盐氮、亚硝酸盐氮、氨氮的测定氯化钾溶液提取法》（ISO/TS 14256-1:2003（E））。附录 A 给出了本标准与 ISO/TS 14256-1:2003（E）章条编号对照一览表，附录 B 给出了本标准与 ISO/TS 14256-1:2003（E）的技术性差异及其原因。

本标准为首次发布。

# 1 适用范围

本标准规定了测定土壤中氨氮、亚硝酸盐氮、硝酸盐氮的氯化钾溶液提取－分光光度法。

本标准适用于土壤中氨氮、亚硝酸盐氮、硝酸盐氮的测定。

当样品量为 $4 0 . 0 \ \mathrm { g }$ 时，本方法测定土壤中氨氮、亚硝酸盐氮、硝酸盐氮的检出限分别为$0 . 1 0 \mathrm { m g / k g }$ 、 $0 . 1 5 ~ \mathrm { { m g / k g } }$ 、 $0 . 2 5 \mathrm { m g / k g }$ ，测定下限分别 $0 . 4 0 \mathrm { m g / k g }$ 、 $0 . 6 0 \mathrm { m g / k g }$ 、 $1 . 0 0 \mathrm { m g / k g }$ 。

# 3 方法原理

# 3.1 氨氮

氯化钾溶液提取土壤中的氨氮，在碱性条件下，提取液中的氨离子在有次氯酸根离子存在时与苯酚反应生成蓝色靛酚染料，在 $6 3 0 \mathrm { n m }$ 波长具有最大吸收。在一定浓度范围内，氨氮浓度与吸光度值符合朗伯-比尔定律。

# 3.2 亚硝酸盐氮

氯化钾溶液提取土壤中的亚硝酸盐氮，在酸性条件下，提取液中的亚硝酸盐氮与磺胺反应生成重氮盐，再与盐酸 N-（1-萘基）-乙二胺偶联生成红色染料，在波长 $5 4 3 \mathrm { n m }$ 波长具有最大吸收。在一定浓度范围内，亚硝酸盐氮浓度与吸光度值符合朗伯-比尔定律。

# 3.3 硝酸盐氮

氯化钾溶液提取土壤中的硝酸盐氮和亚硝酸盐氮，提取液通过还原柱，将硝酸盐氮还原为亚硝酸盐氮，在酸性条件下，亚硝酸盐氮与磺胺反应生成重氮盐，再与盐酸 N-（1-萘基）-乙二胺偶联生成红色染料，在波长 $5 4 3 \mathrm { n m }$ 处具有最大吸收，测定硝酸盐氮和亚硝酸盐氮总量。硝酸盐氮和亚硝酸盐氮总量与亚硝酸盐氮含量之差即为硝酸盐氮含量。

# 5 仪器和设备

5.1 分光光度计：具 $1 0 \mathrm { m m }$ 比色皿。
5.2 pH 计：配有玻璃电极和参比电极。
5.3 恒温水浴震荡器：振荡频率可达 40 次/分钟。
5.4 还原柱：用于将硝酸盐氮还原为亚硝酸盐氮，具体制备方法见附录 C。
5.5 离心机：转速可达 $3 0 0 0 \mathrm { r / m i n }$ ，具 $1 0 0 ~ \mathrm { m l }$ 聚乙烯离心管。
5.6 天平：精度为 $_ { 0 . 0 0 1 \mathrm { ~ g ~ } }$
5.7 聚乙烯瓶： $5 0 0 \mathrm { m l }$ ，具螺旋盖。或采用既不吸收也不向溶液中释放所测组分的其他容器。

5.8 具塞比色管： $2 0 ~ \mathrm { m l }$ 、 $5 0 ~ \mathrm { m l }$ 、 $1 0 0 ~ \mathrm { m l }$ 。
5.9 样品筛： $5 \mathrm { { m m } }$ 。
5.10 一般实验室常用仪器和设备。

# 6 样品

# 6.1 样品的采集

按照 HJ/T 166 的相关规定采集样品。

# 6.2 样品的保存

样品采集后应于 $4 ~ \mathrm { { ^ \circ C } }$ 下运输和保存，并在 3 日内分析完毕。否则，应于 $- 2 0 ~ \mathrm { { ^ circ C } }$ （深度冷冻）下保存，样品中硝酸盐氮和氨氮可以保存数周。

当测定深度冷冻的硝酸盐氮和氨氮含量时，应控制解冻的温度和时间。室温环境下解冻时，需在 4 h内完成样品解冻、匀质化和提取；如果在 $4 ~ \mathrm { { ^ \circ C } }$ 下解冻，解冻时间不应超过 $4 8 \mathrm { { h } }$ 。

注 1：为了缩短样品的解冻时间，应在样品被冷冻前，将其敲碎成小颗粒状。

# 6.3 试样的制备

将采集后的土壤样品去除杂物，手工或仪器混匀，过样品筛。在进行手工混合时应戴橡胶手套。过筛后样品分成两份，一份用于测定干物质含量，测定方法参见 HJ 613；另一份用于测定待测组分含量。

# 6.4 试料的制备

称取 $4 0 . 0 \ \mathrm { g }$ 试样（6.3），放入 $5 0 0 \mathrm { m l }$ 聚乙烯瓶中，加入 $2 0 0 ~ \mathrm { m l }$ 氯化钾溶液（4.1.7），在$2 0 { \pm } 2 \mathrm { ^ \circ C }$ 的恒温水浴震荡器中震荡提取 $^ { 1 \mathrm { ~ h ~ } }$ 。转移约 $6 0 ~ \mathrm { m l }$ 提取液于 $1 0 0 \mathrm { m l }$ 聚乙烯离心管中，在 $3 0 0 0 ~ \mathrm { r / m i n }$ 的条件下离心分离 $1 0 \ \mathrm { m i n }$ 。然后将约 $5 0 ~ \mathrm { m l }$ 上清液转移至 $1 0 0 ~ \mathrm { m l }$ 比色管中，制得试料，待测。

注 2：提取液也可以在 $4 ~ \mathrm { { ^ \circ C } }$ 下，以静置 $^ { 4 \mathrm { h } }$ 的方式代替离心分离，制得试料。

# 8.1 结果计算

# 8.1.1 氨氮

样品中的氨氮含量 $\omega$ （ $\mathrm { m g / k g } )$ ），按照公式（1）进行计算。

$$
\omega = \frac {m _ {1} - m _ {0}}{V} \cdot f \cdot R \tag {1}
$$

$\omega$ -样品中氨氮的含量， $\mathrm { m g / k g }$ ：

$m _ { 1 }$ —从校准曲线上查得的试料中氨氮的含量，μg；

$m _ { 0 }$ —从校准曲线上查得的空白试料中氨氮的含量， $\mu \mathrm { g }$ ；

V——测定时的试料体积， $1 0 . 0 \mathrm { m l }$ ；

$f$ ——试料的稀释倍数；

$R$ ——试样体积（包括提取液体积与土壤中水分的体积）与干土的比例系数， $\mathrm { m l / g }$ ；按照公式（2）进行计算。

$$
R = \frac {\left[ V _ {E S} + m _ {s} \cdot \left(1 - w _ {d m}\right) / d _ {H _ {2} O} \right]}{m _ {s} \cdot w _ {d m}} \tag {2}
$$

VES $V _ { E S }$ 提取液的体积， $2 0 0 \mathrm { m l }$ ；

$m _ { s }$ ms 试样量， $4 0 . 0 \mathrm { g }$ ；

dH2O — $d _ { { \scriptscriptstyle H } _ { 2 } O }$ 水的密度， $1 . 0 \mathrm { g / m l }$ ；

$w _ { d m }$ wdm 土壤中的干物质含量， $\%$ 。

\omega = \frac {m _ {1} - m _ {0}}{V} \cdot f \cdot R \tag {3}

$m _ { 0 }$ —从校准曲线上查得的空白试料中亚硝酸盐氮的含量， $\mu \mathrm { g }$ ；

V——测定时的试料体积， $1 . 0 0 \mathrm { m l }$ ；

$f$ ——试料的稀释倍数；

$R$ ——试样体积（包括提取液体积与土壤中水分的体积）与干土的比例系数， $\mathrm { m l / g }$ ；按照公式（2）进行计算。

# 还原柱的制备

# C.1 镉粉的处理

用浓盐酸（4.3.2）浸泡约10g 镉粉（4.3.3）10min，然后用水冲洗至少五次。再用水（水量盖过镉粉即可）浸泡（约 $1 0 \mathrm { { m i n } } .$ ），加入约 0.5g 硫酸铜，混合 1min，然后用水冲洗至少10 次，直至黑色铜絮凝物消失。重复采用浓盐酸（4.3.2）浸泡混合 1min，然后用水冲洗至少五次。处理好的镉粉，用水浸泡，在 1h 内装柱。

# C.2 还原柱的准备

向还原柱底端加入少许棉花，加水至漏斗三分之二处（L1），缓慢添加处理好的镉粉至L3 处（约为 $1 0 0 ~ \mathrm { { m m } }$ ），添加镉粉的同时，应不断敲打柱子使其填实，最后，在上端加入少许棉花至 L2处。

如果还原柱在 1h 内不使用，应加入氯化铵缓冲溶液贮备液（4.3.19）至L1 处。盖上漏斗盖子，防止蒸发和灰尘进入。在这样的条件下，还原柱可保存一个月。但是，每次使用前要检查还原柱的转化效率。
