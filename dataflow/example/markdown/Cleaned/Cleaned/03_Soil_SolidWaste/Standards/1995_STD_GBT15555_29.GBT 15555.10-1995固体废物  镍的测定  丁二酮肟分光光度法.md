# 中华人民共和国国家标准

# 固体废物镍的测定

# 丁二酮肟分光光度法

GB/T15555.10—1995

Solid waste-Determination of nickel

Dimethylglyoxime spectrophotometric method

# 1 主题内容与适用范围

# 1.1 主题内容

本标准规定了测定固体废物浸出液中镍的丁二酮肟（二甲基乙二醛肟)分光光度法。

# 1.2 适用范围

1.2.1 本标准适用于含镍废渣浸出液中镍的测定。
1.2.2 本标准检测浓度为 $0.1\mathrm{mg / L}$ ，测定上限为 $4\mathrm{mg / L}$
1.2.3 铁、钴、铜离子干扰测定，加入 $\mathrm{Na_2}$ -EDTA溶液，可消除 $300\mathrm{mg / L}$ 铁、 $100\mathrm{mg / L}$ 钴及 $50\mathrm{mg / L}$ 铜，对 $5\mathrm{mg / L}$ 镍测定的干扰。若铁、钴、铜的含量超过上述浓度，则可用丁二酮-正丁醇萃取分离除去（见附录A）。

氰化物亦干扰测定。可在测定前于样品中加入 $2\mathrm{mL}$ 次氯酸钠溶液和 $0.5\mathrm{mL}$ 硝酸加热分解镍氰络合物。

# 2 原理

在柠檬酸铵-氨水介质中，当有氧化剂碘存在下，镍与二丁酮作用，形成组成比为 $1:4$ 的酒红色络合物，可于波长 $530\mathrm{nm}$ 处进行分光光度的测定。

# 5 步骤

# 5.1 样品的保存

浸出液制备后应立即加入浓硝酸(3.1)酸化，至 $\mathrm{pH}$ 值 $1\sim 2$ 。保存时间不要超过一周。

# 5.2 空白试验：

在测定的同时进行空白试验，所用试剂及用量均与测定(5.4、5.5)中所用相同，测定步骤亦相同，但用 $10.0\mathrm{mL}$ 水取代试样。

# 5.3 样品前处理

除非证明样品的消解处理是不必要的，可直接进行显色。否则按下述步骤进行处理。

取适量试样(含镍不超过 $100\mu \mathrm{g}$ )于烧杯中，加 $0.5\mathrm{mL}$ 的硝酸(3.1)，置于电热板上，在近沸状态下蒸发至近干，冷却后，再加 $0.5\mathrm{mL}$ 的硝酸(3.1)和 $0.5\mathrm{mL}$ 的高氯酸(3.3)继续加热消解，蒸发至近干。用硝酸溶液(3.7)溶解，若溶液仍不清澈，则重复上述操作，直到溶液清澈为止。将溶液转移到 $25\mathrm{mL}$ 的容量瓶中，用少量的蒸馏水冲洗烧杯，按(5.4)进行显色。

# 5.4 显色

取适量试样（含镍不超过 $100\mu \mathrm{g}$ ），置于 $25\mathrm{mL}$ 容量瓶中并用水稀释至约 $10\mathrm{mL}$ ，用氢氧化钠溶液（3.8）约 $1\mathrm{mL}$ 使呈中性，加 $2\mathrm{mL}$ 柠檬酸铵溶液（3.9）， $1\mathrm{mL}$ 碘溶液（3.11），加水至约 $20\mathrm{mL}$ ，摇匀，加 $2\mathrm{mL}$ 丁二酮肟溶液（3.12）摇匀。加 $2\mathrm{mLNa}_{2}$ -EDTA溶液（3.14），加水至标线，摇匀。放置 $5\mathrm{min}$ 。

注：加入柠檬酸铵络合铁和锰，防止在氨碱性中形成氢氧化物沉淀，并可使显色液较快达到稳定。并不干扰测定，但使灵敏度稍有降低。

# 5.5 测量

用 $10\mathrm{mm}$ 比色皿，以水为参比液，在 $530\mathrm{nm}$ 波长处测量显色液(5.4)的吸光度并扣除空白(5.2)所测的吸光度。

注：在室温低于 $20^{\circ}\mathrm{C}$ 下显色时，络合物吸光度至少在 $1\mathrm{h}$ 内不变，否则络合物的吸光度稳定性随温度升高而下降。因此，在此情况下，须在较短时间( $15\mathrm{min}$ )内显色测定，且试料测定与绘制校准曲线的显色时间尽量一致。

# 5.6 校准曲线的绘制

往6个 $25~\mathrm{mL}$ 容量瓶中，分别加入0、1.00、2.00、3.00、4.00及 $5.00~\mathrm{mL}$ 镍标准工作溶液（3.20），并加水到 $10~\mathrm{mL}$ ，加 $2\mathrm{mL}$ 柠檬酸铵溶液(3.9)，以下步骤按(5.4)和(5.5)所述进行显色与测量。

以测定的各标准溶液的吸光度扣除空白(零浓度)后的吸光度，与对应的标准溶液的镍含量绘制校准曲线。

# 6 结果的表示

浸出液中镍浓度 $c(\mathrm{mg / L})$ 由回归方程或下式计算：

$$
c = \frac {m}{\bar {V}}
$$

式中： $m$ ——由校准曲线上查得的含镍量， $\mu \mathrm{g}$

$V$ ——试料的体积， $\mathrm{mL}$ 。

# 7 精密度和准确度

7.1 可参考GB11910《水质镍的测定丁二酮肟分光光度法》。
7.2 于镍渣浸出液（含镍 $12.5\mu \mathrm{g}$ ），加标 $10\sim 20\mu \mathrm{g}$ ，其回收率在 $95\% \sim 103\%$ 之间。

# 附件A

# 丁二酮肟-正丁醇萃取分离操作步骤

（补充件）

# A1 萃取分离

A1.1 置试料于 $100\mathrm{mL}$ 分液漏斗中，加 $2\mathrm{mL}$ 柠檬酸铵溶液(3.10)、 $2\mathrm{mL}$ 丁二酮肟乙醇溶液(3.13)，摇匀。加一滴酚酞溶液(3.21)，滴加氨水溶液(3.15)使溶液出现红色，再加 $1\mathrm{mL}$ 氯水-氯化铵缓冲液(3.18)，加水至约 $30\mathrm{mL}$ ，摇匀。

A1.2 用 $10\mathrm{mL}$ 正丁醇(3.6)萃取 $1\sim 2\mathrm{min}$ ，静止分层后，弃尽水相。
A1.3 用 $5\mathrm{mL}$ 氨水溶液(3.16)振摇 $30\mathrm{s}$ ，洗涤有机相一次，弃去水相。
A1.4 加入 $5\mathrm{mL}$ 盐酸溶液(3.17)振摇 $1\sim 2\mathrm{min}$ ，反萃取镍。分层后将水相完全转入 $25\mathrm{mL}$ 容量瓶中，再用 $5\mathrm{mL}$ 水洗涤有机相一次，合并水相。

# A2 显色和测量

于 $25\mathrm{mL}$ 容量瓶中加约 $1\mathrm{mL}$ 氢氧化钠溶液(3.8)使呈中性，加 $0.5\mathrm{mL}$ 柠檬酸铵溶液(3.10)，以下按(5.4)和(5.5)步骤进行显色和测量。

# A3 校准曲线的绘制

向6个 $100~\mathrm{mL}$ 分液漏斗中，分别加入0、0.50、1.00、2.00及 $2.50~\mathrm{mL}$ 镍标准工作溶液(3.20)。以下按A1、A2步骤进行萃取分离，显色和测量，以测量的各标准溶液的吸光度扣除空白的吸光度，与对应的标准溶液的镍含量绘制校准曲线。

注：浸出液的制备方法，参见GB/T15555.1—1995《固体废物总汞的测定冷原子吸收分光光度法》中的附录B。
