# 5.4 运输电子废弃物的工具及设备要求

5.4.1 运输电子废弃物的车辆宜采用厢式货车，且车辆的车厢、底板应平坦完好，周围的栏杆应坚实牢固。
5.4.2 运输电子废弃物的车辆在运行过程中应采取相应措施以导除静电。
5.4.3 运输阴极射线管（CRT）、荧光粉等有害物质时，应使用专用车辆、车厢或船舱等，并设有防雨设施。
5.4.4 电子废弃物的运输包装必须定期检查，如出现破损，应及时更换。
5.4.5 电子废弃物的运输企业或单位应定期将运输车辆、运输工具、罐车罐体和配载容器送质量监督部门认可的机构进行检测、检验，并取得检测、检验合格证明。
5.4.6 装卸电子废弃物的吊钩、吊环、钢丝绳和链条等吊挂用具和起重设备应在使用前检查，并定期检验，严禁降低安全系数使用。

# 5.5 运输人员的安全要求

# 5.5.1 主要负责人

电子废弃物运输企业或单位的主要负责人对本企业或单位运输安全工作负有下列职责：

a）建立、健全本企业或单位的运输生产责任制；
b）组织制定本企业或单位的安全运输规章制度和操作规程；
c）保证本企业或单位安全投入的有效实施；
d）督促、检查本企业或单位的运输安全工作，及时消除安全事故隐患；
e）组织制定并实施本企业或单位的运输安全事故应急救援预案。

# 5.5.2 安全运输管理人员

5.5.2.1 电子废弃物运输企业或单位应配备专职安全运输管理人员或设置安全管理机构。

5.5.2.2 电子废弃物运输企业或单位的安全运输管理人员对本企业或单位运输安全工作负有下列责任：

a）建立本企业或单位安全生产知识的教育和培训计划，每半年应至少组织1次从业人员的集中安全运输培训，并建立教育和培训档案；
b）对本企业或单位的危险货物运输安全进行监督和指导；
c）负责落实企业或单位各项安全运输管理制度和各岗位安全操作规程；
d）负责运输、装卸等现场安全巡视督查，并做好记录；
e）负责安全运输相关数据统计、安全防护设备和劳动保护用品配备、使用及检查；
f）负责车辆及相关设备的安全管理、检修维护，并建立检修技术档案和行车记录档案等。

5.5.2.3 安全运输管理人员发现作业现场存在安全隐患时，应及时向主要负责人报告，对违章指挥、违章操作的，应立即制止。

# 5.5.3 操作人员

5.5.3.1 电子废弃物运输车辆应每车至少配备具有与本企业或单位签订劳动合同的驾驶员和押运人员各1名。
5.5.3.2 驾驶员、押运人员应接受电子废弃物运输企业或单位有关安全知识和应急救援方面的培训，熟悉有关的安全运输规章制度和安全操作规程，掌握本岗位的安全操作技能，了解所运载的电子废弃物

# 电子废弃物的运输安全规范

# 1 范围

本标准规定了电子废弃物的运输类型及方式、运输安全要求、标记代号、运输记录和应急预案。
本标准适用于电子废弃物的运输。

# 3 术语和定义

下列术语和定义适用于本文件。

3.1

# 电子废弃物 electronic waste

废弃的电子电器产品、电子电气设备(以下简称产品和设备)及其废弃零部件、元器件和国家环境保护总局会同有关部门规定纳入电子废物管理的物品、物质。包括工业生产活动中产生的报废产品或者设备、报废的半成品和下脚料，产品或者设备维修、翻新、再制造过程产生的报废品，日常生活或者为日常生活提供服务的活动中废弃的产品或者设备，以及法律法规禁止生产或者进口的产品或者设备。

3.2

# 严密封口 tight seal

容器经过封口后，封口处不外漏固体的封闭形式。

[GB/T26493，定义3.6]

3.3

# 液密封口 liquid seal

容器经过封口后，封口处不渗漏液体的封闭形式。

[GB/T26493，定义3.7]

3.4

# 内部运输 Internal transport
