# 前言

为贯彻《中华人民共和国环境保护法》，保护环境，保障人体健康，规范土壤和沉积物中挥发性芳香烃的测定方法，制定本标准。

本标准规定了测定土壤和沉积物中挥发性芳香烃的顶空/气相色谱法。

本标准为首次发布。

# 土壤和沉积物挥发性芳香烃的测定顶空/气相色谱法

警告：试验中所使用的试剂和标准溶液为易挥发的有毒化合物，配制过程应在通风柜中进行操作；应按规定要求佩带防护器具，避免接触皮肤和衣服。

# 1 适用范围

本标准规定了测定土壤和沉积物中 12种挥发性芳香烃的顶空/气相色谱法。

本标准适用于土壤和沉积物中 12种挥发性芳香烃的测定。其它挥发性芳香烃如果通过验证也适用于本标准。

当取样量为 $2 \mathrm { \ : g }$ 时，12 种挥发性芳香烃的方法检出限为 $3 . 0 { \sim } 4 . 7 ~ \mu \mathrm { g / k g }$ ，测定下限为 $1 2 . 0 \sim$ $1 8 . 8 ~ \mu \mathrm { g / k g }$ ，详见附录 A。

# 3 方法原理

在一定的温度下，顶空瓶内样品中挥发性芳香烃向液上空间挥发，在气液固三相达到热力学动态平衡后。气相中的挥发性芳香烃经气相色谱分离，用火焰离子化检测器检测。以保留时间定性，外标法定量。

# 4 试剂和材料

4.1 实验用水：二次蒸馏水或通过超纯水制备仪制备的水。使用前需经过空白试验检验，确认在目标物的保留时间区间内没有干扰色谱峰出现。
4.2 甲醇（ $\mathrm { C H } _ { 3 } \mathrm { O H } )$ ）：农残级或相当级别。通过空白试验，确认在目标物的保留时间区间内没有干扰色谱峰出现。
4.3 氯化钠（NaCl）：优级纯。

在马弗炉（或箱式电炉）中 $4 0 0 \mathrm { { ^ \circ C } }$ 烘烤 $^ { 4 \mathrm { ~ h ~ } }$ ，置于干燥器中冷却至室温，转移至磨口玻璃瓶中保存。
4.4 磷酸（ $\mathrm { { . H _ { 3 } P O _ { 4 } } , }$ ）：优级纯。
4.5 饱和氯化钠溶液。

量取 $5 0 0 \mathrm { m l }$ 实验用水（4.1），滴加几滴磷酸（4.4）调节至 $\mathsf { p H } \leqslant 2$ ，加入 $1 8 0 ~ \mathrm { g }$ 氯化钠（4.3），溶解并混匀。于 $4 ~ \mathrm { { ^ \circ C } }$ 下避光保存，可保存 6 个月。

4.6 标准贮备液： $\rho { = } 1 0 0 0 ~ \mu \mathrm { g / m l }$ 。

挥发性芳香烃的甲醇标准溶液可直接购买有证标准溶液，也可用标准物质配制。包括苯，甲苯，乙苯，间-二甲苯，对-二甲苯，邻-二甲苯，异丙苯，苯乙烯，氯苯，1,3-二氯苯，1,4-二氯苯，1,2-二氯苯。

在 $4 ~ \mathrm { { ^ \circ C } }$ 以下避光保存或参照制造商的产品说明。使用前应恢复至室温，并摇匀。开封后冷冻密封避光可保存 $1 4 { \mathrm { ~ d ~ } }$ 。如购置高浓度标准贮备液，使用甲醇（4.2）进行适当稀释。

4.7 石英砂（ $\mathrm { S i O } _ { 2 }$ ）：分析纯，20目 ${ \sim } 5 0$ 目。

使用前需通过检验，确认无目标化合物或目标化合物浓度低于方法检出限。

4.8 载气：高纯氮气（ $\geq 9 9 . 9 9 9 \%$ ），经脱氧剂脱氧、分子筛脱水。
4.9 燃气：高纯氢气（ ${ \geqslant } 9 9 . 9 9 9 \%$ ），经分子筛脱水。
4.10 助燃气：空气，经硅胶脱水、活性炭脱有机物。

# 5 仪器和设备

5.1 气相色谱仪：具有分流/不分流进样口，可程序升温，具火焰离子化检测器（FID）。
5.2 色谱柱：石英毛细管柱， $3 0 \mathrm { m }$ （长） $\times 0 . 3 2 \ : \mathrm { m m }$ （内径） $\times 0 . 2 5 \mu \mathrm { m }$ （膜厚），固定相为聚乙二醇。也可使用其它等效毛细管柱。
5.3 自动顶空进样器：带顶空瓶（ $. 2 2 \mathrm { m l }$ ）、密封垫（聚四氟乙烯/硅氧烷）、瓶盖（螺旋盖或一次使用的压盖）。
5.4 往复式振荡器：振荡频率 150次/min，可固定顶空瓶。
5.5 天平：精度为 $0 . 0 1 \ \mathrm { g }$ 的天平。
5.6 微量注射器：5 μl、10 μl、 $2 5 \mu \mathrm { l }$ 、 $1 0 0 \mu \mathrm { l }$ 、500 μl、 $1 0 0 0 \mu \mathrm { l }$
5.7 采样器材：铁铲和不锈钢药勺。
5.8 便携式冷藏箱：容积 $2 0 \mathrm { L }$ ，温度 $4 ~ \mathrm { { ^ \circ C } }$ 以下。
5.9 棕色密实瓶： $2 ~ \mathrm { m l }$ ，具聚四氟乙烯衬垫和实芯螺旋盖。
5.10 采样瓶：具聚四氟乙烯-硅胶衬垫螺旋盖的 $6 0 \mathrm { m l }$ 或 $2 0 0 \mathrm { m l }$ 的螺纹棕色广口玻璃瓶。
5.11 一次性巴斯德玻璃吸液管。
5.12 一般实验室常用仪器和设备。

# 6 样品

# 6.1 样品的采集与保存

按照HJ/T 166的相关规定进行土壤样品的采集和保存。按照 GB 17378.3 的相关规定进行沉积物样品的采集和保存。采集样品的工具应用铁铲和不锈钢药勺。所有样品均应至少采集 3份代表性样品。

用铁铲和不锈钢药勺将样品尽快采集到采样瓶（5.10）中，并尽量填满。快速清除掉样品瓶螺纹及外表面上粘附的样品，密封样品瓶。置于便携式冷藏箱内，带回实验室。

采样瓶中的样品用于样品测定和土壤中干物质含量及沉积物含水率的测定。

注1：必要时，可在采样现场使用用于挥发性芳香烃测定的便携式仪器对样品进行浓度高低的初筛。当样品中挥发性芳香烃浓度大于 $1 0 0 0 \mu \mathrm { g / k g }$ 时，视该样品为高含量样品。

注2：样品采集时切勿搅动土壤及沉积物，以免造成有机物的挥发。

样品送入实验室后应尽快分析。若不能立即分析，在 $4 \mathrm { { } ^ { \circ } C }$ 以下密封保存，保存期限不超过7d。样品存放区域应无有机物干扰。

# 6.2 试样的制备

# 6.2.1 低含量试样

实验室内取出采样瓶（5.10），待恢复至室温后，称取 2g（精确至 $0 . 0 1 \mathrm { g }$ ）样品置于顶空瓶中，迅速向顶空瓶中加入 $1 0 . 0 \mathrm { m l }$ 饱和氯化钠溶液（4.5），立即密封，在往复式振荡器（5.4）上以 150 次/ min 的频率振荡 $1 0 \mathrm { m i n }$ ，待测。

# 6.2.2 高含量试样

高含量试样制备如下：取出采样瓶（5.10），使其恢复至室温。称取 $2 \mathrm { g }$ （精确至 $0 . 0 1 \mathrm { g }$ ）样品置于顶空瓶中，迅速加入 $1 0 . 0 \mathrm { m l }$ 甲醇（4.2），密封，在往复式振荡器（5.4）上以 150 次/ min 的频率振荡 $1 0 \mathrm { m i n }$ 。静置沉降后，用一次性巴斯德玻璃吸液管移取约 1ml 提取液至 $2 \mathrm { m l }$ 棕色密实瓶（5.9）中。该提取液可置于冷藏箱内 $4 \%$ 下保存，保存期为 $1 4 { \mathrm { ~ d ~ } }$ 。

注3：若甲醇提取液中目标化合物浓度较高，可通过加入甲醇进行适当稀释。

在分析之前将提取液恢复到室温后，向空的顶空瓶中加入 2g（精确至 $0 . 0 1 { \mathrm { ~ g } } { \cdot }$ ）石英砂（4.7）、$1 0 . 0 \mathrm { m l }$ 饱和氯化钠溶液（4.5）和 $0 . 0 1 0 { \sim } 0 . 1 0 0 ~ \mathrm { m l }$ 甲醇提取液。立即密封，在往复式振荡器（5.4）上以 150 次/ min 的频率振荡 $1 0 \mathrm { m i n }$ ，待测。

注4：若用高含量方法分析浓度值过低或未检出，应采用低含量方法重新分析样品。

# 6.3 空白试样的制备

# 6.3.1 运输空白试样

采样前在实验室将 $1 0 . 0 \mathrm { m l }$ 饱和氯化钠溶液（4.5）和 $2 \ \mathrm { g }$ （精确至 $_ { 0 . 0 1 \mathrm { ~ g ~ } }$ ）石英砂（4.7）放入顶空瓶中密封，将其带到采样现场。采样时不开封，之后随样品运回实验室，在往复式振荡器（5.4）上以 150次/ min 的频率振荡 $1 0 \mathrm { m i n }$ ，待测。

# 6.3.2 低含量空白试样

称取 $2 \mathrm { g }$ （精确至 $0 . 0 1 \ \mathrm { g }$ ）石英砂（4.7）代替样品，按照 6.2.1步骤制备低含量空白试样。

# 6.3.3 高含量空白试样

称取 $_ { 2 \mathrm { ~ g ~ } }$ （精确至 0.01 g）石英砂（4.7）代替高含量样品，按照 6.2.2 步骤制备高含量空白试样。

# 6.4 土壤干物质含量及沉积物含水率的测定

按照 HJ 613测定土壤中干物质含量；按照 GB 17378.5 测定沉积物样品的含水率。

# 8.2 土壤样品结果计算

8.2.1 低含量样品中挥发性芳香烃的含量（µg/kg），按照公式（1）进行计算。

$$
\omega = \frac {m _ {0}}{m _ {1} \times w _ {d m}} \tag {1}
$$

式中： $\omega$ —样品中目标化合物的含量， $\mu \mathrm { g / k g }$ ；

$\mathfrak { m } _ { 0 }$ —根据校准曲线计算出目标化合物的质量，ng；

$\mathfrak { m } _ { 1 }$ —样品量（湿重），g。

$w _ { d m }$ wdm 样品的干物质含量， $\%$ 。

8.2.2 高含量样品中挥发性芳香烃的含量（ $\bf { \nabla } \cdot \lVert u g / k g \rVert$ ），按照公式（2）进行计算。

$$
\omega = \frac {m _ {0} \times 1 0 . 0 \times f}{m _ {1} \times V _ {S} \times w _ {d m}} \tag {2}
$$

式中： $\omega$ ——样品中目标化合物的含量， $\mu \mathrm { g / k g }$ ；

$\mathfrak { m } _ { 0 }$ —根据校准曲线计算出目标化合物的质量，ng；

10.0 ——提取液体积，ml；

$\mathfrak { m } _ { 1 }$ —样品量（湿重），g；

$V _ { s }$ ——用于顶空测定的甲醇提取液体积，ml；

$w _ { d m }$ wdm 样品的干物质含量， $\%$ ；

$f$ ——萃取液的稀释倍数。

式中： $\omega$ ——样品中目标物的含量， $\mu \mathrm { g / k g }$ ；

$\mathfrak { m } _ { 0 }$ —根据校准曲线计算出目标化合物的质量，ng；

$\mathfrak { m } _ { 1 }$ —样品量（湿重），g；

$V _ { s }$ ——用于顶空测定的甲醇提取液体积，ml；

# 10 质量保证和质量控制

# 10.1 校准曲线

根据目标物的浓度和响应值绘制校准曲线，其相关系数应大于等于0.999，若不能满足要求，需更换色谱柱或采取其他措施，然后重新绘制校准曲线。

# 10.2 校准确认

每批样品分析前或 $2 4 \mathrm { ~ h ~ }$ 之内，利用标准曲线中间浓度点进行校准确认，目标化合物的测定值与标准值间的相对偏差应 $\leqslant 2 0 \%$ ，否则，应重新绘制校准曲线。

# 10.3 样品

10.3.1 实验室空白试验分析结果中所有待测目标化合物浓度均应低于方法检出限。否则，应查明原因，及时消除，至实验室空白测定结果合格后，才能继续进行样品分析。
10.3.2 每批样品至少应采集一个运输空白样品。其分析结果应满足空白试验的控制指标（10.3.1），否则需查找原因，排除干扰后重新采集样品分析。
10.3.3 每一批样品（最多 20 个）应测定一个空白加标样品、基体加标样品和基体加标平行样品，实验室空白加标回收率在 $8 0 . 0 \% \sim 1 2 0 \%$ 之间，基体加标样品分析结果的加标回收率应在 $3 5 . 0 \% \sim 1 1 0 \%$ 范围内，基体加标平行样品分析结果的相对偏差应该在 $20 \%$ 以内。

# 11 废物处理

实验产生的含挥发性芳香烃的危险废物应集中保管，委托有资质的相关单位进行处理。

# 12 注意事项

12.1 为了防止通过采样工具污染，采样工具在使用前要用甲醇、纯净水充分洗净。在采集其他样品时，要注意更换采样工具和清洗采样工具，以防止交叉污染。
12.2 样品的保存和运输过程中，要避免沾污，样品应放在密闭、避光的便携式冷藏箱（5.8）中冷藏贮存。
12.3 在分析过程中必要的器具、材料、药品等应事先分析确认其是否含有对分析测定有干扰目标物测定的物质。器具、材料可采用甲醇清洗，尽可能在空白中除去干扰物质。
