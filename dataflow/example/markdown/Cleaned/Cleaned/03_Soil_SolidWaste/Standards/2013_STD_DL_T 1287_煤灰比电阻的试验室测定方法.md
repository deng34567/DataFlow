# 前言

本标准由中国电力企业联合会提出。

本标准由电力行业电厂化学标准化技术委员会归口。

本标准起草单位：西安热工研究院有限公司、湖北省电力试验研究院。

本标准主要起草人：杜晓光、张太平、马筠、张律。

本标准在执行过程中的意见或建议反馈至中国电力企业联合会标准化管理中心（北京市白广路二条一号，100761）。
