# 1 适用范围

本标准适用于酸性土壤中可交换酸度的测定。

# 3 术语和定义

可交换酸度 exchangeable acidity

用中性盐溶液提取土壤，将土壤胶体上吸附的氢离子和铝离子交换下来，使之进入溶液，然后用氢氧化钠标准溶液滴定，滴定结果称为可交换酸度。

# 4 方法原理

提取原理：用适量氯化钾溶液反复淋洗土壤样品，使得土壤胶体上可交换铝和可交换氢被钾离子交换，形成氢离子和三价铝离子进入溶液。其交换过程用下式表示：

$\mathrm { H } ^ { + } .$ -|土壤胶体|- ${ \bf \partial } . { \bf A l } ^ { 3 + } { + } 3 { \bf K C l }$ |土壤胶体 $- 3 \mathrm { K } ^ { + } + \mathrm { A l } ^ { 3 + } + 3 \mathrm { C l } ^ { - } + \mathrm { H } ^ { + }$

可交换酸度的测定：提取完样品后，取一部分土壤淋洗液，用氢氧化钠标准溶液直接滴定，所得结果为可交换酸度。

可交换铝的测定：提取完样品后，另取一部分土壤提取液，加入适量氟化钠溶液，使氟离子与铝离子形成络合物， ${ \mathrm { A l } } ^ { 3 + }$ 被充分络合，再用氢氧化钠标准溶液滴定，所得结果为可交换氢。可交换酸度与可交换氢的差值为可交换铝。

# 5 试剂和材料

除非另有说明，分析时均使用符合国家标准的分析纯试剂，实验用水为新鲜煮沸蒸馏水。

# 5.1 新鲜煮沸蒸馏水

将蒸馏水在烧杯中煮沸蒸发（蒸发量 $10 \%$ ），加盖冷却后密封备用，应现用现制。

# 5.2 盐酸溶液： $c$ $c \ \mathrm { { ( H C l ) } \ = 1 . 0 m o l / L }$

取 $8 3 \mathrm { m l }$ 浓盐酸（ $\mathrm { \rho _ { \mathrm { \rho } } } \mathrm { = } 1 . 1 9 \mathrm { g / m l }$ ）用水稀释到 1L。

# 5.3 氯化钾溶液： $c$ $\mathrm { : ~ ( K C l ) ~ } { = } 1 . 0 \mathrm { m o l / L }$

称取 $7 4 . 5 5 \mathrm { g }$ 氯化钾，溶于水中，移入 1L 容量瓶中，加水稀释至标线，混匀。

# 5.4 邻苯二甲酸氢钾标准溶液： $c$ $( \mathrm { C _ { 8 } H _ { 5 } K O _ { 4 } } ) \mathrm { \Delta = } 0 . 0 1 \mathrm { m o l / L }$ $\mathrm { C _ { 8 } H _ { 5 } K O _ { 4 } }$

称取已通过 $1 0 5 ^ { \circ } \mathrm { C } \sim 1 1 0 ^ { \circ } \mathrm { C }$ 干燥的基准试剂邻苯二甲酸氢钾 0.5106g 溶于适量水中，移入 250ml

容量瓶中，加水（5.1）稀释至标线，混匀。

5.5 氢氧化钠标准溶液：c $\mathrm { ( N a O H ) = } 0 . 0 1 \mathrm { m o l / L }$

称取 $0 . 4 \mathrm { g }$ 氢氧化钠溶于适量水中，待溶液冷却后移入 1L 容量瓶，稀释至标线，混匀，贮存于聚乙烯塑料容器中。用邻苯二甲酸氢钾标准溶液（5.4）进行标定。

标定方法：吸取邻苯二甲酸氢钾标准溶液（5.4） $2 5 . 0 0 \mathrm { m l }$ 于烧杯中，在烧杯中放入搅拌子，插入电极时用氢氧化钠溶液（5.5）滴定。直到 pH值达到 $7 . 8 0 { \pm } 0 . 0 8 $ ，稳定30 秒。同时做空白试验。连续测定三次，取三次标定结果的平均值。

氢氧化钠标准溶液浓度按照公式（1）进行计算。

$$
c _ {1} = \frac {c _ {2} \times V _ {2}}{V _ {1} - V _ {0}} \tag {1}
$$

式中： $c _ { I }$ ——氢氧化钠标准溶液浓度，mol/L；

$c _ { 2 }$ 邻苯二甲酸氢钾溶液的浓度，mol/L；

$V _ { \theta }$ ——空白试验消耗氢氧化钠溶液的体积，ml；

$V _ { I }$ ——标定时消耗氢氧化钠溶液的体积，ml；

$V _ { 2 }$ ——邻苯二甲酸氢钾溶液的体积，ml。

5.6 氟化钠溶液： $c$ （NaF）=1.0mol/L

称取 $4 2 . 0 \mathrm { g }$ 氟化钠溶于水中并稀释到大约 $9 0 0 \mathrm { m l }$ ，用盐酸溶液（5.2）调节至 pH为 7.0，将溶液移入 $1 0 0 0 \mathrm { m l }$ 容量瓶中，加水稀释至标线，混匀。

5.7 石英砂：30-60 目，使用前在 $3 0 0 \mathrm { { ^ \circ C } }$ 加热 2h。

# 7 样品

7.1 样品的采集和保存

按照 HJ/T166 的相关规定进行土壤样品的采集和保存。

7.2 样品的风干和制备

按照 HJ/T166 的要求制备风干样品。将风干样品过孔径 $2 \mathrm { m m }$ 土壤筛，充分搅拌混匀，采用四分法取其两份，一份交样品库存放，另一份备用。

7.3 试样的制备

称取 $5 . 0 0 \mathrm { g }$ 风干的土样，放在已铺好滤纸的漏斗内，用氯化钾溶液（5.3）少量多次地淋洗，每次加入氯化钾溶液必须待漏斗中的滤液滤干后再进行。滤液承接在 $2 5 0 \mathrm { m L }$ 容量瓶中，近刻度时用 1氯化钾溶液（5.3）定容。

7.4 空白试样的制备

用石英砂代替土壤样品，按照与试样制备相同步骤，制备空白样品提取液。

# 7.5 含水率的测定

按照 HJ613 的规定测定其含水率。

# 8 分析步骤

# 8.1 可交换酸度的测定

# 8.1.1 测定

移取 $1 0 0 \mathrm { m l }$ 试样提取液至烧杯中，煮沸 5min，使可能存在于溶液中的二氧化碳挥发，冷却至室温，以 pH 计为指示，用氢氧化钠溶液滴定至 pH 值为 $7 . 8 0 { \pm } 0 . 0 8 $ ，记录消耗氢氧化钠溶液体积$\mathrm { V } _ { 1 }$ 的毫升数。

# 8.1.2 空白试验

用 8.1.1 方法同时滴定 $1 0 0 \mathrm { m l }$ 空白试样提取液（7.4），记录消耗氢氧化钠溶液体积 V空的用量。

# 8.2 可交换氢的测定

# 8.2.1 测定

移取 $1 0 0 \mathrm { m l }$ 试样提取液至烧杯中，加入 $2 . 5 \mathrm { m l }$ 氟化钠溶液（5.6），煮沸 $5 \mathrm { { m i n } }$ ，赶出二氧化碳，冷却至室温，以 pH 计为指示，用氢氧化钠溶液滴定至 pH 值为 $7 . 8 0 { \pm } 0 . 0 8 $ ，记录消耗氢氧化钠溶液体积 $\mathrm { V } _ { 2 }$ 的毫升数。

# 8.2.2 空白试验

用 8.2.1 方法同时滴定 $1 0 0 \mathrm { m l }$ 空白样品提取液（7.4），记录消耗氢氧化钠溶液体积 $\mathrm { V } _ { 0 }$ 的毫升数。

# 9 结果计算与表示

# 9.1 结果计算

土壤样品中的可交换酸度按照公式（2）进行计算。

$$
E _ {A} = \frac {\left(V _ {1} - V _ {\text {空}}\right) \times c _ {\text {N a O H}} \times 1 0 0 0 \times V}{V _ {s} \times m} \times \frac {1 0 0 + w}{1 0 0} \tag {2}
$$

式中： $E _ { A }$ —烘干土壤中可交换酸度，mmol/kg；

$V _ { I }$ —直接滴定土壤样品消耗氢氧化钠体积，ml；

V 空—空白样品所消耗氢氧化钠体积，ml；

CNaOH—氢氧化钠溶液浓度，mol/L；

V—提取液最终定容体积，ml；

$V _ { S }$ —滴定时移取的提取液体积，ml；

m—风干土质量，g；

w—风干土壤含水率，质量分数。

土壤样品中的可交换氢和可交换铝，按照公式（3）和（4）进行计算。

$$
E _ {H ^ {+}} = \frac {\left(V _ {2} - V _ {0}\right) \times c _ {N a O H} \times 1 0 0 0 \times V}{V _ {s} \times m} \times \frac {1 0 0 + w}{1 0 0} \tag {3}
$$

$$
E _ {A l} = E _ {A} - E _ {H ^ {+}} \tag {4}
$$

式中： ${ E _ { H } } ^ { + } .$ —土壤样品的可交换酸氢，mmol/kg；

$E _ { A l }$ —土壤样品的可交换铝，mmol/kg；

$V _ { 2 }$ ——加入氟化钠后土样消耗氢氧化钠体积，ml；

$V _ { \theta }$ ——加入氟化钠后空白样品消耗氢氧化钠体积，ml；

其他参数的含义见公式（2）。

# 9.2 结果表示

当测定结果小于 $1 . 0 \ \mathrm { m m o l / k g }$ 时，保留到小数点后两位；大于等于 $1 . 0 \ \mathrm { m m o l / k g }$ 时，保留三位有效数字。

# 10 精密度

6 个实验室对可交换酸度分别为 $4 . 5 2 \mathrm { m m o l / k g }$ 和 $7 2 . 6 \ : \mathrm { m m o l / k g }$ 的统一样品进行测定：实验室内相对标准偏差分别为： $2 . 0 \% \sim 4 . 2 \%$ ， $0 . 5 \% \sim 1 . 3 \%$ ；实验室间相对标准偏差分别为： $1 . 9 \% \sim 4 . 1 \%$ ，$0 . 4 \% \sim 1 . 0 \%$ ；重复性限分别为： $0 . 1 3 \mathrm { m m o l / k g }$ ，0.70mmol/kg；再现性限分别为： $0 . 1 9 \mathrm { m m o l / k g }$ ，1.20mmol/kg。

# 11 质量保证和质量控制

11.1 每批样品至少做 2 个空白试验。
11.2 每批样品至少做 $10 \%$ 平行样品。当测定值 $\leq 1 0 . 0 \mathrm { m m o l / k g }$ ，最大允许相对偏差为 $\pm 2 0 \%$ ；测定值在 $1 0 . 0 { \sim } 1 0 0 \mathrm { m m o l / k g }$ ，最大允许相对偏差为 $\pm 1 0 \%$ ；测定值 $\geq 1 0 0 \mathrm { m m o l / k g }$ ，最大允许相对偏差为 $\pm 5 \%$ 。
11.3 pH 计使用前必须用 pH 标准缓冲溶液进行校正。

# 12 注意事项

12.1 土壤样品浸提后应尽快滴定，避免长时间暴露在空气中，造成误差。
12.2 土壤样品的保存与风干过程应在通风无污染的环境中。
12.3 控制滴定速度，应尽快稳定至 pH 值在 7.80 左右。
