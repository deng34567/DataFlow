# 前言

为贯彻《中华人民共和国环境保护法》，保护环境，保障人体健康，规范土壤中有效磷的测定方法，制定本标准。

本标准规定了测定土壤中有效磷的碳酸氢钠浸提-钼锑抗分光光度法。

本标准为首次发布。

本标准的附录 A为资料性附录。

本标准由环境保护部科技标准司组织制订。

# 土壤有效磷的测定碳酸氢钠浸提-钼锑抗分光光度法

# 1 适用范围

本标准规定了测定土壤中有效磷的碳酸氢钠浸提-钼锑抗分光光度法。

本标准适用于石灰性和中性土壤中有效磷的测定。

当取样量为 $2 . 5 0 \ \mathrm { g }$ ，使用 $5 0 \mathrm { m l }$ 碳酸氢钠溶液浸提，采用 $1 0 \mathrm { m m }$ 比色皿时，本方法检出限为 $0 . 5 \mathrm { m g / k g }$ ，测定下限为 $2 . 0 \mathrm { m g / k g }$ 。

# 2 规范性引用文件

本标准内容引用了下列文件或其中的条款。凡是不注明日期的引用文件，其有效版本适用于本标准。

HJ/T166 土壤环境监测技术规范

HJ 613 土壤干物质和水分的测定重量法

NY/T 395-2012 农田土壤环境质量监测技术规范

# 3 术语和定义

土壤有效磷 available phosphorous

缩写为 A-P，也称为速效磷，在植物生长期内能够被植物根系吸收的土壤磷，即在本标准规定的条件下浸提出来土壤溶液中的磷、弱吸附态磷、交换性磷和易溶性固体磷酸盐等。

# 4 方法原理

用 $0 . 5 \mathrm { m o l / L }$ 碳酸氢钠溶液（ $\mathsf { p H } { = } 8 . 5$ ）浸提土壤中的有效磷。浸提液中的磷与钼锑抗显色剂反应生成磷钼蓝，在波长 $8 8 0 \mathrm { n m }$ 处测量吸光度。在一定浓度范围内，磷的含量与吸光度值符合朗伯-比尔定律。

# 5 干扰和消除

当浸提液中砷含量大于 $2 \mathrm { m g / L }$ 有干扰，可用硫代硫酸钠除去；硫化钠含量大于 $2 \mathrm { m g / L }$ 有干扰，在酸性条件下通氮气可以除去；六价铬大于 $5 0 \mathrm { m g / L }$ 有干扰，用亚硫酸钠除去；铁含量为 $2 0 \mathrm { m g / L }$ ，使结果偏低 $5 \%$ 。

# 6 试剂和材料

除非另有说明，分析时均使用符合国家标准的分析纯化学试剂。实验用水为新制备的去离子水或蒸馏水。

6.1 硫酸： $\rho$ $( \mathrm { H } _ { 2 } \mathrm { S O } _ { 4 } ) = 1 . 8 4 \ : \mathrm { g } / \mathrm { m l } .$
6.2 硝酸： $\rho$ $\mathrm { ( H N O _ { 3 } ) } = 1 . 5 1 \ \mathrm { g / m l } .$ 。
6.3 冰乙酸： $\rho$ $( \mathrm { C } _ { 2 } \mathrm { H } _ { 4 } \mathrm { O } _ { 2 } ) = 1 . 0 4 9 \ : \mathrm { g / m l }$ 。
6.4 磷酸二氢钾（ ${ \mathrm { K H } } _ { 2 } { \mathrm { P O } } _ { 4 } .$ ）：优级纯。

取适量磷酸二氢钾于称量瓶中，置于 $1 0 5 \mathrm { { ‰} }$ 烘干 2h，干燥箱内冷却，备用。

6.5 氢氧化钠溶液： $\omega$ $( \mathrm { N a O H } ) = 1 0 \%$ 。

称取 $1 0 \ \mathrm { g }$ 氢氧化钠溶于水中，用水稀释至 $1 0 0 \mathrm { m l }$ ，贮于聚乙烯瓶中。

6.6 硫酸溶液： $c$ （ $( 1 / 2 \mathrm { H } _ { 2 } \mathrm { S O } _ { 4 } ) \mathrm { \Omega } = 2 \mathrm { m o l } / \mathrm { L } \mathrm { \Omega }$ 。

于 $8 0 0 \mathrm { m l }$ 水中，在不断搅拌下缓慢加入 $5 5 ~ \mathrm { m l }$ 硫酸（6.1），待溶液冷却后，加水至 $1 0 0 0 \mathrm { m l }$ ，混匀。

6.7 硝酸溶液： $1 + 5 \ \mathrm { ~ ( ~ } V / V )$ ）。

用硝酸（6.2）配制。

6.8 浸提剂： $c ~ ( \mathrm { N a H C O _ { 3 } } ) ~ { = } 0 . 5 \mathrm { m o l / L }$ $: ~ ( \mathrm { N a H C O } _ { 3 }$

称取 $4 2 . 0 \ \mathrm { g }$ 碳酸氢钠溶于约 $8 0 0 \mathrm { m l }$ 水中，加水稀释至约 $9 9 0 ~ \mathrm { m l }$ ，用氢氧化钠溶液（6.5）调节至 $\mathrm { p H } { = } 8 . 5$ （用 $\mathrm { p H }$ 计测定），加水定容至 1L，温度控制在 $2 5 { \pm } 1 ^ { \circ } \mathrm { C }$ 。贮存于聚乙烯瓶中，该溶液应在 4h 内使用。

注 1：浸提剂温度需控制在 $2 5 { \pm } 1 \mathrm { { } ^ { \circ } C }$ 。具体控制时，最好有 1 小间恒温室，冬季除室温要维持 $2 5 \mathrm { { ^ \circ C } }$ 外，还需将去离子水事先加热至 $2 6 { \sim } 2 7 ^ { \circ } \mathrm { C }$ 后再进行配制。

6.9 酒石酸锑钾溶液： $\rho$ （K（SbO） $\mathrm { C _ { 4 } H _ { 4 } O _ { 6 } { \cdot } 1 / 2 H _ { 2 } O ) \ = 5 \ g / L }$ 。

称取 $0 . 5 \mathrm { g }$ 酒石酸锑钾溶于 $1 0 0 ~ \mathrm { m l }$ 水中。

6.10 钼酸盐溶液

量取 $1 5 3 \mathrm { m l }$ 硫酸（6.1）缓慢注入约 $4 0 0 ~ \mathrm { m l }$ 水中，搅匀，冷却。另取 $1 0 . 0 \mathrm { g }$ 钼酸铵溶于 $3 0 0 \mathrm { m l }$ 约 $6 0 ^ { \circ } \mathrm { C }$ 的水中，冷却。然后将该硫酸溶液缓慢注入钼酸铵溶液中，搅匀，再加入 $1 0 0 \mathrm { m l }$ 酒石酸锑钾溶液（6.9），最后用水定容至 1L。该溶液中含 $1 0 \mathrm { g / L }$ 钼酸铵和 $2 . 7 5 \mathrm { m o l / L }$ 硫酸。该溶液贮存于棕色瓶中，可保存一年。

6.11 抗坏血酸溶液： $\omega$ $( C _ { 6 } \mathrm { H } _ { 8 } \mathrm { O } _ { 6 } ) \ = 1 0 \%$

称取 $1 0 \ \mathrm { g }$ 抗坏血酸溶于水中，加入 $0 . 2 \ \mathrm { g }$ 乙二胺四乙酸二钠 EDTA和 $8 ~ \mathrm { m l }$ 冰乙酸（6.3），加水定容至 $1 0 0 \mathrm { m l }$ 。该溶液贮存于棕色试剂瓶中，在 $4 \mathrm { { } ^ { \circ } C }$ 下可稳定 3 个月。如颜色变黄，则弃去重配。

6.12 磷标准贮备溶液： $\rho$ $\mathrm { { ( P ) = } 1 0 0 \ m g / L } \mathrm { { . } }$ 。

称取 $0 . 4 3 9 4 \mathrm { g }$ 磷酸二氢钾（6.4）溶于约 $2 0 0 \mathrm { m l }$ 水中，加入 $5 \mathrm { m l }$ 硫酸（6.1），然后移至 $1 0 0 0 \mathrm { m l }$ 容量瓶中，加水定容，混匀。该溶液贮存于棕色试剂瓶中，有效期为 1 年。或直接购买市售有证标准物质。

6.13 磷标准使用液： $\rho$ $\mathrm { ~ ( ~ P ~ ) ~ } = 5 . 0 0 \mathrm { m g / L }$ 。

量取 $5 . 0 0 \mathrm { m l }$ 磷标准贮备溶液（6.12）于 $1 0 0 ~ \mathrm { m l }$ 容量瓶中，用浸提剂（6.8）稀释至刻度。临用现配。

6.14 指示剂：2，4-二硝基酚或 2，6-二硝基酚 $\mathrm { ( C _ { 6 } H _ { 4 } N _ { 2 } O _ { 5 } ) }$ ， $\omega { = } 0 . 2 \%$ 。

称取 $0 . 2 \mathrm { g } 2$ ，4-二硝基酚或 2，6-二硝基酚溶于 $1 0 0 \mathrm { m l }$ 水中，该溶液贮存于玻璃瓶中。

# 7 仪器和设备

实验中的玻璃器皿需先用无磷洗涤剂洗净，再用硝酸溶液（6.7）浸泡 $2 4 \mathrm { h }$ ，使用前再依次用自来水和去离子水洗净。

7.1 分光光度计：配备 $1 0 \mathrm { m m }$ 比色皿。
7.2 恒温往复振荡器：频率可控制在 $1 5 0 { \sim } 2 5 0 \mathrm { r / m i n }$ 。
7.3 土壤样品粉碎设备：粉碎机、玛瑙研钵。
7.4 分析天平：精度为 $0 . 0 0 0 1 \mathrm { g }$
7.5 土壤筛：孔径 $1 \mathrm { m m }$ 或 20 目尼龙筛。
7.6 具塞锥形瓶： $1 5 0 \mathrm { m l }$ 。
7.7 一般实验室常用仪器和设备。
7.8 滤纸：经检验不含磷的滤纸。

# 8 样品

# 8.1 采集与保存

按 HJ/T166 的相关规定进行采集和保存土壤样品。

# 8.2 试样的制备

试样的制备按 NY/T 395-2012《农田土壤环境质量监测技术规范》进行土壤处理和制备。

# 8.3 干物质含量的测定

准确称取适量试样（8.2），参照 HJ 613 测定样品干物质的含量。

# 9 分析步骤

# 9.1 试料的制备

称取 $2 . 5 0 \mathrm { g }$ 试样（8.2），置于干燥的 $1 5 0 \mathrm { m l }$ 具塞锥形瓶中，加入 $5 0 . 0 \mathrm { m l }$ 浸提剂（6.8），塞紧，置于恒温往复振荡器上，在 $2 5 { \pm } 1 ^ { \circ } \mathrm { C }$ 下以 $1 8 0 { \sim } 2 0 0 ~ \mathrm { r / m i n }$ 的振荡频率振荡 $3 0 { \pm } 1 \mathrm { m i n }$ ，立即用无磷滤纸（7.8）过滤，滤液应当天分析。

注 2：浸提时最好有 1 小间恒温室，冬季应先开启空调，待室温达到 $2 5 \mathrm { { ^ \circ C } }$ ，且恒温往复振荡器内温度达到 $2 5 \mathrm { { ^ \circ C } }$ 后，再打开振荡器进行振荡计时。

# 9.2 校准

分别量取 0，1.00，2.00，3.00，4.00，5.00， $6 . 0 0 \mathrm { m l }$ 磷标准使用液（6.13）于 7 个 $5 0 \mathrm { m l }$ 容量瓶中，用浸提剂（6.8）加至 $1 0 . 0 \mathrm { m l }$ 。分别加水至 $1 5 { \sim } 2 0 \mathrm { m l }$ 左右，再加入 1 滴指示剂（6.14），然后逐滴加入硫酸溶液（6.6）调至溶液近无色，加入 $0 . 7 5 \mathrm { m l }$ 抗坏血酸溶液（6.11），混匀，30秒后加 5ml 钼酸盐溶液（6.10），用水定容至 $5 0 \mathrm { m l }$ ，混匀。此标准系列中磷浓度依次为 0.00，0.10，0.20，0.30，0.40，0.50，0.60mg/L。

注3：上述操作过程中，会有 $\mathrm { C O } _ { 2 }$ 气泡产生，应缓慢摇动容量瓶，勿使气泡溢出瓶口。

将上述容量瓶置于室温下放置 $3 0 \mathrm { { m i n } } \cdot$ （若室温低于 $2 0 \mathrm { { ^ \circ C } }$ ，可在 $2 5 { \sim } 3 0 ^ { \circ } \mathrm { C }$ 水浴中放置 $3 0 \mathrm { { m i n } ) }$ ）。用 $1 0 \mathrm { m m }$ 比色皿在 $8 8 0 \mathrm { n m }$ 波长处，室温高于 $2 0 \mathrm { { ^ \circ C } }$ 的环境条件下比色，以去离子水为参比，分别测量吸光度。以试剂空白校正吸光度为纵坐标，对应的磷浓度（ $\mathrm { m g / L }$ ）为横坐标，绘制校准曲线。

# 9.3 测定

量取 $1 0 . 0 \mathrm { m l }$ 试液（9.1）于干燥的 $5 0 \mathrm { m l }$ 容量瓶中。然后按照与校准（9.2）相同操作步骤进行显色和测量。

注4：试料中的含磷量较高时，可适当减少试料体积，用浸提剂（6.8）稀释至 $1 0 . 0 \mathrm { m l }$ 。

# 9.4 实验室空白试验

不加入土壤试样，按照 9.1、9.3 相同操作步骤进行显色和测量。

# 10 结果计算与表示

# 10.1 结果计算

土壤样品中有效磷的含量 $\omega$ （ $\mathrm { m g / k g ) }$ ），按照公式（1）进行计算。

$$
\omega = \frac {\left[ \left(A - A _ {0}\right) - a \right] \times V _ {1} \times 5 0}{b \times V _ {2} \times m \times w _ {d m}} \tag {1}
$$

式中： $\omega$ —土壤样品中有效磷的含量， $\mathrm { m g / k g }$ ；

A—试料吸光度值；

$A _ { \boldsymbol { \sigma } }$ ——空白试验的吸光度值；

a—校正曲线的截距；

$V _ { 1 }$ —试料体积， $5 0 \mathrm { m l }$

50——显色时定容体积，ml；

$^ b$ ——校准曲线的斜率；

$V _ { 2 }$ —吸取试料体积，ml；

m—试样量， $2 . 5 0 \mathrm { g }$

$w _ { d m }$ wdm 土壤的干物质含量（质量分数）， $\%$ 。

# 10.2 结果表示

测定结果小数位数与方法检出限保持一致，最多保留三位有效数字。

# 11 精密度和准确度

# 11.1 精密度

6 家实验室分别对土壤有效磷含量为 $4 . 6 \mathrm { m g / k g }$ ， $1 3 . 8 ~ \mathrm { m g / k g }$ ， $2 3 . 3 ~ \mathrm { m g / k g }$ 的统一样品进行了测定，实验室内相对标准偏差分别为： $2 . 5 \% { \sim } 4 . 3 \%$ 、 $1 . 1 \% { \sim } 5 . 2 \%$ 、 $0 . 7 \% \sim 1 . 9 \%$ ；实验室间相对标准偏差分别为： $3 . 7 \%$ 、 $3 . 2 \%$ 、 $1 . 0 \%$ ；重复性限为： $0 . 2 \mathrm { m g / k g }$ 、 $0 . 2 \mathrm { m g / k g }$ 、 $0 . 2 \mathrm { m g / k g }$ ；再现性限为： $0 . 5 ~ \mathrm { { m g / k g } }$ 、 $1 . 2 \mathrm { m g / k g }$ 、 $0 . 7 \mathrm { m g / k g }$ 。

# 11.2 准确度

6 家实验室分别对土壤有效磷含量为（ $1 3 . 8 { \pm } 2 . 3 $ ） $\mathrm { m g / k g }$ 和 $( 2 3 . 3 { \pm } 1 . 4 ) \mathrm { m g / k g }$ 的有证标准物质进行了测定，相对误差分别为： $- 1 0 . 0 \% \sim - 1 . 4 \%$ 、 $- 4 . 7 \% \sim - 2 . 1 \%$ ；相对误差最终值分别为：（ $- 4 . 8 { \pm } 5 . 9$ ） $\%$ 、（ $- 3 . 7 { \pm } 2 . 0 $ ） $\%$ 。

# 12 质量保证和质量控制

12.1 校准曲线的相关系数应大于等于 0.999。
12.2 每批样品应做两个空白试验，其测试结果应低于检测下限。
12.3 每批样品应至少测定 $10 \%$ 的平行双样，样品量少于 10 个时，应至少测定一个平行双样。平行样测定结果允许差应满足一定的要求（见附录 A）。
12.4 每批样品应分析一个有证标准物质，其测定值应在保证值范围内。
12.5 每批样品需做校准曲线。

# 13 注意事项

13.1 所有的采样仪器和设备，分析仪器和设备经处理后都应不含磷。试验中使用的玻璃器皿可用（1+5）盐酸溶液浸泡 2h，或用不含磷的洗涤剂清洗。比色皿用后应以稀硝酸或铬酸洗液浸泡片刻，以除去吸附的钼蓝有色物质。
13.2 由于浸提出来的有效磷受浸提液浓度、水土比例、振荡时间、温度等的影响，建议在有温度控制的实验室完成。
