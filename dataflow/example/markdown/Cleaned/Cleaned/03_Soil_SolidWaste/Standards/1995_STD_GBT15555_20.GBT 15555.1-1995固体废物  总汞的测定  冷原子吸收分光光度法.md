# 中华人民共和国国家标准

# 固体废物总汞的测定

# 冷原子吸收分光光度法

GB/T15555.1—1995

Solid waste-Determination of total mercury

Cold atomic absorption spectrometry

# 1 主题内容与适用范围

1.1本标准规定了测定固体废物浸出液中总汞的高锰酸钾-过硫酸钾消解冷原子吸收分光光度法。
1.2 本标准方法适用于固体废物浸出液中总汞的测定。
1.2.1 在最佳条件下（测汞仪灵敏度高，基线漂移及试剂空白值极小），当试样体积为 $200~\mathrm{mL}$ 时，最低检出浓度可达 $0.05\mu \mathrm{g / L}$ 。在一般情况下，测定范围为 $0.2\sim 50~\mu \mathrm{g / L}$ 。
1.2.2 干扰

碘离子浓度等于或大于 $3.8\mathrm{mg / L}$ 时明显影响精密度和回收率。若有机物含量较高，规定的消解试剂最大量不足以氧化样品中的有机物，则方法不适用。

# 2 原理

在硫酸-硝酸介质及加热条件下，用高锰酸钾和过硫酸钾等氧化剂，将试液中的各种汞化合物消解，使所含的汞全部转化为二价无机汞。

用盐酸羟胺将过量的氧化剂还原，在酸性条件下，再用氯化亚锡将二价汞还原成金属汞。

在室温下通入空气或氮气，使金属汞气化，通入冷原子吸收测汞仪，在 $253.7\mathrm{nm}$ 处测定吸光度值。

# 3 试剂

除非另有说明，分析时均使用符合国家标准，或专业标准的试剂。

3.1无汞蒸馏水：二次重蒸馏水或电渗析去离子水可满足要求。也可将蒸馏水用优级纯盐酸酸化至 $\mathrm{pH}3$ ，然后通过硫基棉纤维管(3.2)除汞。
3.2 疏基棉纤维除汞管：在内径 $6\sim 8\mathrm{mm}$ ，长 $100\mathrm{mm}$ 左右，一端拉细的玻璃管中，或 $500~\mathrm{mL}$ 分液漏斗放液管中，填充 $0.1\sim 0.2\mathrm{g}$ 疏基棉纤维(3.3)，将待净化的试剂或蒸馏水以 $10~\mathrm{mL / min}$ 的速度流过 $1\sim 2$ 次即可除尽汞。
3.3 疏基棉纤维：于棕色广口瓶中，依次加入巯基乙酸 $\mathrm{(CH_2SHCOOH)100mL}$ 、乙酸酐 $60~\mathrm{mL}$ 、 $36\%$ 乙酸 $40~\mathrm{mL}$ 、浓 $\mathrm{H}_2\mathrm{SO}_40.3\mathrm{mL}$ ，充分混匀，冷却至室温后，加入 $30\mathrm{g}$ 长纤维脱脂棉，使之浸泡完全，用水冷却，待反应热散去后，加盖放入 $40\pm 2^{\circ}\mathrm{C}$ 的烘箱中 $2\sim 4\mathrm{d}$ 后取出。用砂芯过滤漏斗抽滤，用无汞重蒸馏水(3.1)充分洗涤至中性后，摊开于 $30\sim 35^{\circ}\mathrm{C}$ 烘干，放于棕色广口磨口瓶中，避光，低温下保存待用。
3.4 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ ， $\rho = 1.84\mathrm{g / mL}$ ，优级纯。
3.5 硝酸 $\mathrm{(HNO_3)}$ ， $\rho = 1.42\mathrm{g / mL}$ ，优级纯。
3.6 盐酸 $(\mathrm{HCl})$ ， $\rho = 1.19\mathrm{g / mL}$ ，优级纯。

3.7 重铬酸钾 $(\mathrm{K}_2\mathrm{Cr}_2\mathrm{O}_7)$ ，优级纯。
3.8 高锰酸钾 $\mathrm{(KMnO_4)}$ 溶液， $5\%$ ：将 $50\mathrm{gKMnO_4}$ （优级纯，若纯度达不到优级纯，要求重结晶提纯 $\mathrm{(KMnO_4)}$ 用水溶解后，定容至 $1000~\mathrm{mL}$ ，贮于棕色瓶中。
3.9 过硫酸钾 $\mathrm{(K_2S_2O_8)}$ 溶液， $5\%$ ：将 $5\mathrm{g}$ 过硫酸钾溶于少量水中，定容至 $100\mathrm{mL}$ 。现配现用。
3.10 盐酸羟胺 $(\mathrm{NH_2OH\cdot HCl})$ 溶液， $20\%$ ：将 $20\mathrm{g}$ 盐酸羟胺用水溶解后稀释至 $100\mathrm{mL}$ 。该溶液不可久贮。
3.11 氯化亚锡 $(\mathrm{SnCl}_2)$ 溶液， $20\%$ ：将 $20\mathrm{gSnCl}_2$ 加入 $20\mathrm{mL}\mathrm{HCl}$ (3.6)中，微微加热溶解，冷却后用水稀释至 $100\mathrm{mL}$ 。以 $2.5\mathrm{L} / \mathrm{min}$ 的流速通入氮气约 $2\mathrm{min}$ 除汞，加几颗锡粒密塞保存。
3.12 经碘化处理的活性碳：称取1份重量碘、2份重量碘化钾和20份重量水，在烧杯中制成溶液，加入约10份重量的粒状活性碳（工业用， $\phi 3\mathrm{mm}$ ，长 $3\sim 7\mathrm{mm}$ )，用力搅拌至溶液脱色后，用G1号砂芯漏斗滤出活性碳，在约 $105^{\circ}\mathrm{C}$ 烘干 $1\sim 2\mathrm{h}$ ，备用。
3.13 汞标准固定液（称固定液）， $0.5\mathrm{g/L}$ ：将 $0.5\mathrm{g}$ 重铬酸钾（3.7）溶于 $950\mathrm{mL}$ 水中，再加入 $50\mathrm{mL}$ $\mathrm{HNO}_3$ （3.5）。
3.14 稀释液：将重铬酸钾(3.7) $0.2\mathrm{g}$ 溶于 $900~\mathrm{mL}$ 水中，加入 $\mathrm{H}_2\mathrm{SO}_4$ (3.4) $28~\mathrm{mL}$ ，冷却后定容至 $1000~\mathrm{mL}$ 。
3.15 汞标准贮备液， $100\mu \mathrm{g / mL}$ ：称取在硅胶干燥器中放置过夜的氯化汞 $(\mathrm{HgCl_2})0.1354\mathrm{g}$ ，用固定液（3.13)溶解后转移至 $1000~\mathrm{mL}$ 容量瓶中，再用固定液定容，摇匀。
3.16 汞中间标准溶液， $10\mu \mathrm{g / mL}$ ：吸取汞标准贮备液(3.15) $10.0\mathrm{mL}$ ，用固定液(3.13)定容至 $100\mathrm{mL}$ 。
3.17 汞标准使用溶液， $0.10 \mu \mathrm{g} / \mathrm{mL}$ ：吸取汞标准中间液(3.16) $10.0 \mathrm{~mL}$ ，用固定液(3.13)定容至 $1000 \mathrm{~mL}$ ，摇匀。于室温下阴凉处保存。通常可使用3个月。
3.18 变色硅胶：干燥用。
3.19 洗液：将 $10\mathrm{g}$ 重铬酸钾溶于 $9\mathrm{L}$ 蒸馏水中，加入 $1000\mathrm{mL}\mathrm{HNO}_3$

# 4 仪器

一般实验室器皿及下列专用仪器及器材：

4.1 测汞仪。
4.2 汞还原器，容积分别有 $25,50,100\mathrm{mL}$ ，具有磨口，带有莲蓬形多孔吹气头的翻气瓶。
4.3 U形管， $\phi 15\times 110\mathrm{mm}$ ，内填 $60\sim 80  \mathrm{mm}$ 的普通粒状变色硅胶(3.18)。
4.4 三通阀。
4.5 汞吸收装置： $250\mathrm{mL}$ 玻璃制干燥塔，内填经碘化处理的活性碳(3.12)。仪器的净化系统，可根据不同测汞仪的特点及具体条件进行连接。更换U形管中的硅胶。

# 5 步骤

5.1 样品的保存：采用表面光洁的硬质玻璃容器，在接收浸出液的器皿中应预先加入适量的固定液（3.13）。样品应在 $40^{\circ}\mathrm{C}$ 下保存，最长不超过28天。
5.2 试样的制备：本法推荐用高锰酸钾-过硫酸钾消解处理试样。

移取 $10\sim 50\mathrm{mL}$ 试液(视其中汞含量而定)于 $125~\mathrm{mL}$ 的锥形瓶中，若取样量不足 $50~\mathrm{mL}$ 时，应补充适量无汞蒸馏水(3.1)至约 $50~\mathrm{mL}$ 。依次加入硫酸(3.4) $1.5\mathrm{mL}$ ，高锰酸钾溶液(3.8) $4\mathrm{mL}$ （如果在 $5\mathrm{min}$ 内紫色褪去，应补加适量的高锰酸钾溶液，使溶液维持紫色不褪)， $4\mathrm{mL}$ 过硫酸钾溶液(3.9)。插入小漏斗，置于沸水中使试液在近沸状态保温 $1\mathrm{h}$ （近沸保温法)，取下冷却。或者向试液中加数粒玻璃珠或者沸石，插入小漏斗，擦干瓶底，在电热板上加热煮沸 $10\mathrm{min}$ （煮沸法)。取下冷却。

在临测定时，边摇边滴加盐酸羟胺溶液(3.10)，直至刚好使过剩的高锰酸钾褪色及生成的二氧化锰全部溶解为止。转移入 $100~\mathrm{mL}$ 容量瓶中，用稀释液(3.14)定容。

5.3 空白试样：每分析一批试样，应同时用无汞蒸馏水代替浸出液试样，按上述试样处理(5.2)步骤，制备二份空白试料。
5.4 校准曲线的绘制

取 $100\mathrm{mL}$ 容量瓶8个，准确吸取汞标准使用溶液(3.17)0、0.50、1.00、1.50、2.00、2.50、3.00和 $4.00\mathrm{mL}$ 注入容量瓶中，每个容量瓶中加入适量固定液补足至 $4.0\mathrm{mL}$ ，加稀释液至标线，摇匀。按(5.5)测量试料步骤逐一进行测量。

以经过空白校正的各测量值为纵坐标，以相应标准溶液的汞浓度 $(\mu \mathrm{g} / \mathrm{L})$ 为横坐标绘制校准曲线。5.5测定：取出汞还原器吹气头，逐个吸取 $10.0\mathrm{mL}$ 经(5.2)处理后的试样或空白溶液(5.3)注入汞还原器中，加入氯化亚锡(3.11) $1\mathrm{mL}$ ，迅速插入吹气头，将三通阀旋至“进样”处，使载气通入汞还原器，记下检测表头的最大读数或记录仪上的峰高。待指针或记录笔重新回零后，将三通阀旋至“校零”处，取出吹气头，弃去废液，用水冲洗汞还原器二次，再用稀释液洗涤一次（氧化可能残留的 $\mathrm{Sn}^{2 + }$ )，仪器读数恢复到零，然后进行另一份试料的测定。

# 6 结果表示

浸出液中(汞)浓度 $c(\mu \mathrm{g} / \mathrm{L})$ 按下式计算：

$$
c (\mu \mathrm {g} / \mathrm {L}) = c _ {1} \times \frac {V _ {0}}{V} \quad \dots \
$$

式中： $c_{1}$ ——被测试料中汞的浓度， $\mu \mathrm{g} / \mathrm{L}$

$V_{0}$ ——制样时定容体积， $\mathrm{mL}$

$V$ ——试料的体积， $\mathrm{mL}$ 。

# 7 精密度和准确度

7.1 可以参考国家标准GB7468《水质总汞的测定冷原子吸收分光光度法》。
7.2室内对汞含量为 $0.081\mu \mathrm{g} / \mathrm{L}$ 的锑渣浸出液和汞含量为 $3.0\mu \mathrm{g} / \mathrm{L}$ 铋渣浸出液，分别平行测定11次，其相对标准偏差分别为 $3.9\%$ 和 $2.4\%$ 。对电镀污泥、尾矿渣、锑渣和砷钙渣等8种不同废渣浸出液进行加标试验(加标量为样品含量的 $0.5\sim 0.8$ 倍)，其回收率在 $95\% \sim 103\%$ 之间。

# 注意事项

（参考件）

A1 当采用抽气或吹气法把汞蒸汽带入吸收池时，载气流速太大，会稀释吸收池内汞蒸汽浓度；太小，导致气化速度减慢，均会降低灵敏度。一般以 $0.7 \sim 1.2 \mathrm{~L} / \mathrm{min}$ 为宜。载气流量应恒定。
A2 温度对测定灵敏度有影响。当室温低于 $10^{\circ}\mathrm{C}$ 时不利于汞的挥发，灵敏度较低。并要注意标准溶液及试样溶液温度的一致性。
A3 反应瓶体积和气液比对测定也有影响。反应瓶体积大小应根据测定试样体积而定。用抽气或吹气鼓泡法进样时，气液比以 $2:1 \sim 3:1$ 最好。经验证明气液比大时，灵敏度有增加的趋势。
A4 在氯化亚锡溶液中加入锡粒，是为了防止其氧化成高价锡而失效。
A5 用盐酸羟胺还原高锰酸钾会产生氯气，必须放置数分钟使氯气逸失，以免干扰汞的测定。
A6 汞容易被吸附在仪器壁及导气管上，联结导管宜用塑料导管而不用橡皮管，所用器皿应充分洗涤。尤其反应瓶内壁及鼓气头上常沾有少量的 $\mathrm{Sn(OH)}_2$ 白色沉淀物，它极易吸附汞，在连续测定中会使测定值越来越低。每测定 $5\sim 10$ 个样，必须用热的稀硝酸冲洗。
A7 为了工作曲线的稳定，绘制时可采用平行样测定。
A8 试剂中的汞及在紫外区有吸收的挥发性杂质，可预先通净化空气或氮气除去。
A9 双光束冷原子吸收测汞仪精密度高，可克服电压波动、光源不稳定的影响，还可以克服环境气氛等因素的影响。

# 附录B

# 浸出液的制备方法

（补充件）

固体废物浸出试验方法标准正在制定中，本法及以下各方法所用的浸出液，先按以下方法制备。待国家标准颁布后，再按国家所颁布的固体废物浸出试验方法标准制备浸出液。

# B1 仪器和材料

B1.1 蒸馏水或去离子水。
B1.2 混合容器：容积为 $2\mathrm{L}$ 的带密封塞的高密聚乙烯瓶。
B1.3 振荡器：往复式水平振荡器。
B1.4 过滤装置：市售成套玻璃砂芯过滤器，纤维滤膜（孔径 $\phi 0.45\mu \mathrm{m}$ ）。

# B2 浸出步骤

B2.1 称取 $100\mathrm{g}$ 试样（以干基计），置于浸取用的混合容器中，加水1L（包括试样的含水量）。
B2.2 将浸取用的混合容器垂直固定在振荡器上，调节振荡频率为 $110 \pm 10$ 次 $/ \mathrm{min}$ ，振幅为 $40 \mathrm{~mm}$ ，在室温下振荡 $8 \mathrm{~h}$ ，静置 $16 \mathrm{~h}$ 。
B2.3 通过过滤装置分离固液相，并立即测定滤液的 $\mathrm{pH}$ 值。滤液应尽快分析，或者按规定分析方法的要求妥善保护贮存浸出液。
