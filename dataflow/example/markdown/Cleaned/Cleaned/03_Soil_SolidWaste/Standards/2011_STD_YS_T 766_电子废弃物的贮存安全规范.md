# 5.4 贮存人员的要求

# 5.4.1 主要负责人

电子废弃物贮存企业或单位的主要负责人对本企业或单位贮存安全工作负有下列职责：

a）建立、健全本企业或单位的贮存安全责任制；
b）组织制定本企业或单位的安全贮存规章制度和操作规程；
c）保证本企业或单位安全投入的有效实施；
d）督促、检查本企业或单位的贮存安全工作，及时消除安全事故隐患；
e）组织制定并实施本企业或单位的贮存安全事故应急救援预案。

# 5.4.2 安全贮存管理人员

5.4.2.1 电子废弃物贮存企业或单位应配备专职安全贮存管理人员或设置安全管理机构。

5.4.2.2 电子废弃物运输企业或单位的安全贮存管理人员对本企业或单位运输安全工作负有下列责任：

a）建立本企业或单位安全贮存知识的教育和培训计划，每半年应至少组织1次从业人员的集中安全贮存培训，并建立教育和培训档案；
b）对本企业或单位的电子废弃物贮存安全进行监督和指导；
c）负责落实企业或单位各项安全贮存管理制度和各岗位安全操作规程；
d) 负责贮存、装卸等现场安全巡视督查，并做好记录；
e）负责安全贮存相关数据统计、安全防护设备和劳动保护用品配备、使用及检查。

5.4.2.3 安全贮存管理人员发现作业现场存在安全隐患时，应及时向企业或单位安全管理机构和主要负责人报告，对违章指挥、违章操作的，应立即制止。

# 5.4.3 操作人员

5.4.3.1 操作人员应接受电子废弃物贮存企业或单位有关安全知识和应急救援方面的培训，熟悉有关的安全运输规章制度和安全操作规程，掌握本岗位的安全操作技能，了解所运载的电子废弃物的性质、危害特性、包装容器的使用特性和发生意外时的应急措施。
5.4.3.2 操作人员必须掌握危险货物贮存的安全知识，并应根据电子废弃物的种类使用劳动防护用品。

# 5.5 贮存记录

贮存场所的管理人员应做好电子废弃物进出登记记录。记录保存时间不得少于3年，且记录上需注明：

a）电子废弃物的类别；
b）电子废弃物的来源；
c）电子废弃物的特性和危险性；
d) 入库日期和存放位置；
e）出库日期及接收单位名称。

# 5.6 其他贮存要求

5.6.1 电子废弃物的贮存设施应参照GB18597、GB18599和HJ527的有关要求进行建设和管理。

# 电子废弃物的贮存安全规范

# 1 范围

本标准规定了电子废弃物的贮存方式、贮存安全要求、安全防护与污染控制和应急预案。
本标准适用于电子废弃物的贮存。

# 3 术语和定义

下列术语和定义适用于本文件。

3.1

# 电子废弃物 electronic waste

指废弃的电子电器产品、电子电气设备(以下简称产品和设备)及其废弃零部件、元器件和国家环境保护总局会同有关部门规定纳入电子废物管理的物品、物质。包括工业生产活动中产生的报废产品或者设备、报废的半成品和下脚料，产品或者设备维修、翻新、再制造过程产生的报废品，日常生活或者为日常生活提供服务的活动中废弃的产品或者设备，以及法律法规禁止生产或者进口的产品或者设备。

3.2

# 电子废弃物的贮存 storage of electronic waste

为收集、运输、拆解、处理和处置之目的，在符合要求的特定场所暂时性存放电子废弃物的活动。

3.3

# 隔开贮存 cut-off storage

在同一建筑或同一区域内，用隔板或墙，将不同的物料隔开的贮存方式。

[GB/T26493，定义3.3]

3.4

# 隔离贮存 segregated storage

在同一房间或同一区域内不同的物料之间分开一定距离用通道保持空间的贮存方式。

[GB/T26493，定义3.4]
