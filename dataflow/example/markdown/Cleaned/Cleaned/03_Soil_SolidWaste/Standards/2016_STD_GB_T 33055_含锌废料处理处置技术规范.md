# 含锌废料处理处置技术规范

# 1 范围

本标准规定了含锌废料处理处置的术语和定义、处理处置方法与环境保护。

本标准适用于化工、冶金、电镀、热镀锌等行业产生的含锌废料的处理处置。

# 3 术语和定义

下列术语和定义适用于本文件。

# 3.1

湿法回收工艺 hydrometallurgical recovery process

利用强酸、强碱或其他溶剂的特性，将含锌废料中的有价金属浸出后再进行分离和回收的处理处置方法。

# 3.2

火法回收工艺 pyrometallurgical recovery process

在高温条件下处理含锌废料，回收其中全部或部分有价元素的处理处置方法。

# 4 处理处置方法

# 4.1 湿法回收

# 4.1.1 适用范围

湿法回收工艺处理处置含锌废料一般用于生产硫酸锌、硝酸锌、氯化锌及金属锌粉等。

# 4.1.2 生产硫酸锌、硝酸锌、氯化锌

# 4.1.2.1 原理

利用湿法回收工艺对含锌废料用酸溶液浸取，其化学反应方程式如下：

$$
\begin{array}{l} \mathrm {Z n} + \mathrm {H} _ {2} \mathrm {S O} _ {4} \longrightarrow \mathrm {Z n S O} _ {4} + \mathrm {H} _ {2} \uparrow \quad \mathrm {Z n O} + \mathrm {H} _ {2} \mathrm {S O} _ {4} \longrightarrow \mathrm {Z n S O} _ {4} + \mathrm {H} _ {2} \mathrm {O} \\ \mathrm {Z n} + 2 \mathrm {H N O} _ {3} \longrightarrow \mathrm {Z n} \left(\mathrm {N O} _ {3}\right) _ {2} + \mathrm {H} _ {2} \uparrow \quad \mathrm {Z n O} + 2 \mathrm {H N O} _ {3} \longrightarrow \mathrm {Z n} \left(\mathrm {N O} _ {3}\right) _ {2} + \mathrm {H} _ {2} \mathrm {O} \\ \mathrm {Z n} + 2 \mathrm {H C l} \longrightarrow \mathrm {Z n C l} _ {2} + \mathrm {H} _ {2} \uparrow \quad \mathrm {Z n O} + 2 \mathrm {H C l} \longrightarrow \mathrm {Z n C l} _ {2} + \mathrm {H} _ {2} \mathrm {O} \\ \end{array}
$$

# 4.1.2.2 工艺流程

含锌废料根据废料情况进行粉碎、漂洗后，加酸溶液进行浸取、过滤，再经净化、除杂、过滤、浓缩（结晶）、脱水、干燥，得到工业硫酸锌、工业硝酸锌、工业氯化锌产品。工艺流程图见图1。

图1锌盐生产工艺流程图

# 4.1.2.3 工艺参数

主要控制工艺参数如下：

浸取温度控制为 $25^{\circ}\mathrm{C}\sim 90^{\circ}\mathrm{C}$
浸取液酸浓度为 $300\mathrm{g / L}\sim 400\mathrm{g / L}$
——浸取固液比约为 $1:6$
浸取时间 $2\mathrm{h}\sim 3\mathrm{h}$

# 4.1.2.4 生产设备

主要设备有：反应釜、排风处理系统、储槽、固液分离设备、耐酸碱泵、干燥设备、包装设备等。

# 4.1.2.5 产品指标

产品应符合HG/T2326、HG/T3582、HG/T2323。

# 4.1.3 生产金属锌粉

# 4.1.3.1 原理

利用湿法回收工艺对以氧化锌为主的含锌废料进行碱浸，对浸出液进行电解得到金属锌。其化学反应方程式和电解反应方程式如下：

$$
\mathrm {Z n O} + 2 \mathrm {N a O H} + \mathrm {H} _ {2} \mathrm {O} \longrightarrow \mathrm {N a} _ {2} \mathrm {Z n} (\mathrm {O H}) _ {4}
$$

阳极反应： $2\mathrm{OH} - 2\mathrm{e}\longrightarrow \left(\frac{1}{2}\right)\mathrm{O}_2\uparrow +\mathrm{H}_2\mathrm{O}$

阴极反应： $\mathrm{Zn(OH)_4^{2 - } + 2e\longrightarrow Zn\downarrow + 4OH^-}$

# 4.1.3.2 工艺流程

以氧化锌为主的含锌废料与碱液在浸出釜中反应；浸出液进行净化，净化液进入电解槽进行电解。废电解液苛化后循环使用，锌进行洗涤烘干后得到锌粉产品。渣进入废渣处理系统。含锌废料生产金属锌粉工艺流程图见图2。

图2 金属锌粉生产工艺流程图

# 4.1.3.3 工艺参数

主要控制工艺参数如下：

浸出工段，初始碱液质量浓度控制为 $220\mathrm{g / L}\sim 250\mathrm{g / L}$ ；浸出温度控制为 $80^{\circ}\mathrm{C}\sim 90^{\circ}\mathrm{C}$ ；浸出时间 $2\mathrm{h}$ 左右；
——净化工段，净化温度控制为 $70^{\circ}\mathrm{C}\sim 90^{\circ}\mathrm{C}$
——电解工段，电解液锌质量浓度为 $35\mathrm{g / L}\sim 40\mathrm{g / L}$ ；电解液碱质量浓度为 $180\mathrm{g / L}\sim 210\mathrm{g / L}$ ；温度 $50~\mathrm{^\circ C}$ 以下；电流密度 $1000\mathrm{A / m^2}$ ；槽电压 $2.7\mathrm{V}\sim 3.2\mathrm{V}$ 。

# 4.1.3.4 生产设备

主要设备有：浸出釜、压滤设备、净化釜、陈化池、电解槽、废液槽、清洗设备、离心设备、干燥机等。

# 4.1.3.5 产品指标

锌粉产品应符合GB/T6890（全锌质量分数 $98\%$ 规格）。

# 4.2 火法回收

# 4.2.1 适用范围

火法回收一般用于低品位(含锌 $20\%$ 以下)含锌废料、废渣中锌的富集，产品以次氧化锌为主。

# 4.2.2 生产次氧化锌

# 4.2.2.1 原理

利用火法回收工艺对低品位含锌废料(炼钢烟尘、浸出渣等，Zn主要以ZnO形态为主)与焦粉按照比例焙烧，高温状态使得锌元素还原后蒸发进入气相，气相中含有氧化性物质，通过降温冷却后在气相又被氧化后进入收尘系统收集，成为次氧化锌，主要化学反应方程式如下：

$$
\begin{array}{l} \mathrm {Z n O} + \mathrm {C O} \longrightarrow \mathrm {Z n (g)} \uparrow + \mathrm {C O} _ {2} \uparrow \\ \mathrm {Z n O} \cdot \mathrm {F e} _ {2} \mathrm {O} _ {3} + \mathrm {C O} \longrightarrow \mathrm {Z n O} + 2 \mathrm {F e O} + \mathrm {C O} _ {2} \uparrow \\ \mathrm {Z n O} + \mathrm {F e} \longrightarrow \mathrm {Z n (g)} \uparrow + \mathrm {F e O} \\ \mathrm {Z n O} \cdot \mathrm {S i O} _ {2} + \mathrm {C O} \longrightarrow \mathrm {Z n} (\mathrm {g}) \uparrow + \mathrm {S i O} _ {2} + \mathrm {C O} _ {2} \uparrow \\ 2 \mathrm {Z n} + \mathrm {O} _ {2} \longrightarrow 2 \mathrm {Z n O} \downarrow \\ \end{array}
$$

# 4.2.2.2 工艺流程

低品位含锌废料根据情况进行粉碎、脱水等前处理后与焦粉按照一定比例混合，混合后在焙烧炉（窑）中进行焙烧，气相进入烟道后进行冷却、利用布袋或其他除尘设备收集次氧化锌粉尘，包装后作为成品出售或作为原料进行精细化加工。固相炉渣经过磁选后收集部分返回钢铁企业回收其中的铁元素，其他交由有资质部门处理。生产次氧化锌工艺流程图见图3。

# 4.2.2.3 工艺参数

主要控制工艺参数如下：

炉窑焙烧温度控制为 $800^{\circ}\mathrm{C}\sim 1300^{\circ}\mathrm{C}$
炉窑负压控制为 $50\mathrm{Pa}\sim 80\mathrm{Pa}$
冷凝器出口温度低于 $500^{\circ}\mathrm{C}$

# 4.2.2.4 生产设备

主要设备有：混料设备、加料机、高温炉（窑）、鼓风机、沉降设备、冷凝器（冷却器）、除尘器、引风机、磁选机、包装设备、成套环保设施等。

# 4.2.2.5 产品指标

次氧化锌产品应符合YS/T73。

# 5 环境保护

5.1 含锌废料在处理处置过程中产生的废气应符合GB31574废气排放要求。
5.2 含锌废料在处理处置过程中产生的废水应符合GB31574废水排放要求。
5.3 含锌废料处理处置过程中产生的废渣，应按GB5085.7的规定进行鉴别，并符合下列规定：

a）经鉴别属于危险废物，应按GB18597和HJ2025的要求进行收集、贮存、运输，并交由有资质单位进行处理；
b）经鉴别属于一般固体废物，应符合GB18599的要求。
