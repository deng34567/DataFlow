# 火力发电厂燃料试验方法第 6 部分：飞灰和炉渣可燃物测定方法

# 1 范围

本文件规定了测定燃煤锅炉排出的飞灰和炉渣中可燃物的灼烧减量法和碳含量测定法（高温燃烧-红外光谱法）的方法提要、仪器设备、测试步骤、方法精密度和结果计算。

灼烧减量法适用于燃煤锅炉运行监督，灼烧减量法中的方法A和碳含量测定法（高温燃烧-红外光谱法）适用于锅炉机组性能试验时的飞灰和炉渣中可燃物的测定。

# 3 术语和定义

DL/T 958界定的以及下列术语和定义适用于本文件。

# 3.1

灼烧减量 Loss on ignition

在规定的条件下加热灼烧后的质量损失。飞灰和炉渣试样在灼烧过程中，减少的物质主要包括水分（游离水和化合水）、矿物质分解和有机物燃烧生成的二氧化碳等。

# 3.2

可燃物 Combustible matter

锅炉燃烧产生的飞灰和炉渣中未完全燃烧的物质，主要包括碳、硫等元素的化合物。

# 4 样品准备

按DL/T 567.3中规定进行飞灰和炉渣样品的采取和制备。

# 5 灼烧减量法

# 5.1 方法提要

称取一定质量的飞灰（炉渣）样品，在充足氧气供应的条件下按规定的升温程序、时间对其进行灼烧，根据其灼烧减量扣除水分含量和碳酸盐二氧化碳含量后，作为可燃物含量。

# 5.2 方法 A（干燥箱 $^ +$ 马弗炉法)

# 5.2.1 水分测定

5.2.1.1 仪器设备参见 GB/T 212-2008 中 3.2.2 的规定。
5.2.1.2 按 GB/T 212-2008 中 3.2 的方法 B（空气干燥法）称取飞灰（炉渣）试样进行水分测定，初始干燥时间应为 1h。
5.2.1.3 应按公式（1）计算飞灰（炉渣）试样中水分含量：

$$
M _ {a d} = \frac {m _ {1}}{m} \times 1 0 0 \tag {1}
$$

式中：

$M _ { a d }$ ad 飞灰（炉渣）试样中空气干燥基水分的质量分数， $\%$ ；

m 称取的飞灰（炉渣）试样的质量，g;

${ \boldsymbol { m } } _ { 1 }$ ——飞灰（炉渣）试样干燥后失去的质量，g。

# 5.2.2 灼烧减量测定

# 5.2.2.1 仪器设备

参见GB/T 212-2008中4.1.2的规定。

# 5.2.2.2 测试步骤

5.2.2.2.1 在预先灼烧至质量恒定的灰皿中，称取飞灰（炉渣）试样（ $1 { \pm } 0 . 1$ ） $\mathbf { g }$ ，称准至 $0 . 0 0 0 2 \mathrm { g }$ ，均匀地摊平在灰皿中。
5.2.2.2.2 将灰皿送入炉温不超过 $1 0 0 ^ { \circ } \mathrm { C }$ 的马弗炉恒温区中，关上炉门并使炉门留有 $1 5 \mathrm { m m }$ 左右的缝隙。在不少于 $3 0 \mathrm { { m i n } }$ 的时间内将炉温缓慢升温至 $5 0 0 ^ { \circ } \mathrm { C }$ ，并在此温度下保持 $3 0 \mathrm { { m i n } }$ 。继续升温到（ $8 1 5 { \pm } 1 0 \ $ ）$^ \circ \mathrm { C }$ ，并在此温度下灼烧 1h。
5.2.2.2.3 从炉中取出灰皿，放在耐热瓷板或石棉板上，在空气中冷却 $5 \mathrm { { m i n } }$ 左右，移入干燥器中冷却至室温（约 $2 0 \mathrm { m i n }$ ）后称量。
