# 1 主题内容与适用范围

1.1 本标准规定了测定土壤中总汞的冷原子吸收分光光度法。
1.2 标准的检出限视仪器型号的不同而异，本方法的最低检出限为 $0.005\mathrm{mg / kg}$ （按称取 $2\mathrm{g}$ 试样计算）。
1.3 易挥发的有机物和水蒸气在 $253.7 \mathrm{~nm}$ 处有吸收而产生干扰。易挥发有机物在样品消解时可除去，水蒸气用无水氯化钙、过氯酸镁除去。

# 2 原理

汞原子蒸气对波长为 $253.7\mathrm{nm}$ 的紫外光具有强烈的吸收作用，汞蒸气浓度与吸光度成正比。通过氧化分解试样中以各种形式存在的汞，使之转化为可溶态汞离子进入溶液，用盐酸羟胺还原过剩的氧化剂，用氯化亚锡将汞离子还原成汞原子，用净化空气做载气将汞原子载入冷原子吸收测汞仪的吸收池进行测定。

# 3 试剂

除非另有说明，分析中均使用符合国家标准或专业标准的优级纯试剂。

3.1 无汞蒸馏水：二次蒸馏水或电渗析去离子水通常可达到此纯度，也可将蒸馏水加盐酸酸化至 $\mathrm{pH}$ 3，然后通过巯基棉纤维管除汞（见附录B)。
3.2 硫酸 $(\mathrm{H_2SO_4})$ ， $\rho = 1.84\mathrm{g / ml_o}$
3.3 盐酸（HCl）， $\rho = 1.19\mathrm{g / ml}$
3.4 硝酸 $(\mathrm{HNO}_3)$ ， $\rho = 1.42\mathrm{g / ml_o}$
3.5 硫酸-硝酸混合液， $1 + 1$
3.6 重铬酸钾 $\mathrm{(K_2Cr_2O_7)}$
3.7 高锰酸钾溶液：将 $20\mathrm{g}$ 的高锰酸钾（ $\mathrm{KMnO}_4$ ，必要时重结晶精制）用蒸馏水（3.1）溶解，稀释至 $1000\mathrm{ml}$ 。
3.8盐酸羟胺溶液：将 $20\mathrm{g}$ 的盐酸羟胺（ $\mathrm{NH_2OH\cdot HCl}$ ）用蒸馏水（3.1）溶解，释释至 $100\mathrm{ml}$
3.9 五氧化二钒 $(\mathrm{V}_2\mathrm{O}_5)$ 。
3.10 氯化亚锡溶液：将 $20\mathrm{g}$ 氯化亚锡（ $\mathrm{SnCl_2\cdot 2H_2O}$ ）置于烧杯中，加入 $20\mathrm{ml}$ 盐酸（3.3），微微加热。待完全溶解后，冷却，再用蒸馏水（3.1）稀释至 $100\mathrm{ml}$ 。若有汞，可通入氮气鼓泡除汞。临用前现配。
3.11 汞标准固定液：将 $0.5\mathrm{g}$ 重铬酸钾（3.6）溶于 $950\mathrm{ml}$ 蒸馏水（3.1）中，再加 $50\mathrm{ml}$ 硝酸（3.4）。
3.12 稀释液：将 $0.2\mathrm{g}$ 重铬酸钾（3.6）溶于 $972.2\mathrm{ml}$ 蒸馏水（3.1）中，再加 $27.8\mathrm{ml}$ 硫酸（3.2）。

3.13 汞标准贮备溶液， $100\mathrm{mg / L}$ ：称取放置在硅胶（3.16）干燥器中充分干燥过的 $0.1354\mathrm{g}$ 氯化汞 $(\mathrm{HgCl_2})$ ，用汞标准固定液（3.11）溶解后，转移到 $1000\mathrm{ml}$ 容量瓶中，再用汞标准固定液（3.11）稀释至标线，摇匀。
3.14 汞标准中间溶液， $10.0\mathrm{mg / L}$ ：吸取汞标准贮备溶液（3.13） $10.00\mathrm{ml}$ ，移入 $100\mathrm{ml}$ 容量瓶，加汞标准固定液（3.11）稀释至标线，摇匀。
3.15 汞标准使用溶液， $0.100\mathrm{mg / L}$ ：吸取汞标准中间溶液（3.14） $1.00\mathrm{ml}$ ，移入 $100\mathrm{ml}$ 容量瓶，加汞标准固定液（3.11）稀释至标线，摇匀。
3.16 变色硅胶： $\phi 3\sim 4\mathrm{mm}$ ，干燥用。
3.17 经碘处理的活性炭：按重量取1份碘，2份碘化钾和20份蒸馏水，在玻璃烧杯中配制成溶液，然后向溶液中加入约10份柱状活性炭（工业用， $\phi 3\mathrm{mm}$ 长 $3\sim 7\mathrm{mm}$ ）。用力搅拌至溶液脱色后，从烧杯中取出活性炭，用玻璃纤维把溶液滤出，然后在 $100\%$ 左右干燥 $1\sim 2\mathrm{h}$ 即可。
3.18 仪器洗液：将 $1.0\mathrm{g}$ 重铬酸钾（3.6）溶于 $900\mathrm{ml}$ 蒸馏水（3.1）中，加入 $100\mathrm{ml}$ 硝酸（3.4）。

# 5样品

将采集的土壤样品（一般不少于 $500\mathrm{g}$ ）混匀后用四分法缩分至约 $100\mathrm{g}$ 。缩分后的土样经风干（自然风干或冷冻干燥）后，除去土样中石子和动植物残体等异物，用木棒（或玛瑙棒）研压，通过 $2\mathrm{mm}$ 尼龙筛（除去 $2\mathrm{mm}$ 以上的砂砾），混匀。用玛瑙研钵将通过 $2\mathrm{mm}$ 尼龙筛的土样研磨至全部通过100目（孔径 $0.149\mathrm{mm}$ ）尼龙筛，混匀后备用。

# 6 分析步骤

# 6.1 试液的制备

# 6.1.1 硫酸一硝酸一高锰酸钾消解法

称取按步骤5制备的土壤样品 $0.5\sim 2\mathrm{g}$ （准确至 $0.0002\mathrm{g}$ ）于 $150~\mathrm{ml}$ 锥形瓶中，用少量蒸馏水（3.1）润湿样品，加硫酸-硝酸混合液（3.5） $5\sim 10\mathrm{ml}$ ，待剧烈反应停止后，加蒸馏水（3.1） $10\mathrm{ml}$ 高锰酸钾溶液（3.7） $10\mathrm{ml}$ ，在瓶口插一小漏斗，置于低温电热板上加热至近沸，保持 $30\sim 60\mathrm{min}$ 。分解过程中若紫色褪去，应随时补加高锰酸钾溶液（3.7），以保持有过量的高锰酸钾存在。取下冷却。在临测定前，边摇边滴加盐酸羟胺溶液（3.8），直至刚好使过剩的高锰酸钾及器壁上的水合二氧化锰全部褪色为止。

注：对有机质含量较多的样品，可预先用硝酸加热回流消解，然后再加硫酸和高锰酸钾继续消解。

# 6.1.2 硝酸一硫酸一五氧化二钒消解法

称取按步骤5制备的土壤样品 $0.5\sim 2\mathrm{g}$ （准确至 $0.0001\mathrm{g}$ ）于 $150~\mathrm{ml}$ 锥形瓶中，用少量蒸馏水（3.1）润湿样品，加入五氧化二钒（3.9）约 $50\mathrm{mg}$ ，硝酸（3.4） $10\sim 20\mathrm{ml}$ ，硫酸（3.2） $5\mathrm{ml}$ ，玻璃珠 $3\sim 5$ 粒，摇匀。在瓶口插一小漏斗，置于电热板上加热至近沸，保持 $30\sim 60\mathrm{min}$ 。取下稍冷，加蒸馏水（3.1） $20\mathrm{ml}$ ，继续加热煮沸 $15\mathrm{min}$ ，此时试样为浅灰白色（若试样色深应适当补加硝酸再进行分解)。取下冷却，滴加高锰酸钾溶液（3.7）至紫色不褪。在临测定前，边摇边滴加盐酸羟胺溶液（3.8），直至刚好使过剩的高锰酸钾及器壁上的水合二氧化锰全部褪色为止。

# 6.2 测定

6.2.1 连接好仪器，更换U形管中硅胶（3.16），按说明书调试好测汞仪及记录仪，选择好灵敏度挡及载气流速。将三通阀（4.5）旋至“校零”端。
6.2.2 取出汞还原器（4.3）吹气头，将试液（含残渣）全部移入汞还原瓶，用蒸馏水洗涤锥形瓶 $3 \sim 5$ 次，洗涤液并入还原瓶，加蒸馏水至 $100 \mathrm{ml}$ 。加入 $1 \mathrm{ml}$ 氯化亚锡溶液（3.11），迅速插入吹气头，然后将三通阀（4.5）旋至“进样”端，使载气通入汞还原器（4.3）。此时试液中汞被还原并汽化成汞蒸气，随载气流入测汞仪的吸收池，表头指针和记录仪笔迅速上升，记下最高读数或峰高。待指针和记录笔重新回零后，将三通阀（4.5）旋至“校零”端，取出吹气头，弃去废液，用蒸馏水（3.1）清洗汞还原器（4.3）二次，再用稀释液（3.11）洗一次，以氧化可能残留的二价锡，然后进行另一试样的测定。

# 6.3 空白试样

每分析一批试样，按步骤（6.1）制备至少两份空白试样，并按步骤（6.2）进行测定。

# 6.4 校准曲线

准确移取汞标准使用溶液（3.15）0.00，0.50，1.00，2.00，3.00和 $4.00\mathrm{ml}$ 于 $150\mathrm{ml}$ 锥形瓶中，加硫酸-硝酸混合液（3.5） $4\mathrm{ml}$ ，加高锰酸钾溶液（3.7）5滴，加蒸馏水（3.1） $20\mathrm{ml}$ ，摇匀。测定前滴加盐酸羟胺溶液（3.8）还原，以下按6.2所述步骤进行测定。

将测得的吸光度为纵坐标，对应的砷含量（ $\mu \mathrm{g}$ ）为横坐标，绘制校准曲线。

# 7 结果的表示

土样中总汞的含量 $c$ （ $\mathrm{Hg}$ ， $\mathrm{mg / kg}$ ）按下式计算：

$$
c = \frac {m}{W (1 - f)}
$$

式中： $m$ ——测得试液中汞量， $\mu \mathrm{g}$

$W$ —称取土样重量，g；

$f$ —土样水份含量， $\%$

# 土样水分含量测定

A1称取通过100目筛的风干土样 $5\sim 10\mathrm{g}$ （准确至 $0.01\mathrm{g}$ ，置于铝盒或称量瓶中，在 $105\%$ 烘箱中烘 $4\sim 5\mathrm{h}$ ，烘干至恒重。

A2 以百分数表示的风干土样水分含量 $f$ 按下式计算：

$$
f (\%) = \frac {W _ {1} - W _ {2}}{W _ {1}} \times 100
$$

式中： $f$ —土样水分含量， $\%$

$W_{1}$ ——烘干前土样重量，g；

$W_{2}$ ——烘干后土样重量， $\mathrm{g}$ 。

# 盐酸羟胺溶液的提纯

盐酸羟胺试剂中常含有汞，必须提纯。当汞含量较低时，可采用巯基棉纤维管除汞法；汞含量高时，先用萃取法除掉大量汞后再用巯基棉纤维管除汞。

B1 疏基棉纤维管除汞法：在内径 $6\sim 8\mathrm{mm}$ 、长 $100\mathrm{mm}$ 左右、一端拉细的玻璃管，或 $500\mathrm{ml}$ 分液漏斗放液管中，填充 $0.1\sim 0.2\mathrm{g}$ 疏基棉纤维，将待净化试剂以 $10\mathrm{ml / min}$ 速度流过一至二次即可除尽汞。

巯基棉纤维（sulfhydryl cotton fiber，缩写S.C.F）的制备：

于棕色磨口广口瓶中，依次加入 $100\mathrm{ml}$ 硫代乙醇酸（ $\mathrm{CH}_2\mathrm{SHCOOH}$ ，分析纯）、 $60\mathrm{ml}$ 乙酸酐 $\left[\mathrm{(CH_3CO)_2O}\right]$ 、 $40\mathrm{ml}36\%$ 乙酸（ $\mathrm{CH}_3\mathrm{COOH}$ ）、 $0.3\mathrm{ml}$ 硫酸（3.2），充分混均，冷却至室温后，加入 $30\mathrm{g}$ 长纤维脱脂棉，使之浸泡完全，用水冷却，待反应热散去后，放入 $40\pm 2\%$ 烘箱中 $2\sim 4$ 天后取出。用耐酸过滤漏斗抽滤，用无汞蒸馏水（3.1）充分洗涤至中性后，摊开，于 $30\sim 35\%$ 下烘干。成品放于棕色磨口广口瓶中，避光，较低温度下保存。

B2萃取法：取 $250\mathrm{ml}$ 盐酸羟胺溶液（3.8）注入 $500\mathrm{ml}$ 分液漏斗中，每次加入 $15\mathrm{ml}$ 含二苯基硫巴腙（双硫腙 $\mathrm{C_{13}H_{12}N_4S}$ ） $0.1\mathrm{g / L}$ 的四氯化碳（ $\mathrm{CCl_4}$ ）溶液，反复萃取，直至含双硫腙的四氯化碳溶液保持绿色不变为止。然后用四氯化碳萃取，以除去多余的双硫腙。
