# 中华人民共和国国家标准

# 固体废物总铬的测定直接吸入火焰原子吸收分光光度法

GB/T15555.6—1995

Solid waste-Determination of total chromiumFlame atomic absorption spectrometry

# 1 主题内容与适用范围

1.1 本标准规定了测定固体废物浸出液中总铬的直接吸入火焰原子吸收分光光度法。
1.2 本标准方法适用于固体废物浸出液中总铬的测定。
1.2.1 测定范围

本方法的测定范围是： $0.08\sim 3.0\mathrm{mg / L}$

# 2 原理

将浸出液经过氧化处理后，直接喷入火焰，在空气-乙炔火焰中形成的铬基态原子对 $357.9\mathrm{nm}$ 或其他的共振线产生吸收。将浸出液的吸光度与标准溶液的吸光度进行比较，测定浸出液中铬的含量。

# 3 试剂

除非另有说明，均使用符合国家标准或专业标准的试剂，去离子水或同等纯度的水。

3.1 蒸馏水或去离子水。
3.2 过硫酸铵 $\left[\left(\mathrm{NH}_{4}\right)_{2} \mathrm{~S}_{2} \mathrm{O}_{8}\right]$ 溶液， $5 \%$
3.3 氯化铵 $\mathrm{(NH_4Cl)}$ 溶液， $10\%$
3.4 硝酸 $(\mathrm{HNO}_3)$ ， $\rho = 1.42\mathrm{g / mL}$ ，优级纯。
3.5 硝酸 $1\%$ ，用（3.4）配制。
3.6 铬标准贮备液， $1.000\mathrm{g / L}$

准确称取 $0.2829\mathrm{g}$ 基准重铬酸钾 $\mathrm{(K_2Cr_2O_7)}$ ，用少量水溶解后，转移入 $100\mathrm{mL}$ 容量瓶中，用水定容后摇匀。
3.7 铬标准使用液， $50\mathrm{mg / L}$

吸取标准贮备液(3.6) $5.0\mathrm{mL}$ 于 $100\mathrm{mL}$ 容量瓶中，用水定容摇匀。

# 4 仪器

4.1 原子吸收分光光度计。
4.2 铬空心阴极灯。
4.3 乙炔钢瓶或乙炔发生器。
4.4 空气压缩机，应备有除水、除油和除尘装置。
4.5 仪器参数：根据仪器说明书要求自己选择测试条件。一般仪器的使用条件如表1所示。

表 1 一般仪器的使用条件


| 元素       | Cr                  |
| -------- | ------------------- |
| 测定波长,nm  | 357.9               |
| 通带宽度,nm  | 0.7                 |
| 火焰性质     | 富燃性火焰               |
| 次灵敏线,nm  | 359.0;360.5;425.4   |
| 燃烧器高度,mm | 10(使空心阴极灯光斑通过亮蓝色部分) |


# 5样品的保存和处理

5.1样品的保存：浸出液如不能很快进行分析，应加浓硝酸(3.4)酸化至 $\mathrm{pH}$ 小于2，并尽快分析，不要超过 $24\mathrm{h}$ 。
5.2 试样的处理

取 $50.0\mathrm{mL}$ 固体废物浸出液于 $150\mathrm{mL}$ 三角瓶中，加入浓硝酸(3.4) $2\mathrm{mL}$ ， $(\mathrm{NH_4})_2\mathrm{S}_2\mathrm{O}_8$ 溶液(3.2) $5\mathrm{mL}$ ，摇匀。在三角瓶口插入小漏斗后置于电热板上加热，煮沸至约剩 $20\mathrm{mL}$ 左右时取下冷却，用少量水冲洗小漏斗和三角瓶内壁，全量转移至 $50\mathrm{mL}$ 容量瓶中，加入 $\mathrm{NH_4Cl}$ 溶液(3.3) $5\mathrm{mL}$ 用水定容，待测。

空白实验：用水代替样品，采用和样品相同的步骤和试剂，在测定试料的同时测定空白值。

# 6 测量

6.1 根据表1所列条件调节仪器，火焰状态、燃烧器高度等达到最佳，用 $2.00\mathrm{mg / L}$ 的标准溶液进行，检查。使用 $357.9\mathrm{nm}$ 线时，其吸光度应达到0.200(不使用标尺扩展)左右，否则应进一步检查、调节。
6.2 分别吸取铬标准使用液(3.7)0,0.50,1.00,1.50,2.00,2.50 mL于50 mL容量瓶中，分别加入 $\mathrm{NH_4Cl}$ 溶液(3.3)5 mL, $\mathrm{HNO_3}$ (3.5)5 mL，用水定容后摇匀。此标准系列含铬为0,0.50,1.00,1.50,2.00,2.50 mg/L。
6.3 用纯水调节仪器零点后，按从稀到浓的顺序吸入标准系列，测定相应的吸光度值。再重复一次。两次吸光度值之差不应超过 $2\%$ ，否则重新检查。调节仪器后再进行测定。用标准系列和相应的吸光度值制作校准曲线。
6.4 用纯水冲洗雾化系统，测定试料空白和试料的吸光度值。试料的吸光度扣除空白吸光度后，从校准曲线上查出试料中铬的浓度。

一般每测定10个试样要进行一次仪器零点校正，并吸入 $2.0\mathrm{mg / L}$ 的标准溶液检查灵敏度是否发生了变化。
6.5 当样品基体成分复杂或者不明时，应制做标准加入法曲线，用于考查样品是否宜用校准曲线法直接定量。

取六只编号的 $50~\mathrm{mL}$ 容量瓶加入 $5\sim 20~\mathrm{mL}$ 的浸出液(视铬的含量而定)，分别加入铬标准使用液(3.7)0,0.25,0.50,0.75,1.00,1.50mL。再分别加入 $\mathrm{NH_4Cl(3.3)5mL}$ $\mathrm{HNO_3(3.5)5mL}$ ，加水稀释到刻度，摇匀。测定吸光度值，用该吸光度值与相应加入标准溶液的浓度并和(6.3)在同一坐标上绘制标准加入曲线。

如果两条曲线平行，则可用校准曲线法直接定量，否则，说明有基体干扰存在，应该用标准加入法定量，或者采用溶剂萃取、共沉淀等分离方法，分离基体干扰成分后，再进行测定。

# 7 结果的表示

浸出液中总铬浓度 $c(\mathrm{mg / L})$ 按下式计算：

$$
c (\mathrm {m g} / \mathrm {L}) = c _ {1} \times \frac {V _ {0}}{V}
$$

式中： $c_{1}$ ——被测试料中总铬的浓度， $\mathrm{mg / L}$

$V_{0}$ ——制样时定容体积， $\mathrm{mL}$

$V$ ——试料的体积， $\mathrm{mL}$ 。

# 8 方法的精密度和准确度

室内测定含总铬 $0.87\mathrm{mg / L}$ 的铬渣浸出液，测定10次，其相对标准偏差为 $1.3\%$ ；加入 $0.50\mathrm{mg / L}$ 标准时的加标回收率为 $97.8\%$ 。

# 注意事项

（参考件）

A1 $7\%$ 以上的 $\mathrm{HCl},\mathrm{HClO}_4$ 对铬的测定有正干扰； $\mathrm{HNO}_3$ 和 $\mathrm{H}_3\mathrm{PO}_4$ 产生负干扰。在空气-乙炔火焰中，产生负干扰的金属元素及干扰顺序为 $\mathrm{Cu} > \mathrm{Ba} > \mathrm{Al} > \mathrm{Mg} > \mathrm{Ca},\mathrm{Na},\mathrm{Sr},\mathrm{Zn},\mathrm{Sn}$ 。且这些干扰情况与铬的存在形态 $(\mathrm{Cr}^{3 + },\mathrm{Cr}_2\mathrm{O}_7^{2 - })$ 无关。

$\mathrm{Fe}^{3+}$ 产生负干扰，而对 $\mathrm{Cr(VI)}$ 有正干扰；锰、镍仅对 $\mathrm{Cr}^{3+}$ 产生负干扰，而不干扰 $\mathrm{Cr(VI)}$ 的测定。当使用贫燃性空气-乙炔火焰时，铁和镍的干扰变小，但铬测定的灵敏度明显降低。在试样中加入 $1\% \sim 2\%$ 的氯化铵可以消除铁、镍、锰等金属元素和 $\mathrm{HCl}, \mathrm{HNO}_3$ 等酸类的干扰。加入 $1\%$ 焦硫酸钾也可消除共存金属成分的干扰。

A2 $\mathrm{Cr(V)}$ 与 $\mathrm{Cr(III)}$ 的灵敏度稍有不同，如果浸出液中 $\mathrm{Cr(III)}$ 和 $\mathrm{Cr(V)}$ 共存，只用 $\mathrm{Cr(V)}$ 作标准系列制作校准曲线，会造成测定误差。由于容易得到 $\mathrm{Cr(V)}$ 的基准试剂，制备标准溶液也很方便。所以用 $\mathrm{(NH_4)_2S_2O_8}$ 将浸出液中可能存在的 $\mathrm{Cr(III)}$ 氧化成 $\mathrm{Cr(V)}$ 再进行测定。
A3 Cr是易形成耐高温氧化物的元素之一。因此，其原子化效率受火焰性质影响较大。必须使用富燃性(还原型)空气-乙炔火焰，否则灵敏度降低。
A4 燃烧器高度对铬的灵敏度也有明显影响。铬的最灵敏区域在燃烧器缝隙上方 $5\sim 13\mathrm{mm}$ 范围，以 $10\mathrm{mm}$ 处为最佳。这时不仅测定灵敏度高，读数稳定，且干扰较小。
A5 铬的测定波长比较多，以 $357.9 \mathrm{~nm}$ 线最为灵敏，可测定低含量的铬。359.4、360.5 nm 线灵敏度稍低于 $357.9 \mathrm{~nm}$ 线，也可以使用。但测定高含量的铬时，建议使用 $425.4 \mathrm{~nm}, 427.5 \mathrm{~nm}$ 或 $429.0 \mathrm{~nm}$ 线。它们都处于可见光区，受分子吸收等光谱干扰较少，灵敏度比 $357.9 \mathrm{~nm}$ 线低 $3 \sim 5$ 倍。

注：浸出液的制备方法，参见GB/T15555.1—1995《固体废物总汞的测定冷原子吸收分光光度法》中的附录B。
