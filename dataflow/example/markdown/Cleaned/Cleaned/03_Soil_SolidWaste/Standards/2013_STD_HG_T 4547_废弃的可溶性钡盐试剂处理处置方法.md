# 废弃的可溶性钡盐试剂处理处置方法

警告——废弃的可溶性钡盐试剂具有毒性，在其处理处置过程中应具备必要的安全措施，并做好个体防护，如溅到皮肤上应立即用肥皂水和清水彻底冲洗，严重者应立即就医。

# 3 术语和定义

# 可溶性钡盐试剂 soluble barium salt reagents

是指 $20^{\circ}\mathrm{C}$ 下，以水为溶剂时溶解度不小于 $1\mathrm{g}$ 的钡盐化学试剂，包括氯化钡、硝酸钡、醋酸钡等。不同的钡化合物的毒性大小与溶解度有关，溶解度越高、毒性越大，可溶性钡盐如氯化钡、硝酸钡、醋酸钡为剧毒。

# 4 废弃的可溶性钡盐试剂的鉴别

（1）取铂丝，用盐酸湿润后，蘸取试验溶液，在无色火焰中燃烧，火焰呈黄绿色，自绿色玻璃中透视，火焰显蓝色。
（2）取试验溶液，加稀硫酸，产生白色沉淀，分离，沉淀在盐酸或硝酸中均不溶解。
（3）与硫酸及高锰酸钾反应

取试验溶液1滴，加高锰酸钾试液及硫酸溶液各1滴生成紫色沉淀，加热 $2\mathrm{min}\sim 3\mathrm{min}$ ，再加过氧化氢溶液1滴，紫色不褪。

# 5 废弃的可溶性钡盐试剂处理处置方法

# 5.1 废弃的可溶性钡盐试剂中钡含量测定

废弃的可溶性钡盐试剂中钡离子含量的测定参见附录A。

# 5.2 处理方法

# 5.2.1 方法原理

钡离子与沉淀剂 $\mathrm{(Na_2SO_4}$ 或 $\mathrm{Na_2CO_3 + H_2SO_4}$ )反应。将废弃的可溶性钡盐试剂中钡离子转化成硫酸钡 $\mathrm{(BaSO_4)}$ 沉淀，使废弃的可溶性钡盐试剂得到无害化处理，排放的废水符合GB8978规定。其化学式如下：

$$
\mathrm {B a} ^ {2 -} + \mathrm {S O} _ {4} ^ {2 -} \longrightarrow \mathrm {B a S O} _ {4} \downarrow
$$

$$
\mathrm {B a} ^ {2 -} + \mathrm {C O} _ {3} ^ {2 -} + 2 \mathrm {H} ^ {-} + \mathrm {S O} _ {4} ^ {2 -} - \rightarrow \mathrm {B a S O} _ {4} \downarrow + \mathrm {C O} _ {2} \uparrow + \mathrm {H} _ {2} \mathrm {O}
$$

# 5.2.2 处理方法介绍

将废弃的可溶性钡盐试剂经过溶解、过滤后，得到可溶性钡盐溶液，再加入沉淀剂（ $\mathrm{Na_2SO_4}$ 或 $\mathrm{Na_2CO_3 + H_2SO_4}$ ），然后经浓缩、过滤、干燥，得到硫酸钡。

# 5.2.4 处理条件

沉淀剂加入量与可溶性钡盐试剂中钡含量比值：一般为 $(1.5:1) \sim (2:1)$ ；

沉淀反应时间：一般为 $1\mathrm{h}\sim 2\mathrm{h}$

# 5.3 处置方法

5.3.1废弃的可溶性钡盐试剂经处理后得到的硫酸钡，按照GB5085.32007规定对其进行浸出毒性鉴别。若浸出液总钡含量超过标准值 $100\mathrm{mg / L}$ ，则为具有浸出毒性的危险废物，将其运至具有资质的专业危险废物处理机构进行处置；若浸出液总钡含量不超过标准值 $100\mathrm{mg / L}$ ，按照GB/128992008规定进行分析，若符合要求，可以作为工业沉淀硫酸钡再利用。
5.3.2 处理过程产生的外排废水经过处理后，符合GB8978规定方可排放。

# 附录A

# （资料性附录）

# 钡含量的测定

# A.1 方法提要

对废弃的可溶性钡盐试剂中的 $\mathrm{Ba}^{2+}$ 进行浸取后，以取适量的浸取液，用乙酸铵调节溶液的 $\mathrm{pH}$ 值，在乙酸-乙酸铵缓冲溶液中重铬酸钾与氯化钡均匀生成铬酸钡沉淀，根据铬酸钡沉淀的质量计算氯化钡的含量。

# A.2 试剂

A.2.1 盐酸溶液： $1 + 11$
A.2.2 重铬酸钾溶液： $50~\mathrm{g / L}$
A.2.3 乙酸铵溶液： $75\mathrm{g / L}$
A.2.4 氨水溶液： $1 + 27$
A.2.5 硝酸银溶液： $10\mathrm{g / L}$

# A.3 仪器、设备

A.3.1 玻璃砂坩埚：滤板孔径 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$
A.3.2 电热恒温干燥箱：控制温度 $133^{\circ}C\pm 2^{\circ}C$

# A.3.3 分析步骤

称取适量试样，按照HJ/T2992007中7.4.2规定的浸出毒性浸出方法进行操作，液固比 $10:1$ $(\mathrm{L / kg})$ 计算出所需浸提剂的体积，得到浸出液，并定容于 $1000~\mathrm{mL}$ 容量瓶中，干过滤，弃去 $10~\mathrm{mL}$ 前滤液。用移液管移取适量滤液，置于 $400~\mathrm{mL}$ 烧杯中，加 $5\mathrm{mL}$ 盐酸溶液，加入 $100~\mathrm{mL}$ 水和 $15~\mathrm{mL}$ 重铬酸钾溶液，加热煮沸，在微沸状态下一边搅拌一边缓慢滴加 $10~\mathrm{mL}$ 乙酸铵溶液（ $3\mathrm{min}\sim 4\mathrm{min}$ 内滴完），保温 $5\mathrm{min}$ ，继续在微沸状态下一边搅拌一边滴加 $15~\mathrm{mL}$ 氨水（ $2\mathrm{min}\sim 3\mathrm{min}$ 内滴完）。在约 $80^{\circ}\mathrm{C}$ 的水浴中静置 $30\mathrm{min}$ 后，取出，迅速冷却至室温，用已于 $133^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 下烘至质量恒定的玻璃砂坩埚抽滤，用含少量氨水的蒸馏水 $(\mathrm{pH}$ 为 $7\sim 8)$ 洗涤沉淀至无氯离子反应(用硝酸银溶液检查)，将玻璃砂坩埚和沉淀于 $133^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 下烘至质量恒定。

# A.3.4 结果计算

钡含量以钡(Ba)的质量分数 $\omega$ 计，按公式(A.1)计算：

$$
w = \frac{0.542\times(m_{1} - m_{0})}{m\times V / 1000}\times 100\%$$

式中：

$m_{1}$ 干燥后铬酸钡及玻璃砂坩埚质量的数值，单位为克 $(\mathrm{g})$

$m_0$ 玻璃砂增蜗质量的数值，单位为克 $(\mathrm{g})$

$V$ 移取浸取液体积的数值，单位为毫升 $(\mathrm{mL})$

$m$ 试料质量的数值，单位为克 $(\mathrm{g})$

0.542 铬酸钡 $\mathrm{(BaCrO_4)}$ 换算成钡（Ba）的系数。

取平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不应大于 $0.2\%$ 。
