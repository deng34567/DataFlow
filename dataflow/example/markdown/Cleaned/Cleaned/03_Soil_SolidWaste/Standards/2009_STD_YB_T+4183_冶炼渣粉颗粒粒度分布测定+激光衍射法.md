# 冶炼渣粉颗粒粒度分布测定激光衍射法

# 1 范围

本标准规定了冶炼渣粉颗粒粒度分布测定——激光衍射法的原理、术语和定义、符号、试验设备、进样方法和分散介质、试验条件、测试步骤、试验结果的重复性、测试报告。

本标准适用于冶炼渣(主要包括钢渣、粒化高炉矿渣、化铁炉渣、铁合金渣等)制备的粉体材料。

# 3 激光衍射法测定原理

一个有代表性的粉体试样，以适当浓度在液体或气体介质中良好分散（即颗粒之间相互分离，不团聚）后通过激光束，这时光束将被试样颗粒散射或阻挡而产生变化了的光信号，该光信号的强弱和空间角度分析与颗粒大小之间有对应关系，反映该关系的米氏光散射理论及相关数据事先保存在与仪器配套的计算机中；该光信号被传感器接收并转换成一组数字化的电信号后输入计算机中，计算机根据接收到的电信号和事先保存在计算机中的米氏光散射理论及相关数据，就可以计算出被测试样的粒度分布。

以液体为介质输送并分散的试样，称为湿法进样；以气体为介质输送并分散的试样，称为干法进样。本标准测定采用湿法进样。

# 4 术语和定义

下列术语和定义适用于本标准。

4.1

遮光比 obscuration

指测量用的照明光束经被测样品颗粒阻挡的部分与照明光强的比值。

4.2

量程范围 range

仪器在一个量程挡内，可以测量的粒度范围。

4.3

冶炼渣粉 smelting slag powder

冶炼渣粉是指冶炼渣经破碎、磁选除铁处理后粉磨达到一定细度的产品。

# 5符号

下列符号适用于本标准。

$D_{10}$ ：表示在累积粒度分布曲线中， $10\%$ 体积的颗粒直径比此值小，单位为微米 $(\mu \mathrm{m})$ 。

$D_{50}$ ：颗粒的中位径，为体积基准，即 $50\%$ 体积的颗粒直径小于或大于这个值，单位为微米 $(\mu \mathrm{m})$ 。

$D_{90}$ ：表示在累积粒度分布曲线中， $90\%$ 体积的颗粒直径比此值小，单位为微米 $(\mu \mathrm{m})$ 。

$D_{(4,3)}$ ：体积平均粒径，是粒径对体积的加权平均，单位为微米 $(\mu \mathrm{m})$ 。

$D_{(3,2)}$ ：表面积平均粒径，是粒径对表面积的加权平均，单位为微米 $(\mu \mathrm{m})$ 。

$X_0$ ：特征粒径，由Rosin-Ramml-Bennet(简称RRB表达式)得到，特指筛余为 $36.8\%$ 时所对应的颗粒粒径，单位为微米 $(\mu \mathrm{m})$ 。

$n$ ：均匀性系数。由RRB表达式得到，表示粒度分布宽窄的参数。

# 6 试验设备

# 6.1 激光粒度分析仪

符合本标准附录A的规定，且应备有搅拌、超声、循环的样品分散系统。

# 6.2 方孔筛

符合GB/T6003.1的规定的 $300\mu \mathrm{m}$ 方孔筛。

# 7 进样方法和分散介质

# 7.1 进样方法

采用湿法进样。

# 7.2 分散介质

采用无水乙醇为分散介质。无水乙醇中乙醇含量应符合色谱纯的要求，即含量大于 $99.5\%$

# 8 试验条件和仪器校准

# 8.1 试验条件

室温在 $10^{\circ}\mathrm{C}\sim 30^{\circ}\mathrm{C}$ 之间，相对湿度不大于 $70\%$ ，室内通风良好，无腐蚀性气体，避免阳光直射。

# 8.2 仪器校准

应按A.1.4的规定对仪器进行校准。

有下列情况之一者进行仪器校准：

首次使用前；

仪器维修后；

测试300个样品后。

# 9 测试步骤

# 9.1 样品处理和样品要求

9.1.1 检验用冶炼渣粉样品应通过 $300\mu \mathrm{m}$ 方孔筛，并确认其金属铁含量小于 $2\%$

9.1.2 在进行冶炼渣粉颗粒粒度分布测定前应将样品混匀后取样，取样方法应符合GB/T19077.1的有关规定。

# 9.2 开机

9.2.1 确认供电状况是否正常。
9.2.2 接通电源，根据仪器生产厂家要求预热仪器，不得少于 $10\mathrm{min}$
9.2.3 输入样品名称、样品编号等有关信息。

# 9.3 测试

9.3.1 在样品池中加入适量的无水乙醇，然后加入准备好的被测样品，按附录A的要求进行试验操作，最后打印出分析结果并保存。

# 激光粒度分析仪

# A.1 技术要求和主要性能指标

# A.1.1 仪器使用环境要求

温度为 $10^{\circ}\mathrm{C}\sim 30^{\circ}\mathrm{C}$ ，相对湿度不大于 $70\%$

# A.1.2 光源

半导体激光器或He-Ne激光器。

# A. 1.3 量程范围

$0.1\mu \mathrm{m}\sim 300\mu \mathrm{m}_{\circ}$

# A.1.4 校准

激光粒度分析仪的校准应采用符合国家颗粒粒度的标准样品进行校准。经供需双方协商，也允许采用国外标准样品进行校准。校准的粒径点为 $D_{10},D_{50},D_{90}$ ，或为从 $2\mu \mathrm{m}$ 到 $60\mu \mathrm{m}$ 之间的5个粒径点，该5个粒径点应大致成倍数关系。在上述粒径点上，对应测得的值与标准颗粒标称值的绝对误差应符合GB/T19077.1中关于准确性的规定。

# A. 1.5.3 测量系统参数

# A.2 安装要求

# A.2.1 对基础设施的要求

# A.2.1.1 安装要求

仪器应安装在少尘、无烟的环境中。室温要稳定，没有明显的气流。没有直射阳光。地面不能有明显的震动。

# A.2.1.2 供电要求

要求 $220\mathrm{V},50\mathrm{Hz} / 60\mathrm{Hz}$ ，有三相插座且接地线良好，并且严禁将零线和地线合接。

# A.2.2 工作台

仪器的测量单元和计算机等应安装在坚实的工作台上。

# A.3 安全注意事项

# A.3.1 激光安全问题

激光粒度分析仪的激光器虽功率不高，但按激光器防护等级，属Ⅱ级防护，直射人人眼仍会造成眼睛的伤害，应严禁直接用眼睛对着激光束。

# A.3.2 电器安全注意事项

仪器插电源线，或在仪器的各单元之间作电气或信号连接时，必须确保电源开关是断开的。否则有可能造成人体触电或仪器损坏。
