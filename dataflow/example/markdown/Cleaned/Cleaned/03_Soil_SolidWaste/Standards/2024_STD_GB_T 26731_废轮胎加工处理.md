# 4 加工处理方法

# 41 通则

废轮胎加工处理方法主要有制取硫化橡胶粉制取再生橡胶裂解处理及原型利用应优先选取制取硫化橡胶粉制取再生橡胶裂解处理等材料回收利用方法其回收利用产物如橡胶粉再生橡胶热裂解再生炭黑、废轮胎(橡胶)再生油,可以替代部分石化基材料,用于制造轮胎、橡胶产品、建材等。

注材料回收是指将废轮胎重新加工成用于各种用途的产品或材料的处理过程

# 42 制取硫化橡胶粉

# 421 概述

将废轮胎破碎、研磨成不同粒径的硫化橡胶粉。制取硫化橡胶粉的工艺流程是在胎圈钢丝分离之后采用逐级粉碎的方式对废轮胎进行处理达到要求的粒径

制取硫化橡胶粉的主要方法包括常温法和冷冻法

# 422 常温法

在常温条件下通过剪切撕裂碾压等方式把废轮胎制成硫化橡胶粉其主要方法有滚切法辊筒法磨盘法锥磨法等工艺流程示意图见图

常温法制取硫化橡胶粉的主要设备有胎圈分离机破碎机粗碎机破胶机磁选机纤维分离机可选细碎机筛选机等

![](images/6e8aadee7904c7674f01b4416ac7755c1ab75b1b97c09b803ba76a68ea1f28e1.jpg)
图1 常温法制取硫化橡胶粉工艺流程示意图(以全钢子午线轮胎为例)

# 423 冷冻法

在橡胶玻璃化温度以下将废轮胎制成粒径较小的精细硫化橡胶粉其主要方法有液氮冷却法空气涡轮膨胀机制冷法等。

冷冻法制取硫化橡胶粉的主要设备有胎圈分离机、破碎机、粗碎机、破胶机、磁选机、纤维分离机、细碎机筛选机冷冻设备等

# 43 制取再生橡胶

# 431 概述

硫化橡胶粉经加热机械并通过化学作用塑化或单独的机械作用塑化使其成为能够再次加工和硫化的新橡胶共混体即再生橡胶在这个过程中橡胶交联网络硫化网络随机发生断裂

废轮胎制取再生橡胶的加工处理设备主要有能量发射器、动态脱硫罐、挤出机、密炼机和炼胶机等。

制取再生橡胶的方法分为物理法和化学法

# 432 物理法

废轮胎经过粉碎处理后达到一定粒径利用外加能量如机械力微波超声波超临界 $\mathrm { C O _ { 2 } }$ 等破坏废轮胎橡胶交联键制取再生橡胶工艺流程示意图见图

# 433 化学法

废轮胎经过粉碎处理后达到一定粒径在一定温度条件下利用化学助剂并借助机械力作用或单独利用化学助剂破坏橡胶交联键制取再生橡胶化学法包括动态脱硫法力化学法等工艺流程示意图见图

动态脱硫法是把橡胶颗粒和化学助剂通过高温高压在动态脱硫罐中进行脱硫的工艺。

力化学法是在一定的温度条件下通过化学助剂的作用并结合机械剪切力降低橡胶颗粒的内聚能和橡胶网络的结合力使橡胶颗粒脱硫的工艺

# 44 裂解处理

# 441 概述

在无氧或贫氧的裂解反应装置中将废轮胎块粒加热进行热裂解或催化裂解获得废轮胎橡胶再生油热裂解再生炭黑热裂解回收钢丝和不凝可燃气等

裂解处理的方法主要为工业连续化裂解方法即将废轮胎块粒或废轮胎块粒和催化剂连续送入裂解反应装置实现连续产出油气固态产物的工业连续化生产方法裂解处理工艺主要分为热裂解工艺和催化裂解工艺工艺流程示意图见图

废轮胎裂解处理的主要设备及装置有加热装置裂解反应装置产品收集装置烟气净化装置粉碎机磁选机及其他辅助设备等

# 442 热裂解工艺

在无氧或贫氧的裂解反应装置中将废轮胎破碎成胶块粒后直接加热至 $3 5 0 \mathrm { ~ \textdegree C } \sim 5 5 0 \mathrm { ~ \textdegree C }$ 或更高的温度)进行裂解的工艺。

# 443 催化裂解工艺

在无氧或贫氧的裂解反应装置中将废轮胎破碎成胶块粒后加热至 $3 8 0 \ { ^ \circ } \mathrm { C } \sim 4 1 0 \ { ^ \circ } \mathrm { C }$ 在催化剂的作用下进行裂解的工艺

# 45 原型利用

直接利用轮胎特有原形或通过部件解体或对其捆绑裁剪冲切等方式将废轮胎改变成有利用价值的物品如制作鞋底垫片船舶的护舷弹性防护网加固坡面消音墙等原型利用的工艺流程示意图见图。
