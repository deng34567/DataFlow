# 有色金属工业固体废物浸出毒性试验方法标准

UDC 628.191 :669.2/.8 :543.06 GB 5086-85

Test method standard for extraction procedure toxicity of solid waste in nonferrous metal industry

为鉴别有色金属工业固体废物的浸出毒性，特制订本标准。

# 1 试样采集与制备

# 1.1 取样

连续生产时，每8h采集不少于 $1\mathrm{kg}$ 的单个样品，混合缩分成日平均样 $1\mathrm{kg}$ ，连续取5天，共 $5\mathrm{kg}$ ，混匀，为代表性样品。

间歇生产时，随同生产采集不少于 $2\mathrm{kg}$ 的单个样品，混合缩分成平均样 $5\mathrm{kg}$ ，为代表性样品。

在固体废物堆取样时，按等距离五点、深度距堆表面为 $0.3\sim 0.5\mathrm{m}$ 处取单个样品各 $4\mathrm{kg}$ ，按四分法混合缩分，取平均样 $5\mathrm{kg}$ ，为代表性样品。

# 1.2 试样制备

样品在室温条件下自然干燥，水分小于 $5\%$ 。污泥样品的水分应小于 $70\%$ 。当污泥样品水分大于 $70\%$ 时，需经过滤，滤液在 $70^{\circ}\mathrm{C}$ 蒸发浓缩，浓缩液返回滤饼，并在室温条件下自然干燥，直至水分小于 $70\%$ 。

当样品粒度小于 $5\mathrm{mm}$ 时，取原样混匀，用四分法缩分，装瓶备用。

当样品粒度有部分大于 $5\mathrm{mm}$ 时，则以 $5\mathrm{mm}$ 筛子筛分， $+5\mathrm{mm}$ 级别破碎筛分，直至全部通过 $5\mathrm{mm}$ 筛孔，混匀，用四分法缩分，装瓶备用。

当处理大块样品时，则破碎筛分，直至全部通过 $5\mathrm{mm}$ 筛孔，样品混匀后用四分法缩分，装瓶备用。

# 2 水分测定

取试样 $20\mathrm{g}$ ，在 $105^{\circ}\mathrm{C}$ 下燥，恒重至重量差小于 $1\mathrm{mg}$ ，计算水分含量。

# 3 浸出方法

# 3.1 仪器与材料

3.1.1 容器： $\phi 130 \times 150$ ，2L，带盖，材质为聚乙烯。
3.1.2 振荡器：往复水平振荡器。
3.1.3 滤膜：孔径 $0.45\mu \mathrm{m}$ 。
3.1.4 滤液瓶：200ml锥形瓶，带磨口塞。
3.2 浸出条件
3.2.1 试样重量： $100\mathrm{g}$ （干基）。
3.2.2 溶剂：蒸馏水， $\mathrm{pH}$ 为 $5.8\sim 6.3$ ，可以用氢氧化钠或盐酸调节。
3.2.3 固液比： $1:10$

3.2.4 振荡频率： $120 \pm 5$ 次 $/ \mathrm{min}$ 。
3.2.5 振幅： $40\mathrm{mm}$ 。
3.2.6 振荡时间： $8\mathrm{h}$ 。
3.2.7 试验温度：室温（ $25 \pm 5^{\circ} \mathrm{C}$ ）。
3.2.8 放置时间： $16\mathrm{h}$

# 3.3 操作步骤

取样 $100 \mathrm{~g}$ (干基), 置于 $2 \mathrm{~L}$ 聚乙烯塑料瓶中, 加蒸馏水 $1 \mathrm{~L}$ (将试样中的水计入), 固液比为 $1: 10$ 。将塑料瓶垂直固定于振荡器上, 调节振荡频率为 $120 \mathrm{~次} / \mathrm{min}$ , 振幅 $40 \mathrm{~mm}$ 。试验温度为室温 $(25 \pm 5^{\circ} \mathrm{C})$ 。振荡时间 $8 \mathrm{~h}$ , 放置 $16 \mathrm{~h}$ 。以玻璃漏斗、 $0.45 \mu \mathrm{m}$ 滤膜过滤, 用于锥形瓶接取滤液 (即浸出液), 分析浸出液中有害物质含量。

# 4 浸出液中有害物质分析

# 4.1 有害物质种类

第一类：Hg、Cd、As、Pb。

必须分析，不得省略。

第二类：Cu、Zn、Cr、Ni、Be、F。

可根据实际情况，适当增减。

浸出液中有害物质的浓度以 $\mathrm{mg / L}$ 计。

# 4.2 分析方法

按现行的《环境监测分析方法》执行。当有害物质的浓度超出《环境监测分析方法》的分析范围时，可采用其他适用的分析方法。

# 5 数据处理与报告

每个样品均做平行浸出试验，浸出液的分析项目，测定二次，从四次分析数据中剔除特异值，然后用算术平均，给出标准差，填写试验报告。
