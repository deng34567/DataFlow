为贯彻《中华人民共和国环境保护法》和《中华人民共和国土壤污染防治法》，保护生态环境，保障人体健康，规范土壤中石油类的测定方法，制定本标准。

警告：实验中所使用的四氯乙烯对人体健康有害，标准溶液配制、样品制备以及测定过程应在通风橱内进行，操作时应按规定要求佩戴防护器具，避免接触皮肤和衣物。

# 1 适用范围

本标准规定了测定土壤中石油类的红外分光光度法。

本标准适用于土壤中石油类的测定。

当取样量为 $_ { 1 0 \mathrm { ~ g ~ } }$ ，提取液体积为 $5 0 \ \mathrm { m l }$ ，使用 $4 0 \ \mathrm { m m }$ 石英比色皿时，方法检出限为$4 \mathrm { m g / k g }$ ，测定下限为 $1 6 \mathrm { m g / k g }$ 。

# 3 术语和定义

下列术语和定义适用于本标准。

# 3.1 石油类 petroleum

指在本标准规定的条件下，能够被四氯乙烯提取且不被硅酸镁吸附，在波数为 2930$\mathrm { c m } ^ { - 1 }$ 、 $2 9 6 0 ~ \mathrm { c m } ^ { - 1 }$ 、 $3 0 3 0 \mathrm { c m } ^ { - 1 }$ 全部或部分谱带处有特征吸收的物质。

# 4 方法原理

土壤用四氯乙烯提取，提取液经硅酸镁吸附，除去动植物油等极性物质后，测定石油类。石油类的含量由波数分别为 $2 9 3 0 ~ \mathrm { c m ^ { - 1 } }$ （ $\mathrm { C H } _ { 2 }$ 基团中C—H键的伸缩振动）、 $2 9 6 0 ~ \mathrm { c m } ^ { - 1 }$ （ $\mathrm { C H } _ { 3 }$ 基团中 C—H 键的伸缩振动）和 $3 0 3 0 ~ \mathrm { c m } ^ { - 1 }$ （芳香环中 C—H 键的伸缩振动）处的吸光度 A2930、$\mathbf { A } _ { 2 9 6 0 }$ 和 $\mathbf { A } _ { 3 0 3 0 }$ ，根据校正系数进行计算。

# 7 样品

# 7.1 样品采集和保存

# 7.4 试样的制备

# 8 分析步骤

# 8.1 校准

分别移取 $2 . 0 0 \mathrm { ~ m l }$ 正十六烷标准使用液（5.10）、 $2 . 0 0 \mathrm { ~ m l }$ 异辛烷标准使用液（5.12）和$1 0 . 0 0 \mathrm { ~ m l }$ 苯标准使用液（5.14）于3个 $1 0 0 \mathrm { ~ m l }$ 容量瓶中，用四氯乙烯（5.1）定容至标线，摇匀。正十六烷、异辛烷和苯标准溶液的浓度分别为 $2 0 . 0 \mathrm { m g / L }$ 、 $2 0 . 0 \mathrm { m g / L }$ 和 $1 0 0 \mathrm { m g / L }$ 。

以 $4 0 \mathrm { m m }$ 石英比色皿加入四氯乙烯（5.1）为参比，分别测量正十六烷、异辛烷和苯标准溶液在 $2 9 3 0 ~ \mathrm { { c m } ^ { - 1 } }$ 、 $2 9 6 0 ~ \mathrm { c m } ^ { - 1 }$ 、 $3 0 3 0 ~ \mathrm { c m } ^ { - 1 }$ 处的吸光度 A 2930、A 2960、 $\mathbf { A } _ { 3 0 3 0 }$ 。将正十六烷、异辛烷和苯标准溶液在上述波数处的吸光度按照公式（1）联立方程式，经求解后分别得到相应的校正系数 $X$ Y $Z$ 和 $F$ 。

$$
\rho_ {1} = X \cdot A _ {2 9 3 0} + Y \cdot A _ {2 9 6 0} + Z \cdot \left(A _ {3 0 3 0} - \frac {A _ {2 9 3 0}}{F}\right) \tag {1}
$$

式中： $\rho _ { 1 }$ ——石油类标准溶液浓度， $\mathrm { m g / L }$

A2930、A 2960、A3030—各对应波数下测得的吸光度；

$X$ ——与 $\mathrm { C H } _ { 2 }$ 基团中 C—H 键吸光度相对应的系数， $\mathrm { m g / L / }$ 吸光度；

Y——与 $\mathrm { C H } _ { 3 }$ 基团中 C—H 键吸光度相对应的系数， $\mathrm { { m g / L / } }$ 吸光度；

Z——与芳香环中 C—H 键吸光度相对应的系数， $\mathrm { m g / L / }$ 吸光度；

$F .$ ——脂肪烃对芳香烃影响的校正因子，即正十六烷在 $2 9 3 0 \mathrm { c m } ^ { - 1 }$ 与 $3 0 3 0 \mathrm { c m } ^ { - 1 }$ 处的吸光度之比。

对于正十六烷和异辛烷，由于其芳香烃含量为零，即 A $A _ { 3 0 3 0 } - \frac { A _ { 2 9 3 0 } } { F } = 0$ 2930A ，则有：

$$
F = \frac {A _ {2 9 3 0} (H)}{A _ {3 0 3 0} (H)} \tag {2}
$$

$$
\rho (H) = X \cdot A _ {2 9 3 0} (H) + Y \cdot A _ {2 9 6 0} (H) \tag {3}
$$

$$
\rho (I) = X \cdot A _ {2 9 3 0} (I) + Y \cdot A _ {2 9 6 0} (I) \tag {4}
$$

由公式（2）可得 $F$ 值，由公式（3）和（4）可得 $X$ 和 $Y$ 值。对于苯，则有：

$$
\rho (B) = X \cdot A _ {2 9 3 0} (B) + Y \cdot A _ {2 9 6 0} (B) + Z \cdot \left(A _ {3 0 3 0} (B) - \frac {A _ {2 9 3 0} (B)}{F}\right) \tag {5}
$$

由公式（5）可得 Z 值。

式中： $\rho ( H ) .$ ——正十六烷标准溶液的浓度， $\mathrm { m g / L }$ ；

$\rho ( I )$ ——异辛烷标准溶液的浓度， $\mathrm { m g / L }$ ；

$\rho ( B )$ ——苯标准溶液的浓度， $\mathrm { m g / L }$ ；

$A _ { 2 9 3 0 } ( H )$ 、 $A _ { 2 9 6 0 } ( H )$ 、 $A _ { 3 0 3 0 } ( H ) \cdot$ ——各对应波数下测得正十六烷标准溶液的吸光度；

A2930(I)、A2960(I)、A3030(I) ——各对应波数下测得异辛烷标准溶液的吸光度；

A2930(B)、A2960(B)、 $A _ { 3 0 3 0 } ( B )$ ——各对应波数下测得苯标准溶液的吸光度。

注：红外测油仪或红外分光光度计出厂时如果设定了校正系数，可以直接按 11.3.1 进行校正系数的检验。

# 9 结果计算与表示

# 9.1 结果计算

土壤中石油类的含量 $w$ （ $\bf { \nabla } \cdot \bf { m g / k g } )$ ），按照公式（6）进行计算：

$$
w = \frac {\rho_ {2} \cdot V}{m \cdot W _ {d m}} \tag {6}
$$

式中：w—土壤中石油类的含量， $\mathrm { m g / k g }$ ；

$\rho _ { 2 }$ —提取液中石油类浓度， $\mathrm { m g / L }$ ；

V——提取液体积，ml；

m—土壤样品质量，g；

$W _ { d m }$ —土壤干物质含量， $\%$ 。

# 9.2 结果表示

测定结果小数点后位数的保留与方法检出限一致，最多保留 3位有效数字。

# 10 精密度和准确度

# 10.1 精密度

六家实验室分别对石油类浓度约为 $3 0 ~ \mathrm { m g / k g }$ 的花园土壤统一样品、 $7 0 ~ \mathrm { m g / k g }$ 的污灌区土壤统一样品和 $1 5 0 \mathrm { m g / k g }$ 的炼铁厂土壤统一样品进行 6次重复测定，实验室内相对标准偏差分别为 $5 . 0 \% \sim 7 . 4 \%$ 、 $3 . 9 \% { \sim } 5 . 9 \%$ 和 $1 . 9 \% { \sim } 4 . 0 \%$ ；实验室间相对标准偏差分别为 $3 . 9 \%$ 、

$2 . 7 \%$ 和 $2 . 4 \%$ ；重复性限分别为 $5 ~ \mathrm { m g / k g }$ 、 $9 ~ \mathrm { m g / k g }$ 和 $1 1 ~ \mathrm { m g / k g }$ ；再现性限分别为 $6 ~ \mathrm { m g / k g }$ 、$1 0 \mathrm { m g / k g }$ 和 $1 4 \mathrm { m g / k g }$ 。

# 10.2 准确度

# 11 质量保证和质量控制

# 11.1 四氯乙烯品质检验

四氯乙烯须避光保存。使用前须按照（5.1）进行四氯乙烯品质检验和判定，确认符合要求后方可使用。

# 11.2 空白试验

每20个样品或每批次（ ${ } \leqslant 2 0 ~ $ 个样品/批）至少做两个实验室空白试验，空白试验结果应低于方法检出限。

# 11.3 校正系数检验

每批样品均应进行校正系数的检验，使用时根据所需浓度，取适量的石油类标准使用液（5.16），以四氯乙烯为溶剂配制适当浓度的石油类标准溶液，与试样测定（8.2）相同的步骤进行测定，按照公式（1）计算石油类标准溶液的浓度。如果测定值与标准值的相对误差在 $\pm 1 0 \%$ 以内，则校正系数可采用，否则重新测定校正系数并检验，直至符合条件为止。

也可使用有证标准物质/样品进行检验。

# 11.4 平行样

每20个样品或每批次（ $\leqslant 2 0$ 个样品/批）样品应测定一个平行样品，平行样的相对偏差应 $\text{‰}$ 。

# 11.5 基体加标

每20个样品或每批次（ ${ } \leqslant 2 0 ~ $ 个样品/批）样品应测定一个基体加标样品，加标回收率应控制在 $7 0 \% \sim 1 1 0 \%$ 之间。

# 12 废物处理

实验中产生的废物应分类收集，并做好相应标识，委托有资质的单位进行处理。

# 13 注意事项

13.1 同一批样品测定所使用的四氯乙烯应来自同一瓶，如样品数量多，可将多瓶四氯乙烯混合均匀后使用。
13.2 样品制备间应清洁、无污染，样品制备过程中应远离有机气体，使用的所有工具都应进行彻底清洗，防止交叉污染。
