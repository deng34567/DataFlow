# 中华人民共和国国家标准

# 固体废物六价铬的测定

# 二苯碳酰二肼分光光度法

GB/T15555.4—1995

Solid waste-Determination of chromium (VI)

1,5-Diphenylcarbohydrazide spectrophotometric method

# 1 主题内容与适用范围

1.1本标准规定了固体废物浸出液中六价铬的测定，用二苯碳酰二肼分光光度法。

1.2 本标准适用于固体废物浸出液中六价铬的测定。

# 1.2.1 测定范围

试料为 $50~\mathrm{mL}$ ，使用 $30~\mathrm{mm}$ 光程比色皿，方法的检出限为 $0.004\mathrm{mg / L}$ 。使用 $10\mathrm{mm}$ 光程比色皿，测定上限为 $1.0\mathrm{mg / L}$ 。

# 1.2.2 干扰

试液有颜色、混浊，或者有氧化性、还原性物质及有机物等均干扰测定。铁含量大于 $1.0\mathrm{mg / L}$ 也干扰测定。钼、汞与显色剂生成络合物有干扰，但是在方法的显色酸度下，反应不灵敏。钒浓度大于 $4.0\mathrm{mg / L}$ 干扰测定，但在显色 $10\mathrm{min}$ 后，可自行退色。

# 2 原理

在酸性溶液中，六价铬与二苯碳酰二肼反应生成紫红色络合物。于最大吸收波长 $540\mathrm{nm}$ 进行分光光度法测定。

# 3 试剂

本标准所用试剂除另有说明外，均用符合国家标准或专业标准的分析纯试剂和蒸馏水或同等纯度的水；

3.1 丙酮 $\mathrm{(C_3H_6O)}$ 。
3.2 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ $\rho = 1.84\mathrm{g / mL}$
3.3 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ ， $\rho = 1.69\mathrm{g / mL}$
3.4 重铬酸钾 $\mathrm{(K_2Cr_2O_7}$ ，优级纯）。
3.5 二苯碳酰二肼 $(\mathrm{C_{13}H_{14}N_4O})$ 。
3.6 硫酸溶液， $1 + 1$

将硫酸(3.2)缓慢加到同体积的水中，边加边搅，待冷却后使用。

3.7 磷酸溶液， $1 + 1$
将磷酸(3.3)与等体积水混匀。
3.8 高锰酸钾 $(\mathrm{KMnO}_4)$ ， $4\%$
3.9 脲素溶液， $20\%$

将尿素 $\left[\left(\mathrm{NH}_{2}\right)_{2} \mathrm{CO}\right] 20 \mathrm{~g}$ , 溶于水中, 并稀释至 $100 \mathrm{~mL}$ 。

3.10 亚硝酸钠， $2\%$

将亚硝酸钠 $(\mathrm{NaNO}_2)2\mathrm{g}$ ，溶于水中，并稀释至 $100~\mathrm{mL}$

3.11 铬标准贮备液， $0.1000\mathrm{mg}\mathrm{Cr}^{6 + } / \mathrm{mL}$

称取于 $120^{\circ}\mathrm{C}$ 下烘 $2\mathrm{h}$ 的重铬酸钾(3.4)0.2829g，用少量水溶解后，移入 $1000\mathrm{mL}$ 容量瓶中，用水稀释至标线，摇匀。

3.12 铬标准溶液， $1.00\mu \mathrm{g / mL}$

吸取 $5.0\mathrm{mL}$ 铬标准贮备溶液(3.11)于 $500~\mathrm{mL}$ 容量瓶中，用水稀释至标线，摇匀。用时现配。

3.13 铬标准溶液， $5.00\mu \mathrm{g / mL}$

吸取 $25.00\mathrm{mL}$ 铬标准贮备溶液(3.11)于 $500~\mathrm{mL}$ 容量瓶中，用水稀释至标线，摇匀。

3.14 显色剂1：

称取二苯碳酰二肼(3.5) $0.2\mathrm{g}$ ，溶于 $50\mathrm{mL}$ 丙酮(3.1)中，加水稀释至 $100\mathrm{mL}$ ，摇匀，于棕色瓶中，在低温下保存。

3.15 显色剂2：

称取二苯碳酰二肼(3.5) $2.0\mathrm{g}$ ，溶于 $50~\mathrm{mL}$ 丙酮(3.1)中，加水稀释至 $100~\mathrm{mL}$ ，摇匀，于棕色瓶中，在低温下保存。

注：显色剂颜色变深，则不能使用。

# 5 步骤

5.1 样品的保存

将浸出液用氢氧化钠调 $\mathrm{pH}$ 值为8。在 $24\mathrm{h}$ 内测定。

5.2 样品的预处理

5.2.1 无还原性物质及有机物、色度等干扰时，可直接取试料测定。
5.2.2 有干扰物质存在时，可按下列步骤处理后再测定。
5.2.2.1 如试样色度影响测定时，可按下述方法校正：

另取一份试料，按(5.3)步骤（只是取 $2.0\mathrm{mL}$ 丙酮代替显色剂），以水作参比测定试料的吸光度。扣除此色度，校正吸光度值。

5.2.2.2 还原性物质的消除

取适量试样于 $50~\mathrm{mL}$ 的比色管中作为试料，中和后用水稀释至标线，加显色剂(3.15)4.0mL，摇均，放 $5\mathrm{min}$ 后，加硫酸(3.6)1.0mL，摇均，放 $10\mathrm{min}$ 后，按(5.3.3)步骤测定，可消除 $\mathrm{Fe}^{2 - }$ 、 $\mathrm{SO}_3^{-2}$ 、 $\mathrm{S}_2\mathrm{O}_3^{-2}$ 等还原性物质的干扰。也可分离三价铬后，用过硫酸铵将还原性物质氧化后再测定。

5.2.2.3 有机物的消除

先用氢氧化锌沉淀分离掉三价铬，再用酸性高锰酸钾氧化分解有机物。取 $50.0\mathrm{mL}$ 试样（六价铬不超过 $10\mu \mathrm{g}$ ）于 $150~\mathrm{mL}$ 锥形瓶中，中和后，放几粒玻璃珠，加入硫酸(3.6) $0.5\mathrm{mL}$ ，磷酸(3.7) $0.5\mathrm{mL}$ ，摇匀，加高锰酸钾溶液(3.8)2滴，如紫红色消退，再加高锰酸钾溶液保持红色不退，加热煮沸至溶液剩 $20~\mathrm{mL}$ 左右，冷却后用中速定量滤纸过滤，于 $50~\mathrm{mL}$ 比色管中，用水洗数次，洗液与滤液合并，向比色管中加脲素溶液(3.9) $1.0~\mathrm{mL}$ ，摇匀，滴加亚硝酸钠溶液(3.10)一滴，摇匀，至溶液红色刚退，稍停片刻，待溶液中气泡全排后，移至 $50~\mathrm{mL}$ 的比色管中，用水稀释至标线，加显色剂(3.14) $2.0~\mathrm{mL}$ ，摇匀，放 $10\mathrm{min}$ 后按(5.3.3)步骤测定。

# 5.2.2.4 次氯酸盐氧化性物质的消除

取适量试样于 $50~\mathrm{mL}$ 的比色管中作为试料，中合后用水稀释至标线，加硫酸(3.6) $0.5\mathrm{mL}$ 、磷酸(3.7) $0.5\mathrm{mL}$ 、脲素(3.9) $1.0\mathrm{mL}$ 摇匀，逐滴加入亚硝酸钠溶液(3.10)，边加边摇，使溶液中气体完全排除后加显色剂(3.14) $2.0\mathrm{mL}$ ，以后按(5.3.3)步骤测定。

# 5.3 测定

5.3.1 吸取(5.2.1)或(5.2.2)的试样于 $50~\mathrm{mL}$ 比色管中（6价铬不超过 $10~\mu \mathrm{g}$ ），中和后用水稀释至标线。
5.3.2 加入硫酸(3.6)0.5mL、磷酸(3.7)0.5mL、摇匀，加显色剂(3.14)2.0mL，摇匀，放置10min。
5.3.3 用10或 $30\mathrm{mm}$ 光程比色皿，于 $540\mathrm{nm}$ 处，以水作参比，测定吸光度，减去空白试验(5.4)的吸光度，从校准曲线(5.5)上查得6价铬的量。

# 5.4 空白试验

以 $50~\mathrm{mL}$ 水代替试料，按照测定步骤(5.3)作空白试验。

# 5.5 校准曲线的绘制

向9支 $50~\mathrm{mL}$ 具塞比色管中，分别加入铬标准溶液(3.12)0.00、0.20、0.50、1.00、2.00、4.00、6.00、8.00、10.00 mL，加水至标线，按(5.3.2)和(5.3.3)步骤显色和吸光度测定，以减去空白的吸光度为纵坐标，6价铬的量 $(\mu \mathrm{g})$ 为横坐标，绘制校准曲线。

# 6 结果的表示

浸出液中六价铬的浓度 $c(\mathrm{mg / L})$ 按下式计算：

$$
c = \frac {m}{V} \tag {1}
$$

式中： $m$ ——从校准曲线上查得试料中六价铬的量， $\mu \mathrm{g}$

$V$ ——试料的体积， $\mathrm{mL}$ 。

# 7 精密度和准确度

7.1 可参考GB7467—87《水质六价铬的测定二苯碳酰二肼分光光度法》。
7.2室内对六价铬浓度为 $185.8\mathrm{mg / L}$ 的铬渣浸出液，6次平均测定的相对标准偏差为 $0.14\%$ 。对六价铬的含量为 $3.176\mu \mathrm{g}$ 的浸出液双样，各加入 $4.00\mu \mathrm{g}$ 的标样，其加标回收率为 $99.4\%$ 和 $103.7\%$ 。

# 注意事项

（参考件）

A1试样中六价铬的浓度高时，可用铬标准溶液(3.15)，并用 $10\mathrm{mm}$ 的光程比色皿测定。
A2 显色酸度在 $0.05\sim 0.3\mathrm{mol / L}(1 / 2\mathrm{H}_2\mathrm{SO}_4)$ 为宜，以 $0.2\mathrm{mol / L}$ 最好。
A3 试料须中和后测定。
A4 所用玻璃仪器均不可用重铬酸钾洗液洗涤。
A5 显色剂的用量一般控制为 $1\mathrm{mol}$ 的6价铬，加入 $1.5\sim 2.0\mathrm{mol}$ 的显色剂。
A6 配制显色剂时若加苯二甲酸酐，在暗处可保存 $30\sim 40\mathrm{d}$
A7 显色剂变为橙色，不可使用。
