# 土壤和沉积物锰的测定火焰原子吸收分光光度法

警示：实验中使用的硝酸、高氯酸具有强氧化性和腐蚀性，盐酸、氢氟酸具有强挥发性和强腐蚀性，试剂配制和样品消解应在通风橱内进行；操作时应按要求佩戴防护器具，避免吸入呼吸道或接触皮肤和衣物。

# 4 原理

土壤和沉积物样品经酸消解后，试样中的锰在空气-乙炔火焰中被原子化，其基态原子对 279.5nm特征谱线产生选择性吸收，吸收强度在一定范围内与锰的浓度成正比。

# 5 干扰和消除

5.1 锰的特征谱线 279.5 nm 附近存在 $2 7 9 . 8 \ \mathrm { n m }$ 和 280.1 nm 两条吸收谱线，对其测定有光谱干扰，选择 $0 . 2 \ \mathrm { n m }$ 的通带宽度可减少干扰。

5.2 试样中浓度低于 $1 0 0 ~ \mathrm { { m g / L } }$ 的钡、铜、镁、镍、铅、锌， $2 0 0 ~ \mathrm { { m g / L } }$ 的钙， $5 0 0 \mathrm { m g / L }$ 的铝， $1 0 0 0 ~ \mathrm { { m g / L } }$ 的铁、钾、钠不干扰锰的测定。

# 6 试剂和材料

6.1 硝酸： $\rho ( \mathrm { { H N O _ { 3 } } ) = 1 . 4 2 g / m L }$ 。
6.2 盐酸： $\rho \left( \mathrm { H C l } \right) { = } 1 . 1 9 ~ \mathrm { g / m L }$ 。
6.3 氢氟酸： $\rho \left( \mathrm { H F } \right) { = } 1 . 4 9 ~ \mathrm { \ g / m L }$ 。
6.4 高氯酸： $\rho \left( \mathrm { H C l } 0 _ { 4 } \right) { = } 1 . 6 8 ~ \mathrm { g / m L }$
6.5 硝酸溶液： $1 + 9 9$ 。
6.6 锰标准贮备液： $\rho \left( \mathrm { M n } \right) { = } 1 0 0 0 \mathrm { m g / L }$ 。可直接购买市售有证标准溶液。
6.7 锰标准使用液： $\rho \left( \mathrm { M n } \right) { = } 1 0 0 ~ \mathrm { m g / L }$ 。准确移取锰标准贮备液（6.6）10.00 mL于 100 mL容量瓶中，用硝酸溶液（6.5）定容，摇匀。
6.8 燃气：乙炔，纯度 ${ \geqslant } 9 9 . \ 9 \%$ 。
6.9 助燃气：空气。

注：除非另有说明，分析时均使用符合国家标准的优级纯试剂，实验用水为符合GB/T 6682中规定的一级水或二级水。

# 9 试验步骤

# 9.1 试样的制备

# 9.1.1 电热板消解

9.1.1.1 称取 $0 . 2 \ \mathrm { g } { \sim } 0 . 3 \ \mathrm { g }$ 样品（8.2），精确到 $0 . 1 \mathrm { \ m g }$ ，置于 $5 0 \ \mathrm { m L }$ 具盖坩埚（7.5）中，用水润湿后加入 6 mL硝酸（6.1），加盖，于通风橱内电热板上 $1 4 0 ~ \mathrm { ~ \textdegree C }$ 加热 $5 0 \mathrm { \ m i n }$ ，使样品初步分解。
9.1.1.2 在初步分解的样品中加入 $2 \ \mathrm { \ m L }$ 硝酸和 6 mL 盐酸（6.2）， $1 5 0 ~ \mathrm { ~ \textdegree C }$ 加热 $6 0 \mathrm { \ m i n }$ ，加入 5 mL氢氟酸（6.3），开盖，继续加热 $5 0 \ \mathrm { \ m i n }$ ，经常摇动坩埚，冷却 $1 0 \mathrm { \ m i n }$ ，加入 2 mL 高氯酸（6.4），于 $1 7 0 \ \mathrm { ~ \textdegree C }$ 加热至冒白烟，当内溶物呈不流动状，停止加热。
9.1.1.3 向坩埚中加入 3 mL 硝酸溶液，温热溶解残渣，冷却后转移至 50 mL 容量瓶中，用硝酸溶液定容，备用。

注：土壤和沉积物样品种类较多，基体差异较大，在消解时可适当调整酸试剂用量。经过消解后的样品若有黑色残渣，表明碳化物未被完全消解，可向坩埚中补加 2 mL硝酸、1 mL氢氟酸和 1 mL高氯酸，在 170 ℃加盖消解 30min后，开盖加热至内溶物呈不流动状。上述过程反复进行直至黑色碳化物消失。

# 9.1.2 微波消解

# 9.4 标准曲线的建立

准确移取锰标准使用液（6.7）0 mL、0.10 mL、0.50 mL、1.00 mL、2.00 mL、3.00 mL于 100 mL容量瓶中，用硝酸溶液定容，摇匀。其中锰的质量浓度分别为 0.00 mg/L、0.10 mg/L、0.50 mg/L、1.00mg/L、 $2 . 0 0 ~ \mathrm { m g / L }$ 、3.00 mg/L。按照仪器测量条件（9.3），用标准曲线零浓度点调节仪器零点，由低浓度到高浓度依次测定标准系列的吸光度，以锰标准系列质量浓度为横坐标，相应的吸光度为纵坐标，建立标准曲线。

注：可根据仪器灵敏度或试样的浓度调整标准系列范围，至少配制 6 个浓度点（含零浓度点）。

注：若试样中锰元素浓度超过标准曲线范围，需用硝酸溶液稀释后重新测定。

# 11 精密度

在同一实验室，由同一操作者使用相同设备，按相同的测试方法，并在短时间内对同一被测对象相互独立进行测试获得的两次独立测试结果的绝对差值不大于这两个测定值的算术平均值的 $1 0 \%$ 。

# 12 质量保证和控制

12.1 每批样品至少做 2 个空白试样，测定结果应低于方法检出限。
12.2 每次分析应建立标准曲线，相关系数应 ${ \geqslant } 0 . 9 9 9$ 。

12.3 每分析 20 个样品或每批样品（少于 20 个样品）应进行标准系列零浓度点和中间浓度点核查。零浓度点测定值应低于方法检出限，中间浓度点测定值与标准值的相对误差应在 $\pm 1 0 \%$ 以内。
12.4 每批样品至少测定 $2 0 \%$ 的平行双样。当样品数量少于 5 个时，应至少测定 1 个平行双样。当样品中锰含量大于 $1 0 0 \mathrm { \ m g / k g }$ ，相对偏差应 $\leqslant 5 \%$ ；当样品中锰含量在 $8 ~ \mathrm { m g / k g } { \sim } 1 0 0 ~ \mathrm { m g / k g }$ 之间，相对偏差应 $\leqslant 1 0 \%$ 。
12.5 每批样品至少测定 $1 0 \%$ 的有证标准样品或基体加标样品，当样品数量少于 10 个时，应至少测定1 个有证标准样品或基体加标样品。有证标准样品的测定结果应在保证值范围内，基体加标样品的回收率应在 $8 0 \% \sim 1 2 0 \%$ 之间。
