# 土壤质量全氮的测定凯氏法

警告：本方法所使用的试剂具有一定的腐蚀性，操作时应尽量避免与这些化学品的直接接触，样品消解过程应在通风橱中进行；热的高氯酸与有机物接触易爆炸，因此消解后的溶液必须冷却后再加入高氯酸。

# 1 适用范围

本标准规定了测定土壤样品中全氮含量的凯氏法。

本标准适用于土壤中全氮的测定。

当取样量为 $1\mathrm{g}$ 时，本标准的方法检出限为 $48\mathrm{mg / kg}$

# 2 规范性引用文件

本标准内容引用了下列文件或其中的条款。凡是不注日期的引用文件，其有效版本适用于本标准。

HJ/T 166 土壤环境监测技术规范

# 3 术语和定义

下列术语和定义适用于本标准。

全氮 total nitrogen

指在本标准规定的条件下，能测定的样品中氮含量的总和，包括有机氮（如蛋白质、氨基酸、核酸、尿素等）、硝态氮、亚硝态氮以及铵态氮，还包括部分联氮、偶氮和叠氮等含氮化合物。

# 4 方法原理

土壤中的全氮在硫代硫酸钠、浓硫酸、高氯酸和催化剂的作用下，经氧化还原反应全部转化为铵态氮。消解后的溶液碱化蒸馏出的氨被硼酸吸收，用标准盐酸溶液滴定，根据标准盐酸溶液的用量来计算土壤中全氮含量。

# 5 试剂和材料

本标准所用试剂除非另有说明，分析时均使用符合国家标准的分析纯化学试剂，实验用水为无氨水(5.1)。

# 5.1 无氨水

每升水中加入 $0.10\mathrm{ml}$ 浓硫酸（5.2）蒸馏，收集馏出液于具塞玻璃容器中，也可使用新制备的去离子水。

5.2 浓硫酸： $\rho(\mathrm{H}_2\mathrm{SO}_4)=1.84\mathrm{g/ml}$ ，优级纯。
5.3 浓盐酸： $\rho(\mathrm{HCl}) = 1.19 \mathrm{~g} / \mathrm{ml}$ 。
5.4 高氯酸： $\rho(\mathrm{HClO}_4) = 1.768 \mathrm{~g} / \mathrm{ml}$ 。

5.5 无水乙醇： $\rho(\mathrm{C}_2\mathrm{H}_6\mathrm{O}) = 0.79\mathrm{g / ml}$ 。
5.6 硫酸钾 $(\mathrm{K}_2\mathrm{SO}_4)$ 。
5.7 五水合硫酸铜 $(\mathrm{CuSO}_4\cdot 5\mathrm{H}_2\mathrm{O})$ 。
5.8 二氧化钛（ $\mathrm{T_iO_2}$ ）：优级纯。
5.9 五水合硫代硫酸钠（ $\mathrm{Na_2S_2O_3\cdot 5H_2O}$ ）。
5.10 氢氧化钠（NaOH）：优级纯。
5.11硼酸（ $\mathrm{H_3BO_3}$ ）：优级纯。
5.12 无水碳酸钠 $(\mathrm{Na}_2\mathrm{CO}_3)$ ：基准试剂。
5.13 催化剂： $200\mathrm{g}$ 硫酸钾(5.6)、 $6\mathrm{g}$ 五水合硫酸铜(5.7)和 $6\mathrm{g}$ 二氧化钛(5.8)于玻璃研钵(6.2)中充分混匀，研细，贮于试剂瓶中保存。
5.14 还原剂：将五水合硫代硫酸钠（5.9）研磨后过 $0.25\mathrm{mm}$ (60目)筛(6.3)，临用现配。
5.15 氢氧化钠溶液： $\rho(\mathrm{NaOH}) = 400 \mathrm{~g} / \mathrm{L}$ 。

称取 $400\mathrm{g}$ 氢氧化钠(5.10)溶于 $500\mathrm{ml}$ 水中，冷却至室温后稀释至 $1000\mathrm{ml}$ 。

5.16硼酸溶液： $\rho (\mathrm{H}_3\mathrm{BO}_3) = 20\mathrm{g / L}$

称取 $20\mathrm{g}$ 硼酸(5.11)溶于水中，稀释至 $1000\mathrm{ml}$ 。

5.17 碳酸钠标准溶液： $c(1/2\mathrm{Na}_2\mathrm{CO}_3) = 0.0500\mathrm{mol/L}$ 。

称取 $2.6498\mathrm{g}$ （于 $250^{\circ}\mathrm{C}$ 烘干4h并置干燥器中冷却至室温）无水碳酸钠（5.12)，溶于少量水中，移入 $1000\mathrm{ml}$ 容量瓶中，用水稀释至标线，摇匀。贮于聚乙烯瓶中，保存时间不得超过一周。

5.18 甲基橙指示液： $\rho = 0.5 \mathrm{~g} / \mathrm{L}$ 。

称取 $0.1\mathrm{g}$ 甲基橙溶于水中，稀释至 $200\mathrm{ml}$

5.19 盐酸标准贮备溶液： $c(\mathrm{HCl}) \approx 0.05 \mathrm{~mol} / \mathrm{L}$ 。

用分度吸管吸取 $4.20 \mathrm{ml}$ 浓盐酸（5.3），并用水稀释至 $1000 \mathrm{ml}$ ，此溶液浓度约为 $0.05 \mathrm{~mol} / \mathrm{L}$ 。其准确浓度按下述方法标定：

用无分度吸管吸取 $25.00\mathrm{ml}$ 碳酸钠标准溶液（5.17）于 $250\mathrm{ml}$ 锥形瓶中，加水稀释至约 $100\mathrm{ml}$ ，加入3滴甲基橙指示液（5.18），用盐酸标准贮备溶液滴定至颜色由桔黄色刚变成桔红色，记录盐酸标准溶液用量。按下式计算其准确浓度：

$$
C = \frac {2 5 . 0 0 \times 0 . 0 5 0 0}{V} \dots \tag {1}
$$

式中： $C$ ——盐酸标准溶液浓度， $\mathrm{mol/L}$ ；

$V$ ——盐酸标准溶液用量，ml。

5.20 盐酸标准溶液： $c(\mathrm{HCl}) \approx 0.01 \mathrm{~mol} / \mathrm{L}$ 。

吸取 $50.00\mathrm{ml}$ 盐酸标准贮备溶液（5.19）于 $250\mathrm{ml}$ 容量瓶中，用水稀释至标线。

5.21 混合指示剂：将 $0.1\mathrm{g}$ 溴甲酚绿和 $0.02\mathrm{g}$ 甲基红溶解于 $100\mathrm{ml}$ 无水乙醇(5.5)中。

# 7 样品

# 7.1 样品的采集和保存

土壤样品采集和保存参照HJ/T166相关规定执行。

# 7.2 试样的制备

将土壤样品置于风干盘中，平摊成 $2\sim 3\mathrm{cm}$ 厚的薄层，先剔除植物、昆虫、石块等残体，用铁锤或瓷质研磨棒压碎土块，每天翻动几次，自然风干。

充分混匀风干土壤，采用四分法，取其两份，一份留存，一份用研磨机（6.1）研磨至全部通过 $2\mathrm{mm}$ （10目）土壤筛（6.3）。取 $10\mathrm{g}\sim 20\mathrm{g}$ 过筛后的土壤样品，研磨至全部通过 $0.25\mathrm{mm}$ （60目）土壤筛（6.3），装于样品袋或样品瓶中。

# 7.3 干物质含量的测定

称取过 $2 \mathrm{~mm}$ （10 目）筛（6.3）后的土壤样品，参照 HJ 613 测定土壤的干物质含量。

# 8 分析步骤

# 8.1 消解

称取适量试样（7.2） $0.2000\mathrm{g} \sim 1.0000\mathrm{g}$ （含氮约 $1\mathrm{mg}$ ），精确到 $0.1\mathrm{mg}$ ，放入凯氏氮消解瓶（6.7）

中，用少量水（约 $0.5\mathrm{ml} \sim 1\mathrm{ml}$ ）润湿，再加入 $4\mathrm{ml}$ 浓硫酸(5.2)，瓶口上盖小漏斗，转动凯氏氮消解瓶（6.7）使其混合均匀，浸泡8小时以上。

使用干燥的长颈漏斗将 $0.5\mathrm{g}$ 还原剂（5.14）加到凯氏氮消解瓶（6.7）底部，置于消解器（或电热板）(6.5)上加热，待冒烟后停止加热。冷却后，加入 $1.1\mathrm{g}$ 催化剂(5.13)，摇匀，继续在消解器（或电热板）(6.5)上消煮。消煮时保持微沸状态，使白烟到达瓶颈1/3处回旋，待消煮液和土样全部变成灰白色稍带绿色后，表明消解完全，再继续消煮1h，冷却。在土壤样品消煮过程中，如果不能完全消解，可以冷却后加几滴高氯酸后再消煮。

注1：消解时温度不能超过 $400^{\circ}\mathrm{C}$ ，以防瓶壁温度过高而使铵盐受热分解，导致氮的损失。

# 8.2 蒸馏

8.2.1 按照图1连接蒸馏装置，蒸馏前先检查蒸馏装置的气密性，并将管道洗净。

8.2.2 把消解液（8.1）全部转入蒸馏瓶中，并用水洗涤凯氏氮消解瓶（6.7） $4\sim 5$ 次，总用量不超过 $80\mathrm{ml}$ ，连接到凯氏氮（或氨氮）蒸馏装置上（6.6）。在 $250\mathrm{ml}$ 锥形瓶（6.9）中加入 $20\mathrm{ml}$ 硼酸溶液（5.16）和3滴混合指示剂（5.21）吸收馏出液，导管管尖伸入吸收液液面以下。将蒸馏瓶成 $45^{\circ}$ 斜置，缓缓沿壁加入 $20\mathrm{ml}$ 氢氧化钠溶液（5.15），使其在瓶底形成碱液层。迅速连接定氮球和冷凝管，摇动蒸馏瓶使溶液充分混匀，开始蒸馏，待馏出液体面积约 $100\mathrm{ml}$ 时，蒸馏完毕。用少量已调节至 $\mathrm{pH}4.5$ 的水洗涤冷凝管的末端。

注2：如果消解后消解瓶中的沉淀物附着在瓶壁上，可加入少量水后使用超声波振荡器将其溶于水中，再完全转移至蒸馏瓶中。

# 8.3 滴定

用盐酸标准溶液（5.20）滴定蒸馏后的馏出液（8.2.2），溶液颜色由蓝绿色变为红紫色，记录所用盐酸标准溶液体积。

注3：如果样品含量大于 $10^{4}\mathrm{mg / kg}$ ，可以改用浓度为 $0.0500\mathrm{mol / L}$ 的盐酸标准贮备溶液（5.19）滴定。

注4：如果使用全自动凯氏定氮仪，按说明书要求进行样品的消解、蒸馏和滴定。

# 8.4 空白试验

凯氏氮消解瓶（6.7）中不加入试样，按照步骤 $8.1\sim 8.3$ 测定，记录所用盐酸标准溶液体积。

# 9 结果计算与表示

土壤中全氮的含量（ $\mathrm{mg/kg}$ ）按公式（2）计算：

$$
\omega_ {N} = \frac {\left(V _ {1} - V _ {0}\right) \times c _ {H C l} \times 1 4 . 0 \times 1 0 0 0}{m \times w _ {d m}} \dots \tag {2}
$$

式中： $\omega_{N}$ ——土壤中全氮的含量， $\mathrm{mg/kg}$ ；

$V_{1}$ ——样品消耗盐酸标准溶液的体积，ml；

$V_{0}$ ——空白消耗盐酸标准溶液的体积，ml；

$\mathcal{C}_{HCl}$ ——盐酸标准溶液的浓度, mol/L;

14.0——氮的摩尔质量，g/mol;

$\mathcal{W}_{dm}$ ——土壤样品的干物质含量， $\%$ ；

m——称取土样的质量，g。

结果保留3位有效数字，按科学计数法表示。

# 10 精密度和准确度

# 10.1 精密度

6个实验室对含氮量分别为 $250\mathrm{mg / kg}$ 、 $500\mathrm{mg / kg}$ 和 $1.50\times 10^{3}\mathrm{mg / kg}$ 的统一样品进行了测定：实验室内相对标准偏差为： $2.0\% \sim 13\%$ 、 $3.1\% \sim 7.3\%$ 、 $2.2\% \sim 4.8\%$ ；实验室间相对标准偏差为： $5.1\%$ 、 $6.9\%$ 、 $5.0\%$ ；重复性限为： $66\mathrm{mg / kg}$ 、 $77\mathrm{mg / kg}$ 、 $122\mathrm{mg / kg}$ ；再现性限为： $71\mathrm{mg / kg}$ 、 $126\mathrm{mg / kg}$ 、 $220\mathrm{mg / kg}$ 。

# 10.2 准确度

6个实验室对含氮量分别为 $265\mathrm{mg / kg}$ 、 $435\mathrm{mg / kg}$ 和 $1.33\times 10^{3}\mathrm{mg / kg}$ 的实际土壤样品进行了加标分析测定：加标回收率分别为 $88.4\% \sim 105\%$ 、 $88.2\% \sim 109\%$ 、 $87.3\% \sim 99.4\%$ ；加标回收率最终值为 $(94.0\pm 11.8)\%$ 、 $(96.0\pm 14.0)\%$ 、 $(92.9\pm 7.8)\%$ 。

4个实验室对含氮量为 $(720\pm 90)\mathrm{mg / kg}$ 的标准样品进行了分析测定，相对误差最终值为 $(-0.5\pm 4.4)$ $\%$ 。

# 11 质量保证和质量控制

# 11.1 空白试验

每批样品应至少做2个全程序空白，空白值（空白样品所消耗的盐酸标准溶液体积）应小于 $0.80\mathrm{ml}$ 。

# 11.2 平行样测定

每批样品应进行 $10\%$ 的平行样品测定，当10个样品以下时，平行样不少于1个。平行双样测量结果的相对偏差应在 $15\%$ 以内。

# 11.3 标准样品测定

每批样品测定质控平行双样，测定值必须落在质控样保证值（在 $95\%$ 的置信水平）范围之内。

# 11.4 样品加标回收率测定

如果没有标准土壤样品，每批样品应进行 $10\% \sim 20\%$ 的回收率测定，样品数不足10个时，适当增加加标频次。实际样品加标回收率应在 $75\% \sim 115\%$ 以内。

# 12 废物处理

对实验过程中产生的废液及分析后的高浓度样品，应放置于适当的密闭容器中保存，并委托有资质的单位进行处理，防止对人员及环境造成危害。
