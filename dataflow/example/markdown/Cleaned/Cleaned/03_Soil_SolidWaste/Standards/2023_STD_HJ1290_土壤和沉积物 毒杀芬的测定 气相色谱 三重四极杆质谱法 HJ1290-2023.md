# 前言

为贯彻《中华人民共和国环境保护法》《中华人民共和国土壤污染防治法》，防治生态环境污染，改善生态环境质量，规范土壤和沉积物中毒杀芬的测定方法，制定本标准。

本标准规定了测定土壤和沉积物中 3 种指示性毒杀芬同类物 P26、P50 和 P62 的气相色谱-三重四极杆质谱法。

本标准的附录 A 和附录 B 为资料性附录。

# 土壤和沉积物毒杀芬的测定气相色谱-三重四极杆质谱法

警告：实验中使用的有机溶剂和标准溶液为有毒有害物质，试剂配制和样品前处理过程应在通风橱内进行；操作时应按要求佩戴防护器具，避免吸入呼吸道或接触皮肤和衣物。

# 1 适用范围

本标准规定了测定土壤和沉积物中毒杀芬的气相色谱-三重四极杆质谱法。

本标准适用于土壤和沉积物中 3 种指示性毒杀芬同类物 P26、P50 和 P62（参见附录 A）的测定。

当取样量为 $1 0 . 0 \ \mathrm { g }$ ，定容体积为 $2 0 \mu \mathrm { l }$ 时，3 种指示性毒杀芬方法检出限均为 $0 . 0 0 3 ~ \mu \mathrm { g / k g }$ ，测定下限均为 $0 . 0 1 2 ~ \mu \mathrm { g / k g }$ 。

# 3 方法原理

土壤或沉积物中毒杀芬目标化合物经有机溶剂提取、净化、浓缩、定容后，用气相色谱分离，三重四极杆质谱仪检测。根据保留时间和特征离子定性，同位素稀释内标法定量。

# 5 仪器和设备

5.1 气相色谱-三重四极杆质谱仪：进样口具备分流/不分流和脉冲/高压进样功能；柱温箱可在$5 0 ~ \mathrm { ~ \textdegree ~ } { }$ 温度区间内程序升温；配有电子轰击离子源，具备多重反应监测功能。
5.2 毛细管色谱柱：低流失石英毛细管柱。 $3 0 \mathrm { m }$ （长） $\times 0 . 2 5 \mathrm { m m }$ （内径） $\times 0 . 2 5 \mu \mathrm { m }$ （膜厚），固定相为 $5 \%$ 苯基- $9 5 \%$ 甲基聚硅氧烷，最高使用温度不低于 $3 5 0 \mathrm { ~ \ } \mathrm { { ~ C ~ } }$ ，或选用其它等效毛细管色谱柱。
5.3 提取装置：索氏提取器或加压流体萃取仪等性能相当的设备。
5.4 浓缩装置：旋转蒸发仪、氮吹仪或其它功能相当的浓缩装置。

5.7 样品瓶：棕色，广口玻璃瓶或聚四氟乙烯衬垫螺口玻璃瓶， $5 0 \mathrm { m l } { \sim } 5 0 0 \mathrm { m l }$ 。
5.8 一般实验室常用仪器和设备。

# 6 样品

# 6.1 样品采集和保存

土壤样品按照 HJ/T 166 的相关规定采集和保存，水体沉积物样品按照 HJ 494、HJ 495 和 HJ/T 91的相关要求采集，海洋沉积物样品按照 GB 17378.3的相关要求采集。样品应于洁净的样品瓶（5.7）中保存，尽快运回实验室分析，运输过程中应密封避光。如不能及时分析，应在 4 ℃以下避光冷藏，保存时间为 $1 4 \textup d$ 。

# 6.2 样品的制备

除去样品中的异物（植物、石子等），风干或用真空冷冻干燥仪（5.5）干燥，研磨。称取两份样品，每份重量约 $1 0 \ \mathrm { g }$ （精确至 $0 . 0 1 \ \mathrm { g }$ ）。

# 6.3 干物质和水分的测定

土壤样品（6.2）干物质含量的测定按照 HJ 613执行，沉积物样品（6.2）含水率的测定按照 GB 17378.5执行。

# 6.4 试样的制备

# 6.4.1 提取

# 6.4.1.1 索氏提取法

称取约 $1 0 \ \mathrm { g }$ 样品（6.2），加入适量硫酸钠（4.18），研磨均化成流砂状，全部转移至索氏提取套筒（4.27），装入提取装置（5.3）中，加入 $1 0 . 0 \mu \mathrm { l }$ 的提取同位素内标使用溶液（4.14）。在圆底溶剂瓶中加入 $2 5 0 ~ \mathrm { m l }$ 正己烷-丙酮混合溶剂（4.4），提取 $1 6 \mathrm { { h } }$ 以上，回流速度控制在 4 次/h ${ \sim } 6$ 次/h。提取完毕，取出圆底溶剂瓶，利用浓缩装置（5.4）将提取液浓缩至 $1 \mathrm { m l } \mathrm { \sim } 2 \mathrm { m l }$ ，待净化。

# 6.4.1.2 加压流体萃取法

称取约 $1 0 \ \mathrm { g }$ （精确至 $0 . 0 1 \ \mathrm { g }$ ）的样品（6.2），加入适量硅藻土（4.25）混匀，研磨均化成流砂状，放入加压流体萃取装置的萃取池中，加入 $1 0 . 0 \mu \mathrm { l }$ 的提取同位素内标使用溶液（4.14）。设定萃取条件，压力 $1 0 . 3 \mathrm { M P a }$ ，温度 $1 5 0 \mathrm { ~ \textdegree C }$ ，提取溶剂为正己烷-丙酮混合溶剂（4.4）， $1 0 0 \%$ 充满萃取池模式，静态萃取时间 $7 \mathrm { { m i n } }$ ，循环三次，收集提取液。利用浓缩装置（5.4）将提取液浓缩至 $1 \mathrm { m l } \mathrm { \sim } 2 \mathrm { m l }$ ，待净化。或按照 HJ 783进行萃取条件的设置和优化。

注1：若经过验证也可使用其他等效提取方法。

注2：样品浓缩液如不能尽快分析，应在 4 ℃以下避光冷藏保存，30d 内完成分析。

# 6.4.2 净化

# 6.4.2.1 硫的净化

样品含硫时，应首先进行硫的去除。

在浓缩后的提取液（6.4.1）中添加 $5 0 ~ \mathrm { m l }$ 的正己烷（4.1），再加入 $1 5 \ \mathrm { g }$ 铜粉（珠）（4.19），充分振荡，直至铜粉（珠）不变色，静置 $3 0 \mathrm { m i n }$ ，用装有硫酸钠（4.18）的三角漏斗过滤，收集滤液，利用浓缩装置（5.4）浓缩至 $\mathrm { 1 m l } \sim 2 \mathrm { m l }$ 。

# 6.4.2.2 酸性硅胶柱净化

在玻璃层析柱（5.6）底部垫一些石英棉（4.24），由下而上依次装填 $1 . 0 \ \mathrm { g }$ 活化硅胶（4.20），$1 2 . 5 \ \mathrm { g }$ 硫酸硅胶Ⅱ（4.22）、 $1 0 . 0 \ \mathrm { g }$ 硫酸硅胶Ⅰ（4.21）、2.5g 活化硅胶（4.20）、 $5 . 0 \ \mathrm { g }$ 硫酸钠（4.18）。填充后用 $5 0 ~ \mathrm { m l }$ 正己烷（4.1）淋洗，保持液面在硫酸钠层。

转移浓缩后的提取液（6.4.1），或脱硫净化后的浓缩液（6.4.2.1）至酸性硅胶柱中，用 $1 \mathrm { m l } \mathrm { \sim } 2 \mathrm { m l }$ 的正己烷（4.1）冲洗盛装提取液/浓缩液的容器壁，反复进行 $2 \sim 3$ 次，洗涤液加到酸性硅胶柱中。继续用 $1 0 0 \mathrm { m l }$ 正己烷（4.1）以 3 ml/min（每秒 1 滴）的流速洗脱，收集洗脱液，利用浓缩装置（5.4）浓缩至 $1 \mathrm { m l } \mathrm { \sim } 2 \mathrm { m l }$ 。

# 6.4.2.3 活化硅胶柱净化

在玻璃层析柱（5.6）底部垫一些石英棉（4.24），由下而上依次装填 $9 . 0 \ \mathrm { g }$ 活化硅胶（4.20）、 $5 . 0 \ \mathrm { g }$ 硫酸钠（4.18）。填充后用 $5 0 \mathrm { m l }$ 正己烷（4.1）淋洗，保持液面在硫酸钠层。

转移酸性硅胶柱净化后的浓缩液（6.4.2.2）至活化硅胶柱中，用 $1 \mathrm { m l } \mathrm { \sim } 2 \mathrm { m l }$ 的正己烷（4.1）冲洗浓缩液的容器壁，反复进行 2次 ${ \sim } 3$ 次，洗涤液加到活化硅胶柱中。继续用 $4 5 ~ \mathrm { m l }$ 正己烷（4.1）以 $3 \mathrm { m l / m i n }$ （每秒 1 滴）的流速洗脱，洗脱液弃去。再用 $5 0 \mathrm { m l }$ 甲苯-正己烷混合溶剂（4.6），以 $3 \mathrm { m l / m i n }$ （每秒 1滴）的流速洗脱，收集洗脱液，使用浓缩装置（5.4）将洗脱液浓缩至 $1 \mathrm { m l } \mathrm { \sim } 2 \mathrm { m l }$ 。将样品转移至进样瓶中，再使用浓缩装置（5.4）将洗脱液浓缩至近干，添加 $5 . 0 \mu \mathrm { l }$ 进样内标使用溶液（4.16），加入 $1 5 ~ \mu \mathrm { l }$ 壬烷（4.9），密封，4 ℃以下避光保存，待测。

注：在满足本方法质量控制要求的前提下，经验证后可使用其它自动或者手动的净化方法。

# 7 分析步骤

# 7.1 仪器参考条件

# 7.1.1 气相色谱仪条件

进样方式：脉冲不分流进样；进样量 $: ~ 1 . 0 ~ \mu \mathrm { l }$ ；进样口温度： $2 5 0 \mathrm { ~ \ } ^ { \circ } \mathrm { C }$ ；传输线温度： $2 9 0 \mathrm { ~ \textdegree C }$ ；

程序升温：100 ℃保持 $1 \mathrm { m i n }$ ，以 15 $\mathrm { { } ^ { \circ } C / \mathrm { { m i n } } }$ 的速度升至 160 ℃保持 $2 \mathrm { m i n }$ ，以 $5 ~ \mathrm { { ^ { \circ } C / m i n } }$ 的速度升至 $2 7 5 \mathrm { ~ \ } ^ { \circ } \mathrm { C }$ 保持 $7 \mathrm { { m i n } }$ ，再以 $1 0 ~ \mathrm { { ^ { \circ } C / m i n } }$ 的速度升至 $3 0 0 { \cdot } \mathrm { \textC }$ ；

# 7.2.5 相对响应因子的计算

各质量浓度点目标化合物相对于提取同位素内标的相对响应因子（ $R R F _ { e s }$ ）由公式（1）计算。

$$
R R F _ {e s} = \frac {\rho_ {e s}}{\rho_ {s}} \times \frac {A _ {s}}{A _ {e s}} \tag {1}
$$

式中： $R R F _ { e s }$ ——目标化合物相对于提取同位素内标的相对响应因子；

$\rho _ { s }$ —标准溶液中目标化合物的浓度， $\mu \mathrm { g / L }$ ；

$\rho _ { e s }$ —标准溶液中提取同位素内标的浓度， $\mu \mathrm { g / L }$ ；

$A _ { s }$ ——标准溶液中待测化合物的监测离子峰面积之和；

$A _ { e s }$ —标准溶液中提取同位素内标的监测离子峰面积之和。

提取同位素内标相对于进样内标的相对响应因子（ $R R F _ { r s }$ ）按公式（2）计算。

$$
R R F _ {r s} = \frac {\rho_ {r s}}{\rho_ {e s}} \times \frac {A _ {e s}}{A _ {r s}} \tag {2}
$$

式中： $R R F _ { r s }$ —提取同位素内标相对于进样内标的相对响应因子；

$\rho _ { e s }$ —标准溶液中提取同位素内标的浓度，μg/L；

$\rho _ { r s }$ —标准溶液中进样内标的浓度， $\mu \mathrm { g / L }$ ；

$A _ { e s }$ —标准溶液中提取同位素内标的监测离子峰面积之和；

$A _ { r s }$ —标准溶液中进样内标的监测离子峰面积之和。

# 8.1 定性分析

根据指示性毒杀芬同类物与其碳标记同类物的保留时间、碎片离子质荷比定性。试样中目标物的保留时间应该与提取同位素内标的保留时间一致，如果保留时间在一天内变化超过 $5 \%$ ，或者与提取同位素内标的相对保留时间变化在 2%以上，应查找原因，重新测定。

# 8.2 定量分析

# 8.2.1 目标化合物的量

对于本标准规定的毒杀芬目标化合物的量（mj），按照公式（3）进行计算：

$$
m _ {j} = \frac {A _ {j}}{A _ {e s , j}} \times \frac {m _ {e s , j}}{\overline {{R R F _ {e s}}}} \tag {3}
$$

式中： $m _ { j }$ —分析样品中目标化合物 $j$ 的量，µg；

$A _ { j }$ ——目标化合物 $j$ 的监测离子峰面积之和；

$A _ { e s , j }$ —提取同位素内标的监测离子峰面积之和；

mes,j — $m _ { e s , j }$ 提取同位素内标的添加量，µg；

$\overline { { R R F _ { e s } } }$ 目标化合物相对提取同位素内标的平均相对响应因子。

# 8.2.2 结果计算

土壤样品中目标化合物的质量浓度（ $\mu \mathrm { g / k g }$ ），按照公式（4）进行计算：

$$
w _ {1, j} = \frac {m _ {j}}{m _ {1} \times w _ {d m}} \times 1 0 0 0 \tag {4}
$$

式中： $w _ { 1 , j }$ —土壤样品中目标化合物 $j$ 浓度， $\mu \mathrm { g / k g }$ ；

$m _ { j }$ mj 试样中目标化合物 $j$ 的量， $\mu \mathrm { g }$ ；

$m _ { 1 }$ —土壤样品质量，g；

$w _ { d m }$ —土壤干物质含量， $\%$ 。

沉积物样品中目标化合物的质量浓度（µg/kg），按照公式（5）进行计算：

$$
w _ {2, \mathrm {j}} = \frac {m _ {j}}{m _ {2} \times \left(1 - w _ {H _ {2} O}\right)} \times 1 0 0 0 \tag {5}
$$

式中： $w _ { 2 , j }$ —沉积物样品中目标化合物 j 浓度，µg/kg；

$m _ { j }$ —试样中目标化合物 j 的量，µg；

m2 沉积物样品质量，g；

wH2O —沉积物水分含量，%。

# 10 质量保证和质量控制

# 10.1 空白试验

# 10.1.1 试剂空白

每 20 个样品或每批次（≤20 个）应至少分析 1 个待测样品溶液所使用的溶剂作为试剂空白。所有试剂空白测试结果应低于方法检出限。

# 10.1.2 实验室空白

每 20个样品或每批次（ $\leqslant 2 0$ 个）应至少分析 1个实验室空白试样，测试结果中的目标化合物浓度不应超过方法的检出限。否则，应检查试剂空白、仪器系统以及前处理过程。

# 10.2 初始校准

采用目标化合物相对提取同位素内标的平均相对响应因子进行校准时，标准系列各点相对响应因子的相对标准偏差应 $\leqslant 2 0 \%$ 。

# 10.3 连续校准

选择中间质量浓度的标准溶液，每 $^ { 1 2 \mathrm { ~ h ~ } }$ 或每批次（ $\leqslant 2 0$ 个）样品至少测定 1 次。目标化合物的测定结果与标准值间的相对误差应在±35%以内。

# 10.4 平行样

每 20个样品或每批次（ $\leqslant 2 0$ 个）应至少分析一个平行样，平行样测定结果的相对偏差绝对值小于$30 \%$ 。

# 10.5 基体加标

每 20个样品或每批次（ $\leqslant 2 0$ 个）应至少分析一个基体加标样，毒杀芬同类物的回收率应在 $5 0 \% \sim$ $120 \%$ 之间。

# 10.6 提取同位素内标回收率

毒杀芬同类物提取同位素内标回收率应在 $3 0 \% \sim 1 3 0 \%$ ，否则须重新取样进行处理。毒杀芬同类物提取同位素内标的回收率按照公式（6）计算。

$$
R = \frac {A _ {e s}}{A _ {r s}} \times \frac {m _ {r s}}{R R F _ {r s}} \times \frac {100 \%}{m _ {e s}} \tag{6}
$$

式中： $R$ ——提取同位素内标的回收率， $\%$ ；

$A _ { e s }$ —提取同位素内标的监测离子峰面积之和；

$A _ { r s }$ —进样内标的监测离子峰面积之和；

$m _ { r s }$ —进样内标的添加量，µg；

RRFrs $\overline { { R R F _ { r s } } }$ 提取同位素内标相对进样内标的平均相对响应因子；

$m _ { e s }$ —提取同位素内标的添加量，µg。
