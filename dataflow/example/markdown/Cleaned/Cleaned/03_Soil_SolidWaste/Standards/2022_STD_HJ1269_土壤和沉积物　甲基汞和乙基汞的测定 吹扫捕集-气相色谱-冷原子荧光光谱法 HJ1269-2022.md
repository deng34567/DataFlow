# 前言

为贯彻《中华人民共和国环境保护法》《中华人民共和国土壤污染防治法》，防治生态环境污染，改善生态环境质量，规范土壤和沉积物中甲基汞和乙基汞的测定方法，制定本标准。

本标准规定了测定土壤和沉积物中甲基汞和乙基汞的吹扫捕集/气相色谱-冷原子荧光光谱法。

本标准的附录 A 为规范性附录，附录 B 为资料性附录。

本标准为首次发布。

# 1 适用范围

本标准规定了测定土壤和沉积物中甲基汞和乙基汞的吹扫捕集/气相色谱-冷原子荧光光谱法。

本标准适用于土壤和沉积物中甲基汞和乙基汞的测定。

取样量为 $0 . 5 \ \mathrm { g }$ ，提取液体积为 $3 0 \mathrm { m l }$ 时，甲基汞和乙基汞的方法检出限均为 $0 . 2 ~ { \mu \mathrm { g / k g } }$ ，测定下限均为 $0 . 8 ~ \mu \mathrm { g / k g }$ 。详见附录 A。

# 3 方法原理

土壤或沉积物样品经碱液提取后，提取液中的甲基汞和乙基汞与四丙基硼化钠发生衍生化反应，生成挥发性的甲基丙基汞和乙基丙基汞，经吹扫捕集、热脱附和气相色谱分离后，再高温裂解为汞蒸气，用冷原子荧光光谱法测定。根据保留时间定性，外标法定量。

# 4 干扰和消除

土壤或沉积物上机液中， $\mathrm { H g } ^ { 2 + }$ 含量低于 1 ng 时，对甲基汞和乙基汞的测定无明显影响。 $\mathrm { H g } ^ { 2 + }$ 含量超过 $1 ~ \mathrm { n g }$ 时，对甲基汞的测定会产生正干扰，分析时可通过减少上机液中提取液所占比例以降低上机液中的 $\mathrm { H g } ^ { 2 + }$ 含量。

# 5 试剂和材料

分析时均使用符合国家标准的优级纯试剂，实验用水为不含目标化合物的纯水。

5.1 甲醇（ $\mathrm { C H } _ { 3 } \mathrm { O H }$ ）：色谱纯。
5.2 乙酸（ $\mathrm { C H _ { 3 } C O O H }$ ）： $\rho { = } 1 . 0 5 ~ \mathrm { g / m l }$ ， $w { \geqslant } 9 9 . 8 \%$ 。
5.3 盐酸（HCl）： $\rho { = } 1 . 1 8 ~ \mathrm { g / m l }$ ， $w \in [ 3 6 . 0 \%$ ， $3 8 . 0 \% ]$ 。
5.4 硝酸（ $\mathrm { H N O } _ { 3 }$ ）： $\rho { = } 1 . 4 \ \mathrm { g / m l }$ ， $w \in [ 6 5 . 0 \%$ ， $6 8 . 0 \% ]$ 。
5.5 氢氧化钾（KOH）。
5.6 四丙基硼化钠 $\mathrm { [ N a B ( C _ { 3 } H _ { 7 } ) _ { 4 } ] }$ ：纯度 $9 8 \%$ ，避光、密封保存。
5.7 四乙基硼化钠 $\mathrm { [ N a B ( C _ { 2 } H _ { 5 } ) _ { 4 } ] }$ ：纯度 $3 9 8 \%$ ，避光、密封保存。
5.8 乙酸钠（ $\mathrm { C H _ { 3 } C O O N a }$ ）。
5.9 乙酸-乙酸钠缓冲溶液： $c ( \mathrm { C H } _ { 3 } \mathrm { C O O H } ) { = } 2 \mathrm { m o l } / \mathrm { L }$ ， $c ( \mathrm { C H } _ { 3 } \mathrm { C O O N a } ) { \mp } 2 \mathrm { m o l } / \mathrm { L } .$

称取 $1 6 . 4 \ \mathrm { g }$ 乙酸钠（5.8）溶于水中，加入 $1 2 \mathrm { m l }$ 乙酸（5.2），定容至 $1 0 0 \mathrm { m l }$ ，缓冲液 $\mathsf { p H }$ 值在 $4 . 5 \sim$ 4.8 之间。

5.10 氢氧化钾-甲醇溶液。

称取 $5 0 . 0 \ \mathrm { g }$ 氢氧化钾（5.5）至 $2 0 0 ~ \mathrm { \ m l }$ 甲醇（5.1）中，超声溶解 $2 0 \mathrm { { m i n } }$ 至溶液为乳白色，贮存于具有螺口的塑料试剂瓶中，使用前混匀。

5.11 四丙基硼化钠溶液： $\rho [ \mathrm { N a B } ( \mathrm { C } _ { 3 } \mathrm { H } _ { 7 } ) _ { 4 } ] { = } 1 0 \ \mathrm { g / L }$ 。

称取 $2 . 0 \ \mathrm { g }$ 氢氧化钾（5.5）至水中溶解，定容至 $1 0 0 ~ \mathrm { m l }$ ，混匀，转移到带盖塑料瓶中，放入冰箱冷却至出现冰晶后，迅速加入 $1 . 0 \ \mathrm { g }$ 四丙基硼化钠（5.6），摇匀，快速分装至多个 $1 . 5 ~ \mathrm { m l }$ 带密封垫的螺口玻璃瓶中，上盖旋紧，于- $\cdot 1 8 \mathrm { ~  ~ { ~ C ~ } } { \pm } 2 \mathrm { ~  ~ { ~ C ~ } ~ }$ 冷冻，可保存 180d。临用时，取出一小瓶试剂，待瓶内冰块融化约一半时使用。

注1：四丙基硼化钠有毒，在空气中暴露易变质，溶液配制时移取四丙基硼化钠应快速并及时密封。冷冻后的四丙基硼化钠溶液随取随用，不可在常温下久置。融化后的四丙基硼化钠溶液不稳定，应一次性使用。

注2：若只分析甲基汞，可使用四乙基硼化钠（5.7）代替四丙基硼化钠（5.6），四乙基硼化钠溶液的配制方法及保存方式与上述步骤相同。

5.12 乙酸-盐酸溶液。

分别量取乙酸（5.2）5 ml、盐酸（5.3）2ml，加入适量水中，用水稀释至 1 L。

5.13 甲基汞标准贮备液： $\rho ( \mathrm { C H _ { 3 } H g ^ { + } } ) { = } 1 . 0 0 \ \mathrm { m g / L }$ 。

购买市售以甲醇、丙酮等为溶剂的有证标准溶液，开封后于4 ℃以下冷藏、避光、密封可保存 $2 4 0 \ : \mathrm { d }$ 。

5.14 乙基汞标准贮备液： $\rho ( \mathrm { C } _ { 2 } \mathrm { H } _ { 5 } \mathrm { H g } ^ { + } ) { = } 1 . 0 0 \mathrm { m g } / \mathrm { L }$ 。

购买市售以甲醇、丙酮等为溶剂的有证标准溶液，开封后于 $4 ~ \mathrm { { ^ \circ C } }$ 以下冷藏、避光、密封可保存 $2 4 0 \ : \mathrm { d }$ 。

5.15 混合标准中间液： $\rho { = } 1 0 . 0 \ \mu \mathrm { g / L }$ 。

分别移取 $5 0 0 \mu \mathrm { l }$ 甲基汞标准贮备液（5.13）和乙基汞标准贮备液（5.14）于 $5 0 ~ \mathrm { m l }$ 容量瓶中，用乙酸-盐酸溶液（5.12）定容至标线，于带内衬聚四氟乙烯垫螺盖的棕色试剂瓶或含氟聚合物瓶中 $4 ~ \mathrm { { ^ \circ C } }$ 以下冷藏、避光、密封可保存 $2 4 0 \mathrm { d }$ 。

5.16 混合标准使用液： $\rho { = } 1 . 0 0 \mu \mathrm { g / L }$ 。

移取适量混合标准中间液（5.15），用乙酸-盐酸溶液（5.12）配制成 $1 . 0 0 \mu \mathrm { g / L }$ 的标准使用液，于带内衬聚四氟乙烯垫螺盖的棕色试剂瓶或含氟聚合物瓶中 $4 \mathrm { ~ \ } \mathrm { ‰ ~ }$ 以下冷藏、避光、密封可保存 $2 4 0 \mathrm { d }$ 。

5.17 石英砂：粒径 $0 . 1 5 \ : \mathrm { m m } { \sim } 0 . 8 3 \ : \mathrm { m m }$ （100 目 ${ \sim } 2 0$ 目）。
5.18 氩气：纯度 ${ \geqslant } 9 9 . 9 9 9 \%$ 。
5.19 氮气：纯度 $\geqslant 9 9 . 9 9 \%$ 。

# 6 仪器和设备

6.1 棕色螺口玻璃瓶： $1 0 0 \mathrm { m l }$ 。
6.2 全自动烷基汞分析仪：包括吹扫捕集装置、气相色谱仪、色谱柱、裂解装置和冷原子荧光光谱仪。吹扫捕集装置可以使用原位吹扫捕集或异位吹扫捕集。捕集管填装有聚 2,6-二苯基-对苯醚吸附剂或其他等效吸附剂，粒径为 $1 5 0 \mu \mathrm { m } { \sim } 1 8 0 \mu \mathrm { m }$ ，具备流量控制器。

色谱柱可以使用填充柱或毛细管柱。填充柱：填料固定液为苯基（ $1 0 \%$ ）甲基聚硅氧烷， $3 4 0 ~ \mathrm { m m }$ （柱长） $\times 1 . 5 9 \mathrm { m m }$ （内径），或其他等效色谱柱；毛细管柱：固定相为 $100 \%$ 二甲基聚硅氧烷， $1 5 \mathrm { m }$ （柱长） $\times 0 . 5 3 \mathrm { m m }$ （内径） $\times 0 . 5 \mu \mathrm { m }$ （膜厚），或其他等效色谱柱。

6.3 真空冷冻干燥仪：空载真空度达 13 Pa 以下。
6.4 离心机：转速可调。
6.5 恒温振荡器。
6.6 涡旋振荡器。
6.7 尼龙筛：孔径 $0 . 1 5 \mathrm { m m }$ （ $\times 1 0 0$ 目）和 2.54 mm（10 目）。

6.10 一般实验室常用仪器和设备。

# 7 样品

# 7.1 样品的采集和保存

土壤样品按照 HJ/T 166和 HJ 25.2的相关要求采集，水体沉积物样品按照 HJ/T91 和 HJ 494 的相关要求采集，海洋沉积物样品按照 GB 17378.3和 HJ 442.4的相关要求采集。

样品采集后，应于棕色螺口玻璃瓶（6.1）中保存，4 ℃以下冷藏可保存 5 d，-15 ℃以下冷冻可保存 15 d。

# 7.2 水分的测定

土壤样品干物质含量测定按照 HJ 613执行，沉积物样品含水率测定按照 GB 17378.5执行。

# 7.3 样品的制备

除去样品中的枝棒、叶片、石子等异物，按照 HJ/T 166 采用四分法粗分，取混匀后的样品，放入真空冷冻干燥仪（6.3）中干燥脱水，干燥后的样品研磨后通过 $2 . 5 4 \mathrm { m m }$ 尼龙筛（6.7）（除去 $2 . 5 4 \mathrm { m m }$ 以上的沙砾），混匀，再将上述样品研磨至全部通过 $0 . 1 5 \mathrm { m m }$ 尼龙筛（6.7），分装后冷藏保存。

# 7.4 试样的制备

准确称取 $0 . 5 \ \mathrm { g }$ 制备的样品（7.3）（精确至 $0 . 1 \mathrm { m g }$ ），放入 $5 0 ~ \mathrm { m l }$ 离心管（6.8）中，加入 $1 5 . 0 \mathrm { m l }$ 氢氧化钾-甲醇溶液（5.10）后，盖紧盖子，用涡旋振荡器（6.6）混匀。将样品倾斜置于恒温振荡器（6.5）中，若采用水浴振荡器应保证水浴液面没过管内溶液，待温度升至 $6 0 ~ \mathrm { ~ \textdegree C }$ 后，采用 $1 5 0 \ \mathrm { r / m i n } { \sim } 1 7 0 \ \mathrm { r / m i n }$ 的频率恒温振荡提取 $^ { 3 \mathrm { ~ h ~ } }$ ，取出样品冷却至室温，加入 $1 5 . 0 \mathrm { m l }$ 实验用水，再次涡旋混匀。将样品放入离心机（6.4），于 $4 0 0 0 \mathrm { r / m i n }$ 离心 $2 \mathrm { m i n }$ ，将上清液从离心管中转移至洁净样品瓶中备用， $2 4 \mathrm { h }$ 之内测定，否则置于 $4 \mathrm { ~ \ : ~ } ^ { \circ } \mathrm { C }$ 以下避光、密封保存，3 d 内完成测定。

注：振荡提取时，待离心管温度升至 $6 0 \textrm { ‰}$ 后，为防止漏液应再次拧紧盖子。提取后应待其充分冷却后方可加水。离心后立即转移上清液，避免土壤或沉积物重新吸附甲基汞和乙基汞。

# 7.5 空白试样的制备

采用石英砂（5.17）代替土壤或沉积物样品，按照与试样的制备（7.4）相同的步骤制备实验室空白试样。

# 8.1 仪器参考条件

# 8.1.1 吹扫捕集热脱附参考条件

b）毛细管柱：初始温度为 $9 3 \mathrm { ~ \ } \mathrm { { ~ ‰ ~ } }$ ，以 $5 ~ \mathrm { { ^ { \circ } C / m i n } }$ 升至 $9 7 \mathrm { ~ \ } ^ { \circ } \mathrm { C }$ ，保持 105 s，以 4.5 ℃/min 升至 $1 0 0 \mathrm { ~ \textdegree C }$ ，保持 $1 0 5 \mathrm { ~ s ~ }$ ；载气流速： $1 5 \mathrm { m l / m i n }$ ；

# 8.2 校准曲线的建立

8.2.1 取 8 个 $4 0 ~ \mathrm { m l }$ 进样瓶（6.9），分别加入实验用水约 $3 5 ~ \mathrm { m l }$ ，再分别加入适量的混合标准使用液（5.16），由低到高依次配制不同浓度的标准系列溶液，目标化合物含量分别为 $0 . 0 0 \mathrm { p g } , 2 . 0 0 \mathrm { p g } , 5 . 0 0 \mathrm { p g } ,$ 、$1 0 . 0 \mathrm { p g }$ 、 $5 0 . 0 { \mathfrak { p g } }$ 、 $1 0 0 { \mathfrak { p g } }$ 、 $5 0 0 \mathrm { p g } . 1 0 0 0 \mathrm { p g }$ （此为参考含量）。

注：原位吹扫采用 $6 0 \mathrm { m l }$ 进样瓶（ $_ { . 6 . 9 \mathrm { ) }  }$ 依次按照上述步骤加入 $4 0 ~ \mathrm { m l }$ 水、混合标准使用液。

8.2.2 向标准系列溶液（8.2.1）中加入 $3 0 0 \mu \mathrm { l }$ 乙酸-乙酸钠缓冲溶液（5.9）及 $5 0 \mu \mathrm { l }$ 四丙基硼化钠溶液（5.11），迅速加入实验用水至瓶满，不留空隙，旋紧样品瓶盖，静置 $2 0 \mathrm { { m i n } }$ 。
8.2.3 按照仪器参考条件（8.1），由低含量到高含量依次测定标准系列溶液。以标准系列溶液中目标化合物的含量为横坐标，以其对应的峰面积或峰高为纵坐标，建立甲基汞和乙基汞的校准曲线。

注：在实际工作中，由于样品浓度未知且差异较大，需要配制的标准系列溶液浓度跨度较大，低浓度样品采用线性回归法容易产生偏差，宜采用校准系数法。也可根据实际情况，分别建立低、高浓度的校准曲线，采用线性回归法计算结果。

# 8.3 试样测定

于 $4 0 ~ \mathrm { m l }$ 进样瓶（6.9）中加入实验用水约 $3 5 ~ \mathrm { m l }$ ，取试样（7.4） $1 5 0 ~ \mu \mathrm { l }$ 至进样瓶（6.9）中，此后按照与 8.2.2相同的操作步骤进行衍生化反应， $\mathsf { p H }$ 值控制在 $4 { \sim } 6$ 之间，按照 8.2.3 的条件和步骤测定。

注：原位吹扫采用 $6 0 \mathrm { m l }$ 进样瓶，依次按照上述步骤加入 $4 0 ~ \mathrm { m l }$ 水、试样、乙酸-乙酸钠缓冲溶液（5.9）和四丙基硼化钠溶液（5.11）。

按照与试样测定（8.3）相同的操作步骤和条件测定空白试样（7.5）。

# 9.2.1 线性回归法

土壤中甲基汞、乙基汞含量（质量分数）按公式（1）计算。

$$
w _ {1} = \frac {m _ {1} \times V _ {1}}{m \times w _ {\mathrm {d m}} \times V _ {2} \times 1 0 0 0} \tag {1}
$$

式中： $w _ { 1 } .$ —土壤中甲基汞或乙基汞的含量， $\mu \mathrm { g / k g }$ ；

$m _ { 1 } .$ —由校准曲线计算样品中甲基汞或乙基汞的质量，pg；

$V _ { 1 } .$ ——提取液体积，ml；

m—提取样品的质量，g；

$\mathcal { W } \mathrm { d m }$ —土壤样品的干物质含量， $\%$ ；

$V _ { 2 } .$ ——提取液的取样体积，ml；

1000——质量单位间的换算系数。

沉积物中甲基汞、乙基汞含量（质量分数）按公式（2）计算。

$$
w _ {2} = \frac {m _ {1} \times V _ {1}}{m \times \left(1 - w _ {\mathrm {H} _ {2} \mathrm {O}}\right) \times V _ {2} \times 1 0 0 0} \tag {2}
$$

式中：w2—沉积物中甲基汞或乙基汞的含量， $\mu \mathrm { g / k g }$ ；

$m _ { 1 }$ —由校准曲线计算样品中甲基汞或乙基汞的质量，pg；

$V _ { 1 }$ ——提取液体积，ml；

$m$ —提取样品的质量，g；

wH2O —沉积物样品的含水率， $\%$ ；

$V _ { 2 } .$ —提取液的取样体积，ml；

1000——质量单位间的换算系数。

# 9.2.2 校准系数法

校准系数按公式（3）计算。

$$
\mathrm {C F} _ {i} = \frac {m _ {i}}{A _ {i} - A _ {0}} \tag {3}
$$

式中： $\mathrm { C F } _ { \dot { \tau } }$ ——第 $i$ 个标准点的校准系数，即单位响应值所对应的目标物含量，pg/峰高或 pg/峰面积；

$m _ { i }$ ——第 $i$ 个标准点的甲基汞或乙基汞的质量，pg；

$A _ { i }$ ——第 $i$ 个标准点的甲基汞或乙基汞的峰高或峰面积；

$A _ { 0 }$ ——空白中甲基汞或乙基汞的峰高或峰面积。

平均校准系数按公式（4）计算。

$$
\overline {{\mathrm {C F}}} = \frac {\sum_ {i = 1} ^ {n} \mathrm {C F} _ {i}}{n} \tag {4}
$$

式中：CF——标准点的平均校准系数， $\mathfrak { p g } /$ 峰高或 pg/峰面积；

n—标准系列点数；

CF——第 i 个标准点的校准系数，即单位响应值所对应的目标物含量，pg/峰高或 pg/峰面积。土壤中甲基汞、乙基汞含量（质量分数） $w _ { 1 }$ 按公式（5）计算。

$$
w _ {1} = \frac {\left(A _ {1} - A _ {0}\right) \times \overline {{\mathrm {C F}}} \times V _ {1}}{m \times w _ {\mathrm {d m}} \times V _ {2} \times 1 0 0 0} \tag {5}
$$

式中： $w _ { 1 } .$ —土壤中甲基汞或乙基汞的含量，μg/kg；

$A _ { 1 } .$ ——样品中甲基汞或乙基汞的峰高或峰面积；

$A _ { 0 }$ ——空白中甲基汞或乙基汞的峰高或峰面积；

CF —标准点的平均校准系数， $\mathrm { p g / }$ 峰高或 $\mathfrak { p g } /$ 峰面积；

$V _ { 1 } .$ ——提取液体积，ml；

m—提取样品的质量，g；

$\mathcal { w } _ { \mathrm { d m } }$ —土壤样品的干物质含量， $\%$ ；

$V _ { 2 } .$ ——提取液的取样体积，ml；

1000——质量单位间的换算系数。

沉积物中甲基汞、乙基汞含量（质量分数） $w _ { 2 }$ 按公式（6）计算。

$$
w _ {2} = \frac {\left(A _ {1} - A _ {0}\right) \times \overline {{\mathrm {C F}}} \times V _ {1}}{m \times \left(1 - w _ {\mathrm {H} _ {2} \mathrm {O}}\right) \times V _ {2} \times 1 0 0 0} \tag {6}
$$

式中： $w _ { 2 } .$ —沉积物中甲基汞或乙基汞的含量， $\mu \mathrm { g / k g }$

$A _ { 1 } .$ ——样品中甲基汞或乙基汞的峰高或峰面积；

$A _ { 0 }$ ——空白中甲基汞或乙基汞的峰高或峰面积；

CF——标准点的平均校准系数， $\mathfrak { p g } /$ 峰高或 $\mathfrak { p g } /$ 峰面积；

$V _ { 1 } .$ ——提取液体积，ml；

$m$ —提取样品的质量，g；

wH2O —沉积物样品的含水率， $\%$ ；

$V _ { 2 } .$ —提取液的取样体积，ml；

# 11 质量保证和质量控制

# 11.1 空白试验

每 20个样品或每批次样品（少于 20个）应至少做 1个空白试样，空白试样的测定值应低于方法检出限。

# 11.2 校准

每次分析样品前均应建立不少于 6 个点的校准曲线，采用线性回归法计算结果，曲线的相关系数 ${ \geqslant } 0 . 9 9 5$ ；采用校准系数法计算结果，校准系数 $\mathrm { { { C F } } } _ { i }$ 的相对标准偏差 $\leqslant 1 5 \%$ 。每 20个样品测定一个校准曲线中间浓度点的标准溶液，其相对误差值应该控制在 $\pm 2 0 \%$ 以内，否则应重新建立校准曲线。

# 11.3 平行样

每 20个或每批次样品（少于 20个样品）应至少测定 1 个平行双样，平行双样测定结果的相对偏差应在 $\pm 3 0 \%$ 以内。

# 11.4 基体加标

每 20个样品或每批次样品（少于 20 个样品）应至少测定 1 个基体加标样品或 1 个有证标准物质。甲基汞加标回收率控制在 $7 5 \% \approx 1 3 0 \%$ 之间；乙基汞加标回收率控制在 $6 5 \% \sim 1 2 0 \%$ 之间。

# 12 废物处置

实验中产生的废弃物应集中收集，分类保管，并做好相应标识，依法委托有资质的单位处理。

# 13 注意事项

13.1 实验所用的器皿（进样瓶、样品管等）应在 $1 0 \%$ 硝酸溶液中浸泡至少 $2 4 \mathrm { { h } }$ ，用水洗净。石英砂在使用前放入马弗炉 400 ℃下灼烧 4h，冷却后再用。
13.2 测定高浓度样品后，须用水多次清洗仪器系统。
13.3 失效的四丙基硼化钠溶液，应放入盛有盐酸（体积比为 1:1）的大烧杯中，于 80 ℃加热分解残留物，待烧杯中溶液体积减少 1/2 时，收集剩余的废酸液，统一处置。
