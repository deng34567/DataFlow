# 废旧电池破碎分选回收技术规范

# 1 范围

本标准规定了废旧锂离子电池和镍氢电池破碎分选回收处理过程的术语和定义、总体要求、破碎分选、安全环保要求。

本标准适用于湿法冶炼处理废旧锂离子电池和镍氢电池（包括废旧小型电池、动力蓄电池包、蓄电池模块、单体电池）的破碎分选。

# 3 术语和定义

下列术语和定义适用于本文件。

废旧电池 waste batteries

失去使用价值被废弃的电池成品和半成品。包括在电池生产、运输、储存、使用过程中产生的不合格产品、报废产品、过期产品，以及电池在生产过程中产生的不合格电芯、报废电芯等。

# 4 总体要求

# 4.1 一般要求

4.1.1 回收利用企业应具有危险废物经营许可证，核准经营方式应包括收集、处置、利用。
4.1.2 回收利用企业应采用自动化进料系统和封闭式破碎分选系统，以提高破碎分选效率及安全性。
4.1.3 禁止将未经任何处理的废旧电池直接焚烧、填埋、丢弃。

# 4.2 场地

4.2.1 厂区建设应符合GBZ1、GBZ2的要求，并按GB/T28001中的规定管理。
4.2.2 破碎分选作业现场应严禁烟火，并按GB50034的规定设计照明装置。

4.2.3 作业场地地面应硬化，作业过程中产生的振动如对建筑物产生影响时，应采取相应的防范措施。

# 4.3 设备

4.3.1 破碎分选设备设施应按照国家有关规定，由具有资质的专业生产单位生产，且安全可靠、节能环保。
4.3.2 破碎设备安装工程施工及验收应按GB50276的规定执行。
4.3.3 破碎设备设施应配备除尘装置，如旋风分离器、布袋除尘装置等。

# 5.2 破碎

# 5.2.1 一般要求

5.2.1.1 不应擅自向破碎设备和分选设备中加入其他物品。
5.2.1.2 严禁带料启动设备，停机前应先停物料。

# 5.2.2 技术要求

5.2.2.1 破碎前应清除铁块等硬性杂质。
5.2.2.2 废旧电池宜采用干法进行破碎，破碎前应进行放电、热解处理。
5.2.2.3 废旧小型电池宜直接破碎。废旧动力蓄电池包和蓄电池模块应拆解为单体电池后根据类型进行破碎，软包单体电池和圆柱形单体电池宜直接破碎，矩形单体电池应拆解为电芯后再破碎。
5.2.2.4 应采用粗破、细破方式进行逐级破碎，破碎粒度应不大于 $2\mathrm{cm}$

# 5.3 分选

# 5.3.1 一般要求

5.3.1.1 分选前应保证物料干燥。
5.3.1.2 应采用多次分选以提高材料回收率。

# 5.3.2 技术要求

5.3.2.1 宜采用筛分、风选、磁选、重选、浮选等技术组合进行分选。
5.3.2.2锂离子电池分选后应分别得到铜粉、铝粉、铁粉和电极材料粉，电极材料粉含有镍、钴、锰中的一种或多种元素。镍氢电池分选后应分别得到铁粉和电极材料粉，电极材料粉含有镍、钴、镧、铈、钕中的多种元素。

5.3.2.3 铜、铝、铁的破碎分选回收率应不低于 $90\%$
5.3.2.4 镍、钴、锰金属元素的破碎分选回收率应不低于 $98.5\%$ 。
5.3.2.5 镧、铈、钕稀土元素的破碎分选回收率应不低于 $95\%$

5.3.2.6 电极材料粉粒度应小于 $1\mathrm{mm}$ 。

# 6 安全要求

# 6.1 操作位置

6.1.1 应具有控制室和自动化控制系统，以保证操作人员安全。
6.1.2 作业现场应先规划好安全人行通道，无关人员不得进入作业现场。
6.1.3 应对进料、出料、输送等过程和外露运动部位采取必要的防护措施。
6.1.4 破碎设备的工作平台要安装护栏，以防人员跌落。

# 6.2 控制系统

6.2.1 每一台设备应设置总停开关，每个操作位置都应有急停装置，防止突发事件引发的危险。
6.2.2 应有报警系统，报警信号应能方便发出和接收。
6.2.3 所有的急停与安全装置应按其功能定期进行检查。

# 6.3 电气设备

6.3.1 设备裸露金属部分均应接地(零)，所用的电气设备应有一套接地故障保护装置。
6.3.2 变压器或高压电缆处，应在四周设置防护栏杆或将其布置在隔离间，并设置相应的安全标志，安全标志应符合GB2894的要求。

# 6.4 防火

6.4.1 破碎分选区应设置火灾报警装置，并按GB50016和GB50140的规定配备消防设施和器材，且灭火剂应放置在显眼、易取的地方，并按期检查，确保干燥。
6.4.2 破碎分选车间、办公区域等作业活动场所的应急通道、出入口，应设置醒目的标志。

# 7 环境保护要求

7.1 废旧镍氢电池破碎分选前储存应满足GB18597的要求。
7.2 破碎分选车间厂区的总粉尘浓度应不大于 $3\mathrm{mg / m^3}$ ，并按GBZ/T192.1的规定进行检测。
7.3 主操作室噪声不应超过 $85\mathrm{dB(A)}$ ，并按GB/T13325的规定进行检测。
7.4 热解废气排放应满足GB9078的要求，放电和破碎分选废气排放应满足GB16297的要求。
