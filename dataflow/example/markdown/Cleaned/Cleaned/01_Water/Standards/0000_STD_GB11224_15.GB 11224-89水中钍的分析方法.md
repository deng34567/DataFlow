# 1 主题内容与适用范围

本标准规定了水中钍的分析方法。

本标准适用于地面水、地下水、饮用水中钍的分析，测定范围： $0.01\sim 0.5\mu \mathrm{g} / \mathrm{L}$ 。

# 2 方法提要

水样中加入镁载体和氢氧化钠后，钍和镁以氢氧化物形式共沉淀。用浓硝酸溶解沉淀，溶解液通过三烷基氧膦萃淋树脂萃取色层柱选择性吸附钍；草酸-盐酸溶液解吸钍；在草酸-盐酸介质中，钍与偶氮肿Ⅲ生成红色络合物，于分光光度计 $660\mathrm{nm}$ 处测量其吸光度。

# 3 干扰

水样中锆、铀总量分别超过 $10\mu \mathrm{g}$ 、 $100\mu \mathrm{g}$ 时，会使结果偏高。

# 4 试剂

所有试剂均为符合国家标准或专业标准的分析纯试剂和蒸馏水或同等纯度的水。

4.1 氯化镁 $(\mathrm{MgCl}_{2} \cdot 6\mathrm{H}_{2}\mathrm{O})$ 。
4.2 盐酸溶液： $10\% (V / V)$ 。
4.3 硝酸：浓度 $65.0\% \sim 68.0\%$ 。
4.4 硝酸溶液： $3\mathrm{mol} / \mathrm{L}$ 。
4.5 硝酸溶液： $1\mathrm{mol} / \mathrm{L}$ 。
4.6 $0.025\mathrm{mol} / \mathrm{L}$ 草酸 $-0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶液。
4.7 0.1mol/L草酸-6mol/L盐酸溶液。
4.8 偶氮肿Ⅲ溶液： $1\mathrm{g / L}$ 。
4.9 氢氧化钠溶液： $10\mathrm{mol} / \mathrm{L}$ 称取 $200\mathrm{g}$ 氢氧化钠，用水溶解，稀释至 $500\mathrm{mL}$ 。贮存于聚乙烯瓶中。
4.10 钍标准溶液： $10\mathrm{mg}$ 钍 $-10\%$ 盐酸溶液，最大相对误差不大于 $0.2\%$ 。
4.10.1 用盐酸溶液（4.2）将上述 $10\mathrm{mg}$ 针标准溶液稀释至 $1000\mathrm{mL}$ ，此溶液为每毫升含 $10\mu \mathrm{g}$ 钉。
4.11 三烷基氧膦（TRPO）萃淋树脂： $50\%$ $(m / m)$ ， $60\sim 75$ 目。

# 5 仪器设备

5.1 玻璃色层交换柱：内径 $7\mathrm{mm}$ 。
5.2 分光光度计。
5.3 离心沉淀机。

# 6 采样

按国家关于核设施水质监测分析取样的规定进行。

# 7 分析步骤

# 7.1 萃取色层柱的准备

# 7.1.1 树脂的处理

用去离子水将三烷基氧磷（4.11）浸泡 $24\mathrm{h}$ 后弃去上层清液。用硝酸溶液（4.4）搅拌下浸泡 $2\mathrm{h}$ 而后用去离子水洗至中性。自然晾干。保存于棕色玻璃瓶中。

# 7.1.2 萃取色层柱的制备

用湿法将树脂装入玻璃色层交换柱(5.1)中，床高 $70\mathrm{mm}$ 。床的上、下两端少量聚四氟乙烯丝填塞，用 $25\mathrm{mL}$ 硝酸溶液（4.5）以 $1\mathrm{mL} / \mathrm{min}$ 流速通过玻璃色层交换柱（5.1）后备用。

# 7.1.3 萃取色层柱的再生

依次用 $20\mathrm{mL}$ 草酸-盐酸溶液（4.6）， $25\mathrm{mL}$ 水， $25\mathrm{mL}$ 硝酸溶液（4.5）依 $1\mathrm{mL} / \mathrm{min}$ 流速通过萃取色层柱后备用。

# 7.2 样品分析

7.2.1 取水样 $10\mathrm{L}$ ，加氢氧化钠溶液（4.9）调节至 $\mathrm{pH}7$ ，加 $5.1\mathrm{g}$ 氯化镁（4.1）。在转速为 $500\mathrm{r / min}$ 搅拌下，缓慢滴加 $10\mathrm{mL}$ 氢氧化钠溶液（4.9）。加完后继续搅拌半小时，放置 $15\mathrm{h}$ 以上。

7.2.2 弃去上层清液，沉淀转入离心管中，在转速为 $2000\mathrm{r / min}$ 下离心 $10\mathrm{min}$ 。弃去上层清液。用约 $6\mathrm{mL}$ 硝酸（4.3）溶解沉淀。溶解液在上述转速下离心 $10\mathrm{min}$ ，上层清液以 $1\mathrm{mL / min}$ 流速通过萃取色层柱。

7.2.3 用 $200\mathrm{mL}$ 硝酸溶液（4.5）以 $1\mathrm{mL} / \mathrm{min}$ 流速洗涤萃取色层柱，然后用 $25\mathrm{mL}$ 水洗涤，洗涤速度为 $0.5\mathrm{mL} / \mathrm{min}$ 。

7.2.4 用 $30\mathrm{mL}$ 草酸-盐酸溶液（4.6）以 $0.3\mathrm{mL} / \mathrm{min}$ 流速解吸钍。收集解吸液于烧杯中，在电砂浴上缓慢蒸干。

7.2.5 将上述烧杯中的残渣用草酸-盐酸溶液（4.7）溶解并转入 $10\mathrm{mL}$ 容量瓶中，加入 $0.50\mathrm{mL}$ 偶氮肿Ⅲ（4.8）。用草酸-盐酸溶液（4.7）稀释至刻度。10min后，将此溶液转入 $3\mathrm{cm}$ 比色皿中。以偶氮肿Ⅲ溶液作参比液，于分光光度计（5.2） $660\mathrm{nm}$ 处测量其吸光度，从工作曲线上查出相应的钍量。

# 7.3 工作曲线绘制

准确移取0，0.05，0.10，0.30，0.50mL针标准溶液（4.10.1）置于一组盛有10L自来水的塑料桶中，按7.2.1~7.2.5进行。以偶氮肿Ⅲ溶液作参比液，于分光光度计（5.2）660nm处测量其吸光度。数据经线性回归处理后，以针量为横坐标，吸光度为纵坐标绘制工作曲线。

# 8 结果计算

试样中钍的浓度按下式计算：

$$
C = \frac {W}{V}
$$

式中： $C$ ——试样中钍的液度， $\mu \mathrm{g} / \mathrm{L}$

$W$ ——从工作曲线上查得的针量， $\mu \mathrm{g}$ ；

$V$ ——试样体积，L。

# 正确使用标准的说明

（参考件）

A1 显色剂偶氮肿Ⅲ溶液的使用期不得超过1个月，否则会影响钍的测定。
A2 在分析中，若需要更换试剂或分光光度计需要调整，更换零件时，必须重作工作曲线。
A3 对于碳酸盐结构地层的水样，由于含碳酸根较高，碳酸根与钍形成五碳酸根络钍阴离子 $\left[\mathrm{Th}(\mathrm{CO}_{3})_{6}^{5-}\right]$ ，从而影响钍的定量沉淀。此时，可在水样中加入过氧化氢，使钍形成溶度积小得多的水合过氧化钍（ $\mathrm{Th}_{2} \mathrm{O}_{7} \cdot 11 \mathrm{H}_{2} \mathrm{O}$ ）沉淀。
A4 用硝酸溶解沉淀时，要缓慢加入，硝酸用量以恰好溶解沉淀为宜。此溶解液在上柱前，一定要离心，防止硅酸盐胶体及其残渣堵塞柱子。
A5 解吸液在蒸至近干时，应防止通风。
A6 本方法的精密度是由四家实验室对四个水平的试样所作的试验确定的。
