# 中华人民共和国国家标准

# 大气降水中亚硝酸盐测定

# N-(1-萘基)-乙二胺光度法

GB 13580.7-92

Determination of nitrite in the wet precipitation—N-(1-naphthyl)-1,2-diaminoethane dihydrochloride spectrophotometry

# 1 主题内容与适用范围

1.1 本标准规定了测定大气降水中亚硝酸盐的N-(1-萘基)-乙二胺光度法。
1.2 本标准适用于大气降水样品中亚硝酸盐的测定。
1.3 本标准最低检出浓度为 $0.04\mathrm{mg / L}$ ，测定范围为 $0.01\sim 0.02\mathrm{mg / L}$ 。

# 2 引用标准

GB13580.2 大气降水样品的采集与保存

# 3 原理

在 $\mathrm{pH}1.7$ 以下，亚硝酸盐和对氨基苯磺酸反应生成重氮盐，再与N-(1-萘基)-乙二胺偶联生成红色染料，于 $540\mathrm{nm}$ 波长处测量吸光度。根据试样吸光度和亚硝酸盐浓度成正比的关系，即可进行定量。

# 4 试剂

4.1 亚硝酸盐 $\left(\mathrm{NO}_{2}^{-}\right)$ 标准贮备液： $1000 \mu \mathrm{g} / \mathrm{mL}$ 。准确称取 $1.4998 \mathrm{~g}$ 亚硝酸钠（干燥器中干燥 $24 \mathrm{~h}$ ），溶于水，并定容至 $1000 \mathrm{~mL}$ 。
4.2 亚硝酸盐标准使用液: $10 \mu \mathrm{g} / \mathrm{mL}$ 。准确吸取 $5.00 \mathrm{~mL}$ 亚硝酸盐的标准贮备液于 $500 \mathrm{~mL}$ 容量瓶中, 用水稀释至刻度。使用时稀释配制。
4.3盐酸溶液： $(1 + 6)$ 。量取 $50~\mathrm{mL}$ 浓盐酸加入到 $300\mathrm{mL}$ 水中，摇匀。
4.4 对氨基苯磺酸溶液: $10 \mathrm{~g} / \mathrm{L}$ 。称取 $5 \mathrm{~g}$ 对氨基苯磺酸, 溶于 $350 \mathrm{~mL}(1+6)$ 盐酸溶液(4.3)中, 用水稀释至 $500 \mathrm{~mL}$ 。此试剂可稳定数月。
4.5盐酸N-(1-萘基)-乙二胺 $1\mathrm{g / L}$ 。称取 $0.5\mathrm{g}$ 盐酸N-(1-萘基)-乙二胺 $\mathrm{(C_{10}H_{17}NHCH_2CH_2NH_2}$ ·2HCl)溶于 $500~\mathrm{mL}$ 水中，贮存于棕色瓶中，在冰箱里保存。此试剂可稳定数周。如变成深棕色则弃去，重新配制。

# 5 仪器

5.1 分光光度计。
5.2 $25\mathrm{mL}$ 比色管。

# 6 样品采集与保存

按GB13580.2执行。

# 7 步骤

7.1 校准曲线的绘制：取 $25\mathrm{mL}$ 比色管6支，分别加入亚硝酸盐标准使用液(4.2)0,0.50,1.00,2.50,5.00,10.0mL，用水稀释至标线。各加入 $1.0\mathrm{mL}$ 对氨基苯磺酸溶液(4.4)，摇匀后放置 $2\sim 8\mathrm{min}$ 。加 $1.0\mathrm{mL}$ N-(1-萘基)-乙二胺溶液(4.5)，摇匀。以水作参比，用 $10\mathrm{mm}$ 吸收池，在 $540\mathrm{nm}$ 波长处测量吸光度，绘制校准曲线。
7.2 样品测定：根据降水中亚硝酸盐含量，吸取 $10.0 \sim 15.0 \mathrm{~mL}$ 样品于 $25 \mathrm{~mL}$ 比色管中，加水至 $25 \mathrm{~mL}$ 。以下按绘制校准曲线的步骤(7.1)进行操作，测量试样的吸光度，从校准曲线上查得亚硝酸盐的含量。

# 8 分析结果的表述

降水中亚硝酸盐（按 $\mathrm{NO}_2^-$ 计）浓度以 $\mathrm{mg / L}$ 表示，按下式计算：

$$
C = \frac {M}{V}
$$

式中： $C$ —样品中亚硝酸盐浓度， $\mathrm{mg / L}$

$M$ ——从校准曲线上查得亚硝酸盐含量, $\mu \mathrm{g}$ ;

$V$ ——取样体积， $\mathrm{mL}$ 。

# 9 精密度和准确度

5个实验室对含 $\mathrm{NO}_2^{-}0.25\mathrm{mg / L}$ ，并含有 $\mathrm{Na}^{+}0.13\mathrm{mg / L}$ 的合成水样进行验证实验。测定结果 $\mathrm{NO}_2^-$ 的相对标准偏差为 $0.13\%$ ，相对误差为 $3.8\%$ 。
