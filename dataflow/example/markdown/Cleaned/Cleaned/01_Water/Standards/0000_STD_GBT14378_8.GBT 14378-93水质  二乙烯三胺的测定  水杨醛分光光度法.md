# 中华人民共和国国家标准

# 水质二乙烯三胺的测定

# 水杨醛分光光度法

GB/T14378-93

Water quality-Determination of diethylenetriamine

Salicylicaldehyde spectrophotometric method

# 1 主题内容与适用范围

# 1.1 主题内容

本标准规定了测定水中二乙烯三胺的水杨醛分光光度法。

# 1.2 适用范围

1.2.1 本方法适用于地面水、航天工业废水中二乙烯三胺的测定。
1.2.2 二乙烯三胺的测定范围： $0.4\sim 3.2\mathrm{mg / L}$ 。水样中二乙烯三胺含量大于 $3.2\mathrm{mg / L}$ 时，可稀释后按本方法测定。
1.2.3水中存在偏二甲基肼、硝基甲烷、 $\mathrm{NH_4^+}$ 等干扰物，其浓度为二乙烯三胺浓度5倍以内时，干扰很小，可不计；水中存在二甲苯胺、三乙胺、 $\mathrm{NO}_3^-$ 、 $\mathrm{NO}_2^-$ 等干扰物，其浓度为二乙烯三胺浓度10倍以内时，干扰很小，可不计；甲醛含量高于 $0.8\mathrm{mg / L}$ 时，会产生负干扰。

# 2 方法原理

二乙烯三胺和水杨醛的碱性反应产物，在 $\mathrm{pH}$ 为3.5左右时可与硫酸钴产生化学反应，生成黄色化合物，颜色的深度与二乙烯三胺的含量成正比，用分光光度计在 $390\mathrm{nm}$ 波长处测定。

# 3 试剂

除另有说明，本标准所用试剂均为符合国家标准或专业标准试剂、蒸馏水或等纯度的水。

3.2 乙醇： $95\%$ 以上。
3.3 二乙烯三胺 $(\mathrm{C_4H_{13}N_3})$ ： $\rho = 0.9586\mathrm{g / mL}$ ，纯度 $98\%$ 以上。
3.4 硫酸溶液： $c(\mathrm{H}_2\mathrm{SO}_4) = 1.0\mathrm{mol / L}$

3.5 氢氧化钠溶液： $0.2\mathrm{g} / 100\mathrm{mL}$

称取氢氧化钠 $(\mathrm{NaOH})0.2\mathrm{g}$ ，溶于 $100~\mathrm{mL}$ 水中。

3.6 水杨醛-乙醇溶液。

吸取水杨醛 $\mathrm{(C_6H_4OHCHO)}$ 0.4mL，溶于 $100~\mathrm{mL}$ 乙醇中，此液配制后暗处保存可使用3d。

# 4 仪器

# 4.1 分光光度计：配 $3\mathrm{cm}$ 比色皿。

# 4.2 玻璃仪器：

# 4.2.1 比色管： $25~\mathrm{mL}$ 具塞比色管。

# 4.2.2 容量瓶： $1000\mathrm{mL},500\mathrm{mL},100\mathrm{mL}$

# 4.2.3 移液管： $10\mathrm{mL},5\mathrm{mL},1\mathrm{mL}$

# 5 分析步骤

# 5.1 标准曲线绘制

5.1.1 分别吸取 $0.00, 0.50, 1.00, 1.50, 2.00, 2.50, 3.00, 4.00 \mathrm{~mL}$ 二乙烯三胺标准溶液(3.10)，注入一组 $25 \mathrm{~mL}$ 的比色管中，加入水，水与二乙烯三胺标准溶液之体积和为 $10 \mathrm{~mL}$ 。加入氢氧化钠溶液(3.5) $0.2 \mathrm{~mL}$ ，混匀后，加入水杨醛-乙醇溶液(3.6) $1.5 \mathrm{~mL}$ ，充分振荡后放置 $6 \mathrm{~min}$ ，加入硫酸钴溶液(3.7) $0.2 \mathrm{~mL}$ ，充分振荡后，放置 $4 \mathrm{~min}$ ，加入乙酸-乙酸钠缓冲溶液 $2.0 \mathrm{~mL}$ ，摇匀后，用水稀释至标线。

5.1.2 放置 $10\mathrm{min}$ 后，在分光光度计 $390\mathrm{nm}$ 处，以空白液为参比液，使用 $3\mathrm{cm}$ 光程比色皿测定吸光度。

5.1.3 根据测得的吸光度与相应的二乙烯三胺含量，绘制标准曲线，求出回归方程 $Y = bX + a$ 。

# 5.2 测定

采集水样经澄清后，用 $c(\mathrm{H}_2\mathrm{SO}_4) = 1.0\mathrm{mol / L}$ 硫酸调 $\mathrm{pH}$ 值为7左右，吸取水样 $10.0\mathrm{mL}$ 于 $25\mathrm{mL}$ 具塞比色管中，按 $5.1.1\sim 5.1.3$ 方法测定吸光度。从标准曲线上查得或按回归方程算出相应的二乙烯三胺含量。

# 5.3水中存在胖时的测定方法

5.3.1 取不同含量的肼溶液于 $25\mathrm{mL}$ 具塞比色管中，用 $5.1.1\sim 5.1.3$ 方法制作肼的校正曲线（注意比色前需用定量滤纸过滤）。
5.3.2 按 $5.1.1\sim 5.1.3$ 方法测出水样的吸光度值 $A_{1}$
5.3.3 水样中肼的含量 $C_1$ 的测定；可参照附录A（参考件）。
5.3.4 在肼的校正曲线上，查出 $C_1$ 含量的相对应的吸光度值 $A_{2}$ 。
5.3.5 $A_{3} = A_{1} - A_{2}$ ，在二乙烯三胺的标准曲线上查出 $A_{3}$ 吸光度值相对应的含量，即为水样中二乙烯三胺的含量。

# 6 结果的表述

按下式计算水中二乙烯三胺的浓度：

$$
C = \frac {W}{V}
$$

式中： $C$ ——水中二乙烯三胺浓度， $\mathrm{mg / L}$

$W$ ——标准曲线中查得或按回归方程算出的样品中三乙胺含量； $\mu \mathrm{g}$ ；
$V$ ——分析时所取水样的体积， $\mathrm{mL}$ 。

# 7. 精密度

二乙烯三胺浓度在 $0.4\mathrm{mg / L}$ 时，相对标准偏差为 $5.5\%$ ，二乙烯三胺浓度在 $3.2\mathrm{mg / L}$ 时，相对标准偏差为 $1.25\%$ 。

# 8注意事项

8.1 温度会对测量结果有影响，温度高时，测量值偏低。水样的二乙烯三胺含量测定应与标准曲线制作同时进行。
8.2 本方法适宜温度范围 $5\sim 30^{\circ}\mathrm{C}$

# 对二甲氨基苯甲醛分光光度法

# A1 方法原理

水中微量肼在硫酸介质中与对二甲胺基苯甲醛-乙醇溶液反应，生成黄色联氮化合物。在测定范围内黄色深度与肼含量成正比。符合郎伯-比尔定律，黄色络合物最大吸收波长是 $458~\mathrm{nm}$ 。

水中偏二甲基肼、硝酸盐、氨、氟对本方法无干扰。

一甲基肼含量在肼含量3倍以内干扰可不计。

肼测定范围： $0.002\sim 0.30\mathrm{mg / L}$ ，水样中肼含量大于 $0.30\mathrm{mg / L}$ 时，可稀释后按本方法测定。

# A2 仪器与器皿

A2.1 分光光度计：721型配有光程 $1\mathrm{cm}$ 和 $5\mathrm{cm}$ 比色皿。
A2.2 容量瓶： $1000\mathrm{mL}$ ，4只； $500\mathrm{mL}$ ，2只； $50\mathrm{mL}$ ，16只。
A2.3 棕色容量瓶： $500~\mathrm{mL}$ ，2只
A2.4 分度吸管： $10\mathrm{mL},3$ 只； $5\mathrm{mL},3$ 只； $1\mathrm{mL},2$ 只
A2.5 酸式滴定管： $50~\mathrm{mL},1$ 只； $15~\mathrm{mL},1$ 只

# A3 试剂

A3.1 对二甲氨基苯甲醛：分析纯。
A3.2 硫酸肼：分析纯。
A3.3 乙醇：分析纯，含量 $95\%$
A3.4 硫酸：分析纯。
A3.5 硫酸： $c(1 / 2\mathrm{H}_2\mathrm{SO}_4) = 0.5\mathrm{mol / L}$

在 $1000\mathrm{mL}$ 容量瓶中加入 $500\mathrm{mL}$ 蒸馏水，缓缓注入 $13.9\mathrm{mL}$ 浓硫酸，用蒸馏水稀释至刻度、摇匀。

A3.6 硫酸： $c(1 / 2\mathrm{H}_2\mathrm{SO}_4) = 1.0\mathrm{mol / L}$

在 $1000\mathrm{mL}$ 容量瓶中加入 $500\mathrm{mL}$ 蒸馏水，缓缓注入 $27.8\mathrm{mL}$ 浓硫酸，用蒸馏水稀释至刻度、摇匀。

A3.7 硫酸： $c(1 / 2\mathrm{H}_2\mathrm{SO}_4) = 9.0\mathrm{mol / L}$

在 $1000\mathrm{mL}$ 容量瓶中加入 $500\mathrm{mL}$ 蒸馏水，缓缓注入 $250\mathrm{mL}$ 浓硫酸，用蒸馏水稀释至刻度、摇匀。

A3.8 显色剂：对二甲氨基苯甲醛-乙醇溶液。

称取 $2.0\mathrm{g}$ 对二甲氨基苯甲醛于 $250~\mathrm{mL}$ 棕色容量瓶中，加入 $100~\mathrm{mL}$ 乙醇，缓缓注入 $9\mathrm{mol} / \mathrm{L}$ 硫酸 $10~\mathrm{mL}$ 。摇匀。

A3.9 胖标准贮备液。

准确称取 $4.066\mathrm{g}$ 硫酸肼，用 $0.5\mathrm{mol} / \mathrm{L}$ 硫酸溶液溶解后，移入 $1000\mathrm{mL}$ 容量瓶中，用 $0.5\mathrm{mol} / \mathrm{L}$ 硫酸稀释至刻度、摇匀。此溶液 $1\mathrm{mL}$ 含肼 $1000\mu \mathrm{g}$ 。

A3.10 胖标准工作液

吸取 $1\mathrm{mL}$ 肼标准贮备液于 $1000\mathrm{mL}$ 容量瓶中，用 $0.5\mathrm{mol} / \mathrm{L}$ 硫酸稀释至刻度，摇匀。此溶液 $1\mathrm{mL}$ 含肼 $1\mu \mathrm{g}$ 。

# A4 分析步骤

# A4.1 标准曲线绘制

# A4.1.1 水样中不存在一甲基肼和偏二甲基肼时肼标准曲线绘制

取 $50~\mathrm{mL}$ 容量瓶8只，依次加入 $0.0,0.10,0.20,0.40,0.50,1.00,1.50,2.00\mathrm{mL}$ 胺标准工作液，再向每只容量瓶中加入 $1\mathrm{mol} / \mathrm{L}$ 硫酸 $10\mathrm{mL}$ ，加蒸馏水至总体积约为 $43~\mathrm{mL}$ ，加显色剂 $5\mathrm{mL}$ 。然后用蒸馏水稀释至刻度，摇匀，放置 $15\sim 30\mathrm{min}$ ，用 $5\mathrm{cm}$ 光程比色皿以蒸馏水作参比，在波长 $458~\mathrm{nm}$ 处测吸光度，减去空白液的吸光度为每种溶液的净吸光度，在坐标纸上绘制肼含量 $(\mu \mathrm{g})$ -吸光度曲线，求出回归方程 $Y = bX + a$ 。

# A4.1.2 水样中存在一甲基肼和偏二甲基肼时肼标准曲线绘制

取 $50~\mathrm{mL}$ 容量瓶8只，依次加入 $0.00,0.20,0.40,0.50,1.00,1.50,2.00~\mathrm{mL}$ 胺标准工作液，加 $1\mathrm{mol} / \mathrm{L}$ 硫酸 $30~\mathrm{mL}$ ，加蒸馏水至体积约为 $43~\mathrm{mL}$ 。再加显色剂 $5\mathrm{mL}$ ，然后用蒸馏水稀释至刻度，摇匀，放置 $15\sim 30\mathrm{min}$ ，按A4.1.1方法绘制曲线。

# A4.2 样品分析

# A4.2.1 采样

水样用玻璃瓶采集，采集后加酸或加碱调 $\mathrm{pH}$ 至中性，尽快测定。

# A4.2.2 样品前处理

A4.2.2.1 水样中不存在一甲基肼和偏二甲基肼时，取A4.2.1中性水样约 $400~\mathrm{mL}$ 置于一只清洁、干燥的 $500~\mathrm{mL}$ 容量瓶中，慢慢加入浓硫酸 $2.8~\mathrm{mL}$ ，再加此中性水样至刻度，摇匀，冷却即为样品溶液（I）。
A4.2.2.2 水样中存在一甲基肼和偏二甲基肼时，取A4.2.1中性水样约 $400~\mathrm{mL}$ 置于一只清洁、干燥的 $500~\mathrm{mL}$ 容量瓶中，慢慢加入浓硫酸 $8.3~\mathrm{mL}$ ，再用此水样加至刻度，摇匀，冷却即为样品溶液（I）。

# A4.2.3 比色测定

取 $50~\mathrm{mL}$ 清洁、干燥的容量瓶3只，分别加入 $5\mathrm{mL}$ 显色剂，然后分别加入样品溶液（I）或样品溶液（Ⅱ）至刻度，摇匀，放置 $15\sim 30\mathrm{min}$ ，在 $458\mathrm{nm}$ 波长处用 $5\mathrm{cm}$ 光程比色皿，以蒸馏水做参比液，测定吸光度，减去空白试验测得的吸光度后，从A4.1.1或A4.1.2标准曲线上查得胖的含量 $(\mu \mathrm{g})$ 或按回归方程计算得胖的含量 $(\mu \mathrm{g})$ 。

# A5 计算

$$
C = W / V \tag {A1}
$$

式中： $C$ ——水样中肼的浓度， $\mu \mathrm{g} / \mathrm{L}$

$W$ ——标准曲线上查得或按回归方程计算出的水样中胖的含量， $\mu \mathrm{g}$ ；

$V$ ——样品溶液体积， $\mathrm{mL}$ 。

测样品溶液（I）时， $V$ 为 $44.75\mathrm{mL}$

测样品溶液（Ⅱ）时， $V$ 为 $44.24\mathrm{mL}$

# A6 测定误差

在 $c(1 / 2\mathrm{H}_2\mathrm{SO}_4) = 1\mathrm{mol / L}$ 硫酸 $10\mathrm{mL}$ 的条件下测肼时，相对标准偏差为 $0.8\%$ ；在 $c(1 / 2\mathrm{H}_2\mathrm{SO}_4)$ $= 1\mathrm{mol / L}$ 硫酸 $30\mathrm{mL}$ 的条件下测肼时，相对标准偏差为 $1.5\%$ 。

# A7 注意事项

A7.1 本分析方法灵敏度及显色所需时间均受环境温度的影响，因此，水样中肼含量的测定应与绘制工作曲线同时进行。
A7.2 当被测样品溶液中含氯量较高时，可在绘制肼标准曲线的每个肼标准工作液中加入 $1\mathrm{mol} / \mathrm{L}$ 盐酸 $1\mathrm{mL}$ 。然后按A4.1.1或A4.1.2操作。
