# 1 主题内容与适用范围

1.1 本标准规定了测定废水中银的原子吸收分光光度法。
1.2 本标准适用于感光材料生产、胶片洗印、镀银、冶炼等行业排放废水及受银污染的地面水中银的测定。
1.3 本标准的最低检出浓度为 $0.03\mathrm{mg / L}$ ，测定上限为 $5.0\mathrm{mg / L}$ 。经稀释或浓缩测定范围可以扩展。
1.4 大量氯化物、溴化物、碘化物、硫代硫酸盐对银的测定有干扰，但试样经消解处理后，干扰可被消除。

# 2 原理

将消解处理后的试液吸入火焰，火焰类型为空气-乙炔，氧化型(蓝色)。在火焰中，银离子形成基态原子，对波长为 $328.1\mathrm{nm}$ 的特征电磁辐射产生吸收。将测得试样的吸光度和标准溶液的吸光度相比较，确定试样中银的浓度。

# 3 试剂

除非另有说明，分析时均使用符合国家标准或专业标准分析纯试剂，去离子水或同等纯度的水。

3.1 硝酸 $(\mathrm{HNO}_3)$ ， $\rho = 1.42\mathrm{g / mL}$
3.2 高氯酸 $\mathrm{(HClO_4)}$ ， $\rho = 1.68\mathrm{g / mL}$
3.3 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ ， $\rho = 1.84\mathrm{g / mL}$
3.4 过氧化氢 $(\mathrm{H}_2\mathrm{O}_2)$ ， $30\% (m / V)$
3.5 硝酸溶液， $1 + 1$
3.6 银标准贮备液, $1000 \mathrm{mg} / \mathrm{L}$ : 准确称取 $0.1575 \mathrm{~g}$ 硝酸银 $\left(\mathrm{AgNO}_{3}\right)$ , 溶于适量水中, 加入 $2 \mathrm{~mL}$ 硝酸溶液 (3.5), 转入 $100 \mathrm{~mL}$ 容量瓶中, 用水稀释至标线, 摇匀。贮于棕色细口瓶中, $4^{\circ} \mathrm{C}$ 下存放。此溶液可保存半年。
3.7 银标准溶液, $50.0 \mathrm{mg} / \mathrm{L}$ : 准确吸取银标准贮备溶液(3.6) $10.00 \mathrm{~mL}$ , 置于 $200 \mathrm{~mL}$ 棕色容量瓶中, 加入 $4 \mathrm{~mL}$ 硝酸溶液(3.5), 用水稀释至标线。此溶液可稳定两周。

# 4 仪器

4.1 一般实验室仪器和以下仪器。
4.2 原子吸收分光光度计。
4.3. 银空心阴极灯。

4.4 乙炔的供气装置。
4.5 空气压缩机，加除油、水及杂质装置。

# 5 采样及样品

采用聚乙烯瓶这类合适的容器贮存样品，用硝酸(3.1)将水样酸化至 $\mathrm{pH}1\sim 2$ ，并尽快分析。感光材料的生产、胶片洗印及镀银等行业的废水，样品采集后不加酸，并立即进行分析。含银水样应避免光照。

# 6 步骤

# 6.1 试料

取 $50~\mathrm{mL}$ 均匀样品置于 $150~\mathrm{mL}$ 烧杯中，如含银浓度大于 $5\mathrm{mg / L}$ ，可适量少取样品，加水至 $50~\mathrm{mL}$ 。

# 6.2 测定

6.2.1 前处理：在试料(6.1)中，依次加入硝酸(3.1) $10\mathrm{mL}$ ，硫酸(3.3) $1\mathrm{mL}$ ，过氧化氢(3.4) $1\mathrm{mL}$ ，在电热板上蒸至冒白烟。冷却后，加入 $2\mathrm{mL}$ 高氯酸(3.2)，加盖表面皿，继续加热至冒白烟并蒸至近干，冷却后，加硝酸溶液(3.5) $2\mathrm{mL}$ 溶解残渣，然后小心用水洗入 $50\mathrm{mL}$ 容量瓶中，稀释至标线，摇匀，备测。
6.2.2 测定：遵照仪器使用说明书调节仪器至最佳工作条件，测定试液的吸光度。
6.2.3 空白试验：用水代替试料按步骤(6.2.1)做空白试验，并按步骤(6.2.2)进行测定。

# 6.3 校准曲线的绘制

在 $50~\mathrm{mL}$ 容量瓶中，加入 $2\mathrm{mL}$ 硝酸溶液(3.5)及银标准溶液(3.7)，配制至少五个标准工作溶液，其浓度范围应包括试料中被测银的浓度，按(6.2.2)中的条件测其吸光度。

用减去空白的吸光度与相对应的银含量 $(\mathrm{mg / L})$ 绘制校准曲线。

# 7 结果的表示

7.1 银含量 $C(\mathrm{mg / L})$ 按下式计算：

$$
C = f C ^ {\prime}
$$

式中： $C$ ——试样中银的含量， $\mathrm{mg / L}$

$C^{\prime}$ ——由校准曲线上查得的含银浓度， $\mathrm{mg / L}$

$f$ ——稀释比（定容体积比取样体积）。

7.2 银含量，用回归方程计算。

# 8 精密度和准确度

四个实验室分析用蒸馏水配制的含银 $1\mathrm{mg / L}$ 的统一样品（样品加氨水和碘化氰(CNI)保存）。

# 8.1·重复性

重复性相对标准偏差为 $1.2\%$

# 8.2 再现性

再现性相对标准偏差为 $2.6\%$

# 8.3 准确度

相对误差为 $+1.1\%$ 。

# 9特殊情况的说明

9.1 试样在消解过程中不宜蒸干，否则银有损失。
9.2 当样品成分复杂，含有机质较多或有沉淀时，应用硝酸-高氯酸反复消解几次，直至溶液澄清为止。
9.3 有沉淀或悬浮物的样品，如胶片洗印度水等，应尽量取均匀试样制备试料。

9.4 即使用浓硝酸将样品酸化到 $\mathrm{pH}1 \sim 2$ ，也不宜贮存，应尽快分析。不宜酸化的样品，如原产、胶片洗印等行业废水，采样后，应立即分析。
