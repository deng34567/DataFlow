# 中华人民共和国国家标准

# 大气降水中硫酸盐测定

GB 13580.6-92

Determination of sulfate in the wet precipitation

# 1 主题内容与适用范围

1.1 本标准规定了测定大气降水中硫酸盐的硫酸钡浊度法和铬酸钡-二苯碳酰二肼光度法。
1.2 本标准适用于大气降水样品中硫酸盐的测定。
1.3 本标准分两篇：第一篇硫酸钡浊度法，第二篇铬酸钡-二苯碳酰二肼光度法。硫酸钡浊度法最低检出浓度为 $0.4\mathrm{mg / L}$ ，测定范围为 $1.0\sim 70\mathrm{mg / L}$ 。

铬酸钡-二苯碳酰二肼光度法最低检出浓度为 $0.1\mathrm{mg / L}$ ，测定范围为 $0.5\sim 10\mathrm{mg / L}$ 。

# 2 引用标准

GB13580.2 大气降水样品的采集与保存

# 第一篇硫酸钡浊度法

# 3 原理

降水中硫酸盐和钡离子生成细的悬浮体，使溶液浑浊。其浑浊程度与样品中硫酸盐的含量成比例。测量硫酸钡悬浮液吸光度，即可由校准曲线求出硫酸盐的含量。

# 4 试剂

4.1 硫酸盐标准贮备液： $1000 \mu \mathrm{g} / \mathrm{mL}$ 。称取 $1.8140 \mathrm{~g}$ 硫酸钾（ $105^{\circ} \mathrm{C}$ 烘干 $2 \mathrm{~h}$ ），溶于水，定容至 $1000 \mathrm{~mL}$ 。
4.2 硫酸盐标准使用液: $100\mu \mathrm{g} / \mathrm{mL}$ 。吸取硫酸盐标准贮备液(4.1) $10.00\mathrm{mL}$ 于 $100\mathrm{mL}$ 容量瓶, 用水稀释到刻度。
4.3 明胶溶液： $3.0\mathrm{g/L}$ 。称取 $3.0\mathrm{g}$ 明胶，搅拌溶解在 $1000\mathrm{mL}70^{\circ}\mathrm{C}$ 水中，冷却后，放在冰箱中冷藏 $24\mathrm{h}$ 以上。
4.4 氯化钡-明胶溶液。称取 $3.0\mathrm{g} / \mathrm{L}$ 明胶溶液(4.3) $500\mathrm{mL}$ ，溶解 $10.0\mathrm{g}$ 氯化钡 $(\mathrm{BaCl}_2\cdot 2\mathrm{H}_2\mathrm{O})$ ，用盐酸(HCl)调至 $\mathrm{pH}1\sim 2$ 。此溶液可稳定大约1个月。

# 5 仪器

5.1 分光光度计或光电比色计。
5.2 电磁搅拌器。

# 6样品采集与保存

按GB13580.2执行。

# 7 步骤

7.1 校准曲线的绘制：取 $50~\mathrm{mL}$ 平底烧杯10个，分别加入硫酸盐标准使用液(4.2)0,0.10,0.20,0.60,1.00,1.50,2.00,3.00,4.00,5.00mL，分别加入水9.00,8.90,8.80,8.40,8.00,7.50,7.00,6.00,5.00,4.00mL，再准确加入明胶溶液(4.3)1.00mL，将配好标准系列溶液的烧杯，逐个放在磁力搅拌器(5.2)上，搅拌10s，迅速加入 $2\mathrm{mL}$ 氯化钡-明胶溶液(4.4)，同时用秒表准确计时，搅拌2.0min。静置1.5min后，于 $420\mathrm{nm}$ 波长处，用 $20\mathrm{mm}$ 吸收池，以水作参比测一吸光度。用吸光度对相应硫酸盐含量绘制校准曲线。
7.2 样品测定：根据降水中硫酸盐含量，准确吸取 $5.00\sim 9.00\mathrm{mL}$ 样品，加入适量水，使体积为 $9.00\mathrm{mL}$ ，以下按绘制校准曲线的步骤(7.1)进行操作。由吸光度从校准曲线上查出硫酸盐的含量。

# 8 分析结果的表述

降水中硫酸盐(按 $\mathrm{SO}_4^{2-}$ 计)浓度以 $\mathrm{mg/L}$ 表示，按式(1)计算：

$$
C = \frac {M}{V} \quad \dots \tag {1}
$$

式中： $C$ ——样品中硫酸盐浓度， $\mathrm{mg / L}$

$M$ ——从校准曲线上查得硫酸盐含量， $\mu \mathrm{g}$ ；

$V$ ——取样体积， $\mathrm{mL}$ 。

# 9 精密度和准确度

5个实验室对含 $\mathrm{SO}_4^{2-}6.00\mathrm{mg/L},\mathrm{F}^{-}0.20\mathrm{mg/L},\mathrm{Cl}1.00\mathrm{mg/L},\mathrm{NO}_3^{-}1.2\mathrm{mg/L},\mathrm{K}^{+}0.44\mathrm{mg/L}$ 的合成水样进行测定，测定结果 $\mathrm{SO}_4^{2-}$ 的相对标准偏差为 $0.74\%$ ，相对误差为 $-1.2\%$ 。

# 第二篇铬酸钡-二苯碳酰二肼光度法

# 10 原理

在弱酸性溶液中，硫酸根与铬酸钡悬浮液发生下述交换反应：

$$
\mathrm {S O} _ {4} ^ {2 -} + \mathrm {B a C r O} _ {4} \longrightarrow \mathrm {B a S O} _ {4} \downarrow + \mathrm {C r O} _ {4} ^ {2 -}
$$

在乙醇-氨性溶液介质中，分离除去过量的铬酸钡。交换释放出的 $\mathrm{CrO_4^{2-}}$ 用二苯碳酰二肼显色，于波长 $545\mathrm{nm}$ 处测定吸光度，间接确定 $\mathrm{SO_4^{2-}}$ 浓度。

# 11 试剂

11.1 硫酸盐标准贮备液： $1000 \mu \mathrm{g} / \mathrm{mL}$ 。称取 $1.8140 \mathrm{~g}$ 硫酸钾（ $105^{\circ} \mathrm{C}$ 烘干 $2 \mathrm{~h}$ ），溶于水，定容至 $1000 \mathrm{~mL}$ 。
11.2 硫酸盐标准使用液: $10 \mu \mathrm{g} / \mathrm{mL}$ 。吸取硫酸盐标准贮备液 $5.00 \mathrm{~mL}$ 于 $500 \mathrm{~mL}$ 容量瓶中, 用水稀释至刻度。
11.3 铬酸钡的制备：称取 $5\mathrm{g}$ 氯化钡、 $3\mathrm{g}$ 重铬酸钾，分别溶解在 $50\mathrm{mL}$ 水中，混匀，加入 $16.7\mathrm{mL}$ 盐酸，加水至 $500\mathrm{mL}$ ，加热 $(70\sim 80^{\circ}\mathrm{C})$ 使之溶解，加3滴 $0.1\%$ 溴百酚蓝，用 $6\mathrm{mol} / \mathrm{L}$ 氨水中和至溶液为蓝色，析出沉淀。用温水倾洗沉淀 $2\sim 3$ 次，再用冷水充分洗涤，经滤膜过滤，沉淀在 $105^{\circ}\mathrm{C}$ 干燥 $2\mathrm{h}$ ，研细备用。
11.4 铬酸钡悬浮液：称取 $0.5\mathrm{g}$ 精制的铬酸钡(11.3)于 $200\mathrm{mL}$ 含有 $0.42\mathrm{mL}$ 盐酸和 $14.7\mathrm{mL}$ 冰乙酸的水中，得悬浮液。贮于聚乙烯瓶中，使用前充分摇匀。
11.5 氨水溶液： $6\mathrm{mol} / \mathrm{L}$ 。吸取 $45\mathrm{mL}$ 浓氨水用水稀释到 $100\mathrm{mL}$ 。

11.6 氨-氯化钙溶液：称取 $1.10\mathrm{g}$ 氯化钙，用少量稀盐酸溶解，加 $6\mathrm{mol} / \mathrm{L}$ 氨水溶液至 $400\mathrm{mL}$ 。
11.7 盐酸溶液： $2\mathrm{mol} / \mathrm{L}$ 。吸取 $16.8\mathrm{mL}$ 浓盐酸用水稀释到 $100\mathrm{mL}$ 。
11.8 二苯碳酰二肼溶液: $5 \mathrm{~g} / \mathrm{L}$ 。称取 $0.5 \mathrm{~g}$ 二苯碳酰二肼溶于 $100 \mathrm{~mL}$ 乙醇中, 加 $1 \mathrm{~mL}$ 盐酸溶液 (11.7) 作为稳定剂。于冷暗处保存。

# 12 仪器

12.1 分光光度计。
12.2 过滤装置（见GB13580.2图1）。

# 13 样品采集与保存

按GB13580执行。

# 14 步骤

14.1 校准曲线的绘制：取 $25\mathrm{mL}$ 比色管7支，分别加入硫酸盐标准使用液(11.2)0,0.50,1.00,2.00,4.00,6.00,10.00mL，分别加水至 $10.0\mathrm{mL}$ ，再加入 $2.0\mathrm{mL}$ 铬酸钡悬浮液(11.4)，1.0mL氨-氯化钙溶液(2.6)， $10\mathrm{mL}95\%$ 乙醇(每加一种试剂均需摇匀)，加水至刻度，摇匀。其 $\mathrm{SO}_4^{2-}$ 含量分别为：5.0，10.0,20.0,40.0,60.0,100 $\mu$ g。置于冷水(15℃)中冷却 $10\mathrm{min}$ ，取出，经滤膜过滤于干燥比色管中1，吸取 $5.00\mathrm{mL}$ 滤液于 $10\mathrm{mL}$ 比色管中，加入二苯碳酰二肼溶液(11.8)10mL，盐酸溶液(11.7)1.0mL，加水至刻度，摇匀。 $10\mathrm{min}$ 后，于 $545\mathrm{nm}$ 波长，用 $10\mathrm{mm}$ 吸收池，以空白试验溶液作参比，测量吸光度。以吸光度对硫酸盐含量作图，绘制校准曲线。

注：1）滤膜上的铬酸钡毒性大，应留作专门处理，防止对环境造成污染。

14.2 样品测定：根据降水中硫酸盐含量，吸取 $1.00\sim 10.00\mathrm{mL}$ 样品于 $25\mathrm{mL}$ 比色管中，按绘制校准曲线的步骤(14.1)进行测定，由测得的吸光度从校准曲线上查得硫酸盐的含量。

# 15 分析结果的表述

降水中硫酸盐（按 $\mathrm{SO}_4^{2-}$ 计）浓度以 $\mathrm{mg/L}$ 表示，按式(2)计算：

$$
C = \frac {M}{V} \quad \dots \tag {2}
$$

式中： $C$ ——样品中硫酸盐浓度， $\mathrm{mg / L}$

$M$ ——从校准曲线上查得硫酸盐含量， $\mu \mathrm{g}$ ；

$V$ ——取样体积， $\mathrm{mL}$ 。

# 16 精密度和准确度

5个实验室对含 $\mathrm{SO}_4^{2 - }6.00\mathrm{mg / L},\mathrm{F}^{-}0.20\mathrm{mg / L},\mathrm{Cl}^{-}1.00\mathrm{mg / L},\mathrm{NO}_3^{-}1.20\mathrm{mg / L},\mathrm{K}^+$ $2.98\mathrm{mg / L},\mathrm{Na}^{+}0.44\mathrm{mg / L}$ 的合成水样，进行测定，测定结果 $\mathrm{SO}_4^{2 - }$ 的相对标准偏差为 $3.2\%$ ，相对误差为 $-1.2\%$ 。
