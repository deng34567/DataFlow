# 水质偏二甲基肼的测定

# 氨基亚铁氰化钠分光光度法

GB/T14376-93

Water quality-Determination of asymmetrical dimethyl hydrazine

Amino ferrocyanide sodium spectrophotometric method

# 1 主题内容与适用范围

# 1.1 主题内容

本标准规定了测定水中偏二甲基肼的氨基亚铁氰化钠分光光度法。

# 1.2 适用范围

1.2.1 本方法适用于地面水、航天工业废水中偏二甲基肼的测定。
1.2.2 偏二甲基肼的测定范围： $0.01\sim 1.0\mathrm{mg / L}$ 。水样中偏二甲基肼含量大于 $1.0\mathrm{mg / L}$ 时，可稀释后按本方法测定。
1.2.3 氨、尿素对本方法测定基本无干扰。肼、一甲基肼、甲醛含量在偏二甲基肼含量5倍以上干扰测定。

# 2 方法原理

微量的偏二甲基肼与氨基亚铁氰化钠在弱酸性水溶液中生成红色络合物。在测定范围内，颜色的深度与偏二甲基肼的含量成正比，用分光光度计在 $500\mathrm{nm}$ 处测定。

# 3.13 氨基亚铁氰化钠(TPF)( $\mathrm{Na}_{3}[\mathrm{Fe}(\mathrm{CN})_{5}\mathrm{NH}_{3} \cdot 3\mathrm{H}_{2}\mathrm{O}]$ )。

称取亚硝基铁氰化钠 $45\mathrm{g}$ 于 $250~\mathrm{mL}$ 锥形瓶中，缓慢加入氨水 $140.0~\mathrm{mL}$ ，边加边摇，至全部溶解。将此瓶放入 $0^{\circ}\mathrm{C}$ 左右的冰箱中过夜，加入甲醇 $250~\mathrm{mL}$ ，即析出黄色结晶，将结晶滤出、抽干，再用甲醇洗一次。所得结晶放入装有氯化钙的棕色真空干燥器中，干燥 $4\sim 6\mathrm{h}$ 。干燥后的氨基亚铁氰化钠装入棕色细口瓶中，于暗处保存，有效期一年。

# 3.16 偏二甲基肼标准贮备液： $1\mathrm{mg / L}$

# 3.17 偏二甲基肼工作液： $1\mu \mathrm{g / mL}$

# 6.1 校准曲线绘制

# 6.2 测定

当水样中亚硝酸根含量大于偏二甲基肼2倍以上时，应在加入缓冲溶液后，分别再向各管加入 $5\%$ 氨基磺酸胺 $0.2\mathrm{mL}$ ，然后再加入显色剂 $1.0\mathrm{mL}$ 。

6.2.1 水样中偏二甲基肼含量在 $50~\mu \mathrm{g / L}$ 以上时的测定：

6.2.1.1 先调水样 $\mathrm{pH}$ 在 $3.0\sim 8.0$ 。水样混浊时，要用4号砂芯漏斗过滤。

取适当体积的上述预处理过的水样于具塞刻度管中，用蒸馏水准确稀释至 $25\mathrm{mL}$ ，然后按6.1条步骤进行测定。根据测定的吸光度，按校准曲线的回归方程或查标准工作曲线，算出偏二甲基肼含量。

6.2.1.2 按式(1)计算水样的偏二甲基肼含量：

$$
C = \frac {C _ {0} \times n}{V} \tag {1}
$$

式中： $C$ ——水样中偏二甲基肼含量， $\mathrm{mg / L}$

$C_0$ ——6.2.1.1中算出的偏二甲基肼含量， $\mu \mathrm{g}$ ；

$n$ ——稀释倍数；

$V$ ——比色时所取水样体积， $\mathrm{mL}$ 。

6.2.2 水样中偏二甲基肼含量小于 $50\mu \mathrm{g} / \mathrm{L}$ 时的测定：

6.2.2.1 水样的富集（蒸馏浓缩法）：

蒸馏装置如下图。

在 $1000\mathrm{mL}$ 圆底烧瓶中，加入氯化钠 $90\mathrm{g}$ 和氢氧化钠 $120\mathrm{g}$ ，以流量 $50\mathrm{mL} / \mathrm{min}$ 通氮气 $15\mathrm{min}$ ，取水样 $500\mathrm{mL}$ 仔细的注入烧瓶中，轻轻摇动几下，然后加热蒸馏。继续通氮，接通冷却水，用具塞刻度管在冷凝管口收集冷凝液。当水样即将沸腾时，调节调压器电压在 $150\mathrm{V}$ 左右，以控制电炉加热温度，使每 $25\mathrm{mL}$ 馏分的流出时间在 $11\sim 15\mathrm{min}$ 。连续收集3管馏分，每管准确收集 $25\mathrm{mL}$ 。按6.2.1测得每管内的偏二甲基肼含量。

# 6.2.2.2 按式(2)计算水样的偏二甲基肼含量：

$$
C = \frac {W _ {1} + W _ {2} + W _ {3}}{5 0 0 \times N} \tag {2}
$$

式中： $C$ ——水样中偏二甲基肼含量， $\mathrm{mg / L}$

$W_{1}, W_{2}, W_{3}$ ——分别为3只装有馏分具塞刻度管中偏二甲基肼含量， $\mu \mathrm{g}$ ；

500—蒸馏水样的体积， $\mathrm{mL}$

$N$ ——回收率， $71\%$ 。
