# 中华人民共和国国家标准

# 水质钙和镁的测定

# 原子吸收分光光度法

GB 11905-89

Water quality-Determination of calcium and magnesium-Atomic absorption spectrophotometric method

引言

本标准等效采用国际标准ISO7980—1986《水质一钙和镁的测定一原子吸收分光光度法》。

# 1 主题内容与适用范围

# 1.1 主题内容

本标准规定了测定水中钙和镁的原子吸收分光光度法。

# 1.2 适用范围

本标准适用于测定地下水，地面水和废水中的钙、镁。

本标准适用的校准溶液浓度范围(见表1)与仪器的特性有关，随着仪器的参数变化而变化。通过样品的浓缩和稀释还可使测定实际样品浓度范围得到扩展。

$\mathrm{mg / L}$

表 1 测定范围及最低检出浓度


| 元素  | 最低检出浓度 | 测定范围     |
| --- | ------ | -------- |
| 钙   | 0.02   | 0.1~6.0  |
| 镁   | 0.002  | 0.01~0.6 |


# 1.3 干扰

原子吸收法测定钙镁的主要干扰有铝、硫酸盐、磷酸盐、硅酸盐等，它们能抑制钙、镁的原子化，产生干扰，可加入锶、镧或其它释放剂来消除干扰。火焰条件直接影响着测定灵敏度，必须选择合适的乙炔量和火焰观测高度。试样需检查是否有背景吸收，如有背景吸收应予以校正。

# 2 定义

2.1 可滤态钙、镁：未酸化的样品中能通过 $0.45\mu \mathrm{m}$ 滤膜的钙镁成分。
2.2 不可滤态钙、镁总量：未经过滤的样品经消解后测得的钙、镁浓度，或样品中可滤态和不可滤态两部分钙、镁浓度总和。

# 3 原理

将试液喷入火焰中，使钙、镁原子化，在火焰中形成的基态原子对特征谱线产生选择性吸收。由测得的样品吸光度和校准溶液的吸光度进行比较，确定样品中被测元素的浓度。选用 $422.7 \mathrm{~nm}$ 共振线的吸收测定钙，用 $285.2 \mathrm{~nm}$ 共振线的吸收测定镁。

# 4 试剂

除另有说明外，分析时均使用符合国家标准或专业标准的分析纯试剂，去离子水或同等纯度的水。

4.1. 硝酸 $(\mathrm{HNO}_3)$ , $p = 1.40 \mathrm{~g} / \mathrm{mL}$ 。
4.2 高氯酸 $(\mathrm{HClO}_4)$ ， $\rho = 1.68\mathrm{g / mL}$ ，优级纯。
4.3 硝酸溶液，1+1。
4.4 燃料：乙炔，用钢瓶气供给，也可用乙炔发生器供给，但要适当纯化。
4.5 助燃气：空气，一般由气体压缩机供给，进入燃烧器以前应经过适当过滤，以除去其中的水，油和其他杂质。
4.6 镧溶液, $0.1 \mathrm{~g} / \mathrm{mL}$ : 称取氧化镧 $\left(\mathrm{La}_{2} \mathrm{O}_{3}\right) 23.5 \mathrm{~g}$ , 用少量硝酸溶液 (4.3) 溶解, 蒸至近干, 加 $10 \mathrm{~mL}$ 硝酸溶液 (4.3) 及适量水, 微热溶解, 冷却后用水定容至 $200 \mathrm{~mL}$ 。
4.7 钙标准贮备液, $1000 \mathrm{mg} / \mathrm{L}$ ; 准确称取 $105 \sim 110^{\circ} \mathrm{C}$ 烘干过的碳酸钙 $\left(\mathrm{CaCO}_{3}, \mathrm{G}, \mathrm{R}\right) 2.4973 \mathrm{~g}$ 于 $100 \mathrm{~mL}$ 烧杯中, 加入 $20 \mathrm{~mL}$ 水, 小心滴加硝酸溶液 (4.3) 至溶解, 再多加 $10 \mathrm{~mL}$ 硝酸溶液 (4.3), 加热煮沸, 冷却后用水定容至 $1000 \mathrm{~mL}$ 。
4.8 镁标准贮备液, $100\mathrm{mg / L}$ : 准确称取 $800^{\circ}\mathrm{C}$ 灼烧至恒重的氧化镁 $(\mathrm{MgO}, \mathrm{S}, \mathrm{P}) 0.1658\mathrm{g}$ 于 $100\mathrm{mL}$ 烧杯中, 加 $20\mathrm{mL}$ 水, 滴加硝酸溶液 (4.3) 至完全溶解, 再多加 $10\mathrm{mL}$ 硝酸溶液 (4.3), 加热煮沸, 冷却后用水定容至 $1000\mathrm{mL}$ 。
4.9钙、镁混合标准溶液，钙 $50\mathrm{mg / L}$ 、镁 $5.0\mathrm{mg / L}$ ：准确吸取钙标准贮备液(4.7)和镁标准贮备液(4.8)各 $5.0\mathrm{mL}$ 于 $100\mathrm{mL}$ 容量瓶中，加入 $1\mathrm{mL}$ 硝酸溶液(4.3)，用水稀释至标线。

# 5 仪器

原子吸收分光光度计及相应的辅助设备。

# 6样品

# 6.1 样品的保存

采集代表性水样贮存于聚乙烯瓶中。采样瓶先用洗涤剂洗净，再在硝酸溶液(4.3)浸泡至少 $24\mathrm{h}$ ，然后用去离子水冲洗干净。

# 6.2 试样的制备

6.2.1 分析可滤态钙、镁时，如水样有大量的泥沙，悬浮物，样品采集后应及时澄清，澄清液通过 $0.45\mu \mathrm{m}$ 有机微孔滤膜过滤，滤液加硝酸(4.1)酸化至 $\mathsf{pH}$ 为 $1\sim 2$ 。
6.2.2 分析不可滤态钙、镁总量时，采集后立即加硝酸(4.1)酸化至 $\mathsf{pH}$ 为 $1\sim 2$ 。如果样品需要消解，则校准溶液，空白溶液也要消解。消解步骤如下：取 $100\mathrm{mL}$ 待处理样品，置于 $200\mathrm{mL}$ 烧杯中，加入 $5\mathrm{mL}$ 硝酸(4.1)，在电热板上加热消解，蒸至 $10\mathrm{mL}$ 左右，加入 $5\mathrm{mL}$ 硝酸(4.1)和 $2\mathrm{mL}$ 高氯酸(4.2)，继续消解，蒸至 $1\mathrm{mL}$ 左右，取下冷却，加水溶解残渣，通过中速滤纸，滤入 $50\mathrm{mL}$ 容量瓶中，用水稀释至标线（消解中使用的高氯酸易爆炸，要求在通风柜中进行）。

# 7 步骤

# 7.1 试料

准确吸取经预处理的试样(6.2)， $1.00\sim 10.00\mathrm{mL}$ （含钙不超过 $250\mu \mathrm{g}$ ，镁不超过 $25\mu \mathrm{g}$ ）于 $50~\mathrm{mL}$ 容量瓶中，加入 $1\mathrm{mL}$ 硝酸溶液(4.3)和 $1\mathrm{mL}$ 镧溶液(4.6)用水稀释至标线，摇匀。

# 7.2 空白试验

在测定的同时应进行空白试验。空白试验时用 $50~\mathrm{mL}$ 水取代试样。所用试剂及其用量，步骤与试料测定完全相同。

# 7.3 测定

7.3.1 根据表2选择波长和调节火焰至最佳工作条件，测定试料(7.1)的吸光度。

表 2 波长及火焰类型


| 元素  | 特征谱线波长, nm | 火焰类型      |
| --- | ---------- | --------- |
| 钙   | 422.7      | 乙炔-空气,氧化型 |
| 镁   | 285.2      | 乙炔-空气,氧化型 |


7.3.2 根据试料吸光度，在校准曲线上查出(或用回归方程计算出)试料中的钙、镁浓度。

# 8 结果的表示

$$
X = f c
$$

式中： $X$ ——钙或镁含量，以Ca或 $\mathrm{Mg}$ 计， $\mathrm{mg / L}$

$f$ ——试料定容体积与试样体积之比；

$c$ ——由校准曲线查得的钙、镁浓度， $\mathrm{mg / L}$ 。

# 9 精密度与准确度

5个实验室分析(1986年12月)统一分发的合成水样结果。水样中含钙 $40.64\mathrm{mg / L}$ , 含镁 $8.39\mathrm{mg / L}$ 。

# 9.1 重复性

重复性相对标准偏差：钙为 $1.29\%$ ，镁为 $1.52\%$ 。

# 9.2 再现性

再现性相对标准偏差：钙为 $1.72\%$ ，镁为 $1.70\%$ 。

# 9.3 准确度

相对误差：钙为 $+0.05\%$ ，镁为 $-0.30\%$ 。
