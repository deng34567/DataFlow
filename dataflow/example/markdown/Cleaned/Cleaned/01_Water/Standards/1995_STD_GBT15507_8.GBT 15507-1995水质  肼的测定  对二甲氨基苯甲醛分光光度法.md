# 中华人民共和国国家标准

# 水质胍的测定

# 对二甲氨基苯甲醛分光光度法

GB/T15507—1995

Water quality-Determination of hydrazine-p-Dime

tylaminobenzaldehyde spectrophotometric method

# 1 主题内容和适用范围

# 1.1 主题内容

本标准规定了测定水中肼的对二甲氨基苯甲醛分光光度法。

# 1.2 适用范围

1.2.1 本标准适用于地面水和工业废水中肼的测定。
1.2.2 试料体积 $1\sim 10\mathrm{mL}$ ，比色皿光程为 $1.5\mathrm{cm}$ 时，本方法检测限以肼计为 $0.002\mathrm{mg / L}$ ，测定上限为 $1.00\mathrm{mg / L}$ 。更高浓度的样品，稀释后再进行测定。
1.2.3 干扰：氨基脲、硫脲、脲素分别高达 $20,50,200\mathrm{mg / L}$ 以上时干扰测定；一甲基肼为肼含量的3倍以上， $\mathrm{NO}_2^-$ 大于 $1\mathrm{mg / L}$ 时产生干扰。

# 2 原理

在酸性溶液条件下，肼与对二甲氨基苯甲醛作用，生成对二甲氨基苄连氮黄色化合物，于波长 $458~\mathrm{nm}$ 处进行分光光度测定。

# 3 试剂

除非另有说明，本标准所用试剂均为符合国家标准或专业标准的分析纯试剂，试验中均应使用蒸馏水或同等纯度的水。

3.1 盐酸 $(\mathrm{HCl})$ ： $\rho = 1.19\mathrm{g / mL}$
3.2 盐酸溶液： $c(\mathrm{HCl}) = 1\mathrm{mol / L}$
3.3 乙醇 $95\%$
3.4对二甲氨基苯甲醛溶液：称取 $4\mathrm{g}$ 对二甲氨基苯甲醛溶于 $200\mathrm{mL}95\%$ 乙醇和 $20\mathrm{mL}$ HCl[3.1]酸中。
3.5叠氮化钠溶液： $155\mathrm{g / L}$ 。称取不含肼的叠氮化钠 $(\mathrm{NaN}_3$ ，注意剧毒)15.5g溶于水中，稀释至 $100\mathrm{mL}$ 。如叠氮化钠中含肼，可按下述步骤精制：

将叠氮化钠溶于适量水中，滤去不溶物，滤液置烧杯中，将烧杯放在电热板上加热，不断搅拌，使叠氮化钠结晶析出，待大部分叠氮化钠析出后，用滤纸过滤，经无水乙醇脱水后，将此叠氮化钠放入烘箱中于 $105 \pm 2^{\circ} \mathrm{C}$ 干燥 $2 \mathrm{~h}$ ，放在干燥器内冷却，然后放入试剂瓶中备用。

注意：叠氮化钠剧毒，精制操作应在通风柜内进行。

3.6 肼标准贮备溶液： $100\mathrm{mg / L}$ 。称取0.3280盐酸肼 $(\mathrm{N}_2\mathrm{H}_4\cdot 2\mathrm{HCl})$ 或0.4060g硫酸肼 $(\mathrm{N}_2\mathrm{H}_4^{\prime}\cdot \mathrm{H}_2\mathrm{SO}_4)$ ，用HCl溶液[3.2]溶解，定量移入 $1000\mathrm{mL}$ 容量瓶中，并用HCl[3.2]稀释至标线备

用。

3.7 肼标准溶液： $1.00\mu \mathrm{g / mL}$ 。吸取胼标准贮备液[3.6]10.0 mL，移入 $1000\mathrm{mL}$ 容量瓶中，用HCl溶液[3.2]稀释至标线。

# 4 仪器

一般实验室常规仪器和设备。

4.1 分光光度计：带光程 $1.5\mathrm{cm}$ 比色皿。
4.2 具塞比色管： $25~\mathrm{mL}$

# 5. 采样与保存

5.1 采样与贮存样品均使用玻璃瓶。
5.2 水样现场固定：用盐酸固定， $\mathrm{pH}$ 值调至小于2，可保存 $24\mathrm{h}$ 。

# 6 步骤

# 6.1 校准曲线的制备

取8支 $25~\mathrm{mL}$ 具塞比色管，分别加入肼标准溶液[3.7]，0，0.5，1.00，2.00，4.00，6.00，8.00， $10.00~\mathrm{mL}$ ，加入蒸馏水至 $10~\mathrm{mL}$ ，加入 $10~\mathrm{mL}$ 对二甲氨基苯甲醛溶液，加蒸馏水稀释至 $25~\mathrm{mL}$ 标线，混匀，放置 $20\mathrm{min}$ ，用 $10~\mathrm{cm}$ 光程的比色皿于 $458~\mathrm{nm}$ 波长处测定吸光度。以肼含量为横坐标，吸光度为纵坐标，绘制校准曲线。

# 6.2 样品测定

用无分度吸管吸取 $10\mathrm{mL}$ 待测水样于 $25\mathrm{mL}$ 具塞比色管中，检查水样 $\mathrm{pH}$ 值，如水样不呈中性需用稀盐酸或氢氧化钠溶液，将水样调至7左右，加入 $10\mathrm{mL}$ 对二甲氨基苯甲醛溶液，加蒸馏水稀释至 $25\mathrm{mL}$ 标线，混匀， $20\mathrm{min}$ 后用 $10\mathrm{cm}$ 光程的比色皿于 $458\mathrm{nm}$ 波长处测定吸光度，从校准曲线查出肼含量。

# 6.3 空白试验

取 $10\mathrm{mL}$ 蒸馏水代替水样，按水样相同的步骤进行空白测定。

# 7 结果的表示

试样中肼含量 $C(\mathrm{mg / L})$ 按下式计算：

$$
C = \frac {m}{V}
$$

式中： $m$ 从校准曲线上查得试样肼含量， $\mu \mathrm{g}$

$V$ ——分析试样体积， $\mathrm{mL}$ 。

# 8 精密度和准确度

8个实验室对 $0.100,0.500,0.800\mathrm{mg / L}$ 的标准溶液，按[6.2]分析步骤测定，结果如下：

# 8.1 重复性

三个浓度的实验室内相对标准偏差分别为 $3.7\% ,0.9\% ,0.6\%$

# 8.2 再现性

三个浓度的实验室间相对标准偏差分别为 $4.4\%$ 、 $1.3\%$ 、 $0.9\%$ 。

# 8.3 准确度

加标回收范围为 $96\% \sim 104\%$

# 附录A

（补充件）

# A1 水样处理

水样一般不需预处理，如水样中含有微小的固体颗粒，可先将水样用快速滤纸过滤，弃去开始滤出的数毫升水样后，滤液待用，或离心去除杂质。

# A2 排除干扰

水样中一般不含铬化合物，如个别水样有铬共存时，可将水样先加 $2\mathrm{mLHCl}[3.2]$ 于水样管中，再加 $1\%$ 的碘化钾 $0.4\mathrm{mL}$ 混匀，放置 $10\mathrm{min}$ 后，再按操作步骤[6.2]进行。如：水样中含 $\mathrm{NO}_2^-$ 含量高于 $1\mathrm{mg / L}$ 时可向水样中加入酚酞指示剂1滴，用 $20\%$ 氢氧化钠溶液调至水样成红色，再加入 $2\mathrm{mL}$ 叠氮化钠溶液[3.5]于通风柜中加 $\mathrm{HCl}[3.2]$ 至红色消失，然后加入 $1.3\mathrm{mLHCl}[3.1]$ 充分振荡使反应进行完全，再按操作步骤[6.2]进行。如：水样中存在一甲基肼和偏二甲基肼时，水样的酸度调到 $1\mathrm{N}$ 后，再按操作步骤[6.2]进行。

# A3 计算

如测定结果以水合肼计，将结果乘以1.56即可。因1.56份 $\mathrm{N}_2\mathrm{H}_4\cdot \mathrm{H}_2\mathrm{O}$ 中含有1份 $\mathrm{N}_2\mathrm{H}_4$
