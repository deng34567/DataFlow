# 中华人民共和国国家标准

# 大气降水中硝酸盐测定

# Determination of nitrate in the wet precipitation

GB 13580.8-92

# 1 主题内容与适用范围

1.1 本标准规定了测定大气降水中硝酸盐的紫外光度法和镉柱还原法。
1.2 本标准适用于大气降水样品中硝酸盐的测定。
1.3本标准分两篇：第一篇紫外光度法，第二篇镉柱还原法。紫外光度法最低检出浓度为 $0.2\mathrm{mg / L}$ ，测定范围为 $0.4\sim 10\mathrm{mg / L}$ 。镉柱还原法最低检出浓度为 $0.004\mathrm{mg / L}$ ，测定范围为 $0.01\sim 0.2\mathrm{mg / L}$ 。

# 2 引用标准

GB13580.2 大气降水样品的采集与保存

# 第一篇紫外光度法

# 3 原理

利用硝酸根离子 $220\mathrm{nm}$ 波长处的吸收而定量测定硝酸盐。溶解的有机物在 $220\mathrm{nm}$ 波长处也有吸收，而硝酸盐在 $275\mathrm{nm}$ 处没有吸收。因此，在 $275\mathrm{nm}$ 处作一次测量，以校正有机物对硝酸盐测定的影响。 $\mathrm{NO}_2^-$ 有干扰可用氨磺酸消除。

# 4 试剂

4.1 硝酸盐标准贮备液： $1000\mu \mathrm{g / mL}$ 。准确称取 $1.6306\mathrm{g}$ 硝酸盐 $\mathrm{(KNO_3}$ ，置干燥器中干燥 $24\mathrm{h}$ ），溶于水，并定容至 $1000\mathrm{mL}$ 。
4.2 硝酸盐标准使用液: $10 \mu \mathrm{g} / \mathrm{mL}$ 。准确吸取硝酸盐标准贮备液 $5.00 \mathrm{~mL}$ 于 $500 \mathrm{~mL}$ 容量瓶中, 用水稀释到标线, 摇匀。
4.3 氨磺酸水溶液： $10\mathrm{g/L}$ 。称取 $1\mathrm{g}$ 氨磺酸溶解在 $100\mathrm{mL}$ 水中，避光保存于冰箱内。
4.4盐酸溶液： $(1 + 11)$ 。取 $10\mathrm{mL}$ 盐酸加 $110\mathrm{mL}$ 水，混匀。

# 5 仪器

5.1 紫外分光光度计。
5.2 $25\mathrm{mL}$ 具塞比色管。

# 6 样品采集与保存

按GB13580.2执行。

# 7步骤

7.1 校准曲线的绘制：取 $25\mathrm{mL}$ 比色管7支，分别加入硝酸盐标准使用液(4.2)0,1.00,2.00,4.00,5.00,10.00,20.00mL。向各管中加 $0.1\mathrm{mL}$ 氨磺酸溶液(4.3),1mL盐酸溶液(4.4)，用水稀释至刻度，摇匀。用 $10\mathrm{mm}$ 石英吸收池，于 $220\mathrm{nm}$ 和 $275\mathrm{nm}$ 波长处，以水作参比，分别测量其吸光度。以 $\Delta A = A_{220} - 2A_{275}$ 对硝酸盐含量作图，绘制校准曲线。
7.2 样品测量：根据降水中硝酸盐含量，吸取 $10.0 \sim 20.0 \mathrm{~mL}$ 样品于 $25 \mathrm{~mL}$ 比色管中，按绘制校准曲线的步骤(7.1)进行操作，测定吸光度。从校准曲线上查出硝酸盐的含量。

# 8 分析结果的表述

降水中硝酸盐（按 $\mathrm{NO}_3^-$ 计）浓度以 $\mathrm{mg / L}$ 表示，按式(1)计算：

$$
C = \frac {M}{V} \quad \dots \
$$

式中： $C$ ——样品中硝酸盐浓度， $\mathrm{mg / L}$

$M$ ——从校准曲线上查得硝酸盐含量， $\mu \mathrm{g}$ ；

$V$ ——取样体积， $\mathrm{mL}$ 。

# 9 精密度和准确度

5个实验室对 $\mathrm{NO}_3^-$ 1.20 mg/L, $\mathrm{F}^{-}0.20\mathrm{mg / L}$ $\mathrm{Cl}^{-}1.00\mathrm{mg / L}$ $\mathrm{SO}_4^{2 - }6.00\mathrm{mg / L}$ $\mathrm{K}^{+}2.98\mathrm{mg / L}$ $\mathrm{Na}^{+}0.44\mathrm{mg / L}$ 的合成水样进行测定，测得结果 $\mathrm{NO}_3^-$ 的相对标准偏差为 $7.1\%$ ，相对误差为 $0.83\%$ 。

# 第二篇锯柱还原光度法

# 10 原理

在 $\mathrm{pH}8\sim 10$ 的条件下，硝酸盐经镉柱被还原成亚硝酸盐。亚硝酸盐与对氨基苯磺酸重氮化，再与N-(1-萘基)-乙二胺盐酸盐偶合，形成红色偶氮染料，于 $540\mathrm{nm}$ 波长处进行光度测量。经镉柱还原测得的是硝酸盐和亚硝酸盐的总量，减去不经过镉柱还原而直接测得的亚硝酸盐含量，即可得出硝酸盐含量。

# 11 试剂

11.1 硝酸盐标准贮备液: $1000 \mu \mathrm{g} / \mathrm{mL}$ 。准确称取 $1.6306 \mathrm{~g}$ 硝酸钾 $\left(\mathrm{KNO}_{3}\right.$ , 置干燥器中干燥 $24 \mathrm{~h}$ ), 溶于水, 并定容到 $1000 \mathrm{~mL}$ 。
11.2 硝酸盐标准使用液: $10 \mu \mathrm{g} / \mathrm{mL}$ 。准确吸取硝酸盐标准贮备液 $5.00 \mathrm{~mL}$ 于 $500 \mathrm{~mL}$ 容量瓶中, 用水稀释至刻度, 摇匀。
11.3 盐酸溶液： $(1 + 6)$ 。取 $10 \mathrm{~mL}$ 盐酸加到 $60 \mathrm{~mL}$ 水中，摇匀备用。
11.4 对氨基苯磺酸溶液： $10\mathrm{g/L}$ 。称取 $5\mathrm{g}$ 对氨基苯磺酸，溶于 $350\mathrm{mL}(1+6)$ 盐酸溶液（11.3）中，用水稀释到 $500\mathrm{mL}$ 水。此溶液可稳定数日。
11.5 N-(1-萘基)-乙二胺盐酸盐溶液： $1\mathrm{g/L}$ 。称取 $0.5\mathrm{gN}$ -（1-萘基）-乙二胺盐酸盐溶于 $500\mathrm{mL}$ 水中，贮存于棕色瓶中，在冰箱里保存。此试剂可稳定数周。如变成深棕色，应弃去重配。
11.6 氯化铵溶液： $20\mathrm{g} / 100\mathrm{mL}$ 。称取 $20\mathrm{g}$ 氯化铵溶解于 $100\mathrm{mL}$ 水中。
11.7 氯化铵溶液： $5\mathrm{g/L}$ 。称取 $0.5\mathrm{g}$ 氯化铵溶解于 $100\mathrm{mL}$ 水中。
11.8 锯汞齐柱的制备：玻璃管底部填小块玻璃棉，管内充满氯化铵溶液(11.7)，慢慢加入镉汞齐填料，使填料达到管的上口部（注意避免在填料中间引起气泡）。用 $300 \sim 500 \mathrm{~mL}$ 氯化铵溶液(11.7)，控制流速为每分钟 $6 \sim 8 \mathrm{~mL}$ ，使其流过新制备的镉汞齐柱。淋洗结束，管内充一定量的氯化铵溶液（如图所示）。

11.9 硝酸溶液： $1\% (V / V)$ 。取 $10\mathrm{mL}$ 硝酸于 $1000\mathrm{mL}$ 水中，摇匀。
11.10 氯化汞溶液： $10\mathrm{g/L}$ 。称 $1\mathrm{g}$ 氯化汞，溶于水并稀释到 $100\mathrm{mL}$ 。

11.11 盐酸溶液： $1\% (V / V)$ 。取 $1\mathrm{mL}$ 盐酸于 $100\mathrm{mL}$ 水中，摇匀。
11.12 镧汞齐柱的再生：倒出玻璃管中的镉汞齐填料，用水漂去悬浮状物，晾干。用硝酸溶液(11.9)充分洗涤，然后用蒸馏水洗到中性，晾干，按 $50\mathrm{g}$ 镉粒加 $50\mathrm{mL}$ 氯化汞的比例倒入氯化汞溶液(11.10)，搅拌几分钟后，倾去氯化汞溶液，用水洗 $3\sim 5$ 次，用硝酸溶液(11.9)洗一次(活化用)，立即用水清洗，再用盐酸溶液(11.11)洗一次，并用水洗至洗出液不含硝酸根离子为止，装柱。

# 12 仪器

12.1 分光光度计。
12.2 镯还原柱（有商品出售，若不便购买，可按本标准11.8方法制备）。

# 13 样品采集与保存

按GB13580.2执行。

# 14 步骤

14.1 校准曲线的绘制：取 $100\mathrm{mL}$ 容量瓶6个，分别加入硝酸盐标准使用液(11.2)0,0.50,1.00,1.50,2.00,2.50mL，分别加入2.0mL氯化铵(11.6)，用水稀释至刻度，摇匀。将第一份溶液倒入镉还原柱中，控制流速，用 $25\mathrm{mL}$ 具塞比色管接收还原液。弃去开始流出液 $30\mathrm{mL}$ 。接收随后流出的 $25.0\mathrm{mL}$ 还原液，待显色测定。当溶液流至填料表面时，倒入第二份溶液，按上述步骤继续进行第二份校准溶液的还原操作。以下类推。
14.2 显色：在上述 $25 \mathrm{~mL}$ 溶液中，分别加入 $0.5 \mathrm{~mL}$ 对氨基苯磺酸溶液(11.4)，摇匀后，放置 $2 \sim 8 \mathrm{~min}$ ，加 $1.0 \mathrm{mL} \mathrm{~N}$ -（1-萘基)-乙二胺盐酸盐溶液(11.5)，立即摇匀。至少放置 $10 \mathrm{~min}$ ，但不要超过 $2 \mathrm{~h}$ 。以水作参比，用 $10 \mathrm{~mm}$ 吸收池，在 $540 \mathrm{~nm}$ 波长处测量吸光度。绘制校准曲线。
14.3 样品测定：根据降水中硝酸盐的含量，吸取 $20.0\sim 50.0\mathrm{mL}$ 样品于 $100\mathrm{mL}$ 容量瓶中，按作标准曲线的步骤进行还原(14.1)和显色反应(14.2)，由测得的吸光度从校准曲线上查得硝酸盐的含量。

# 15 分析结果的表述

降水中硝酸盐（按 $\mathrm{NO}^{-}$ 计）浓度以 $\mathrm{mg / L}$ 表示，按式(2)计算：

$$
C = \frac {M}{V} \times 1 0 0 0 - 1. 2 9 2 C _ {\mathrm {N O} _ {2} ^ {-}}$$

式中： $C$ —样品中硝酸盐浓度， $\mathrm{mg / L}$

$M$ ——从校准曲线上查得硝酸盐含量， $\mu \mathrm{g}$ ；

$V$ ——取样体积， $\mathrm{mL}$ ；

$C_{\mathrm{NO}_2^-}$ ——水样中原有亚硝酸盐离子浓度1)， $\mathrm{mg / L}$

1.292——是 $\mathrm{NO}_2^-$ 折合成 $\mathrm{NO}_3^-$ 浓度的换算因子。

注：1）亚硝酸盐浓度按GB13580.7进行测定。

# 16 精密度和准确度

5个实验室对含 $\mathrm{NO}_3^-$ 为 $1.20\mathrm{mg / L},\mathrm{F}^{-}0.20\mathrm{mg / L},\mathrm{Cl}^{-}1.00\mathrm{mg / L},\mathrm{SO}_4^{2 - }6.00\mathrm{mg / L},\mathrm{K}^{+}2.98\mathrm{mg / }$ $\mathrm{L},\mathrm{Na}^{+}0.44\mathrm{mg / L}$ 的合成水样进行验证试验，测定结果 $\mathrm{NO}_3^-$ 的相对标准偏差为 $2.5\%$ ，相对误差为 $5.0\%$ 。
