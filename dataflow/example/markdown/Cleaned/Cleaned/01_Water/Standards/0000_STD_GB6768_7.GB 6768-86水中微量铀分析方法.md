本标准包括：1固体荧光法（磷酸三丁酯萃取－铀试剂Ⅲ反萃取或三正辛基氧膦萃取）；2液体激光荧光法；3分光光度法（磷酸三丁酯萃取－铀试剂Ⅲ反萃取）。前两种方法适用于天然水和排放废水中微量铀的测定，第三种方法适用于排放废水中微量铀的测定。

# 1 固体荧光法

测定范围为 $0.05 \sim 100 \mu \mathrm{g} / \mathrm{L}$ ；回收率大于 $90\%$ ；相对标准偏差优于 $\pm 20\%$ 。

# 1.1 原理

在硝酸介质中，铀酰离子与硫氰酸根生成的络合物被磷酸三丁酯定量萃取后，经铀试剂Ⅲ反萃取后，以固体荧光法测定铀或在硝酸介质中，铀酰离子被三正辛基氧膦定量萃取分离后，以固体荧光法测定铀。

# 1.2 试剂

除非另有说明，分析时均使用符合国家标准或专业标准的分析纯试剂和蒸馏水或同等纯度的水。

# 1.2.1 铀标准贮备溶液（ $1.00\mathrm{mg / ml}$ ）

将基准或光谱纯八氧化三铀于温度为 $850^{\circ}\mathrm{C}$ 马弗炉内灼烧 $0.5\mathrm{h}$ ，取出冷却。称取 $0.1179\mathrm{g}$ 于 $50\mathrm{ml}$ 烧杯内，用 $2\sim 3$ 滴水润湿后，加入 $5\mathrm{ml}$ 硝酸，于电热板上加热溶解并蒸至近干，然后用 $\mathrm{pH}2$ 的硝酸酸化水溶解，转入 $100\mathrm{ml}$ 容量瓶内，稀释至刻度；

# 1.2.2 铀标准溶液（临用时配制）

用 $\mathrm{pH}2$ 的硝酸酸化水将 $1.00\mathrm{mg / ml}$ 铀标准贮备溶液逐级稀释成不同浓度的铀标准溶液；

# 1.2.3 $20\%$ 磷酸三丁酯-二甲苯溶液

取一定体积的磷酸三丁酯 $\left[\left(\mathrm{C}_{4} \mathrm{H}_{9} \mathrm{O}\right)_{3} \mathrm{PO}\right]$ ，用等体积 $5 \%$ 碳酸钠 $\left(\mathrm{Na}_{2} \mathrm{CO}_{3} \cdot 10 \mathrm{H}_{2} \mathrm{O}\right)$ 溶液洗涤 $2 \sim 3$ 次，再用水洗至中性。取洗涤过的磷酸三丁酯与二甲苯 $\left[\mathrm{C}_{6} \mathrm{H}_{4} \left(\mathrm{CH}_{3}\right)_{2}\right]$ 按体积比 $1:4$ 混匀；

# 1.2.4 6M硫氰酸钾溶液

称取 $291\mathrm{g}$ 硫氰酸钾（KSCN），用水溶解，稀释至 $500\mathrm{ml}$

# 1.2.5 2M酒石酸溶液

称取 $150\mathrm{g}$ 酒石酸，用水溶解，稀释至 $500\mathrm{ml}$

# 1.2.6 $7.5\%$ 乙二胺四乙酸二钠溶液

称取 $7.5 \mathrm{~g}$ 乙二胺四乙酸二钠（简称EDTA二钠），加少量水，滴加氨水使之完全溶解后，用水稀释至 $100 \mathrm{ml}$ ；

# 1.2.7 $40\%$ 硝酸铵溶液

称取 $400\mathrm{g}$ 硝酸铵，用水溶解，稀释至 $1000\mathrm{ml}$

# 1.2.8 $5\%$ 碳酸钠溶液

称取 $50\mathrm{g}$ 无水碳酸钠，用水溶解，稀释至 $1000\mathrm{ml}$

# 1.2.9 $0.002\%$ 铀试剂Ⅲ溶液

称取 $0.1000 \mathrm{~g}$ 铀试剂Ⅲ，用 $\mathrm{pH} 2$ 的硝酸酸化水溶解后，转入 $100 \mathrm{ml}$ 容量瓶中，稀释至刻度，此溶

液浓度为 $0.1\%$ 。取 $10.0 \mathrm{ml}$ 该溶液于 $500 \mathrm{ml}$ 容量瓶中, 用 $\mathrm{pH} 2$ 的硝酸酸化水稀释至刻度。贮于棕色瓶内;

1.2.10 氟化钠，粉末状，优级纯；
1.2.11 混合熔剂，将98份氟化钠粉末与2份氟化锂粉末均匀混合；
1.2.12 0.1M三正辛基氧膦-环己烷溶液

称取 $19.33\mathrm{g}$ 三正辛基氧膦，溶于环己烷中，并稀释至 $500\mathrm{ml}$

# 1.2.13 混合掩蔽剂溶液

$6\% 1,2-$ 环己二胺四乙酸(简称CyDTA)溶液：称取 $30\mathrm{g}$ CyDTA，放入 $500\mathrm{ml}$ 烧杯中，加水 $400\mathrm{ml}$ ，滴加 $20\%$ 氢氧化钠溶液至其完全溶解。然后，用 $1\mathrm{mol} / \mathrm{L}$ 硝酸或氢氧化钠溶液调至中性，再用水稀释至 $500\mathrm{ml}$
$3\%$ 氟化钠溶液：称取 $15\mathrm{g}$ 氟化钠，用水溶解，稀释至 $500\mathrm{ml}$

取等体积的 $6\% 1,2-$ 环己二胺四乙酸溶液和 $3\%$ 氟化钠溶液混匀，即成混合掩蔽剂溶液。

# 1.3 仪器设备

1.3.1 光电荧光光度计：具有激发波长范围 $320 \sim 370 \mathrm{~nm}$ ，在 $530 \sim 570 \mathrm{~nm}$ 波长处测量发射的荧光，能够探测 $0.5 \mathrm{ng}$ 或更少的铀；

1.3.2 酒精喷灯或液化石油气灯：温度可达到 $1100^{\circ} \mathrm{C}$ ；
1.3.3 铂丝环：将直径为 $0.5\mathrm{mm}$ 的铂丝一端熔入玻璃棒，另一端绕成内径为 $3\mathrm{mm}$ 的圆环；
1.3.4 铂皿：内径 $10\mathrm{mm}$ ，深 $2\mathrm{mm}$ 。
1.3.5 氟化钠压片器；
1.3.6 马弗炉：温度 $1000^{\circ}\mathrm{C}$ 。

# 1.4 步骤

# 1.4.1 标准曲线绘制

将氟化钠或混合熔剂分别和1.2.2中所配制的铀标准溶液烧制熔珠或熔片，其操作方法见附录A中的A.3。烧制熔珠的条件为：火焰（氧化焰）温度 $980\sim 1050^{\circ}\mathrm{C}$ ，全熔后持续 $20\sim 30s$ ，退火 $5\sim 10s$ 冷却 $15\mathrm{min}$ 后，在光电荧光光度计上测定其荧光强度。用荧光强度与对应的铀浓度作图，绘制成四条不同量级的标准曲线。

# 1.4.2 样品分析

1.4.2.1 磷酸三丁酯萃取-铀试剂Ⅲ反萃取

取 $100 \mathrm{ml}$ （视铀含量而定）水样放入 $150 \mathrm{ml}$ 分液漏斗中，依次加入 $2 \mathrm{ml} 6 \mathrm{M}$ 硫氰酸钾溶液、 $2 \mathrm{ml} 2 \mathrm{M}$ 酒石酸溶液、 $6 \mathrm{ml} 17.5 \%$ 乙二胺四乙酸二钠溶液\*，每加入一种试剂均应摇匀。用 $1:1$ 硝酸或 $1:1$ 氢氧化铵溶液调节 $\mathrm{pH}$ 为 $2 \sim 3$ （精密 $\mathrm{pH}$ 试纸指示），加入 $5 \mathrm{ml} 20 \%$ 磷酸三丁酯-二甲苯溶液，充分振荡 $5 \mathrm{~min}$ ，静置分层后，弃去水相。用 $5 \mathrm{ml} 40 \%$ 硝酸铵溶液洗涤有机相二次，每次振荡 $2 \mathrm{~min}$ ，弃去水相。加 $2.00 \mathrm{ml} 0.002 \%$ 铀试剂Ⅲ溶液，振荡 $3 \mathrm{~min}$ ，静置分层后，将下层水相全部或定量分取部分与氟化钠或混合熔剂烧制熔珠或熔片。以下操作同1.4.1。

# 1.4.2.2 三正辛基氧膦萃取

取 $100\mathrm{ml}$ （视铀含量而定）水样放入 $150\mathrm{ml}$ 分液漏斗内，加入 $7\mathrm{ml}$ 硝酸和 $0.5\mathrm{ml}$ 混合掩蔽剂溶液，摇匀，加入 $1.00\mathrm{ml}0.1\mathrm{M}$ 三正辛基氧磷一环己烷溶液。充分振荡 $5\mathrm{min}$ ，静置分层后，弃去水相，定量分取有机相部分与氟化钠或混合熔剂烧制熔珠或熔片，以下操作同1.4.1。

# 1.5 结果的表示

铀含量按式（1）计算：

$$
c = \frac {(A - A _ {0}) V _ {1}}{V _ {0} \cdot V _ {2} \cdot R}$$

式中： $c$ ——水样中铀的浓度， $\mu \mathrm{g} / \mathrm{L}$

$A$ ——从标准曲线上查得样品熔珠或熔片的铀量， $\mu \mathrm{g}$ ；

$A_0$ ——方法本底铀量， $\mu \mathrm{g}$ ；

$V_{1}$ ——反萃取（或萃取）液总体积， $\mathrm{ml}$

$V_{2}$ ——用于测定的反萃取（或萃取）液体积， $\mathrm{ml}$

$V_{0}$ ——分析用水样体积，L；

$R$ ——全程回收率， $\%$ 。

# 2 液体激光荧光法

测定范围为 $0.02\sim 20\mu \mathrm{g} / \mathrm{L}$ ；相对标准偏差优于 $\pm 15\%$ ；全程回收率大于 $90\%$ 。

# 2.1 原理

直接向水样中加入荧光增强剂，使之与水样中铀酰离子生成一种简单的络合物，在激光（波长 $337\mathrm{nm}$ ）辐射激发下产生荧光。采用标准铀加入法定量地测定铀。

水样中常见干扰离子的含量为：锰（Ⅱ）小于 $1.5\mathrm{ppm}$ 、铁（Ⅲ）小于 $6\mathrm{ppm}$ 、铬（Ⅵ）小于 $6\mathrm{ppm}$ 、腐植酸小于 $3\mathrm{ppm}$ 。

# 2.2 试剂

除非另有说明，分析时均使用符合国家标准或专业标准的分析纯试剂和蒸馏水或同等纯度的水。

2.2.1 荧光增强剂：荧光增强倍数不小于100倍；
2.2.2 铀标准贮备溶液（1.00mg/ml）

同1.2.1;

2.2.3 $10.0\mu \mathrm{g / ml}$ 铀标准溶液

取 $1.00\mathrm{ml}$ 铀标准贮备溶液，用 $\mathrm{pH}2$ 的硝酸酸化水稀释至 $100\mathrm{ml}$

2.2.4 $0.500\mu \mathrm{g} / \mathrm{ml}$ 铀标准溶液

取 $5.00\mathrm{ml}10.0\mu \mathrm{g} / \mathrm{ml}$ 的铀标准溶液，用 $\mathrm{pH}2$ 的硝酸酸化水稀释至 $100\mathrm{ml}$

2.2.5 $0.100\mu \mathrm{g} / \mathrm{ml}$ 铀标准溶液

取 $1.00 \mathrm{ml} 10.0 \mu \mathrm{g} / \mathrm{ml}$ 的铀标准溶液, 用 $\mathrm{pH} 2$ 的硝酸酸化水稀释至 $100 \mathrm{ml}$ 。

# 2.3 仪器设备

2.3.1 铀分析仪：最低检出限 $0.05\mu \mathrm{g} / \mathrm{L}$

2.3.2 微量注射器： $50\mu \mathrm{l}$ （或 $0.1\mathrm{ml}$ 玻璃移液管）。

# 2.4 步骤

取 $5.00\mathrm{ml}\mathrm{pH}$ 为 $3.0\sim 11.0$ 的被测水样（如铀含量较高，可用水适当稀释）于石英比色皿内，调节补偿器旋钮直至表头指示为零（不为零时，可记录读数 $N_{0}$ ），向样品内加入 $0.5\mathrm{ml}$ 荧光增强剂，充分混匀，测定荧光强度为 $\sqrt{N_{1}}$ ，再向样品内加 $0.050\mathrm{ml}0.100\mu \mathrm{g} / \mathrm{ml}$ 铀标准溶液（高档测量应加入 $0.050\mathrm{ml}0.500\mu \mathrm{g} / \mathrm{ml}$ 铀标准溶液），充分混匀，测定荧光强度为 $\dot{N}_{2}$ 。

# 2.5 结果的表示

铀含量按式（2）计算：

$$
c = \frac {\left(N _ {1} - N _ {0}\right) c _ {1} V _ {1} K}{\left(N _ {2} - N _ {1}\right) V _ {0} R} \times 1 0 0 0 \dots \tag {2}
$$

式中： $c$ ——水样中铀的浓度， $\mu \mathrm{g} / \mathrm{L}$

$N_{0}$ ——样品未加荧光增强剂前的荧光强度；

$N_{1}$ ——加荧光增强剂后样品的荧光强度；

$N_{2}$ ——样品加标准铀后的荧光强度；

$c_{1}$ ——加入标准铀溶液的浓度， $\mu \mathrm{g} / \mathrm{ml}$ ；

$V_{1}$ ——加入标准铀溶液的体积，ml；

$V_{0}$ ——分析用水样的体积， $\mathrm{ml}$

$K$ ——水样稀释倍数；

$R$ ——全程回收率， $\%$ 。

# 3 分光光度法

测定范围为 $2\sim 100\mu \mathrm{g} / \mathrm{L}$ ；相对标准偏差优于 $\pm 10\%$ ；全程回收率大于 $90\%$ 。

# 3.1 原理

在硝酸介质中，铀酰离子与硫氰酸根生成的络合物被磷酸三丁酯定量萃取后，经铀试剂Ⅲ反萃取，以分光光度法测定铀。

# 3.4.1 工作曲线绘制

在7个 $150\mathrm{ml}$ 分液漏斗内，分别加入0.00、1.00、2.00、4.00、6.00、8.00、 $10.00\mu \mathrm{g}$ 铀标准溶液用水加至 $100\mathrm{ml}$ ，摇匀。依次加入 $2\mathrm{ml}6\mathrm{M}$ 硫氰酸钾溶液、 $2\mathrm{ml}2\mathrm{M}$ 酒石酸溶液、 $6\mathrm{ml}7.5\%$ 乙二胺四乙酸二钠溶液，每加入一种试剂均应摇匀。用 $1:1$ 硝酸或 $1:1$ 氢氧化铵溶液调节 $\mathrm{pH}$ 为 $2\sim 3$ （精密 $\mathrm{pH}$ 试纸指示)。加入 $10\mathrm{ml}20\%$ 磷酸三丁酯一煤油溶液，充分摇荡 $5\mathrm{min}$ ，静置分层后弃去水相。用 $5\mathrm{ml}40\%$ 硝酸铵溶液洗涤有机相二次，每次振荡 $2\mathrm{min}$ ，弃尽洗涤液，加入 $10.0\mathrm{ml}0.002\%$ 铀试剂Ⅲ溶液，振荡 $2\mathrm{min}$ ，静置分层后，将水相转入 $10\mathrm{ml}$ 离心试管内，离心 $3\mathrm{min}$ ，移入 $3\mathrm{cm}$ 比色皿内，于 $655\mathrm{nm}$ 波长处，以试剂空白作参比，测定光密度，并绘制工作曲线。

# 正确使用标准的说明

（参考件）

A.1 水样的处理及保存参见核设施水质监测分析取样规定。

A.2 所用酸在没有注明浓度时均指浓酸（浓硝酸密度1.42，浓盐酸密度1.19）。

A.3 固体荧光法中，烧制熔珠的方式可酌情选择下列方式中任意一种：

a. 将 $80 \pm 5 \mathrm{mg}$ 氟化钠用压片器压制成片，取 $0.050 \mathrm{ml}$ 含铀溶液滴在片上烧制熔珠；
b. 用 $80 \pm 5 \mathrm{mg}$ 氟化钠与含铀溶液蒸至近干后烧制熔珠；
c. 取 $0.1 \mathrm{ml}$ 含铀溶液滴入加有 $100 \mathrm{mg} 98 \% \mathrm{NaF} - 2 \% \mathrm{LiF}$ 混合熔剂的铂盘内，于烘箱内 $105^{\circ} \mathrm{C}$ 温度下烘干，转入马弗炉内在 $900^{\circ} \mathrm{C}$ 温度下熔融 $5 \mathrm{~min}$ ，取出冷至室温后，测量其荧光强度。不管采用何种方式，每种样品均需烧制3个熔珠或熔片，而且分析样品烧制熔珠或熔片的方式必须与标准曲线烧制熔珠或熔片的方式一致。

标准曲线必须进行直线回归处理，并定期（最多不得超过三个月）进行校正。在分析中，更换制作标准曲线时所用的任何试剂或光电荧光光度计进行调整，更换零件等，都必须重作标准曲线。

烧制熔珠的熔剂也可采用 $98\% \mathrm{NaF}$ 和 $2\% \mathrm{LiF}$ 的混合熔剂。

A.4 固体荧光法中, 当样品铀含量大于 $0.1 \mu \mathrm{g} / \mathrm{L}$ 时, 可用 $1.00 \mathrm{ml} 20 \%$ 磷酸三丁酯-二甲苯溶液萃取后, 直接取 $0.050 \mathrm{ml} 20 \%$ 磷酸三丁酯-二甲苯溶液滴在氟化钠片上烧制熔珠。
A.5 固体荧光法中，磷酸三丁酯的稀释剂可用煤油、甲苯、二甲苯中任意一种。
A.6 液体激光荧光法中，当加入荧光增强剂进行样品测量时，如样品产生沉淀，必须将被测样品经稀释或其他方法处理，不再产生沉淀后，方可进行测量。
A.7 液体激光荧光法中，2.5结果的表示是按简化公式计算铀含量，如果进行精确测量可用下式计算：

$$
c = \frac {N _ {1} \left(V _ {0} + V _ {2}\right) - N _ {0} V _ {0}}{N _ {2} \left(V _ {0} + V _ {1} + V _ {2}\right) - N _ {1} \left(V _ {0} + V _ {2}\right)} \cdot \frac {K c _ {1} V _ {1}}{V _ {0} R} \times 1 0 0 0
$$

式中： $V_{2}$ —加添的荧光增强剂溶液体积， $\mathrm{m}1$

其他符号的含义同式（2）。

A.8 固体荧光法和分光光度法中，用 $40\%$ 硝酸铵溶液洗涤有机相时，如含铁量太高，不易洗至无色，可采用 $40\%$ 硝酸铵加 $5\%$ 抗坏血酸的混合溶液洗涤。
