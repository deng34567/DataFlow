# 1 主题内容与适用范围

# 1.1 主题内容

本标准规定了测定水质中黑索今的分光光度法。

# 1.2 适用范围

1.2.1 本标准适用于弹药装药工业废水中黑索今含量的测定。
1.2.2 对 $50~\mathrm{mL}$ 试料，比色皿光程 $10\mathrm{mm}$ ，黑索今的最低检出浓度为 $0.05\mathrm{mg / L}$ ，测定范围为 $0.1\sim$ $10.0\mathrm{mg / L}$ 。
1.2.3 在被测溶液中如有环四甲撑四硝胺(奥托今)，对黑索今测定有干扰。

# 2 原理

黑索今在硫酸溶液中加热分解生成甲醛，甲醛与乙酰丙酮及氨作用，生成黄色的3,5-二乙酰基-1,4-二氢卢剔啶，在 $412\mathrm{nm}$ 处进行分光光度测定。

# 3 试剂

除另有说明，分析中所用试剂均应符合国家标准分析纯试剂。所用水均使用蒸馏水或同等纯度的水。

3.1 浓硫酸： $\rho = 1.84\mathrm{g / mL}$
3.2 乙酰丙酮溶液。
3.2.1 称取 $25\mathrm{g}$ 乙酸铵溶于 $80~\mathrm{mL}$ 水中，加冰乙酸 $(\rho = 1.05\mathrm{g / mL})3\mathrm{mL}$ ，以水稀释至 $100~\mathrm{mL}$ 。此溶液为 $\mathrm{pH}6$ 缓冲溶液。
3.2.2 吸取 $0.2\mathrm{mL}$ 乙酰丙酮试剂加入上述(3.2.1)pH6缓冲溶液中，摇匀。低温下保存，有效期为一个月。
3.3 硝酸溶液： $50\% (V / V)$ 。

量取 $10\mathrm{mL}$ 硝酸 $(\rho = 1.39\mathrm{g / mL})$ 与 $10\mathrm{mL}$ 水混合。

3.4 硫酸亚铁铵 $\left[\mathrm{FeSO}_{4} \cdot \left(\mathrm{NH}_{4}\right)_{2} \cdot \mathrm{SO}_{4} \cdot 6 \mathrm{H}_{2} \mathrm{O}\right]$ 。
3.5 硫酸亚铁铵溶液： $10\mathrm{g / L}$

称取 $1.3\mathrm{g}$ 六合水硫酸亚铁铵(3.4)溶于 $100\mathrm{mL}$ 水中。

3.6 黑索今标准溶液。

准确称取精制黑索今(经丙酮重结晶二次) $0.0400\mathrm{g}$ 于 $1000\mathrm{mL}$ 烧杯中，以约 $10\mathrm{mL}$ 丙酮溶解后，缓缓加入预先加热的水约 $800\mathrm{mL}$ ，置沸水浴中加热。使其溶解后，继续加热至无丙酮气味。取出冷却至

室温，移入 $1000\mathrm{mL}$ 棕色容量瓶中，以水稀释至刻线，摇匀。此溶液为贮备液，浓度为 $40\mathrm{mg / L}$ 。低温 $(2\sim 5^{\circ}C)$ 保存，保存期5d。

用稀释法配制 $20.0\mathrm{mg / L}$ 或 $2.00\mathrm{mg / L}$ 黑索今标准使用液，临用时现配。

# 5 采样和样品

# 5.1 采样

样品应采集于棕色玻璃瓶中，低温 $(2\sim 5^{\circ}C)$ 保存，有效期5d。

# 5.2 样品

样品中含 $\mathrm{Cr^{6+}}$ 、甲醛时，按下述情况进行预处理。

5.2.1 样品中含有 $\mathrm{Cr}^{6+}$ , 可加入适量硫酸亚铁铵溶液(3.5)。 $\mathrm{Cr}^{6+}$ 浓度小于 $10 \mathrm{mg} / \mathrm{L}$ 时, 加入量为 $1 \mathrm{~mL}$ 。
5.2.2 样品中含有甲醛时，量取 $200\mathrm{mL}$ 试样于 $500\mathrm{mL}$ 烧杯中，以硝酸溶液(3.3)调 $\mathrm{pH}$ 值为4。然后在电炉上煮沸并不断搅拌，当水样蒸发至原体积一半时，停止加热。待溶液冷却至室温后，以水定容为原体积 $200\mathrm{mL}$ ，待测定。

# 6 分析步骤

# 6.1 校准曲线的绘制

# 6.1.1 高浓度校准曲线

吸取 $20.0\mathrm{mg / L}$ 浓度的黑索今标准使用液0,2.50,5.00,10.00,15.00,20.00,25.00mL,分别移入 $250~\mathrm{mL}$ 全玻璃蒸馏瓶中。然后加水至 $45.00~\mathrm{mL}$ ，加入浓硫酸(3.1) $10.00~\mathrm{mL}$ ，摇匀。加数粒玻璃珠，防爆沸。

吸取乙酰丙酮溶液(3.2.2) $10.00\mathrm{mL}$ ，移入 $50~\mathrm{mL}$ 具塞比色管中，用以接受馏出液。

按蒸馏装置(4.6)连接各部分，并检查使其密合。冷凝器的出口插入接受管(50 mL 具塞比色管)中。

打开冷凝水，加热蒸馏，蒸馏速度为 $2\sim 3\mathrm{mL / min}$ 。当馏出液和乙酰丙酮溶液总体积至 $50~\mathrm{mL}$ 时，取下接受管，停止蒸馏。

将馏出液与乙酰丙酮溶液摇匀，于沸水浴中加热 $5\mathrm{min}$ 。取下冷却后，用 $10\mathrm{mm}$ 比色皿，于 $412\mathrm{nm}$ 波长处，以试剂水溶液作参比测量吸光度。以吸光度为纵坐标，对应的黑索今含量 $(\mathrm{mg})$ 为横坐标绘制校准曲线。

# 6.1.2 低浓度校准曲线

吸取 $2.00\mathrm{mg / L}$ 浓度的黑索今标准使用液0,2.50,5.00,10.00,15.00,20.00,25.00mL分别置于 $250~\mathrm{mL}$ 全玻璃蒸馏瓶中。加水分别至 $45.00~\mathrm{mL}$ ，除比色皿用 $30~\mathrm{mm}$ 外，其余步骤同高浓度校准曲线。

# 6.2 样品测定

# 6.2.1 非生化法处理废水样的测定

取试样 $45.00\mathrm{mL}$ 置于 $250\mathrm{mL}$ 全玻璃蒸馏瓶中（如果试样中黑索今浓度高于 $10\mathrm{mg / L}$ 时，可取适量试样，加水至 $45.00\mathrm{mL}$ ）。以下按6.1.1步骤操作。

# 6.2.2 生化法处理废水样的测定

用量筒取试样 $200\mathrm{mL}$ 于 $500\mathrm{mL}$ 烧杯中，然后加入六合水硫酸亚铁按(3.4)1g，用玻璃棒搅拌均匀。放置 $10\mathrm{min}$ 后用中速定量滤纸过滤，滤液承接于 $500\mathrm{mL}$ 三角瓶内。待测定。

取二个 $250~\mathrm{mL}$ 蒸馏瓶并编号为A、B，然后分别加入上述滤液 $45.00~\mathrm{mL}$ （如试样浓度大于 $10~\mathrm{mg / L}$ 时，可取适量滤液，加水至 $45.00~\mathrm{mL}$ ）。向A、B蒸馏瓶中加入硝酸(3.3)1～2滴，使其 $\mathrm{pH}$ 值为4。A蒸馏瓶不加浓硫酸；B蒸馏瓶加入浓硫酸(3.1)10.00mL，摇匀。其余步骤按6.1.1操作。

# 7 结果的表示

# 7.1 非生化法处理废水样

黑索今含量按式(1)计算：

$$
c _ {1} = \frac {m}{V _ {1}} \times 1 0 0 0 \tag {1}
$$

式中： $c_{1}$ ——黑索今含量， $\mathrm{mg / L}$

$m$ ——校准曲线查得黑索今含量， $\mathrm{mg}$ ；

$V_{1}$ ——试样的体积， $\mathrm{mL}$ 。

# 7.2 生化法处理废水样

黑索今含量按式(2)计算：

$$
c _ {2} = \frac {m _ {2} - m _ {1}}{V _ {2}} \times 1 0 0 0 \quad \dots \tag {2}
$$

式中： $c_{2}$ ——黑索今含量， $\mathrm{mg / L}$

$m_{1}$ ——A蒸馏瓶试样在校准曲线查得黑索今含量， $\mathrm{mg}$

$m_{2}$ ——B蒸馏瓶试样在校准曲线查得黑索今含量， $\mathrm{mg}$

$V_{2}$ ——试样的体积， $\mathrm{mL}$ 。

# 8 精密度和准确度

5个实验室测定黑索今浓度为 $2.00\mathrm{mg / L}$ 的试样。

8.1 重复性

实验室内相对标准偏差为 $3.1\%$

8.2 再现性

实验室间相对标准偏差为 $2.3\%$

8.3 准确度

加标回收率为 $94\% \sim 105\%$
