# 中华人民共和国国家标准

# 大气降水中氟、氯、亚硝酸盐、硝酸盐、

# 硫酸盐的测定离子色谱法

GB 13580.5-92

Determination of fluoride, chloride, ntrite nitrate,

sulfate in the wet precipitation-Ion chromatography

# 1 主题内容与适用范围

1.1 本标准规定了测定大气降水中氟、氯、亚硝酸盐、硝酸盐、硫酸盐的离子色谱法。
1.2本标准适用于大气降水样品中氟、氯、亚硝酸盐、硝酸盐、硫酸盐的测定。
1.3本标准进样品量为： $50~\mu \mathrm{L}$ ，最低检出浓度分别为： $\mathrm{F}^{-}0.03\mathrm{mg / L},\mathrm{Cl}^{-}0.03\mathrm{mg / L},\mathrm{NO}_2^-$ $0.05\mathrm{mg / L},\mathrm{NO}_3^{-}0.10\mathrm{mg / L},\mathrm{SO}_4^{2 - }0.10\mathrm{mg / L}$

# 2 引用标准

GB13580.2 大气降水样品的采集与保存

# 3 原理

离子色谱法测定阴离子是利用离子交换原理进行分离，由抑制柱扣除淋洗液背景电导，然后利用电导检测器进行测定。根据混合标准溶液中各阴离子出峰的保留时间以及峰高可进行定性和定量测定各种阴离子。

# 4 试剂

所用的水均应为电导率小于 $1\mu \mathrm{s} / \mathrm{cm}$ 的去离子水。

4.1 氟化物标准溶液： $1000\mu \mathrm{g} / \mathrm{mL}$ 。准确称取 $2.2100\mathrm{g}$ 氟化钠(NaF, $105\mathrm{C}$ 干燥 $2\mathrm{h}$ ），溶于水，并定容至 $1000\mathrm{mL}$ 。
4.2 氯化物标准溶液: $1000 \mu \mathrm{g} / \mathrm{mL}$ 。准确称取 $1.6480 \mathrm{~g}$ 氯化钠 $(\mathrm{NaCl}, 105^{\circ} \mathrm{C}$ 烘 $2 \mathrm{~h})$ , 溶于水, 并定容至 $1000 \mathrm{~mL}$ 。
4.3 亚硝酸盐标准溶液： $1000\mu \mathrm{g} / \mathrm{mL}$ 。准确称取 $1.5000\mathrm{g}$ 亚硝酸钠 $\mathrm{(NaNO_2}$ ，干燥器中干燥 $24\mathrm{h}$ ），溶于水，并定容至 $1000\mathrm{mL}$ 。
4.4硝酸盐标准溶液： $1000\mu \mathrm{g / mL}$ 。准确称取 $1.6305\mathrm{g}$ 硝酸钾 $\mathrm{(KNO_3}$ ，干燥器中干燥 $24\mathrm{h})$ ，溶于水，并定容至 $1000~\mathrm{mL}$ 。
4.5 硫酸盐标准溶液： $1000\mu \mathrm{g / mL}$ 。准确称取 $1.8140\mathrm{g}$ 硫酸钾 $\mathrm{(K_2SO_4,105°C}$ 烘 $2\mathrm{h})$ ，溶于水，并定容至 $1000\mathrm{mL}$ 。
4.6淋洗液 $(0.003\mathrm{mol} / \mathrm{L}$ 碳酸氢钠-0.0025mol/L碳酸钠)：称取 $2.5203\mathrm{g}$ 碳酸氢钠 $\mathrm{(NaHCO_3)}$ 和 $2.6498\mathrm{g}$ 无水碳酸钠 $\mathrm{(Na_2CO_3)}$ 溶于水，并定容至 $1000\mathrm{mL}$ 。装入专用的塑料桶中或按有关仪器说明书配制。淋洗液应经 $0.45\mu \mathrm{m}$ 滤膜过滤后使用。
4.7 硫酸溶液： $0.0125\mathrm{mol} / \mathrm{L}$ 。量取浓硫酸 $1.4\mathrm{mL}$ ，在不断搅拌下，慢慢倒入水中，稀释至 $2000\mathrm{mL}$

# 5 仪器

# 5.1 离子色谱仪。

5.2 双笔记录仪（或积分仪），或与离子色谱仪配套的记录仪。
5.3 前置柱。
5.4 分离柱。
5.5 抑制柱或国产电化学抑制器。
5.6 仪器工作条件：可根据不同型号仪器说明书选择，以下仅为一例供参考。

a. 主机量程: $10 \sim 30 \mu \mathrm{s}$ ;
b. 泵流速: $2.0 \mathrm{~mL} / \mathrm{min}$ ;
c. 分离柱温度: $25^{\circ} \mathrm{C}$ ;
d. 进样体积： $50~\mu \mathrm{L}$

# 6样品采集与保存

按GB13580.2执行。

# 7 步骤

# 7.1 校准曲线的绘制

根据降水样品中各离子的相对含量，配制五种离子的混合标准系列。按前述仪器工作条件(3.3)开动仪器，待基线稳定后，注入标准系列样品。记录仪按一定顺序记录各离子的峰高，可根据溶液中离子的浓度和相应的峰高绘制校准曲线。

# 7.2 样品测定

按绘制校准曲线的程序(6.1)测定样品峰高，由样品峰高从校准曲线上查得相应浓度。

# 8 分析结果的表述

降水中氟化物（按 $\mathbf{F}^{-}$ 计），氯化物（按 $\mathrm{Cl}^{-}$ 计），亚硝酸盐（按 $\mathrm{NO}_2^-$ 计），硝酸盐（按 $\mathrm{NO}_3^-$ 计），硫酸盐（按 $\mathrm{SO}_4^{2-}$ 计）的浓度以 $\mathrm{mg/L}$ 表示。可按下式计算：

$$
C = M \cdot D
$$

式中： $C$ —样品中待测离子含量， $\mathrm{mg / L}$

$M$ ——由校准曲线上查得样品中待测离子的含量， $\mathrm{mg / L}$ ；

$D$ ——样品稀释倍数。

# 9 精密度和准确度

5个实验室对含 $\mathrm{F}^{-}0.20\mathrm{mg / L},\mathrm{SO}_4^{2 - }6.00\mathrm{mg / L},\mathrm{Cl}^{-}1.00\mathrm{mg / L},\mathrm{NO}_3^{-}1.20\mathrm{mg / L},$ $\mathrm{K}^{+}2.98\mathrm{mg / L},\mathrm{Na}^{+}0.44\mathrm{mg / L}$ 的合成水样及含 $\mathrm{NO}_2^{-}0.25\mathrm{mg / L},\mathrm{Na}^{+}0.13\mathrm{mg / L}$ 的合成水样进行验证实验。测定结果是： $\mathrm{F}^{-}$ 的相对标准偏差为 $1.1\%$ ，相对误差为 $-0.5\% ;\mathrm{Cl}^{-}$ 的相对标准偏差为 $2.1\%$ 相对误差为 $-0.2\% ;\mathrm{NO}_3^{-}$ 的相对标准偏差为 $1.7\%$ ，相对误差为 $-0.83\% ;\mathrm{SO}_4^{2 - }$ 的相对标准偏差为 $0.8\%$ ，相对误差为 $-1.7\% ;\mathrm{NO}_2^{-}$ 的相对标准偏差为 $3.2\%$ ，相对误差为 $1.3\%$ 。
