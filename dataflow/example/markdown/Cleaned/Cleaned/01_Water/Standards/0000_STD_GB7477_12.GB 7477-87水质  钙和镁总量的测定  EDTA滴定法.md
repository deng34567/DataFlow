# 中华人民共和国国家标准

# 水质钙和镁总量的测定EDTA滴定法

UDC 614.777:543

.24:546.41

:546.46

GB 7477-87

Water quality-Determination of the sum of calcium and magnesium-EDTA titrimetric method

本标准等效采用ISO6059—1984《水质钙与镁总量的测定EDTA滴定法》。

# 1 适用范围

本标准规定用EDTA滴定法测定地下水和地面水中钙和镁的总量。本方法不适用于含盐量高的水，诸如海水。本方法测定的最低浓度为 $0.05\mathrm{mmol / L}$ 。

# 2 原理

在pH10的条件下，用EDTA溶液络合滴定钙和镁离子。铬黑T作指示剂，与钙和镁生成紫红或紫色溶液。滴定中，游离的钙和镁离子首先与EDTA反应，跟指示剂络合的钙和镁离子随后与EDTA反应，到达终点时溶液的颜色由紫变为天蓝色。

# 3 试剂

分析中只使用公认的分析纯试剂和蒸馏水，或纯度与之相当的水。

# 3.1 缓冲溶液（pH10）。

3.1.1 称取 $1.25\mathrm{g}$ EDTA 二钠镁 $\left(\mathrm{C}_{10}\mathrm{H}_{12}\mathrm{N}_{2}\mathrm{O}_{8}\mathrm{Na}_{2}\mathrm{Mg}\right)$ 和 $16.9\mathrm{g}$ 氯化铵 $\left(\mathrm{NH}_{4}\mathrm{Cl}\right)$ 溶于 $143\mathrm{ml}$ 浓的氨水 $\left(\mathrm{NH}_{3} \cdot \mathrm{H}_{2}\mathrm{O}\right)$ 中，用水稀释至 $250\mathrm{ml}$ 。因各地试剂质量有出入，配好的溶液应按3.1.2方法进行检查和调整。
3.1.2 如无EDTA二钠镁，可先将16.9g氯化铵溶于143ml氨水。另取0.78g硫酸镁（ $\mathrm{MgSO_4\cdot 7H_2O}$ ）和1.179gEDTA二钠二水合物（ $\mathrm{C_{10}H_{14}N_2O_8Na_2\cdot 2H_2O}$ ）溶于50ml水，加入2ml配好的氯化铵、氨水溶液和0.2g左右铬黑T指示剂干粉（3.4）。此时溶液应显紫红色，如出现天蓝色，应再加入极少量硫酸镁使变为紫红色。逐滴加入EDTA二钠溶液（3.2）直至溶液由紫红转变为天蓝色为止（切勿过量）。将两溶液合并，加蒸馏水定容至250ml。如果合并后，溶液又转为紫色，在计算结果时应减去试剂空白。

# 3.2 EDTA二钠标准溶液： $\approx 10\mathrm{mmol / L}$ 。

# 3.2.1 制备

将一份EDTA二钠二水合物在 $80^{\circ}\mathrm{C}$ 干燥 $2\mathrm{h}$ ，放入干燥器中冷至室温，称取 $3.725\mathrm{g}$ 溶于水，在容量瓶中定容至 $1000\mathrm{ml}$ ，盛放在聚乙烯瓶中，定期校对其浓度。

# 3.2.2 标定

按第6章的操作方法，用钙标准溶液(3.3)标定EDTA二钠溶液(3.2.1)。取 $20.0\mathrm{ml}$ 钙标准溶液(3.3）稀释至 $50\mathrm{ml}$ 。

# 3.2.3 浓度计算

EDTA二钠溶液的浓度 $c_{1}$ （ $\mathrm{mmol / L}$ ）用式（1）计算：

$$
c _ {1} = \frac {c _ {2} V _ {2}}{V _ {1}} \tag {1}
$$

式中： $c_{2}$ ——钙标准溶液（3.3）的浓度， $\mathrm{mmol / L}$ ；

$V_{2}$ ——钙标准溶液的体积， $\mathrm{ml}$ ；

$V_{1}$ ——标定中消耗的EDTA二钠溶液体积， $\mathbf{m}1$ 。

# 3.3 钙标准溶液： $10\mathrm{mmol / L}$ 。

将一份碳酸钙（ $\mathrm{CaCO}_3$ ）在 $150^{\circ}\mathrm{C}$ 干燥 $2\mathrm{h}$ ，取出放在干燥器中冷至室温，称取 $1.001\mathrm{g}$ 于 $500\mathrm{ml}$ 锥形瓶中，用水润湿。逐滴加入 $4\mathrm{mol} / \mathrm{L}$ 盐酸至碳酸钙全部溶解，避免滴入过量酸。加 $200\mathrm{ml}$ 水，煮沸数分钟赶除二氧化碳，冷至室温，加入数滴甲基红指示剂溶液（ $0.1\mathrm{g}$ 溶于 $100\mathrm{ml}60\%$ 乙醇），逐滴加入 $3\mathrm{mol} / \mathrm{L}$ 氨水至变为橙色，在容量瓶中定容至 $1000\mathrm{ml}$ 。此溶液 $1.00\mathrm{ml}$ 含 $0.4008\mathrm{mg}$ （ $0.01\mathrm{mmol}$ ）钙。

# 3.4 铬黑T指示剂。

将 $0.5\mathrm{g}$ 铬黑T $\left[\mathrm{HOC}_{10}\mathrm{H}_{6}\mathrm{N}:\mathrm{N}_{10}\mathrm{H}_{4}(\mathrm{OH})(\mathrm{NO}_{2})\mathrm{SO}_{3}\mathrm{Na}\right.$ ，又名媒染黑11，学名：1-（1-羟基-2-萘基偶氮）-6-硝基-2-萘酚-4-磺酸钠盐，sodium salt of 1-(1-hydroxy-2-naphthylazo)-6-nitro-2-naphthol-4-sulfonic acid]溶于 $100\mathrm{ml}$ 三乙醇胺 $\left[\mathrm{N}(\mathrm{CH}_{2}\mathrm{CH}_{2}\mathrm{OH})_{3}\right]$ ，可最多用 $25\mathrm{ml}$ 乙醇代替三乙醇胺以减少溶液的粘性，盛放在棕色瓶中。或者，配成铬黑T指示剂干粉，称取 $0.5\mathrm{g}$ 铬黑T与 $100\mathrm{g}$ 氯化钠（NaCl，GB1266-77）充分混合，研磨后通过 $40\sim 50$ 目，盛放在棕色瓶中，紧塞。

# 3.5 氢氧化钠：2 mol/L 溶液。

将 $8\mathrm{g}$ 氢氧化钠（ $\mathrm{NaOH}$ ）溶于 $100\mathrm{ml}$ 新鲜蒸馏水中。盛放在聚乙烯瓶中，避免空气中二氧化碳的污染。

# 3.6 氰化钠（NaCN）。

注意：氰化钠是剧毒品，取用和处置时必须十分谨慎小心，采取必要的防护。含氰化钠的溶液不可酸化。

# 3.7 三乙醇胺 $\left[\mathbf{N}\left(\mathbf{CH}_{2} \mathbf{CH}_{2} \mathbf{OH}\right)_{3}\right]$ 。

# 4 仪器

常用的实验室仪器及：

滴定管： $50\mathrm{ml}$ ，分刻度至 $0.10\mathrm{ml}$ 。

# 5 采样和样品保存

采集水样可用硬质玻璃瓶（或聚乙烯容器），采样前先将瓶洗净。采样时用水冲洗3次，再采集于瓶中。

采集自来水及有抽水设备的井水时，应先放水数分钟，使积留在水管中的杂质流出，然后将水样收集于瓶中。采集无抽水设备的井水或江、河、湖等地面水时，可将采样设备浸入水中，使采样瓶口位于水面下 $20\sim 30\mathrm{cm}$ ，然后拉开瓶塞，使水进入瓶中。

水样采集后（尽快送往实验室），应于24h内完成测定。否则，每升水样中应加 $2\mathrm{ml}$ 浓硝酸作保存剂（使 $\mathsf{pH}$ 降至1.5左右）。

# 6 步骤

# 6.1 试样的制备

一般样品不需预处理。如样品中存在大量微小颗粒物，需在采样后尽快用 $0.45\mu \mathrm{m}$ 孔径滤器过滤。样品经过滤，可能有少量钙和镁被滤除。

试样中钙和镁总量超出 $3.6 \mathrm{mmol} / \mathrm{L}$ 时，应稀释至低于此浓度，记录稀释因子 $F$ 。

如试样经过酸化保存，可用计算量的氢氧化钠溶液（3.5）中和。计算结果时，应把样品或试样

由于加酸或碱的稀释考虑在内。

# 6.2 测定

用移液管吸取 $50.0\mathrm{ml}$ 试样于 $250\mathrm{ml}$ 锥形瓶中，加 $4\mathrm{ml}$ 缓冲溶液（3.1）和3滴铬黑T指示剂溶液或 $50\sim 100\mathrm{mg}$ 指示剂干粉（3.4），此时溶液应呈紫红或紫色，其 $\mathsf{pH}$ 值应为 $10.0\pm 0.1$ 。为防止产生沉淀，应立即在不断振摇下，自滴定管加入EDTA二钠溶液（3.2），开始滴定时速度宜稍快，接近终点时应稍慢，并充分振摇。最好每滴间隔 $2\sim 3\mathrm{s}$ ，溶液的颜色由紫红或紫色逐渐转为蓝色，在最后一点紫的色调消失，刚出现天蓝色时即为终点，整个滴定过程应在 $5\mathrm{min}$ 内完成。记录消耗EDTA二钠溶液体积的毫升数。

如试样含铁离子为 $30\mathrm{mg / L}$ 或以下，在临滴定前加入 $250\mathrm{mg}$ 氰化钠（3.6），或数毫升三乙醇胺（3.7）掩蔽。氰化物使锌、铜、钴的干扰减至最小。加氰化物前必须保证溶液呈碱性。

试样如含正磷酸盐和碳酸盐，在滴定的pH条件下，可能使钙生成沉淀，一些有机物可能干扰测定。

如上述干扰未能消除，或存在铝、钡、铅、锰等离子干扰时，需改用原子吸收法测定。

# 7 结果的表示

钙和镁总量 $c(\mathrm{mmol} / \mathrm{L})$ 用式（2）计算：

$$
c = \frac {c _ {1} V _ {1}}{V _ {0}} \tag {2}
$$

式中： $c_{1}$ ——EDTA二钠溶液浓度， $\mathrm{mmol / L}$ ；

$V_{1}$ ——滴定中消耗EDTA二钠溶液的体积，ml；

$V_{0}$ ——试样体积， $\mathfrak{m}1$ 。

如试样经过稀释，采用稀释因子 $F$ 修正计算。

关于硬度的计算，见附录A。1mmol/L的钙镁总量相当于100.1mg/L以CaCO $_3$ 表示的硬度。

# 8 精度

本方法的重复性为 $\pm 0.04\mathrm{mmol / L}$ ，约相当于 $\pm 2$ 滴EDTA二钠溶液。

# 附录A

# 水硬度的概念

（参考件）

硬度，不同国家有不同的定义概念，如总硬度、碳酸盐硬度、非碳酸盐硬度。

# A.1 定义

A.1.1 总硬度——钙和镁的总浓度。
A.1.2 碳酸盐硬度——总硬度的一部分，相当于跟水中碳酸盐及重碳酸盐结合的钙和镁所形成的硬度。
A.1.3 非碳酸盐硬度——总硬度的另一部分，当水中钙和镁含量超出与它们结合的碳酸盐和重碳酸盐含量时，多余的钙和镁就跟水中氯化物、硫酸盐、硝酸盐结成非碳酸盐硬度。

# A.2 硬度的表示方法

A.2.1 德国硬度——1德国硬度相当于CaO含量为 $10\mathrm{mg / L}$ 或为 $0.178\mathrm{mmol / L}$ 。
A.2.2 英国硬度——1 英国硬度相当于 $\mathrm{CaCO}_{3}$ 含量为 1 格令/英加仑，或为 $0.143 \mathrm{~mmol} / \mathrm{L}$ 。
A.2.3 法国硬度——1法国硬度相当于 $\mathrm{CaCO}_3$ 含量为 $10\mathrm{mg / L}$ 或为 $0.1\mathrm{mmol / L}$ 。
A.2.4 美国硬度——1 美国硬度相当于 $\mathrm{CaCO}_3$ 含量为 $1\mathrm{mg / L}$ 或为 $0.01\mathrm{mmol / L}$ 。
