GB 9803-88

Water quality--Determination of pentachlorophenol

by safranine-T spectrophotometric method

# 1 适用范围

本标准适用于含五氯酚工业废水以及被五氯酚污染的水体中五氯酚的测定。其测定浓度范围为 $0.01 \sim 0.5 \mathrm{mg} / \mathrm{L}$ ；挥发酚类化合物（以苯酚计）低于 $150 \mathrm{mg} / \mathrm{L}$ 对测定无干扰。最低检出浓度为 $0.01 \mathrm{mg} / \mathrm{L}$ 。

# 2 原理

用蒸馏法蒸馏出五氯酚，从而与高沸点酚类和其他色素等干扰物分离。被蒸馏出的五氯酚在硼酸盐缓冲液 $(\mathrm{pH}9.3)$ 存在下，可与藏红T生成紫红色络合物，用乙酸异戊酯萃取，置于波长 $535\mathrm{nm}$ 下，测定吸光度。

# 3 试剂

本标准所用试剂除另有说明外，均为分析纯试剂。所用的水指蒸馏水或具有同等纯度的去离子水。

3.1 五氯酚 $(\mathrm{C_6Cl_5OH})$ ，化学纯。

3.2 藏红T(Safranine-T)，编号82555，染料。

3.3 乙酸异戊酯 $(\mathrm{CH}_3\mathrm{COOC}_5\mathrm{H}_{11})$

3.4 四硼酸钠 $(\mathrm{Na}_2\mathrm{B}_1\mathrm{O}_7\cdot 10\mathrm{H}_2\mathrm{O})$ 。

3.5 $10\% (m / l)$ 氢氧化钠溶液。

3.6 无水硫酸钠。

3.7 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)\cdot \rho = 1.70\mathrm{g / mL}(85.0\%)$

3.8 $10\% (m / l^{-})$ 硫酸铜溶液。

3.9 $0.1\% (m / \Gamma)$ 甲基橙指示剂溶液(methylorange)。

3.10 五氯酚贮备液：准确称取五氯酚(3.1)0.5000g于小烧杯中，逐滴加入氢氧化钠溶液(3.5)10mL，边加边搅拌，然后，加入少量水，使之全部溶解后，移入500mL棕色容量瓶中，并用水稀释至标线其浓度为1.000mg/L，置冰箱内，避光保存。

# 3.11 五氯酚使用溶液

取五氯酚贮备液(3.10) $1.00\mathrm{mL}$ 于 $100\mathrm{mL}$ 棕色容量瓶中，加水至标线，其浓度为 $0.010\mathrm{mg / mL}$ ，临用时配制。

# 3.12 藏红T硼酸盐缓冲液

称取 $0.1\mathrm{g}$ 藏红T（3.2）， $19.1\mathrm{g}$ 四硼酸钠(3.4)共溶于水中，并稍微加热至全溶，稀释至 $1000\mathrm{mL}$ 每次用 $30\sim 50\mathrm{mL}$ 乙酸异戊酯(3.3)反复萃取，弃去有机相，直至乙酸异戊酯萃取液的透光率约为 $93\%$ 为

止[以乙酸异戊酯（3.3)为参比液]，将此液贮于冷暗处。

# 4 仪器

4.1 $500~\mathrm{mL}$ 全玻璃磨口蒸馏器。
4.2 $250~\mathrm{mL}$ 分液漏斗。
4.3 分光光度计。
4.4 实验室常规玻璃器具与设备。

# 5样品

# 5.1 样品采集

样品采集后应即时加磷酸(3.7)酸化至 $\mathrm{pH}4$ 以下。因五氯酚在酸性条件下稳定，并加入适量硫酸铜 $(1\mathrm{g} / \mathrm{L})$ ，以抑制微生物对五氯酚的分解作用。

# 5.2 样品运输及保存

样品存放在硬质玻璃瓶中，运输途中要避光，因五氯酚能被光降解。贮于冰箱中避光保存。采集后在 $24\mathrm{h}$ 内进行测定。

# 6 步骤

# 6.1 空白试验

最大试份体积为 $100~\mathrm{mL}$ ，相对标准偏差不得大于 $10\%$ 时，测定下限为 $1\mu \mathrm{g}$ 。

取水 $100~\mathrm{mL}$ ，采用与试样完全相同的步骤进行平行测定。

# 6.2 测定

# 6.2.1 预蒸馏

取 $200\mathrm{mL}$ （或取适量稀释至 $200\mathrm{mL}$ 试样，移入蒸馏器(4.1)中，补加 $25\mathrm{mL}$ 水，加入数粒玻璃珠，再加 $2\sim 3$ 滴甲基橙指示剂溶液(3.9)，用磷酸(3.7)调节至 $\mathrm{pH}4$ 以下(溶液呈粉红色。若采样时已加过，且酸量已够，则不加)，加硫酸铜溶液(3.8) $3\sim 5\mathrm{mL}$ （若采样时已加过，则不加)，连接冷凝器，加热蒸馏，收集馏出液 $200\mathrm{mL}$ 。

# 6.2.2 显色

取馏出液(6.2.1) $100.0\mathrm{mL}$ 于 $250~\mathrm{mL}$ 分液漏斗(4.2)中，加入藏红T-硼酸盐缓冲液(3.12) $5\mathrm{mL}$ ，混匀。

# 6.2.3 萃取

准确加入乙酸异戊酯(3.3) $10.0\mathrm{mL}$ 于分液漏斗中，塞紧剧烈振摇 $1.5\mathrm{min}$ ，静置 $15\mathrm{min}$ ，弃去水相，加入无水硫酸钠(3.6) $0.3\sim 0.5\mathrm{g}$ ，置暗处 $30\mathrm{min}$ 。用滤纸擦干漏斗颈管内壁，弃去最初流出的萃取液后，直接放入 $20\mathrm{mm}$ 的比色皿中。

# 6.2.4 分光光度测定

于 $535\mathrm{nm}$ 波长处，以空白试验(6.1)为参比液，测量乙酸异戊酯萃取液(6.2.3)的吸光度。

# 6.3 校准

# 6.3.1 校准系列的制备

于一组7个分液漏斗(4.2)中，分别加入 $50~\mathrm{mL}$ 水，依次加入0；0.20；0.50；1.00；2.00；3.50；5.00 mL五氯酚使用溶液(3.11)，再分别加水至 $100~\mathrm{mL}$ 。

按6.2.2～6.2.4程序进行测定。

# 6.3.2 校准曲线的绘制

以零管为参比液，由校准系列测得的吸光度值，绘制吸光度对五氯酚含量 $(\mu \mathrm{g})$ 的校准曲线。

# 6.4 结果表示

五氯酚浓度和五氯酚钠浓度按下式计算：

$$
\text {五氯酚 (m g / L)} = \frac {m}{V} \tag {1}
$$

$$
\text {五氯酚钠} (\mathrm {m g / L}) = \frac {m}{V} \times 1. 0 8 \dots \
$$

式中： $m$ ——五氯酚质量 $(\mu \mathrm{g})$ ，由试份测得的吸光度值(6.2.4)，再从相应的五氯酚校准曲线(6.3.2)确定 $m$ 值；

$V$ ——试份体积， $\mathrm{mL}$

1.08——五氯酚钠的换算系数。

# 6.5 精密度和准确度

由4个实验室参加本法验证试验。

# 6.5.1 实验室内

含 $0.02 \sim 0.027 \mathrm{mg} / \mathrm{L}$ 五氯酚河水，加标回收率均值为 $95.5\%$ ，变率系数为 $3.4\%$ ；浓度范围在 $2.39 \sim 2.72 \mathrm{mg} / \mathrm{L}$ 五氯酚的加标工业废水回收率均值为 $96.7\%$ ，变异系数为 $5.5\%$ 。

# 6.5.2 实验室间

分析含 $0.050\mathrm{mg / L}$ 五氯酚的统一样品，实验室间总相对标准偏差为 $4.5\%$ 。加标回收率置信范围 $(100\pm 3.8)\% ,(\alpha = 0.05)$ 。

# 方法说明

（参考件）

A1 藏红 $\mathrm{T}^{\mathrm{C},\mathrm{T}}$ Colour Index, 编号82555, 分子式: $\mathrm{C}_{20}\mathrm{H}_{19}\mathrm{N}_{1}\mathrm{Cl} = 350.80$ 。又名: 番红花红 T, 蓝光藏红花红 T, 蓝基桃红, 沙黄等。英文名: Safranine T, Safranine O, Safranine A 等为一种氧化还原指示剂及染料。

藏红T的质量直接影响藏红T-硼酸盐缓冲溶液的纯化。

A2 五氯酚与藏红T生成的紫红色络合物的乙酸异戊酯萃取液(显色液)对光不稳定，应避光。比色时应减少单色光的光照次数，故使用本法应避开直射灯光和日光下进行。
A3 萃取后的乙酸异戊酯色层（显色液），应在 $30\mathrm{min}$ 内与水相分离，以免在水相中停留时间过长而退色。
