# 中华人民共和国国家标准

# 水质总碑的测定

# 二乙基二硫代氨基甲酸银

# 分光光度法

UDC 614.777:543

.42:546.19

GB7485-87

Water quality-Determination of total

arsenic—Silver diethyldithiocarbamate

spectrophotometric method

本标准参照采用ISO6595《水质——总碑的测定——二乙基二硫代氨基甲酸银分光光度法》。

本标准根据我国实际情况对ISO6595标准方法作如下主要修改：

a. 吸收液中的有机碱、麻黄碱或吡啶改为三乙醇胺。
b. 试份的预处理由高锰酸钾-过硫酸钾法改为直接测定和硝酸-硫酸加热消解法。

# 1 适用范围

1.1 本标准规定二乙基二硫代氨基甲酸银分光光度法测定水和废水中的砷。

当试样取最大体积 $50\mathrm{m}1$ 时，本方法可测上限浓度为含砷 $0.50\mathrm{mg / L}$ 。用无砷水适当稀释试样，也可测定较高浓度的砷。

1.2 最低检出浓度

试样为 $50\mathrm{m}1$ ，用 $10\mathrm{mm}$ 比色皿，可检测含砷 $0.007\mathrm{mg / L}$ 。

1.3 干扰

锑、铋干扰测定（参见附录A）。铬、钴、铜、镍、汞、银以及铂，它们浓度高达 $5\mathrm{mg / L}$ 时也不干扰测定。

# 2 定义

下列定义适用于本标准。

总碑：指单体形态、无机和有机化合物中碑的总量。

# 3 原理

锌与酸作用，产生新生态氢；在碘化钾和氯化亚锡存在下，使五价砷还原为三价；三价砷被初生态氢还原成砷化氢（肿）；用二乙基二硫代氨基甲酸银－三乙醇胺的氯仿液吸收肿，生成红色胶体银，在波长 $530\mathrm{nm}$ 处，测量吸收液的吸光度。

# 4 试剂

除非另有说明，分析时均使用符合国家标准或专业标准的分析纯试剂和蒸馏水或同等纯度的水，试剂和水中砷的含量可忽略不计。

4.1 二乙基二硫代氨基甲酸银（C5H10NS2Ag）。
4.2 三乙醇胺 $\left[\left(\mathrm{HOCH}_{2} \mathrm{CH}_{3}\right)_{3} \mathrm{~N}\right)$ 。
4.3 氯仿（CHCl3）。

4.4 无砷锌粒（10～20目）。
4.5 盐酸（HC1）： $\rho = 1.19\mathrm{g / ml}$ 。
4.6 硝酸（HNO3）： $\rho = 1.40\mathrm{g / m}1$
4.7 硫酸（ $\mathrm{H}_2\mathrm{SO}_4$ ）： $\rho = 1.84\mathrm{g / ml}$ 。
4.8 硫酸溶液 $(\frac{1}{2}\mathrm{H}_2\mathrm{SO}_4)$ ： $2\mathrm{mol} / \mathrm{L}$ 。
4.9 氢氧化钠（NaOH）溶液：2mol/L，贮存在聚乙烯瓶中。
4.10 碘化钾（KI）溶液： $150\mathrm{g / L}$ 。

将 $15\mathrm{g}$ 碘化钾（KI）溶于水中并稀释到 $100\mathrm{ml}$ 。贮存在棕色玻璃瓶中。此溶液至少一个月内是稳定的。

4.11 氯化亚锡溶液。

将 $40\mathrm{g}$ 氯化亚锡（ $\mathrm{SnCl}_2\cdot 2\mathrm{H}_2\mathrm{O}$ ）溶于 $40\mathrm{ml}$ 盐酸（4.5）中。溶液澄清后，用水稀释到 $100\mathrm{ml}$ 。加数粒金属锡保存。

4.12 硫酸铜溶液： $150\mathrm{g / L}$ 。

将 $15\mathrm{g}$ 硫酸铜（ $\mathrm{CuSO_4\cdot 5H_2O}$ ）溶于水中并稀释到 $100\mathrm{ml}$ 。

4.13 乙酸铅溶液： $80\mathrm{g / L}$ 。

将 $8\mathrm{g}$ 乙酸铅 $\left[\left(\mathrm{Pb}\left(\mathrm{CH}_{3} \mathrm{COO}\right)_{2} \cdot 3 \mathrm{H}_{2} \mathrm{O}\right)\right]$ 溶于水中并稀释到 $100 \mathrm{ml}$ 。

4.14 乙酸铅棉花。

将 $10\mathrm{g}$ 脱脂棉浸于 $100\mathrm{ml}$ 乙酸铅溶液（4.13）中，浸透后取出风干。

4.15 吸收液

将 $0.25\mathrm{g}$ 二乙基二硫代氨基甲酸银（4.1）用少量氯仿（4.3）溶成糊状，加入 $2\mathrm{ml}$ 三乙醇胺（4.2），再用氯仿（4.3）稀释到 $100\mathrm{ml}$ 。用力振荡使尽量溶解。静置暗处 $24\mathrm{h}$ 后，倾出上清液或用定性滤纸过滤。贮于棕色玻璃瓶中。贮存在冰箱中是稳定的。

4.16 砷标准溶液： $100.0\mathrm{mg / L}$ 。

将三氧化二砷（ $\mathrm{AS}_2\mathrm{O}_3$ ）在硅胶上预先干燥至恒重，准确称量 $0.1320\mathrm{g}$ ，溶于 $5\mathrm{ml}$ 氢氧化钠溶液（4.9）中，溶解后加入 $10\mathrm{ml}$ 硫酸溶液（4.8），转移至 $1000\mathrm{ml}$ 容量瓶中。用水稀释到刻度。

此标准溶液含砷 $100.0\mu \mathrm{g} / \mathrm{ml}$ 。

4.17 砷标准溶液： $1.00\mathrm{mg / L}$ 。

取 $10.00\mathrm{mI}$ 砷标准溶液（4.16）于 $1000\mathrm{mI}$ 容量瓶中，用水稀释到刻度。

注：需特别注意本分析方法所用的砷，在溶液转移和处置中要特别小心，整个操作应在良好的通风环境中进行，并严防人口。

# 5 仪器

一般实验室仪器和：

5.1 分光光度计： $10\mathrm{mm}$ 比色皿。
5.2 砷化氢发生装置，此仪器由下述部件组装而成：

5.2.1 砷化氢发生瓶：容量为 $150\mathrm{mL}$ 、带有磨口玻璃接头的锥形瓶。
5.2.2 导气管：一端带有磨口接头，并有一球形泡（内装乙酸铅棉花（4.14）；一端被拉成毛细管，管口直径不大于 $1\mathrm{mm}$ 。
5.2.3 吸收管：内径为 $8\mathrm{mm}$ 的试管，带有 $5.0\mathrm{m}1$ 刻度。

注：吸收液柱高保持 $8\sim 10\mathrm{cm}$ 。

# 6 测定步骤

6.1 试份

取 $50\mathrm{ml}$ 试样于砷化氢发生瓶（5.2.1）中，如预料砷的含量超过 $0.5\mathrm{mg / L}$ ，取适量的试样，并

用水稀释到 $50\mathrm{ml}$ 。

# 6.2 空白试验

在测定的同时应进行空白试验，所用试剂及其用量与在测定中所用的相同，包括任何预处理的步骤亦相同。但用 $50\mathrm{m}1$ 水取代试份。

# 6.3 测定

# 6.3.1 预处理

除非证明试样的消解处理是不必要的，可直接制备试份（6.1），加入 $4\mathrm{ml}$ 硫酸（4.7）进行显色（6.3.2）和测定（6.3.3），否则，要按下述步骤进行预处理：

于砷化氢发生瓶（5.2.1）中，加入 $4\mathrm{ml}$ 硫酸（4.7）和 $5\mathrm{ml}$ 硝酸（4.6）。在通风橱内煮沸消解至产生白色烟雾。如溶液仍不清澈，可再加 $5\mathrm{ml}$ 硝酸（4.6），继续加热至产生白色烟雾，直至溶液清澈为止（其中可能存在乳白色或淡黄色酸不溶物）。冷却后，小心加入 $25\mathrm{ml}$ 水，再加热至产生白色烟雾，赶尽氮氧化物，冷却后，加水使总体积为 $50\mathrm{ml}$ 。

注：在消解破坏有机物的过程中，勿使溶液变黑，否则砷可能有损失。

# 6.3.2 显色

6.3.2.1 于砷化氢发生瓶（6.3.1）中，加 $4\mathrm{ml}$ 碘化钾（4.10），摇匀，再加 $2\mathrm{ml}$ 氯化亚锡溶液（4.11），混匀，放置 $15\mathrm{min}$ 。

6.3.2.2 取 $5.0\mathrm{ml}$ 吸收液（4.15）至吸收管（5.2.3）中，插入导气管（5.2.2）。

6.3.2.3 加 $1\mathrm{ml}$ 硫酸铜溶液（4.12）和 $4\mathrm{g}$ 无砷锌粒（4.4）于砷化氢发生瓶中，并立即将导气管与发生瓶连接，保证反应器密闭。

6.3.2.4 在室温下，维持反应 $1\mathrm{h}$ ，使脾完全释出。加氯仿（4.3）将吸收体积补足到 $5.0\mathrm{ml}$ 。

注：① 砷化氢剧毒，整个反应应在通风橱内或通风良好的室内进行。

② 在完全释放砷化氢后，红色生成物在 $2.5 \mathrm{~h}$ 内是稳定的，应在此期间内进行分光光度测定。

# 6.3.3 光度测定

用 $10\mathrm{mm}$ 比色皿，以氯仿（4.3）为参比液，在 $530\mathrm{nm}$ 波长下测量吸收液（6.3.2.4）的吸光度，减去空白试验（6.2）所测得的吸光度，从校准曲线（6.4.3）上查出试份中的含砷量。

# 6.4 校准

# 6.4.1 标准工作溶液的制备

往8个砷化氢发生瓶（5.2.1）中，分别加入0、1.00、2.50、5.00、10.00、15.00、20.00及 $25.00\mathrm{m}1$ 砷标准溶液（4.17），并用水加到 $50\mathrm{m}1$ 。

# 6.4.2 显色与测定

于上述砷化氢发生瓶中，分别加入 $4\mathrm{m}1$ 硫酸（4.7），以下步骤按6.3.2和6.3.3进行。

# 6.4.3 校准曲线的绘制

减去试剂空白的吸光度，来修正对应的每个标准溶液的吸光度。以修正的吸光度为纵坐标，与之对应的标准溶液的砷含量（ $\mu \mathrm{g}$ ）为横坐标作图。

要经常绘制校准曲线，至少在每次使用新试剂时，要绘制一次。

# 7 结果的表示

# 7.1 计算方法

砷含量 $c(\mathrm{mg / L})$ 由下式计算：

$$
c = \frac {m}{V}
$$

式中： $m$ ——校准曲线查得的试份砷含量， $\mu \mathrm{g}$ ；

$\pmb{V}$ ——试份体积， $\mathbf{m}1$ 。

# 7.2 结果表示

7.2.1 取平行测定结果的算术平均值为测定结果。
7.2.2 报告碑的含量。根据有效数字的规则，结果以二位或三位有效数字表示。

# 7.3.1 重复性

实验室内相对标准偏差为 $2\%$ 。

# 7.3.2 再现性

实验空间相对标准偏差为 $3\%$ 。

# 7.3.3 准确度

# 附录A

（补充件）

# A.1 锋的干扰及其消除

锑盐在试验条件下，还原生成氢化物，又能与吸收液作用产生红色胶体银。试份中锑的含量大于 $0.1\mathrm{mg / L}$ 时，干扰砷的测定。加入 $2\mathrm{ml}$ 氯化亚锡溶液（4.11）和 $5\mathrm{ml}$ 碘化钾溶液（4.10），可抑制 $300\mu \mathrm{g}$ 锑盐的干扰。如锑浓度很高，本方法不适用。

A.2 硝酸浓度为 $0.01\mathrm{mol} / \mathrm{L}$ 时，开始有负干扰，故不适合作保存剂。若试份中有硝酸，分析前要加硫酸再加热分解之。
A.3 硫化物对测定有干扰，可通过乙酸铅棉花去除。若棉花变黑，应更换。
A.4 吸收液中的氯仿沸点较低，在吸收脾的过程中，易挥发损失，影响脾的吸收。当室温较高时，建议将发生瓶和吸收管降温，并不断补加氯仿于吸收管中，使之尽可能保持一定高度的液层。
A.5 锌粒的规格（粒度）对砷化氢的发生有影响，表面粗糙的锌粒还原效率高，规格以 $10 \sim 20$ 目为宜，粒度较大，应适当增加用量。
A.6 夏天高温季节，还原反应激烈，可适当减少硫酸溶液（4.7）的用量，或将砷化氢发生瓶放入冷水中，使反应缓和。
