# 1 适用范围

本标准适用于测定饮用水、地下水和清洁地面水中的硝酸盐氮。

# 1.1 测定范围

本方法适用于测定硝酸盐氮浓度范围在 $0.02 \sim 2.0 \mathrm{mg} / \mathrm{L}$ 之间。浓度更高时，可分取较少的试份测定。

# 1.2 最低检出浓度

采用光程为 $30\mathrm{mm}$ 的比色皿，试份体积为 $50\mathrm{ml}$ 时，最低检出浓度为 $0.02\mathrm{mg / L}$ 。

# 1.3 灵敏度

当使用光程为 $30\mathrm{mm}$ 的比色皿，试份体积为 $50\mathrm{ml}$ ，硝酸盐氮含量为 $0.60\mathrm{mg / L}$ 时，吸光度约0.6单位。

使用光程为 $10\mathrm{mm}$ 的比色皿，试份体积为 $50\mathrm{m}1$ ，硝酸盐氮含量为 $2.0\mathrm{mg / L}$ 时，其吸光度约0.7单位。

# 1.4 干扰

水中含氯化物、亚硝酸盐、铵盐、有机物和碳酸盐时，可产生干扰。含此类物质时，应作适当的前处理，以消除对测定的影响。

# 2 原理

硝酸盐在无水情况下与酚二磺酸反应，生成硝基二磺酸酚，在碱性溶液中，生成黄色化合物，于 $410\mathrm{nm}$ 波长处进行分光光度测定。

# 3 试剂

本标准所用试剂除另有说明外，均为分析纯试剂，实验中所用的水，均应用蒸馏水或同等纯度的水。

3.1 硫酸： $\rho = 1.84\mathrm{g / m}^3$ 。
3.2 发烟硫酸 $(\mathrm{H}_2\mathrm{SO}_4\cdot \mathrm{SO}_3)$ ：含 $13\%$ 三氧化硫 $(\mathrm{SO}_3)$ 。

注：（1）发烟硫酸在室温较低时凝固，取用时，可先在 $40\sim 50^{\circ}\mathrm{C}$ 隔水浴中加温使熔化，不能将盛装发烟硫酸的玻璃瓶直接置入水浴中，以免瓶裂引起危险。

（2）发烟硫酸中含三氧化硫（ $\mathrm{SO}_3$ ）浓度超过 $13\%$ 时，可用硫酸（3.1）按计算量进行稀释。

3.3 酚二磺酸 $(\mathrm{C_6H_3(OH)(SO_3H)_2})$

称取25g苯酚置于500m1锥形瓶中，加150m1硫酸（3.1）使之溶解，再加75m1发烟硫酸(3.2），充分混和。瓶口插一小漏斗，置瓶于沸水浴中加热2h，得淡棕色稠液，贮于棕色瓶中，密塞保存。

注：（1）当苯酚色泽变深时，应进行蒸馏精制。

（2）无发烟硫酸时，亦可用硫酸（3.1）代替，但应增加在沸水浴中加热时间至 $6\mathrm{h}$ ，制得的试剂尤应注意防止吸收空气中的水分，以免因硫酸浓度的降低，影响硝基化反应的进行，使测定结果偏低。

3.4 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ ： $\rho = 0.90\mathrm{g / ml}$ 。
3.5 硝酸盐氮标准溶液： $c_{\mathrm{N}} = 100\mathrm{mg / L}$ 。

将0.7218g经 $105\sim 110^{\circ}C$ 干燥 $2\mathrm{h}$ 的硝酸钾（KNO3）溶于水中，移入 $1000\mathrm{ml}$ 容量瓶，用水稀释至标线，混匀。加 $2\mathrm{ml}$ 氯仿作保存剂，至少可稳定6个月。

每毫升本标准溶液含 $0.10\mathrm{mg}$ 硝酸盐氮。

3.6 硝酸盐氮标准溶液： $c_{\mathrm{N}} = 10.0\mathrm{mg / L}$ 。

吸取 $50.0\mathrm{ml}$ 硝酸盐氮标准溶液（3.5），置蒸发皿内，加氢氧化钠溶液（3.9）使调至 $\mathsf{pH8}$ ，在水浴上蒸发至干。加 $2\mathrm{ml}$ 酚二磺酸试剂（3.3），用玻璃棒研磨蒸发皿内壁，使残渣与试剂充分接触，放置片刻，重复研磨一次，放置 $10\mathrm{min}$ ，加入少量水，定量移入 $500\mathrm{ml}$ 容量瓶中，加水至标线，混匀。

每毫升本标准溶液含 $0.010 \mathrm{mg}$ 硝酸盐氮。

贮于棕色瓶中，此溶液至少稳定6个月。

注：本标准溶液应同时制备两份，如发现浓度存在差异时，应重新吸取硝酸盐氮标准溶液（3.5）进行制备。

# 3.7 硫酸银溶液

称取 $4.397\mathrm{g}$ 硫酸银（ $\mathrm{Ag_2SO_4}$ ）溶于水，稀释至 $1000\mathrm{m}1$ 。

1.00ml此溶液可去除 $1.00\mathrm{mg}$ 氯离子（C1-）。

3.8 硫酸溶液： $0.5\mathrm{mol} / \mathrm{L}$ 。
3.9 氢氧化钠溶液： $0.1\mathrm{mol} / \mathrm{L}$ 。

3.10 EDTA二钠溶液。

称取 $50\mathrm{gEDTA}$ 二钠盐的二水合物 $(\mathrm{C_{10}H_{14}N_2O_3Na_2\cdot 2H_2O})$ ，溶于 $20\mathrm{ml}$ 水中，使调成糊状，加入 $60\mathrm{ml}$ 氨水（3.4）充分混合，使之溶解。

3.11 氢氧化铝悬浮液。

称取125g硫酸铝钾（KA1（SO4）2·12H2O）或硫酸铝铵（NH4Al（SO4）2·12H2O）溶于1L水中，加热到60℃，在不断搅拌下徐徐加入55ml氨水（3.4），使生成氢氧化铝沉淀，充分搅拌后静置，弃去上清液。反复用水洗涤沉淀，至倾出液无氯离子和铵盐。最后加入300ml水使成悬浮液。

使用前振摇均匀。

3.12 高锰酸钾溶液： $3.16\mathrm{g / L}$ 。

# 4 仪器

常用实验室仪器及：

4.1 瓷蒸发皿： $75\sim 100\mathrm{ml}$ 容量。
4.2 具塞比色管：50ml。
4.3 分光光度计：适用于测量波长 $410\mathrm{nm}$ ，并配有光程 $10\mathrm{mm}$ 和 $30\mathrm{mm}$ 的比色皿。

# 5 采样和样品

按照国家标准规定及根据待测水的类型提出的特殊建议进行采样。

实验室样品可贮于玻璃瓶或聚乙烯瓶中。

硝酸盐氮的测定应在水样采集后立即进行，必要时，应保存在 $4^{\circ}\mathrm{C}$ 下，但不得超过 $24\mathrm{h}$ 。

# 6 步骤

6.1 试份体积的选择

最大试份体积为 $50\mathrm{ml}$ ，可测定硝酸盐氮浓度至 $2.0\mathrm{mg / L}$ 。

# 6.2 空白试验

取 $50\mathrm{ml}$ 水，以与试份测定完全相同的步骤、试剂和用量，进行平行操作。

# 6.3 干扰的排除

# 6.3.1 带色物质

取 $100\mathrm{ml}$ 试样移入 $100\mathrm{ml}$ 具塞量筒中，加 $2\mathrm{ml}$ 氢氧化铝悬浮液（3.11），密塞充分振摇，静置数分钟澄清后，过滤，弃去最初滤液的 $20\mathrm{ml}$ 。

# 6.3.2 氯离子

取 $100\mathrm{ml}$ 试样移入 $100\mathrm{ml}$ 具塞量筒中，根据已测定的氯离子含量，加入相当量的硫酸银溶液（3.7），充分混合，在暗处放置 $30\mathrm{min}$ ，使氯化银沉淀凝聚，然后用慢速滤纸过滤，弃去最初滤液 $20\mathrm{ml}$ 。

注：（1）如不能获得澄清滤液，可将已加过硫酸银溶液后的试样在近 $80^{\circ}\mathrm{C}$ 的水浴中加热，并用力振摇，使沉淀充分凝聚，冷却后再进行过滤。

（2）如同时需去除带色物质，则可在加入硫酸银溶液并混匀后，再加入 $2\mathrm{ml}$ 氢氧化铝悬浮液，充分振摇，放置片刻待沉淀后，过滤。

# 6.3.3 亚硝酸盐

当亚硝酸盐氮含量超过 $0.2\mathrm{mg / L}$ 时，可取 $100\mathrm{ml}$ 试样，加 $1\mathrm{ml}$ 硫酸溶液（3.8），混匀后，滴加高锰酸钾溶液（3.12），至淡红色保持 $15\mathrm{min}$ 不褪为止，使亚硝酸盐氧化为硝酸盐，最后从硝酸盐氮测定结果中减去亚硝酸盐氮量。

# 6.4 测定

# 6.4.1 蒸发

取 $50.0\mathrm{ml}$ 试份入蒸发皿中，用 $\mathbf{pH}$ 试纸检查，必要时用硫酸溶液（3.8）或氢氧化钠溶液(3.9)，调节至微碱性（ $\mathrm{pH}\approx 8$ ），置水浴上蒸发至干。

# 6.4.2 硝化反应

加 $1.0\mathrm{ml}$ 酚二磺酸试剂（3.3），用玻璃棒研磨，使试剂与蒸发皿内残渣充分接触，放置片刻，再研磨一次，放置 $10\mathrm{min}$ ，加入约 $10\mathrm{ml}$ 水。

# 6.4.3 显色

在搅拌下加入 $3\sim 4\mathrm{ml}$ 氨水（3.4），使溶液呈现最深的颜色。如有沉淀产生，过滤；或滴加EDTA二钠溶液（3.10），并搅拌至沉淀溶解。将溶液移入比色管（4.2）中，用水稀释至标线，混匀。

# 6.4.4 分光光度测定

于 $410\mathrm{nm}$ 波长，选用合适光程长的比色皿，以水为参比，测量溶液的吸光度。

# 7 结果的表示

# 7.1 计算方法

试份中硝酸盐氮的吸光度 $A_{\mathrm{r}}$ 用式（1）计算：

$$
A _ {r} = A _ {s} - A _ {b} \dots \tag {1}
$$

式中： $A_{s}$ ——试份溶液（6.4）的吸光度；

$A_{\mathrm{b}}$ ——空白试验溶液（6.2）的吸光度。

注：对某种特定样品，As和Ab应在同一种光程长的比色皿中测定。

硝酸盐氮含量 $c_{\mathrm{NMg}} / \mathrm{L}$ 表示。

7.1.1 未经去除氯离子的试样，按式（2）计算：

$$
c _ {N} = \frac {m}{V} \times 1 0 0 0 \dots \tag {2}
$$

式中： $m$ ——硝酸盐氮质量， $\mathbf{mg}$ ，由 $A$ 值和相应比色皿光程的校准曲线（6.5.2）确定；

$\nu$ ——试份体积， $\mathbf{m}1$ ；

1000——换算为每升试样计。

7.1.2 经去除氯离子的试样，按式（3）计算：

$$
c _ {N} = \frac {m}{V} \times 1 0 0 0 \times \frac {V _ {1} + V _ {2}}{V _ {1}} \dots \tag {3}
$$

式中： $V_{1}$ ——供去氯离子的试样取用量， $\mathrm{m}1$ ；

$V_{2}$ ——硫酸银溶液加入量，ml。

# 7.2 精密度和准确度

7.2.1 经5个实验室的分析方法协作试验结果如下：

# 7.2.1.1 实验室内

浓度范围为 $0.2 \sim 0.4 \mathrm{mg} / \mathrm{L}$ 的加标地面水，最大总相对标准偏差 $6.4\%$ ，回收率平均值 $78\%$ 。

浓度范围 $1.8 \sim 2.0 \mathrm{mg} / \mathrm{L}$ 的加标地面水，最大总相对标准偏差 $5.4\%$ ，回收率平均值 $98.6\%$ 。

# 7.2.1.2 实验室间

a. 分析含硝酸盐氮 $1.20 \mathrm{mg} / \mathrm{L}$ 的统一分发标准样，实验室间总相对标准偏差为 $9.4\%$ ，相对误差为 $-6.7\%$ 。
b. 52个实验室测定含硝酸盐氮 $1.59 \mathrm{mg} / \mathrm{L}$ 的合成水样，相对标准偏差为 $11.0\%$ ，相对误差为 $8.8\%$ 。
