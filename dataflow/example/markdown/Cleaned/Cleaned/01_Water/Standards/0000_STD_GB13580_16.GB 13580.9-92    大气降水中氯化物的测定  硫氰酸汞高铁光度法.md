# 中华人民共和国国家标准

# 大气降水中氯化物的测定

# 硫氰酸汞高铁光度法

GB13580.9-92

Determination of chloride in wet precipitation

Ferrithiocyanate spectrophotometry

# 1 主题内容与适用范围

1.1 本标准规定了大气降水中氯化物的硫氰酸汞高铁光度法。
1.2 本标准适用于大气降水样品中氯化物测定。
1.3 本标准最低检出浓度为 $0.03\mathrm{mg / L}$ ，测定范围为 $0.4\sim 6.0\mathrm{mg / L}$ 。

# 2 引用标准

GB13580.2 大气降水样品的采集与保存

# 3 原理

氯离子与硫氰酸汞反应，交换出的硫氰酸根离子与三价铁离子反应，生成红色硫氰酸铁络合物，于波长 $460\mathrm{nm}$ 处进行分光光度测定。

# 4 试剂

4.1 氯化物标准贮备液： $1000\mu \mathrm{g / mL}$ 。称取 $2.1028\mathrm{g}$ 氯化钾 $(105\%)$ 烘 $2\mathrm{h}$ ，溶于水，定容至 $1000\mathrm{mL}$ 。
4.2 氯化物标准使用液： $10\mu \mathrm{g} / \mathrm{mL}$ 。吸取氯化物标准贮备液(4.1) $5.00\mathrm{mL}$ 于 $500\mathrm{mL}$ 容量瓶中，用水稀释到刻度。
4.3 高氯酸溶液： $(1 + 2)$ 。量取 $100 \mathrm{~mL}$ 高氯酸加入到 $200 \mathrm{~mL}$ 水中，摇匀。
4.4 硫酸铁铵溶液： $60\mathrm{g/L}$ 。称 $6\mathrm{g}$ 硫酸铁铵 $\left(\mathrm{NH}_{4}\mathrm{FeSO}_{4} \cdot 12\mathrm{H}_{2}\mathrm{O}\right)$ ，用高氯酸溶液（4.3）溶解，稀释到 $100\mathrm{mL}$ 。如浑浊应过滤。
4.5 硫氰酸钾溶液： $40\mathrm{g/L}$ 。称 $4\mathrm{g}$ 硫氰酸钾(KCNS)溶于水，稀释到 $100\mathrm{mL}$ 。
4.6 硝酸溶液： $0.5\mathrm{mol} / \mathrm{L}$ 。吸取 $7.0\mathrm{mL}$ 硝酸用水稀释到 $200\mathrm{mL}$ 。
4.7 硫氰酸汞-乙醇溶液: $4\mathrm{g} / \mathrm{L}$ 。称 $0.4\mathrm{g}$ 硫氰酸汞, 用无水乙醇配成 $100\mathrm{mL}$ 溶液。用定性滤纸过滤, 收集滤液于棕色瓶中备用。

硫氰酸汞的制备：称 $5\mathrm{g}$ 硝酸汞 $\mathrm{[Hg(NO_3)_2\cdot H_2O]}$ 溶于 $200\mathrm{mL}$ 硝酸溶液(4.6)中，加 $3\mathrm{mL}$ 硫酸铁铵溶液(4.4)，在搅拌下，滴加硫氰酸钾溶液(4.5)至试验呈微橙红色为止。生成硫氰酸汞白色沉淀，用G3砂芯漏斗(4.3)过滤，并用水充分洗涤(用倾洗法)，将沉淀放入干燥器中自然干燥，贮于棕色瓶中。

# 5 仪器

5.1 分光光度计。
5.2 $10\mathrm{mL}$ 具塞比色管。

# 5.3 G3砂芯漏斗。

# 6 样品采集与保存

按GB13580.2执行。

# 7 步骤

7.1 校准曲线的绘制：取 $10\mathrm{mL}$ 干燥比色管8支，分别加入氯化物标准使用液(4.2)0,0.30,0.50,1.00,1.50,2.00,2.50,3.00 mL，各加水至 $10\mathrm{mL}$ ，再向各管加入 $2.00\mathrm{mL}$ 硫酸铁铵溶液(4.4)，加入硫氰酸汞溶液(4.6)1.00 mL，混匀。室温下放置 $20\mathrm{min}$ 。于波长 $460\mathrm{nm}$ 处，用 $20\mathrm{mm}$ 吸收池，以水作参比测量吸光度。绘制氯化物的校准曲线。
7.2样品测定：根据降水中氯化物的含量，吸取 $5.00\mathrm{mL}$ 样品于 $10\mathrm{mL}$ 干燥比色管中，加水至 $10\mathrm{mL}$ 以下按绘制校准曲线的步骤(7.1)进行操作，由测得的吸光度，从校准曲线上查得氯化物含量。

# 8 分析结果的表述

降水中氯化物（按 $\mathrm{Cl}^-$ 计)的浓度以 $\mathrm{mg / L}$ 表示，用下式计算：

$$
C = \frac {M}{V}
$$

式中： $C$ ——样品中氯化物的浓度， $\mathrm{mg / L}$

$M$ ——从校准曲线上查得氯化物的含量, $\mu \mathrm{g}$ ;

$V$ ——取样体积， $\mathrm{mL}$ 。

# 9 精密度和准确度

6个实验室对含 $\mathrm{Cl}^{-}1.00\mathrm{mg / L},\mathrm{F}^{-}0.20\mathrm{mg / L},\mathrm{SO}_{4}^{2 - }6.00\mathrm{mg / L},\mathrm{NO}_{3}^{-}1.20\mathrm{mg / L},\mathrm{K}^{+}2.98\mathrm{mg / L},\mathrm{Na}^{+}0.44\mathrm{mg / L}$ 的合成水样进行测定，测定结果 $\mathrm{Cl}^{-}$ 的相对标准偏差为 $4.1\%$ ，相对误差为 $-3.0\%$
