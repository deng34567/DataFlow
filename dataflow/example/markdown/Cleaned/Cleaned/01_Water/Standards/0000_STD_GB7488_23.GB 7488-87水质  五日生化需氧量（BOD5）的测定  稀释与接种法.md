# 中华人民共和国国家标准

# 水质五日生化需氧量（ $\mathbf{BOD}_5$ ）的测定稀释与接种法

UDC 614.777 :577.121.7

GB7488-87

Water quality-Determination of biochemical oxygen demand after 5 days $(\mathbf{BOD}_5)$ -Dilution and seeding method

本标准参照采用国际标准ISO5815—1983，本国家标准规定采用稀释与接种法作为测定水中生化需氧量的标准方法，这是一种经验性的常规方法。

# 1 适用范围

本方法适用于 $\mathbf{BOD}_5$ 大于或等于 $2\mathrm{mg / L}$ 并且不超过 $6000\mathrm{mg / L}$ 的水样。 $\mathbf{BOD}_5$ 大于 $6000\mathrm{mg / L}$ 的水样仍可用本方法，但由于稀释会造成误差，有必要要求对测定结果做慎重的说明。

本试验得到的结果是生物化学和化学作用共同产生的结果，它们不象单一的、有明确定义的化学过程那样具有严格和明确的特性，但是它能提供用于评价各种水样质量的指标。

本试验的结果可能会被水中存在的某些物质所干扰，那些对微生物有毒的物质，如杀菌剂、有毒金属或游离氯等，会抑制生化作用。水中的藻类或硝化微生物也可能造成虚假的偏高结果。

# 2 定义

生物化学需氧量（BOD）：在规定条件下，水中有机物和无机物在生物氧化作用下所消耗的溶解氧（以质量浓度表示）。

# 3 原理

将水样注满培养瓶，塞好后应不透气，将瓶置于恒温条件下培养5天。培养前后分别测定溶解氧浓度，由两者的差值可算出每升水消耗掉氧的质量，即BOD,值。

由于多数水样中含有较多的需氧物质，其需氧量往往超过水中可利用的溶解氧（DO）量，因此在培养前需对水样进行稀释，使培养后剩余的溶解氧（DO）符合规定。

一般水质检验所测BOD,只包括含碳物质的耗氧量和无机还原性物质的耗氧量。有时需要分别测定含碳物质耗氧量和硝化作用的耗氧量。常用的区别含碳和氮的硝化耗氧的方法是向培养瓶中投加硝化抑制剂,加入适量硝化抑制剂后,所测出的耗氧量即为含碳物质的耗氧量。在5天培养时间内,硝化作用的耗氧量取决于是否存在足够数量的能进行此种氧化作用的微生物,原污水或初级处理的出水中这种微生物的数量不足,不能氧化显著量的还原性氮,而许多二级生化处理的出水和受污染较久的水体中,往往含有大量硝化微生物,因此测定这种水样时应抑制其硝化反应。

在测定BOD,的同时，需用葡萄糖和谷氨酸标准溶液完成验证试验。

# 4 试剂

分析时，只采用公认的分析纯试剂和蒸馏水或同等纯度的水（在全玻璃装置中蒸馏的水或去离子水），水中含铜不应高于 $0.01\mathrm{mg / L}$ ，并不应有氯、氯胺、苛性碱、有机物和酸类。

# 4.1 接种水

如试验样品本身不含有足够的合适性微生物，应采用下述方法之一，以获得接种水：

a. 城市废水，取自污水管或取自没有明显工业污染的住宅区污水管。

这种水在使用前，应倾出上清液备用。
b. 在 $1 \mathrm{~L}$ 水中加入 $100 \mathrm{~g}$ 花园土壤，混合并静置 $10 \mathrm{~min}$ 。取 $10 \mathrm{ml}$ 上清液用水稀释至 $1 \mathrm{~L}$ 。
c. 含有城市污水的河水或湖水。
d. 污水处理厂出水。
e. 当待分析水样为含难降解物质的工业废水时，取自待分析水排放口下游约 $3 \sim 8 \mathrm{~km}$ 的水或所含微生物适宜于待分析水并经实验室培养过的水。

# 4.2 盐溶液

下述溶液至少可稳定一个月，应贮存在玻璃瓶内，置于暗处。一旦发现有生物滋长迹象，则应弃去不用。

# 4.2.1 磷酸盐：缓冲溶液。

将 $8.5\mathrm{g}$ 磷酸二氢钾 $\left(\mathrm{KH}_{2} \mathrm{PO}_{4}\right)$ 、 $21.75\mathrm{g}$ 磷酸氢二钾 $\left(\mathrm{K}_{2} \mathrm{HPO}_{4}\right)$ 、 $33.4\mathrm{g}$ 七水磷酸氢二钠 $\left(\mathrm{Na}_{2} \mathrm{HPO}_{4} \cdot 7 \mathrm{H}_{2} \mathrm{O}\right)$ 和 $1.7\mathrm{g}$ 氯化铵 $\left(\mathrm{NH}_{4} \mathrm{Cl}\right)$ 溶于约 $500 \mathrm{ml}$ 水中，稀释至 $1000 \mathrm{ml}$ 并混合均匀。

此缓冲溶液的 $\mathbf{pH}$ 应为7.2。

# 4.2.2 七水硫酸镁： $22.5\mathrm{g} / \mathrm{L}$ 溶液。

将 $22.5\mathrm{g}$ 的七水硫酸镁（ $\mathrm{MgSO_4\cdot 7H_2O}$ ）溶于水中，稀释至 $1000\mathrm{ml}$ 并混合均匀。

# 4.2.3 氯化钙： $27.5\mathrm{g} / \mathrm{L}$ 溶液。

将 $27.5\mathrm{g}$ 的无水氯化钙（ $\mathrm{CaCl}_2$ ）（若用水合氯化钙，要取相当的量）溶于水，稀释至 $1000\mathrm{ml}$ 并混合均匀。

# 4.2.4 六水氯化铁（Ⅲ）： $0.25\mathrm{g} / \mathrm{L}$ 溶液。

将 $0.25\mathrm{g}$ 六水氯化铁（Ⅲ）（ $\mathrm{FeCl}_3 \cdot 6\mathrm{H}_2\mathrm{O}$ ）溶解于水中，稀释至 $1000\mathrm{ml}$ 并混合均匀。

# 4.3 稀释水

取每种盐溶液（4.2.1、4.2.2、4.2.3和4.2.4）各 $1\mathrm{ml}$ ，加入约 $500\mathrm{ml}$ 水中，然后稀释至 $1000\mathrm{ml}$ 并混合均匀，将此溶液置于 $20^{\circ}\mathrm{C}$ 下恒温，曝气 $1\mathrm{h}$ 以上，采取各种措施，使其不受污染，特别是不被有机物质、氧化或还原性物质或金属污染\*，确保溶解氧浓度不低于 $8\mathrm{mg / L}$ 。

此溶液的5日生化需氧量不得超过 $0.2\mathrm{mg / L}$ 。

此溶液应在 $\mathbf{8h}$ 内使用。

# 4.4 接种的稀释水

根据需要和接种水的来源，向每升稀释水（4.3）中加 $1.0 \sim 5.0 \mathrm{ml}$ 接种水（4.1），将已接种的稀释水在约 $20^{\circ} \mathrm{C}$ 下保存，8h后尽早应用。

已接种的稀释水的5天（ $20^{\circ}\mathrm{C}$ ）耗氧量应在每升 $0.3 \sim 1.0 \mathrm{mg}$ 之间。

# 4.5 盐酸（HCl）溶液： $0.5\mathrm{mol} / \mathrm{L}$ 。

4.6 氢氧化钠（NaOH）溶液： $20\mathrm{g} / \mathrm{L}$ 。
4.7 亚硫酸钠 $(\mathrm{Na}_2\mathrm{SO}_3)$ 溶液： $1.575\mathrm{g} / \mathrm{L}$ ，此溶液不稳定，需每天配制。

# 4.8 葡萄糖-谷氨酸标准溶液。

将一些无水葡萄糖 $\left(\mathbf{C}_{6} \mathbf{H}_{12} \mathbf{O}_{6}\right)$ 和一些谷氨酸 $\left(\mathrm{HOOC}-\mathrm{CH}_{2}-\mathrm{CH}_{2}-\mathrm{CHNH}_{2}-\mathrm{COOH}\right)$ 在 $103^{\circ} \mathrm{C}$ 下干燥 $1 \mathrm{~h}$ , 每种称量 $150 \pm 1 \mathrm{mg}$ , 溶于蒸馏水中, 稀释至 $1000 \mathrm{ml}$ 并混合均匀。

此溶液于临用前配制。

# 5 仪器

使用的玻璃器皿要认真清洗，不能吸有毒的或生物可降解的化合物，并防止沾污。

常用的实验室设备如下：

5.1 培养瓶：细口瓶的容量在 $250 \sim 300 \mathrm{ml}$ 之间，带有磨口玻璃塞，并具有供水封用的钟形口，最好是直肩的。
5.2 培养箱：能控制在 $20 \pm 1^{\circ} \mathrm{C}$ 。
5.3 测定溶解氧仪器*。
5.4 用于样品运输和贮藏的冷藏手段（ $0 \sim 4^{\circ} \mathrm{C}$ ）。
5.5 稀释容器：带塞玻璃瓶，刻度精确到毫升，其容积大小取决于使用稀释样品的体积。

# 6 样品的贮存

样品需充满并密封于瓶中，置于 $2 \sim 5^{\circ} \mathrm{C}$ 保存到进行分析时。一般应在采样后 $6 \mathrm{~h}$ 之内进行检验。若需远距离转运，在任何情况下贮存皆不得超过 $24 \mathrm{~h}$ 。

样品也可以深度冷冻贮存。

# 7 操作步骤

# 7.1 样品预处理

# 7.1.1 样品的中和

如果样品的 $\mathbf{pH}$ 不在 $6 \sim 8$ 之间，先做单独试验，确定需要用的盐酸溶液（4.5）或氢氧化钠溶液（4.6）的体积，再中和样品，不管有无沉淀形成。

# 7.1.2 含游离氯或结合氯的样品

加入所需体积的亚硫酸钠溶液（4.7），使样品中自由氯和结合氯失效，注意避免加过量。

# 7.2 试验水样的准备

将试验样品温度升至约 $20^{\circ} \mathrm{C}$ ，然后在半充满的容器内摇动样品，以便消除可能存在的过饱和氧。

将已知体积样品置于稀释容器（5.5）中，用稀释水（4.3）或接种稀释水（4.4）稀释，轻轻地混合，避免夹杂空气泡。稀释倍数可参考下表。

测定BOD,时建议稀释的倍数


| 预期BOD,值,mg/L | 稀释比   | 结果取整到 | 适用的水样 |
| ------------ | ----- | ----- | ----- |
| 2~6          | 1~2之间 | 0.5   | R     |
| 4~12         | 2     | 0.5   | R, E  |
| 10~30        | 5     | 0.5   | R, E  |
| 20~60        | 10    | 1     | E     |
| 40~120       | 20    | 2     | S     |
| 100~300      | 50    | 5     | S, C  |
| 200~600      | 100   | 10    | S, C  |
| 400~1200     | 200   | 20    | I, C  |
| 1000~3000    | 500   | 50    | I     |
| 2000~6000    | 1000  | 100   | I     |


表中：R：河水；
E：生物净化过的污水；
S：澄清过的污水或轻度污染的工业废水；

C：原污水；

I：严重污染的工业废水。

若采用的稀释比大于100, 将分两步或几步进行稀释。若需要抑制硝化作用, 则加入ATU或TCMP试剂。

若只需要测定有机物降解的耗氧，必需抑制硝化微生物以避免氮的硝化过程，为此目的，在每升稀释样品中加入 $2\mathrm{ml}$ 浓度为 $500\mathrm{mg / L}$ 的烯丙基硫脲（ATU）（ $\mathrm{C_4H_8N_2S}$ ）溶液或一定量的固定在氯化钠（NaCl）上的2-氯代-6-三氯甲基吡啶（TCMP）（ $\mathrm{Cl - C_5H_3N - CCl_3}$ ），使TCMP在稀释样品中浓度大约为 $0.5\mathrm{mg / L}$ 。

恰当的稀释比应使培养后剩余溶解氧至少有 $1\mathrm{mg / L}$ 和消耗的溶解氧至少 $2\mathrm{mg / L}$ 。

当难于确定恰当的稀释比时，可先测定水样的总有机碳(TOC)或重铬酸盐法化学需氧量(COD)，根据TOC或COD估计BOD，可能值，再围绕预期的BOD值，做几种不同的稀释比，最后从所得测定结果中选取合乎要求条件者。

# 7.3 空白试验

用接种稀释水（4.4）进行平行空白实验测定。

# 7.4 测定

按采用的稀释比（见7.2）用虹吸管充满两个培养瓶至稍溢出。

将所有附着在瓶壁上的空气泡赶掉，盖上瓶盖，小心避免夹空气泡。

将瓶子分为二组，每组都含有一瓶选定稀释比的稀释水样和一瓶空白溶液（见7.3）。

放一组瓶于培养箱（5.2）中，并在暗中放置5天。

在计时起点时，测量另一组瓶的稀释水样和空白溶液中的溶解氧浓度。

达到需要培养的5天时间时，测定放在培养箱中那组稀释水样和空白溶液的溶解氧浓度。

# 7.5 验证试验

为了检验接种稀释水、接种水和分析人员的技术，需进行验证试验。将 $20\mathrm{ml}$ 葡萄糖-谷氨酸标准溶液（4.8）用接种稀释水（4.4）稀释至 $1000\mathrm{ml}$ ，并且按照7.4的步骤进行测定。

得到的BOD,应在 $180\sim 230\mathrm{mg / L}$ 之间，否则，应检查接种水。如果必要，还应检查分析人员的技术。

本试验同试验样品同时进行。

# 8 结果的表示

8.1 被测定溶液若满足以下条件，则能获得可靠的测定结果。

培养5天后：

$$
\text {剩余} \mathrm {D O} > 1 \mathrm {m g} / \mathrm {L} ；
$$

$$
\text {消耗} \mathbf {D O} > 2 \mathbf {m g} / \mathbf {L} 。
$$

若不能满足以上条件，一般应舍掉该组结果。

8.2 五日生化需氧量 $\mathbf{\Phi}^{\star}$ （ $\mathbf{B}\mathbf{O}\mathbf{D}_{5}$ ）以每升消耗氧的毫克数表示，由下式算出：

$$
\mathrm {B O D} _ {3} = \left[ \left(c _ {1} - c _ {2}\right) - \frac {V _ {1} - V _ {e}}{V _ {1}} \left(c _ {3} - c _ {4}\right) \right] \frac {V _ {1}}{V _ {e}}
$$

式中： $c_{1}$ ——在初始计时时一种试验水样的溶解氧浓度， $\mathbf{mg} / \mathbf{L}$ ；

$c_{2}$ ——培养5天时同一种水样的溶解氧浓度， $\mathbf{mg} / \mathbf{L}$ ；

$c_{3}$ ——在初始计时时空白溶液的溶解氧浓度， $\mathbf{mg} / \mathbf{L}$

$c_{4}$ ——培养5天时空白溶液的溶解氧浓度， $\mathbf{mg} / \mathbf{L}$

$V_{\mathrm{e}}$ ——制备该试验水样用去的样品体积， $\mathbf{m}!$ ；

$V_{\mathrm{t}}$ ——该试验水样的总体积，ml。

若有几种稀释比所得数据皆符合8.1所要求的条件，则几种稀释比所得结果皆有效，以其平均值

表示检测结果。
