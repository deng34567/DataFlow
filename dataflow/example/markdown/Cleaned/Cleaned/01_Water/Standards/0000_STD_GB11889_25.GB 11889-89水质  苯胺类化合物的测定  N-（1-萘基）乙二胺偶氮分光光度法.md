# 中华人民共和国国家标准

# 水质苯胺类化合物的测定 N-(1-萘基)乙二胺偶氮分光光度法

GB 11889-89

Water quality-Determination of aniline compounds-Spectrophotometric method with N-(1-naphthyl)ethylenediamine

# 1 主题内容与适用范围

本标准规定了测定水中苯胺类化合物的N-(1-萘基)乙二胺重氮偶合比色法。

本标准适用于地面水、染料、制药等废水中芳香族伯胺类化合物的测定。

试料体积为 $25\mathrm{mL}$ ，使用光程为 $10\mathrm{mm}$ 的比色皿，本方法的最低检出浓度为含苯胺 $0.03\mathrm{mg / L}$ ，测定上限浓度为 $1.6\mathrm{mg / L}$ 。

在酸性条件下测定，苯酚含量高于 $200\mathrm{mg / L}$ 时，对本方法有正干扰。

# 2 原理

苯胺类化合物在酸性条件下 $(\mathrm{pH}1.5\sim 2.0)$ 与亚硝酸盐重氮化，再与N-(1-萘基)乙二胺盐酸盐偶合，生成紫红色染料，进行分光光度法测定，测量波长为 $545~\mathrm{nm}$ 。

# 3 试剂

分析中只使用公认的分析纯试剂和蒸馏水或纯度与之相当的水。

3.1 蒸馏水。
3.2 硫酸氢钾 $\mathrm{(KHSO_4)}$
3.3 无水碳酸钠 $\mathrm{(Na_2CO_3)}$ 。
3.4 亚硝酸钠 $\left(\mathrm{NaNO}_{2}\right)$ , $50 \mathrm{~g} / \mathrm{L}$ : 称取 $5 \mathrm{~g}$ 亚硝酸钠, 溶于少量水中, 稀释至 $100 \mathrm{~mL}$ (应配少量, 贮于棕色瓶中, 置冰箱内保存)。
3.5 氨基磺酸铵 $\left(\mathrm{NH}_{4} \mathrm{SO}_{3} \mathrm{NH}_{2}\right)$ , $25 \mathrm{~g} / \mathrm{L}$ : 称取 $2.5 \mathrm{~g}$ 氨基磺酸铵, 溶于少量水中, 稀释至 $100 \mathrm{~mL}$ (贮于棕色瓶中, 置冰箱内保存)。
3.6 N-(1-萘基)乙二胺盐酸盐， $20\mathrm{g/L}$ ：称取 $2\mathrm{gN}$ -(1-萘基)乙二胺盐酸盐，溶于水中，稀释至 $100\mathrm{mL}$ （详见附录A）。
3.7 硫酸标准溶液，浓度 $c\left(\frac{1}{2}\mathrm{H}_2\mathrm{SO}_4\right) = 0.05\mathrm{mol} / \mathrm{L}$ 。
3.8 精密 $\mathbf{pH}$ 试纸 $0.5\sim 5.0$
3.9 苯胺 $\left(\mathrm{C}_{6} \mathrm{H}_{5} \mathrm{NH}_{2}\right)$ 标准贮备液：于 $25 \mathrm{~mL}$ 容量瓶中加入 $0.05 \mathrm{~mol} / \mathrm{L}$ 硫酸溶液(3.7) $10 \mathrm{~mL}$ ，称量（称准至 $0.0001 \mathrm{~g}$ ），加入 $3 \sim 5$ 滴苯胺试剂，再称量，用 $0.05 \mathrm{~mol} / \mathrm{L}$ 硫酸溶液(3.7)稀释至标线，摇匀。计算出每毫升溶液中所含苯胺的量，此为贮备液，置冰箱内保存（可用两个月）。
3.10 苯胺标准使用溶液：将标准贮备液(3.9)用 $0.05 \mathrm{~mol} / \mathrm{L}$ 硫酸溶液(3.7)稀释成浓度为 $1.00 \mathrm{~mL}$ 溶液含苯胺 $10.0 \mu \mathrm{g}$ 的标准使用溶液(临用时配)。

注：如果苯胺试剂为无色透明液，可直接称量配制。若试剂颜色发黄，应重新蒸馏或标定苯胺含量后使用（详见GB691《化学试剂苯胺》）。

# 4 仪器

4.1 分光光度计：能在波长 $545\mathrm{nm}$ 处操作，配有光程为 $10\mathrm{mm}$ 的比色皿。
4.2 $25\mathrm{mL}$ 具塞刻度试管。

# 5 采样与样品

# 5.1 采样

采集 $500\mathrm{mL}$ 水样于硬质玻璃瓶中(保存不得超过 $24\mathrm{h}$ )，若取样后不能及时进行测定，需置 $4^{\circ}\mathrm{C}$ 下保存(不得超过两周)。

# 5.2 试料制备

将水样(5.1)用经水冲洗过的中速滤纸过滤，弃去初滤液 $20~\mathrm{mL}$ ，用硫酸氢钾(3.2)或无水碳酸钠(3.3)调节 $\mathsf{pH}$ 值为6，作为试料。

注：若水样颜色深，可用聚己内酰胺粉末脱色(6.4.1)。颜色不深的水样可不脱色，而以样品溶液（不加显色剂）为参比溶液。

# 6 分析步骤

# 6.1 校准曲线的绘制

取7个 $25\mathrm{mL}$ 具塞刻度试管(4.2)，分别加入苯胺标准使用溶液(3.10)0.0,0.25,0.50,1.00,2.00,3.00,4.00mL，各加水(3.1)至 $10\mathrm{mL}$ 。然后按照测定的步骤(6.2)进行操作。

以测得的吸光度减去试剂空白试验(零浓度)的吸光度，和对应的苯胺含量绘制校准曲线。

# 6.2 测定

吸取试料(5.2)(含苯胺 $0.5\sim 30\mu \mathrm{g}$ )于 $25\mathrm{mL}$ 具塞刻度试管中，加水(3.1)稀释至 $10\mathrm{mL}$ ，加硫酸氢钾(3.2) $50\mathrm{mg}$ ，摇匀(可预先取另一份相同体积的该水样，用精密 $\mathsf{pH}$ 试纸(3.8)控制其 $\mathsf{pH}$ 值为 $1.5\sim$ 2.0为参考值)。加1滴 $5\%$ 亚硝酸钠溶液(3.4)，摇匀，放置 $3\mathrm{min}$ ，加入氨基磺酸铵溶液(3.5) $0.5\mathrm{mL}$ ，充分振荡后，放置 $3\mathrm{min}$ ，待气泡除尽(以消除过量的亚硝酸钠对测定的影响)。加入N-(1-萘基)乙二胺盐酸盐溶液(3.6) $1.0\mathrm{mL}$ ，用水稀释至 $25\mathrm{mL}$ ，摇匀，放置 $30\mathrm{min}$ ，于 $545\mathrm{nm}$ 波长处，用 $10\mathrm{mm}$ 比色皿，以水为参比测量吸光度。以试料的吸光度减去空白试验(6.3)的吸光度(试料和校准曲线发色时间一致即可)，由校准曲线(6.1)上查出相应的苯胺含量。

# 6.3 空白试验

按6.2进行空白试验，用水(3.1)代替试料，并加入与测定时相同体积的试剂。

# 6.4 去干扰试验

# 6.4.1 脱色

污染严重或颜色深的水样，可取水样于比色管中，用硫酸氢钾(3.2)或无水碳酸钠(3.3)调节 $\mathsf{pH}$ 值为 $1.5\sim 2.0$ ，加水样体积一半的聚己内酰胺粉末，加塞摇 $1\sim 2\mathrm{min}$ ，放置后再摇几次，用中速滤纸过滤，取滤液进行测定。

# 6.4.2 补偿法

对于颜色较浅(或深色时取少量)的水样采用过滤后不加N-(1-萘基)乙二胺溶液(3.6)，其余则加入与测定时相同体积的试剂，以此溶液作参比，消除试料原有色度的影响。

# 7 结果的表示

苯胺含量 $c(\mathrm{mg / L})$ ，按下式计算：

$$
c = \frac {m}{V}
$$

式中： $m$ ——由校准曲线查得的试料中含苯胺量， $\mu \mathrm{g}$

$V$ ——试料的体积， $\mathrm{mL}$ 。

# 8 精密度

经9个实验室测定苯胺统一标样(制药废水1986.7)按6.2步骤测定结果如下：

# 附录A

# 本标准一般说明

（参考件）

A1 显色剂N-(1-萘基)乙二胺盐酸盐，三级品色深，配制时在水浴上温热至清亮时全部溶解，过滤后贮于棕色瓶中，冰箱保存，不易多配，当溶液浑浊时应重新配制。
A2 温度对反应的影响比较大, 最佳温度在 $22 \sim 30^{\circ} \mathrm{C}$ , 高于或低于此温度范围可在恒温水浴中发色或采用同时制作标准曲线的办法 (在环境温度多变时, 最好采用标样加入法), 以消除温度影响。
A3文献中指出含酚量高于 $200\mathrm{mg / L}$ 时对本方法有正干扰。为消除干扰可将废水样进行预蒸馏，然后按分析步骤(6.2)进行测定。
A4 确定了酸度条件为 $c\left(\frac{1}{2}\mathrm{H}_2\mathrm{SO}_4\right) = 0.005\sim 0.05\mathrm{mol / L}$ 硫酸介质发色时间为 $30\mathrm{min}$ 。
A5 吸收波长: 工业废水中苯胺类化合物相当复杂, 苯胺的最大吸收波长为 $556 \mathrm{~nm}$ , 而其他苯胺类化合物如: 对硝基苯胺吸收波长为 $545 \mathrm{~nm}$ , 邻氯对硝基苯胺吸收波长为 $530 \mathrm{~nm}, 2, 4$ -二硝基苯胺吸收波长为 $520 \mathrm{~nm}$ , 综合考虑废水中苯胺类化合物的特点后, 确定吸收波长为 $545 \mathrm{~nm}$ 。
A6 采样的说明：关于(5.1)中指出采样存放不得超过两周，主要考虑到染料废水成分复杂，变化很大，通过1983年北京市组织的对染料厂废水中苯胺类的验证数据：


| 苯胺类测定值，mg/L | 1#、3#、4#、5#、6#实验室同时进行 | 2#实验室放置两周后进行 |
| ----------- | --------------------- | ------------ |
| 苯胺类测定值，mg/L | 平均值=0.82              | 平均值=0.23     |


由此可知水样贮存时间不可过长。
