# 水质铵的测定

# 纳氏试剂比色法

UDC 614.777

:543.432

GB 7479-87

Water quality-Determination of ammonium-

Nessler's reagent colorimetric method

# 1 适用范围

1.1 本标准适用于生活饮用水、地面水和废水。
1.2 样品中含有悬浮物、余氯、钙镁等金属离子、硫化物和有机物时，会产生干扰，含有此类物质时，要作适当的预处理，以消除对测定的影响。

# 1.3 范围

最大试份体积为 $50\mathrm{ml}$ 时，铵氮浓度 $c_{\mathrm{N}}$ 可达 $2\mathrm{mg / L}$ 。

# 1.4 最低检出浓度

# 1.4.1 目视法

试份体积为 $50\mathrm{ml}$ 时，最低检出浓度为 $0.02\mathrm{mg / L}$ 。

# 1.4.2 分光光度法

试份体积为 $50\mathrm{ml}$ ，使用光程长为 $10\mathrm{mm}$ 比色皿时，最低检出浓度为 $0.05\mathrm{mg / L}$ 。

# 1.5 灵敏度

使用 $50\mathrm{m}1$ 试份，光程长为 $10\mathrm{mm}$ 比色皿， $c_{\mathbf{N}} = 1.0\mathrm{mg / L}$ ，给出的吸光度约为0.2个单位。

# 2 原理

以游离态的氨或铵离子等形式存在的铵氮与纳氏试剂反应生成黄棕色络合物，该络合物的色度与铵氮的含量成正比，可用目视比色或者用分光光度法测定。

# 3 试剂

分析中只使用公认的分析纯试剂和按3.1制备的水。

# 3.1 水：无氨，按下述方法之一制备。

# 3.1.1 离子交换法

将蒸馏水通过一个强酸性阳离子交换树脂（氢型）柱，流出液收集在带有磨口玻璃塞的玻璃瓶中。每升流出液中加入 $10\mathrm{g}$ 同类树脂，以利保存。

# 3.1.2 蒸馏法

在 $1000\mathrm{ml}$ 蒸馏水中，加入 $0.1\mathrm{ml}$ 硫酸（ $\rho = 1.84\mathrm{g / ml}$ ），并在全玻璃蒸馏器中重蒸馏。弃去前 $50\mathrm{ml}$ 馏出液，然后将约 $800\mathrm{ml}$ 馏出液收集在带有磨口玻璃塞的玻璃瓶中。每升收集的馏出液中加入 $10\mathrm{g}$ 强酸性阳离子交换树脂（氢型），以利保存。

# 3.2 纳氏试剂。

3.2.1 二氯化汞-碘化钾-氢氧化钾（ $\mathrm{HgCl}_2 - \mathrm{KI} - \mathrm{KOH}$ ）。

称取 $15\mathrm{g}$ 氢氧化钾（KOH），溶于 $50\mathrm{ml}$ 水中，冷至室温。

称取 $5\mathrm{g}$ 碘化钾（KI），溶于 $10\mathrm{ml}$ 水中，在搅拌下，将 $2.5\mathrm{g}$ 二氯化汞（ $\mathrm{HgCl}_2$ ）粉末分次少量加

人于碘化钾溶液中，直到溶液呈深黄色或出现微米红色沉淀溶解缓慢时，充分搅拌混和，并改为滴加二氯化汞饱和溶液，当出现少量朱红色沉淀不再溶解时，停止滴加。

在搅拌下, 将冷的氢氧化钾溶液缓慢地加入到上述二氯化汞和碘化钾的混合液中, 并稀释至 $100 \mathrm{ml}$ , 于暗处静置 $24 \mathrm{~h}$ , 倾出上清液, 贮于棕色瓶中, 用橡皮塞塞紧。存放暗处, 此试剂至少可稳定一个月。

# 3.2.2 碘化汞-碘化钾-氢氧化钠（HgI2-KI-NaOH）。

称取 $16\mathrm{g}$ 氢氧化钠（NaOH），溶于 $50\mathrm{ml}$ 水中，冷至室温。

称取 $7\mathrm{g}$ 碘化钾（KI）和 $10\mathrm{g}$ 碘化汞（ $\mathrm{HgI}_2$ ），溶于水中，然后将此溶液在搅拌下，缓慢地加入到氢氧化钠溶液中，并稀释至 $100\mathrm{ml}$ 。贮于棕色瓶内，用橡皮塞塞紧。于暗处存放，有效期可达一年。

# 3.3 酒石酸钾钠溶液

称取 $50\mathrm{g}$ 酒石酸钾钠（ $\mathrm{KNaC_4H_6O_6\cdot 4H_2O}$ ），溶于 $100\mathrm{ml}$ 水中，加热煮沸，以驱除氨，充分冷却后稀释至 $100\mathrm{ml}$ 。

# 3.4铵氮标准溶液： $c_{\mathbf{N}} = 1000\mu \mathrm{g / ml}$

称取 $3.819 \pm 0.004 \mathrm{~g}$ 氯化铵（ $\mathrm{NH}_{4} \mathrm{Cl}$ ，在 $100 \sim 105^{\circ} \mathrm{C}$ 下燥 $2 \mathrm{~h}$ ），溶于水中，移入 $1000 \mathrm{ml}$ 容量瓶中，稀释至刻度。

# 3.5 铵氮标准溶液： $c_{\mathrm{N}} = 10\mu \mathrm{g / ml}$ 。

吸取 $10.00\mathrm{ml}$ 铵氮标准溶液（3.4）于 $1000\mathrm{ml}$ 容量瓶中，稀释至刻度。临用前配制。

# 3.6 $10\% (m / V)$ 硫酸锌溶液。

称取 $10\mathrm{g}$ 硫酸锌 $(\mathrm{ZnSO}_4\cdot 7\mathrm{H}_2\mathrm{O})$ ，溶于水中，稀释至 $100\mathrm{ml}$ 。

# 3.7 $25\% (m / V)$ 氢氧化钠溶液。

称取 $25\mathrm{g}$ 氢氧化钠（ $\mathrm{NaOH}$ ），溶于水中，冷至室温，稀释至 $100\mathrm{ml}$ 。

# 3.8 $0.35\% (m / V)$ 硫代硫酸钠溶液。

称取3.5g硫代硫酸钠（ $\mathbf{Na}_2\mathbf{S}_2\mathbf{O}_3$ 或 $\mathbf{Na}_2\mathbf{S}_2\mathbf{O}_3 \cdot 5\mathbf{H}_2\mathbf{O}$ ），溶于水，再稀释至 $1000\mathrm{ml}$ 。

# 3.9 淀粉-碘化钾试纸。

称取 $1.5 \mathrm{~g}$ 可溶性淀粉于烧杯中，用少量水调成糊状，加入 $200 \mathrm{ml}$ 沸水，搅拌混匀放冷。加 $0.5 \mathrm{~g}$ 碘化钾（KI）和 $0.5 \mathrm{~g}$ 碳酸钠（ $\mathrm{Na}_{2} \mathrm{CO}_{3}$ ），用水稀释至 $250 \mathrm{ml}$ 。将滤纸条浸渍后，取出凉干，装棕色瓶中密封保存。

# 4 仪器

常用实验室仪器及分光光度计。

# 5 采样及样品

# 5.1 实验室样品

实验室样品采集在聚乙烯瓶或玻璃瓶内，应尽快分析，不然要在 $2 \sim 5^{\circ} \mathrm{C}$ 下存放，用硫酸（ $\rho = 1.84 \mathrm{~g} / \mathrm{ml}$ ）将样品酸化至 $\mathrm{pH} < 2$ 亦有利于保存，但酸化样品会吸收空气中的氨而被污染，应注意防止。

# 5.2 试份

清洁样品可直接从中取 $50\mathrm{m}1$ 作为试份。

含有悬浮物或色度深的样品在预处理后（6.1），再从中取 $50\mathrm{ml}$ （或取适量，稀释至 $50\mathrm{ml}$ ）作为试份。

# 6 步骤

# 6.1 预处理

样品中含有悬浮物、余氯、钙镁等金属离子、硫化物和有机物时，对比色测定有干扰，处理方法如下：

# 6.1.1 除余氯

加入适量的硫代硫酸钠溶液（3.8），每 $0.5\mathrm{ml}$ 可除去 $0.25\mathrm{mg}$ 余氯。也可用淀粉-碘化钾试纸（3.9）检验是否除尽余氯。

# 6.1.2 凝聚沉淀

$100\mathrm{ml}$ 样品中加入 $1\mathrm{ml}$ 硫酸锌溶液（3.6）和 $0.1\sim 0.2\mathrm{ml}$ 氢氧化钠溶液（3.7），调节 $\mathsf{pH}$ 约为10.5，混匀，放置使之沉淀，倾取上清液作试份。必要时，用经水冲洗过的中速滤纸过滤，弃去初滤液 $20\mathrm{ml}$ 。

# 6.1.3 络合掩蔽

加人酒石酸钾钠溶液（3.3），可消除钙镁等金属离子的干扰。

# 6.1.4 蒸馏法

用凝聚沉淀和络合掩蔽后，样品仍浑浊和带色，则应采用蒸馏法（见附录A）。

# 6.1.5 低pH下煮沸

蒸馏中，某些有机物很可能与氨同时被馏出，对测定仍有干扰，其中有些物质（如甲醛）可在比色前于低pH下采用煮沸而除之。

# 6.2 测定

取试份于 $50\mathrm{ml}$ 比色管中，加入 $1\mathrm{ml}$ 酒石酸钾钠溶液（3.3），摇匀，再加入纳氏试剂 $1.5\mathrm{ml}$ （3.2.1）或 $1.0\mathrm{ml}$ （3.2.2），摇匀。放置 $10\mathrm{min}$ 后进行比色。若色度很低采用目视比色，一般在波长 $420\mathrm{nm}$ 下，用光程长 $20\mathrm{mm}$ 比色皿，以水作参比，测定试份的吸光度。

# 6.3 空白试验

用 $50\mathrm{m}1$ 水代替试份，按6.2进行处理。

注：此步骤只用于分光光度法。

# 6.4 校准

# 6.4.1 目视比色法

在6个 $50\mathrm{m}1$ 比色管中，分别加入0、0.10、0.30、0.50、0.70、1.00m1铵氮标准溶液（3.5），再加水至刻度，按6.2显色后进行目视比色。

# 6.4.2 分光光度法

在8个 $50\mathrm{m}1$ 比色管中，分别加入0、0.50、1.00、2.00、3.00、5.00、7.00、10.00m1铵氮标准溶液（3.5），再加水至刻度。按6.2显色后进行分光光度测定。

将上面系列标准溶液测得的吸光度扣除试剂空白（零浓度）的吸光度，便得到校正吸光度，以校正吸光度为纵坐标，铵氮质量 $m_{\mathrm{N}}$ 为横坐标，绘制校准曲线。

# 7 结果的表示

# 7.1 目视比色法

将试份的色度与标准溶液（6.4.1）的色度比较后，得到试份中的铵氮质量 $m_{\mathbb{N}}$ ，除以试份的体积 $V$ ，便可得到试份的铵氮含量 $c_{\mathbb{N}}(\mathbf{mg} / \mathbf{L})$ 。

# 7.2 分光光度法

# 7.2.1 计算方法

试份中铵氮吸光度 $A_{\mathrm{r}}$ 用式（1）计算：

$$
A _ {r} = A _ {s} - A _ {b} \dots \tag {1}
$$

式中： $A_{s}$ ——试份测定（6.2）吸光度；

$A_{b}$ ——空白试验（6.3）吸光度。

铵氮含量 $c_{\mathrm{N}}$ （ $\mathrm{mg / L}$ ）用式（2）计算：

$$
c _ {N} = \frac {m _ {N}}{V} \dots \tag {2}
$$

式中： $m_{\mathrm{N}}$ ——铵氮质量， $\mu \mathrm{g}$ ，由 $A_{\mathrm{r}}$ 值和相应比色皿光程的校准曲线（6.4.2）确定。

$\pmb{V}$ ——试份体积， $\mathbf{m}1$ 。

# 附录A

# 样品的蒸馏预处理

（补充件）

# A.1 试剂

所用的试剂为公认的分析纯试剂，所用的水应为无氨水。

A.1.1 硼酸 $(\mathbf{H}_3\mathbf{BO}_3)$ ： $20\mathrm{g / L}$ 溶液。
A.1.2 氢氧化钠（NaOH）： $40\mathrm{g} / \mathrm{L}$ 溶液。
A.1.3 轻质氧化镁（MgO）：不含碳酸盐，在 $500^{\circ} \mathrm{C}$ 下加热氧化镁，以除去碳酸盐。
A.1.4 盐酸（HCl， $\rho = 1.18\mathrm{g / ml}$ ）： $1\mathrm{mol / L}$ 溶液。
A.1.5 防沫剂：如石蜡碎片。
A.1.6 溴百里酚蓝（bromthymol blue）： $0.5\mathrm{g/L}$ 指示液。

# A.2 仪器

常用实验室仪器及：

蒸馏器：由一个 $500 \sim 800 \mathrm{ml}$ 的蒸馏烧瓶及防喷头和一个垂直放置的冷凝管组装而成。冷凝管末端可连接一适当长度的滴管，使出口尖端浸入吸收液液面下约 $2 \mathrm{~cm}$ 。

蒸馏器清洗：向蒸馏烧瓶中加入 $350\mathrm{ml}$ 水，加几粒防爆沸颗粒，装好仪器，蒸馏到至少收集 $100\mathrm{ml}$ 水时，将馏出液及瓶内残留液弃去。

# A.3 蒸馏操作

将 $50\mathrm{ml}$ 硼酸溶液（A.1.1）移入接收瓶内，确保冷凝管出口在硼酸溶液液面之下。

量取 $300\mathrm{ml}$ 样品，移入蒸馏烧瓶中，加几滴溴百里酚蓝指示液（A.1.6），必要时，用氢氧化钠溶液（A.1.2）或盐酸溶液（A.1.4）调整 $\mathsf{pH}$ 至6.0（指示剂呈黄色）\~7.4（指示剂呈蓝色）之间，加水使总体积约为 $350\mathrm{ml}$ 。向蒸馏烧瓶中加入0.25g轻质氧化镁（A.1.3）及少许防爆沸颗粒[对一些工业废水样品，必要时，加入防沫剂（A.1.5）]，立即将蒸馏烧瓶与冷凝管连接好。

加热蒸馏，使馏出液速率约为 $10\mathrm{ml} / \mathrm{min}$ ，待馏出液约为 $200\mathrm{ml}$ 时，停止蒸馏。

将馏出液定容至原体积（300ml）。

注：分取试份供纳氏试剂比色测定时，应先用氢氧化钠溶液（A.1.2）调节至中性。
