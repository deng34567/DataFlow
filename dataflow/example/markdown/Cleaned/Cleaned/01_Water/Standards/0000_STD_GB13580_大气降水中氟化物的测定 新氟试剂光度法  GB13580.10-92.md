# 中华人民共和国国家标准

# 大气降水中氟化物的测定新氟试剂光度法

GB 13580.10-92

Determination of fluoride in the wet precipitation Fluor reagent spectrophotometry

# 1 主题内容与适用范围

1.1 本标准规定了测定大气降水中氟化物的氟试剂光度法。
1.2 本标准适用于大气降水样品中氟化物的测定。
1.3 本标准最低检出浓度为 $0.05\mathrm{mg / L}$ ，测定范围为 $0.06\sim 1.5\mathrm{mg / L}$ 。

# 2 引用标准

GB 13580.2 大气降水样品的收集与保存

# 3 原理

在 $\mathrm{pH}4.1$ 的乙酸盐缓冲介质中，氟离子与氟试剂及硝酸铜反应生成蓝色的三元络合物，络合物颜色的深度与氟离子浓度成正比，于波长 $620\mathrm{nm}$ 处测定吸光度。

# 4 试剂

4.1 氟化物标准贮备液： $1000\mu \mathrm{g} / \mathrm{mL}$ 。准确称取 $0.2210\mathrm{g}$ 氟化钠（ $\mathrm{NaF}$ ，在 $105\%$ 烘 $2\mathrm{h}$ ），溶于水，定容至 $1000\mathrm{mL}$ 。贮存于白色塑料瓶中。
4.2 氟化物标准使用液: $10 \mu \mathrm{g} / \mathrm{mL}$ 。准确吸取 $10.0 \mathrm{~mL}$ 氟化物标准贮备液(4.1)于 $100 \mathrm{~mL}$ 容量瓶中, 稀释刻度。
4.3 氢氧化钠溶液： $1\mathrm{mol} / \mathrm{L}$ 。称取 $4\mathrm{g}$ 氢氧化钠(NaOH)溶于水，稀释至 $100\mathrm{mL}$ 。
4.4盐酸溶液： $1\mathrm{mol} / \mathrm{L}$ 。取 $8.4\mathrm{mL}$ 浓盐酸用水稀释到 $100\mathrm{mL}$ 。
4.5 氟试剂溶液: $0.001 \mathrm{~mol} / \mathrm{L}$ 。称取 $0.193 \mathrm{~g}$ 氟试剂（茜素-3-甲胺-N-二乙酸），用少量水润湿，滴加氢氧化钠溶液（4.3），使之溶解，再加 $0.13 \mathrm{~g}$ 乙酸钠 $\left(\mathrm{NaC}_{2} \mathrm{H}_{3} \mathrm{O}_{2} \cdot 3 \mathrm{H}_{2} \mathrm{O}\right)$ ，用盐酸溶液（4.4）调 $\mathrm{pH} 5.0$ ，用水稀释至 $500 \mathrm{~mL}$ 。
4.6硝酸镧 $\mathrm{[La(NO_3)_3\cdot 6H_2O]}$ 溶液： $0.001\mathrm{mol / L}$ 。用少量盐酸溶液(4.4)溶解，用乙酸钠 $\mathrm{(NaC_2H_3O_2}$ $3\mathrm{H}_{2}\mathrm{O})$ 调 $\mathsf{pH}$ 为4.1，用水稀释到 $1000\mathrm{mL}$ 。
4.7 缓冲溶液：称 $35\mathrm{g}$ 无水乙酸钠 $\mathrm{(NaC_2H_3O_2)}$ ，溶于 $800~\mathrm{mL}$ 水中，加冰乙酸 $75~\mathrm{mL}$ ，用水稀释1000 $\mathrm{mL}$ 。用酸度计校正 $\mathsf{pH}$ 值。
4.8 混合显色剂：氟试剂溶液(4.5)，pH4.1的缓冲溶液(4.7)，丙酮，硝酸镧溶液(4.6)按体积为3:1:3:3混合即得(使用时现配)。

# 5 仪器

5.1 分光光度计。
5.2 酸度计。
5.3 $25\mathrm{mL}$ 具塞比色管。

# 6样品采集与保存

按GB13580.2执行。

# 7 步骤

7.1 校准曲线的绘制：取 $25\mathrm{mL}$ 比色管6支，分别加入氟化物标准使用液(4.2)0,0.20,0.30,0.50,1.00,2.00mL，各加水至 $10.00\mathrm{mL}$ ，摇匀。再向各管加 $10\mathrm{mL}$ 混合显色剂(4.8)，用水稀释至刻度，摇匀。放置 $0.5\mathrm{h}$ ，于波长 $620\mathrm{nm}$ 处，用 $30\mathrm{mm}$ 吸收池，以氟化物0浓度的空白试验溶液作参比，测量吸光度。以吸光度对氟化物含量作图，绘制校准曲线。
7.2 样品测定：根据降水中氟化物的含量，吸取 $5.00 \sim 10.0 \mathrm{~mL}$ 样品于 $25 \mathrm{~mL}$ 比色管中，加水至 $10 \mathrm{~mL}$ ，摇匀。以下按绘制校准曲线的步骤(7.1)进行操作，由测得的吸光度从校准曲线上查得氟化物的含量。

# 8 分析结果的表述

降水中氟化物（按 $\mathbf{F}^{-}$ 计)的浓度以 $\mathrm{mg / L}$ 表示，按下式计算：

$$
C = \frac {M}{V}
$$

式中： $C$ ——样品中氟化物的浓度， $\mathrm{mg / L}$

$M$ ——从校准曲线上查得氟化物的含量, $\mu \mathrm{g}$ ;

$V$ ——取样体积， $\mathrm{mL}$ 。

# 9 精密度和准确度

5个实验室对含 $\mathbf{F}^{-}0.20\mathrm{mg / L},\mathrm{Cl}^{-}1.00\mathrm{mg / L},\mathrm{SO}_{4}^{2 - }6.00\mathrm{mg / L},\mathrm{NO}_{3}^{-}1.20\mathrm{mg / L},\mathrm{K}^{+}$ $2.98\mathrm{mg / L},\mathrm{Na}^{+}0.44\mathrm{mg / L}$ 的合成水样进行测定，测定结果 $\mathbf{F}^{-}$ 的相对标准偏差为 $1.3\%$ ，相对误差为 $-0.5\%$ 。
