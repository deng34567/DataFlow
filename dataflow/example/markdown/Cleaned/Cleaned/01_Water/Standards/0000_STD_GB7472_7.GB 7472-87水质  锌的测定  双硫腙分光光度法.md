# 水质锌的测定

# 双硫腙分光光度法

UDC 614.777:543

.42:546.47

GB 7472-87

Water quality-Determination of zinc-Spectrophotometric method with dithizone

# 1 适用范围

1.1 测定物质：本标准规定了用双硫腙分光光度法测定水中的锌。
1.2 样品类型：本方法适用于测定天然水和某些废水中微量锌。有关干扰问题见附录A。
1.3 范围：本方法适用于测定锌浓度在 $5\sim 50\mu \mathrm{g} / \mathrm{L}$ 的水样。
1.4检出限：当使用光程长 $20\mathrm{mm}$ 比色皿，试份体积为 $100\mathrm{ml}$ 时，检出限为 $5\mu \mathrm{g} / \mathrm{L}$ 。
1.5 灵敏度：本方法用四氯化碳萃取，在最大吸光波长 $535\mathrm{nm}$ 测量时，其摩尔吸光度约为 $9.3 \times 10^{4}\mathrm{L} / \mathrm{mol} \cdot \mathrm{cm}$ 。

# 2 定义

本标准规定水样经酸消解处理后，测定水样中总锌量。

# 3 原理

在 $\mathsf{pH}$ 为 $4.0\sim 5.5$ 的乙酸盐缓冲介质中，锌离子与双硫腙形成红色螯合物，用四氯化碳萃取后进行分光光度测定。水样中存在少量铅、铜、汞、镉、钴、铋、镍、金、钯、银、亚锡等金属离子时，对锌的测定有干扰，但可用硫代硫酸钠作掩蔽剂和控制 $\mathsf{pH}$ 值而予以消除，其反应为：

$$
Z n ^ {2 + 2} S = \begin{array}{c} H C _ {6} H _ {5} \\ | | \\ N - N - H \\ \hline \end{array} \longrightarrow S = \begin{array}{c} H C _ {6} H _ {5} \\ | | \\ N - N \\ \hline \end{array} \begin{array}{c} C _ {6} H _ {5} \\ | | \\ N = N \\ \hline \end{array} \begin{array}{c} C _ {6} H _ {5} \\ | | \\ N = N \\ \hline \end{array} \begin{array}{c} C = S + 2 H \\ | | \\ N = N \\ \hline \end{array}
$$

# 4 试剂

本标准所用试剂除另有说明外，均为分析纯试剂，实验中均用不含锌的水。

无锌水：将普通蒸馏水通过阴阳离子交换柱以除去水中锌。

4.1 四氯化碳（CCl4）。
4.2 高氯酸 $\left(\mathrm{HClO}_4\right)$ ： $\rho = 1.75\mathrm{g / ml}$ 。
4.3 盐酸（HCl）： $\rho = 1.18\mathrm{g / ml}$ 。
4.3.1 盐酸： $6\mathrm{mol} / \mathrm{L}$ 溶液。

取 $500\mathrm{m}1$ 盐酸（4.3）用水稀释至 $1000\mathrm{ml}$ 。

4.3.2 盐酸： $2\mathrm{mol} / \mathrm{L}$ 溶液。

取 $100\mathrm{ml}$ 盐酸（4.3）用水稀释到 $600\mathrm{ml}$ 。

4.3.3 盐酸： $0.02\mathrm{mol} / \mathrm{L}$ 溶液。

取 $10\mathrm{ml}$ 盐酸（4.3.2）溶液用水稀释到 $1000\mathrm{ml}$ 。

4.4 乙酸（CH3COOH）。

4.5 氨水 $(\mathrm{NH}_3, \mathrm{H}_2\mathrm{O})$ ： $\rho = 0.90\mathrm{g / ml}$ 。

4.5.1 氨水： $1 + 100$ 溶液。

取 $10\mathrm{ml}$ 氨水（4.5）用水稀释至 $1000\mathrm{ml}$ 。

4.6 硝酸 $\left(\mathrm{HNO}_{3}\right): \rho = 1.4 \mathrm{~g} / \mathrm{ml}$ 。

4.6.1 硝酸： $2\% (V / V)$ 溶液。

取 $20\mathrm{ml}$ 硝酸（4.6）用水稀释到 $1000\mathrm{ml}$ 。

4.6.2 硝酸： $0.2\% (V / V)$ 溶液。

取 $2\mathrm{ml}$ 硝酸（4.6）用水稀释至 $1000\mathrm{ml}$ 。

4.7 乙酸钠缓冲溶液。

将68g三水乙酸钠（CH3COONa·3H2O）溶于水中，并稀释至250ml。另取1份乙酸（4.4）与7份水混合。将上述两种溶液按等体积混合，混合液再用双硫腙四氯化碳溶液（4.9）重复萃取数次，直到最后的萃取液呈绿色，然后再用四氯化碳（4.1）萃取以除去过量的双硫腙。

4.8 硫代硫酸钠溶液。

将 $25\mathrm{g}$ 五水硫代硫酸钠（ $\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3\cdot 5\mathrm{H}_2\mathrm{O}$ ）溶于 $100\mathrm{ml}$ 水中，每次用 $10\mathrm{ml}$ 双硫腙四氯化碳溶液（4.9）萃取，直到双硫腙溶液呈绿色为止，然后再用四氯化碳（4.1）萃取以除去多余的双硫腙。

4.9 双硫腙： $0.1\% (m / V)$ 四氯化碳贮备溶液。

称取 $0.25\mathrm{g}$ 双硫腙（ $\mathbf{C}_{13}\mathbf{H}_{12}\mathbf{N}_{4}\mathbf{S}$ ）溶于 $250\mathrm{ml}$ 四氯化碳（4.1），贮于棕色瓶中，放置在冰箱内。如双硫腙试剂不纯，可按下述步骤提纯：

称取 $0.25\mathrm{g}$ 双硫腙溶于 $100\mathrm{ml}$ 四氯化碳中滤去不溶物，滤液置分液漏斗中，每次用 $20\mathrm{ml}$ 氨水（4.5.1）提取五次，此时双硫腙进入水层，合并水层，然后用盐酸（4.3.1）中和。再用 $250\mathrm{ml}$ 四氯化碳（4.1）分三次提取，合并四氯化碳层。将此双硫腙四氯化碳溶液流放入棕色瓶中，保存于冰箱内备用。

4.10 双硫腙： $0.01\% (m / V)$ 四氯化碳溶液。

临用前将双硫腙溶液（4.9）用四氯化碳（4.1）稀释10倍。

4.11 双硫腙： $0.004\% (m / V)$ 四氯化碳溶液。

取 $40\mathrm{ml}$ 双硫腙四氯化碳溶液（4.10），用四氯化碳（4.1）稀释到 $100\mathrm{ml}$ 。当天配制。

4.12 双硫腙： $0.0004\%$ （ $m / V$ ）四氯化碳溶液。

取 $10\mathrm{ml}$ 双硫腙四氯化碳溶液（4.11），用四氯化碳（4.1）稀释至 $100\mathrm{ml}$ （此溶液的透光度在500 nm 波长处用 $10\mathrm{mm}$ 比色皿测量时，应为 $70\%$ ）。当天配制。

4.13 柠檬酸钠溶液。

将10克二水柠檬酸钠 $\left(\mathrm{C}_{6} \mathrm{H}_{5} \mathrm{O}_{7} \mathrm{Na}_{2} \cdot 2 \mathrm{H}_{2} \mathrm{O}\right)$ 溶解在 $90 \mathrm{ml}$ 水中，按上面介绍方法（4.7）用双硫腙四氯化碳萃取纯化。此试剂用于玻璃器皿的最后洗涤。

4.14 锌标准贮备溶液。

称取 $0.1000\mathrm{g}$ 锌粒（纯度 $99.9\%$ ）溶于 $5\mathrm{ml}$ 盐酸（4.3.2）中，移入 $1000\mathrm{ml}$ 容量瓶中，用水稀释至标线。此溶液每毫升含 $100\mu \mathrm{g}$ 锌。

4.15 锌标准溶液。

取锌标准贮备溶液（4.14） $10.00\mathrm{ml}$ 置于 $1000\mathrm{ml}$ 容量瓶中，用水稀释至标线。此溶液每毫升含 $1.00\mu \mathrm{g}$ 锌。

# 5 仪器

5.1 分光光度计：光程 $10\mathrm{mm}$ 或更长的比色皿。
5.2 分液漏斗：容量为125和 $150\mathrm{ml}$ ，最好配有聚四氟乙烯活塞。
5.3 玻璃器皿：所有玻璃器皿均先后用 $1 + 1$ 硫酸和无锌水浸泡和清洗。

# 6 采样和样品

# 6.1 实验室样品

根据水样的类型提出的特殊建议进行采样，采用聚乙烯瓶采样。使用前用硝酸（4.6.1）溶液浸泡24h，然后用无锌水冲洗干净，水样采集后，每1000ml水样立即加入2.0ml硝酸（4.6）加以酸化（pH约1.5）。

# 6.2 试样

除非证明水样的消化处理是不必要的，例如：不含悬浮物的地下水和清洁地面水可直接测定。否则要按下述二种方法处理：

6.2.1 比较浑浊的地面水，每 $100\mathrm{ml}$ 水样加入 $1\mathrm{ml}$ 硝酸（4.6），置于电热板上微沸消解 $10\mathrm{min}$ ，冷却后用快速滤纸过滤，滤纸用硝酸（4.6.2）洗涤数次，然后用硝酸（4.6.2）稀释至一定体积，供测定用。
6.2.2 含悬浮物和有机质较多的地面水或废水，每 $100\mathrm{ml}$ 水样加入 $5\mathrm{ml}$ 硝酸（4.6），在电热板上加热消解到 $10\mathrm{ml}$ 左右，冷却，再加入 $5\mathrm{ml}$ 硝酸（4.6）和 $2\mathrm{ml}$ 高氯酸（4.2），继续加热消解，蒸发至近干。用硝酸（4.6.2）温热溶解残渣，冷却后，用快速滤纸过滤，滤纸用硝酸（4.6.2）洗涤数次，滤液用硝酸（4.6.2）稀释定容，供测定用。每分析一批样品要平行操作两个空白。

# 6.3 试份

如果水样中锌的含量不在测定范围内，可将试样作适当的稀释减少取试样量，如锌的含量太低，也可取较大量试样置于石英皿中进行浓缩。如果取加酸保存的试样，则要取一份试样放在石英皿中蒸发至干，以除去过量酸（注意：不要用氢氧化物中和，因为此类试剂中的含锌量往往过高），然后加无锌水，加热煮沸 $5\mathrm{min}$ 。用稀盐酸或经纯制的氨水调节试样的 $\mathsf{pH}$ 在 $2\sim 3$ 之间。最后以无锌水定容。

# 7 步骤

# 7.1 测定

# 7.1.1 显色萃取

取 $10\mathrm{ml}$ （含锌量在 $0.5\sim 5\mu \mathrm{g}$ 之间）试份（6.3），置于 $60\mathrm{ml}$ 分液漏斗中，加入 $5\mathrm{ml}$ 乙酸钠缓冲溶液（4.7）及 $1\mathrm{ml}$ 硫代硫酸钠溶液（4.8），混匀后，再加 $10.0\mathrm{ml}$ 双硫腙四氯化碳溶液（4.12），振摇四min，静置分层后，将四氯化碳层通过少许洁净脱脂棉过滤入 $20\mathrm{mm}$ 比色皿中。

# 7.1.2 吸光度的测量

立即在 $535\mathrm{nm}$ 的最大吸光波长处测量溶液的吸光度，采用合适的（如 $20\mathrm{mm}$ ）光程长的比色皿，参比皿中放入四氯化碳（4.1）（注意：第一次采用本方法时，应检验最大吸光波长，以后的测定中均使用此波长），由测量得吸光度扣去空白试验（7.2）吸光度之后，从校准曲线上查出测量锌量，然后按8.1的公式计算样品中锌的含量。

# 7.2. 空白试验

用适量（如 $10 \pm 0.5 \mathrm{ml}$ ）无锌水代替试份，按6.3和7.1的方法进行处理。

# 7.3 校准

# 7.3.1 制备一组校准溶液

向一系列 $125\mathrm{m}1$ 分液漏斗中，分别加入锌标准溶液（4.15）0、0.50、1.00、2.00、3.00、4.00、

5.00ml,各加适量无锌水补充到10ml,向各分液漏斗中加入5ml乙酸钠溶液（4.7）和1ml硫代硫酸钠溶液（4.8），混匀后备用萃取。

# 7.3.2 显色萃取

上述溶液（7.3.1）用 $10.0\mathrm{ml}$ 双硫腙四氯化碳溶液（4.12）摇动萃取 $4\mathrm{min}$ ，静置分层后，将四氯化碳层通过少许洁净脱脂棉过滤人 $20\mathrm{mm}$ 比色皿中。

# 7.3.3 吸光度的测量

立即在 $535 \mathrm{~nm}$ 的最大吸光波长处测量溶液的吸光度，采用 $20 \mathrm{~mm}$ 光程长的比色皿，用纯四氯化碳作参比。

# 7.3.4 校准曲线的绘制

从7.3.3测得的吸光度扣去试剂空白（零浓度）的吸光度后，绘制吸光度对锌量的曲线。这条校准线应为通过原点的直线。

# 7.3.5 校准次数

应定期检查校准曲线，特别是分析一批水样或每使用一批新试剂时要检查一次。

# 8 结果表示

# 8.1 计算方法

锌的浓度 $c(\mathrm{mg / L})$ 由下式计算：

$$
c = \frac {m}{V}
$$

式中： $m$ ——从校准曲线上求得锌量， $\mu \mathrm{g}$

$V$ ——用于测定的水样体积， $\mathbf{m} \mathbf{l}$ 。

# 8.2 报告结果

结果以二位有效数字表示。

# 9 精密度和准确度

46个实验室曾用本方法分析过一个合成水样，其中含锌 $650\mu \mathrm{g} / \mathrm{L}$ 。其他离子含量（以 $\mu \mathrm{g} / \mathrm{L}$ 计）为：铝500，镉50，铬110，铜470，铁300，铅70，锰120和银150。得到的相对标准偏差为 $18.2\%$ 。相对误差为 $25.9\%$ 。

# 附录A

# 干扰及其消除

（补充件）

水中存在少量铋、镉、钴、铜、金、铅、汞、镍、钯、银和亚锡等金属离子时，对本方法均有干扰，但可用硫代硫酸钠掩蔽剂和控制溶液的pH值来消除这些干扰。三价铁、余氯和其他氧化剂会使双硫腙变成棕黄色，由于锌普遍存在于环境中，而锌与双硫腙反应又非常灵敏，因此需要采取特殊措施防止污染。实验中如出现高而无规律的空白值，这种现象往往是起源于含氧化锌的玻璃、表面被污染的玻璃器皿、橡胶制品、活塞润滑剂、试剂级化学药品或蒸馏水，因此需要保留一套专供测定锌用的玻璃器皿，单独放置。

本方法只适用于一般轻度受重金属污染的废水。
