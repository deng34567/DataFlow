# 中华人民共和国国家标准

# 大气降水中钙、镁的测定

# 原子吸收分光光度法

GB 13580.13-92

Determination of calcium and magnesium in the wet precipitation

Atomic absorption spectrophotometry

# 1 主题内容与适用范围

1.1 本标准规定了测定大气降水中钙、镁的原子吸收分光光度法。
1.2 本标准适用于大气降水样品中钙、镁的测定。
1.3 本标准最低检出浓度钙为 $0.02\mathrm{mg / L}$ ，镁为 $0.0025\mathrm{mg / L}$ ，测定范围钙为 $0.2\sim 7\mathrm{mg / L}$ ，镁为 $0.02\sim 0.5\mathrm{mg / L}$ 。

# 2 引用标准

GB13580.2 大气降水样品的收集与保存

# 3 原理

火焰原子吸收分光光度法是根据某元素的基态原子对该元素的特征光谱辐射产生选择性吸收来进行测定。将降水试样喷入空气-乙炔火焰中，分别于波长 $422.7\mathrm{nm}$ 和 $285.2\mathrm{nm}$ 处测量钙、镁的吸光度，用校准曲线法进行测定。样品若有Al、Be、Ti等元素存在会产生负干扰，可加入释放剂氯化镧、硝酸镧或氯化锶予以消除。

# 4 试剂

4.1 盐酸溶液： $(1 + 1)$ 。取 $100\mathrm{mL}$ 盐酸加到 $100\mathrm{mL}$ 水中，摇匀。
4.2钙标准贮备液： $500\mu \mathrm{g / mL}$ 。准确称取0.6250g碳酸钙(180℃烘2h)于烧杯中，加20mL水悬浮，缓慢加入少量(1+1)盐酸(4.1)，小心溶解，加热驱除 $\mathrm{CO}_{2}$ ，冷却后，定容至 $500~\mathrm{mL}$ 。
4.3 镁标准贮备液： $100\mu \mathrm{g} / \mathrm{mL}$ 。准确称取 $0.1658\mathrm{g}$ 氧化镁（在 $1000^{\circ}\mathrm{C}$ 高温炉中灼烧 $1\mathrm{h}$ ，在保干器中冷却后称量）于烧杯中，用少量 $(1 + 1)$ 盐酸(4.1)溶解，移入 $1000\mathrm{mL}$ 容量瓶中，用水稀释至刻度。
4.4钙、镁混合标准使用液： $50~\mu \mathrm{g / mL}$ 钙， $5\mu \mathrm{g / mL}$ 镁。分别吸取钙、镁标准贮备液 $10.0\mathrm{mL}$ 和5.00mL于 $100\mathrm{mL}$ 容量瓶中，加 $2\mathrm{mL}$ 硝酸溶液(4.5)，用水稀释到刻度；
4.5 硝酸溶液： $(1 + 1)$ 。取 $100 \mathrm{~mL}$ 硝酸加入 $100 \mathrm{~mL}$ 水中，摇匀。
4.6 硝酸溶液： $5\%$ 。取 $5\mathrm{mL}$ 硝酸加水至 $100\mathrm{mL}$ ，摇匀。
4.7 硝酸镧溶液： $10\%$ 。称 $23.5\mathrm{g}$ 氧化镧，用少量硝酸(4.5)微热溶解，加硝酸溶液(4.6)至 $200\mathrm{mL}$ 。

# 5 仪器

5.1 原子吸收分光光度计。
5.2 钙、镁空心阴极灯。

国家环境保护局1992-06-20批准

1993-03-01实施

5.3 $500\mathrm{mL},100\mathrm{mL}$ 容量瓶。
5.4 $10\mathrm{mL}$ 具塞比色管。

# 6样品采集与保存

按GB13580.2执行。

# 7 步骤

7.1 根据不同型号的原子吸收光度计说明书，选择最佳仪器参数，开机预热，待仪器稳定后进行测量。
7.2 校准曲线的绘制：取 $10\mathrm{mL}$ 比色管7支，分别加入钙、镁混合标准使用液(4.4)0,0.10,0.20,0.40,0.60,0.80,1.00mL，在各管中加水到刻度，再加硝酸钡溶液(4.7)0.2mL，摇匀。顺次喷入空气-乙炔火焰中，测量吸光度，分别以钙、镁的吸光度对其相应的含量作图，绘制校准曲线。
7.3样品测定：吸取 $10.0\mathrm{mL}$ 降水样品于干燥的 $10\mathrm{mL}$ 比色管中（若降水样品中钙、镁浓度分别超过 $5.0\mathrm{mg / L}$ ，和 $0.5\mathrm{mg / L}$ ，可酌情少取样品，然后加水到 $10\mathrm{mL}$ ，加 $0.2\mathrm{mL}$ 硝酸镧溶液(4.7)，摇匀。按作校准曲线的步骤(7.2)测量吸光度，从校准曲线上查得钙、镁含量。

# 8 分析结果的表述

降水中钙、镁浓度以 $\mathrm{mg / L}$ 表示，按下式计算：

$$
C = \frac {M}{V}
$$

式中： $C$ —样品中钙、镁的浓度， $\mathrm{mg / L}$

$M$ ——从校准曲线上查得钙(镁)含量, $\mu \mathrm{g}$ ;

$V$ ——取样体积， $\mathrm{mL}$ 。

# 9 精密度和准确度

8个实验室对含 $\mathrm{Ca^{2 + }2.00mg / L,Mg^{2 + }0.40mg / L,K^{+}1.00mg / L,Na^{+}1.20mg / L,Cl^{-}2.76mg / L}$ 的合成水样进行验证实验，测得结果是：钙的相对标准偏差为 $1.0\%$ ，相对误差为 $0.4\%$ ；镁的相对标准偏差为 $3.5\%$ ，相对误差为 $0\%$ 。
