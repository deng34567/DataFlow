# 1 定义

总铬的测定是将三价铬氧化成六价铬后，用二苯碳酰二肼分光光度法测定。当铬含量高时（大于 $1\mathrm{mg / L}$ ），也可采用硫酸亚铁铵滴定法。

# 第一篇高锰酸钾氧化-二苯碳酰二肼分光光度法

# 2 适用范围

2.1 本标准适用于地面水和工业废水中总铬的测定。
2.2 测定范围

试份体积为 $50\mathrm{ml}$ ，使用光程长为 $30\mathrm{mm}$ 的比色皿，本方法的最小检出量为 $0.2\mu \mathrm{g}$ 铬，最低检出浓度为 $0.004\mathrm{mg / L}$ ，使用光程为 $10\mathrm{mm}$ 的比色皿，测定上限浓度为 $1.0\mathrm{mg / L}$ 。

2.3 干扰

铁含量大于 $1\mathrm{mg} / \mathrm{L}$ 显黄色，六价钼和汞也和显色剂反应，生成有色化合物，但在本方法的显色酸度下，反应不灵敏，钼和汞的浓度达 $200\mathrm{mg} / \mathrm{L}$ 不干扰测定。钒有干扰，其含量高于 $4\mathrm{mg} / \mathrm{L}$ 时即干扰显色。但钒与显色剂反应后 $10\mathrm{min}$ ，可自行褪色。

# 3 原理

在酸性溶液中，试样的三价铬被高锰酸钾氧化成六价铬。六价铬与二苯碳酰二肼反应生成紫红色化合物，于波长 $540\mathrm{nm}$ 处进行分光光度测定。

过量的高锰酸钾用亚硝酸钠分解，而过量的亚硝酸钠又被尿素分解。

# 4 试剂

测定过程中，除非另有说明，均使用符合国家标准或专业标准的分析纯试剂和蒸馏水或同等纯度的水，所有试剂应不含铬。

4.4 硝酸（ $\mathrm{HNO}_3$ ， $\rho^* = 1.42\mathrm{g / ml}$ ）。
4.5 氯仿 $\left(\mathrm{CHCl}_3\right)$ 。
4.6 高锰酸钾：40g/L溶液。

4.7 尿素：200g/L溶液。

4.8 亚硝酸钠：20g/L溶液。

4.9 氢氧化铵：1 + 1 溶液。

氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O},\rho = 0.90\mathrm{g / ml})$ 与等体积水混合。

4.10 铜铁试剂： $50\mathrm{g} / \mathrm{L}$ 溶液。

称取铜铁试剂 $\left[\mathrm{C}_{6} \mathrm{H}_{5} \mathrm{~N}\right] \left(\mathrm{NO}\right) \left(\mathrm{ONH}_{4}\right) 5 \mathrm{~g}$ , 溶于冰水中并稀释至 $100 \mathrm{ml}$ , 临用时新配。

4.11 铬标准贮备溶液： $0.1000\mathrm{g / L}$ 。

4.12 铬标准溶液： $1\mathrm{mg / L}$ 。

吸取 $5.00\mathrm{ml}$ 铬标准贮备液（4.11）置于 $500\mathrm{ml}$ 容量瓶中，用水稀释至标线，摇匀。此溶液 $1\mathrm{ml}$ 含 $1.00\mu \mathrm{g}$ 铬。使用当天配制。

4.13 铬标准溶液： $5.00\mathrm{mg / L}$ 。

吸取 $25.00\mathrm{ml}$ 铬标准贮备液（4.11），置于 $500\mathrm{ml}$ 容量瓶中，用水稀释至标线，摇匀。此溶液 $1\mathrm{ml}$ 含 $5.00\mu \mathrm{g}$ 铬。使用当天配制。

4.14 显色剂：二苯碳酰二肼， $2\mathrm{g} / \mathrm{L}$ 丙酮溶液。

称取二苯碳酰二肼 $\left(\mathrm{C}_{13}\mathrm{H}_{14}\mathrm{N}_{4}\mathrm{O}\right)0.2\mathrm{g}$ ，溶于 $50\mathrm{ml}$ 丙酮（4.1）中，加水稀释至 $100\mathrm{ml}$ ，摇匀。贮于棕色瓶，置冰箱中。色变深后，不能使用。

# 6 采样与样品

实验室样品应该用玻璃瓶采集。采集时，加入硝酸调节样品pH值小于2。在采集后尽快测定，如放置，不得超过 $24\mathrm{h}$ 。

# 7 步骤

7.1 样品的预处理

7.1.1 一般清洁地面水可直接用高锰酸钾氧化后测定。

7.1.2 硝酸-硫酸消解：样品中含有大量的有机物需进行消解处理。

取 $50.0\mathrm{ml}$ 或适量样品（含铬少于 $50\mu \mathrm{g}$ ），置 $100\mathrm{ml}$ 烧杯中，加入 $5\mathrm{ml}$ 硝酸（4.4）和 $3\mathrm{ml}$ 硫酸（4.2），蒸发至冒白烟，如溶液仍有色，再加入 $5\mathrm{ml}$ 硝酸（4.4），重复上述操作，至溶液清澈，冷却。

用水稀释至 $10\mathrm{ml}$ ，用氢氧化铵溶液（4.9）中和至 $\mathsf{pH}$ 为 $1\sim 2$ ，移入 $50\mathrm{ml}$ 容量瓶中，用水稀释至标线，摇匀，供测定。

7.1.3·铜铁试剂-氯仿萃取除去钼、钒、铁、铜。

取 $50.0\mathrm{ml}$ 或适量样品（铬含量少于 $50\mu \mathrm{g}$ ），置 $100\mathrm{ml}$ 分液漏斗中，用氢氧化铵溶液（4.9）调至中性（加水至 $50\mathrm{ml}$ ）。加入 $3\mathrm{ml}$ 硫酸溶液（4.2.1）。

用冰水冷却后，加入 $5\mathrm{ml}$ 铜铁试剂（4.10）后振摇 $1\mathrm{min}$ ，置冰水中冷却 $2\mathrm{min}$ 。每次用 $5\mathrm{ml}$ 氯仿（4.5）共萃取三次，弃去氯仿层。

将水层移入锥形瓶中，用少量水洗涤分液漏斗，洗涤水亦并入锥形瓶中。加热煮沸，使水层中氯仿挥发后，按7.1.2和7.2处理。

# 7.2 高锰酸钾氧化三价铬

7.2.1 取 $50.0\mathrm{ml}$ 或适量（铬含量少于 $50\mu \mathrm{g}$ ）样品或经7.1.2、7.1.3处理的试样，置于 $150\mathrm{ml}$ 锥形瓶中，用氢氧化铵溶液（4.9）或硫酸溶液（4.2.1）调至中性，加入几粒玻璃珠，加入 $0.5\mathrm{ml}$ 硫酸溶液（4.2.1）、 $0.5\mathrm{ml}$ 磷酸溶液（4.3）（加水至 $50\mathrm{ml}$ ），摇匀，加2滴高锰酸钾溶液（4.6），如紫红色消褪，则应添加高锰酸钾溶液保持紫红色。加热煮沸至溶液体积约剩 $20\mathrm{ml}$ 。

取下冷却，加入 $1\mathrm{ml}$ 尿素溶液（4.7），摇匀。用滴管滴加亚硝酸钠溶液（4.8），每加一滴充分摇匀，至高锰酸钾的紫红色刚好褪去。稍停片刻，待溶液内气泡逸出，转移至 $50\mathrm{ml}$ 比色管中。

注：①也可用叠氮化钠还原过量的高锰酸钾。即在氧化步骤完成后取下，趁热逐滴加入浓度为 $2\mathrm{g} / \mathrm{L}$ 的叠氮化钠溶液，每加一滴立即摇匀，煮沸，重复数次，至紫红色完全褪去，继续煮沸 $1\mathrm{min}$ 。

警告：叠氮化钠是易爆危险品。

② 如样品中含有少量铁（ $\mathrm{Fe}^{3+}$ ）干扰测定，可将7.2.1中加入 $0.5\mathrm{ml}$ 硫酸（4.2.1）、 $0.5\mathrm{ml}$ 磷酸溶液（4.3）改为加入 $1.5\mathrm{ml}$ 磷酸溶液（4.3）。

# 7.3 测定

取 $50\mathrm{ml}$ 或适量（含铬量少于 $50\mu \mathrm{g}$ ）经7.2步骤处理的试份置 $50\mathrm{ml}$ 比色管中，用水稀释至刻线，加入 $2\mathrm{ml}$ 显色剂（4.14），摇匀。 $10\mathrm{min}$ 后，在 $540\mathrm{nm}$ 波长下，用10或 $30\mathrm{mm}$ 光程的比色皿，以水做参比，测定吸光度。减去空白试验吸光度，从校准曲线（7.5）上查得铬的含量。

# 7.4 空白试验

按与试样完全相同的处理步骤进行空白试验，仅用 $50\mathrm{ml}$ 水代替试样。

# 7.5 校准

向一系列 $150\mathrm{ml}$ 锥形瓶中分别加入0、0.20、0.50、1.00、2.00、4.00、6.00、8.00和 $10.00\mathrm{ml}$ 铬标准溶液（4.12或4.13），用水稀释至 $50\mathrm{ml}$ 。然后按照测定试样的步骤（7.1、7.2、7.3）进行处理。

从测得的吸光度减去空白试验的吸光度后，绘制以含铬量对吸光度的曲线。

$$

式中： $m$ ——从校准曲线上查得的试份中含铬量， $\mu \mathrm{g}$

# 第二篇硫酸亚铁铵滴定法

# 9 适用范围

本标准适用于水和废水中高浓度（大于 $1\mathrm{mg / L}$ ）总铬的测定。

# 10. 原理.

在酸性溶液中，以银盐作催化剂，用过硫酸铵将三价铬氧化成六价铬。加入少量氯化钠并煮沸，除去过量的过硫酸铵及反应中产生的氯气。以苯基代邻氨基苯甲酸做指示剂，用硫酸亚铁铵溶液滴定，使六价铬还原为三价铬，溶液呈绿色为终点。根据硫酸亚铁铵溶液的用量，计算出样品中总铬的含量。

钒对测定有干扰，但在一般含铬废水中钒的含量在允许限以下。

# 11 试剂

在测定过程中，除非另有说明，均使用符合国家标准或专业标准的分析纯试剂和蒸馏水或同等纯度的水。

11.1 $5\% (V / V)$ 硫酸溶液。

取硫酸（4.2） $100\mathrm{ml}$ 缓慢加入到 $2\mathrm{L}$ 水中，混匀。

11.2 磷酸 $(\mathrm{H}_3\mathrm{PO}_4, \rho = 1.69\mathrm{g / ml})$ 。
11.3 硫酸-磷酸混合液：取 $150\mathrm{ml}$ 硫酸（4.2）缓慢加入到 $700\mathrm{ml}$ 水中，冷却后，加入 $150\mathrm{ml}$ 磷酸（11.2）混匀。
11.4 过硫酸铵 $\left[\left(\mathrm{NH}_{4}\right)_{2} \mathrm{~S}_{2} \mathrm{O}_{8}\right]$ ； $250 \mathrm{~g} / \mathrm{L}$ 溶液。
11.5 铬标准溶液：称取于 $110^{\circ} \mathrm{C}$ 干燥 $2 \mathrm{~h}$ 的重铬酸钾 $\left(\mathrm{K}_{2} \mathrm{Cr}_{2} \mathrm{O}_{7}\right.$ ，优级纯） $0.5658 \pm 0.0001 \mathrm{~g}$ ，用水溶解后，移入 $1000 \mathrm{ml}$ 容量瓶中，加入稀释至标线，摇匀。此溶液 $1 \mathrm{ml}$ 含 $0.2 \mathrm{mg}$ 铬。
11.6 硫酸亚铁铵溶液。

称取硫酸亚铁铵 $\left[\left(\mathrm{NH}_{4}\right)_{2} \mathrm{Fe}\left(\mathrm{SO}_{4}\right)_{2} \cdot 6 \mathrm{H}_{2} \mathrm{O}\right] 3.95 \pm 0.01 \mathrm{~g}$ ，用 $500 \mathrm{ml}$ 硫酸溶液(11.1)溶解，过滤至 $2000 \mathrm{ml}$ 容量瓶中，用硫酸溶液（11.1）稀释至标线。临用时，用铬标准溶液（11.5）标定。

标定：吸取三份各 $25.0\mathrm{ml}$ 铬标准溶液（11.5）置 $500\mathrm{ml}$ 锥形瓶中，用水稀释至 $200\mathrm{ml}$ 左右。加入 $20\mathrm{ml}$ 硫酸-磷酸混合液（11.3），用硫酸亚铁铵溶液（11.6）滴定至淡黄色。加入3滴苯基代邻氨基苯甲酸指示剂（11.12），继续滴定至溶液由红色突变为亮绿色为终点，记录用量 $V$ 。

三份铬标准溶液所消耗硫酸亚铁铵溶液的毫升数的极差值不应超过 $0.05\mathrm{mi}$ ，取其平均值。

按式（2）计算：

$$
T = \frac {0 . 2 0 \times 2 5 . 0}{V} = \frac {5 . 0}{V} \dots \tag {2}
$$

式中： $T$ ——硫酸亚铁铵溶液对铬的滴定度， $\mathrm{mg / ml}$ 。

11.7 硫酸锰： $10\mathrm{g} / \mathrm{L}$ 溶液。

将硫酸锰（ $\mathrm{MnSO_4\cdot 2H_2O}$ ） $1\mathrm{g}$ 溶于水稀释至 $100\mathrm{ml}$ 。

11.8 硝酸银： $5\mathrm{g} / \mathrm{L}$ 溶液。

将硝酸银（ $\mathrm{AgNO}_3$ ）0.5g溶于水并稀释至 $100\mathrm{ml}$ 。

11.9 无水碳酸钠： $50\mathrm{g} / \mathrm{L}$ 溶液。

将无水碳酸钠（ $\mathrm{Na}_2\mathrm{CO}_3$ ） $5\mathrm{g}$ 溶于水并稀释至 $100\mathrm{ml}$ 。

11.10 氢氧化铵： $1 + 1$ 溶液。

取氨水（ $\rho = 0.90\mathrm{g / ml}$ ）加入等体积水中，混匀。

11.11 氯化钠： $10\mathrm{g} / \mathrm{L}$ 溶液。

将氯化钠（NaCl）1g溶于水并稀释至 $100\mathrm{ml}$ 。

# 11.12 苯基代邻氨基苯甲酸指示剂。

称取苯基代邻氨基苯甲酸（phenylan thranilic acid）0.27g溶于 $5\mathrm{ml}$ 碳酸钠溶液（11.9）中，用水稀释至 $250\mathrm{ml}$ 。

# 12 步骤

# 12.1 测定

吸取适量样品于 $150\mathrm{ml}$ 烧杯中，按7.1.2步骤消解后转移至 $500\mathrm{ml}$ 锥形瓶中（如果样品清澈、无色，可直接取适量样品于 $500\mathrm{ml}$ 锥形瓶中）。用氢氧化铵溶液（11.10）中和至溶液 $\mathsf{pH}$ 为 $1\sim 2$ 。加入 $20\mathrm{ml}$ 硫酸-磷酸混合液（11.3）、 $1\sim 3$ 滴硝酸银溶液（11.8）、 $0.5\mathrm{ml}$ 硫酸锰溶液（11.7）、 $25\mathrm{ml}$ 过硫酸铵溶液（11.4），摇匀，加入几粒玻璃珠。加热至出现高锰酸盐的紫红色，煮沸 $10\mathrm{min}$ 。

取下稍冷，加入 $5\mathrm{ml}$ 氯化钠溶液（11.11），加热微沸 $10\sim 15\mathrm{min}$ ，除尽氯气。取下迅速冷却，用水洗涤瓶壁并稀释至 $220\mathrm{ml}$ 左右。加入3滴苯基代邻氨基苯甲酸指示剂（11.12），用硫酸亚铁铵溶液（11.6）滴定至溶液由红色突变为绿色即为终点，记下用量 $V_{1}$ 。

注：① 应注意掌握加热煮沸时间，若加热煮沸时间不够，过量的过硫酸铵及氯气未除尽，会使结果偏高；若煮沸时间太长，溶液体积小，酸度高，可能使六价铬还原为三价铬，使结果偏低。

$②$ 苯基代邻氨基苯甲酸指示剂，在测定样品和空白试验时加入量要保持一致。

按12.1步骤进行空白试验，仅用和样品体积相同的水代替样品。

# 13 结果表示

# 13.1 计算方法

总铬含量 $c_{2}(\mathrm{mg} / \mathrm{L})$ 按式（3）计算：

$$
c _ {2} = \frac {\left(V _ {1} - V _ {0}\right) T \times 1 0 0 0}{V} \dots \tag {3}
$$

式中： $V_{1}$ ——滴定样品时，硫酸亚铁铵溶液（11.6）用量，ml；

$V_{0}$ ——空白试验时，硫酸亚铁铵溶液（11.6）用量，ml；

$T$ ——硫酸亚铁铵溶液（11.6）对铬的滴定度， $\mathrm{mg / ml}$ ；

$\nu$ ——样品的体积， $\mathfrak{m}\mathfrak{l}$ 。
