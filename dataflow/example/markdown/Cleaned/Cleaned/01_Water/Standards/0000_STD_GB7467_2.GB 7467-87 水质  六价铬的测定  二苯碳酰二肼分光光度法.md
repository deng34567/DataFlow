# 1 适用范围

1.1 本标准适用于地面水和工业废水中六价铬的测定。
1.2 测定范围

试份体积为 $50\mathrm{ml}$ ，使用光程长为 $30\mathrm{mm}$ 的比色皿，本方法的最小检出量为 $0.2\mu \mathrm{g}$ 六价铬，最低检出浓度为 $0.004\mathrm{mg / L}$ ，使用光程为 $10\mathrm{mm}$ 的比色皿，测定上限浓度为 $1.0\mathrm{mg / L}$ 。

1.3 干扰

含铁量大于 $1\mathrm{mg / L}$ 显色后呈黄色。六价钼和汞也和显色剂反应，生成有色化合物，但在本方法的显色酸度下，反应不灵敏，钼和汞的浓度达 $200\mathrm{mg / L}$ 不干扰测定。钒有干扰，其含量高于 $4\mathrm{mg / L}$ 即干扰显色。但钒与显色剂反应后 $10\mathrm{min}$ ，可自行褪色。

# 2 原理

在酸性溶液中，六价铬与二苯碳酰二肼反应生成紫红色化合物，于波长 $540\mathrm{nm}$ 处进行分光光度测定。

# 3 试剂

测定过程中，除非另有说明，均使用符合国家标准或专业标准的分析纯试剂和蒸馏水或同等纯度的水，所有试剂应不含铬。

3.1 丙酮。
3.2 硫酸

3.2.1 $1 + 1$ 硫酸溶液。

将硫酸 $\left(\mathrm{H}_{2} \mathrm{SO}_{4}, \rho = 1.84 \mathrm{~g} / \mathrm{ml}\right.$ ，优级纯）缓缓加入到同体积的水中，混匀。

3.3 磷酸： $1 + 1$ 磷酸溶液。

将磷酸（ $\mathrm{H}_3\mathrm{PO}_4$ ， $\rho = 1.69\mathrm{g / ml}$ ，优级纯）与水等体积混合。

3.4 氢氧化钠： $4\mathrm{g} / \mathrm{L}$ 氢氧化钠溶液。

将氢氧化钠（NaOH）1g溶于水并稀释至 $250\mathrm{ml}$ 。

3.5 氢氧化锌共沉淀剂
3.5.1 硫酸锌： $8\% (m / V)$ 硫酸锌溶液。

称取硫酸锌（ $\mathrm{ZnSO_4\cdot 7H_2O}$ ） $8\mathrm{g}$ ，溶于 $100\mathrm{ml}$ 水中。

3.5.2 氢氧化钠： $2\% (m / V)$ 溶液。

称取 $2.4\mathrm{g}$ 氢氧化钠，溶于 $120\mathrm{ml}$ 水中。

用时将3.5.1和3.5.2两溶液混合。

3.6 高锰酸钾： $40\mathrm{g} / \mathrm{L}$ 溶液。

称取高锰酸钾（ $\mathrm{KMnO}_4$ ） $4\mathrm{g}$ ，在加热和搅拌下溶于水，最后稀释至 $100\mathrm{ml}$ 。

# 3.7 铬标准贮备液。

称取于 $110^{\circ} \mathrm{C}$ 干燥 $2 \mathrm{~h}$ 的重铬酸钾 $\left(\mathrm{K}_{2} \mathrm{Cr}_{2} \mathrm{O}_{7}\right.$ ，优级纯） $0.2829 \pm 0.0001 \mathrm{~g}$ ，用水溶解后，移入1000 ml容量瓶中，用水稀释至标线，摇匀。此溶液 $1 \mathrm{ml}$ 含 $0.10 \mathrm{mg}$ 六价铬。

# 3.8 铬标准溶液。

吸取 $5.00\mathrm{ml}$ 铬标准贮备液（3.7）置于 $500\mathrm{ml}$ 容量瓶中，用水稀释至标线，摇匀。此溶液 $1\mathrm{ml}$ 含 $1.00\mu \mathrm{g}$ 六价铬。使用当天配制此溶液。

# 3.9 铬标准溶液。

吸取 $25.00\mathrm{ml}$ 铬标准贮备液（3.7）置于 $500\mathrm{ml}$ 容量瓶中，用水稀释至标线，摇匀。此溶液 $1\mathrm{ml}$ 含 $5.00\mu \mathrm{g}$ 六价铬。使用当天配制此溶液。

# 3.10 尿素： $200\mathrm{g} / \mathrm{L}$ 尿素溶液。

将尿素 $\left[\left(\mathrm{NH}_{2}\right)_{2} \mathrm{CO}\right] 20 \mathrm{~g}$ 溶于水并稀释至 $100 \mathrm{ml}$ 。

# 3.11 亚硝酸钠： $20\mathrm{g} / \mathrm{L}$ 溶液。

将亚硝酸钠 $(\mathrm{NaNO}_2)2\mathrm{g}$ 溶于水并稀释至 $100\mathrm{ml}$ 。

# 3.12 显色剂（I）。

称取二苯碳酰二肼（ $\mathbf{C}_{13}\mathbf{H}_{14}\mathbf{N}_{4}\mathbf{O}$ ） $0.2\mathrm{g}$ ，溶于 $50\mathrm{ml}$ 丙酮（3.1）中，加水稀释至 $100\mathrm{ml}$ ，摇匀，贮于棕色瓶，置冰箱中。色变深后，不能使用。

# 3.13 显色剂（Ⅱ）。

称取二苯碳酰二肼 $2\mathrm{g}$ ，溶于 $50\mathrm{ml}$ 丙酮（3.1）中，加水稀释至 $100\mathrm{ml}$ ，摇匀。贮于棕色瓶，置冰箱中。色变深后，不能使用。

注：显色剂（I）也可按下法配制：称取 $4.0\mathrm{g}$ 苯二甲酸酐（ $\mathrm{C_6H_4O}$ ），加到 $80\mathrm{ml}$ 乙醇中，搅拌溶解（必要时可用水浴微温），加入 $0.5\mathrm{g}$ 二苯碳酰二肼，用乙醇稀释至 $100\mathrm{ml}$ 。此溶液于暗处可保存六个月。使用时要注意加入显色剂后立即摇匀，以免六价铬被还原。

# 5 采样与样品

实验室样品应该用玻璃瓶采集。采集时，加入氢氧化钠，调节样品 $\mathsf{pH}$ 值约为8。并在采集后尽快测定，如放置，不要超过 $24\mathrm{h}$ 。

# 6 步骤

# 6.1 样品的预处理

6.1.1 样品中不含悬浮物，是低色度的清洁地面水可直接测定。
6.1.2 色度校正：如样品有色但不太深时，按6.3步骤另取一份试样，以 $2\mathrm{ml}$ 丙酮（3.1）代替显色剂，其他步骤同6.3。试份测得的吸光度扣除此色度校正吸光度后，再行计算。
6.1.3 锌盐沉淀分离法：对混浊、色度较深的样品可用此法前处理。

取适量样品（含六价铬少于 $100\mu \mathrm{g}$ ）于 $150\mathrm{ml}$ 烧杯中，加水至 $50\mathrm{ml}$ 。滴加氢氧化钠溶液（3.4），调节溶液 $\mathsf{pH}$ 值为 $7\sim 8$ 。在不断搅拌下，滴加氢氧化锌共沉淀剂（3.5）至溶液 $\mathsf{pH}$ 值为 $8\sim 9$ 。将此溶液转移至 $100\mathrm{ml}$ 容量瓶中，用水稀释至标线。用慢速滤纸干过滤，弃去 $10\sim 20\mathrm{ml}$ 初滤液，取其中 $50.0\mathrm{ml}$ 滤液供测定。

注：当样品经锌盐沉淀分离法前处理后仍含有机物干扰测定时，可用酸性高锰酸钾氧化法破坏有机物后再测定。即取 $50.0\mathrm{ml}$ 滤液于 $150\mathrm{ml}$ 锥形瓶中，加入几粒玻璃珠，加入 $0.5\mathrm{ml}$ 硫酸溶液（3.2.1）、 $0.5\mathrm{ml}$ 磷酸溶液（3.3），摇匀。加入2滴高锰酸钾溶液（3.6），如紫红色消褪，则应添加高锰酸钾溶液保持紫红色。加热煮沸至溶液体积约剩 $20\mathrm{ml}$ 。取下稍冷，用定量中速滤纸过滤，用水洗涤数次，合并滤液和洗液至 $50\mathrm{ml}$ 比色管中。加入 $1\mathrm{ml}$ 尿素溶液（3.10）。摇匀，用滴管滴加亚硝酸钠溶液（3.11），每加一滴充分摇匀，至高锰酸钾的紫红色刚好褪去。稍停片刻，待溶液内气泡逸尽，转移至 $50\mathrm{ml}$ 比色管中，用水稀释至标线，供测定用。

6.1.4 二价铁、亚硫酸盐、硫代硫酸盐等还原性物质的消除：取适量样品（含六价铬少于 $50\mu \mathrm{g}$ ）于 $50\mathrm{ml}$ 比色管中，用水稀释至标线，加入 $4\mathrm{ml}$ 显色剂（Ⅱ）（3.13），混匀，放置 $5\mathrm{min}$ 后，加入 $1\mathrm{ml}$ 硫酸溶液（3.2）摇匀。 $5\sim 10\mathrm{min}$ 后，在 $540\mathrm{nm}$ 波长处，用 $10$ 或 $30\mathrm{mm}$ 光程的比色皿，以水做参比，测定吸光度。扣除空白试验测得的吸光度后，从校准曲线查得六价铬含量。用同法做校准曲线。
6.1.5 次氯酸盐等氧化性物质的消除：取适量样品（含六价铬少于 $50\mu \mathrm{g}$ ）于 $50\mathrm{ml}$ 比色管中，用水稀释至标线，加入 $0.5\mathrm{ml}$ 硫酸溶液（3.2）、 $0.5\mathrm{ml}$ 磷酸溶液（3.3）、 $1.0\mathrm{ml}$ 尿素溶液（3.10），摇匀，逐滴加入 $1\mathrm{ml}$ 亚硝酸钠溶液（3.11），边加边摇，以除去由过量的亚硝酸钠与尿素反应生成的气泡，待气泡除尽后，以下步骤同6.3（免去加硫酸溶液和磷酸溶液）。

# 6.2 空白试验

按同试样完全相同的处理步骤进行空白试验，仅用 $50\mathrm{ml}$ 水代替试样。

# 6.3 测定

取适量（含六价铬少于 $50\mu \mathrm{g}$ ）无色透明试份，置于 $50\mathrm{ml}$ 比色管中，用水稀释至标线。加入 $0.5\mathrm{ml}$ 硫酸溶液（3.2）和 $0.5\mathrm{ml}$ 磷酸溶液（3.3），摇匀。加入 $2\mathrm{ml}$ 显色剂（I）（3.12），摇匀， $5\sim 10\mathrm{min}$ 后，在 $540\mathrm{nm}$ 波长处，用10或 $30\mathrm{mm}$ 的比色皿，以水做参比，测定吸光度，扣除空白试验测得的吸光度后，从校准曲线（6.4）上查得六价铬含量。

注：如经锌盐沉淀分离、高锰酸钾氧化法处理的样品，可直接加入显色剂测定。

# 6.4 校准

向一系列 $50\mathrm{ml}$ 比色管中分别加入0、0.20、0.50、1.00、2.00、4.00、6.00、8.00和 $10.0\mathrm{ml}$ 铬标准溶液（3.8或3.9）（如经锌盐沉淀分离法前处理，则应加倍吸取），用水稀释至标线。然后按照测定试样的步骤（6.1和6.3）进行处理。

从测得的吸光度减去空白试验的吸光度后，绘制以六价铬的量对吸光度的曲线。

# 7 结果的表示

# 7.1 计算方法

六价铬含量 $c(\mathrm{mg / L})$ 按下式计算：

$$
c = \frac {m}{V}
$$

式中： $m$ ——由校准曲线查得的试份含六价铬量， $\mu \mathrm{g}$

$V$ ——试份的体积， $\mathfrak{m}\mathfrak{l}$ 。

六价铬含量低于 $0.1\mathrm{mg / L}$ ，结果以三位小数表示；六价铬含量高于 $0.1\mathrm{mg / L}$ ，结果以三位有效数字表示。
