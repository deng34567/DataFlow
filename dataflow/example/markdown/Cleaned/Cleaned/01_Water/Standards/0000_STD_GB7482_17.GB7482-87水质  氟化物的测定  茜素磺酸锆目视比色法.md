# 水质氟化物的测定茜素磺酸锆目视比色法

UDC 614.777:543

.432:546.16

GB7482-87

Water quality-Determination of fluoride Visual colorimetry with zirconium alizarinsulfonate

# 1 适用范围

本标准规定茜素磺酸锆目视比色法测定饮用水、地面水、地下水及废水中的氟化物。

取 $50\mathrm{m}1$ 试样，直接测定时，本方法最低检出浓度为含氟 $0.05\mathrm{mg / L}$ ，检测上限为 $2.5\mathrm{mg / L}$ （高含量样品可经稀释后分析）。

当样品中干扰离子超过下列浓度（以 $\mathbf{m}\mathbf{g} / \mathbf{L}$ 计）时，需要预先蒸馏：总碱度（ $\mathrm{CaCO_3}$ ）400、氯化物500、硫酸盐200、铝0.1、磷酸盐1.0、铁2.0、浊度25度、色度25度。

# 2 原理

在酸性溶液中，茜素磺酸钠和锆盐生成红色络合物，当样品中有氟离子存在时，能夺取络合物中锆离子，生成无色的氟化锆离子，释放出黄色的茜素磺酸钠，根据溶液由红色褪至黄色的色度不同，与标准比色定量。

# 3 试剂

除非另有说明，分析时均使用符合国家标准或专业标准的分析纯试剂和蒸馏水或同等纯度的水。

3.1 高氯酸（HC1O4）： $70\sim 72\%$
3.2 盐酸 $(\mathbf{HC1}):\rho_{20} = 1.19\mathbf{g} / \mathbf{m}1$ 。
3.3 硫酸 $\left(\mathrm{H}_{2} \mathrm{SO}_{4}\right): \rho = 1.84 \mathrm{~g} / \mathrm{m} l$ 。
3.4 亚砷酸钠（NaAsO₂）：5g/L。

称取 $0.5\mathrm{g}$ 亚砷酸钠，溶解于少量水中，稀释至 $100\mathrm{ml}$ 。

注：亚砷酸钠剧毒，防止进入口中。

3.5 氟化物标准溶液，含氟 $100.0\mu \mathrm{g} / \mathrm{ml}$ ：取氟化钠于 $105\%$ 烘 $2\mathrm{h}$ ，于干燥器中冷却后，精确称取 $0.2210\mathrm{g}$ ，用水溶解，转入 $1000\mathrm{ml}$ 容量瓶中，并用水稀释至刻度。
3.6 氟化物标准溶液，含氟 $10.00\mu \mathrm{g} / \mathrm{ml}$ ：取 $10.00\mathrm{ml}$ 氟化物标准溶液（3.5），于 $100\mathrm{ml}$ 容量瓶中，用水稀释至刻度。

# 3.7 茜素磺酸锆酸性溶液

3.7.1 茜素磺酸锆溶液：称取 $0.3\mathrm{g}$ 氯氧化锆（ $\mathrm{ZrOCl}_2 \cdot 8\mathrm{H}_2\mathrm{O}$ ）于 $100\mathrm{ml}$ 烧杯中，用 $50\mathrm{ml}$ 水溶解后移入 $1000\mathrm{ml}$ 容量瓶中。另称取 $0.07\mathrm{g}$ 茜素磺酸钠（ $\mathrm{C}_1\mathrm{H}_2\mathrm{O},\mathrm{SNa}\cdot \mathrm{H}_2\mathrm{O}$ ）溶于 $50\mathrm{ml}$ 水中，在不断摇动下，缓慢注入氯氧化锆溶液中。充分摇动后，放置澄清。
3.7.2 混合酸溶液：量取 $101\mathrm{ml}$ 盐酸（3.2）用水稀释到 $400\mathrm{ml}$ ，另量取 $33.3\mathrm{ml}$ 硫酸（3.3），在不断搅拌下，缓慢加入到 $400\mathrm{ml}$ 水中，冷却后将两酸合并。
3.7.3 将混合酸（3.7.2）倾入盛有茜素磺酸锆溶液（3.7.1）的容量瓶中，用水稀释到刻度，摇

匀，此溶液在 $1\mathrm{h}$ 后由红变黄即可使用。

此溶液避光保存，可稳定6个月。

# 5 采样和样品

采集和贮存样品均使用聚乙烯瓶。

# 5.1 试样的制备

除非证明试样的预处理是不必要的，可直接制备试份（5.2）进行比色，否则应按下述步骤进行蒸馏预处理：

取 $50.0\mathrm{ml}$ 水样于蒸馏瓶（3）中，加 $10\mathrm{ml}$ 高氯酸（3.1），摇匀，按图连接好装置，加热，待蒸馏瓶内溶液温度升到约 $130^{\circ}\mathrm{C}$ 时，开始通入蒸汽，并维持温度在 $130\sim 140^{\circ}\mathrm{C}$ ，蒸馏速度为 $5\sim 6\mathrm{ml} / \mathrm{min}$ 。待接收瓶（1）中馏出液（ $\pmb{W}$ ）体积约 $200\mathrm{ml}$ 时停止蒸馏，并用水稀释至 $200\mathrm{ml}$ ，留测定用。

# 5.2 试份

取 $50\mathrm{m}1$ 试样或馏出液（ $\pmb{W}$ ）置于比色管（4.1）中，氟含量高于 $2.5\mathrm{mg} / \mathrm{L}$ 时，可量取少量试样或馏出液（ $\pmb{W}$ ），用水稀释到 $50\mathrm{m}1$ 。

# 6 步骤

# 6.1 空白试验

用 $50\mathrm{m}1$ 水代替试份，所用试剂及其用量与在测定（6.2）中所用的相同，并采用相同的步骤。

# 6.2 测定

如果试样中含有余氯，按每毫克余氯加入1滴（0.05ml）亚砷酸钠溶液（3.4），混匀，将余氯除去。加2.5ml茜素磺酸锆酸性溶液（3.7.3）于试份（5.2）中混匀，放置1h，与标准系列（6.3）进行比色。

注：应调节温度，使试份与标准比色系列之间的温差不超过 $2^{\circ}\mathrm{C}$ 。

# 6.3 校准

在一系列比色管（4.1）中分别加入不同体积的氟化物标准溶液（3.6），并用水稀释至 $50\mathrm{m}1$ ，以下步骤按（6.2）进行。选择的标准溶液中，至少有两个低于和两个高于试份中氟化物的浓度，通常以 $50\mu \mathrm{g} / \mathrm{L}$ 或 $100\mu \mathrm{g} / \mathrm{L}$ 的氟为间隔是合适的。

# 7 结果的表示

样品中氟化物的浓度 $c_{1}(\mathbf{mg} / \mathbf{L})$ （以F计）按下式计算：

$$
c _ {1} = \frac {c}{V _ {2}} \times \frac {2 0 0}{V _ {1}}
$$

式中： $c$ 由标准系列给出试份含氟量， $\mu \mathrm{g}$

$V_{2}$ ——试份的体积， $\mathbf{m}1$ ；

$V_{1}$ ——试样体积，ml。

# 8 精密度与准确度

# 8.1 重复性

对自来水作8次测定，得相对标准偏差为 $1.5\%$

对湖水作6次测定，得相对标准偏差为 $3.0\%$

# 8.2 再现性和准确度

20个实验室测定下列水样的结果：

含氟量为 $830\mu \mathrm{g} / \mathrm{L}$ 的无干扰物质的统一水样：室间相对标准偏差为 $4.9\%$ ，相对误差为 $3.6\%$ 。含氟量为 $570\mu \mathrm{g} / \mathrm{L}$ 的含有干扰物质的统一水样：室间相对标准偏差为 $11.1\%$ ，相对误差为0。

# A.1 共存离子影响

样品中有硫酸盐、磷酸盐、铁、锰的存在，能使测定结果偏高，铝可与氟离子形成稳定的络合物 $(\mathrm{ALF}_{6})^{8-})$ ，使测定结果偏低。

A.2 茜素磺酸钠（1,2-二羟基蒽醌-3-磺酸钠，结构式如下）是易溶于水的橙色粉末，水溶液呈浅褐色，pH3.7时呈黄色，pH5.2时呈紫色，试剂配制后与锆盐最好分别保存，使用时再按比例混合，以保持试剂的灵敏度。

![](images/14655cf88dd703392d6a9bf7b4f99699ce44af09a905ea72001d6c8c64b22318.jpg)

A.3 茜素磺酸锆与氟离子在作用过程中颜色的形成，受各种因素的影响，因此在分析时，要控制样品。空白和标准系列加入试剂的量、反应温度、放置时间等条件必须一致。
A.4 如果水样中有机物含量高时，为避免与高氯酸发生爆炸，用硫酸代替高氯酸（酸与水样的体积比为 $1:1$ ）进行蒸馏，控制温度在 $145 \pm 5^{\circ} \mathrm{C}$ 。
