# 中华人民共和国国家标准

# 水质铁（Ⅱ、Ⅲ）氰络合物的测定三氯化铁分光光度法

GB/T13899-92

Water quality—Determination of ferro and ferric cyanic complex—Ferric trichloride spectrophotometry

# 1 主题内容与适用范围

# 1.1 主题内容

本标准规定了测定火工品工业废水中铁（Ⅱ、Ⅲ）氰络合物的三氯化铁分光光度法。

# 1.2 适用范围

1.2.1 本标准适用于火工品生产厂工厂排出口废水中铁（Ⅱ、Ⅲ）氰络合物含量的测定。
1.2.2 当取样体积为 $25\mathrm{mL}$ 时，铁（Ⅱ、Ⅲ）氰络合物的最低检出浓度为 $0.4\mathrm{mg / L}$ ；测定浓度范围为 $2\sim 10\mathrm{mg / L}$ 。

# 2 原理

亚铁氰根与三氯化铁在酸性介质中生成蓝色的亚铁氰化铁，在 $700\mathrm{nm}$ 波长处进行光度测定。铁氰化物被对甲氨基苯酚硫酸盐还原生成亚铁氰化物后也同时被测定。

# 3 试剂

本标准所用试剂均为分析纯试剂；所用的水为去离子水或具有同等纯度的水。

3.1 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ 溶液： $1 + 2(V / V)$ 。
3.2 碳酸钠 $\mathrm{(Na_2CO_3)}$ 溶液： $100\mathrm{g / L}$
3.3 对甲氨基苯酚硫酸盐 $\left(\mathrm{C}_{14} \mathrm{H}_{18} \mathrm{O}_{2} \mathrm{~N}_{2} \cdot \mathrm{H}_{2} \mathrm{SO}_{4}\right)$ 溶液： $5 \mathrm{~g} / \mathrm{L}$ 。
3.4 明胶溶液： $5\mathrm{g} / \mathrm{L}$
3.5 硫酸镁 $(\mathrm{MgSO_4\cdot 7H_2O})$ 溶液： $50~\mathrm{g / L}$
3.6 高锰酸钾 $\mathrm{(KMnO_4)}$ 溶液： $5\mathrm{g / L}$
3.7 三氯化铁溶液: $6 \mathrm{~g} / \mathrm{L}$ 。称取 $10.0 \mathrm{~g}$ 三氯化铁 $\left(\mathrm{FeCl}_{3} \cdot 6 \mathrm{H}_{2} \mathrm{O}\right)$ , 溶于 $1000 \mathrm{~mL}$ 浓度为 $c(\mathrm{HCl}) =$

1.5mol/L的盐酸(HCl)溶液中。
3.8 氯化钠(NaCl)溶液： $80\mathrm{g / L}$ 。
3.9 盐酸羟胺 $(\mathrm{NH}_2\mathrm{OH}\cdot \mathrm{HCl})$ 溶液： $10\mathrm{g / L}$
3.10 硫代硫酸钠 $(\mathrm{Na}_{2}\mathrm{S}_{2}\mathrm{O}_{3}\cdot 5\mathrm{H}_{2}\mathrm{O})$ 溶液： $25\mathrm{g / L}$
3.11 氢氧化钠(NaOH)溶液： $100\mathrm{g / L}$ 。
3.12 铁氰化钾 $\left[\mathrm{K}_{3} \mathrm{Fe}(\mathrm{CN})_{6}\right]$ 标准溶液。
3.12.1 铁氰化钾标准贮备液：称取 $0.7768\mathrm{g}$ 经 $105\mathrm{C}$ 烘干 $2\mathrm{h}$ 的铁氰化钾，溶于适量水中，加入 $5\mathrm{mL}$ 氢氧化钠溶液(3.11)，移入 $500\mathrm{mL}$ 棕色容量瓶中，以水稀释至标线，混匀，保存于暗处。 $1.00\mathrm{mL}$ 此溶液含 $1.00\mathrm{mg}$ 铁氰酸根。

3.12.2 铁氰化钾标准使用液：量取 $5.00 \mathrm{~mL}$ 铁氰化钾标准贮备液于 $100 \mathrm{~mL}$ 棕色容量瓶中，加水稀释至标线，混匀。 $1.00 \mathrm{~mL}$ 此溶液含 $50.0 \mu \mathrm{g}$ 铁氰酸根。使用时配制。

# 4 仪器

一般实验室仪器和具 $30\mathrm{mm}$ 比色皿的分光光度计。

# 5 采样与样品

水样采集于玻璃瓶中，采集后，于每升水样中加入 $1\mathrm{mL}$ 硫代硫酸钠溶液(3.10)，以氢氧化钠溶液(3.11)调整其 $\mathrm{pH} > 12$ ，于 $2\sim 5\mathrm{C}$ 下冷藏。水样应于 $24\mathrm{h}$ 内进行测定。

# 6 分析步骤

# 6.1 试料

分别量取两份同体积的均匀试样[其体积不大于 $30~\mathrm{mL}$ ，准确至 $0.1~\mathrm{mL}$ ，含铁（Ⅱ、Ⅲ）氰络合物的量小于 $0.25~\mathrm{mg}$ ] 作为试料。

# 6.2 空白试验

用同体积水代替试样，加入试剂的量及试验步骤与6.3条测定相同，进行空白试验。

# 6.3 测定

# 6.3.1 前处理

取试料于 $200\mathrm{mL}$ 烧杯中，加水至 $30\mathrm{mL}$ ，以硫酸溶液(3.1)调整 $\mathrm{pH}$ 值至 $6\sim 8$ 后，加入 $0.5\mathrm{mL}$ 碳酸钠溶液(3.2)，滴加高锰酸钾溶液(3.6)至试液呈现不褪的粉红色后，再过量 $0.2\mathrm{mL}$ ，加水至 $50\mathrm{mL}$ ，烧杯内放入数粒玻璃珠，盖以表面皿，移至电炉上加热，微沸约 $15\mathrm{min}$ ，至溶液体积约为 $20\mathrm{mL}$ 时，取下，加入 $2\mathrm{mL}$ 硫酸镁溶液(3.5)，混匀后，静置至室温。用水冲洗表面皿，将烧杯内容物以慢速滤纸过滤于 $50\mathrm{mL}$ 容量瓶中，洗净烧杯与滤纸，使溶液体积不超过 $40\mathrm{mL}$ 。

# 6.3.2 显色

于容量瓶中滴加3滴硫酸(3.1)，使溶液呈酸性后，加入 $2\mathrm{mL}$ 对甲氨基苯酚硫酸盐溶液(3.3)， $0.3\mathrm{mL}$ 明胶溶液(3.4)， $2\mathrm{mL}$ 三氯化铁溶液(3.7)，用水稀释至标线，混匀。于暗处静置 $40\mathrm{min}$ 。

# 6.3.3 测量

以空白试验溶液为参比，用 $30\mathrm{mm}$ 比色皿，于 $700\mathrm{nm}$ 波长处测定显色溶液吸光度。

从校准曲线(6.5.2)上查出试料中含铁氰酸根的量。

# 6.4 干扰的排除

6.4.1 当试料含银(I)时，在加入试剂前，预先加入氯化钠溶液(3.8)1 mL，使之形成氯化银沉淀。
6.4.2 当试料中汞（Ⅱ）的含量超过铁（Ⅱ、Ⅲ）氰络离子的0.05倍时，在用高锰酸钾-碳酸钠溶液消解之前，按其加入量，先以高锰酸钾(3.6)-氢氧化钠(3.11)溶液消解，使汞（Ⅱ）水解沉淀，经过滤除去后，再次调其 $\mathrm{pH}$ 值为 $6\sim 8$ ，接下操作。
6.4.3 试料如含有铬而在消解后因呈铬（VI）存在使试液呈现黄色时，在加入对甲氨基苯酚硫酸盐之前，滴加盐酸羟胺溶液(3.9)至黄色基本消失，但注意不宜过量。
6.4.4 显色时，室温如果低于 $20^{\circ}\mathrm{C}$ ，需适当延长放置时间。

# 6.5 校准

# 6.5.1 标准工作溶液的制备、显色和测量

分别量取0,0.50,1.00,2.00,3.00,4.00,5.00mL铁氰化钾标准使用液(3.12.2)于 $200~\mathrm{mL}$ 烧杯中，按6.3条测定步骤操作，以空白试验(零浓度)溶液为参比，进行其他各浓度标准工作溶液吸光度的测定。同时也可以水为参比，测定空白试验溶液吸光度。

# 6.5.2 校准曲线的绘制

用测定的吸光度和对应的铁氰酸根的量绘制校准曲线。

# 7. 结果的表示

铁 $(\mathbb{I},\mathbb{I})$ 氰络合物含量 $c$ 以铁氰酸根计，按下式计算：

$$
c = \frac {m}{V}
$$

式中： $c$ ——水样中铁（Ⅱ、Ⅲ）氰络合物的含量， $\mathrm{mg / L}$

$m$ ——由校准曲线上查得试料中铁氰酸根的量, $\mu \mathrm{g}$ ;

$V$ ——试料的体积， $\mathrm{mL}$ 。

# 8 精密度和准确度

五个实验室对浓度 $2\sim 6\mathrm{mg / L}$ 范围的火工品工业废水及加标水样按第6章分析步骤进行测定。

# 8.1 精密度

相对标准偏差范围为 $0.8\% \sim 3.6\%$

# 8.2 准确度

加标回收率范围为 $96\% \sim 108\%$
