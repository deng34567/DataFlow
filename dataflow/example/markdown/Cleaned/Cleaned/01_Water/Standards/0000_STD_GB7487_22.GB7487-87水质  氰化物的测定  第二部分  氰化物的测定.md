氰化物属于剧毒物，在操作氰化物及其溶液时，要特别小心。避免沾污皮肤和眼睛，吸取溶液一定要用安全移液管或用洗耳球吸溶液，切勿吸入口中！

除氰化物剧毒外，吡啶也具有毒性，应注意安全使用。

氰化物可能以氰氢酸、氰离子和络合氰化物的形式存在于水中，这些氰化物可作为总氰化物和氰化物加以测定。

本标准适用于饮用水、地面水、生活污水和工业废水。

活性氯等氧化物干扰，使结果偏低，可在蒸馏前加亚硫酸钠溶液排除干扰，见GB7486—876.1.7a。

硫化物干扰，可在蒸馏前加碳酸铅或碳酸镉排除干扰，见GB7486-876.1.7c。

亚硝酸离子干扰，可在蒸馏前加适量氨基磺酸排除干扰，见GB7486-876.1.7b。

少量油类对测定无影响，中性油或酸性油大于 $40\mathrm{mg / L}$ 时干扰测定，可加入水样体积的 $20\%$ 量的正已烷，在中性条件下短时间萃取排除干扰。

本标准分四篇：

第一篇氰化氢的释放和吸收；

第二篇硝酸银滴定法；

第三篇异烟酸－吡唑啉酮比色法；

第四篇吡啶－巴比妥酸比色法。

硝酸银滴定法最低检测浓度为 $0.25\mathrm{mg / L}$ ，检测上限为 $100\mathrm{mg / L}$ 。

异烟酸-吡唑啉酮比色法最低检测浓度为 $0.004\mathrm{mg / L}$ ，检测上限为 $0.25\mathrm{mg / L}$ 。

吡啶-巴比妥酸比色法最低检测浓度为 $0.002\mathrm{mg / L}$ （用72型分光光度计吸光度为0.020左右），检测上限为 $0.45\mathrm{mg / L}$ （10mm比色皿）、 $0.15\mathrm{mg / L}$ （30mm比色皿）。

# 第一篇氰化氢的释放和吸收

# 1 定义

在 $\mathbf{pH}4$ 的介质中，硝酸锌存在下，加热蒸馏，能形成氰化氢的氰化物，包括全部简单氰化物（碱金属的氰化物和碱土金属的氰化物）和锌氰络合物，不包括铁氰化物、亚铁氰化物、铜氰络合物、镍氰络合物、钴氰络合物。

# 2 原理

向水样中加入酒石酸和硝酸锌，在 $\mathbf{pH}4$ 的条件下，加热蒸馏，简单氰化物和部分络合氰化物（如锌氰络合物）以氰化氢形式被蒸馏出，并用氢氧化钠吸收。

# 3 试剂

3.1 $15\% (m / V)$ 酒石酸溶液。

称取 $150\mathrm{g}$ 酒石酸（ $\mathbf{C}_4\mathbf{H}_6\mathbf{O}_6$ ，tartaric acid）溶于 $1000\mathrm{ml}$ 水中。

3.2 $0.05\% (m / V)$ 甲基橙指示剂。
3.3 $10\% (m / V)$ 硝酸锌 $\left[\mathrm{Zn}\left(\mathrm{NO}_{3}\right)_{2} \cdot 6 \mathrm{H}_{2} \mathrm{O}\right]$ 溶液。
3.4 乙酸铅试纸。

称取 $5\mathrm{g}$ 乙酸铅 $\left[\mathrm{Pb}\left(\mathrm{C}_{2}\mathrm{H}_{3}\mathrm{O}_{2}\right)_{2} \cdot 3\mathrm{H}_{2}\mathrm{O}\right]$ 溶于水中，并稀释至 $100\mathrm{ml}$ 。将滤纸条浸入上述溶液中， $1\mathrm{h}$ 后，取出晾干，盛于广口瓶中，密塞保存。

3.5 碘化钾-淀粉试纸。

称取 $1.5\mathrm{g}$ 可溶性淀粉，用少量水搅成糊状，加入 $200\mathrm{mL}$ 沸水，混匀，放冷，加 $0.5\mathrm{g}$ 碘化钾和 $0.5\mathrm{g}$ 碳酸钠，用水稀释至 $250\mathrm{mL}$ ，将滤纸条浸渍后，取出晾干，盛于棕色瓶中，密塞保存。

3.6 1 + 5 硫酸溶液。
3.7 $1.26\% (m / V)$ 亚硫酸钠（ $\mathrm{Na}_{2}\mathrm{SO}_{3}$ ）溶液。
3.8 氨基磺酸（ $\mathrm{NH_2SO_2OH}$ ，sulfamic acid）。
3.9 $4\% (m / V)$ 氢氧化钠（NaOH）溶液。
3.10 $1\% (m / V)$ 氢氧化钠（NaOH）溶液。

# 5 采样和样品

5.1 采集水样时，必须立即加氢氧化钠固定，一般每升水样加0.5g固体氢氧化钠，当水样酸度高时，应多加固体氢氧化钠，使样品的 $\mathsf{pH} > 12$ ，并将样品存于聚乙烯塑料瓶或硬质玻璃瓶中。
5.2 当水样中含有大量硫化物时，应先加碳酸镉 $(\mathrm{CdCO}_3)$ 、碳酸铅 $(\mathrm{PbCO}_3)$ 固体粉末，除去硫化物后，再加氢氧化钠固定。否则，在碱性条件下，氰离子和硫离子作用形成硫氰酸离子而干扰测定。

注：检验硫化物方法，可取1滴水样或样品，放在乙酸铅试纸（3.4）上，若变黑色（硫化铅），说明有硫化物存在。

5.3 如果不能及时测定样品，采样后应在24h内要分析样品，必须将样品存放在冷暗的冰箱内。

# 6 步骤

# 6.1 氰化氢释放和吸收

6.1.1 量取 $200\mathrm{ml}$ 样品，移入 $500\mathrm{ml}$ 蒸馏瓶（2）中（若氰化物含量高，可少取样品，加水稀释至 $200\mathrm{ml}$ ），加数粒玻璃珠。
6.1.2 往接收瓶（4）内加入 $10\mathrm{ml}$ 氢氧化钠（3.10），作为吸收液。

当样品中存在亚硫酸钠和碳酸钠时，可用 $4\%$ 氢氧化钠溶液（3.9）作为吸收液。

6.1.3 馏出液导管（5）上端接冷凝管的出口，下端插入接收瓶（4）的吸收液中，检查连接部位，使其严密。
6.1.4 将 $10\mathrm{ml}$ 硝酸锌溶液（3.3）加入蒸馏瓶（2）内，加入 $7\sim 8$ 滴甲基橙指示剂（3.2），迅

速加入 $5\mathrm{ml}$ 酒石酸溶液（3.1），立即盖好瓶塞，使瓶内溶液保持红色，打开冷凝水，馏出液以 $2\sim$ $4\mathrm{ml / min}$ 速度进行加热蒸馏。

6.1.5 接收瓶（4）内溶液近 $100\mathrm{ml}$ 时停止蒸馏，用少量水洗涤馏出液导管（5），取出接收瓶（4），用水稀释至标线，此碱性馏出液“C”待测定氰化物用。

# 6.2 空白试验

按步骤6.1.1至6.1.5操作，用实验用水代替样品，进行空白试验，得到空白试验馏出液“D”待测定氰化物用。

# 第二篇硝酸银滴定法

# 7 原理

经蒸馏得到的碱性馏出液“C”，用硝酸银标准溶液（8.4）滴定，氰离子与硝酸银作用形成可溶性的银氰络合离子 $[\mathbf{Ag}(\mathbf{CN})_2]$ ，过量的银离子与试银灵指示剂（8.1）反应，溶液由黄色变为橙红色，进行比色测定。

# 8 试剂

# 8.1 试银灵指示剂。

称取0.02g试银灵（对二甲氨基亚苄基罗丹宁，paradimethylaminobenzalrhodanine）溶于100ml丙酮中。贮存在棕色瓶中并于暗处可稳定一个月。

# 8.2 铬酸钾 $\left(\mathrm{K}_{2} \mathrm{CrO}_{4}\right)$ 指示剂。

称取 $10\mathrm{g}$ 铬酸钾溶于少量水中，滴加硝酸银溶液（8.4）至产生橙红色沉淀为止，放置过夜后，过滤，用水稀释至 $100\mathrm{ml}$ 。

# 8.3 氯化钠（NaCl）标准溶液：0.01mol/L。

将氯化钠置瓷坩埚内，经 $500\sim 600^{\circ}C$ 灼烧至爆烈声后，在干燥器内冷却，称取 $0.5844\mathrm{g}$ 于烧杯内，用水溶解，移入 $1000\mathrm{ml}$ 容量瓶，并稀释至标线，混合摇匀。

# 8.4 硝酸银标准溶液： $0.01\mathrm{mol} / \mathrm{L}$ 。

8.4.1 称取 $1.699\mathrm{g}$ 硝酸银溶于水中，稀释至 $1000\mathrm{ml}$ ，贮于棕色试剂瓶中，摇匀，待标定后使用。

# 8.4.2 硝酸银溶液的标定：

a. 吸取 $0.01 \mathrm{~mol} / \mathrm{L}$ 氯化钠标准溶液(8.3) $10.00 \mathrm{ml}$ , 于 $150 \mathrm{ml}$ 具柄瓷皿或锥形瓶（9.2）中, 加 $50 \mathrm{ml}$ 水, 同时另取一具柄瓷皿或锥形瓶（9.2）, 加入 $60 \mathrm{ml}$ 水作空白试验。
b. 向溶液中加入 $3 \sim 5$ 滴铬酸钾指示剂（8.2），在不断搅拌下，从滴定管加入待标定的硝酸银溶液（8.4.1）直至溶液由黄色变成浅砖红色为止。记下读数 $(V)$ 。同样滴定空白溶液，读数 $(V_{0})$ 。

硝酸银浓度 $c_{1}$ （mol/L）按式（1）计算：

$$
c _ {1} = \frac {c \times 1 0 . 0 0}{V - V _ {0}} \quad \dots \
$$

式中： $c$ ——氯化钠标准溶液浓度， $\mathrm{mol} / \mathrm{L}$ ；

$\pmb{\nu}$ ——滴定氯化钠标准溶液时硝酸银溶液用量， $\mathbf{m}1$

$V_{0}$ ——滴定空白溶液时硝酸银溶液用量， $\mathfrak{m}1$ 。

8.5 硝酸银标准溶液： $0.001\mathrm{mol} / \mathrm{L}$ 。
8.6 $2\% (m / V)$ 氢氧化钠（NaOH）溶液。

# 9 仪器

9.1 $10\mathrm{ml}$ 棕色酸式滴定管。
9.2 $150\mathrm{ml}$ 具柄瓷皿或 $250\mathrm{ml}$ 锥形瓶。

# 10 步骤

# 10.1 测定

10.1.1 取 $100\mathrm{ml}$ 馏出液“C”（如试样中氰化物含量高时，可少取试样，用水稀释至 $100\mathrm{ml}$ ）于具柄瓷皿或锥形瓶（9.1）中。
10.1.2 加入 $0.2 \mathrm{ml}$ 试银灵指示剂（8.1），摇匀，用硝酸银标准溶液（8.4）滴定至溶液由黄色变为橙红色为止。记下读数 $V_{\mathrm{a}}$ 。

# 10.2 空白试验

另取 $100\mathrm{ml}$ 空白试验馏出液“D”于锥形瓶（9.2）中，按10.1.2进行滴定，记下读数 $V_{0}$ 。

注：若样品氰化物浓度小于 $1\mathrm{mg / L}$ ，可用 $0.001\mathrm{mol / L}$ 硝酸银标准溶液（8.5）滴定。

# 11 结果的表示

# 11.1 计算方法

氰化物含量 $c_{2}(\mathrm{mg} / \mathrm{L})$ 以氰离子（CN）计，按式（2）计算：

$$
c _ {2} = \frac {c \left(V _ {1} - V _ {0}\right) \times 5 2 . 0 4 \times \frac {V _ {1}}{V _ {2}} \times 1 0 0 0}{V} \dots \tag {2}
$$

式中： $c$ ——硝酸银标准溶液浓度， $\mathrm{mol} / \mathrm{L}$ ；

$V_{\mathrm{s}}$ ——测定试样时硝酸银标准溶液用量， $\mathrm{ml}$ ；

$V_{0}$ ——空白试验硝酸银标准溶液用量，ml；

$\pmb{\nu}$ ——样品体积，ml；

$V_{1}$ ——试样（馏出液“C”）的体积，ml；

$V_{2}$ ——试份（测定时，所取馏出液“C”）的体积， $\mathfrak{m}1$ ；

52.04——相当于 $1 \mathrm{~L}$ 的 $1 \mathrm{~mol} / \mathrm{L}$ 硝酸银标准溶液的氰离子（ $2 \mathrm{CN}^{-}$ ）质量， $\mathbf{g}$ 。

# 第三篇异烟酸－吡唑啉酮比色法

# 12 原理

在中性条件下，样品中的氰化物与氯胺T反应生成氯化氰，再与异烟酸作用，经水解后生成戊烯二醛，最后与吡唑啉酮缩合生成蓝色染料，此染料与氰化物的含量成正比，进行比色测定。

# 13 试剂

13.1 $2\% (m / V)$ 氢氧化钠（NaOH）溶液。
13.2 $0.1\% (m / V)$ 氢氧化钠（NaOH）溶液。
13.3 磷酸盐缓冲溶液（ $\mathbf{pH} = 7$ ）。

称取 $34.0\mathrm{g}$ 无水磷酸二氢钾（ $\mathrm{KH_2PO_4}$ ）和 $35.5\mathrm{g}$ 无水磷酸氢二钠（ $\mathrm{Na_2HPO_4}$ ）于烧杯内，加水溶解后，稀释至 $1000\mathrm{ml}$ ，摇匀，存于冰箱。

13.4 $1\% (m / V)$ 氯胺T溶液。

临用前，称取 $1.0\mathrm{g}$ 氯胺 $\mathrm{T}$ （ $\mathrm{C},\mathrm{H},\mathrm{Cl}\mathrm{N}\mathrm{a}\mathrm{O}_{2}\mathrm{S}\cdot 3\mathrm{H}_{2}\mathrm{O}$ ，chloramine-T）溶于水，并稀释至 $100\mathrm{ml}$ ，摇匀，贮存于棕色瓶中。

13.5 异烟酸-吡唑啉酮溶液。

13.5.1 异烟酸溶液。

称取 $1.5\mathrm{g}$ 异烟酸 $\left(\mathrm{C}_{6} \mathrm{H}_{5} \mathrm{NO}_{2}\right.$ ，iso-nicotinic acid)溶于 $24 \mathrm{ml} 2 \%$ 氢氧化钠溶液（13.1）中，加水稀释至 $100 \mathrm{ml}$ 。

13.5.2 吡唑啉酮溶液。

称取 $0.25\mathrm{g}$ 吡唑啉酮（3-甲基-1-苯基-5-吡唑啉酮， $\mathbf{C}_{10}\mathbf{H}_{10}\mathbf{O}\mathbf{N}_{2},3-$ methy-1-phenyl-5-pyrazolone）溶于 $20\mathrm{ml}\mathbf{N}$ ，N-二甲基甲酰胺〔HCON（CH）2，N，N-dimethyl formamide]中。

临用前，将吡唑啉酮溶液（13.5.2）和异烟酸溶液（13.5.1）按 $1 + 5$ 混合。

13.6 氰化钾（KCN）标准溶液。

13.6.1 氰化钾贮备溶液的配制和标定：

称取0.25g氰化钾（KCN，注意剧毒！）溶于氢氧化钠（13.2）中，并稀释至 $100\mathrm{ml}$ ，摇匀，避光贮存于棕色瓶中。

吸取 $10.00\mathrm{ml}$ 氰化钾贮备溶液于锥形瓶（9.2）中，加入 $50\mathrm{ml}$ 水和 $1\mathrm{ml}2\%$ 氢氧化钠溶液（13.1），加入 $0.2\mathrm{ml}$ 试银灵指示剂（8.1），用硝酸银标准溶液（8.4）滴定，溶液由黄色刚变为橙红色为止，记录硝酸银标准溶液用量 $(V_{1})$ 。同时另取 $10.00\mathrm{ml}$ 实验用水代替氰化钾贮备液作空白试验，记录硝酸银标准溶液用量 $(V_{0})$ 。

氰化物含量 $c_{3}(\mathrm{mg / ml})$ 以氰离子（CN-）计，按式（3）计算：

$$
c _ {3} = \frac {c \left(V _ {1} - V _ {0}\right) \times 5 2 . 0 4}{1 0 . 0 0} \dots \tag {3}
$$

式中： $c$ ——硝酸银标准溶液浓度， $\mathrm{mol} / \mathrm{L}$

$V_{1}$ ——滴定氰化钾贮备溶液时硝酸银标准溶液用量， $\mathfrak{m}1$ ；

$V_{0}$ ——空白试验硝酸银标准溶液用量， $\mathfrak{m}1$ ；

52.04 ——相当于 $1 \mathrm{~L}$ 的 $1 \mathrm{~mol} / \mathrm{L}$ 硝酸银标准溶液的氰离子（ $2 \mathrm{CN}^{-}$ ）的质量，g；

10.00 ——氰化钾贮备液体积，ml。

13.6.2 氰化钾标准中间溶液（1.00ml含 $10.00\mu \mathrm{g}$ 氰离子）。

先按式（4）计算出配制 $500\mathrm{ml}$ （1.00ml含 $10.0\mu \mathrm{g}$ 的氰离子）氰化钾标准中间溶液时，所需氰化钾贮备溶液（13.6.1）的体积（ $V$ ，ml）：

$$
V = \frac {1 0 . 0 0 \times 5 0 0}{T \times 1 0 0 0} \dots \tag {4}
$$

式中： $T\times 1000 - 1\mathrm{ml}$ 氰化钾贮备溶液中氰化物含量， $\mu \mathrm{g}$

10.00 —— 1 ml 氰化钾标准中间溶液含 $10.00 \mu \mathrm{g}$ 氰离子；

500——氰化钾标准中间溶液体积，ml。

准确吸取 $V$ （ml）氰化钾贮备溶液（13.6.1）于 $500\mathrm{ml}$ 棕色容量瓶中，用氢氧化钠溶液（13.2）稀释至标线，摇匀。

13.6.3 氰化钾标准使用溶液（1.00ml含1.00μg氰离子）。

临用前吸取 $10.00\mathrm{ml}$ 氰化钾标准中间溶液（13.6.2）于 $100\mathrm{ml}$ 棕色容量瓶中，用氢氧化钠溶液（13.2）稀释至标线，摇匀。

14.1 分光光度计或比色计。

14.2 25ml具塞比色管。

# 15 步骤

15.1 校准

15.1.1 取8支具塞比色管（14.2），分别加入氰化钾标准使用溶液（13.6.3）0、0.20、0.50、1.00、2.00、3.00、4.00和5.00ml，各加氢氧化钠溶液（13.2）至10ml。

# 16.1 计算方法

氰化物含量 $c_{4}(\mathrm{mg} / \mathrm{L})$ 以氰离子（CN-）计，按式（5）计算：

$$

式中： $m_{\mathrm{a}}$ ——从校准曲线上查出试份的氰化物含量， $\mu \mathrm{g}$

$V_{1}$ ——样品的体积， $\mathbf{m}1$ ；

# 第四篇吡啶－巴比妥酸比色法

# 17 原理

在中性条件下，氰离子和氯胺T的活性氯反应生成氯化氰，氯化氰与吡啶反应生成戊烯二醛（glutacondialdehyde），戊烯二醛与两个巴比妥酸分子缩合生成红紫色染料，进行比色测定。

# 18 试剂

18.1 1 + 3 盐酸 (HCl)。
18.2 吡啶-巴比妥酸溶液。

临用前，称取0.18g巴比妥酸（ $\mathrm{C_4H_4N_2O_3}$ ，barbituric acid），加入3ml吡啶（ $\mathrm{C_5H_5N}$ ，pyridine）及10ml盐酸（18.1），待溶解后，加水至100ml，摇匀，贮存在棕色瓶中。本溶液若有不溶物可过滤，存于暗处可稳定一天，存放于冰箱中可稳定一周。

18.3 磷酸盐缓冲溶液（pH7）。

称取 $2.79\mathrm{g}$ 无水磷酸二氢钾（ $\mathrm{KH}_2\mathrm{PO}_4$ ）和 $4.14\mathrm{g}$ 无水磷酸氢二钠（ $\mathrm{Na}_2\mathrm{HPO}_4$ ），溶于水中，稀释至 $1000\mathrm{ml}$ 。

18.4 0.5mol/L盐酸溶液。
18.5 $0.1\% (m / V)$ 酚酞指示剂。

# 19 仪器

19.1 分光光度计或比色计。
19.2 $25\mathrm{m}1$ 具塞比色管。

# 20 步骤

# 20.1 校准

20.1.1 取8支具塞比色管（19.2），分别加入氰化钾标准使用溶液（13.6.3）0、0.20、0.50、1.00、2.00、3.00、4.00和5.00ml，各加氢氧化钠溶液（13.2）至10ml。
20.1.2 向各管中加入 1 滴酚酞指示剂（18.5），用盐酸（18.4）调节溶液红色刚消失为止。
20.1.3 加入 $5\mathrm{ml}$ 磷酸盐缓冲溶液，摇匀，迅速加入 $0.2\mathrm{ml}$ 氯胺T溶液（13.4），立即盖塞子，混匀，放置 $3\sim 5\mathrm{min}$ ，再加入 $5\mathrm{ml}$ 吡啶-巴比妥酸溶液（18.2），加水稀释至标线，混匀。
