# 中华人民共和国国家标准

# 水质一甲基肼的测定对二甲氨基苯甲醛分光光度法

GB/T14375-93

Water quality-Determination of monomethyl hydrazine

p-Dimethylaminobenzaldehyde spectrophotometric method

# 1 主题内容与适用范围

# 1.1 主题内容

本标准规定了测定水中一甲基肼的对二甲氨基苯甲醛分光光度法。

# 1.2 适用范围

1.2.1 本方法适用于地面水、航天工业废水中一甲基肼的测定。
1.2.2 一甲基肼的测定范围为 $0.02\sim 0.80\mathrm{mg / L}$ 。水样中一甲基肼含量大于 $0.80\mathrm{mg / L}$ 时，可稀释后测定。
1.2.3 肼干扰一甲基肼的测定。偏二甲基肼含量高于一甲基肼时，可用校正曲线预以校正。

# 2 方法原理

在酸性条件下，水中微量一甲基肼与对二甲氨基苯甲醛反应生成黄色缩合物，在测定范围内颜色的深度与一甲基肼的含量成正比，用分光光度计在 $470~\mathrm{nm}$ 处测定。

# 3 试剂

除另有说明外，均使用符合国家标准或专业标准的分析纯试剂和蒸馏水或等纯度的水。

3.1 硫酸： $\rho = 1.84\mathrm{g / mL}$
3.2 乙醇： $95\%$ 以上。
3.3 一甲基肼 $\mathrm{(CH_3NHNH_2)}$ ：纯度 $98\%$ 以上。
3.4 硫酸溶液： $c(\mathrm{H}_2\mathrm{SO}_4) = 1.00\mathrm{mol / L}$
3.5 硫酸溶液： $c(\mathrm{H}_2\mathrm{SO}_4) = 0.05\mathrm{mol / L}$
3.6 对二甲氨基苯甲醛溶液：称取对二甲氨基苯甲醛 $\left[\left(\mathrm{CH}_{3}\right)_{2} \mathrm{NC}_{6} \mathrm{H}_{4} \mathrm{CHO}\right] 5.0 \mathrm{~g}$ ，加入硫酸溶液（3.4） $20 \mathrm{~mL}$ ，混匀后加入乙醇（3.2） $100 \mathrm{~mL}$ ，使其溶解。
3:7 氨基磺酸铵或氨基磺酸溶液： $1\mathrm{g} / 100\mathrm{mL}$ 。

称取氨基磺酸铵 $\mathrm{(NH_4SO_3NH_2)}$ 或氨基磺酸 $\mathrm{(NH_2SO_3H)}$ $1.0\mathrm{g}$ ，溶于 $100~\mathrm{mL}$ 水中。

3.8 一甲基肼贮备液： $10\mathrm{mg / mL}$

吸取硫酸溶液 $(3.4)5\sim 10\mathrm{mL}$ 于 $25\mathrm{mL}$ 容量瓶中，称准至 $0.0001\mathrm{g}$

用注射器吸取一甲基肼(3.3) $0.3\mathrm{mL}$ ，注入上述容量瓶中，轻轻摇动瓶子，再次称重至 $0.0001\mathrm{g}$ ，用硫酸溶液(3.4)稀释至标线。

3.9 一甲基肼溶液： $200\mu \mathrm{g / mL}$

吸取溶液(3.8) $2.0\mathrm{mL}$ ，移入 $100\mathrm{mL}$ 容量瓶中，用硫酸溶液(3.5)稀释至标线，在 $2\sim 5^{\circ}\mathrm{C}$ 下保存。

3.10 一甲基肼标准溶液： $2\mu \mathrm{g / mL}$

吸取一甲基肼溶液 $(3.9)5\mathrm{mL}$ ，移入 $500~\mathrm{mL}$ 容量瓶中，用硫酸溶液(3.5)稀释至标线，混匀。

# 4 仪器

4.1 分光光度计：配 $2\mathrm{cm}$ 比色皿。
4.2 玻璃仪器：
4.2.1 比色管： $25\mathrm{mL}$
4.2.2 容量瓶： $500~\mathrm{mL},100~\mathrm{mL},25~\mathrm{mL}$

# 5 分析步骤

# 5.1 校准曲线的绘制

5.1.1 不存在亚硝酸盐时标准曲线的绘制：

5.1.1.1取一组 $25\mathrm{mL}$ 的容量瓶，分别注入 $0.00,0.04,0.08,0.20,0.40,0.80,2.00,4.00\mathrm{mL}$ 一甲基肼标准溶液(3.10)，加入乙醇(3.2) $4.5\mathrm{mL}$ ，加入显色剂(3.6) $5.0\mathrm{mL}$ 用硫酸溶液(3.5)稀释至标线，摇匀。

5.1.1.2 放置 $40\mathrm{min}$ 后，在分光光度计 $470\mathrm{nm}$ 处，以试剂空白液为参比液，使用 $2\mathrm{cm}$ 光程比色皿测定吸光度。

5.1.1.3 根据测得的吸光度与相应的一甲基肼含量，绘制校准曲线，求出回归方程 $Y = bX + a$ 。

5.1.2 存在亚硝酸盐时校准曲线的绘制：按5.1.1.1取一甲基肼标准溶液后，加入 $0.2\mathrm{mL}$ 氨基磺酸铵溶液(3.7)，其余步骤与5.1.1相同。

5.2 测定

5.2.1 采样和样品处理

5.2.1.1 采样：用玻璃瓶采样。

5.2.1.2 样品处理：量取 $500~\mathrm{mL}$ 样品，用硫酸(3.1)，将水样调制 $\mathrm{pH}$ 值1.0左右。

5.2.2 水样中无亚硝酸盐和其他肼类时的测定方法。

5.2.2.1 吸取水样 $15\mathrm{mL}$ 于 $25\mathrm{mL}$ 比色管中，加入乙醇(3.2) $4.5\mathrm{mL}$ ，对二甲氨基苯甲醛溶液(3.6) $5.0\mathrm{mL}$ ，摇匀后用硫酸溶液(3.5)稀释至标线。放置 $40\mathrm{min}$ 。

5.2.2.2 于 $470 \mathrm{~nm}$ 波长，用 $2 \mathrm{~cm}$ 光程的比色皿，以蒸馏水为参比液，测定溶液的吸光度，扣除空白吸光度，从校准曲线上查得或按回归方程算得相应的一甲基肼含量 $(\mu \mathrm{g})$ 。

# 5.2.3 水中存在亚硝酸盐时的测定方法

吸取水样 $15\mathrm{mL}$ 于 $25\mathrm{mL}$ 比色管中，加入氨基磺酸铵(3.7) $0.2\mathrm{mL}$ ，其余步骤按5.2.2进行。在 $25\mathrm{mL}$ 定容体积中，若 $\mathrm{NO}_2^-$ 总量超过 $20\mu \mathrm{g}$ ，应同时测加标回收率2个，测得值除以回收率为一甲基肼含量。平行样之间的相对偏差不超过 $10\%$ 。

5.2.4 水中存在偏二甲基肼时的测定方法
5.2.4.1 水样中偏二甲基肼含量高于一甲基肼时，应先按偏二甲基肼测定方法测出偏二甲基肼的含量。
5.2.4.2 按5.1.1或5.1.2制作偏二甲基肼校正曲线，并在曲线上查得偏二甲基肼含量相应的吸光度 $A_{1}$ 。
5.2.4.3 按5.2.2或5.2.3操作，记取吸光度 $A_{2}$ 。
5.2.4.4 $A_{2} - A_{1} = A_{3}$ ，用 $A_{3}$ 值在校准曲线上查得或用回归方程计算出水样中一甲基肼含量。

# 6 结果的表述

一甲基肼含量 $(C,\mathrm{mg / L})$ 按下式计算：

$$
C = \frac {W}{V}
$$

式中： $W$ ——校准曲线上查得或按回归方程算出的水样中一甲基肼含量， $\mu \mathrm{g}$

$V$ ——水样体积， $\mathrm{mL}$ 。

# 7 精密度

一甲基肼浓度低于 $0.10\mathrm{mg / L}$ 时，相对标准偏差不大于 $15\%$ ； $0.10\sim 0.80\mathrm{mg / L}$ 时，相对标准偏差不大于 $3.6\%$ 。

# 8注意事项

8.1 本方法灵敏度随温度升高而降低，水样的一甲基肼含量测定应与校准曲线制作同时进行。
8.2 本方法温度适应范围 $15\sim 30^{\circ}C$
