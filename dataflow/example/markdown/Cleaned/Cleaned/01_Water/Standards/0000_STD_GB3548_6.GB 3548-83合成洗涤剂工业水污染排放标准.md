# 合成洗涤剂工业污染物排放标准

GB 3548-83

Effluent standard for pollutants from synthetic detergent industry

本标准为贯彻《中华人民共和国环境保护法（试行）》，防治合成洗涤剂工业废水和粉尘对环境的污染而制定。

本标准适用于全国合成洗涤剂企业。

# 1. 标准的分级

# 1.1 合成洗涤剂工业污染物排放标准分为二级

第一级：指所有新建、扩建、改建企业，自标准实施之日起立即执行的标准。

第二级：指所有现有企业，自标准实施之日起立即执行的标准。

# 3 其它规定

3.1 表1至表3所列污染物标准值，除PH值外，其他项目均以月平均值为考核依据。抽检时连续取样12小时以上可代表一天，并以当时产量计算，其最大值不应超过日平均值。

# 3.2 排水量

排水量是指生产每吨产品的排水体积。

凡能够按日准确测定并记录排水量的企业，按排水量计算。否则可按所用清水计算。排水量的计算范围包括：直接生产排水，厂区生活排水，但不包括锅炉耗水和电站耗水。

# 3.3 非单一产品生产企业的污染负荷

非单一产品生产企业的污染物容许排放量等于各类产品的日产量乘以每吨产品的排放标准的总和。

3.4 当地方执行本标准不适用于当地环境特点（如集中生活水源，经济渔业区等）时，可以按国家有关规定制订地方污染物排放标准。

# 4 标准的监测

4.1 制定本标准依据的监测分析方法是《合成洗涤工业污染物监测分析方法》。
