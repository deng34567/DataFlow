# 水质铜的测定

# 2,9-二甲基-1,10-菲啰啉分光光度法

UDC 614.777:543.42

:546.56

GB7473-87

Water quality-Determination of copper

2, 9- Dimethy-1, 10-phenanthroline

spectrophotometric method

# 1 适用范围

本标准适用于地面水、生活污水和工业废水中铜的测定。

在被测溶液中，如有大量的铬和锡、过量的其他氧化性离子、以及氰化物、硫化物和有机物等对测定铜有干扰。加入亚硫酸使铬酸盐和络合的铬离子还原，可以避免铬的干扰。加入盐酸羟胺溶液\*，可以消除锡和其他氧化性离子的干扰。通过消解过程，可以除去氰化物、硫化物和有机物的干扰。

取 $50\mathrm{m}1$ 试份，比色皿光程 $10\mathrm{mm}$ ，铜的最低检测浓度为 $0.06\mathrm{mg / L}$ ，测定上限为 $3\mathrm{mg / L}$ 。

# 2 定义

2.1 可溶性铜：未经酸化的水样，通过 $0.45\mu \mathrm{m}$ 滤膜后测得的铜浓度。
2.2 总铜：未经过滤的水样，经剧烈消解后测得的铜浓度。

# 3 原理

用盐酸羟胺把二价铜离子还原为亚铜离子，在中性或微酸性溶液中，亚铜离子和2，9-二甲基-1，10-菲啰啉反应生成黄色络合物，可被多种有机溶剂（包括氯仿-甲醇混合液）萃取，在波长 $457\mathsf{nm}$ 处测量吸光度。

在 $25\mathrm{ml}$ 有机溶剂中，含铜量不超过 $0.15\mathrm{mg}$ 时，显色符合比耳定律。在氯仿-甲醇混合液中，该颜色可保持数日。

# 4 试剂

在测定过程中，均使用去离子水或全玻璃蒸馏器制得的重蒸馏水。除另有说明外，均使用公认的分析纯试剂。

4.1 硫酸 $\left(\mathrm{H}_{2} \mathrm{SO}_{4}\right): \rho_{20} = 1.84 \mathrm{~g} / \mathrm{ml}$ ，优级纯。
4.2 硝酸（HNO3）： $\rho_{20} = 1.40\mathrm{g / ml}$ ，优级纯。
4.3 盐酸（HC1）： $\rho_{20} = 1.19\mathrm{g / ml}$ 。
4.4 氯仿（CHCl3）。
4.5 甲醇（CH3OH）：99.5%（V/V）。
4.6 盐酸羟胺： $100\mathrm{g} / \mathrm{L}$ 溶液。

将 $50\mathrm{g}$ 盐酸羟胺（ $\mathrm{NH_2OH\cdot HCl}$ ）溶于水并稀释至 $500\mathrm{m}1$ 。

4.7 柠檬酸钠： $375\mathrm{g} / \mathrm{L}$ 溶液。

将 $150\mathrm{g}$ 柠檬酸钠（ $\mathrm{Na}_{3}\mathrm{C}_{6}\mathrm{H}_{5}\mathrm{O}_{7}\cdot 2\mathrm{H}_{2}\mathrm{O}$ ）溶解于 $400\mathrm{ml}$ 水中，加入 $5\mathrm{ml}$ 盐酸羟胺溶液（4.6）和 $10\mathrm{ml}2$ ，9-二甲基-1，10-菲啰啉溶液（4.9），用 $50\mathrm{ml}$ 氯仿（4.4）萃取以除去其中的杂质铜，弃去氯仿层。

4.8 氢氧化铵： $5\mathrm{mol} / \mathrm{L}$ 溶液。

量取 $330\mathrm{m}1$ 氢氧化铵（ $\mathrm{NH_4OH}$ ： $\rho_{20} = 0.90\mathrm{g / ml}$ ），用水稀释至 $1000\mathrm{m}1$ ，贮存于聚乙烯瓶中。

4.9 2，9-二甲基-1，10-菲啰啉： $1\mathrm{g} / \mathrm{L}$ 溶液。

将 $100\mathrm{mg}2$ ，9-二甲基-1，10-菲啰啉 $(\mathbf{C}_{14}\mathbf{H}_{12}\mathbf{N}_{2}\cdot \frac{1}{2}\mathbf{H}_{2}\mathbf{O})$ 溶于 $100\mathrm{ml}$ 甲醇(4.5)中。这种溶液在普通贮存条件下，可稳定一个月以上。

4.10 铜：相当于 $0.20\mathrm{mg / ml}$ 铜的标准溶液。

称取 $0.2000 \pm 0.0001 \mathrm{~g}$ 抛光的电解铜丝或铜箔（纯度 $99.9\%$ 以上），置于 $250 \mathrm{~mL}$ 锥形瓶中，加入 $10 \mathrm{~mL}$ 水和 $5 \mathrm{~mL}$ 硝酸（4.2），直到反应速度变慢时微微加热，使全部铜溶解。接着，煮沸溶液以驱除氮的氧化物。冷却后加入 $50 \mathrm{~mL}$ 水，定量转移到 $1000 \mathrm{~mL}$ 容量瓶中，用水稀释至标线并混匀。

4.11 铜：相当于 $20.0\mu \mathrm{g} / \mathrm{m}1$ 铜的标准溶液。

吸取 $50.0\mathrm{ml}$ 铜标准溶液（4.10）置于 $500\mathrm{ml}$ 容量瓶中，用水稀释至标线并混匀。

4.12 铜：相当于 $2.0\mu \mathrm{g} / \mathrm{m}1$ 铜的标准溶液。

吸取 $10.00\mathrm{m}1$ 铜标准溶液（4.11）置于 $100\mathrm{m}1$ 容量瓶中，用水稀释至标线并混匀。

4.13 刚果红试纸或变色范围 $4 \sim 6$ 的 $\mathbf{pH}$ 试纸。

# 5 仪器

常用实验室设备及：

5.1 分光光度计：配有光程 $10\mathrm{mm}$ 和 $50\mathrm{mm}$ 比色皿。

5.2 125ml锥形分液漏斗：具有磨口玻璃塞，活塞上不得涂抹油性润滑剂。

5.3 $25\mathrm{m}1$ 容量瓶。

# 6 采样和样品

# 6.1 实验室样品

采集 $1000\mathrm{ml}$ 水样，立即进行测定，若不能立即测定，为了防止铜离子吸附在采样容器壁上，向每升水样中加入 $5\mathrm{ml}1 + 1$ 盐酸（4.3），酸化至 $\mathsf{pH}$ 约为1.5。采样容器宜用塑料桶。

# 6.2 试样

从水样中取两份均匀试样，每份 $100\mathrm{ml}$ 置于 $250\mathrm{ml}$ 烧杯中，作为7.2.1消解试样。

# 7 步骤

# 7.1 校准

取7个分液漏斗（5.2），分别加入铜标准溶液（4.11）0、1.00、2.00、3.00、4.00、6.00、8.00、10ml，加水至体积为50ml，加入1ml硫酸（4.1），按测定步骤7.2.2进行。

以氯仿（4.4）作参比，从测得铜标准溶液的吸光度中减去试剂空白（零浓度）吸光度后与相对应的铜含量 $(\mu \mathrm{g})$ 绘制校准曲线。

如果试份中铜的含量低于 $20\mu \mathrm{g}$ ，还需要绘制一条浓度系列更低的校准曲线。吸取铜标准溶液（4.12） $1.00 \sim 10.00 \mathrm{ml}$ ，按上述程序进行操作。测量吸光度时，改用 $50 \mathrm{~mm}$ 比色皿。

# 7.2 测定

# 7.2.1 消解

向每份试样（6.2）中加入 $1\mathrm{ml}$ 硫酸（4.1）和 $5\mathrm{ml}$ 硝酸（4.2），并放入几粒沸石后，置电热板上加热消解（注意勿喷溅）至冒三氧化硫白色浓烟为止。如果溶液仍然带色，冷却后加入 $5\mathrm{ml}$ 硝酸（4.2），继续加热消解至冒白色浓烟为止。必要时，重复上述操作，直到溶液无色。

冷却后加入约 $80\mathrm{ml}$ 水，加热至沸腾并保持 $3\mathrm{min}$ ，冷却后滤入 $100\mathrm{ml}$ 容量瓶内，用水洗涤烧杯和滤纸，用洗涤水补加至标线并混匀。

将第2份消解后的试样（D）保存起来，用于校核试验（7.4）。把第一份消解后的试样按下述步骤进行萃取和测定。

# 7.2.2 萃取和测定

从 $100\mathrm{ml}$ 消解试样溶液中吸取 $50.0\mathrm{ml}$ 或适量体积的试份（含铜量不超过 $0.15\mathrm{mg}$ ），置于分液漏斗（5.2）中，必要时，用水补足至 $50\mathrm{ml}$ ，加入 $5\mathrm{ml}$ 盐酸羟胺溶液（4.6）和 $10\mathrm{ml}$ 柠檬酸钠溶液（4.7），充分摇匀。按每次 $1\mathrm{ml}$ 的加入量加入氢氧化铵溶液（4.8），把 $\mathbf{pH}$ 值调到大约为4，每次加入少量的（或稀的）氢氧化铵溶液至刚果红试纸正好变红色（或 $\mathbf{pH}$ 试纸显示 $4\sim 6$ ）。

加入 $10\mathrm{m}12$ ，9-二甲基-1，10-菲啰啉溶液（4.9）和 $10\mathrm{m}1$ 氯仿（4.4），轻轻旋摇并放气，旋紧活塞后剧烈摇动30s以上，将黄色络合物萃入氯仿中，静置分层后，用滤纸吸去分液漏斗放液管内的水珠并塞入少量脱脂棉，把氯仿层放入容量瓶（5.3）中。再加入 $10\mathrm{m}1$ 氯仿于水相中，重复上述步骤再萃取一次。合并两次萃取液，用甲醇（4.5）稀释至标线并混匀。

将萃取液放入 $10\mathrm{mm}$ 比色皿内（如含铜量低于 $20\mu \mathrm{g}$ ，用 $50\mathrm{mm}$ 比色皿），在 $457\mathrm{nm}$ 处以氯仿（4.4）为参比，测量试份的吸光度。

用试份的吸光度减去空白试验（7.3）的吸光度后，从校准曲线（7.1）上查得铜的含量。

# 7.3 空白试验

用 $100\mathrm{ml}$ 水代替试样，按7.2.1和7.2.2步骤规定进行处理。空白与试样在相同条件下同时进行测定。

# 7.4 校准试验

从第2份消解试样（D）中吸取适量体积的溶液，加入铜标准溶液数毫升，使试份体积不超过 $50\mathrm{ml}$ ，含铜量不超过 $0.15\mathrm{mg}$ ，按7.2.2步骤进行萃取和测定，重复进行操作，以确定有无干扰影响。

# 8 结果表示

# 8.1 计算方法

含铜量 $c$ （ $\mathbf{mg} / \mathbf{L}$ ）按下式计算：

$$
c = \frac {m}{V}
$$

式中： $m$ ——试份测得铜量， $\mu \mathrm{g}$

$\pmb{\nu}$ ——萃取用的试份体积， $\mathbf{m}\mathbf{l}$ 。

计算结果以两位小数表示。
