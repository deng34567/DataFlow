# 纺织染整工业水污染物排放标准

# Discharge standard of water pollutants for dyeing and finishing of textile industry

1992-05-18 发布

1992-07-01实施

为贯彻《中华人民共和国环境保护法》、《中华人民共和国水污染防治法》和《中华人民共和国海洋环境保护法》，促进纺织染整行业生产工艺和污染治理技术的进步，防治水污染，制定本标准。

# 1 主题内容与适用范围

# 1.1 主题内容

本标准按照纺织染整企业的废水排放去向，分年限规定了纺织染整工业水污染物最高允许排放浓度及排水量。

# 1.2 适用范围

本标准适用于纺织染整工业企业的排放管理，以及建设项目的环境影响评价、设计、竣工验收及其建成后的排放管理。

本标准不适用于洗毛、麻脱胶、煮茧和化纤原料蒸煮等工序所产生的废水。

# 3 术语

# 3.1 染整 dyeing and finishing

对纺织材料(纤维、纱、线和织物)进行以化学处理为主的工艺过程。染整包括预处理、染色、印花和整理。俗称印染。

# 3.2 纺织品 textile

纺织工业产品, 包括各类机织物、无纺织布、各种缝纫包装用线、绣花线、绒线以及绳类、带类等。

# 5 监测

# 5.1 采样点

采样点应在企业废水排放口(六价铬在车间或车间处理设施排出口采样), 排放口应设置废水水量计量装置和永久性标志。

# 5.2 采样频率

按生产周期确定监测频率，生产周期在 $8\mathrm{h}$ 以内的，每 $2\mathrm{h}$ 采集一次；生产周期大于 $8\mathrm{h}$ 的，每 $4\mathrm{h}$ 采集一次。最高允许排放浓度按日均值计算。

# 5.3 排水量

排水量不包括冷却水及生产区非生产用水，其最高允许排水量按月均值计算。

# 5.4 统计

企业原材料使用量、产品产量等，以法定月报表或年报表为准。

# 附录A

# 废水中二氧化氯监测分析方法

# 连续滴定碘量法

（参考件）

# A1 适用范围

本法适用于亚漂设备及含有大量亚氯酸盐的废水。

# A2 原理

二氧化氯和亚氯酸根均是氧化剂，它们都能氧化碘离子而析出碘，继而用硫代硫酸钠滴定-碘量法，但在不同的 $\mathsf{pH}$ 值条件下，氧化数变化不同。

在 $\mathrm{pH} = 7, \mathrm{ClO}_2 + \mathrm{I}^- \rightarrow \mathrm{ClO}_2^- + \frac{1}{2}\mathrm{I}_2$ ，氧化数由 $4 \rightarrow 3$

在 $\mathsf{pH} = 1\sim 3,\mathrm{ClO}_2 + 5\mathrm{HI}\rightarrow \mathrm{H}^+ +\mathrm{Cl}^- +2\mathrm{H}_2\mathrm{O} + 2\frac{1}{2}\mathrm{I}_2$ ，氧化数由 $4\rightarrow -1$

$\mathrm{HClO}_2 + 4\mathrm{HI} \rightarrow 2\mathrm{I}_2 + \mathrm{HCl} + 2\mathrm{H}_2\mathrm{O}$ ，氧化数由 $3 \rightarrow 1$

因此，可一次采样，控制不同 $\mathsf{pH}$ 值连续滴定来测定二氧化氯和亚氯酸根。

# A3 试剂

A3.1 硫代硫酸钠标准液: $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.1\mathrm{mol} / \mathrm{L}$ 。溶解 $25\mathrm{g}$ 硫代硫酸钠 $(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3\cdot 5\mathrm{H}_2\mathrm{O})$ 于 $1\mathrm{L}$ 新煮沸的蒸馏水中, 至少存放二周之后, 用碘酸钾或重铬酸钾标定。最初必须存放一段时间, 是为了使所含的亚硫酸氢盐离子氧化。使用煮沸的蒸馏水, 并加入几毫升三氯甲烷, 以使细菌分解作用减小到最低程度, 以下述两种方法中任选一种来标定。

A3.2 碘酸盐溶液: 溶解 $3.249 \mathrm{~g}$ 无水碘酸氢钾(一级试剂)或 $3.567 \mathrm{~g}$ 碘酸钾(在 $103 \pm 2^{\circ} \mathrm{C}$ 温度下干燥 $1 \mathrm{~h}$ )于蒸馏水中, 转入 $1000 \mathrm{~mL}$ 容量瓶稀至标线, 即为 $c = 0.10000 \mathrm{~mol} / \mathrm{L}$ 溶液, 贮存于具玻璃塞瓶内。

于 $80~\mathrm{mL}$ 蒸馏水中，边搅拌边加入 $1\mathrm{mL}$ 浓硫酸， $10.00\mathrm{mL}c = 0.10000\mathrm{mol} / \mathrm{L}$ 的碘酸氢钾和 $1\mathrm{g}$ 碘化钾，立即用 $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.1\mathrm{mol} / \mathrm{L}$ 溶液滴至淡黄色，加入 $1\mathrm{mL}0.5\mathrm{g} / 100\mathrm{mL}$ 淀粉指示剂，继续滴到蓝色消失为止。

A3.3 重铬酸盐溶液: 溶解 $4.904 \mathrm{~g}$ 无水重铬酸钾(一级试剂)于蒸馏水中, 转入 $1000 \mathrm{~mL}$ 容量瓶并稀至标线, 即为 $c(1 / 6 \mathrm{~K}_{2} \mathrm{Cr}_{2} \mathrm{O}_{7}) = 0.1000 \mathrm{~mol} / \mathrm{L}$ 的溶液, 贮存于具玻璃塞瓶内, 用 $10.00 \mathrm{~mL}$ 重铬酸钾标准溶液代替碘酸盐标准溶液, 在暗处放置 $6 \mathrm{~min}$ 后用 $c(\mathrm{Na}_{2} \mathrm{~S}_{2} \mathrm{O}_{3}) = 0.1 \mathrm{~mol} / \mathrm{L}$ 溶液滴定, 方法同前。

$$
\text {硫代硫酸钠的浓度 (m o l / L)} = \frac {1}{\text {所消耗硫代硫酸钠毫升数}}
$$

A3.4 硫代硫酸钠标准滴定液：用新煮沸过的蒸馏水将上述硫代硫酸钠标准液稀释至0.0100或0.0500mol/L。
A3.5 $0.5\mathrm{g} / 100\mathrm{mL}$ 淀粉指示剂：于 $0.5\mathrm{g}$ 淀粉中，加入少许冷水调成糊状，倾入 $100\mathrm{mL}$ 沸腾的蒸馏水中搅拌，然后沉淀过夜。应用上层清液，加入 $0.125\mathrm{g}$ 水杨酸， $0.4\mathrm{g}$ 氯化锌防腐。
A3.6 碘化钾晶体。
A3.7 $c(\mathrm{NaOH}) = 0.1\mathrm{mol / L}$ 氢氧化钠溶液：溶解 $4\mathrm{g}$ 氢氧化钠于 $1\mathrm{L}$ 蒸馏水中。
A3.8 (1+1)硫酸。
A3.9 缓冲溶液 $(\mathrm{pH} = 7)$ ：称取 $34.0\mathrm{g}$ 磷酸二氢钾和 $35.5\mathrm{g}$ 磷酸氢二钠于烧杯中，加水溶解后稀释至

1L。

# A4 测定步骤

量取 $0.5 \mathrm{~mL}$ (或适量) 水样, 用 $0.1 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠调至近中性, 加缓冲液 $5 \mathrm{~mL}$ 和 $1 \mathrm{~g}$ 碘化钾, 用 $0.0100 \mathrm{~mol} / \mathrm{L}$ 硫代硫酸钠溶液滴至淡黄色, 加 $1 \mathrm{~mL} 0.5 \mathrm{~g} / 100 \mathrm{~mL}$ 淀粉指示剂, 继续滴至蓝色消失, 记下读数 $a$ , 加 $3 \mathrm{~mL} (1 + 1)$ 硫酸 $(\mathrm{pH}$ 调至 $1 \sim 3)$ , 溶液又呈蓝色, 继续滴至无色, 消耗硫代硫酸钠标液为 $b$ 毫升, 若亚氯酸盐含量很高, 可改用 $0.0500 \mathrm{~mol} / \mathrm{L}$ 或适当浓度硫代硫酸钠标液滴定。

# A5 计算公式

$$
\text {二氧化氯} (\mathrm {C l O} _ {2}, \mathrm {m g} \mathrm {L}) = \frac {a \cdot c}{V} \times 6 7 4 5 0
$$

$$
\text {亚氯酸根} \left(\mathrm {C l O} _ {2} ^ {-}, \mathrm {m g} / \mathrm {L}\right) = \frac {(b - 4 a) \cdot c}{\mathrm {r}} \times \frac {1}{4} \times 6 7 4 5 0
$$

式中： $V$ ——水样体积， $\mathrm{mL}$

$c$ ——硫代硫酸钠标准滴定液浓度, $\mathrm{mol} / \mathrm{L}$ :

$a$ ——第一次滴定所消耗硫代硫酸钠标准滴定液体积. $\mathrm{mL}$ ；

$b$ ——第二次滴定所消耗硫代硫酸钠标准滴定液体积. mL.
