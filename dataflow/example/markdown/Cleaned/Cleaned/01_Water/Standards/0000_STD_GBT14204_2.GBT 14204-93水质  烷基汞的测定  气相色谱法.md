# 1 主题内容和适用范围

本标准规定了测定水中烷基汞(甲基汞，乙基汞)的气相色谱法。

本标准适用于地面水及污水中烷基汞的测定。

本方法用巯基棉富集水中的烷基汞，用盐酸氯化钠溶液解析，然后用甲苯萃取，用带电子捕获检测器的气相色谱仪测定，实际达到的最低检出浓度随仪器灵敏度和水样基体效应而变化，当水样取 $1\mathrm{L}$ 时，甲基汞通常检测到 $10\mathrm{ng / L}$ ，乙基汞检测到 $20\mathrm{ng / L}$ 。

样品中含硫有机物(硫醇,硫醚,噻酚等)均可被富集萃取,在分析过程中积存在色谱柱内,使色谱柱分离效率下降,干扰烷基汞的测定。定期往色谱柱内注入二氯化汞苯饱和溶液,可以去除这些干扰,恢复色谱柱分离效率。

2.2.6 乙酸酐：分析纯。
2.2.7 乙酸：分析纯。
2.2.8 硫代乙醇酸：化学纯。
2.2.9 脱脂棉。
2.2.10 氯化钠(NaCl)：分析纯。
2.2.11 硫酸铜：分析纯。
2.2.12 硫酸铜溶液: $w(\mathrm{CuSO}_4) = 25 \mathrm{~g} / 100 \mathrm{~mL}$ 。 $\mathrm{CuSO}_4 \cdot 5 \mathrm{H}_2\mathrm{O} 50 \mathrm{~g}$ 溶于 $200 \mathrm{~mL}$ 无汞蒸馏水 (2.2.14)。

2.2.15 二氯化汞柱处理液：称量 $0.1\mathrm{g}$ 二氯化汞，在 $100\mathrm{mL}$ 容量瓶中用苯溶解，稀释至标线，此溶液为二氯化汞饱和苯溶液。

2.2.17 烷基汞标准溶液：见5.2.2的有关内容。
2.2.18 甲醇：分析纯。
2.2.19 无水乙醇：分析纯。
2.2.20 盐酸溶液： $w = 5\%$
2.2.21 盐酸溶液： $c(\mathrm{HCl}) = 0.1\mathrm{mol / L}$
2.2.22 氢氧化钠溶液： $c(\mathrm{NaOH}) = 5\mathrm{mol} / \mathrm{L}$

带有电子捕获检测器的气相色谱仪。

# 3.2 色谱仪汽化室

全玻璃系统汽化室。

# 3.3 色谱柱

# 3.3.1 色谱柱类型

硬质玻璃填充柱：长度 $1.0 \sim 1.8 \mathrm{~m}$ ，内径： $2 \sim 4 \mathrm{~mm}$ 。

# 3.3.2 填充物

# 3.3.2.1 载体

Chromosorb WAWDMCS, $80\sim 100$ 目，或其他等效载体。涂渍固定液之前，在 $90\mathrm{C}$ 烘 $1.5\mathrm{h}$ 。

# 3.3.2.2 固定液

a. DEGS(丁二酸二乙二醇酯)：最高使用温度 $200^{\circ}\mathrm{C}$ ；或OV-17(苯基 $50\%$ 甲基硅酮)：最高使用温度 $350^{\circ}\mathrm{C}$ 。
b. 液相载荷量：5%DEGS；2%OV-17。
c. 涂渍固定液的方法：静态法。

称取一定量的固定液，例如：称 $0.5\mathrm{g}$ 的DEGS(3.3.2.2)，溶解在二氯甲烷(2.3.2)中，待完全溶解后，倒入刚烘过的载体(3.3.2.1) $9.5\mathrm{g}$ ，使溶有DEGS的二氯甲烷刚好浸没载体，待溶剂完全挥发后，烘干 $(100^{\circ}\mathrm{C})$ ，即涂渍完毕。

# 3.3.3 色谱柱的填充方法

用硅烷化玻璃毛塞住色谱柱的一端，接缓冲瓶和减压系统，柱的另一端接软管连漏斗，将填充物缓缓倒入漏斗，同时开启减压系统，轻轻震动柱体（建议使用超声波水浴）以确保填充紧密，填充完成后，用硅烷化玻璃毛塞住色谱柱另一端，注意：在柱的两端都要空出 $2\mathrm{cm}$ ，填充玻璃毛，以防固定液在进样器和检测器的高温下分解。填充好的色谱柱接检测器一端应与填充时减压吸气一端一致。

# 3.3.4 色谱柱的老化

将填好的色谱柱一端接在仪器进样口上，另一端不接入检测器。通载气 $30~\mathrm{mL / min}$ ，柱温维持 $200^{\circ}\mathrm{C}$ ，老化 $24\mathrm{h}$ ，柱温降至 $160^{\circ}\mathrm{C}$ ，注入柱处理液每次 $20~\mu \mathrm{L}$ ，共五次，间隔 $5\mathrm{min}$ 。继续老化 $24\mathrm{h}$ 。接检测器，柱温设在使用温度，使用前检查，以基线走直为准。（约 $10\sim 20\mathrm{min}$ ）。

# 4样品

# 4.1 样品采集和保存

样品采集在塑料瓶(3.3.10)中，如在数小时内样品不能进行分析，应在样品瓶中预先加入硫酸铜(2.2.11)，加入量为每升 $1\mathrm{g}$ （水样处理时不再加硫酸铜溶液），水样在 $2\sim 5\%$ 条件下贮存。

# 4.2 试样的预处理

4.2.1 取均匀水样 $1\mathrm{L}$ ，置于 $2\mathrm{L}$ 分液漏斗(3.3.11)中，加入 $1\mathrm{mL}$ 硫酸铜溶液(2.2.12)，使用 $2\mathrm{mol} / \mathrm{L}$ 盐酸溶液(2.2.4)，或 $6\mathrm{mol} / \mathrm{L}$ 氢氧化钠(2.2.22)，调 $\mathrm{pH}$ 为 $3\sim 4$ ，接巯基棉管，让水样流速保持在 $20\sim 25\mathrm{mL} / \mathrm{min}$ ，待吸附完毕，用洗耳球压出吸附管内残存的水滴，然后加入 $3.0\mathrm{mL}$ 解析液(2.2.16)，将巯基棉上吸附的烷基汞解析到 $10\mathrm{mL}$ 具塞离心管(3.3.12)中(用吸耳球压出最后一滴解析液)，向试管中加入 $1.0\mathrm{mL}$ 甲苯(苯)(2.2.3)，加塞，振荡提取 $1\mathrm{min}$ ，静置分层，用离心机 $2500\mathrm{r} / \mathrm{min}$ 离心 $3\sim 5\mathrm{min}$ ，离心分离有机相与盐酸解析液，取有机相进行色谱测定；或者分层后吸出有机相，加入少量无水硫酸钠(2.2.13)脱水，进行色谱测定。

# 4.2.2 污水试样的处理

取污水水样 $>100\mathrm{mL}$ 置于锥形瓶中，用 $2\mathrm{mol} / \mathrm{L}$ 盐酸溶液(2.2.4)酸化至 $\mathrm{pH} < 1$ ，加入 $1\mathrm{g}$ 硫酸酮(2.2.11)充分搅拌后，调 $\mathrm{pH} = 3$ ，静置，用快速滤纸过滤，收集滤液 $100\mathrm{mL}$ 转移到分液漏斗中，在漏斗下口塞一些玻璃毛过滤，接巯基棉管富集，解析步骤同上。

5.2.2.4 标准溶液的使用

a. 色谱测定使用的标准样品，进样后出单一峰，没有其他物质干扰。标准溶液（溶剂甲苯或苯配制）用于确定烷基汞的保留时间(RT)，并考察仪器的线性范围。
b. 每次分析样品时，都要用标准进行校准，一般每测定十个样品校准一次，当使用 $0.02\mathrm{mg / L}$ 标准溶液，连续进样两次，两峰峰高(或峰面积)相对偏差 $\leq 4\%$ ，可认为仪器稳定。
c. 在同一次分析中, 标准样品进样体积要与被测样品进样体积相同, 使用外标法定量时, 标准样品的响应值应与被测样品的响应值接近。
d. 实际分析工作中使用的标准样品的制备：取基体加标标准溶液(5.2.2.3) $1.0\mathrm{mL}$ ，加解析液(2.2.16) $3\mathrm{mL}$ ，加 $1.0\mathrm{mL}$ 甲苯(苯)，振荡萃取 $1\mathrm{min}$ ，离心分离。制备过程与试样预处理(4.2.1)步骤中，用甲苯(苯)萃取解析液一致，以减小系统误差。

5.3 校准数据的表示

试样中组分按式(1)校准：

$$
X _ {i} = E _ {i} \times \frac {A _ {i}}{A _ {E}} \tag {1}
$$

式中： $X_{i}$ ——试样中组分 $i$ 的含量；

$E_{i}$ ——标准试样中组分 $i$ 的含量；

$A_{i}$ ——试样中组分 $i$ 的峰面积， $\mathrm{cm}^2$

$A_{E}$ ——标准试样中组分 $i$ 的峰面积， $\mathrm{cm}^2$ 。

5.5.3 定量分析
5.5.3.1 色谱峰的测量

a. 以峰的起点和拐点的联线做为峰底，从峰高最大值对时间轴作垂线，对应的时间即为保留时间（RT）。从峰顶到峰底间的线段为峰高。
b. 积分仪自动求出RT，给出峰面积。

5.5.3.2 计算

a. 使用记录仪：

$$
C = \frac {m \cdot h _ {1} \cdot V _ {1} \cdot K}{h _ {2} \cdot V _ {2} \cdot V _ {3}} \tag {2}
$$

式中： $C$ ——样品中甲（乙）基汞浓度， $\mu \mathrm{g} / \mathrm{L}$

$m$ ——标准物重量，ng；

$h_1$ ——样品峰高， $\mathrm{mm}$

$V_{1}$ 提取液体积， $\mu \mathrm{L}$

$K$ ——稀释因子；

$h_2$ ——标准物峰高， $\mathrm{mm}$

$V_{2}$ ——提取液进样体积， $\mu \mathrm{L}$

$V_{3}$ ——水样体积， $\mathrm{mL}$ 。

# 6.2.3 检测限

当气相色谱仪设在仪器的最大灵敏度时，以噪声的3倍作为仪器的检测限。

甲基汞： $1.0\times 10^{-12}\mathrm{g}$ ；乙基汞： $1.5\times 10^{-12}\mathrm{g}$ 。

本方法要求仪器的灵敏度不低于 $10^{-12}\mathrm{g}$ 。按照载气(2.1)的标准，可达到本方法对仪器灵敏度的要求。

# 巯基棉(S.C.F)的制备

（补充件）

# A1 Nishi法

在一个玻璃烧杯中，依次加入 $100\mathrm{mL}$ 硫代乙醇酸(2.2.8)， $60\mathrm{mL}$ 乙酸酐(2.2.6)， $40\mathrm{mL}$ 乙酸(2.2.7)， $0.3\mathrm{mL}$ 硫酸(2.2.5)，充分混匀，冷却至室温后，加入 $30\mathrm{g}$ 脱酯棉(2.2.9)，浸泡完全，压紧，冷至室温，降温后加盖，放在 $37\sim 40^{\circ}\mathrm{C}$ 烘箱中 $48\sim 96\mathrm{h}$ 。取出后放在耐酸漏斗上过滤，用无汞蒸馏水(2.2.14)洗至中性，置于 $35\sim 37^{\circ}\mathrm{C}$ 烘箱中烘干。取出置于棕色干燥器中，避光保存。每批巯基棉的性能必须做回收率测定。回收率 $>85\%$ ，才可使用。

# A2 S.C.F 回收率测定

取基体加标标准液 $(0.2\mu \mathrm{g / mL})1.0\mathrm{mL}$ ，加入1L试剂水中，按4.2.1步骤处理，与基体加标标准液 $(0.2\mu \mathrm{g / mL})1.0\mathrm{mL}$ 的甲苯(苯)萃取液比较，计算回收率。

# 附录C

# 溶剂冲洗进样技术

（补充件）

用清洁的样品溶剂冲洗进样器几次，把少量样品溶剂 $(1\mu \mathrm{L})$ 抽入进样器，再抽入 $0.5\mu \mathrm{L}$ 空气，然后将进样器针头插入样品容器内，慢慢地抽入 $2\sim 4\mu \mathrm{L}$ 样品，使针头离开样品，将进样器柱塞慢慢提起，样品完全抽入针筒内，并抽入 $0.5\mu \mathrm{L}$ 空气，此时可见两个液体柱两个空气柱：溶剂和样品，中间由空气柱隔开。样品量可由针筒刻度准确计量，针头内不含样品。快速进样。这种进样方式重复性好，可保证同一样品连续进样两针，响应值相对偏差 $\leqslant 4\%$ 。

# 附录D

# 积分仪的使用

（参考件）

# D1 积分仪的调正

按使用说明书的要求，设定适当的衰减和纸速。

# D2 色谱峰的测量

完成进样后，启动积分仪，积分仪自动求出色谱峰的 $RT$ 值和相应的峰面积。

# D3 计算（外标法）

计算 $RF$ 因子：每个浓度水平的化合物的响应值与注入质量的比值为 $RF$ 值。当采用五个浓度水平的标准溶液测定的 $RF$ 因子，其相对标准偏差 $< 20\%$ 时，用 $RF$ 因子的平均值可以代替标准曲线。

$$
R F = X / A$$

式中： $X$ ——已知浓度的标准样品， $\mathrm{ng} / \mu \mathrm{L}$

$A$ ——峰面积积分值。

定量计算公式：

$$
X _ {i} = \frac {1}{k} \times \frac {R F \times A _ {i}}{m} \times 1 0 0 \tag {D2}
$$

式中： $X_{i},A_{i}$ ——同式(1)；

$k$ ——样品浓缩或稀释倍数；

$m$ ——样品的重量。

# 附录E

# 质量控制

（参考件）

E1 应用本方法的实验室都要执行质量控制计划。质量控制的目的是考察实验室的能力，然后通过加标样品分析考查实验室水平。要求实验室建立实验数据档案，保留反映分析工作水平的一切数据，定期检查现有工作水平是否在方法的准确度和精密度范围之内。

E1.1 进行样品分析之前，分析人员必须证明有能力用本方法取得可接受的准确度和精密度。这种能力的评定见E2。

E1.2 实验室至少要对全部样品的 $10\%$ 作加标分析，加标浓度应当超过样品背景浓度值的2倍，实验方为有效。使用本方法的基体加标溶液，配制所需要的加标浓度，以监测实验室的持续水平。操作步骤见E4。

E2 用下述操作来检验分析人员是否具有能力，以达到方法要求的准确度和精密度。

E2.1 测定统一的质量控制样品(QC), QC样品的浓度应比选定的浓度大1000倍。QC样品是以0.1mol/L盐酸为溶剂, 含有一定量烷基汞的溶液, 封装在棕色安瓿瓶中。

注：QC样品可以从北京市环境监测中心得到。

E2.2 踢开QC样品安瓿瓶，用移液管向至少四个 $1000\mathrm{mL}$ 的试剂水中各加入 $1.0\mathrm{mLQC}$ 样品，按4.2条的内容分析各份样品。
E2.3 对分析结果计算平均回收率 $(R)$ 和回收率的标准偏差 $(S)$ 。
E2.4 将 E2.3 的计算结果与本方法的平均回收率 $(X)$ 和标准偏差 $(P)$ 相比较。如果 $S > 2P$ 或 $|X - R| > 2P$ ，应查找可能存在的问题并重新实验，直到达到方法要求。
E2.5 根据实验室间验证的结果，确定了方法的 $(X)$ 和 $(P)$ 的指标，分析人员在熟悉了方法要求后，必须先满足这些指标，然后才能分析样品。
E3 分析人员必须计算分析方法的性能指标，确定实验室对各加标浓度(高浓度、低浓度)和待测化合物的分析水平。
E3.1 计算分析方法回收率的控制上限和控制下限：

控制上限 $(\mathrm{UCL}) = R + 3S$

控制下限 $(\mathrm{LCL}) = R - 3S$

式中 $R$ 和 $S$ 按E2.3计算。UCL和LCL用来绘制观察分析水平变化趋势图。

E3.2 实验室必须建立该方法分析样品数据的档案，保留表示实验室在分析烷基汞方面准确度的记录。
E4 要求实验室将部分样品重复分析以测定加标回收率, 至少应对全部样品的 $10\%$ 进行加标回收测定。至少每月作一次加标分析。加标样品要 E1.2 的要求进行加标。在加标实验中, 如果某一种烷基汞的回收率未落在方法控制限内, 同一批处理的样品中烷基汞的数据就是可疑的。实验室应监测这种可疑数据的出现频率, 以保证这一频率维持在 $5\%$ 以下。
E5 做实验方法全程序空白，以证明所有玻璃器皿和试剂的干扰都在控制之下，当更换实验全程序中使用的任何一种物品（试剂、巯基棉和玻璃器皿），必须做一次全程序空白实验。
E6 建议实验室采取进一步的质量保证措施, 对出现可疑数据的样品要反复做, 并重新取样, 来监测采样技术的精密度。当对一种烷基汞的定性有疑问时, 可采用不同极性的色谱柱确证, 或采用其他确证方法, 比如 GC/MS。
