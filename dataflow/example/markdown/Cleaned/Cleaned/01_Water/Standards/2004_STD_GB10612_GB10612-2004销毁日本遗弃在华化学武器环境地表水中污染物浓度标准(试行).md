# 销毁日本遗弃在华化学武器

# 环境地表水中污染物浓度标准（试行）

Concentration limitations of pollutants in surface water for the destruction of

chemical weapons abandoned by Japan in China

# 1 范围

本标准规定了销毁日本遗弃在华化学武器地表水中污染物的浓度限值及监测方法。

本标准适用于销毁日本遗弃在华化学武器各项设施建设或作业的地表水环境影响评价以及销毁作业过程和后续环境管理与评价。

# 3 技术内容

3.1 销毁日本遗弃在华化学武器地表水中污染物的浓度不得超过表1所列的限值，其他污染物的浓度限值按GB3838中的规定执行。
3.2 当地表水中污染物浓度超过表1规定的限值时，应划定隔离区，采取措施或限制其使用功能。

表 1 销毁日本遗弃在华化学武器地表水中污染物浓度限值


| 序号                                                                                                                                     | 污染物名称        | 浓度限值/(mg/L)                       |
| -------------------------------------------------------------------------------------------------------------------------------------- | ------------ | --------------------------------- |
| 1                                                                                                                                      | 芥子气a         | 0.000 2                           |
| 2                                                                                                                                      | 路易氏剂b        | 0.000 2                           |
| 3                                                                                                                                      | 芥子气与路易氏剂混合剂c | 按公式计算: cHD / (SHD + cL) / (S) ≤ 1 |
| 4                                                                                                                                      | 二苯氰肿d        | 0.000 02                          |
| 5                                                                                                                                      | 苯氯乙酮         | 0.000 4                           |
| 6                                                                                                                                      | 氰化物(以CN-计)   | 0.01                              |
| 7                                                                                                                                      | 总砷           | 0.01                              |
| a指芥子气、芥子砜和芥子亚砜的总量。b指路易氏剂、氯乙烯肿酸和氯乙烯氧肿的总量。c公式中:cHD:芥子气、芥子砜和芥子亚砜的浓度;SHD:芥子气浓度限值;cL:路易氏剂、氯乙烯肿酸和氯乙烯氧肿的浓度;SL:路易氏剂浓度限值。d指二苯氰肿、二苯氯肿和氧联双二苯肿的总量。 |              |                                   |


# 4 监测

4.1 监测点设置和监测频次按HJ/T91的规定执行。
4.2 表1中污染物的分析方法按表2执行，可采用最低检测浓度更低的方法实施监测。其他污染物的分析方法按GB3838和HJ/T91中的规定方法执行。
