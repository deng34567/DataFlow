# 水质亚硝酸盐氮的测定分光光度法

UDC 614.777

:543.42

GB 7493-87

Water quality-Determination of nitrogen

(nitrite) -Spectrophotometric method

本标准等效采用ISO6777—1984《水质亚硝酸盐氮测定分子吸收分光光度法》。

本标准根据我国标准的格式对ISO6777—1984标准技术上稍作修改和补充。

# 1 适用范围

本标准规定了用分光光度法测定饮用水、地下水、地面水及废水中亚硝酸盐氮的方法。

# 1.1 测定上限

当试份取最大体积（50ml）时，用本方法可以测定亚硝酸盐氮浓度高达 $0.20\mathrm{mg / L}$ 。

# 1.2 最低检出浓度

采用光程长为 $10\mathrm{mm}$ 的比色皿，试份体积为 $50\mathrm{ml}$ ，以吸光度0.01单位所对应的浓度值为最低检出限浓度，此值为 $0.003\mathrm{mg / L}$ 。

采用光程长为 $30\mathrm{mm}$ 的比色皿，试份体积为 $50\mathrm{ml}$ ，最低检出浓度为 $0.001\mathrm{mg / L}$ 。

# 1.3 灵敏度

采用光程长为 $10\mathrm{mm}$ 的比色皿，试份体积为 $50\mathrm{ml}$ 时，亚硝酸盐氮浓度 $c_{\mathrm{N}} = 0.20\mathrm{mg / L}$ ，给出的吸光度约为0.67单位。

# 1.4 干扰

当试样 $\mathrm{pH} \geqslant 11$ 时，可能遇到某些干扰，遇此情况，可向试份中加入酚酞溶液（3.12）1滴，边搅拌边逐滴加入磷酸溶液（3.4），至红色刚消失。经此处理，则在加入显色剂后，体系 $\mathrm{pH}$ 值为 $1.8 \pm 0.3$ ，而不影响测定。

试样如有颜色和悬浮物，可向每 $100\mathrm{ml}$ 试样中加入 $2\mathrm{ml}$ 氢氧化铝悬浮液（3.9），搅拌，静置，过滤，弃去 $25\mathrm{ml}$ 初滤液后，再取试份测定。

水样中常见的可能产生干扰物质的含量范围见附录A。其中氯胺、氯、硫代硫酸盐、聚磷酸钠和三价铁离子有明显干扰。

# 2 原理

在磷酸介质中， $\mathsf{pH}$ 值为1.8时，试份中的亚硝酸根离子与4-氨基苯磺酰胺(4-aminobenzene sulfonamide）反应生成重氮盐，它再与N-（1-萘基）-乙二胺二盐酸盐[N-（1-naphthyl-1，2'-diaminoethane dihydrochloride]偶联生成红色染料，在540nm波长处测定吸光度。如果使用光程长为 $10\mathrm{mm}$ 的比色皿，亚硝酸盐氮的浓度在 $0.2\mathrm{mg / L}$ 以内其呈色符合比尔定律。

# 3 试剂

在测定过程中，除非另有说明，均使用符合国家标准或专业标准的分析纯试剂，实验用水均为无亚硝酸盐的二次蒸馏水。

# 3.1 实验用水

采用下列方法之一进行制备：

3.1.1 加入高锰酸钾结晶少许于 $1 \mathrm{~L}$ 蒸馏水中，使成红色，加氢氧化钡（或氢氧化钙）结晶至溶液呈碱性，使用硬质玻璃蒸馏器进行蒸馏，弃去最初的 $50 \mathrm{ml}$ 馏出液，收集约 $700 \mathrm{ml}$ 不含锰盐的馏出液，待用。
3.1.2 于 $1\mathrm{L}$ 蒸馏水中加入硫酸（3.3） $1\mathrm{ml}$ 、硫酸锰溶液[每 $100\mathrm{ml}$ 水中含有36.4g硫酸锰（ $\mathrm{MnSO}_4\cdot \mathrm{H}_2\mathrm{O}$ ）] $0.2\mathrm{ml}$ ，滴加 $0.04\%$ （ $V / V$ ）高锰酸钾溶液至呈红色（约 $1\sim 3\mathrm{ml}$ ），使用硬质玻璃蒸馏器进行蒸馏，弃去最初的 $50\mathrm{ml}$ 馏出液，收集约 $700\mathrm{ml}$ 不含锰盐的馏出液，待用。
3.2 磷酸： $15\mathrm{mol} / \mathrm{L}$ ， $\rho = 1.70\mathrm{g / ml}$ 。
3.3 硫酸： $18\mathrm{mol} / \mathrm{L}$ ， $\rho = 1.84\mathrm{g / ml}$ 。
3.4 磷酸： $1 + 9$ 溶液 $(1.5\mathrm{mol} / \mathrm{L})$ 。

溶液至少可稳定6个月。

# 3.5 显色剂

$500\mathrm{ml}$ 烧杯内置人 $250\mathrm{ml}$ 水和 $50\mathrm{ml}$ 磷酸(3.2)，加入 $20.0\mathrm{g}4-$ 氨基苯磺酰胺 $\mathrm{(NH_2C_6H_4SO_2NH_2)}$ 。再将 $1.00\mathrm{gN} - (1 - \text{萘基}) -$ 乙二胺二盐酸盐 $\mathrm{(C_{10}H_7NHC_2H_4NH_2\cdot 2HCl)}$ 溶于上述溶液中，转移至 $500\mathrm{ml}$ 容量瓶中，用水稀至标线，摇匀。

此溶液贮存于棕色试剂瓶中，保存在 $2 \sim 5^{\circ} \mathrm{C}$ ，至少可稳定一个月。

注：本试剂有毒性，避免与皮肤接触或吸入体内。

3.6 亚硝酸盐氮标准贮备溶液： $c_{\mathrm{N}} = 250\mathrm{mg / L}$ 。

# 3.6.1 贮备溶液的配制

称取 $1.232\mathrm{g}$ 亚硝酸钠 $\left(\mathrm{NaNO}_{2}\right)$ ，溶于 $150\mathrm{ml}$ 水中，定量转移至 $1000\mathrm{ml}$ 容量瓶中，用水稀释至标线，摇匀。

本溶液贮存在棕色试剂瓶中，加入 $1\mathrm{ml}$ 氯仿，保存在 $2\sim 5\%$ ，至少稳定一个月。

# 3.6.2 贮备溶液的标定

在 $300\mathrm{ml}$ 具塞锥形瓶中，移入高锰酸钾标准溶液（3.10） $50.00\mathrm{ml}$ 、硫酸（3.3） $5\mathrm{ml}$ ，用 $50\mathrm{ml}$ 无分度吸管，使下端插入高锰酸钾溶液液面下，加入亚硝酸盐氮标准贮备溶液 $50.00\mathrm{ml}$ ，轻轻摇匀，置于水浴上加热至 $70\sim 80^{\circ}\mathrm{C}$ ，按每次 $10.00\mathrm{ml}$ 的量加入足够的草酸钠标准溶液（3.11），使高锰酸钾标准溶液褪色并使过量，记录草酸钠标准溶液用量 $V_{2}$ ，然后用高锰酸钾标准溶液（3.10)滴定过量草酸钠至溶液呈微红色，记录高锰酸钾标准溶液总用量 $V_{1}$ 。

再以 $50\mathrm{ml}$ 实验用水代替亚硝酸盐氮标准贮备溶液，如上操作，用草酸钠标准溶液标定高锰酸钾溶液的浓度 $c_{1}$ 。

按式（1）计算高锰酸钾标准溶液浓度 $c_{1}$ （ $1 / 5\mathrm{KMnO}_4\mathrm{mol} / \mathrm{L}$ ）：

$$
c _ {1} = \frac {0 . 0 5 0 0 \times V _ {4}}{V _ {3}} \dots \tag {1}
$$

式中： $V_{3}$ ——滴定实验用水时加入高锰酸钾标准溶液总量， $\mathrm{ml}$ ；

$V_{4}$ ——滴定实验用水时加入草酸钠标准溶液总量， $\mathrm{ml}$ ；

0.0500——草酸钠标准溶液浓度 $c(1 / 2\mathrm{Na}_2\mathrm{C}_2\mathrm{O}_4)$ ，mol/L。

按式（2）计算亚硝酸盐氮标准贮备溶液的浓度 $c_{\mathrm{N}}$ （ $\mathrm{mg / L}$ ）：

$$
\begin{array}{l} c _ {N} = \frac {\left(V _ {1} c _ {1} - 0 . 0 5 0 0 V _ {2}\right) \times 7 . 0 0 \times 1 0 0 0}{5 0 . 0 0} \\ = 1 4 0 V _ {1} c _ {1} - 7. 0 0 V _ {2} \dots \tag {2} \\ \end{array}
$$

式中： $V_{1}$ ——滴定亚硝酸盐氮标准贮备溶液时加入高锰酸钾标准溶液总量， $\mathbf{m}1$

$V_{2}$ ——滴定亚硝酸盐氮标准贮备溶液时加入草酸钠标准溶液总量，ml；

$c_{1}$ ——经标定的高锰酸钾标准溶液的浓度，mol/L；

7.00——亚硝酸盐氮（1/2N）的摩尔质量；

50.00——亚硝酸盐氮标准贮备溶液取样量，ml；

0.0500——草酸钠标准溶液浓度 $c(1 / 2\mathrm{Na}_{2}\mathrm{C}_{2}\mathrm{O}_{4})$ ， $\mathrm{mol} / \mathrm{L}$ 。

3.7 亚硝酸盐氮中间标准液： $c_{\mathrm{N}} = 50.0\mathrm{mg / L}$ 。

取亚硝酸盐氮标准贮备溶液（3.6）50.00ml置250ml容量瓶中，用水稀释至标线，摇匀。

此溶液贮于棕色瓶内，保存在 $2 \sim 5^{\circ} \mathrm{C}$ ，可稳定一星期。

3.8 亚硝酸盐氮标准工作液： $c_{\mathrm{N}} = 1.00\mathrm{mg / L}$ 。

取亚硝酸盐氮中间标准液（3.7） $10.00\mathrm{ml}$ 于 $500\mathrm{ml}$ 容量瓶内，水稀释至标线，摇匀。

此溶液使用时，当天配制。

注：亚硝酸盐氮中间标准液和标准工作液的浓度值，应采用贮备溶液标定后的准确浓度的计算值。

# 3.9 氢氧化铝悬浮液

溶解125g硫酸铝钾 $\left[\mathrm{KAl}\left(\mathrm{SO}_{4}\right)_{2} \cdot 12 \mathrm{H}_{2} \mathrm{O}\right]$ 或硫酸铝铵 $\left[\mathrm{NH}_{4} \mathrm{Al}\left(\mathrm{SO}_{4}\right)_{2} \cdot 12 \mathrm{H}_{2} \mathrm{O}\right]$ 于1L一次蒸馏水中，加热至 $60^{\circ} \mathrm{C}$ ，在不断搅拌下，徐徐加入 $55 \mathrm{ml}$ 浓氢氧化铵，放置约 $1 \mathrm{~h}$ 后，移入1L量筒内，用一次蒸馏水反复洗涤沉淀，最后用实验用水洗涤沉淀，直至洗涤液中不含亚硝酸盐为止。澄清后，把上清液尽量全部倾出，只留稠的悬浮物，最后加入 $100 \mathrm{ml}$ 水。使用前应振荡均匀。

3.10 高锰酸钾标准溶液： $c(1 / 5\mathrm{KMnO}_4) = 0.050\mathrm{mol} / \mathrm{L}$ 。

溶解1.6g高锰酸钾（KMnO₄）于1.2L水中（一次蒸馏水），煮沸0.5~1h，使体积减少到1L左右，放置过夜，用G-3号玻璃砂芯滤器过滤后，滤液贮存于棕色试剂瓶中避光保存。高锰酸钾标准溶液浓度按3.6.2第二段所述方法进行标定和计算。

3.11 草酸钠标准溶液： $c(1 / 2\mathrm{Na}_2\mathrm{C}_2\mathrm{O}_4) = 0.0500\mathrm{mol} / \mathrm{L}$ 。

溶解经 $105^{\circ} \mathrm{C}$ 烘干 $2 \mathrm{~h}$ 的优级纯无水草酸钠 $\left(\mathrm{Na}_{2} \mathrm{C}_{2} \mathrm{O}_{4}\right) 3.3500 \pm 0.0004 \mathrm{~g}$ 于 $750 \mathrm{ml}$ 水中, 定量转移至 $1000 \mathrm{ml}$ 容量瓶中, 用水稀释至标线, 摇匀。

3.12 酚酞指示剂： $c = 10\mathrm{g / L}$ 。

0.5g酚酞溶于 $95\%$ （ $V / V$ ）乙醇50ml中。

# 4 仪器

所有玻璃器皿都应用 $2\mathrm{mol} / \mathrm{L}$ 盐酸仔细洗净，然后用水彻底冲洗。

常用实验室设备及分光光度计。

# 5 采样和样品

# 5.1 采样和样品保存

实验室样品应用玻璃瓶或聚乙烯瓶采集，并在采集后尽快分析，不要超过 $24\mathrm{h}$ 。

若需短期保存（1～2天），可以在每升实验室样品中加入 $40\mathrm{mg}$ 氯化汞，并保存于 $2\sim 5\mathrm{C}$ 。

# 5.2 试样的制备

实验室样品含有悬浮物或带有颜色时，需按照1.4第二段所述的方法制备试样。

# 6 步骤

# 6.1 试份

试份最大体积为 $50.0\mathrm{ml}$ ，可测定亚硝酸盐氮浓度高至 $0.20\mathrm{mg / L}$ 。浓度更高时，可相应用较少量的样品或将样品进行稀释后，再取样。

# 6.2 测定

用无分度吸管将选定体积的试份移至 $50\mathrm{ml}$ 比色管（或容量瓶）中，用水稀释至标线，加入显色剂（3.5） $1.0\mathrm{ml}$ ，密塞，摇匀，静置，此时 $\mathsf{pH}$ 值应为 $1.8 \pm 0.3$ 。

加入显色剂20min后、2h以内，在540nm的最大吸光度波长处，用光程长10mm的比色皿，以实验用水做参比，测量溶液吸光度。

注：最初使用本方法时，应校正最大吸光度的波长，以后的测定均应用此波长。

# 6.3 空白试验

按6.2所述步骤进行空白试验，用 $50\mathrm{m}1$ 水代替试份。

# 6.4 色度校正

如果实验室样品经5.2的方法制备的试样还具有颜色时，按6.2所述方法，从试样中取相同体积的第二份试份，进行测定吸光度，只是不加显色剂（3.5），改加磷酸（3.4）1.0ml。

# 6.5 校准

在一组六个 $50\mathrm{ml}$ 比色管（或容量瓶）内，分别加入亚硝酸盐氮标准工作液（3.8）0、1.00、3.00、5.00、7.00和 $10.00\mathrm{ml}$ ，用水稀释至标线，然后按6.2第二段开始到末了叙述的步骤操作。

从测得的各溶液吸光度，减去空白试验吸光度，得校正吸光度 $A_{r}$ ，绘制以氮含量（ $\mu \mathrm{g}$ ）对校正吸光度的校准曲线，亦可按线性回归方程的方法，计算校准曲线方程。

# 7 结果表示

# 7.1 计算方法

试份溶液吸光度的校正值 $A_{r}$ 按式（3）计算：

$$
A _ {r} = A _ {s} - A _ {b} - A _ {c} \dots \tag {3}
$$

式中： $A_{s}$ ——试份溶液测得吸光度；

$A_{b}$ ——空白试验测得吸光度；

$A_{\mathrm{c}}$ ——色度校正测得吸光度。

由校正吸光度 $A_{\mathrm{r}}$ 值，从校准曲线上查得（或由校准曲线方程计算）相应的亚硝酸盐氮的含量 $m_{\mathrm{N}}(\mu \mathrm{g})$ 。

试份的亚硝酸盐氮浓度按式（4）计算：

$$
c _ {\mathrm {N}} = \frac {m _ {\mathrm {N}}}{V} \dots \tag {4}
$$

式中： $c_{\mathrm{N}}$ —亚硝酸盐氮浓度， $\mathrm{mg / L}$ ；

$m_{\mathbb{N}}$ ——相应于校正吸光度 $A_{r}$ 的亚硝酸盐氮含量， $\mu \mathrm{g}$ ；

$V$ ——取试份体积， $\mathfrak{m}\mathfrak{l}$ 。

试份体积为 $50\mathrm{ml}$ 时，结果以三位小数表示。

# 7.2 精密度和准确度

7.2.1 取平行双样测定结果的算术平均值为测定结果。

7.2.2 23个实验室测定亚硝酸盐氮浓度为 $7.46 \times 10^{-2} \mathrm{mg} / \mathrm{L}$ 的试样，重复性为 $1.1 \times 10^{-3} \mathrm{mg} / \mathrm{L}$ ，再现性为 $3.7 \times 10^{-3} \mathrm{mg} / \mathrm{L}$ ，加标百分回收率范围为 $96\% \sim 104\%$ 。

15个实验室测定亚硝酸盐氮浓度为 $6.19 \times 10^{-2} \mathrm{mg} / \mathrm{L}$ 的试样，重复性为 $2.0 \times 10^{-3} \mathrm{mg} / \mathrm{L}$ ，再现性为 $3.7 \times 10^{-3} \mathrm{mg} / \mathrm{L}$ ，加标百分回收率范围为 $93\% \sim 103\%$ 。
