# 销毁日本遗弃在华化学武器

# 空气中氰溴甲苯的测定高效液相色谱法（试行）

Determination of bromobenzylcyanide in ambient air for the destruction of chemical weapons abandoned by Japan in China High performance liquid chromatography

警告：氰溴甲苯是强催泪性物质，对眼睛、皮肤、呼吸道等刺激强烈。操作时应按规定要求佩带防护器材，在通风良好的环境中进行操作；测试后的残渣残液应做妥善的安全处理。

# 1 范围

本标准规定了测定空气中氰溴甲苯的高效液相色谱方法。

本标准适用于销毁日本遗弃在华化学武器全过程工作区及环境空气中氰溴甲苯的测定。

当采样体积为 $100\mathrm{L}$ 时，本标准的最低检测浓度为 $0.001\mathrm{mg / m^3}$ 。

# 3 原理

采用玻璃纤维滤膜阻留、硅胶吸附的组合式采样，以二氯甲烷为溶剂索氏提取。提取液经浓缩后，用高效液相色谱/紫外检测器于波长 $237\mathrm{nm}$ 处进行测定。

# 4 试剂和材料

4.1 甲醇：HPLC级。
4.2 水：二次蒸馏水，用 $0.45\mu \mathrm{m}$ 的微孔滤膜过滤。
4.3 二氯甲烷：HPLC级。
4.4 玻璃棉： $400\%$ 加热 $4\mathrm{h}$
4.5 氰溴甲苯：含量不低于 $99.0\%$
4.6 氰溴甲苯标准贮备液 $(1.0\mathrm{mg / ml})$ ：称取 $10\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ ）的氰溴甲苯（4.5），用甲醇溶解，在容量瓶中定容至 $10.0\mathrm{ml}$ 。 $4\%$ 避光保存。
4.7 标准工作液：先用甲醇将标准贮备液（4.6）稀释成 $100.0\mu \mathrm{g / ml}$ 、 $10.0\mu \mathrm{g / ml}$ 的标准溶液，然后用该溶液配制五个或五个以上浓度的标准工作液。标准工作液浓度的确定应参照样品浓度范围，以样品浓度在曲线中段为宜。 $4\%$ 避光保存。
4.8 玻璃纤维滤膜：过滤效率不低于 $99.99\%$
4.9 硅胶：60~80目，填装前，在200℃条件下通氮气老化1h。
4.10 硅胶采样管：取长 $10\mathrm{cm}$ ，内径 $6\mathrm{mm}$ 的硬质玻璃管，内填充 $0.5\mathrm{g}$ 硅胶吸附剂（4.9），两端充填少许玻璃棉固定，管两头分别用硅橡胶塞塞紧。装填完成后，应测试空白。

# 5 仪器

5.1 高效液相色谱仪：具紫外检测器。
5.2 色谱柱：十八烷基硅烷键合硅胶（ODSC18）柱， $150\mathrm{mm}\times \phi 6\mathrm{mm}$
5.3 样品采样及预处理时使用的仪器
5.3.1 空气采样器：流量范围 $0\sim 1.5\mathrm{L / min}$
5.3.2 索氏提取器：带 $60\mathrm{ml}$ 圆底烧瓶。
5.3.3 K-D浓缩器。
5.3.4 恒温水浴锅。
5.3.5 微量注射器： $25\mu \mathrm{l}$

# 6样品

# 6.1 样品采集

采样系统以带玻璃纤维滤膜的采样头在前端阻留，紧接硅胶采样管（4.10)，再与空气采样器(5.3.1）串联。以 $0.5\sim 1.0\mathrm{L / min}$ 的流量采样。

# 6.2 样品的贮存

将玻璃纤维滤膜尘面朝里折叠，塑料袋密封；硅胶采样管的两端用胶塞密封，迅速送回实验室， $4\mathrm{C}$ 避光保存，7d内完成分析。

# 6.3 样品的提取

将采好样品的采样管及玻璃纤维滤膜一起，用玻璃纤维滤筒包好，小心放入索氏提取器（5.3.2）中。在放有 $1\sim 2$ 粒干净沸石的圆底烧瓶中加入 $50\mathrm{ml}$ 二氯甲烷（4.3），将烧瓶连接在提取器上，在 $55\%$ 恒温水浴中提取 $6\mathrm{h}$ 。提取速度要求达到 $4\sim 6$ 个循环/h。提取结束后，提取液冷却至室温，并转移至K-D浓缩瓶。

# 6.4 样品的浓缩

将盛有提取液的K-D浓缩瓶置于 $50\sim 55\%$ 的水浴上浓缩至约 $1\mathrm{ml}$ 。用微量注射器吸取少量二氯甲烷（4.3）冲洗玻璃接口并定容至 $2.0\mathrm{ml}$ ，待色谱分析用。

# 7 操作步骤

# 7.1 色谱条件

7.1.1 柱温：室温。
7.1.2 流速： $1.0\mathrm{ml / min}$
7.1.3 流动相（体积分数）： $70\%$ 甲醇 $+30\%$ 水。
7.1.4 检测器：紫外检测器，测定波长 $237\mathrm{nm}$

# 7.2 标准曲线的绘制

7.2.1 以氰溴甲苯的色谱峰面积 $A$ （或峰高 $H$ ）为纵坐标，相应浓度 $c(\mu \mathrm{g / ml})$ 为横坐标，绘制标准曲线。
7.2.2 每天用浓度居中的标准工作液（其浓度必须大于10倍最低检测浓度）做常规校正，组分响应值变化超过 $15\%$ 时，需重新绘制标准曲线。

# 7.3 测定

7.3.1 进样

进样量： $25\mu \mathrm{l}$

7.3.2 空白实验：每批样品或试剂有变动时，都应有相应的现场空白实验。现场空白样品应经历样品制备和测定的所有步骤。

# 7.3.4 定量分析

以外标法进行定量。

空气中氰溴甲苯的浓度以公式（1）计算：

$$
X = \frac {c _ {i} V _ {1}}{V _ {2}} \tag {1}
$$

式中： $X$ ——空气中氰溴甲苯浓度， $\mathrm{mg / m^3}$

$c_{i}$ ——浓缩液中氰溴甲苯的浓度， $\mu \mathrm{g / ml}$

$V_{1}$ ——浓缩液体积， $\mathrm{ml}$

$V_{2}$ ——换算成标准状态下的采样体积， $\mathrm{L}$ 。

# 8 结果的表示

# 8.1 定性结果

根据标准溶液色谱图保留时间进行样品中氰溴甲苯的鉴定。

# 8.2 定量结果

# 8.2.1 浓度的表示方法

按式（1）计算空气中氰溴甲苯的浓度，以 $\mathrm{mg / m^3}$ 表示。

# 8.2.2 精密度

结果相对标准偏差不大于 $15\%$ 。

# 8.2.3 准确度

样品加标回收率不小于 $70\%$ 。

# 8.3 检测限

按仪器基线噪声3倍所对应的氰溴甲苯的量作为仪器的检测限，本方法要求仪器的最低检测限为 $1 \times 10^{-9} \mathrm{~g}$ 。
