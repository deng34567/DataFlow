# 1 范围

本标准规定了测定空气中苯氯乙酮的气相色谱法。

本标准适用于销毁日本遗弃在华化学武器全过程空气中苯氯乙酮的测定。

当采样 $360\mathrm{L}$ 时，空气中苯氯乙酮最低检出浓度为 $0.026\mathrm{mg / m^3}$ 。

# 3 方法原理

采用硅胶吸附空气中的苯氯乙酮，经无水乙醇解吸后，用气相色谱法分离、测定。气相色谱法选用石英毛细管色谱柱、氢火焰离子化检测器（FID)。以色谱保留时间定性，以正十二烷为内标定量。

# 4 试剂和材料

除非另有规定，仅使用确认为分析纯的试剂。

# 4.1 试剂

# 4.1.1 无水乙醇。

# 4.1.2 内标物：正十二烷。

# 4.1.3 标准样品：苯氯乙酮，纯度大于 $98\%$

# 4.1.4 标准溶液配制

# 4.1.4.1 苯氯乙酮贮备溶液

称取苯氯乙酮（4.1.3） $0.05\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ）于 $50\mathrm{ml}$ 容量瓶中，用无水乙醇（4.1.1）溶解、定容。于 $4\%$ 下保存，保存时间 $30\mathrm{d}$ 。

# 4.1.4.2 苯氯乙酮中间溶液

准确移取一定体积苯氯乙酮贮备溶液（4.1.4.1）于 $50\mathrm{ml}$ 容量瓶中，用无水乙醇（4.1.1）稀释、定容，配制成浓度为 $100\mu \mathrm{g / ml}$ 的苯氯乙酮中间溶液。

# 4.1.4.3 正十二烷贮备溶液

称取正十二烷（4.1.2） $0.05\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ）于 $50\mathrm{ml}$ 容量瓶中，用无水乙醇（4.1.1）溶解、定容。于 $4^{\circ}\mathrm{C}$ 保存，保存时间 $30\mathrm{d}$ 。

# 4.1.4.4 正十二烷中间溶液

准确移取一定体积正十二烷贮备溶液（4.1.4.3）于 $50\mathrm{ml}$ 容量瓶中，用无水乙醇（4.1.1）稀释、定容，配制成浓度为 $250\mu \mathrm{g / ml}$ 的正十二烷中间溶液。

# 4.1.4.5 系列标准溶液

准确移取不等体积的苯氯乙酮中间溶液（4.1.4.2）于 $10\mathrm{ml}$ 容量瓶中，分别加入同一体积正十二烷中间溶液（4.1.4.4），用无水乙醇（4.1.1）定容，配制成苯氯乙酮浓度为 $1.0\sim 50\mu \mathrm{g / ml}$ 的系列（至少5个）标准溶液。内标物正十二烷浓度为 $25\mu \mathrm{g / ml}$ 。

# 4.2 气体

4.2.1 载气：氮气，纯度为 $99.999\%$
4.2.2 燃气：氢气，纯度为 $99.99\%$
4.2.3 助燃气：经净化的空气。

# 4.3 材料

硅胶：粒径为 $0.4\sim 0.8\mathrm{mm}$ 的粗孔硅胶。

# 5 仪器和装置

# 5.1 分析仪器

气相色谱仪：氢火焰离子化检测器（FID）。

# 5.2 色谱柱

5.2.1 类型：石英毛细管色谱柱。
5.2.2 尺寸： $30\mathrm{m}\times 0.25\mathrm{mm}\times 0.25\mu \mathrm{m}$ （长度 $\times$ 内径 $\times$ 膜厚）。
5.2.3 固定相：（5%苯基）甲基聚硅氧烷。
5.2.4 分离度：在设定条件下色谱柱总的分离度大于1.5。

# 5.3 采样仪器

空气采样器：按GB/T17061—1997中的第4章的规定执行。

# 5.4 其他仪器及装置

5.4.1 微量注射器： $1.0,10\mu \mathrm{l}_{\circ}$
5.4.2 多用调速振荡机。
5.4.3 样品瓶： $2\mathrm{ml}$
5.4.4 微孔滤膜过滤器：有机系列，孔径为 $0.45\mu \mathrm{m}$ 。
5.4.5 吸附管：硅胶管，按GB/T17061—1997中的第3章的规定执行。

# 6样品

# 6.1 采样方法

采样时将吸附管（5.4.5）连接在空气采样器（5.3）上，以 $6\mathrm{L} / \mathrm{min}$ 流速采气 $360\mathrm{L}$ ，采样后将吸附管密封并做好标识和记录。

# 6.2 样品保存

经采样的吸附管于 $4\%$ 下保存，保存时间 $3\mathrm{d}$

# 6.3 解吸

吸附管（6.1）用 $9\mathrm{ml}$ 无水乙醇（4.1.1）分三次解吸。其步骤如下：

a）将吸附管（6.1）中硅胶等取出，放入装有 $5.0\mathrm{ml}$ 无水乙醇（4.1.1）的 $25\mathrm{ml}$ 具塞磨口锥形瓶中，浸泡 $10\mathrm{min}$ ，置于振荡机（5.4.2）上振荡 $5\mathrm{min}$ ，取出 $2.0\mathrm{ml}$ 解吸液放入 $10\mathrm{ml}$ 具塞刻度试管内；
b) 在锥形瓶中加入 $2.0 \mathrm{ml}$ 无水乙醇，振荡 $5 \mathrm{~min}$ ，取出 $2.0 \mathrm{ml}$ 解吸液；
c）重复6.3b）操作一次，合并解吸液并用微孔滤膜（5.4.4）过滤后用无水乙醇定容。

# 6.4 试样制备

准确移取 $0.9\mathrm{ml}$ 解吸液（6.3c）于样品瓶（5.4.3）中，加入 $0.1\mathrm{ml}$ 正十二烷中间溶液(4.1.4.4)，摇匀，待测。

# 7 分析步骤

# 7.1 色谱条件

# 7.1.1 仪器温度

进样口温度： $250~\mathrm{^\circ C}$

检测器温度： $260~\mathrm{^\circ C}$

柱温：初温 $80\%$ ，程序升温 $10\% / \mathrm{min}$ ，终温 $150\%$ 。

# 7.1.2 载气和辅助气体及流速

氮气（4.2.1）流速为 $25\mathrm{ml / min}$

氢气（4.2.2）流速为 $30\mathrm{ml / min}$

空气（4.2.3）流速为 $400\mathrm{ml / min}$

# 7.1.3 进样方式

不分流进样，不分流时间为 $0.7\mathrm{min}$

# 7.1.4 进样量

$1.0\mu \mathrm{l}_{\circ}$

# 7.2 校准

# 7.2.1 标准曲线绘制

# 7.2.1.1 曲线绘制

色谱进样量为 $1.0\mu \mathrm{l}$ ，测定系列标准溶液（4.1.4.5）中苯氯乙酮与正十二烷的色谱峰面积比 $(A_{i} / A_{s})$ ，取三次测定平均值为结果，要求三次测定的峰面积比相对标准偏差不大于 $10\%$ 。以苯氯乙酮浓度（ $c_{i}$ ）为纵坐标，苯氯乙酮和正十二烷色谱峰面积之比（ $A_{i} / A_{s}$ ）为横坐标，绘制标准曲线并进行一元线性回归，得回归方程式（1）：

$$
c _ {i} = a + b A _ {i} / A _ {\mathrm {s}} \tag {1}
$$

式中： $c_{i}$ ——系列标准溶液中苯氯乙酮浓度， $\mu \mathrm{g / ml}$

$a$ ——标准曲线截距；

$b$ ——标准曲线斜率；

$A_{i} / A_{s}$ ——系列标准溶液中苯氯乙酮和正十二烷色谱峰面积比。

# 7.2.1.2 标准曲线校正

在样品测定前后，或分析了 $3\sim 5$ 个样品后，应插入一个标准曲线中浓度适中的标准溶液样品，其测定值与原配制值的相对误差不大于 $\pm 5\%$ ，否则应重新绘制标准曲线。

# 7.2.2 检测限

以二倍噪音所对应的苯氯乙酮的质量作为仪器的检测限，仪器的检测限为 $1.0 \times 10^{-10} \mathrm{~g}$ 。

# 7.2.3 空白

场地空白。

# 8 测定

# 8.1 定性分析

考查试样色谱图的组分分离状况，用色谱保留时间作为定性依据，被测组分与苯氯乙酮标准样品其色谱保留时间窗口为 $\pm 10\mathrm{s}$ 。本方法，苯氯乙酮的色谱保留时间为 $6.203\mathrm{min}$ ，正十二烷的色谱保留时间为 $5.025\mathrm{min}$ ，其色谱图见图1。

# 8.2 定量分析

# 8.2.1 分析

按7.2.1分析步骤，对试样（6.4）进行测定。

# 8.2.2 计算

空气中苯氯乙酮的浓度按公式（2）计算：

$$
c = \frac {c _ {i} V _ {t}}{0 . 9 V _ {\mathrm {n d}}} \tag {2}
$$

式中： $c$ ——空气中苯氯乙酮的质量浓度， $\mathrm{mg / m^3}$

$c_{i}$ ——由公式（1）计算的解吸液中苯氯乙酮的浓度， $\mu \mathrm{g} / \mathrm{ml}$ ；

$V_{t}$ ——解吸液的体积，ml；

$V_{\mathrm{nd}}$ ——标准状态下空气采样的体积，L；

0.9 ——试样制备中浓度换算系数。

# 9 结果表示

# 9.1 含量的表示

空气中苯氯乙酮含量以质量浓度表示， $\mathrm{mg / m^3}$

# 9.2 最低检出浓度

当采样 $360\mathrm{L}$ 时，空气中苯氯乙酮最低检测浓度为 $0.026\mathrm{mg / m^3}$ 。

# 9.3 精密度

相对标准偏差不大于 $20\%$ 。

# 9.4 准确度

按本方法，当空气中苯氯乙酮浓度在 $0.03\sim 0.3\mathrm{mg / m^3}$ 时，相对误差不大于 $20\%$ 。
