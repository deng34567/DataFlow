# 01什么是声环境功能区？

声环境功能区是为有效指导声环境保护工作的开展，按照《城市区域环境噪声适用区划分技术规范》（GB/T15190）对城市规划区内不同声环境功能的区域进行划分，以作为噪声污染防治的法定依据。

![](images/8cec9ce43968159d4e63daca006d74acefbbf4ba247e7bb90aff6beecd2be655.jpg)

# 02

# 声环境功能区分类及对应的噪声限值

根据我国现行《声环境质量标准》（GB3096）规定，声环境功能区分为五种类型：

![](images/f94047ba4292215f38d8612c85c4bd77e1638d5b38643caed1ba2e3c8c5720a0.jpg)

# 0类声环境功能区：

指康复疗养区等特别需要安静的区域。

![](images/79077250840ee427fbd91ace00a9314b551d1cd558fa0a4a8c0a36a2a66e8030.jpg)

# 1类声环境功能区：

指以居民住宅、医疗卫生、文化教育、科研设计、行政办公为主要功能，需要保持安静的区域。

![](images/3f5a6b8df3a75f15dfbf456498545ed665ed47166b8a6781c8990f18be6959c6.jpg)

# 2类声环境功能区：

指以商业金融、集市贸易为主要功能，或者居住、商业、工业混杂，需要维护住宅安静的区域。

![](images/4279ed1653e445ed46ae17ae8fc217adf23ebaead6fae427b651dfa6f95369e5.jpg)

# 3类声环境功能区：

指以工业生产、仓储物流为主要功能，需要防止工业噪声对周围环境产生严重影响的区域。

![](images/e63d746d189aebf6419400f7e622fb11fdfc0e4cf3751c663f1f7b792917d3b5.jpg)

# 4类声环境功能区：

指交通干线两侧一定距离之内，需要防止交通噪声对周围环境产生严重影响的区域，包括4a类和4b类两种类型。4a类为高速公路、一级公路、二级公路、城市快速路、城市主干路、城市次干路、城市轨道交通（地面段）、内河航道两侧区域；4b类为铁路干线两侧区域。

# 各类声环境功能区的噪声限值


| 时段声环境功能区类别 |     | 昼间  | 夜间  |
| ------------ | --- | --- | --- |
| 0类           |     | 50  | 40  |
| 1类           |     | 55  | 45  |
| 2类           |     | 60  | 50  |
| 3类           |     | 65  | 55  |
| 4类           | 4a类 | 70  | 55  |

# 03

# 海淀区声环境功能区划

海淀区声环境功能区划调整后包含1、2、4类声环境功能区。

# ·1类声环境功能区

海淀区辖区范围内，除2类声环境功能区和4类声环境功能区以外的地区，面积378.97平方公里。

# ·2类声环境功能区

（1）中关村永丰科技园：区域边界由北侧顺时针为玉河路-玉河南路-区界-唐家岭路-邓庄南路-六里屯社区路-永丰西路-永澄南路-永澄北路-宏丰渠东路，面积约16.10平方公里。
（2）中关村上地科技园：区域边界由北侧顺时针为后厂村路-京包铁路-上地南路-信息路-农大南路-树村郊野公园东侧边界-北大生物城西二路-马连洼南路-北大生物城西一路-马连洼北路-上地西路-东北旺南路-西北旺南路-西北旺东路，面积约5.91平方公里。
（3）清河、西三旗地区：区域边界由北

侧顺时针为海淀昌平区界-西小口路-黑泉路-永泰庄北路-后屯路-后屯中街-后屯东路-西小□路-安宁庄中街-安宁庄东路-毛纺路-朱房路-京包铁路，面积约为13.67平方公里。
（4）中关村翠湖科技园：区域边界由北侧顺时针为北清路-凝翠路-三星庄路-翠湖北环路-稻香湖路-翠湖南路-翠湖东路-京密引水渠-西六环路，面积14.70平方公里。

# ·4类声环境功能区

根据海淀区交通干线建设现状，此次涉及划分4类声环境功能区的道路等级为次干级以上的道路221条，城市轨道交通（地面段）4段，铁路3条；西郊机场为4a类场站，清河站为4b类场站。

针对两次声环境功能区划调整之间建成通车的交通干线，交通干线建设规划未实施前应按照当前声环境功能区类别管理，规划实施后交通干线两侧及时调整为4类声环境功能区。
