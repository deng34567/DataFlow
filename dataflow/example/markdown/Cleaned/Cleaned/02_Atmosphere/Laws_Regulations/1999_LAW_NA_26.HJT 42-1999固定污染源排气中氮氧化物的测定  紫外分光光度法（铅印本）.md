# 固定污染源排气中氮氧化物的测定紫外分光光度法

HJ/T42—1999

Stationary source emission—Determination of nitrogen oxide—Ultraviolet spectrophotometric method

# 1 适用范围

1.1 本标准适用于固定污染源有组织排放的氮氧化物测定。
1.2 当采样体积为1L时，方法的氮氧化物检出限为 $10\mathrm{mg / m^3}$ ；定量测定的浓度下限为 $34\mathrm{mg / m^3}$ ；在不作稀释的情况下，测定的浓度上限为 $1730\mathrm{mg / m^3}$ 。

# 2 方法原理

样品气体被收集在一个盛有稀硫酸-过氧化氢吸收液的瓶中，氮氧化物受到氧化和被吸收，成为 $\mathrm{NO}_3^-$ 存在于吸收液中，于 $210\mathrm{nm}$ 处测定 $\mathrm{NO}_3^-$ 的光吸收。

下列标准所包含的条文，通过在本标准中引用而构成为本标准的条文：

# 4 试剂与材料

除非另有说明，分析中均使用符合国家标准的分析纯试剂和蒸馏水。

4.4 过氧化氢溶液： $c = 3\%$ $(m / V)$ 。

4.1 硫酸： $\rho = 1.84\mathrm{g / ml}$
4.2 双氧水： $30\% (m / V)$ 。
4.3 硝酸钾：基准试剂。

在 $100\mathrm{ml}$ 容量瓶中，加入 $10.0\mathrm{ml}$ 双氧水（4.2），用水稀释至标线。

# 4.5 硝酸钾标准贮备液

准确称取事先于 $105\sim 110^{\circ}\mathrm{C}$ 干燥 $2\mathrm{h}$ 的硝酸钾（4.3） $2.198\mathrm{g}$ ，溶解于水并转移至 $1000\mathrm{ml}$ 容量瓶中，用水稀释至刻度。此溶液每毫升相当于 $\mathrm{NO}_21000\mu \mathrm{g}$ 。

# 4.6 硝酸钾标准使用液

准确吸取硝酸钾标准贮备液（4.5） $10\mathrm{ml}$ ，至 $1000\mathrm{ml}$ 容量瓶中，用水稀释至刻度。此溶液每毫升相当于 $\mathrm{NO}_210\mu \mathrm{g}$ 。

# 4.7 吸收液

于 $1000\mathrm{ml}$ 容量瓶中加入约 $500\mathrm{ml}$ 水，缓慢加入 $2.8\mathrm{ml}$ 硫酸（4.1)，再加入 $6\mathrm{ml}3\%$ 过氧化氢溶液(4.4)，摇匀后用水稀释至刻度。该溶液应贮存于棕色容量瓶中，使用时避免阳光直射，存放于暗处，可使用一周。

# 5 仪器

5.1 分光光度计：具紫外部分和 $1\mathrm{cm}$ 石英比色皿。
5.2 采样装置
5.2.1 采样管

具有适当尺寸的硬质玻璃或聚四氟乙烯管料，并备有可加热防止水气凝结的加温夹套。

5.2.2 样品吸收瓶

1L或2L的圆底烧瓶。其瓶口应可与一三支三通阀磨口相接，壁厚可承受一个大气压力。其外部具有泡沫塑料包壳，尺寸如图所示。

5.2.3 玻璃三支三通阀

可与样品吸收瓶（5.2.2）磨口相接。

5.2.4 负压表

测量精度应可达到 $0.5\mathrm{kPa}$ 或更好；最大量程达 $90\mathrm{kPa}$ 以上。在无法得到负压表的情况下，可采用具有相同功效的U型管压力表。

5.2.5 真空抽气泵

能将样品吸收瓶抽真空至绝对压力等于或低于 $10\mathrm{kPa}$ 。

5.2.6 温度计

可以从 $-5 \sim 50^{\circ} \mathrm{C}$ 每间隔 $1^{\circ} \mathrm{C}$ 进行测量。

5.2.7 连接管

内衬聚四氟乙烯薄膜的硅橡胶管。

5.3 玻璃棉。
5.4 真空活塞油脂。
5.5 玻璃二通阀。

# 6 样品采集和保存

6.1 采样位置和采样点

按GB16157—1996中9.1.1和9.1.2设置采样位置和采样点。

6.2 采样时间和频次

按GB16297—1996中8.2设置采样时间和采样频次。

6.3 采样装置的连接

将二通（5.5）和三通阀（5.2.3）的活塞及与吸收瓶相接的磨口部分涂好真空活塞油脂，采样管头部塞适量玻璃棉，吸收瓶内装好 $25.0\mathrm{ml}$ 吸收液（4.7）后，按下图相连接。连接管要尽可能短，并应保证密封性和可靠性。

6.4 采样前的检查

采样前首先要检查采样管头部是否已塞好适量玻璃棉，各连接点是否稳妥，然后检查样品吸收瓶的密封性。启动泵抽真空至一定程度后，关闭二通阀活塞，观察负压表指针，若在 $1\mathrm{min}$ 内负压下降超过1.3kPa，应检查漏气原因并重新连接。否则可进行采样。

6.5 样品采集

将吸收瓶内的真空度抽至负 $70\sim 90\mathrm{kPa}$ ，旋转三通阀活塞，使吸收瓶关闭，采样管抽取排气筒内的气体清洗管道约 $3\mathrm{min}$ ，再旋转三通阀活塞，使采样管内的气体迅速进入吸收瓶内。关闭吸收瓶，将吸收瓶连三通阀一起从采样装置中拆下，摇动 $5\mathrm{min}$ 后（注意避免阳光直射）带回实验室。

# 7 分析步骤

# 7.1 绘制校准曲线

取5只 $100\mathrm{ml}$ 容量瓶，分别加入0.00、5.00、10.0、15.0和 $20.0\mathrm{ml}$ $\mathrm{KNO}_3^-$ 标准使用液（4.6)，向每只容量瓶加入 $5.00\mathrm{ml}$ 吸收液（4.7)，用水稀释至刻度。在紫外分光光度计 $210\mathrm{nm}$ 处，以空白为参比测定吸光度。以吸光度为纵坐标，对应的 $\mathrm{NO}_2$ 含量（ $\mu \mathrm{g}$ ）为横坐标，绘制校准曲线，并计算校准曲线的线性回归方程。

# 7.2 样品分析

带回实验室的样品放置 $16\mathrm{h}$ (6.6) 以后，再摇动 $2\mathrm{min}$ ，然后开启三通阀活塞并取下三通阀，从每一个样品中准确吸取 $5.00\mathrm{ml}$ 吸收液，注入到 $100\mathrm{ml}$ 容量瓶中，用水稀释至刻度。以下按绘制校准曲线相同步骤进行样品分析。

# 8 计算和结果表示

# 8.1 采样体积计算

$$
V _ {\mathrm {n d}} = \left(V _ {\mathrm {b}} - V _ {i}\right) \times \frac {2 7 3}{1 0 1 3 0 0} \times \left(\frac {P _ {\mathrm {f}} - P _ {\mathrm {f v}}}{2 7 3 + t _ {\mathrm {f}}} - \frac {P _ {i} - P _ {i v}}{2 7 3 + t _ {i}}\right)
$$

式中： $V_{\mathrm{b}}$ ——吸收瓶和阀的容积（应采用重量法校准），L；

$V_{i}$ —吸收液体积，L；

$P_{\mathrm{f}}$ ——采样前吸收瓶内的压力，Pa；

$P_{i}$ ——采样后吸收瓶内的压力，Pa；

$t_{\mathrm{f}}$ ——采样前吸收瓶内的温度， $^{\circ}\mathrm{C}$ ；

$t_i$ ——采样后吸收瓶内的温度， $^{\circ}\mathrm{C}$ ；

$P_{\mathrm{fv}}$ ——温度为 $t_{\mathrm{f}}$ 度时的饱和水蒸气压力，Pa；

$P_{iy}$ ——温度为 $t_i$ 度时的饱和水蒸气压力，Pa；

$V_{\mathrm{nd}}$ ——换算成标准状态的采样体积，L。

# 8.2 样品中氮氧化物的计算

$$
c = 5 \times \frac {W}{V _ {\mathrm {n d}}}
$$

式中： $c$ ——样品中氮氧化物的浓度， $\mathrm{mg / m^3}$

$W$ ——由校准曲线查得，或由回归方程计算得到的氮氧化物（以 $\mathrm{NO}_2$ 计）质量， $\mu \mathrm{g}$ ；

$V_{\mathrm{nd}}$ ——同8.1。

5——在吸收液体积为 $25 \mathrm{ml}$ ，取其中 $5 \mathrm{ml}$ 进行测定情况下的系数。如果所取进行测定的体积有变化，该系数亦应作相应变化。

# 8.3 氮氧化物“排放浓度”的计算

按GB16157—1996中11.1.2或11.1.4计算氮氧化物的“排放浓度”。

# 8.4 氮氧化物“排放速率”的计算

按GB16157—1996中11.4计算氮氧化物的“排放速率”。

# 9 精密度和准确度

# 9.1 方法的精密度

五个实验室分别测定氮氧化物浓度为 $367\mathrm{mg / m^3}$ 的统一样品，测定重复性标准偏差为 $10\mathrm{mg / m^3}$ ；重复性相对标准偏差为 $2.9\%$ ；重复性为 $28\mathrm{mg / m^3}$ 。测定的再现性标准偏差为 $11\mathrm{mg / m^3}$ ；再现性相对标准偏差为 $3.3\%$ ；再现性为 $32\mathrm{mg / m^3}$ 。

五个实验室共同采集和分别测定某硝酸生产排放废气样品，测定结果的相对标准偏差为 $3.8\%$ 。

# 9.2 方法的准确度

五个实验室分别测定氮氧化物浓度为 $367\mathrm{mg / m^3}$ 的统一样品，测定总均值的相对误差为 $4.9\%$ ；各实验室测定均值的相对误差于 $2.4\% \sim 6.8\%$ 之间。

# 10 说明

对于浓度过高的样品，可以在采样前降低吸收瓶的抽真空程度，或减少取出进行分析的吸液体积；对于浓度过低的样品，可增加取出分析的吸收液体积，从而扩大测定范围，但同时应注意使标准系列与样品溶液的基本组分相同。
