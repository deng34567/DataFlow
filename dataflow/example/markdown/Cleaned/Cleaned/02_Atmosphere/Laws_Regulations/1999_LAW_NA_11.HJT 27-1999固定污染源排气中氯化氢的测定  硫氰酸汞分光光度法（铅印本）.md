# 固定污染源排气中氯化氢的测定硫氰酸汞分光光度法

# Stationary source emission—determination of hydrogen chloride—Mercuric thiocyanate spectrophotometric method

1999-08-18 发布

2000-01-01 实施

# 国家环境保护总局标准

# 固定污染源排气中氯化氢的测定硫氰酸汞分光光度法

HJ/T27—1999

Stationary source emission—Determination of hydrogen chloride—Mercuric thiocyanate spectrophotometric method

# 1 适用范围

1.1 本标准适用于固定污染源有组织排放和无组织排放的氯化氢测定。
1.2 在无组织排放样品分析中，当采气体积为 $60\mathrm{L}$ 时，氯化氢的检出限为 $0.05\mathrm{mg / m^3}$ ，定量测定的浓度范围为 $0.16\mathrm{mg / m^3}\sim 0.80\mathrm{mg / m^3}$ ；在有组织排放样品分析中，当采气体积为 $10\mathrm{L}$ 时，氯化氢的检出限为 $0.9\mathrm{mg / m^3}$ ，定量测定的浓度范围为 $3.0\mathrm{mg / m^3}\sim 24\mathrm{mg / m^3}$ 。
1.3 在本标准规定的显色条件下，当采气体积为 $100\mathrm{L}$ 时，氟化氢（HF）浓度高于 $0.2\mathrm{mg / m^3}$ ，硫化氢 $\mathrm{(H_2S)}$ 浓度高于 $0.1\mathrm{mg / m^3}$ ，以及氰化氢（HCN）浓度高于 $0.1\mathrm{mg / m^3}$ 时，将对氯化氢的测定产生干扰。

# 2 原理

用稀氢氧化钠溶液吸收氯化氢（HCl）。吸收溶液中的氯离子和硫氰酸汞反应，生成难电离的二氯化汞分子，置换出的硫氰酸根与三价铁离子反应生成橙红色硫氰酸铁络离子，根据颜色深浅，用分光光度法测定。反应式为：

$$
2 \mathrm {C l} ^ {-} + \mathrm {H g} (\mathrm {S C N}) _ {2} \longrightarrow \mathrm {H g C l} _ {2} + 2 \mathrm {S C N} ^ {-}
$$

$$
\mathrm {S C N} ^ {-} + \mathrm {F e} ^ {3 +} \longrightarrow \mathrm {F e} (\mathrm {S C N}) ^ {2 +}
$$

（橙红色）

# 4 试剂和材料

除非另有说明，分析时均使用符合国家标准的分析纯试剂和去离子水。

4.1 氢氧化钠。
4.2 硫氰酸汞。
4.3 硫酸铁铵： $\mathrm{[NH_4Fe(SO_4)_2\cdot 12H_2O]}$
4.4 氯化钾：优级纯，于 $110^{\circ}\mathrm{C}$ 烘干 $2\mathrm{h}$ 。
4.5 高氯酸： $\rho = 1.76\mathrm{g / ml}$
4.6 无水乙醇。
4.7 硫氰酸汞-乙醇溶液： $c = 0.04\mathrm{g} / 100\mathrm{ml}$

称取 $0.04\mathrm{g}$ 硫氰酸汞(4.2)，用无水乙醇（4.6）配成 $100\mathrm{ml}$ 溶液，放置一周后将上清液吸至另一

棕色细口瓶中备用。

4.8 高氯酸溶液： $1 + 1.5$

用量筒量取高氯酸(4.5) $50~\mathrm{ml}$ ，缓慢倒入 $75~\mathrm{ml}$ 水中，搅拌均匀后装入干净的试剂瓶中。

4.9 硫酸铁铵溶液： $c = 3.0\mathrm{g} / 100\mathrm{ml}$

称取 $3.0\mathrm{g}$ 硫酸铁铵(4.3)，用高氯酸溶液(4.8)溶解并稀释至 $100\mathrm{ml}$ ，如混浊应过滤。

4.10 氢氧化钠吸收液： $c(\mathrm{NaOH}) = 0.05\mathrm{mol / L}$

称取氢氧化钠(4.1)2.0g，溶于 $1000\mathrm{ml}$ 水中。

4.11 氯化钾标准储备液： $c(\mathrm{KCl}) = 1000\mu \mathrm{g} / \mathrm{ml}$

称取 $2.045\mathrm{g}$ 氯化钾(4.4)，溶解于水，移入 $1000\mathrm{ml}$ 容量瓶中，用氢氧化钠吸收液(4.10)稀释至刻度，摇匀。

4.12 氯化钾标准使用液： $c(\mathrm{KCl}) = 10.0\mu \mathrm{g / ml}$

移取 $10.0\mathrm{ml}$ 氯化钾标准储备液(4.11)于 $1000\mathrm{ml}$ 容量瓶中，用氢氧化钠吸收液(4.10)稀释至刻度，摇匀。

4.13 乙酸纤维微孔滤膜： $0.3\mu \mathrm{m}$

# 6.1.2 连接采样装置

参照GB16157—1996中9.3图28，按采样管、滤膜夹、吸收瓶、流量计量装置、抽气泵的顺序连接好采样装置，并按GB16157—1996中9.4的要求检查其气密性和可靠性。

# 6.1.3 采集样品

串联两支各装 $25\mathrm{ml}$ 氢氧化钠吸收液(4.10)的多孔玻板吸收瓶(5.3.3)，以 $0.5\mathrm{L / min}$ 流量，采样 $5\sim$ $30\mathrm{min}$ 。在采样过程中，根据排气温度和湿度调节采样管保温夹套温度，以避免水汽于吸收瓶之前凝结。

# 6.2 无组织排放样品采集

# 6.2.1 采样位置和采样点

按GB16297—1996中附录C的规定，确定无组织排放监控点的位置，或按其他特定要求确定采样点。

# 6.2.2 环境空气样品采集

按引气管、滤膜夹、吸收管、流量计量装置、抽气泵的顺序连接好采样装置，并检查其气密性和可靠性。将 $0.3\mu \mathrm{m}$ 滤膜(4.13)装在滤膜夹(5.3.2)内，后面串联两支各装 $5\mathrm{ml}$ 吸收液(4.10)的吸收管(5.3.3)，以 $1\mathrm{L / min}$ 流量采气 $30\sim 60\mathrm{min}$ 。长时间采样，需适当加水补充水分蒸发。

# 6.3 样品保存

如果样品采集后不能当天测定，应将试样密封后置于冰箱 $3\sim 5\%$ 保存，保存期不超过 $48\mathrm{h}$ 。

# 8 计算和结果表示

# 8.1 样品中的氯化氢浓度计算

有组织排放和无组织排放样品中氯化氢浓度按下式计算：

$$
\text {氯化氢 (H C l , m g / m ^ {3})} = \left(\frac {W _ {1}}{V _ {1}} + \frac {W _ {2}}{V _ {2}}\right) \times \frac {V _ {\mathrm {t}}}{V _ {\mathrm {n d}}}
$$

式中： $W_{1},W_{2}$ ——分别为从第一、二吸收管所取样品溶液中氯化氢含量， $\mu \mathrm{g}$

$V_{1}, V_{2}$ ——分别为测定时从第一、二吸收管中所取溶液的体积，ml；

$V_{\mathrm{t}}$ ——定容体积，ml；

$V_{\mathrm{nd}}$ ——标准状况下的干采气体积 $(0^{\circ}\mathrm{C}, 101.325\mathrm{kPa})$ ， $\mathrm{L}$ 。

按GB16157—1996中10.1或10.2计算 $V_{\mathrm{nd}}$ 。

# 8.2 氯化氢有组织排放的“排放浓度”计算

按GB16157—1996中11.1.2或11.1.4计算氯化氢的“排放浓度”。

# 8.3 氯化氢有组织排放的“排放速率 $(\mathrm{kg / h})$ ”计算

按GB16157—1996中11.4计算氯化氢的“排放速率”。

# 8.4 氯化氢的“无组织排放监控浓度值”计算

# 8.4.1 按下式计算一个无组织排放监控点的氯化氢平均浓度

$$
c = \frac {\sum_ {i = 1} ^ {n} c _ {i}}{n}
$$

式中： $c$ ——一个无组织排放监控点的氯化氢平均浓度；

$c_{i}$ ——一个样品中的氯化氢浓度；

$n$ ——一个无组织排放监控点采集的样品数目。

# 8.4.2 “无组织排放监控浓度值”的计算

按GB16297—1996附录C中C2.3计氯化氢的“无组织排放监控浓度值”。

# 9 精密度和准确度

# 9.1 方法的精密度

五个实验室分别测定浓度为 $2.80 \mathrm{mg} / \mathrm{m}^{3}$ 的标准样品，得到方法的重复性标准偏差为 $0.045 \mathrm{mg} / \mathrm{m}^{3}$ ，重复性相对标准偏差为 $1.6\%$ ，重复性为 $0.13 \mathrm{mg} / \mathrm{m}^{3}$ ；方法的再现性标准偏差为 $0.060 \mathrm{mg} / \mathrm{m}^{3}$ ，再现性相对标准偏差为 $2.1\%$ ，再现性为 $0.17 \mathrm{mg} / \mathrm{m}^{3}$ 。

# 9.2 方法的准确度

五个实验室分别测定浓度为 $2.80\mathrm{mg / m^3}$ 的标准样品，测定结果的总均值为 $2.75\mathrm{mg / m^3}$ ，总均值的相对误差及其置信区间范围为 $1.3\% \pm 1.4\%$ ，加标回收率的总均值及其置信区间为 $96.7\% \pm 5.4\%$ 。

# 10 说明

10.1 由于分析时所用的试剂及去离子水均含有微量 $\mathrm{Cl}^-$ ，使试剂空白液吸光度较高。
10.2 在采集无组织排放样品时，滤膜夹与第一吸收管、第一吸收管与第二吸收管之间不可用乳胶管连接，应采用聚四氟乙烯或聚乙烯塑料管以内接外套法连接，即将塑料管插入滤膜夹出口及吸收管管口，用聚四氟乙烯胶带缠好，接口处再套一小段硅橡胶管。
10.3 在采集有组织排放样品时，采样管与第一吸收瓶、第一吸收瓶与第二吸收瓶之间不可用乳胶管连接，应用聚乙烯管或用聚四氟乙烯塑料管以内接外套法连接，即将塑料管插入吸收瓶管口，用聚四氟乙烯

胶带缠好后，外面再用一段硅橡胶管接好管口。

10.4 废气中含有固体氯化物颗粒时，应在第一吸收瓶之前增加滤膜夹。
10.5 用过的吸收瓶、具塞比色管、连接管等，将溶液倒出后，直接用去离子水洗涤，不要用自来水洗涤；在操作过程中应注意防尘：手指不要触摸吸收瓶管口、比色管磨口处，以防氯化物沾污。
10.6 采样分析时，样品溶液、标准溶液、和空白对照必须用同一批试剂同时操作，所加试剂量也要求准确。
10.7 试剂空白液吸光度较高而且不够稳定时，应多次测定其吸光度，在获得稳定数值后，再绘制校准曲线及测定样品。
