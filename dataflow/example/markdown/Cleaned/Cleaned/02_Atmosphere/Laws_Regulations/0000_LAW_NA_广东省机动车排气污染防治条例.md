# 第一章总则

第一条了防治机排气污染，保和改善大气境，保为动车护环障人体健康，促社、、境的展，根据《大气污染进会经济环协调发防治法》及有法律、法，合本省，制定本例。关规结实际条

第二条本例适用于本省行政域由燃机或者引条区内内驱动牵的机向大气境排放污染物所造成的污染防治。动车环

前款所机不包括路机和拖拉机。称动车铁车

第三条以上生境主管部本行政域的机县级态环门对区内动车排气污染防治施一督管理，并同有主管部的机实统监对级关门动车排气污染防治督管理工作行和指。监进协调导

公安、交通、市督管理、能源等主管部根据各自运输场监门职责对动车实监，机排气污染防治施督管理。

第四条任何位和人机排气污染超行可以投单个对动车标为诉和。生境主管部及有主管部向社公布投举报态环门关门应当会诉举报电话讯规对诉举报处、通地址，并按照定投和予以理和答复。

# 第二章排气污染防治

第五条省人民政府可以根据机排气污染防治的需要，依动车照家有定，定新提前行家段性机排放国关规决对车执国阶动车标准，并可以根据大气污染防治要求，珠江三角洲地以及其他大对区气污染防治重城市制定更格的机排气污染防治定。点严动车规

对车执国阶动车标应新提前行家段性机排放准的，省人民政府

当会车应提前向社公告并采取措施保障相配套的用燃料的供，省生态环门应当国录应动车境主管部按照家公布的目提前公布符合相机排放准的型。标车

第六条以上人民政府机排气污染防治入道县级应当将动车纳路与交通划，大力展公共交通，鼓、支持使用清洁能源的机规发励动车节动车开发产销和能型低污染排放机的、生、售、使用。

县级对区内营运单节以上人民政府本行政域位改用能型低污染排放机工作行划和施，加客汽、动车进规组织实强对营运载车载货汽等排气污染重的机的督管理，并可以根据大气污车严营运动车监染防治的需要，限制特定域摩托、低速等的保有区内车货车车辆量。

第七条机排气污染重城市的人民政府采取措施动车严应当预防光化等污染事件，并制定相的急案。学烟雾应应预

第八条公安机交通管理部助生境主管部关门应当协态环门对机排气污染行督管理，有机排气果入机动车进监将关动车检测结纳动车内将动车环关内态环交通管理的容，并机保有容与生境主管部门实资施源共享。

第九条省市督管理部同省生境主管部适场监门应当会态环门时车标对动车车产实制定用燃料地方准，并机和用燃料的生、加工施量督。质监

第十条公安机交通管理部未到本行政域行行关门对达区现执的家段性机污染物排放准的机，不予理注登国阶动车标动车办册记。

在用机所有人申定期合格志的，所交的机动车请检验标验动车应当动车册记时国规动车标符合机注登家定的机污染物排放准。

跨地以上市入的在用级迁车，迁粤东粤粤区入西北地，符合迁入地在用排放准要求的，公安机交通管理部理登车标关门应当办记迁区应当迁现执国动；入珠江三角洲域的，符合入地行行的家机车标应阶大气污染物排放准中相段排放限值。国规家另有定的除外。

第十一条禁止排放不合格的机上道路行。检验动车驶

第十二条禁止生、售、口超家定的污染物排放产销进过国规标国动车车发动准或者家已明令淘汰的机及用机。

第十三条禁止生、售、口不符合家或者地方准的产销进国标车用燃料。

第十四条新建加油站及新登油罐按照家准配套记车应当国标安油气回收系；已建加油站及在用油罐在家准定装统车应当国标规的期限完成油气回收合治理施安。内综设装

第十五条机的所有人、人加机的动车驾驶应当强对动车维护保，保持机排气污染控制置的正常行，避免置失效造养动车装运装成机排气污染超定准，不得擅自拆除、置或者更改在动车过规标闲用机排气污染控制置。动车装

# 第三章排气污染检测

第十六条省生境主管部和省市督管理部可以根态环门场监门

据家有定，适合本省的机排气污染方法，国关规选择实际动车检测确定相的排放限值，省人民政府批准后公布、施。应报实

第十七条省和地以上市生境主管部机排级态环门应当将动车放机检验录会构名向社公告。

机排放机动车检验须计检检测设备够构必具有量定合格的、能与地地以上市生境主管部及公安机交通管理部当级态环门关门实时传输检测数设备国规检测术员排气据的和符合家定要求的技人。

机排气污染定期与机安动车检测应当动车术检验进全技同步行。

第十八条机排放机动车检验应当规构遵守下列定：

（一）依据法定的方法、准机排气污染行检测检测标对动车进检测；

（三）出具真实动车检测结、准确的机排气污染果；
