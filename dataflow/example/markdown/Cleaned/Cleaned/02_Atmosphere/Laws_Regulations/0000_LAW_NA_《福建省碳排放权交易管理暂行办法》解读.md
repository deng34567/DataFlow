# 《福建省碳排放权交易管理暂行办法》

# 解读

一、《福建省碳排放交易管理行法》（以下权暂办简称《行法》）的制定背景和立法目的？暂办

《行法》第一定暂办条规 :“为温了建立室气体排放控制市机制，范碳排放交易活，推家生文明场规权动进国态试验区设关规结实际办建，根据有法律法，合本省，制定本法。”

解：福建省生优比明，但也面加快展读态势较显临发与源境束力，有源境制度以适资环约趋紧压现资环难应转方式构、推色展需要，已成我省可持调结动绿发为实现续发设绿会战展，建色低碳社的重要挑。2016年8月12日，中共中央公、院公印了《家生文明办厅国务办厅发国态试验区实办发（福建）施方案》（中 ﹝2016﹞58号），明确支持我省深化碳排放交易，出台碳排放交易权试点权实施，立碳排放交易平台，展碳排放交易，细则设权开权实施林碳交易，探索林碳交易模式。省委、省业汇试点业汇政府我省碳排放交易市建高度重，要求我省要对权场设视

积鉴试点经验极借其他先期省市，在 2016 年底前初步建成具有福建特色的碳排放交易市。权场

根据碳排放交易其他省市，通地方国内权试点经验过立法，与企范、碳排放配的清、碳排放对参业围额缴义务报关处罚规场告与核查要求、相的措施作出定，市才能正常。因此，了落党中央、院和我省工作运转为贯彻实国务部署，建立室气体排放控制市机制，范碳排放交温场规权易活，充分碳排放交易市在少碳排放与动发挥权场实现减经济会发双赢进国态试验社展“ ”中的重要作用，推家生文明区设时权暂办建，及出台《福建省碳排放交易管理行法》是十分必要和迫切的。

# 二、《行法》主要范了哪些容？暂办规内

《行法》共暂办 40 条个，主要在以下几方面作了明确规权围义则定：（一）碳排放交易的适用范、定及原；（二）碳排放交易的管理与分工；（三）碳排放配权职责额权场报管理；（四）碳排放市交易；（五）碳排放告、核查与配清；（六）碳排放交易市与者的法律额缴权场参责任。

# 三、碳排放交易适用范、定及原？权围义则

《行法》第二定暂办条规 :“本省行政域的碳排放区内权交易及其督管理活，适用本法。本法所碳排放监动办办称权设总点交易，是指由省人民政府定年度碳排放量以及重排放位的排，重排放位通市机制履行单减义务点单过场义务报报的碳排放控制机制，主要包括碳排放告送、核查、配核、交易以及履等。”第三定：“碳排放交易额发约条规权坚导场运结开持政府引与市作相合，遵循公、公平、公正和信原。”诚则

解：碳排放交易市是政府把碳排放空作一读权场间为种公共源，以一定的分配方式分配重排放位，并资给点单在一定下行市化交易的碳排放控制机制。建立碳规则进场排放交易制度是我省生文明建的一全新的权态试验区设项探索，因此，《行法》有必要明确碳排放交易的适暂办权用范、定和原。围义则

# 四、碳排放交易的管理与分工？权职责

《行法》第四定暂办条规 :“省、的市人民政府展设区发改革部是本行政域碳排放交易的主管部，本门区权门负责

行政域碳排放交易市的督管理。省人民政府金融区权场监工作机构是全省碳排放交易所的管理部，权场统筹门负责碳排放交易所准入管理、督查、置等督权场监检风险处监管理工作。省、市人民政府和信息化、政、住设区经济财房和城建、交通、林、海洋与、有乡设运输业渔业国资产监统计质术监门职督管理、、价格、量技督等部按照各自责,协权关监同做好碳排放交易相的督管理工作。根据碳排放交易主管部的授或者委托，碳排放交易的技权门权权术支位碳排放送系、注登系的建和撑单负责报统册记统设运行等相工作。”维护关

解：碳排放量控制的任主体是各市人民政读总责设区府，碳排放交易的配量也分配到各市，因此权额总将设区赋设区职够证实总予各市必要的管理能，能保落碳排放量控制任目和相政策措施，且充分各市极责标关调动设区积性，同，也符合党中央、院政放的方。于时国务简权针对覆范、配量、配分配方法等重要的，盖围额总额顶层设计则权门统筹负责则会由省人民政府碳排放交易主管部，否出政出多，市以一。同，在碳排放交易活现门场难统时权动还记点单数中涉及碳排放信息登、重排放位碳排放据采集、碳排放交易据研究分析、省室气体清制权数级温单编

等技工作需要技支位支持。此，《行法》术术撑单为暂办对权术撑单规碳排放交易的技支位作了明确定。

# 五、碳排放交易覆范？权盖围问题

《行法》第五定：“省人民政府碳排放交易暂办条规权主管部照院碳排放交易主管部的相定，门参国务权门关规结产业结实际况纳权合本省构等情，公布入碳排放交易的温类业围点单标室气体种、行范和重排放位确定准。”第九条规设区权门定：“ 的市人民政府碳排放交易主管部按照省人民政府碳排放交易主管部公权门标布的准，提出本行政区点单单权域重排放位名，由省人民政府碳排放交易主管部定并门审会向社公布。”

解：探索建立碳排放交易市是一性读权场项专业强很的工作，需要先易后难稳进将强，步推，先碳排放量大、度高的工企入碳排放交易市，并业业纳权场扩逐步大到建筑、交通等行，业终实现盖温最全覆。根据我省室气体排放行特等因业点纳权场素，确定我省首批入碳排放交易市的重排放位点单为2013至2015年中任意一年合能源综消费总达量 1万吨煤标电以上的力、石化、化工、建材、钢铁纸、有色、造、航空、陶瓷等9个业业行企。

# 六、碳排放量总如何确定？

《行法》第十定：“省人民政府碳排放交易暂办条规权主管部根据本省室气体控制体目，合门温总标结经济长增、产业转级点单况设型升、重排放位情等因素，定年度碳排放配量，制定碳排放配分配和管理，省人民额总额细则报政府批准后向社公会布。”

解：碳排放量控制是读总础实总基，只有行量控制，碳排放才成权会为资会为产稀缺源，才成一种可交易的品。同，建立碳排放交易市是极时权场积场进节利用市机制促能降碳的重要手段，其目的是了为进续发促可持展，因此，总标应该紧结经济会发设量目密合我省社展需要，合理定。

# 七、碳排放配权额如何分配？

《行法》第十一定：“省人民政府碳排放交暂办条规权易主管部根据行门应当业减潜点基准水平、排力和重排放单历经关业门位史碳排放水平等因素，征求相行主管部意见额设区后，制定碳排放配具体分配方案；的市人民政府碳排放交易主管部根据分配方案核定本行政域重权门区点排放位的单费额数过册记统免分配配量，通注登系向本行

政域的重排放位区点单费发额实动态免放。碳排放配行管理，每年确定一次。”第十二定：“碳排放配初期采条规额取免费时偿偿分配方式，适引入有分配机制，逐步提高有分配的比例。省人民政府碳排放交易主管部通有分权门过偿配取得的收益缴级财库实两条线入省政金，行收支管理，相工作所需支出由省政关级财统筹进减安排，用于促本省少碳排放及相的能力建。”关设

解：碳排放初读权额权场始配分配是碳排放交易市建设将权额给业会产的核心，只有碳排放配合理分配企，才生交易需求。我省碳排放交易市建权场设刚起步，初始配额费发减轻来业绪顺动免放有利于自企的抵触情，利推碳排放交易市建。碳排放交易市建的成权场设随着权场设熟和完善，有分配具有碳价格、偿发现发业节减激企能碳的积时进偿动业创极性，适推有分配，有利于推企新低碳技术强发挥权场，加碳排放管理，有利于充分碳排放交易市手段在促进绿发我省色低碳展中的作用。

《行法》第十暂办条规权产八定：“碳排放交易的品包括碳排放配、家核自额国证减励创愿排量以及本省鼓探索

新的碳排放交易相权关产条规权品等。”第十九定：“碳排放交易的主体包括入碳排放配管理的重排放位以及纳额点单其他符合交易且自规则参权愿与碳排放交易的公民、法人或者其他。”第二十明确，碳排放交易在省人组织条权应当民政府确定的交易机构行，省人民政府碳排放交易内进权主管部其行督管理。第二十一明确，交易门对业务进监条机构制定交易，建立交易系，建立信息应当规则统披露制度，建立和行管理制度。第二十三定：“碳排放执风险条规配的交易价格由交易与方根据市额参场关供求系确定，禁止通过纵发虚乱权操供求和布假信息等方式扰碳排放交易市场条权秩序”。第三十一明确，省人民政府碳排放交易主管部门应当会条向社公布的信息。第三十二明确信用管理问题规档纳，定建立信用案，入信用管理体系，建立守信激励惩关内条和失信戒制度的有容。第三十三明确，履行责点单报项类别任的重排放位可以优先申的目。

# 九、第三方核查机构如何管理？

《行法》第二十六定：“省人民政府碳排放暂办条规权交易主管部建立第三方核查机构门录库强动态名，加管理，通性过竞争购对点磋商等采方式确定第三方核查机构，重排放位的碳排放告行第三方核查。重排放位单报进点单应当开配合第三方核查机构展核查工作，并按照要求提供相关绝挠应当材料，不得拒、干扰和阻。第三方核查机构按照省人民政府碳排放交易主管部的要求展碳排放核权门开查工作，出具核查告，并履行保报义务报应当密。核查告真实权门对报准确。省人民政府碳排放交易主管部核查告进费从级预内资行抽查。核查、抽查用省算基建金中予以安排。”

解：碳排放交易的主要读权产额额数品是排放配，配量多少、真实详细监测计认与否除了要建立可操作的划，真编报制排放告外，主要依靠核查机构的核查。因此，核查机构的能力、资质职业业务关操守和程序直接系交易市场时为进的公平。同，了促第三方核查机构的公平、公正，

《行法》在第二十暂办条点单对结七明确重排放位核查果有异议处条规的理方式，第二十八定委托第三方核查机构测点单认时算重排放位上年度排放量确的方式方法。同，在第三十六第三方核查机构的法律任作出明确定条对责规。

# 十、于重排放位履清关点单约缴问题

《行法》第二十暂办条规点单应当九定：“重排放位在每年6月底前向设区权门的市人民政府碳排放交易主管部提交不少于上年度确排放量的排放配，履行经认额上年度的配额额缴义务条规励点单足清。”第三十定：“鼓重排放位使用家或者省人民政府碳排放交易主管部核的经国权门证林碳目自业汇项减经认愿排量抵消其确的碳排放量，也可以使用除林碳业汇领国证减外其他域家核自愿排量抵消其部分确的碳排放量，具体经认办规抵消法另行定。”

解：建立履制度是读约权场形成碳排放交易市的根本保。证设约额点只有置履机制，才能促使配不足的重排放单过场购买额额卖额位通市配，而配富余的可以出配，确保市的正常。重排放位通场运转点单过缴上与排放量相等的配额约时励点单购买完成履，同也鼓重排放位使用一定比例的林碳等自业汇减约为愿排量完成履。此外，了充分
