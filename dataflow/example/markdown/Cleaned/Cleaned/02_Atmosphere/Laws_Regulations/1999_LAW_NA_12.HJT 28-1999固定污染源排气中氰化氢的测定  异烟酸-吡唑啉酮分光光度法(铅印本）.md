# 1 适用范围

1.1 本标准适用于固定污染源有组织排放和无组织排放的氰化氢测定。
1.2 在氰化氢无组织排放的空气样品分析中，当采样体积为 $30\mathrm{L}$ 时，方法的检出限为 $2\times 10^{-3}\mathrm{mg / m^3}$ 定量测定的浓度范围为 $0.0050\sim 0.17\mathrm{mg / m^3}$ 。在有组织排气样品分析中，当采样体积为 $5\mathrm{L}$ 时，方法的检出限为 $0.09\mathrm{mg / m^3}$ ，定量测定浓度范围为 $0.29\sim 8.8\mathrm{mg / m^3}$ 。
1.3 硫化氢和氧化剂（如 $\mathrm{Cl}_2$ ）存在对测定有干扰。

# 2 方法原理

用氢氧化钠溶液吸收氰化氢（HCN），在中性条件下，与氯胺T作用生成氯化氰（CNCl)，氯化氰与异烟酸反应，经水解生成戊烯二醛，再与吡唑啉酮进行缩聚反应，生成蓝色化合物，用分光光度法测定。反应方程式如下：

称取 $3.4\mathrm{g}$ 硝酸银（4.5），溶解于水中并稀释至 $1000\mathrm{ml}$ 。贮于棕色细口瓶中。

标定：吸取氯化钠标准溶液 $10.00\mathrm{ml}$ ，置于 $150\mathrm{ml}$ 锥形瓶中，加水 $50\mathrm{ml}$ 及 $4\sim 5$ 滴铬酸钾指示剂，用硝酸银标准溶液滴定，直至溶液由黄色变成浅砖红色。记下读数 $(V)$ 。平行滴定所消耗硝酸银标准溶液体积之差应不大于 $0.04\mathrm{ml}$ 。

取水 $60\mathrm{ml}$ ，同法做空白滴定。按下式计算硝酸银标准溶液的浓度：

$$
c \left(\mathrm {A g N O} _ {3}\right) = \frac {c (\mathrm {N a C l}) \times 1 0 . 0 0}{V - V _ {0}}
$$

式中： $c$ （ $\mathrm{AgNO_3}$ ）——硝酸银标准溶液的浓度，mol/L；

$c$ (NaCl)——氯化钠标准溶液的浓度，mol/L；

$V$ 、 $V_{0}$ ——分别为滴定氯化钠标准溶液、空白溶液时消耗硝酸银标准溶液体积， $\mathrm{ml}$ 。

4.22 氰化钾标准贮备液

液（4.16）稀释至 $100\mathrm{ml}$ ，混匀后避光贮存于棕色细口瓶中。该溶液每毫升约相当于 $1.0\mathrm{mg}$ 氰化氢。

标定方法：吸取 $10.00\mathrm{ml}$ 氰化钾贮备液，置于 $150\mathrm{ml}$ 锥形瓶中，加 $50\mathrm{ml}$ 水和 $2\%$ 氢氧化钠溶液（4.15） $1.0\mathrm{ml}$ ，加 $2\sim 3$ 滴试银灵指示剂，用硝酸银标准溶液滴定至溶液由淡黄色变为橙红色，记录消耗硝酸银标准溶液体积 $(V)$ 。平行滴定所消耗硝酸银溶液体积之差应不超过 $0.04\mathrm{ml}$ 。

$$

$$

式中： $c$ —氰化钾贮备液中相当于氰化氢的浓度， $\mathrm{mg / ml}$

$c(\mathrm{AgNO}_3)$ ——硝酸银标准溶液的浓度， $\mathrm{mol / L}$

$V_{\mathrm{v}}$ 、 $V_{0}$ ——分别为滴定氰化钾贮备液、空白溶液所消耗硝酸银标准溶液体积， $\mathrm{ml}$ 。

54.04——相当于 $1\mathrm{L}1\mathrm{mol} / \mathrm{L}$ 硝酸银标准溶液的氰化氢（HCN）质量，g。

# 6.3 样品保存

如果样品采集后不能当天测定，应将试样密封后置于 $2\sim 5^{\circ}\mathrm{C}$ 下保存，保存期不超过 $48\mathrm{h}$ 。在采样、运输和贮存过程中应避免日光照射。

# 6.4 采样防护

采集有组织排气的人员必须两人以上，并戴好防毒面具才能进入现场采样。

# 8 计算和结果表示

# 8.1 无组织排放样品浓度计算

$$
\text {氰化氢} \left(\mathrm {H C N}, \mathrm {m g / m ^ {3}}\right) = \frac {W}{V _ {\mathrm {n d}}}
$$

式中： $W$ ——样品溶液中氰化氢含量， $\mu \mathrm{g}$

$V_{\mathrm{nd}}$ ——换算成标准状态下的干采气体积，L。

# 8.2 有组织排放样品浓度计算

$$
\text {氰化氢 (H C N , m g / m ^ {3})} = \frac {W}{V _ {\mathrm {n d}}} \times \frac {V _ {\mathrm {t}}}{V _ {\mathrm {a}}}
$$

式中： $W$ ——测定时所取样品溶液中氰化氢含量， $\mu \mathrm{g}$

$V_{\mathrm{nd}}$ ——换算成标准状态下的干采气体积，L；

$V_{\mathrm{t}}$ ——样品溶液总体积， $\mathrm{ml}$

$V_{\mathrm{a}}$ ——测定时所取样品溶液体积， $\mathrm{ml}$ 。

按GB16157—1996中10.1或10.2计算 $V_{\mathrm{nd}}$ 。

8.3 氰化氢有组织排放的“排放浓度”计算

按照GB16157—1996中11.1.2或11.1.4计算氰化氢的“排放浓度”。

8.4 氰化氢有组织排放的“排放速率”计算

按照GB16157—1996中的11.4计算氰化氢的排放速率。

8.5 氰化氢的“无组织排放监控浓度值”计算

8.5.1 按下式计算一个无组织排放监控点的氰化氢平均浓度

$$
c = \frac {\sum_ {i = 1} ^ {n} c _ {i}}{n}
$$

式中： $c$ ——一个无组织排放监控点的氰化氢平均浓度；

$c_{i}$ ——一个样品的氰化氢浓度；

$n$ ——一个无组织排放监控点采集的样品数目。

8.5.2 “无组织排放监控浓度值”的计算

按GB16297—1996附录C中C2.3计算氰化氢的“无组织排放监控浓度值”。

# 9 精密度和准确度

# 9.1 精密度

五个实验室测定HCN浓度为 $0.37\mathrm{mg / m^3}$ 的统一样品，得到方法的重复性标准偏差为 $0.01\mathrm{mg / m^3}$ 重复性相对标准偏差为 $3.1\%$ ，重复性为 $0.030\mathrm{mg / m^3}$ ；方法的再现性标准偏差为 $0.016\mathrm{mg / m^3}$ ，再现性相对标准偏差为 $4.1\%$ ，再现性为 $0.040\mathrm{mg / m^3}$ 。

五个实验室测定实际样品的相对标准偏差于 $0.81\% \sim 11\%$ 之间。

# 9.2 准确度

五个实验室分别测定浓度为 $0.37\mathrm{mg / m^3}$ 的统一样品，测定总均值的相对误差为 $2.90\%$ ；各实验室测定均值的相对误差于 $\pm 5.4\%$ 之间。

五个实验室测定统一样品的加标回收率于 $93.2\% \sim 103\%$ 之间；测定实际样品的加标回收率于 $90.1\% \sim 102\%$ 之间。

# 10 说明

10.1 氰化氢是易挥发的有毒物质，在操作过程中，除了加试剂外，比色管都应盖严。
10.2 绘制校准曲线和样品测定时的温度之差应不超过 $3^{\circ}\mathrm{C}$ 。
10.3 如能获得硝酸银基准试剂，可用直接法配制硝酸银标准溶液，免去标定步骤。
10.4 为降低试剂空白值，实验中以选用无色的 $\mathbf{N}$ ， $\mathbf{N}^{\prime}$ 二甲基甲酰胺为宜。
10.5 含氰化钾的废液应加三价铁盐或漂白粉处理后排放。
10.6 氧化剂（如 $\mathrm{Cl}_2$ ）和硫化氢存在对测定有干扰，检验试样中是否存在氧化剂和硫化氢及消除干扰的方法见本标准附录A。

# 检验试样中氧化剂和硫化物及消除干扰的方法

A1 检验试样中是否存在氧化剂（如有效氯），可取1滴试样滴在淀粉-碘化钾试纸上，若变蓝说明氧化剂存在。若有显著量的氧化剂存在，氰化氢的测定不能进行。若氧化剂存在量小，可向样品溶液中加入一定量的亚硫酸钠溶液消除其干扰。具体做法是：量取两份相同体积的同一样品，向其中一份样品中投入淀粉-碘化钾试纸 $1\sim 3$ 片，加硫酸酸化，用亚硫酸钠溶液滴定至淀粉-碘化钾试纸由蓝色变至无色为止，记录用量。另一份样品，不加试纸和硫酸，仅加上述同量的亚硫酸钠溶液，并将此溶液进行氰化氢测定。A2 检验试样中是否存在硫化物，可取1滴试样滴在乙酸铅试纸上，若变黑，说明有硫化物存在。若试样中含有少量硫化物，可用预蒸馏的方法予以去除，蒸馏前加入 $2\mathrm{ml}0.02\mathrm{mol} / \mathrm{L}$ 硝酸银溶液。
