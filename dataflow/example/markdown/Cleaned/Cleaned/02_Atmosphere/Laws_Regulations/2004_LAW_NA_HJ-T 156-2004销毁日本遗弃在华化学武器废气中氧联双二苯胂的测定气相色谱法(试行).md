# 1 范围

本标准规定了测定废气中氧联双二苯胂的气相色谱分析方法。

本标准适用于销毁日本遗弃在华化学武器全过程废气中氧联双二苯胂浓度的测定。

当采样体积为 $30\mathrm{L}$ 时，废气中氧联双二苯肿的最低检测浓度为 $5\times 10^{-2}\mathrm{mg / m^3}$ 。

# 3 方法原理

废气中的氧联双二苯胂经乙腈吸收液吸收后，加入衍生化试剂1-丁基硫醇转化成含硫化合物，经气相色谱毛细管柱分离，用硫型脉冲火焰光度检测器检测。以保留时间定性，外标法定量。衍生化反应式如下：

$$
\left(\mathrm {C} _ {6} \mathrm {H} _ {5}\right) _ {2} \mathrm {A s} \mathrm {—} \mathrm {O} \mathrm {—} \mathrm {A s} \left(\mathrm {C} _ {6} \mathrm {H} _ {5}\right) _ {2} + 2 \mathrm {S H C H} _ {2} \mathrm {C H} _ {2} \mathrm {C H} _ {2} \mathrm {C H} _ {3} \longrightarrow 2 \left(\mathrm {C} _ {6} \mathrm {H} _ {5}\right) _ {2} \mathrm {A s} \mathrm {—} \mathrm {S} \mathrm {—} \mathrm {C H} _ {2} \mathrm {C H} _ {2} \mathrm {C H} _ {2} \mathrm {C H} _ {3}
$$

# 4 试剂和材料

# 4.1 载气和辅助气体

4.1.1 载气：氮气，纯度为 $99.999\%$
4.1.2 燃气：氢气，纯度为 $99.99\%$
4.1.3 助燃气：净化后的空气。

# 4.2 试剂

4.2.1 标准样品：氧联双二苯肿，纯度不小于 $98\%$
4.2.2 吸收液：乙腈，分析纯。
4.2.3 衍生化试剂：1-丁基硫醇，纯度不小于 $98\%$ 。

4.2.4 贮备溶液：称取氧联双二苯肿（4.2.1）约 $20\mathrm{mg}$ （精确至 $0.0001\mathrm{g}$ ）于 $50\mathrm{ml}$ 容量瓶中，用乙腈（4.2.2）溶解，定容。此贮备液密封后在 $4\%$ 时可保存 $30\mathrm{d}$ 。
4.2.5 氧联双二苯肿系列标准溶液：将氧联双二苯肿贮备溶液用乙腈（4.2.2）稀释后，配制成浓度分别为 $100\sim 1000\mathrm{ng / ml}$ 的氧联双二苯肿系列标准溶液。

# 5 仪器

# 5.1 色谱仪

气相色谱仪，具硫型脉冲火焰光度检测器（PFPD-S）。

# 5.2 进样器

微量进样器：5.0、 $10.0\mu \mathrm{l}$ 。

# 5.3 试样预处理使用的仪器

试样预处理使用的仪器有：

a）具塞样品瓶： $4\mathrm{ml}$
b) 超声波振荡器；
c）微量移液器：50.0、100.0、500.0 $\mu \mathrm{l}$ 。

# 5.4 采样时使用的仪器

# 5.4.1 采样装置

具加热采样管的烟气采样装置。

# 5.4.2 吸收管

采用GB/T16157—1996中9.3.4b）规定的大型气泡式吸收管， $25\mathrm{ml}$

# 5.4.3 滤膜

丙纶滤膜或玻璃纤维滤膜（作批次干扰检验），直径 $20\mathrm{mm}$ 。

# 5.4.4 滤膜夹

滤膜夹，直径 $20\mathrm{mm}$

# 5.4.5 连接管

聚四氟乙烯软管或硅橡胶软管。

# 6样品

# 6.1 样品采集

# 6.1.1 采样位置和采样点

采样位置及采样点数目，按GB/T16157—1996中9.1有关规定执行。

# 6.1.2 采样频次和时间

按GB19060—2003有关规定执行。

# 6.1.3 操作步骤

按GB/T16157—1996中9.3连接采样系统。将滤膜（5.4.3）和内装 $10\mathrm{ml}$ 吸收液（4.2.2）的吸收管（5.4.2）串联之后，与烟气采样装置相接，用冰盐水控制水浴温度为 $0\%$ 左右，以 $1.0\mathrm{L / min}$ 的流量采集废气 $30\mathrm{min}$ 。采样系统示意图见图1。

# 6.2 样品保存

采样后的样品应尽快分析，密封样品于 $4\%$ 下可保存 $7\mathrm{d}$ 。

同时作现场空白试验。

# 7 分析步骤

# 7.1 色谱条件

# 7.1.1 色谱柱

固定液为 $14\%$ 氰苯基 $86\%$ 二甲基聚硅氧烷毛细管柱： $30\mathrm{m}\times 0.32\mathrm{mm}\times 0.25\mu \mathrm{m}$ （长 $\times$ 内径 $\times$ 膜厚）。

# 7.1.2 气相色谱仪

气相色谱仪条件如下：

a）载气：氢气，流速为 $2.5\mathrm{ml} / \mathrm{min}$
b)气化室温度： $280~\mathrm{^\circ C}$
c）柱温：初温 $150\%$ 、保持 $1\mathrm{min}$ ，以 $15\%$ /min的速率升至 $280\%$ 、保持 $5\mathrm{min}$
d) 进样方式: 不分流进样, 不分流时间 $1 \mathrm{~min}$ ;
e）进样体积： $2\mu \mathrm{l}$
f) 检测器：温度为 $220\,^{\circ}\mathrm{C}$ ，氢气流速为 $10\, \mathrm{ml/min}$ ，空气1流速为 $11\, \mathrm{ml/min}$ ，空气2流速为 $8\, \mathrm{ml/min}$ ，数据采集方式为线性。

# 7.2 校准

# 7.2.1 仪器稳定性检验

仪器各部分温度、气体流量、压力稳定正常，连续二次测试与试样被测成分含量相近的标准样品，其峰面积相对偏差不大于 $10\%$ ，即认为仪器处于稳定状态。

# 7.2.2 标准曲线的绘制

分别取4.2.5系列标准溶液 $2.0\mathrm{ml}$ 于 $4\mathrm{ml}$ 具塞样品瓶中，加入 $35\mu \mathrm{l}$ 衍生化试剂，振荡 $2\mathrm{min}$ 后，放置 $30\mathrm{min}$ 使衍生完全。将衍生后的系列标准溶液分别进样三次、每次进样 $2\mu \mathrm{l}$ ，测定峰面积，绘制峰面积与浓度的标准曲线。氧联双二苯脾的浓度按公式（1）计算：

$$
Y = a + b X \tag {1}
$$

式中： $Y$ 一一氧联双二苯脾的浓度， $\mathrm{ng / ml}$

$a$ ——标准曲线的截距；

$b$ ——标准曲线的斜率；

$X$ ——氧联双二苯胂的峰面积。

# 7.2.3 标准曲线的校正

标准曲线应每天校正一次。选标准曲线中间浓度进行校正，当偏差大于 $10\%$ 时，应重新绘制标准曲线。

# 7.2.4 仪器的检测限

在仪器稳定条件下，配制氧联双二苯肿乙腈溶液（浓度为 $15\mathrm{ng / ml}$ ），以产生二倍噪声峰高所对应的氧联双二苯肿含量计算，本仪器的检测限为 $3\times 10^{-11}\mathrm{g}$ 。

# 7.3.2 定量分析

# 7.3.2.1 分析

用外标法定量分析。

# 7.3.2.2 计算

气体样品中氧联双二苯脾的浓度按公式（2）计算：

$$
c = \frac {Y V _ {1}}{V} \times 1 0 ^ {- 3} \tag {2}
$$

式中： $c$ ——废气中氧联双二苯胂的浓度， $\mathrm{mg / m^3}$

$Y$ ——由公式（1）计算吸收液中氧联双二苯胂的浓度， $\mathrm{ng / ml}$ ；

$V_{1}$ ——样品溶液的体积， $\mathrm{ml}$

$V$ ——标准状态下干废气的采样体积, $\mathrm{L}$ 。

# 8 结果表示

# 8.1 定性结果

根据色谱图中氧联双二苯肿衍生物的保留时间来确定被测样品中的氧联双二苯肿成分。试样色谱峰的保留时间与标准样品的保留时间作比较，保留时间窗口为 $\pm 10\mathrm{s}$ 。应在色谱图上标明属于氧联双二苯肿衍生产物的峰。

# 8.2 定量结果

# 8.2.1 浓度的表示

公式（2）中的 $c$ 表示废气中氧联双二苯脾的浓度，以 $\mathrm{mg / m^3}$ 表示。

# 8.2.2 精密度

相对标准偏差不大于 $20\%$ 。

# 8.2.3 准确度

相对误差不大于 $20\%$ 。

# 8.2.4 最低检测浓度

采样体积为 $30\mathrm{L}$ ，废气中氧联双二苯肼的最低检测浓度为 $5\times 10^{-2}\mathrm{mg / m^3}$ 。
