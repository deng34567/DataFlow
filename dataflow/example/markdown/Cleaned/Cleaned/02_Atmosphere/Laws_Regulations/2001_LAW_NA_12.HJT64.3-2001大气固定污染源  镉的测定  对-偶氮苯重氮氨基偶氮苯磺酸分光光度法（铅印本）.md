# 对-偶氮苯重氮氨基偶氮苯磺酸分光光度法

# 1 主题内容与适用范围

1.1 本标准适用于大气固定污染源有组织和无组织排放中镉及其化合物的测定。
1.2 最低检出浓度

采气体积为 $2\mathrm{m}^3$ ，定容体积为 $25.0\mathrm{ml}$ ，使用光程 $10\mathrm{mm}$ 比色皿，本方法最低检出浓度为 $1.0\times 10^{-4}\mathrm{mg / m^3}$ 。

# 3 定义

本标准系指大气固定污染源中的镉，经滤筒或滤膜采集的颗粒物，经硝酸-高氯酸体系溶出的镉及其化合物。

# 4 原理

将采集样品后的滤膜或滤筒用硝酸-高氯酸消解制成样品溶液。在 $\mathrm{pH}9.5\sim 11.5$ 的弱碱性溶液中，非离子表面活性剂存在下，镉离子与对-偶氮苯重氮氨基偶氮苯磺酸（缩写ADAAS，结构式： $\mathrm{HOS}_{3}-\mathrm{O}-\mathrm{N}=\mathrm{N}-\mathrm{O}-\mathrm{N}=\mathrm{N}-\mathrm{NH}-\mathrm{O}-\mathrm{N}=\mathrm{N}-\mathrm{O}-\mathrm{O}$ ）作用生成稳定的红色络合物。于波长 $532\mathrm{nm}$ 处有最大吸光度。

# 5 试剂

除非另有说明，分析所用试剂均为符合国家标准的分析纯；水为蒸馏水或同等纯度的水。

5.1 硝酸 $\left(\mathrm{HNO}_{3}\right)$ ， $\rho_{20} = 1.42$ 。
5.2 高氯酸 $\mathrm{(HClO_4)}$ ， $\rho_{20} = 1.68$

5.4 硝酸溶液， $1 + 1$
5.5 甲醛（HCHO）， $36\% \sim 38\%$ 。
5.6 甲醛溶液， $2\% (V / V)$ ：

量取 $2\mathrm{ml}$ 甲醛（5.5）溶于 $98\mathrm{ml}$ 水中。

5.7 二甲基甲酰胺 $\mathrm{(HCON(CH_3)_2)}$ ，简写DMF。
5.8 氯化铵 $\mathrm{(NH_4Cl)}$ ，固体。
5.9 氢氧化钠（NaOH）溶液， $c_{(\mathrm{NaOH})} = 1.0\mathrm{mol / L}$
5.10 氢氧化钠（NaOH）溶液， $c_{(\mathrm{NaOH})} = 0.1\mathrm{mol / L}$

5.11 ADAAS显色剂， $0.04\% (m / V)$

称取 $80\mathrm{mg}$ 纯化的对-偶氮苯重氮氨基偶氮苯磺酸，溶于 $80\mathrm{mlDMF}$ （5.7）中，加入 $120\mathrm{ml}$ 水，滴加 $2\sim 3$ 滴氢氧化钠（5.9），摇匀。

5.12 曲力通X-100溶液， $2\%$ $(V / V)$

量取 $2.0\mathrm{ml}$ [Triton X-100, $(\mathrm{CH}_3)_3\mathrm{CCH}_2\mathrm{C}$ $(\mathrm{CH}_3)_2\mathrm{-C}_6\mathrm{H}_4$ $(\mathrm{OCH}_2\mathrm{CH}_2)_n\mathrm{OH}$ , $n\approx 10)$ ], 溶于 $98\mathrm{ml}$ 水中, (可加热使其溶解)。

5.13 氨水一氯化铵缓冲溶液， $\mathrm{pH} = 10.0$

称取 $20\mathrm{g}$ 氯化铵（5.8)，溶于适量水中，加入 $120\mathrm{ml}$ 氨水（5.3)，加水稀释至约 $500\mathrm{ml}$ ，摇匀。用氯化铵和氨水调至溶液 $\mathrm{pH} = 10.0$ ，用 $\mathrm{pH}$ 计校准。

5.14 氰化钾-酒石酸钾钠混合掩蔽剂：

称取 $0.15\mathrm{g}$ 氰化钾（KCN）溶于 $100\mathrm{ml}$ 氢氧化钠溶液（5.10）中，并加入 $5\mathrm{g}$ 酒石酸钾钠 $\mathrm{(C_4H_4O_6KNa\cdot 4H_2O)}$ 。此溶液含 $0.15\%$ 氰化钾和 $5\%$ 酒石酸钾钠。

注：氰化钾为剧毒物质，使用和保存必须严格按规定要求。

5.15 镉标准贮备液：

称取 $0.5000\mathrm{g}$ 金属镉 $(99.99\%)$ ，置于 $100\mathrm{ml}$ 烧杯中，用硝酸溶液（5.4） $10\mathrm{ml}$ 加热溶解，移入 $500\mathrm{ml}$ 容量瓶中以水定容。此溶液每毫升含 $1.000\mathrm{mg}$ 镉。

5.16 镉标准中间液：

吸取镉标准贮备液（5.15） $10.00\mathrm{ml}$ 移入 $100\mathrm{ml}$ 容量瓶，加入 $1\mathrm{ml}$ 硝酸（5.1）后用水定容至标线，此溶液每毫升含 $100.0\mu \mathrm{g}$ 镉。

5.17 镉标准使用液：

吸取镉标准中间液（5.16） $2.00\mathrm{ml}$ 移入 $100\mathrm{ml}$ 容量瓶，用水定容至标线，此溶液每毫升含2.00 $\mu \mathrm{g}$ 镉，使用时当天配制。

# 8 步骤

# 8.1 校准曲线绘制

取6支 $25\mathrm{ml}$ 具塞比色管，依次加入镉标准使用溶液（5.17）0、1.00、2.00、3.00、4.00、5.00ml，然后向各管中分别加入 $2.0\mathrm{ml}$ 氨缓冲液（5.13）， $0.5\mathrm{ml}$ 曲力通X—100溶液（5.12）， $1.0\mathrm{ml}$ 氰化钾-酒石酸钾钠混合掩蔽剂（5.14），用洗瓶冲洗一下比色管内壁，摇匀（不必用力摇晃，以免生成大量泡沫）。片刻后，加入 $1.5\mathrm{ml}$ ADAAS显色剂（5.11）， $1.0\mathrm{ml}$ 甲醛溶液（5.6），加水至标线，摇匀。放置 $10\mathrm{min}$ 后，于 $532\mathrm{nm}$ 波长处，用 $10\mathrm{mm}$ 比色皿，以试剂空白溶液为参比测定吸光度。以吸光度对镉含量（ $\mu \mathrm{g}$ ）回归，得回归方程，绘制标准曲线

# 8.2 样品测定

# 8.2.1 样品处理

将采样后的滤膜或滤筒剪碎（切勿抖落尘粒），放入 $150\mathrm{ml}$ 锥形瓶中，罩上玻璃漏斗（6.1），加30ml硝酸溶液（5.4），5ml高氯酸（5.2）（对于滤筒样品可加大硝酸溶液用量，以浸没样品为适宜）。将锥形瓶置于电热板上加热（如果样品消解不完全，可在刚开始冒高氯酸烟时将锥形瓶取下，趁热滴加硝酸（5.1）数滴，继续加热至样品颜色变浅）。高氯酸冒白色浓烟至样品近干时，取下冷却。

冷却后的锥形瓶中加入少量水，于电热板上继续加热，溶解盐类。

将放冷后的样品溶液过滤，用水分数次洗涤残渣（可用套硅橡皮头的玻璃棒研磨瓶壁，以减少损失）。

# 8.2.2 干扰

对于 $5\mu \mathrm{g}$ 镉，下列含量的共存离子不影响测定： $200\mu \mathrm{g}$ 汞、 $1500\mu \mathrm{g}$ 铜、 $500\mu \mathrm{g}$ 铁、 $500\mu \mathrm{g}$ 铝、 $200\mu \mathrm{g}$ 钴、 $200\mu \mathrm{g}$ 镍。

铁、铝含量大于 $500\mu \mathrm{g}$ 的样品，采用碘化钠介质，MIBK萃取分离，萃取操作步骤参见附录A。

# 8.2.3 显色测定

将上述过滤液收集于 $150\mathrm{ml}$ 锥形瓶中，蒸发至近干，移入 $25\mathrm{ml}$ 比色管中（溶液体积不宜超过18ml)，加入NaOH溶液（5.10)，调节 $\mathrm{pH}$ 至近中性；加入 $2.0\mathrm{ml}$ 氨缓冲溶液（5.13)， $0.5\mathrm{ml}$ 曲力通X—100溶液(5.12)， $1.0\mathrm{ml}$ 氰化钾-酒石酸钾钠混合掩蔽剂(5.14)，缓慢摇匀后，再加 $1.5\mathrm{ml}$ ADAAS显色剂（5.11)， $1.0\mathrm{ml}$ 甲醛溶液(5.6)，加水至标线，摇匀。放置 $10\mathrm{min}$ 后，于 $532\mathrm{nm}$ 波长处，用 $10\mathrm{mm}$ 比色皿，以试剂空白作参比，测定吸光度。

# 8.3 空白试验

用同批号的滤膜或滤筒，按样品消解步骤和条件进行处理，并显色测定空白值。

# 9 计算

# 9.1 计算污染物排放浓度

根据所测吸光度，由回归方程求得试样中镉含量，由下式计算有组织排放或无组织排放大气污染源中镉含量 $(\mathrm{mg} / \mathrm{m}^3)$ 。

$$
c (\mathrm {C d}, \mathrm {m g} / \mathrm {m} ^ {3}) = \frac {(W - W _ {0})}{V _ {\mathrm {n d}} \times 1 0 0 0} \times \frac {S _ {\mathrm {t}}}{S _ {\mathrm {a}}}
$$

式中： $W$ —样品溶液中镉含量， $\mu \mathrm{g}$

$W_{0}$ ——空白试验溶液中镉含量， $\mu \mathrm{g}$

$S_{\mathrm{t}}$ ——样品滤膜总面积， $\mathrm{cm}^2$

$S_{\mathrm{a}}$ ——测定时所取样品滤膜面积， $\mathrm{cm}^2$

$V_{\mathrm{nd}}$ ——标准状态下的采气体积， $\mathrm{m}^3$ 。

注：对滤筒样品， $S_{\mathrm{t}} = S_{\mathrm{a}}$

# 9.2 计算污染物排放速率

根据GB16297—1996《大气污染物综合排放标准》附录A和附录B的有关条文计算污染物排放

速率。

选取标准参考物GSD—12（镉含量推荐值为 $4.0\mathrm{mg / kg}$ ）作为统一样品，经6个实验室分析：

重复性相对标准偏差： $3.4\%$

11 说明

11.1 氰化钾为剧毒物质，操作应十分小心。加入氰化钾-酒石酸钾钠掩蔽剂前，应保持待测体系为碱性，以免产生HCN气体，毒害人体。
11.2 样品消解时，高氯酸烟应尽量赶尽，否则，过滤时易破损滤纸，直接显色时还影响溶液酸度。
11.3 对于用有机滤膜采集的样品或有机物含量高的样品，消解时应先用硝酸氧化有机物，然后加入高氯酸，以避免爆炸。
11.4 氨-氯化铵缓冲体系可掩蔽锌、铝、锰、铅、铁等离子，氰化钾可掩蔽汞、铜、银、钴、镍等离子，酒石酸钾钠可掩蔽钙、镁、铁、铝等离子，防止在碱性介质中形成氢氧化物沉淀。但当分析含铁 $5\mathrm{mg}$ 以上样品（如土壤样品）时，上述掩蔽体系无法防止氢氧化物沉淀。类似情况，可采用萃取分离方法消除干扰。

# A4 测定步骤

# A4.1 萃取分离

将前述消解后的样品溶液（8.2）过滤收集于 $125\mathrm{ml}$ 分液漏斗中，加入 $3.0\mathrm{ml}$ 碘化钠-抗坏血酸溶液（A2.3）， $10\mathrm{ml}$ MIBK（A2.1）。萃取 $2\sim 3\mathrm{min}$ ，放掉水相。再用 $5\mathrm{ml}$ 洗萃液（A2.4）振摇数次，静置，弃去水相。然后，加入 $6.0\mathrm{ml}$ 反萃液（A2.5），反萃 $2\mathrm{min}$ ，水相放入 $25\mathrm{ml}$ 比色管中，加少量水洗涤有机相 $2\sim 3$ 次，合并放入比色管（溶液体积不宜超过 $20\mathrm{ml}$ ）。

# A4.2 比色测定

向比色管中（A4.1）分别加入 $0.5\mathrm{ml}$ 曲力通X—100溶液（5.12）， $1.5\mathrm{ml}$ ADAAS显色剂(5.11)， $1.0\mathrm{ml}$ 甲醛溶液（5.6)，加水至标线定容，摇匀后，放置 $10\mathrm{min}$ 。于 $532\mathrm{nm}$ 波长处， $10\mathrm{mm}$ 比色皿，以相同操作的试剂空白作参比测定溶液吸光度，并由校准曲线（8.1）查得样品溶液中镉的含量。
