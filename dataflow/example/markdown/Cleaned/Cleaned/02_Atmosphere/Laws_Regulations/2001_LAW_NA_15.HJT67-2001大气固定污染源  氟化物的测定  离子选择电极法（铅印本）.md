# 离子选择电极法

# 1 范围

# 1.1 适用范围

本标准适用于大气固定污染源有组织排放中氟化物的测定。不能测定碳氟化物，如氟利昂。

# 1.2 测定范围

当采样体积为 $150\mathrm{L}$ 时，检出限为 $6\times 10^{-2}\mathrm{mg / m^3}$ ；测定范围为 $1\sim 1000\mathrm{mg / m^3}$ 。

下列标准所包含的条文，通过在本标准中引用而构成为本标准的正式条文。本标准出版时，所示版本均为有效。所有标准都会被修订，使用本标准的各方，应探讨使用下列标准最新版本的可能性。

GB/T16157—1996 固定污染源排气中颗粒物的测定和气态污染物采样方法。

# 3 定义

氟化物系指气态氟与尘氟的总和。本标准中的气态氟用氢氧化钠溶液吸收，尘氟指溶于盐酸溶液的与颗粒物共存的氟化物。

# 4 原理

使用滤筒、氢氧化钠溶液为吸收液采集尘氟和气态氟，滤筒捕集尘氟和部分气态氟，用盐酸溶液浸溶后制备成试样，用氟离子选择电极测定；当溶液的总离子强度为定值而且足够大时，其电极电位与溶液中氟离子活度的对数成线性关系。

# 7 样品的采集和保存

# 7.1 样品的采集

污染源中尘氟和气态氟共存时，采用烟尘采样方法进行等速采样，在采样管的出口串联三个装有 $75\mathrm{ml}$ 吸收液的大型冲击式吸收瓶，分别捕集尘氟和气态氟。

若污染源中只存在气态氟时，可采用烟气采样方法，在采样管出口串联两个装有 $50\mathrm{ml}$ 吸收液的多孔玻板吸收瓶，以 $0.5\sim 2.0\mathrm{L / min}$ 的流速采集 $5\sim 20\mathrm{min}$ 。

采样管与吸收瓶之间的连接管，选用聚四氟乙烯管，并应尽量短。

注：连接管也可使用聚乙烯塑料管和橡胶管。

采样点数目，采样点位设置及操作步骤，按GB/T16157—1996《固定污染源排气中颗粒物的测定和气态污染物采样方法》有关规定进行。采样频次和时间，按GB16297—1996《大气污染物综合排放标准》有关规定进行。

# 7.2 样品的保存

采样结束后，将滤筒取出，编号后放入干燥洁净的器皿中，并按采样要求，做好记录。吸收瓶中的样品全部转移至聚乙烯瓶中，并用少量水洗涤三次吸收瓶，洗涤液并入聚乙烯瓶中。编号做好记录，采样管与连接管先用 $50\mathrm{ml}$ 吸收液（5.2.1）洗涤，再用 $400\mathrm{ml}$ 水冲洗，全部并入聚乙烯瓶中，编号做好记录。

样品常温下可保存一周。

在聚乙烯烧杯中各放入一根搅拌子，加入三滴溴甲酚绿指示剂（5.5），用盐酸溶液（5.1.2）调节 $\mathrm{pH}$ 值，使溶液刚刚变为蓝绿色为止（此时溶液的 $\mathrm{pH}$ 值为5.5左右），加TISAB溶液 $10.00\mathrm{ml}$ ，加水使总体积为 $40.0\mathrm{ml}$ 。置于磁力搅拌器上，插入按要求已清洗好的氟离子选择电极及甘汞电极；从低到高浓度依次测定。

每个样品搅拌数分钟，待读数稳定后（即每分钟电极电位变化小于 $0.1\mathrm{mV}$ ），停止搅拌，静置1分钟读取毫伏值。

用半对数坐标纸，以等距坐标表示毫伏值，对数坐标表示氟含量 $(\mu \mathrm{g})$ ，绘制校准曲线。或作直线回归，取得回归方程。

电极的实际斜率：温度在 $20\sim 25\mathrm{C}$ 之间，氟离子浓度每改变10倍，电极电位变化 $58\pm 2\mathrm{mV}$ 。

# 9 计算

# 9.1 浓度的计算

# 9.1.1 气态氟和尘氟浓度

气态氟浓度： $c$ （F， $\mathrm{mg / m^3}$ ） $= W\times \frac{V_{\mathrm{t}}}{V_{\mathrm{a}}}\times \frac{1}{V_{\mathrm{nd}}}$

空白滤筒氟含量： $d(\mu \mathrm{g}) = W\times \frac{V_{\mathrm{t}}}{V_{\mathrm{a}}}$

尘氟浓度： $c(\mathrm{F}_{\text{尘}}^{-},\mathrm{mg / m}^{3}) = \left(W\times \frac{V_{\mathrm{t}}}{V_{\mathrm{a}}} -d\right)\times \frac{1}{V_{\mathrm{nd}}}$

式中： $W$ ——测定时所取样品溶液中氟含量， $\mu \mathrm{g}$

$V_{\mathrm{t}}$ ——样品溶液总体积，ml；

$V_{\mathrm{a}}$ ——测定时所取样品溶液体积，ml；

$V_{\mathrm{nd}}$ ——标准状态下的采样体积，L。

$V_{\mathrm{nd}}$ 的换算参照GB/T16157—1996《固定污染源排气中颗粒物测定和气态污染源采样方法》中的第10部分（采样体积计算）；

$d$ ——空白滤筒的氟含量， $\mu \mathrm{g}$ 。

# 9.1.2 总氟浓度：

$$
c _ {\text {总}} = c _ {\text {气}} + c _ {\text {尘}}
$$
