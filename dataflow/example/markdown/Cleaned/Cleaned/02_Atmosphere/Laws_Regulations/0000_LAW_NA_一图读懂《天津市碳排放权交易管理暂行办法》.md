# 第一章总则

# 第一条

![](images/0bd014f1d49134e85491b6ea4087746c1335ba768a9cb575721cb96e447d8c9a.jpg)

为规范本市碳排放权交易及相关活动，加强对温室气体排放的控制，促进经济社会绿色低碳发展，推进生态文明建设，根据《碳排放权交易管理暂行条例》、《天津市碳达峰碳中和促进条例》等有关规定，结合本市实际，制定本办法。

# 第二条

本办法适用于本市行政区域内开展的地方碳排放权交易及相关活动。

纳入全国碳排放权交易市场的碳排放权交易及相关活动、纳入全国温室气体自愿减排交易市场的温室气体自愿减排交易及相关活动，按照国家有关规定执行。

![](images/47bd9c2b0ae78de4a3cb768668fcf141cfdbb4ab67c486bf5fff3534357d7e81.jpg)

# 第三条

本市碳排放权交易及相关活动的管理，坚持政府引导与市场调节相结合，遵循公开、公平、公正和诚信的原则。
