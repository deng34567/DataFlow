# 1 范围

本标准规定了测定废气中光气的高效液相色谱法。

本标准适用于销毁遗弃化学武器全过程中废气中光气含量的测定。

本方法的最低检测浓度为 $0.003\mathrm{mg / m^3}$ 。

# 3 原理

用苯胺吸收液吸收光气，光气与苯胺反应生成1,3-二苯基脲，然后将苯胺吸收液注入高效液相色谱仪，经色谱柱将1,3-二苯基脲与可能干扰光气测定的共存物分离；然后用紫外检测器在波长 $257\mathrm{nm}$ 处测定峰高或峰面积。用外标法定量。

$$
\mathrm {C O C l} _ {2} + 2 \left\{\left\langle \mathrm {C} \right\rangle \mathrm {N H} _ {2} \right\rangle \longrightarrow \left\{\left\langle \mathrm {C} \right\rangle \mathrm {N H} \mathrm {C} \right\rangle \mathrm {N H} \left\langle \mathrm {C} \right\rangle + 2 \mathrm {H C l}
$$

# 4 试剂和材料

4.1 苯胺：分析纯。
4.2 甲醇：HPLC级。
4.3 水：二次蒸馏水。
4.4 1,3-二苯基脲：色谱纯。
4.5 吸收液：取新蒸馏的苯胺 $0.25\mathrm{g}$ ，溶解于 $1000\mathrm{ml}$ 水（4.3）中， $4\%$ 避光保存。
4.6 1,3-二苯基脲标准储备液（ $100\mu \mathrm{g} / \mathrm{ml}$ ）：称取1,3-二苯基脲（4.4） $0.0215\mathrm{g}$ ，用甲醇（4.2）溶解，移入 $100\mathrm{ml}$ 容量瓶，用甲醇稀释定容至标线。此标准储备液相当于每 $1\mathrm{ml}$ 含 $100\mu \mathrm{g}$ 光气。置冰箱中 $4\%$ 保存。
4.7 1,3-二苯基脲标准溶液 $(1.0\mu \mathrm{g} / \mathrm{ml})$ ：移取1，3-二苯基脲标准储备液（4.6） $0.50\mathrm{ml}$ 于 $50\mathrm{ml}$

容量瓶中，用吸收液（4.5）稀释至标线。此标准溶液相当于每 $1\mathrm{ml}$ 含 $1.00\mu \mathrm{g}$ 光气。现用现配。

# 5 仪器

5.1 多孔玻板吸收管： $50~\mathrm{ml}$
5.2 烟气采样器：流量范围 $0\sim 1.5\mathrm{L / min}$
5.3 高效液相色谱仪：具紫外检测器。
5.4 色谱柱：十八烷基硅烷键合硅胶（ODSC18）柱， $150\mathrm{mm}\times \phi 6\mathrm{mm}$
5.5 微量注射器： $10\mu \mathrm{l}$

# 6样品

# 6.1 样品采集方法

6.1.1 试样采集方法：串联两个各装有 $10\mathrm{ml}$ 吸收溶液（4.5）的多孔玻板吸收管（5.1），以 $1.0\mathrm{L}/\mathrm{min}$ 的采样速度采气 $20\mathrm{L}$ 。记录现场温度和大气压。

采样方法：按照GB/T16157—1996中第9章执行。

6.1.2 现场空白采集方法：串联两个各装有 $10\mathrm{ml}$ 吸收溶液（4.5）的多孔玻板吸收管（5.1）带至采样现场，敞开放置，但不抽气。

# 6.2 样品的保存

采样结束后，应立即分析；如无法尽快分析，则将吸收管封闭， $4\mathrm{^\circ C}$ 避光保存， $48\mathrm{h}$ 内分析完毕。

# 6.3 样品处理方法

将吸收管中的吸收液移入 $25\mathrm{ml}$ 的比色管中，用少量甲醇（4.2）洗涤 $2\sim 3$ 次，洗涤液并入比色管中，最后用吸收液（4.5）定容，待测定。

# 7 分析测定步骤

# 7.1 色谱条件

7.1.1 柱温：室温。
7.1.2 流动相流量： $1.0\mathrm{ml / min}$
7.1.3 流动相（体积分数）： $70\%$ 甲醇 $+30\%$ 水。
7.1.4 检测器：紫外检测器，测定波长 $257~\mathrm{nm}$

# 7.2 标准曲线的绘制

7.2.1 标准工作液的配制：用标准溶液（4.7）配制五个或五个以上浓度的标准工作溶液。标准工作液浓度的确定应参照废气污染物样品浓度范围，以样品浓度在曲线中段为宜。 $4^{\circ}\mathrm{C}$ 避光保存。
7.2.2 标准曲线绘制：以1,3-二苯基脲色谱峰面积 $A$ （或峰高 $H$ ）为纵坐标，相应浓度 $c(\mu \mathrm{g / ml})$ 为横坐标，绘制标准曲线。
7.2.3 标准曲线校正：每天用浓度居中的标准工作液（其检测浓度必须大于10倍最低检出浓度）做常规校正，组分响应值变化应在 $15\%$ 之内，若偏差过大，需重新绘制标准曲线。

# 7.3.4.2 计算：

以扣除空白后实测样品中1,3-二苯基脲峰面积（或峰高），从标准曲线上读取样品吸收液中光气的浓度。

废气中光气的浓度按公式（1）计算：

$$
c = \frac {c _ {t} V _ {t}}{V _ {\mathrm {s}}} \tag {1}
$$

式中： $c$ ——废气中光气浓度， $\mathrm{mg / m^3}$

$c_{t}$ ——样品吸收溶液中光气浓度， $\mu \mathrm{g / ml}$

$V_{t}$ ——吸收溶液体积， $\mathrm{ml}$

$V_{s}$ ——标准状态下的采气体积， $\mathrm{L}$ 。
