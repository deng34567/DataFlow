# 固定污染源排气中光气的测定苯胺紫外分光光度法

HJ/T31—1999

Stationary source emission-Determination of phosgene Aniline ultraviolet spectrophotometric method

# 1 适用范围

1.1 本标准适用于固定污染源有组织排放和无组织排放的光气测定。
1.2 在无组织排放样品分析中，当采样体积为 $60\mathrm{L}$ 时，光气的检出限为 $0.02\mathrm{mg / m^3}$ ，定量测定的浓度范围为 $0.06\sim 1.0\mathrm{mg / m^3}$ 。

在有组织排放样品分析中，当采样体积为 $15\mathrm{L}$ 时，光气的检出限为 $0.4\mathrm{mg / m^3}$ ，定量测定的浓度范围为 $1.2\sim 20\mathrm{mg / m^3}$ 。

1.3 在本标准规定的条件下，氯气浓度大于 $1600\mathrm{mg / m^3}$ 时对光气测定有干扰。

# 2 方法原理

含光气（ $\mathrm{COCl}_2$ ）的气体先经装有硫代硫酸钠和无水碳酸钠的双联玻璃球，以除去氯、二氧化氮、氨等干扰气，而后被苯胺溶液吸收，生成1，3-二苯基脲，用溶剂在酸性条件下萃取，在波长 $257\mathrm{nm}$ 处测定吸光度，其值与光气含量成正比。反应式如下：

$$
\left\langle \right\rangle - \mathrm {N H} _ {2} + \mathrm {C O C l} _ {2} \longrightarrow \left\langle \right\rangle - \mathrm {N} - \mathrm {C} - \mathrm {N} - \left\langle \right\rangle + 2 \left\langle \right\rangle - \mathrm {N H} _ {2} \cdot \mathrm {H C l}
$$

# 4 试剂与材料

除非另有说明，分析中均使用符合国家标准的分析纯试剂和去离子水。

4.1 硫酸： $\rho = 1.84\mathrm{g / ml}$
4.2 甲醇。
4.3 硫代硫酸钠。
4.4 无水碳酸钠。
4.5 苯胺

常压蒸馏，收集 $183\sim 184^{\circ}\mathrm{C}$ 的馏分。

4.6 正己烷

在 $1000\mathrm{ml}$ 分液漏斗中，按正己烷与浓硫酸（4.1） $50 + 3(V / V)$ 的比例，将浓硫酸（4.1）缓慢加

入试剂中，上塞，倒置分液漏斗，用力震荡数分钟（注意经常放气），静置分层后弃去浓硫酸。再重复上述步骤，直至弃去的浓硫酸为浅色止。酸洗结束后再用去离子水水洗，步骤同上，直至弃去的去离子水 $\mathrm{pH}$ 为7左右，然后进行蒸馏，收集 $68\sim 69^{\circ}\mathrm{C}$ 的馏分。

# 4.7 二氯甲烷

以同（4.6）的步骤进行酸洗和水洗，然后进行常压蒸馏，收集 $39\sim 40^{\circ}\mathrm{C}$ 的馏分。

# 4.8 异戊醇

常压蒸馏，收集 $129\sim 130^{\circ}\mathrm{C}$ 的馏分。

# 4.9 混合萃取剂

取正己烷（4.6）、二氯甲烷（4.7）、异戊醇（4.8）按 $1 + 1 + 0.2(V / V / V)$ 比例混合配制。

# 4.10 硫酸溶液： $1 + 1$

用量筒量取 $250\mathrm{ml}$ 浓硫酸（4.1），缓慢地倒入（边搅拌） $250\mathrm{ml}$ 水中。

# 4.11 吸收液

称取新蒸馏的苯胺（4.5） $0.25\mathrm{g}$ ，溶解于 $1000\mathrm{ml}$ 水中，此溶液置于冰箱中可保存一个月。

# 4.12 1,3-二苯基脲标准储备液：相当于 $100.0\mu \mathrm{g}$ （光气） $/ \mathrm{ml}$ 。

称取 $127.5\mathrm{mg}1,3-$ 二苯基脲，溶于甲醇（4.2)，移入 $500~\mathrm{ml}$ 容量瓶，用甲醇稀释至刻度。

# 4.13 1,3-二苯基脲标准使用液；相当于 $1.00\mu \mathrm{g}$ （光气） $/ \mathrm{ml}$ 。

临用时，吸取标准储备液（4.12） $1.00\mathrm{ml}$ ，于 $100\mathrm{ml}$ 容量瓶中，用吸收液稀释至刻度。

# 8 计算和结果表示

8.1 样品中光气含量 $c$ ，以 $\mathrm{mg / m^3}$ 表示，由式（1）计算：

$$
c = \frac {W \cdot V _ {\mathrm {s}}}{V _ {\mathrm {n d}} \cdot V _ {\mathrm {o}}} \tag {1}
$$

式中： $W$ ——测定时所分取的样品溶液中光气含量 $(\mu \mathrm{g})$ ，此值可由校准曲线上查得，也可由回归方程求得；

$V_{s}$ ——样品溶液总体积，ml；

$V_{\circ}$ ——分析时所分取的样品溶液体积，ml；

$V_{\mathrm{nd}}$ ——所采气体换算成标准状态下的干采气体积， $\mathrm{L}$ 。

按GB16157—1996中10.1或10.2计算 $V_{\mathrm{nd}}$ 值。

8.2 光气有组织排放的“排放浓度”计算

按GB16157—1996中11.1.2或11.1.4计算光气的“排放浓度”。

8.3 光气有组织排放的“排放速率 $(\mathrm{kg / h})$ ”计算

按GB16157—1996中11.4计算光气的“排放速率”。

8.4 光气的“无组织排放监控浓度值”计算

8.4.1 按（2）式计算某一个无组织排放监控点的光气平均浓度

$$
c = \frac {\sum_ {i = 1} ^ {n} c _ {i}}{n} \tag {2}
$$

式中： $c$ ——一个无组织排放监控点的光气平均浓度；

$c_{i}$ ——一个样品中的光气浓度；

$n$ ——一个无组织排放监控点采集的样品数目。

8.4.2 “无组织排放监控浓度值”的计算

按GB16297—1996附录C中C2.3计光气的“无组织排放监控浓度值”。

# 9 精密度和准确度

# 9.1 精密度

五个实验室分别测定浓度为 $4.00\mathrm{mg / L}$ 的统一样品，测定的重复性标准偏差为 $0.096\mathrm{mg / L}$ ，重复性相对标准偏差为 $2.4\%$ ，重复性为 $0.27\mathrm{mg / L}$ ；再现性标准偏差为 $0.11\mathrm{mg / L}$ ，再现性相对标准偏差为 $2.8\%$ ，再现性为 $0.31\mathrm{mg / L}$ 。

# 9.2 准确度

五个实验室分别测定浓度为 $4.00\mathrm{mg / L}$ 的统一样品，各实验室测定均值的相对误差于 $0.5\% \sim 2.2\%$ 之间，平均相对误差为 $1.5\%$ 。

5. 五个实验室分别对若干个无组织排放样品于实验室进行加标回收率测定，得加标回收率分布于

94. $0\% \sim 104\%$ 之间；有组织排放样品的加标回收率分布于 $100\% \sim 102\%$ 之间。

# 10 说明

10.1 在现场采样时，如排气管道处于正压且浓度较高时，应于采样孔装防喷阀门，操作人员应在上风向并佩戴防毒口罩操作，严防光气中毒。
10.2 当氯气浓度大于 $1600\mathrm{mg / m^3}$ 时，对光气测定产生正干扰，为消除干扰，应于吸收管（瓶）前装一双联玻璃球管（5.6），两球内分别加入 $10\mathrm{g}$ 硫代硫酸钠（4.3）和 $5\mathrm{g}$ 无水碳酸钠（4.4）。玻璃球管两端分别用无碱玻璃棉（5.7）塞住，中间用无碱玻璃棉（5.7）隔开。
10.3 分析步骤中萃取时，为使有机相与水相分离更完全，可在振摇后将上层有机相移入 $10\mathrm{ml}$ 离心管中，以 $2000\mathrm{r / min}$ 离心分离 $2\mathrm{min}$ 后取澄清液比色。
