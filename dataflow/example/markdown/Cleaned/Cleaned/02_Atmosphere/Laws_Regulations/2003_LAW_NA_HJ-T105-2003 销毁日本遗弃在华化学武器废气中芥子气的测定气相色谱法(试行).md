# 1 范围

本标准规定了测定废气中芥子气的气相色谱分析方法。

本标准适用于销毁日本遗弃在华化学武器全过程废气中芥子气浓度的测定。

当采集气体体积为 $20\mathrm{L}$ 时，废气中芥子气的最低检出浓度为 $2\times 10^{-4}\mathrm{mg / m^3}$ 。

# 3 方法原理

废气中的芥子气用正庚烷吸收后，用旋转蒸发器浓缩，经气相色谱毛细管柱分离，用脉冲火焰光度检测器（PFPD-S）检测。以保留时间定性，外标法定量。

# 4 试剂和材料

# 4.1 载气和辅助气体

4.1.1 载气：氮气，纯度为 $99.999\%$
4.1.2 燃气：氢气，纯度为 $99.99\%$
4.1.3 助燃气：净化后的空气。

# 4.2 试剂

除非另有说明，在分析中仅使用确认为分析纯的试剂。

4.2.1 标准样品：芥子气，纯度不小于 $98\%$
4.2.2 吸收液：正庚烷 $(\mathrm{C_7H_{16}})$ 。
4.2.3 贮备溶液：称取芥子气约 $20\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ ）于 $100\mathrm{ml}$ 容量瓶中，用正庚烷溶解并稀释至刻度、摇匀。此贮备液密封后在 $4^{\circ}\mathrm{C}$ 时可保存 $30\mathrm{d}$ 。
4.2.4 芥子气系列标准溶液：将芥子气贮备溶液稀释后，配制成浓度分别为30.0、60.0、90.0、120.0、180.0、240.0、360.0 ng/ml芥子气系列标准溶液。

# 5 仪器

# 5.1 色谱仪

气相色谱仪，硫型脉冲火焰光度检测器（PFPD-S）。

# 5.2 进样器

微量进样器：5.0、10.0 $\mu \mathrm{l}$ 。

# 5.3 试样预处理仪器

使用的仪器有：

a）样品瓶： $2\mathrm{ml}$ 或 $4\mathrm{ml}$
b) 振荡器；
e）微量移液器：50.0、100.0、500.0 $\mu \mathrm{l}$
d）带刻度具塞浓缩管： $10\mathrm{ml}$
e）旋转蒸发器。

# 5.4 采样仪器

# 5.4.1 采样装置

具加热采样管的烟气采样装置，流量范围为 $0\sim 2.0\mathrm{L / min}$ 的烟气采样器。

# 5.4.2 吸收管

采用GB/T16157—1996中9.3.4b）大型气泡式吸收管， $25\mathrm{ml}$

# 6样品

# 6.1 样品采集

气体样品的采集按GB/T16157—1996执行。将装入 $10\mathrm{ml}$ 吸收液（4.2.2）的吸收管（5.4.2）连接好后，置于冰盐水浴中，与烟气采样器（5.4.1）相接，以 $1.0\mathrm{L / min}$ 的流量采集废气 $20\mathrm{min}$ 。同时记录采样器流量、流量计前的压力、温度及采样时间和地点。

# 6.2 样品保存

采样后吸收液应尽快分析，密封样品在 $4\%$ 下保存。

# 6.3 试样制备

# 6.3.1 芥子气吸收样品的转移

将吸收管中的吸收液转入 $10\mathrm{ml}$ 浓缩管（5.3d）中，摇匀待用。

# 6.3.2 芥子气吸收样品的浓缩

将装有芥子气吸收样品的浓缩管（6.3.1）与旋转蒸发器连接好，调整水浴温度为 $30\mathrm{^\circ C}$ ，开动水泵，在真空度为 $0.095\mathrm{MPa}$ 左右，将样品浓缩至 $1\mathrm{ml}$ 以下，定容至 $1.0\mathrm{ml}$ ，取浓缩后试样于色谱样品瓶中，进行色谱分析。

# 6.4 空白试验

同时作现场空白试验。

# 7 分析步骤

# 7.1 色谱条件

# 7.1.1 色谱柱

固定液为 $14\%$ 氰丙基苯 $86\%$ 甲基聚硅氧烷或同等极性的毛细管柱： $30\mathrm{m}\times 0.32\mathrm{mm}\times 0.25\mu \mathrm{m}$ （长 $\times$ 内径 $\times$ 膜厚）。

# 7.1.2 气相色谱仪

气相色谱仪条件如下：

a）载气（氮气）流速： $1.3\mathrm{ml} / \mathrm{min}$
b) 气化室温度: $250^{\circ} \mathrm{C}$ ;
c) 柱温: 初温 $70^{\circ} \mathrm{C}$ 、保持 $0.5 \mathrm{~min}$ , 以 $20^{\circ} \mathrm{C} / \mathrm{min}$ 的速率升至 $250^{\circ} \mathrm{C}$ 、保持 $5 \mathrm{~min}$ ;

d) 进样方式: 不分流进样, 不分流时间 $0.75 \mathrm{~min}$ ;
e）进样体积： $2\mu \mathrm{l}$
f) PFPD 检测器 S 模式;
g）检测器温度： $220~\mathrm{^\circ C}$
h）氢气流速： $11.5\mathrm{ml / min}$
i) 空气1流速： $10\mathrm{ml} / \mathrm{min}$
j) 空气2流速： $10\mathrm{ml / min}$
k) 数据采集方式：线性。

# 7.2 校准

# 7.2.1 仪器稳定性检验

待仪器各部分温度、气体流量、压力稳定正常时，连续二次测试与试样被测成分含量相近的标准样品，其峰面积相对偏差不大于 $10\%$ ，即认为仪器处于稳定状态。

# 7.2.2 标准曲线绘制

待色谱仪基线稳定后，将系列标准溶液分别进样三次、每次进样 $2\mu \mathrm{l}$ ，进行色谱分析。计算峰面积平均值，将数据回归后，得出回归方程，见公式（1）：

$$
Y = a + b X \tag {1}
$$

式中： $Y$ ——芥子气的浓度， $\mathrm{ng / ml}$

$a$ ——标准曲线的截距；

$b$ ——标准曲线的斜率；

$X$ ——芥子气的峰面积。

# 7.2.3 标准曲线的校正

标准曲线应每天校正一次。选标准曲线中间浓度进行校正，当相对偏差大于 $10\%$ 时，应重新绘制标准曲线。

# 7.2.4 检测限

以产生二倍噪声峰高所对应的芥子气含量作为仪器的检测限，本仪器的检测限为 $1 \times 10^{-11} \mathrm{~g}$ 。

# 7.3.2 定量分析

# 7.3.2.1 分析

将6.1采集的样品按6.3进行处理，用外标法定量。

# 7.3.2.2 计算

废气样品中芥子气的浓度按公式（2）计算：

$$
c = \frac {Y V _ {1}}{V} \times 1 0 ^ {- 3} \tag {2}
$$

式中： $c$ ——废气中芥子气的浓度， $\mathrm{mg / m^3}$

Y——浓缩后吸收液中芥子气的浓度， $\mathrm{ng / ml}$ ；

$V_{1}$ ——浓缩后吸收液体积，ml；

$V$ ——换算成标准状态下的采样体积， $\mathrm{L}$ 。

# 8 结果表示

# 8.1 定性结果

根据色谱图中芥子气的保留时间来确定被测样品中的芥子气成分。试样色谱峰的保留时间与标准样品的保留时间作比较，保留时间窗口为 $\pm 10\mathrm{s}$ 。应在色谱图上标明属于芥子气的峰。

# 8.2 定量结果

# 8.2.1 含量的表示方法

公式（2）中的 $c$ 表示废气中芥子气的浓度，以 $\mathrm{mg / m^3}$ 表示。

# 8.2.2 精密度

相对标准偏差不大于 $20\%$ 。

# 8.2.3 准确度

相对误差不大于 $20\%$ 。

# 8.2.4 最低检出浓度

采样体积为 $20\mathrm{L}$ ，吸收液浓缩样品体积为 $1.0\mathrm{ml}$ 时，废气中芥子气的最低检出浓度为 $2\times 10^{-4}\mathrm{mg / m^3}$ 。
