# 前言

为贯彻《中华人民共和国环境保护法》、《中华人民共和国大气污染防治法》，实施《环境空气质量标准》（GB 3095-2012），规范环境空气中颗粒物（ $\mathrm { P M } _ { 1 0 }$ 和 $\mathrm { P M } _ { 2 . 5 }$ ）连续自动监测系统的性能和质量，制定本标准。

本标准规定了环境空气颗粒物（ $\mathrm { P M } _ { 1 0 }$ 和 $\mathrm { P M } _ { 2 . 5 }$ ）连续自动监测系统（以下简称 $\mathrm { P M } _ { 1 0 }$ 和$\mathrm { P M } _ { 2 . 5 }$ 连续监测系统）的技术要求、性能指标和检测方法。

本标准中附录 A 为规范性附录。

本标准为首次发布。

本标准由环境保护部科技标准司组织制订。

本标准主要起草单位：中国环境监测总站。

本标准环境保护部 2013 年 7 月 30 日批准。

本标准自 2013 年 8 月 1 日起实施。

本标准由环境保护部解释。

# 3 术语和定义

下列术语和定义适用于本标准。

3.1

空气动力学当量直径 aerodynamic diameter

指单位密度（ $\scriptstyle \cdot \rho _ { 0 } = 1 \mathrm { g } / \mathrm { c m } ^ { 3 }$ ）的球体，在静止空气中作低雷诺数运动时，达到与实际粒子相同的最终沉降速度时的直径。

3.2

切割器 particle separate device

指具有将不同粒径粒子分离功能的装置。

3.3

$50 \%$ 切割粒径（ $( D a _ { 5 0 }$ ） $50 \%$ cutpoint diameter

指切割器对颗粒物的捕集效率为 $50 \%$ 时所对应的粒子空气动力学当量直径。

3.4

颗粒物（粒径小于等于 $1 0 \mu \mathrm { m }$ ） particulate matter（ $\mathrm { P M } _ { 1 0 }$ ）

指环境空气中空气动力学当量直径小于等于 $1 0 \mu \mathrm { m }$ 的颗粒物，也称可吸入颗粒物。

3.5

颗粒物（粒径小于等于 $2 . 5 \mu \mathrm { m }$ ） particulate matter（ $\mathrm { P M } _ { 2 . 5 }$ ）

指环境空气中空气动力学当量直径小于等于 $2 . 5 \mu \mathrm { m }$ 的颗粒物，也称细颗粒物。

3.6

标准状态 standard state

指温度为 273K，压力为 $1 0 1 . 3 2 5 \mathrm { { k P a } }$ 时的状态。本标准污染物浓度值均为标准状态下浓度值。

3.7

参比方法 reference method

国家发布的标准方法。

3.8

捕集效率的几何标准偏差 $( \sigma _ { g }$ ） geometric standard deviation of sampling efficiency

切割器对颗粒物的捕集效率有以下两种表述方法：

（1）捕集效率为 $1 6 \%$ 时对应的粒子空气动力学当量直径 $D _ { a I 6 }$ 与捕集效率为 $50 \%$ 时对应的粒子空气动力学当量直径 $D _ { a 5 0 }$ 的比值；
（2）捕集效率为 $50 \%$ 时对应的粒子空气动力学当量直径 $D _ { a 5 0 }$ 与捕集效率为 $84 \%$ 时对应的粒子空气动力学当量直径 $D _ { a 8 4 }$ 的比值。

上述两个比值均应符合 $\sigma _ { g } = 1 . 5 { \pm } 0 . 1 \ : ( \mathrm { P M } _ { 1 0 }$ $\mathrm { \bf { P M } } _ { 1 0 }$ 连续监测系统)、 $\sigma _ { g } = 1 . 2 { \pm } 0 . 1 $ ( $\mathrm { P M } _ { 2 . 5 }$ 连续监测系统)的要求。计算公式见（1）、（2）式：

$$
\sigma_ {g} = \frac {D _ {a 1 6}}{D _ {a 5 0}} \dots \tag {1}
$$

$$
\sigma_ {g} = \frac {D _ {a 5 0}}{D _ {a 8 4}} \dots \tag {2}
$$

式中： $\sigma _ { g }$ --------捕集效率的几何标准偏差， $\%$ ；

$D _ { a l 6 }$ --------切割器对颗粒物的捕集效率为 $1 6 \%$ 时对应的粒子空气动力学当量直径， $\mu \mathrm { m }$ ；

$D _ { a 5 0 } .$ --------切割器对颗粒物的捕集效率为 $50 \%$ 时对应的粒子空气动力学当量直径， $\mu \mathrm { m }$ ；

$D _ { a 8 4 } .$ --------切割器对颗粒物的捕集效率为 $84 \%$ 时对应的粒子空气动力学当量直径， $\mu \mathrm { m }$ 。

3.9

仪器平行性 parallelism of monitors

指每一批次数据结果的均方根。

3.10

气溶胶传输效率 aerosol transport efficiency

指通过采样管进入监测仪的气溶胶量与进入采样管前气溶胶总量的百分比。

# 4 系统组成与原理

# 4.1系统的组成

$\mathrm { P M } _ { 1 0 }$ 和 $\mathrm { P M } _ { 2 . 5 }$ 连续监测系统包括样品采集单元、样品测量单元、数据采集和传输单元以及其他辅助设备。

# 4.1.1 样品采集单元

样品采集单元由采样入口、切割器和采样管等组成。将环境空气颗粒物进行切割分离，并将目标颗粒物输送到样品测量单元。

# 4.1.2 样品测量单元

样品测量单元对采集的环境空气 $\mathrm { P M } _ { 1 0 }$ 或 $\mathrm { P M } _ { 2 . 5 }$ 样品进行测量。

# 4.1.3 数据采集和传输单元

数据采集和传输单元采集、处理和存储监测数据，并能按中心计算机指令传输监测数据和系统工作状态信息。

# 4.1.4 其它辅助设备

其它辅助设备包括安装仪器设备所需要的机柜或平台、安装固定装置、采样泵等。

# 4.2方法原理

$\mathrm { P M } _ { 1 0 }$ 和 $\mathrm { P M } _ { 2 . 5 }$ 连续监测系统所配置监测仪器的测量方法为 $\beta$ 射线吸收法或微量振荡天平法。

# 5 技术要求

# 5.1 外观要求

5.1.1 $\mathrm { P M } _ { 1 0 }$ 和 $\mathrm { P M } _ { 2 . 5 }$ 连续监测系统应有产品铭牌，铭牌上应标有仪器名称、型号、生产单位、出厂编号、制造日期等信息。

5.1.2 $\mathrm { P M } _ { 1 0 }$ 和 $\mathrm { P M } _ { 2 . 5 }$ 连续监测系统外观应完好无损，无明显缺陷，各零、部件连接可靠，各操作键、按钮灵活有效。

# 5.2 工作条件

5.2.1 切割器在以下环境条件中应能正常工作：

环境温度： $( - 3 0 { \sim } 5 0 ) \ \mathrm { ~ \textbar { ~ } C }$ ；

大气压：（80～106）kPa。

5.2.2 监测仪在以下环境条件中应能正常工作：

环境温度：（ $( 1 5 { \sim } 3 5 ) \ \mathrm { ~ \textdegree C }$ ；

相对湿度： $\text{‰}$ ；

大气压：（ $8 0 \sim 1 0 6$ ）kPa。

# 5.2.3 供电电压

AC（ $2 2 0 { \pm } 2 2$ ）V，（ $5 0 { \pm } 1$ ）Hz。

注：低温、低压等特殊环境条件下，仪器设备的配置应满足当地环境条件的使用要求。

# 5.3 安全要求

# 5.3.1 绝缘电阻

在环境温度为（ $1 5 { \sim } 3 5$ ） $\mathrm { { } ^ { \circ } C }$ ，相对湿度 $\text{‰}$ 条件下，监测仪电源端子对地或机壳的绝缘电阻不小于 20MΩ。

# 5.3.2 绝缘强度

在环境温度为（ $( 1 5 \small { \sim } 3 5 ) \ \mathrm { ~ } ^ { \circ } \mathrm { C }$ ，相对湿度 ${ \leqslant } 8 5 \%$ 条件下，监测仪在 1500V（有效值）、 $5 0 \mathrm { H z }$ 正弦波实验电压下持续 1min，不应出现击穿或飞弧现象。

# 5.3.3 $\beta$ 射线源安全

$\mathrm { P M } _ { 1 0 }$ 和 $\mathrm { P M } _ { 2 . 5 }$ 连续监测系统所配置监测仪器的测量方法为 $\beta$ 射线吸收法时，使用的 $\beta$ 射线源应符合放射性安全标准。

# 5.4 功能要求

# 5.4.1 滤膜要求

在规定膜面流速下， $\mathrm { P M } _ { 1 0 }$ 采样滤膜要求对 $0 . 3 \mu \mathrm { m }$ 颗粒物的截留效率 $9 9 9 \%$ ， $\mathrm { P M } _ { 2 . 5 }$ 采样滤膜要求对 $0 . 3 \mu \mathrm { m }$ 颗粒物的截留效率 $3 9 9 . 7 \%$ ；

5.4.2 具备显示和设置系统时间的功能及时间标签功能，数据为设置时段的平均值；
5.4.3 具备记录或输出测量过程中的环境大气压、环境温度、流量和浓度等数据的功能；
5.4.4 具备数字信号输出功能，数据采集和存储记录要求应符合附录 A 的要求；
5.4.5 具备三个月以上数据的存储能力。
5.4.6 仪器掉电后，能自动保存数据；恢复供电后系统可自动启动，恢复运行状态并正常开始工作。

# 6 性能指标

# 6.1 $\mathrm { P M } _ { 1 0 }$ 连续监测系统

# 6.1.1 浓度测量范围

（ $0 \sim 1 0 0 0$ ） $\mu \mathrm { g } / \mathrm { m } ^ { 3 }$ 或（ $0 \sim 1 0 0 0 0$ ） $\mu \mathrm { g } / \mathrm { m } ^ { 3 }$ （可选），最小显示单位 $0 . 1 \mu \mathrm { g } / \mathrm { m } ^ { 3 }$ 。

# 6.1.2 切割性能

$50 \%$ 切割粒径： $D a _ { 5 0 } { = } \ ( 1 0 { \pm } 0 . 5 ) \ \mu \mathrm { m } ;$

捕集效率的几何标准偏差： $\sigma _ { g } { = } 1 . 5 { \pm } 0 . 1$ 。

# 6.1.3 时钟误差

（1）在监测仪正常工作状态下测试 6h，时钟误差 $\pm 2 0 \mathrm { s }$ 。
（2）断开监测仪的供电总计 5 次（各次断电的持续时间分别为 20s、40s、2min、7min和 $2 0 \mathrm { { m i n } }$ ，且在每次断电之间应保证不少于 $1 0 \mathrm { { m i n } }$ 正常电力供应），测试 6h，时钟误差 $\pm 2 \mathrm { { m i n } }$ 。

# 6.1.4 温度测量示值误差

在（ $- 3 0 { \sim } 5 0$ ） $\mathrm { { } ^ { \circ } C }$ 范围内，温度测量示值误差 $\pm 2 ^ { \circ } \mathrm { C }$ 。

# 6.1.5 大气压测量示值误差

在（ $8 0 \sim 1 0 6 $ ）kPa 范围内，大气压测量示值误差≤1kPa。

# 6.1.6 流量稳定性

24h 内，每一次测试时间点流量变化 $\pm 1 0 \%$ 设定流量，24h 平均流量变化 $\pm 5 \%$ 设定流量。

# 6.1.7 校准膜重现性

监测仪校准膜重现性 $\pm 2 \%$ （标称值）。

# 6.1.8 电压变化稳定性

供电电压变化 $\pm 1 0 \%$ ，监测仪标准膜测量值的变化 $\pm 5 \%$ （标称值）。

# 6.1.9 仪器平行性

三台（套）仪器平行性 $\leqslant 1 0 \%$ 。

# 6.1.10参比方法比对测试

使用参比方法进行至少 10 组有效数据的比对测试，测试结果进行线性回归分析，符合以下要求：

斜率： $1 { \pm } 0 . 1 5$ ；

截距：（0±10） $\mu \mathrm { g } / \mathrm { m } ^ { 3 }$ ；

相关系数 ${ \geqslant } 0 . 9 5$ 。

# 6.1.11 有效数据率

连续运行至少 90 天，有效数据率不低于 $8 5 \%$ 。

# $6 . 2 \mathrm { P M } _ { 2 . 5 }$ 连续监测系统

# 6.2.1 浓度测量范围

（ $0 \sim 1 0 0 0$ ） $\mu \mathrm { g } / \mathrm { m } ^ { 3 }$ 或（0～10000） $\mu \mathrm { g } / \mathrm { m } ^ { 3 }$ （可选），最小显示单位 $0 . 1 \mu \mathrm { g } / \mathrm { m } ^ { 3 }$

# 6.2.2 切割性能

$50 \%$ 切割粒径： $D a _ { 5 0 } { = } ~ ( 2 . 5 { \pm } 0 . 2 ) ~ { \mu \mathrm { m } }$

捕集效率的几何标准偏差： $\sigma _ { g } { = } 1 . 2 { \pm } 0 . 1$ 。

# 6.2.3 时钟误差

（1）在监测仪正常工作状态下测试 6h，时钟误差 $\pm 2 0 \mathrm { s }$ 。
（2）断开监测仪的供电总计 5 次（各次断电的持续时间分别为 20s、40s、2min、7min和 $2 0 \mathrm { { m i n } }$ ，且在每次断电之间应保证不少于 $1 0 \mathrm { { m i n } }$ 正常电力供应），测试 6h，时钟误差 $\pm 2 \mathrm { { m i n } }$ 。

# 6.2.4 温度测量示值误差

在（ $- 3 0 { \sim } 5 0$ ） $\mathrm { { } ^ { \circ } C }$ 范围内，温度测量示值误差 $\pm 2 ^ { \circ } \mathrm { C }$ 。

# 6.2.5 大气压测量示值误差

在（ $8 0 \sim 1 0 6$ ）kPa 范围内，大气压测量示值误差≤1kPa。

# 6.2.6 流量测试

在监测仪正常工作条件下，使用标准流量计在采样入口处检测流量，符合以下指标：

（1）平均流量偏差 $\pm 5 \%$ 设定流量；
（2）流量相对标准偏差 $\leqslant 2 \%$ ；
（3）平均流量示值误差 $\leqslant 2 \%$ 。

# 6.2.7 校准膜重现性

监测仪校准膜重现性 $\pm 2 \%$ （标称值）。

# 6.2.8 环境气压、环境温度及供电电压变化的影响

监测仪分别在不同的气压、温度和供电电压等 6种环境条件下进行测试，其流量性能指标应符合 6.2.6 要求。

# 6.2.9 气溶胶传输效率

采样管气溶胶传输效率 $9 \%$ 。

# 6.2.10切割器加载测试

在一个维护周期内，加载后的切割器切割性能指标符合 6.2.2 要求。

# 6.2.11 仪器平行性

三台（套）仪器平行性 $\leqslant 1 5 \%$ 。

# 6.2.12参比方法比对测试

使用参比方法进行至少 23 组有效数据的比对测试，测试结果进行线性回归分析，符合以下要求：

斜率： $1 { \pm } 0 . 1 5$ ；

截距：（0±10） $\mu \mathrm { g } / \mathrm { m } ^ { 3 }$ ；

相关系数 $\geqslant 0 . 9 3$ 。

# 6.2.13 有效数据率

连续运行至少 90 天，有效数据率不低于 $8 5 \%$ 。

# 7 检测方法

# 7.1 $\mathrm { P M } _ { 1 0 }$ 连续监测系统

# 7.1.1 切割性能

切割性能测试可使用分流测试法或静态箱测试法。

# 7.1.1.1 分流测试法

发生单一粒径、均匀、稳定的气溶胶粒子，分别测试待测切割器上游的气溶胶浓度和切割器下游的气溶胶浓度，计算不同粒径气溶胶的捕集效率；拟合捕集效率与粒径的关系得到该切割器的 $50 \%$ 切割粒径和捕集效率的几何标准偏差。

# （1）气溶胶的生成

# （2）分流法测试

a）将待测切割器去除进气部件，通过分流管连接流量适配器、待测切割器和气溶胶检测仪器，切割器应竖直放置。
b）采用单分散固态气溶胶发生器，发生表 1 中空气动力学当量直径 $( 3 \pm 0 . 5$ ） $\mu \mathrm { m }$ 的雾化单分散固态气溶胶颗粒。
c）采用气溶胶检测仪器测量单分散固态气溶胶的粒径，确认其稳定、均匀，符合要求。
d）采用气溶胶检测仪器分别测定切割器上、下游的气溶胶浓度。记录为 $C _ { I I I }$ 和 $C _ { 2 I I }$ 。
e）分别依次生成表 1 中所列的 8 种粒径的雾化单分散固态气溶胶颗粒。重复以上 c）～d）的操作，直至 8 种粒径的雾化单分散固态气溶胶颗粒测试完毕，得到 $C _ { I i j }$ 和 $C _ { 2 i j }$ 。
f）重复 e）的操作三次，按公式（3）计算得到 8 组 24 个捕集效率的数据。

$$
\eta_ {i j} = \frac {C _ {2 i j}}{C _ {1 i j}} \times 100 \% \tag{3}
$$

式中： $\eta _ { i j }$ ------每个粒径点单次测量的捕集效率， $\%$ ；

$C _ { I i j }$ -----切割器上游固态单分散颗粒物单次测量浓度， $/ \mathrm { m } ^ { 3 }$ ；

$C _ { 2 i j }$ -----切割器下游固态单分散颗粒物单次测量浓度， $/ \mathrm { m } ^ { 3 }$ ；

i--------发生的气溶胶粒径点（ $i { = } 1 { \sim } 8 )$ ）；

j--------每个粒径点测量的次数 $\left( j { = } 1 { \sim } 3 \right)$ ）。

g）按公式（4）分别计算得到 8 个粒径点捕集效率的平均值。

$$

# A.2 数据记录要求

A.2.1 系统应至少能显示记录颗粒物的标况浓度、工况浓度、工况流量，标况累积体积以及环境温度、环境大气压等实时数据。

A.2.2 小时数据应至少记录该时间段内 $\mathrm { P M } _ { 1 0 }$ 或 $\mathrm { P M } _ { 2 . 5 }$ 的标况浓度、工况浓度以及标况累积体积的测量值；环境温度和环境大气压的平均值。

A.2.3 分钟数据应至少记录该时刻的环境温度、环境大气压、工况流量的测量值。

A.2.4 应统计记录当日小时数据的最大值、最小值和日均值。

# A.3 数据处理要求

# A.3.1 颗粒物标况浓度小时数据

颗粒物标况浓度小时数据按公式（A1）计算：

$$
C _ {s n, i} = \frac {m _ {i} \times 1 0 0 0}{V _ {s n , i}} \dots \tag {A1}
$$

式中：Csn,i - $C _ { s n , i }$ -监测仪第 $i$ 小时颗粒物标况浓度， $\mu \mathrm { g } / \mathrm { m } ^ { 3 }$ ；

$m _ { i }$ - -监测仪第 i 小时所采集到的颗粒物质量， $\mu \mathrm { g }$ ；

$V _ { s n , i }$ -- 监测仪第 i 小时的标况累积体积，L。

# A.3.2 颗粒物工况浓度小时数据

颗粒物工况浓度小时数据按公式（A2）计算：

$$
C _ {s, i} = \frac {m _ {i} \times 1 0 0 0}{V _ {s , i}} \dots \tag {A2}
$$

式中： $C _ { s , i ^ { - } }$ - 监测仪第 i 小时颗粒物工况浓度， $\mu \mathrm { g } / \mathrm { m } ^ { 3 }$ ；

$V _ { s , i }$ -- 监测仪第 i 小时的工况累积体积，L。

# A.3.3 采样流量转换计算

采样流量转换计算按公式（A3）计算：

$$
Q _ {s n} = Q \times \frac {2 7 3}{2 7 3 + t} \times \frac {B _ {a}}{1 0 1 . 3 2 5}. \tag {A3}
$$

式中： $\varrho _ { s n }$ --- -标况流量，L/min；

$\mathcal { Q }$ - -工况流量，L/min；

$B _ { a }$ - -环境大气压力，kPa；

-环境温度，℃。

# A.3.4 颗粒物标况浓度日均值

颗粒物标况浓度日均值按公式（A4）计算：

$$
\overline {{C}} = \frac {\sum_ {i = 1} ^ {n} C _ {s n , i}}{n}. \tag {A4}
$$

式中：C - -监测仪日均值浓度， $\mu \mathrm { g } / \mathrm { m } ^ { 3 }$ ；

$C _ { s n , i }$ -----监测仪当日第 $i$ 小时标况浓度， $\mu \mathrm { g } / \mathrm { m } ^ { 3 }$

n- 当日监测小时数（ $2 0 { \leqslant } n { \leqslant } 2 4$ ）。
