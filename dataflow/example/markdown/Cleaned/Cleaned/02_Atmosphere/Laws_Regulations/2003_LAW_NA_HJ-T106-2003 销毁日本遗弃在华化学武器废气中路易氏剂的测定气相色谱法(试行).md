# 1 范围

本标准规定了测定废气中路易氏剂的气相色谱分析方法。

本标准适用于销毁日本遗弃在华化学武器全过程废气中路易氏剂浓度的测定。

当采集气体体积为 $20 \mathrm{~L}$ 时, 废气中路易氏剂的最低检出浓度为 $4 \times 10^{-4} \mathrm{mg} / \mathrm{m}^{3}$ 。

# 3 方法原理

废气中的路易氏剂用盐酸水溶液吸收，路易氏剂和其水解产物氯乙烯亚砷酸（CVAA）被1，3-二巯基丙烷衍生后，用固相微萃取纤维萃取，采用气相色谱毛细管柱分离、硫型脉冲火焰光度检测器（PFPD-S）测定，以保留时间定性，外标法定量。由含量计算废气中路易氏剂的浓度。反应方程式如下：

# 4 试剂和材料

# 4.1 载气和辅助气体

4.1.1 载气：氮气，纯度为 $99.999\%$
4.1.2 燃气：氢气，纯度为 $99.99\%$
4.1.3 助燃气：净化后的空气。

# 4.2 试剂

除非另有说明，在分析中仅使用确认为分析纯的试剂和蒸馏水或相当纯度的水。

4.2.1 标准样品：路易氏剂，纯度不小于 $98\%$
4.2.2 衍生化试剂：1，3-二巯基丙烷（简称PDT)，纯度不小于 $98\%$ 。
4.2.3 衍生化试剂溶液：准确移取1，3-二巯基丙烷 $2.0\mu \mathrm{l}$ 于 $5.0\mathrm{ml}$ 容量瓶中，用丙酮定容、摇匀，得体积分数为 $4\times 10^{-4}$ 的衍生化试剂溶液。此溶液需现用现配，配制后用氮气保护，于 $4\%$ 避光条件下可保存 $2\mathrm{d}$ 。

4.2.4 无水甲醇。
4.2.5 丙酮。
4.2.6 盐酸。
4.2.7 吸收液： $\mathrm{pH}$ 为1.0的盐酸溶液，装入试剂瓶中，密封保存。
4.2.8 路易氏剂贮备溶液：称取路易氏剂约 $40\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ ）于 $100\mathrm{ml}$ 容量瓶中，用无水甲醇稀释至刻度、摇匀，于 $4\%$ 下保存。
4.2.9 路易氏剂系列标准溶液：于六个 $4\mathrm{ml}$ 固相微萃取样品瓶中分别加入 $3\mathrm{ml}$ 盐酸溶液（ $\mathrm{pH} = 1$ ），将贮备溶液（4.2.8）稀释250倍，用微量移液器准确移入稀释后的贮备溶液若干微升，再分别加入 $30.0\mu\mathrm{l}$ 衍生化试剂溶液（4.2.3），振荡 $30\mathrm{~s}$ 后，保持 $15\mathrm{~min}$ ，配制成浓度分别为 $2.0$ 、 $4.0$ 、 $8.0$ 、 $16.0$ 、 $24.0$ 、 $32.0\mathrm{ng/ml}$ 的路易氏剂系列标准溶液，此标准溶液需现用现配。

# 5 仪器

# 5.1 色谱仪

气相色谱仪，具硫型脉冲火焰光度检测器（PFPD-S）。

# 5.2 试样预处理仪器

使用的仪器有：

a）样品瓶： $4\mathrm{ml}$
b）振荡器；
c）固相微萃取仪（带磁力加热搅拌工作台、进样手柄、搅拌磁子和PDMS型膜厚 $100\mu \mathrm{m}$ 的固相微萃取纤维）；
d）微量移液器：50.0、100.0、500.0 $\mu \mathrm{l}$ 。

# 5.3 采样仪器

# 5.3.1 采样装置

具加热采样管的烟气采样装置，流量范围为 $0\sim 2.0\mathrm{L / min}$ 的烟气采样器。

# 5.3.2 吸收管

采用GB/T16157—1996中9.3.4b）所示大型气泡式吸收管， $25\mathrm{ml}$

# 6样品

# 6.1 样品采集

气体样品的采集按GB/T16157—1996执行。将装入 $10\mathrm{ml}$ 吸收液（4.2.7）的吸收管（5.3.2）连接好后，置于冰盐水浴中，与烟气采样器（5.3.1）相接，以 $1.0\mathrm{L / min}$ 的流量采集废气 $20\mathrm{min}$ 。同时记录采样器流量、流量计前的压力、温度及采样时间和地点。

# 6.2 样品的保存

采样后吸收液应尽快分析，密封样品在 $4\%$ 下保存。

# 6.3 试样的制备

# 6.3.1 路易氏剂的衍生

用移液管取 $3.0\mathrm{ml}$ 采样后的吸收液（6.1）于 $4\mathrm{ml}$ 固相微萃取样品瓶中，加入 $30\mu \mathrm{l}$ 衍生化试剂溶液（4.2.3），于振荡器（5.2b）上振荡 $30\mathrm{s}$ 使体系充分混匀，室温放置 $15\mathrm{min}$ ，待用。

# 6.3.2 固相微萃取法富集样品

在固相微萃取样品瓶（5.2a）中放入搅拌磁子，将固相微萃取纤维插入样品瓶内的吸收液中进行萃取，保持固相微萃取纤维完全浸入吸收液中。开动搅拌器，萃取 $15\mathrm{min}$ 后，取出固相微萃取手柄，插入色谱进样口解吸 $3\mathrm{min}$ ，进行色谱分析。

# 6.4 空白试验

同时作现场空白试验。

# 7 分析步骤

# 7.1 色谱条件

# 7.1.1 色谱柱

固定液为 $5\%$ 苯基 $95\%$ 聚二甲基硅氧烷或同等极性的石英毛细管柱： $30\mathrm{m}\times 0.32\mathrm{mm}\times 0.25\mu \mathrm{m}$ （长 $\times$ 内径 $\times$ 膜厚）。

# 7.1.2 气相色谱仪

气相色谱仪条件如下：

a）载气（氮气）流速： $1.2\mathrm{ml / min}$
b)气化室温度： $250~\mathrm{^\circ C}$
c）柱温：初温 $70\%$ 、保持 $3\mathrm{min}$ ，以 $15\% / \mathrm{min}$ 的速率升至 $260\%$ 、保持 $5\mathrm{min}$
d) 进样方式: 不分流进样, 不分流时间 $3 \mathrm{~min}$ ;
e) PFPD 检测器 S 模式;
f) 检测器温度: $220^{\circ} \mathrm{C}$ ;
g）氢气流速： $11.5\mathrm{ml / min}$
h）空气1流速： $10\mathrm{ml / min}$
i）空气2流速： $10\mathrm{ml / min}$
j) 数据采集方式：线性。

# 7.2 校准

# 7.2.1 仪器稳定性检验

待仪器各部分温度、气体流量、压力稳定正常时，连续二次测试与试样被测成分含量相近的标准样品，其峰面积相对偏差不大于 $10\%$ ，即认为仪器处于稳定状态。

# 7.2.2 标准曲线的绘制

待色谱仪基线稳定后，将系列标准溶液（每个浓度平行三个样品）用固相微萃取头分别富集、解吸并分析，计算平行三个样品的峰面积平均值，将数据回归后，得出回归方程，见公式（1）：

$$
Y = a + b X \tag {1}
$$

式中： $Y$ ——路易氏剂的质量， $\mathrm{ng}$

$a$ ——标准曲线的截距；

$b$ ——标准曲线的斜率；

$X$ ——路易氏剂的峰面积。

# 7.2.3 标准曲线的校正

标准曲线应每天校正一次。选标准曲线中间浓度进行校正，当相对偏差大于 $10\%$ 时，应重新绘制标准曲线。

# 7.2.4 检测限

以二倍信噪比峰高所对应的路易氏剂含量作为仪器的检测限，本仪器的检测限为 $1 \times 10^{-11} \mathrm{~g}$ 。

# 7.3.2 定量分析

# 7.3.2.1 分析

将6.1采集的样品按6.3进行衍生化反应和样品的富集，同时解吸并进样，用外标法定量。

# 7.3.2.2 计算

废气样品中路易氏剂的浓度按公式（2）计算：

$$
c = \frac {Y V _ {1}}{V _ {2} V} \times 1 0 ^ {- 3} \tag {2}
$$

式中： $c$ ——废气中路易氏剂的浓度， $\mathrm{mg/m^3}$ ；

$Y$ ——路易氏剂的质量， $\mathrm{ng}$ ；

$V_{1}$ —吸收液的体积， $\mathrm{ml}$

$V_{2}$ ——固相微萃取样品的体积， $\mathrm{ml}$

$V$ ——换算成标准状态下的采样体积， $\mathrm{L}$ 。

# 8 结果表示

# 8.1 定性结果

根据色谱图中路易氏剂的保留时间来确定被测样品中的路易氏剂成分。试样色谱峰的保留时间与标准样品的保留时间作比较，保留时间窗口为 $\pm 10\mathrm{s}$ 。应在色谱图上标明属于路易氏剂的峰。

# 8.2 定量结果

# 8.2.1 含量的表示方法

公式（2）中的 $c$ 表示废气中路易氏剂的浓度，以 $\mathrm{mg / m^3}$ 表示。

# 8.2.2 精密度

相对标准偏差不大于 $20\%$ 。

# 8.2.3 准确度

相对误差不大于 $20\%$ 。

# 8.2.4 最低检出浓度

采样体积为 $20\mathrm{L}$ 时，废气中路易氏剂的最低检出浓度为 $4\times 10^{-4}\mathrm{mg / m^3}$ 。
