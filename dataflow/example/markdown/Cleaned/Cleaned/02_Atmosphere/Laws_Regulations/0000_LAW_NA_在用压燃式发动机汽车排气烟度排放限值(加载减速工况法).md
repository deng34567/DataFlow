# 前言

本标准按照GB/T1.1-2009给出的规则起草。

本标准由海南省国土环境资源厅提出并归口。

本标准起草单位：海南省环境科学研究院、北京理工大学。

本标准主要起草人：谢东海、岳平、葛蕴珊、李清贵、薛英、陈表娟、刘统亮。

# 引言

为贯彻《中华人民共和国环境保护法》和《中华人民共和国大气污染防治法》，控制在用汽车污染物排放，改善海南省环境空气质量，根据《中华人民共和国大气污染防治法》第七条的规定，制定本标准。

本标准的技术内容是根据GB3847-2005《车用压燃式发动机和压燃式发动机汽车排气烟度排放限值及测量方法》进行制定，并参考了北京市地方标准DB11/121-2010及香港环保署的柴油车加载减速烟度排放法规(CAP.374中77F(1)(a)部分，2000年6月修订版）中相关技术内容。

本标准排放限值是按照HJ/T241-2005《确定压燃式发动机在用汽车加载减速法排气烟度排放限值的原则和方法》，并结合海南省车辆排放状况和大气污染控制目标制定的，排放限值在标准实施后，将定期根据实测数据进行调整。

# 在用压燃式发动机汽车排气烟度排放限值（加载减速工况法）

# 1 范围

本标准规定了在用压燃式发动机汽车加载减速法的术语和定义、排气烟度排放限值、测量方法、测量结果判定等要求。

本标准适用于装用压燃式发动机、最大总质量大于 $400\mathrm{kg}$ 、最大设计速度大于或者等于 $50\mathrm{km / h}$ 的在用汽车排气烟度检测。对紧密型多驱动轴车辆，双层客车、铰接客车、半挂牵引车、全挂牵引车、铰接列车和最大总质量大于 $10000\mathrm{kg}$ 的或车长大于 $12\mathrm{m}$ 的专用货车及全时四轮驱动车辆，可按自由加速法进行排气烟度的检测。

# 3术语和定义

GB3847、GB18285和HJ/T292界定的以及下列术语和定义适用本标准。为了便于使用，以下重复列出了GB3847、GB18285和HJ/T292中的某些术语和定义。

3.1

在用汽车

已经登记注册并取得号牌的汽车。

[GB3847-2005，定义3.13]

3.2

压燃式发动机

采用压燃原理工作的发动机（如：柴油机）。

[GB3847-2005，定义3.2]

3.3

轻型汽车

最大总质量不超过3500kg的M1类、M2类和N1类车辆。

[GB18285-2005，定义3.1]

3.4

M1、M2、N1类车辆

M1类车至少有四个车轮，或有三个车轮且厂定最大总质量超过 $1000\mathrm{kg}$ ，除驾驶员座位外，乘客座位不超过8个的载客车辆。
M2类车至少有四个车轮，或有三个车轮且厂定最大总质量超过 $1000\mathrm{kg}$ ，除驾驶员座位外，乘客座位超过8个，且厂定且最大总质量不超过 $5000\mathrm{kg}$ 的载客车辆。

N1类车至少有四个车轮，或有三个车轮且厂定最大总质量超过 $1000\mathrm{kg}$ ，厂定最大总质量不超过3500kg的载货车辆。

[GB18285-2005，定义3.2]

3.5

重型汽车

最大总质量超过3500kg的车辆。

[GB18285-2005，定义3.3]

3.6

第一类轻型汽车

设计乘员数不超过6人（包括司机），且最大总质量≤2500kg的M1类车。

[GB18285-2005，定义3.4]

3.7

第二类轻型汽车

在本标准适用范围内，除第一类车以外的其他所有轻型汽车。

[GB18285-2005，定义3.5]

3.8

最大总质量（GVM）

汽车制造厂规定的技术上允许的车辆最大质量。

[GB18285-2005，定义3.9]

3.9

全时四轮驱动车辆

四个车轮一直保持有驱动力的汽车。可分成固定扭矩分配（前后50:50比例分配）和变扭矩分配（前后动力分配比例可变）两大类。

3.10

紧密型多驱动轴车辆

车辆的驱动轴数大于等于3的车辆。

3.11

轮边功率

汽车在底盘测功机上运转时驱动轮实际输出功率的测量值。

[GB3847-2005，定义3.7]

3.12

最大轮边功率（MaxHP）

进行GB3847规定的功率扫描过程中得到的实测轮边功率最大值。

[GB3847-2005，定义3.8]

3.13

光吸收系数（k）

表示光束被单位长度的排烟衰减的一个系数，它是单位体积的微粒数n，微粒的平均投影面积a和微粒的消光系数Q三者的乘积。

[GB3847-2005，定义3.9]

# 3.14

发动机最大转速（MaxRPM）

进行GB3847规定的测试试验中，油门踏板处于全开位置时测量得到的发动机最大转速。

[GB3847-2005，定义3.10]

# 3.15

实测最大轮边功率时的转鼓线速度（VeIMaxHP）

进行GB3847规定的功率扫描试验中，实际测量得到的最大轮边功率点的转鼓线速度。

[GB3847-2005，定义3.11]

# 3.16

不透光烟度计

按GB3847附录G规定的、用于连续测量汽车排气的光吸收系数的仪器。

[GB3847-2005，定义3.4]

# 3.17

加载减速工况（LugDownMode）

柴油车发动机油门处于全开位置，通过对柴油车驱动轮强制加载迫使柴油车减速运行的工况。

[HJ/T292，定义3.11]

# 4.2 车辆预检和筛选要求

车辆预先检查要符合GB3847-2005附件JA规定的预检要求。如果发现受检车辆的车况太差，不适合进行加载减速法检测，必须先进行维修后才能进行检测。

# 5 测量方法

本标准测量方法采用加载减速法，按GB3847-2005附录J的规定执行，检测报告见附录A。

# 6 测量结果判定

6.1 如果在3个工况点（即：VelMaxHP点、90% VelMaxHP点和80% VelMaxHP点）测得的光吸收系数k中，有一项超过表1中规定的排放限值，则判定受检车辆排放检验不合格。
6.2如果受检车辆在功率扫描过程中测得的实际最大轮边功率值低于制造厂规定的发动机标定功率值的 $50\%$ ，则判定受检车辆排放检验不合格。
6.3 受检车辆同时满足光吸收系数和轮边功率的限值，则判定排放检验合格。
