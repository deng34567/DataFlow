# 空气质量一氧化碳的测定非分散红外法

Air quality-Determination of carbon monoxide Non-dispersive infrared spectrometry

UDC 661.92 :543.06

GB 9801-88

# 1 适用范围

本标准适用于测定空气质量中的一氧化碳。

测定范围为 $0\sim 62.5\mathrm{mg / m^3}$ ，最低检出浓度为 $0.3\mathrm{mg / m^3}$ 。

# 2 原理

样品气体进入仪器，在前吸收室吸收 $4.67\mu \mathrm{m}$ 谱线中心的红外辐射能量，在后吸收室吸收其他辐射能量。两室因吸收能量不同，破坏了原吸收室内气体受热产生相同振幅的压力脉冲，变化后的压力脉冲通过毛细管加在差动式薄膜微音器上，被转化为电容量的变化，通过放大器再转变为与浓度成比例的直流测量值。

# 3 仪器

3.1 一氧化碳红外分析仪：量程 $0\sim 62.5\mathrm{mg / m^3}$
3.2 记录仪： $0\sim 10\mathrm{mV}$ 。
3.3 流量计： $0\sim 1\mathrm{L / min}$
3.4 采气袋、止水夹、双联球。
3.5 氮气：要求其中一氧化碳浓度已知，或是制备霍加拉特加热管除去其中一氧化碳。
3.6 一氧化碳标定气：浓度应选在仪器量程的 $60\% \sim 80\%$ 的范围内。

# 4 采样

4.1 使用仪器现场连续监测将样品气体直接通入仪器进气口。
4.2 现场采样实验室分析时，用双联球将样品气体挤入采气袋中，放空后再挤入，如此清洗 $3\sim 4$ 次，最后挤满并用止水夹夹紧进气口。记录采样地点、采样日期和时间、采气袋编号。

# 5 分析

# 5.1 仪器调零

开机接通电源预热 $30\mathrm{min}$ ，启动仪器内装泵抽入氮气，用流量计控制流量为 $0.5\mathrm{L / min}$ 。调节仪器调零电位器，使记录器指针指在所用氮气的一氧化碳浓度的相应位置。

使用霍加拉特管调零时，将记录器指针调在零位。

# 5.2 仪器标定

在仪器进气口通入流量为 $0.5\mathrm{L / min}$ 的一氧化碳标定气，调节仪器灵敏度电位器，使记录器指针调

在一氧化碳浓度的相应位置。

# 5.3 样品分析

接上样品气体到仪器进气口，待仪器读数稳定后直接读取指示格数。

# 6 计算

按下式计算一氧化碳浓度：

$$
c = 1. 2 5 \times n
$$

式中： $c$ ——样品气体中一氧化碳浓度， $\mathrm{mg / m^3}$

$n$ ——仪器指示的一氧化碳格数；

1.25——一氧化碳换算成标准状态下的 $\mathrm{mg / m^3}$ 换算系数。

# 7 精密度和准确度

四个实验室对二种不同浓度的一氧化碳六次测定的重复性变异系数小于测量量程的 $1\%$ 。
