本标准规定了燃煤锅炉飞灰中氨含量测定的分光光度法。

# 3 原理

飞灰加水并碱化蒸馏，馏出液用硼酸溶液吸收，吸收液中的铵离子与纳氏试剂反应生成的络合物于波长 $4 2 0 \mathrm { n m }$ 处测量吸光度，根据飞灰量和测得的铵离子量计算飞灰中的氨含量。

# 4 试剂和材料

除非另有说明，分析时均使用符合国家标准的分析纯化学试剂。（悬置段，放总则？）

4.1 水：符合 GB/T 6682中一级水的技术要求。

4.2 轻质氧化镁：在 $5 0 0 \mathrm { { } ^ { \circ } C }$ 下加热氧化镁 $( \mathrm { M g O } ) 0 . 5 \mathrm { h }$ ，以除去碳酸盐。

4.3纳氏试剂（碘化汞-碘化钾-氢氧化钠（ $\mathrm { H g I } _ { 2 }$ -KI-NaOH）溶液）：称取 $1 6 . 0 \mathrm { g }$ 氢氧化钠(NaOH)，溶于 $5 0 \mathrm { m L }$ 水，冷却至室温。称取 $7 . 0 \mathrm { g }$ 碘化钾(KI)和 $1 0 . 0 \mathrm { g }$ 碘化汞 $\mathrm { ( H g I } _ { 2 } )$ ，一并溶于少量水后，在搅拌下缓慢加入到 $5 0 \mathrm { m L }$ 氢氧化钠溶液中，用水稀释至 $1 0 0 \mathrm { m L }$ 。贮于聚乙烯瓶内，用橡胶塞或聚乙烯盖子盖紧，避光存放，有效期 1 年。

4.4 酒石酸钾钠溶液： $5 0 0 \mathrm { g / L }$ ，称取 $5 0 . 0 \mathrm { g }$ 酒石酸钾钠 $\mathrm { ( K N a C _ { 4 } H _ { 6 } O _ { 6 } \bullet 4 H _ { 2 } O ) }$ 溶于水中，加热煮沸，充分冷却后稀释至 $1 0 0 ~ \mathrm { m L }$ 。
4.5 氢氧化钠溶液：1mol/L，称取 ${ 4 \mathrm { g } }$ 氢氧化钠（NaOH）溶于水，稀释至 $1 0 0 ~ \mathrm { m L }$ 。
4.6 硼酸溶液： $2 0 ~ \mathrm { g / L }$ ，称取 $2 0 \ \mathrm { g }$ 硼酸 $\left( \mathrm { H } _ { 3 } \mathrm { B O } _ { 3 } \right)$ )溶于水，稀释至 1L。
4.7 铵离子标准溶液
4.7.1 铵离子标准贮备溶液： $1 0 0 0 \mathrm { m g / L }$ ，称取 $2 . 9 6 5 4 \mathrm { g }$ 优级纯氯化铵（ $\mathrm { N H } _ { 4 } \mathrm { C l }$ ， $1 0 5 ^ { \circ } \mathrm { C } { \sim } 1 1 0 ^ { \circ } \mathrm { C }$ 干燥 2h），溶于水，在 $1 0 0 0 \mathrm { m L }$ 容量瓶中用水定容，贮于聚丙烯或高密度聚乙烯瓶。该溶液在 $2 ^ { \circ } \mathrm { C } { \sim } 5 ^ { \circ } \mathrm { C }$ 下存放，有效期 1 个月。
4.7.2 铵离子标准工作溶液： $1 0 \mathrm { m g / L }$ ，取 $5 . 0 0 \mathrm { m L }$ 铵离子标准贮备溶液(4.7.1)于 $5 0 0 \mathrm { m L }$ 容量瓶中，用水定容。

# 5 仪器与设备

5.1 可见分光光度计：配有 $2 0 \mathrm { m m }$ 比色皿。
5.2 氨氮蒸馏装置：由调温加热器、 $5 0 0 \mathrm { m L }$ 凯氏烧瓶、氮球、冷凝管、导管和锥形瓶组成，冷凝管末端可连接一段适当长度的滴管，使出口尖端浸入吸收液液面下。也可使用 $5 0 0 \mathrm { m L }$ 蒸馏烧瓶代替凯氏烧瓶。如图 1 所示。

# 6 样品采集与预处理

# 6.1 样品的采集与保存

烟道气飞灰的采集应按照 DL/T414 规定进行；其中采样位置应根据 GB/T16157 规定，以保证样品的代表性。采集后的样品应用自封袋密封保存，并在 7 天内分析。

# 6.2 样品预蒸馏

锥形瓶内加入 $5 0 \mathrm { m L }$ 硼酸溶液（4.6），确保冷凝管出口在硼酸溶液的液面之下。称取混合均匀的飞灰样 $2 0 \mathrm { g }$ ，精确到 $0 . 0 0 0 1 \mathrm { g }$ ，移入凯氏烧瓶，加入 $2 5 0 \mathrm { m L }$ 水，加入 $0 . 2 5 \mathrm { g }$ 轻质氧化镁（4.2）及数粒玻璃珠，立即连接氮球和冷凝管。加热蒸馏，控制加热温度，确保凯氏烧瓶内液体处于稳定沸腾且不爆沸，馏出液速率一般控制在 10mL/min 左右,待馏出液总量不少于 $1 0 0 \mathrm { m L }$ 时，停止蒸馏。将锥形瓶内液体全部转移至 $2 5 0 \mathrm { m L }$ 容量瓶并用水定容。此溶液为待测试样原液，应在 24 小时内完成分析。

# 7 试验步骤

# 7.1 仪器的准备

按照分光光度计使用说明书的要求，接通电源并调试仪器使之处于正常工作状态。

# 7.2 校准曲线的建立

在 8 支 $5 0 \mathrm { m L }$ 比色管中，分别加入铵离子标准工作溶液（4.7.2) 0.00、0.50、1.00、2.00、4.00、6.00、8.00 和 $1 0 . 0 0 \mathrm { m L }$ ，加水（4.1）稀释至刻度，混匀后加入 $1 . 0 \mathrm { m L }$ 酒石酸钾钠溶液（4.4），混匀。再依次加入 $1 . 0 \mathrm { m L }$ 纳氏试剂（4.3），混匀。放置 $1 0 \mathrm { { m i n } }$ ，在波长 $4 2 0 \mathrm { n m }$ 下，用 $2 0 \mathrm { m m }$ 比色皿，以水作参比，在 $2 0 \mathrm { { m i n } }$ 内完成吸光度测量。

以铵离子质量（ $\mu \ g$ ）为横坐标，以空白校正后的吸光度为纵坐标，建立校准曲线。

# 7.3 样品分析

取 $2 5 \mathrm { m L }$ 待测试样原液(6.3)到 $5 0 \mathrm { m L }$ 比色管，加入一定量的氢氧化钠溶液（4.5），调节水样 $\mathsf { p H }$ 值至 ${ 7 } { \sim } 8$ （可用 pH试纸检测水样 pH值），用水稀释至 $5 0 \mathrm { m L }$ 标线，再按校准曲线相同的步骤测量吸光度。当待测试样原液吸光度超出校准曲线范围时，应稀释待测试样原液后测定。

# 7.4 空白实验

用 $2 0 \mathrm { m L }$ 水（4.1）代替飞灰，按与样品相同的分析步骤进行吸光度测量。

# 8 结果计算

8.1 待测试样原液铵离子的质量浓度按公式（1）计算：

$$
C _ {0} = \frac {\mathrm {A} _ {S} \times f - A _ {b} - a}{b \times 2 5} \tag {1}
$$

式中：C0——待测试样原液中铵离子的质量浓度 (以 $\mathrm { N H 4 + }$ 计) ，单位为毫克每升（mg/L）；

As—待测试样的校正吸光度；

A b—空白试验的校正吸光度；

a 校准曲线的截距；

b —校准曲线的斜率；

25——测定时移入比色管的待测试液体积，单位为毫升（mL）；

f 测定时待测试样原液的稀释倍数，未稀释时为 1。

8.2 按公式（2）计算飞灰中的氨含量。

$$
C = \frac {C _ {0} \times 2 5 0}{m} \times 0. 9 4 4 \tag {2}
$$

式中： C 飞灰氨含量，单位为毫克每千克（ $\mathrm { m g / k g ) }$ ）；

C0 待测试样原液的铵离子浓度，单位为毫克每升（ $\mathrm { m g / L } \mathrm { \Omega }$ ）；

m 飞灰的质量，单位为克（g）；

250——待测试样原液的体积，单位为毫升（mL）;

0.944—铵离子与氨的换算系数。

计算结果按照 GB/T8170标准规定修约，取重复测定的两个结果的算术平均值作为报告值。两次结果相对标准偏差在氨含量不大于 $1 0 \mathrm { m g / k g }$ 时，应控制在 $7 \%$ 以内，在氨含量大于$1 0 \mathrm { m g / k g }$ 时应控制在 $2 . 5 \%$ 以内。
