# 中华人民共和国国家标准

GB 9804-88

# 烟度卡标准

# Standard for smokemetric tablet

![](images/50abbf3223a903f7a1052144d026391e41840edc823770c28b67f98320b349ba.jpg)

1988-08-15发布

1988-12-01实施

# 1 适用范围

本标准适用于新生产和在用的滤纸式烟度计（其抽气泵的抽气量为 $330 \pm 15 \mathrm{~mL}$ ；滤纸的有效工作面直径为 $32 \mathrm{~mm}$ )示值的分度和校准。

# 2 符号、定义

# 2.1 符号

$R_{\mathrm{b}}$ ——表示烟度卡的烟度；

BSU——表示烟度的单位。

# 2.2 定义

烟度卡的烟度定义为：

$$
R _ {\mathrm {b}} = (1 - \rho_ {\mathrm {v}}) \times 1 0
$$

式中： $\rho_{\mathrm{v}}$ ——烟度卡相对于滤纸的反射比；

10——烟度计的满量程。

# 3 技术内容

# 3.1 尺寸规格

# 3.1.1 尺寸

烟度卡的有效工作面积为 $36\mathrm{mm}\times 36\mathrm{mm}$

# 3.1.2 规格

烟度卡的规格有两种：其中一种为6级一套，烟度标定值从 $0.5\sim 9.0$ BSU；另一种为3级一套，烟度标定值均为4.5BSU。标定值允许误差为 $\pm 0.2$ BSU。

注：烟度卡的烟度值也可根据各使用部门的特定要求确定。

# 3.2 性能要求

烟度卡表面的涂料应满足光谱选择性、漫反射特性和反射均匀性等基本光学性能和化学稳定性能的要求。

# 3.3 标定条件

烟度卡的烟度值应由国家技术监督局和国家环境保护局共同确认的部门标定；标定装置的光谱条件和几何条件应符合反射测量的有关规定；标定值总不确定度应不大于0.1BSU。

烟度卡实物标签应有实物标准的认证标志，方可有效。标志见图。

3.4 使用和保存
3.4.1 使用要求

烟度卡使用时应避免摩擦和机械碰撞；同时应避免用手指触及涂层表面。

3.4.2 使用有效期

烟度卡使用一年后应对烟度值重新进行标定，若烟度卡表面涂层有明显污染和划痕应报废。

3.4.3 保存条件

烟度卡不使用时应放入包装袋内保存，同时应避免潮湿环境和直射阳光的影响。
