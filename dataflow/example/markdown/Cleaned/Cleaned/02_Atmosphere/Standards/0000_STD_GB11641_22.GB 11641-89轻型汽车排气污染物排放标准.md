# 中华人民共和国国家标准

# 轻型汽车排气污染物排放标准

GB 11641-89

Emission standard for exhaust pollutants from light-duty vehicle

# 1 主题内容与适用范围

本标准规定了轻型汽车冷起动后排气污染物标准值。

本标准适用于装点燃式四冲程发动机及压燃式发动机，最大总质量为 $400\sim 3500\mathrm{kg}$ ，最大设计速度等于或大于 $50\mathrm{km / h}$ 的轿车、客车和货车。

# 2 引用标准

GB11642 轻型汽车排气污染物测试方法

# 3 术语

# 3.1 基准质量

基准质量是指整车整备质量加 $100\mathrm{kg}$ 质量。

# 3.2 最大总质量

最大总质量是指汽车制造厂规定的技术上允许的最大质量。

# 3.3 排气污染物

排气污染物是指汽车排气管排出的一氧化碳(CO)、碳氢化合物(HC)及氮氧化合物 $\mathrm{(NO_x)}$ 。后者以二氧化氮 $\mathrm{(NO_2)}$ 当量表示。

# 4 试验分类及试验方法

# 4.1 试验分类

4.1.1 试验分型式认证试验与产品一致性检查试验。

4.1.2 型式认证试验，制造厂应提交一辆该厂生产车型的代表车辆，进行本标准4.2条规定的试验。

4.1.3 产品一致性的检查试验，从已经本标准型式认证试验合格的成批生产的车辆中任意抽取一辆，进行本标准4.2条规定的试验。

# 4.2 试验方法

试验方法按GB11642的规定。

4.2.1 被试车辆应放在底盘测功机上，每次试验共经历 $13\mathrm{min}$ ，包括4个循环，中间不得间断，每个循环应包括15种工况（怠速、加速、等速、减速等）。

4.2.2 用定容取样法(CVS)取样。用下列方法分析各种排放物：

一氧化碳(CO)：用不分光红外线吸收型(NDIR)分析仪；

二氧化碳 $(\mathrm{CO}_{2})$ ：用不分光红外线吸收型(NDIR)分析仪；

碳氢化合物(HC)：用氢火焰离子化型(FID)分析仪。对压燃式发动机用加热式氢火焰离子化(HFID)型分析仪；

氮氧化合物 $\left(\mathrm{NO}_{\mathrm{x}}\right)$ ：用化学发光型(CLD)或不分光紫外线共谱吸收型(NOUNR)分析仪。

4.2.3 除下述5.1.2.2及5.1.2.3外，试验应该重复三次。

# 5.1.2 试验结果、试验次数与标准值

$V_{1}, V_{2}, V_{3}$ 分别代表三种污染物1、2、3次试验结果； $L_{i}$ 代表 $i$ 种污染物的标准值。

5.1.2.1 试验应该重复进行三次，对给定基准质量的车辆，每次试验结果应小于表1标准值。即三次试验中，每次每种污染物 $V_{1} < L_{i}$ ； $V_{2} < L_{i}$ ， $V_{3} < L_{i}$ ，则认为合格。
5.1.2.2 如果三种污染物的 $V_{1} \leqslant 0.70L_{i}$ ，则只进行一次试验，就认为合格。
5.1.2.3 如果三种污染物的 $V_{1} \leqslant 0.85 L_{i}$ , 且其中有一种污染物 $V_{1} > 0.70 L_{i}$ , 则要进行第二次试验, 如果三种污染物的 $V_{2} \leqslant L_{i}$ , 每种污染物的 $V_{1} + V_{2} < 1.70 L_{i}$ , 则只要进行二次试验就认为合格。
5.1.2.4 如果三次重复试验结果，每一种污染物的算术平均值 $\frac{1}{3} (V_1 + V_2 + V_3) < L_i$ 并允许在三次试验中有一次一种污染物 $V < 1.10L_i$ ，则也认为合格。
5.1.2.5 如果三次重复试验结果，每一种污染物的算术平均值 $\frac{1}{3} (V_{1} + V_{2} + V_{3}) = (1.00 \sim 1.10) L_{i}$ 时，可根据制造厂的要求，将试验追加到10次，再根据10次试验结果的平均值作出结论。当 $V = \frac{1}{10} (V_{1} + V_{2} + V_{3} + \dots + V_{10}) < L_{i}$ 时，认为合格，否则认为不合格。

5.2.2 从成批生产的车辆中抽取一辆车，按4.2条规定进行试验，试验结果、试验次数与表2标准值的关系，按5.1.2.1～5.1.2.4规定确定。
5.2.3 如果从成批生产的车辆中抽取的一辆车不能达到表2标准值的要求，制造厂可以要求从成批生产的车辆中抽取若干辆样车，包括原来抽样的那辆车进行测定。制造厂应确定样车的数量 $n$ ，除原来取出的一辆样车外，其余样车每辆均应进行1次4.2条规定的试验。

对原来取出的一辆车应以三次试验的算术平均值作为试验结果。将在 $n$ 辆样车上进行的三种排气污染物排放测定试验所取得的结果分别求出算术平均值 $(\overline{X})$ 和标准偏差 $S$ 。如能满足下列条件则该成批生产的产品被认为是一致的：

$$
\bar {X} + K \cdot S \leqslant L _ {i}
$$

$$
S = \sqrt {\Sigma \frac {(X - \bar {X}) ^ {2}}{n - 1}}
$$

式中： $L_{i}$ ——为表2规定的三种排气污染物标准值；

$K$ ——根据 $n$ 确定的统计因素，其数值列入表3；

$X$ ——为从 $n$ 辆样车中任何一辆单独取得的试验结果。
