# 空气质量总悬浮微粒的测定

# 重量法

Air quality-Determination of total suspended

particulates—Gravimetric method

# 1 适用范围

本标准为贯彻执行GB3095—82《大气环境质量标准》而制定。

本标准适合于空气中总悬浮微粒的测定。

总悬浮微粒指空气动力学直径小于 $100\mu \mathrm{m}$ 的微粒。

# 2 原理

在已称重的滤膜上采集大气样品，总悬浮微粒被阻留在滤膜上，从滤膜的增重和采样体积，计算总悬浮微粒的浓度。

# 3 仪器

3.1 采样器：采样器流量范围 $100\sim 120\mathrm{L / min}$
3.2 采样头：采样头有效直径为 $80~\mathrm{mm}$
3.3百页箱：百页箱外形尺寸为 $l\times b\times h$ ， $\mathrm{mm}:200\times 200\times 200$ 。几何尺寸如图1。

图1 百页箱

3.4 分析天平：感量 $0.1\mathrm{mg}$
3.5 滤膜：使用的滤膜阻留率不得低于 $90\%$ ，可选用超细纤维玻璃滤膜或过氯乙烯滤膜。

# 4 采样

# 4.1 滤膜准备

将大张滤膜按采样头尺寸用不锈钢剪子剪成圆片，放入专用纸袋中，在干燥器内放置 $24\mathrm{h}$ ，称量滤膜重量，放回干燥器内 $1\mathrm{h}$ 后再次称重，二次称量之差不大于 $0.4\mathrm{mg}$ 即为恒重，装入专用纸袋备用。

准备的滤膜数量应多于采样所需数量的 $5\sim 7$ 张。

# 4.2 采样

用不锈钢镊子取出滤膜，平放在采样头的网板上，拧紧采样头套圈后装入采样器，放好百页箱开始采样。记录采样开始和停止的时间、流量。采样完毕将已采样滤膜尘面向里，对折两次成扇形放回纸袋。记下采样日期和采样地点。

用过氯乙烯滤膜采样，装膜时需揭去两边衬纸，采样完毕将对折好的滤膜仍夹在衬纸之间，装入纸袋。

测定任何一次浓度，每次需更换滤膜，采样时间不得少于 $1\mathrm{h}$ 。

测定日平均浓度，样品采集在一张滤膜上，采样时间不得少于 $6\mathrm{h}$ ，采样间断次数不得少于2次。

采样高度入口距离地面 $1.5\sim 2\mathrm{m}$

# 5 称重

采样后滤膜纸袋放入干燥器内，按4.1条再次称至恒重。

# 6 计算

按公式(1)计算总悬浮微粒的浓度：

$$
c = \frac {\left(W - W _ {0}\right) \times 1 0 0 0}{V} \dots \tag {1}
$$

式中： $c$ ——总悬浮微粒浓度， $\mathrm{mg / m^3}$

$W$ ——样品和滤膜重量，g；

$W_{0}$ ——空白滤膜重量，g；

$V$ ——换算成标准状态下的采样体积， $\mathrm{m}^3$ 。

# 附录A

# 滤膜称量温度校正方法

（参考件）

# A1 滤膜称重空气湿度的影响

湿度较大地区可根据实验室条件，还可选择恒温恒湿称量、恒湿称量、校正空白称量。

# A2 校正空白称量

空白滤膜放置天平室内平衡 $24\mathrm{h}$ 后称至恒重，记下天平室内温度和相对湿度。

采样后滤膜同样在天平室平衡 $24\mathrm{h}$ 后称至恒重。预留空白滤膜亦同时再次称量，按(A1)式计算总悬浮微粒浓度：

$$
c = \frac {\left(W - W _ {0} \frac {B}{A}\right) \times 1 0 0 0}{V} \dots \tag {A1}
$$

式中： $A$ 与正文 $W_{0}$ 同条件称重的空白校正膜的平均重量，g；

$B$ 与正文 $W$ 同条件称重的空白校正膜的平均重量，g；

其他符号意义同正文(1)式。
