# 前言

为贯彻《中华人民共和国水污染防治法》、《中华人民共和国大气污染防治法》和《中华人民共和国环境噪声防治条例》，加强对污水排放口、废气排放口和噪声排放源的监督管理，制定本标准。

本标准从1996年7月1日起实施。

本标准由国家环境保护局科技标准司提出。

本标准由国家环境保护局负责解释。

# 1 主题内容与适用范围

本标准规定了污水排放口、废气排放口和噪声排放源环境保护图形标志及其功能。

本标准适用于环境保护行政主管部门对污水排放口、废气排放口和噪声排放源的监督管理。

# 3术语

# 3.1 图形符号

以特定图形或图象为主要特征的符号。

# 3.2 标志

给人以行为指示的符号和（或）说明性文字。

# 3.3.提示图形符号

本标准所指提示图形符号是用于向人们提供某种环境信息的符号。

# 3.4 警告图形符号

本标准所指警告图形符号是用于提醒人们注意污染物排放可能造成危害的符号。

# 5标志牌的使用与维护

# 5.1 标志牌的设置

标志牌应设在与之功能相应的醒目处。

# 5.2 实施监督

本标准由各级环境保护行政主管部门统一监督实施。

标志牌制作由国家环境保护局统一监制。

# 5.3 检查与维修

标志牌必须保持清晰、完整。当发现形象损坏、颜色污染或有变化、退色等不符合本标准的情况，

2

应及时修复或更换。检查时间至少每年一次。
