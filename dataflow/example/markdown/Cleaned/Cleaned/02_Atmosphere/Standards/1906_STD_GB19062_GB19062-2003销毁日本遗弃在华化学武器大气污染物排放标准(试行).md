# 销毁日本遗弃在华化学武器

# 土壤污染控制标准（试行）

Controlling standards of soil pollution for the destruction of

chemical weapons abandoned by Japan in China

# 1 范围

本标准规定了日本遗弃在华化学武器污染土壤中各种有害物质的控制值。

本标准适用于日本遗弃在华化学武器污染的土壤和销毁处理全过程中有关的各种被污染的土壤。

# 3 对污染土壤的控制

3.1 在销毁日本遗弃在华化学武器全过程中，对可能被化学毒剂和有毒相关物污染的土壤，必须进行监测，土壤中有害物质含量超过控制限值的，必须进行无害化处理，采取措施限制其使用。土壤中各有害物质的污染控制限值见表1。

单位： $\mathrm{mg / kg}$

表 1 土壤中各污染物控制限值 (干基)


| 序号                                                                                            | 污染物项目        | 控制限值                           |
| --------------------------------------------------------------------------------------------- | ------------ | ------------------------------ |
| 1                                                                                             | 芥子气          | 0.02                           |
| 2                                                                                             | 路易氏剂a        | 0.04                           |
| 3                                                                                             | 芥子气与路易氏剂混合剂b | 按公式计算: cHD / SHD + cL / SL ≤ 1 |
| 4                                                                                             | 二苯氰肿c        | 0.05                           |
| 5                                                                                             | 苯氯乙酮         | 0.3                            |
| 6                                                                                             | 总氰化物（以CN-计）  | 0.2                            |
| 7                                                                                             | 总砷           | 30                             |
| a指路易氏剂、氯乙烯肿酸和氯乙烯氧肿的总量。b公式中：cHD：芥子气含量；SHD：芥子气含量限值；cL：路易氏剂含量；SL：路易氏剂含量限值。c指二苯氰肿、二苯氯肿、氧联双二苯肿的总量。 |              |                                |


3.2 判断土壤中砷污染状况应对比当地未被污染的土壤中砷的背景值进行。若当地土壤中砷的背景值低于表1中污染控制限值，而被监控土壤中污染物的含量高于背景值时，按3.1的规定执行。若当地土壤中砷的背景值高于表1中污染控制限值，而被监控土壤中污染物的含量高于背景值，必须进行无害化处理，采取措施限制其使用。

# 4 监测

# 4.1 土壤样品的采集

污染土壤样品的采样点选择、采样量、采样方法、注意事项等，参照采用《环境监测分析方法》（1983年版，原城乡建设环境保护部环境保护局）、中国环境监测总站编著《土壤元素的近代分析方法》（中国环境科学出版社，1992）“第二章土壤样品的采集”中的方法。

# 4.2 监测方法

本标准中各种污染物的监测方法见表2，可采用最低检测浓度更低的方法实施监测。

表 2 各污染物的监测方法


| 序号  | 项目              | 测定方法             | 方法标准编号          |
| --- | ---------------- | ---------------- | --------------- |
| 1   | 芥子气              | 气相色谱-质谱法         | HJ/T 108-2003   |
| 2   | 路易氏剂、氯乙烯氧肿、氯乙烯肿酸 | 气相色谱法            | HJ/T 146-2003   |
| 3   | 二苯氰肿、二苯氯肿、氧联双二苯肿 | 气相色谱-质谱法         | HJ/T 148-2003   |
| 4   | 苯氯乙酮             | 气相色谱法            | HJ/T 118-2003   |
| 5   | 总氰化物（按CN-计）      | 异烟酸-吡唑啉酮分光光度法    | HJ/T 141-2003   |
| 6   | 总砷               | 二乙基二硫代氨基甲酸银分光光度法 | GB/T 17134-1997 |

本标准由国家环境保护总局监督实施。
