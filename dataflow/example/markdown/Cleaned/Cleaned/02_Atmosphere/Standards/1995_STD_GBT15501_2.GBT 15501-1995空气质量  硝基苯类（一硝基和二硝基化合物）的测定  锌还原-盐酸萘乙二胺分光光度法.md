# 中华人民共和国国家标准

# 空气质量硝基苯类（一硝基和二硝基化合物)的测定锌还原-盐酸萘乙二胺分光光度法

GB/T15501—1995

Air quality-Determination of nitrobenzene(mononitro- and dinitro-compound)-Reduction by zincN-(1-Naphthyl)ethylene diamine dihydrochloride spectrophotometric method

# 1 主题内容及适用范围

# 1.1 主题内容

本标准规定了测定工业废气和环境空气中硝基苯类化合物的锌还原-盐酸萘乙二胺分光光度法。

# 1.2 适用范围

1.2.1 本标准适用于制药、染料、香料等行业排放废气中能还原为苯胺(芳香伯胺)类化合物的一硝基和二硝基苯类化合物的测定。
1.2.2 在采样体积为 $0.5\sim 10.0\mathrm{L}$ 时，测定范围为 $6\sim 1000\mathrm{mg / m^3}$

# 2·原理

用稀乙醇溶液吸收的硝基苯，在常温酸性条件下，由锌粉反应产生的初生态氢还原成苯胺，经重氮化后与N-盐酸萘乙二胺偶合反应生成紫红色偶氮染料，该染料的色度与硝基苯的含量成正比，在 $550\mathrm{nm}$ 波长处用分光光度法测定，反应式如下：

# 3 试剂

# 3.1 不含有机物的蒸馏水

3.9 硝基苯标准溶液：
3.9.1 硝基苯提纯：硝基苯 $\left(\mathrm{C}_{6} \mathrm{H}_{5} \mathrm{O}_{2} \mathrm{~N}\right)$ 重蒸馏，取 $210 \sim 211^{\circ} \mathrm{C}$ 馏分。
3.9.2 硝基苯标准储备液：

于已加有 $10\mathrm{ml}$ 无水乙醇(3.10)并准确称重的 $50.0\mathrm{ml}$ 棕色容量瓶中，迅速加入 $1\sim 2$ 滴硝基苯(3.9.1)，称重后再加入 $35\mathrm{ml}$ 无水乙醇(3.10)，以水定容摇匀，以差减法计算硝基苯储备液浓度。 $2\sim$ $5^{\circ}\mathrm{C}$ 存放，使用一月。

3.9.3 硝基苯标准中间液：

将硝基苯标准储备液(3.9.2)用 $10\%$ 乙醇溶液(3.2)稀释为约 $100\mu \mathrm{g / ml}$ ，中间溶液，临用前配制。

3.9.4 硝基苯标准使用液：

将硝基苯标准中间液(3.9.3)用 $10\%$ 乙醇溶液(3.2)稀释为约 $20~\mu \mathrm{g / ml}$ 使用溶液，临用前配制。

# 4 仪器

4.1 采样器：流量范围为 $0.2\sim 1.0\mathrm{L / min}$ 的空气采样器（备有流量测量装置）。

4.8 采样引气管：聚四氟乙烯管，内径 $6\sim 7\mathrm{mm}$ ，引气管前端带有玻璃纤维滤料。
4.9 空盒气压表。
4.10 水银温度计： $0\sim 100^{\circ}\mathrm{C}$

# 5样品

5.1 样品的采集
采样系统由采样引气管(4.8)，采样吸收管(4.3)，空气采样器(4.1)串联组成，吸收管体积为 $50\mathrm{ml}$ 或 $125\mathrm{ml}$ ，内装乙醇吸收液(3.2)分别为 $20\mathrm{ml}$ 或 $50\mathrm{ml}$ ，以 $0.5\sim 1.0\mathrm{L / min}$ 的流量，采气 $5\sim 20\mathrm{min}$ 。
5.2 样品的保存
采集好的样品应避光保存，以防止硝基苯挥发， $2\sim 5\mathrm{C}$ 存放，2天内分析完毕。

# 6.1.2 还原

于上述比色管(6.1.1)中加 $2\mathrm{g} / 100\mathrm{ml}$ 硫酸铜溶液(3.3)1滴， $(1 + 1)$ 盐酸溶液(3.6)1.0ml，用 $10\%$ 乙醇溶液(3.2)稀释，于 $10.0\mathrm{ml}$ 定溶，加 $0.2\sim 0.3\mathrm{g}$ 锌粉(3.8)，颠倒混匀，打开管塞，放置 $30\mathrm{min}$ 干滤纸干漏斗过滤，弃去最初的 $1\mathrm{ml}$ 滤液，取 $2.0\mathrm{ml}$ 滤液于 $25.0\mathrm{ml}$ 比色管(4.4)中，用水稀释定容至 $10.0\mathrm{ml}$ 刻线， $\mathrm{pH}$ 约为2。

# 6.1.3 显色

将上述盛标准色列的 $25.0\mathrm{ml}$ 比色管(6.1.2)置于 $15\sim 20^{\circ}\mathrm{C}$ 水浴中，加入 $0.25\mathrm{g} / 100\mathrm{ml}$ 亚硝酸钠溶液(3.4) $0.5\mathrm{ml}$ ，摇匀放置 $10\mathrm{min}$ ，再加 $2.5\mathrm{g} / 100\mathrm{ml}$ 氨基磺酸铵溶液(3.5) $0.5\mathrm{ml}$ ，摇匀振荡两次，放置 $10\mathrm{min}$ ，驱尽气泡后加入 $0.75\mathrm{g} / 100\mathrm{ml}$ 盐酸萘乙二胺溶液(3.7) $1.0\mathrm{ml}$ ，摇匀放置 $45\mathrm{min}$ ，从水浴取出于室温平衡，在波长 $550\mathrm{nm}$ 处，以水为参比，用 $1\mathrm{cm}$ 吸收池，测定各管的吸光度 $A$ 。

# 6.1.4 校准曲线（工作曲线）的绘制

将上述系列标准溶液测得的吸光度 $A(6.1.3)$ 扣除试剂空白(零浓度)的吸光度 $A_{0}$ ，便得到校准吸光度 $y$ ，以校准吸光度 $y$ 为纵坐标，以硝基苯含量 $x(\mu \mathrm{g})$ 为横坐标，绘制校准曲线，或用最小二乘法计算其回归方程式(3)。注意“零”浓度不参与计算

$$
y = b x + a \quad \dots \tag {3}
$$

式中： $a$ ——校准曲线截距；

$b$ ——校准曲线斜率。

由斜率倒数求得校准因子： $B_{\mathrm{s}} = 1 / b$

# 7 结果表示

# 7.1 计算公式

试样中硝基苯的吸光度 $y$ 用式（4）计算。

$$
y = A _ {\mathrm {s}} - A _ {\mathrm {b}} \quad \dots \tag {4}
$$

式中： $A_{s}$ ——样品测定(6.2)吸光度；

$A_{\mathrm{b}}$ ——空白试验(6.3)吸光度。

试样中硝基苯含量 $x(\mu \mathrm{g})$ 用式（5）计算。

$$
x = \frac {y - a}{b} \times \frac {V _ {1}}{V _ {2}} \text {或} x = (y - a) B _ {a} \times \frac {V _ {1}}{V _ {2}}$$

式中： $V_{1}$ ——定容体积，ml；

$V_{2}$ ——测定取样体积， $\mathrm{ml}$ 。

废气或空气中硝基苯浓度 $c(\mathrm{mg} / \mathrm{m}^3)$ 用式(6)计算。

$$
c = \frac {x}{V _ {\mathrm {n d}}} \tag {6}
$$

式中： $V_{\mathrm{nd}}$ ——所采气体标准状态体积， $(0^{\circ}\mathrm{C},101.325\mathrm{kPa})\mathrm{L}$

# 7.2 精密度和准确度

经六个实验室分析含硝基苯3.05、6.10、9.16 mg/L的三个统一样品，重复性标准偏差为0.046、0.128、0.078 mg/L，重复性相对标准偏差为1.5%、2.1%、0.85%，再现性标准偏差为0.14、0.18、0.16 mg/L，再现性相对标准偏差为4.6%、3.0%、1.8%，加标回收率为99.2%~100.1%。在四个实样分析中加标回收率为94.8%~106.3%。

# 8注意事项

日光照射和气温过高，都会引起吸收液和硝基苯的挥发，以至浓度变化，因此在采样、样品输送和存放过程中都应采取避光和低温的措施。
