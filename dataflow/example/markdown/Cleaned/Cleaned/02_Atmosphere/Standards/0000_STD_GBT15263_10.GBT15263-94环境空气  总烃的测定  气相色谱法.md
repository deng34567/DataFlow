# 环境空气总烃的测定气相色谱法

# Ambient air—Determination of total hydrocarbons—Gas chromatographic method

1994-10-26 发布

1995-06-01实施

# 1 适用范围

1.1 本标准适用于空气中总烃的测定。
1.2 本方法用注射器采集空气样品，以带氢火焰离子化检测器的气相色谱仪直接进行测定。样品经过空柱或玻璃微珠阻尼柱导入检测器，测定总烃含量。
1.3 方法检测范围：

所用的仪器不同，方法的测定范围不同。总烃测定下限为 $0.14\mathrm{mg / m^3}$ 。

以氮气为载气测定空气中总烃含量时，总烃峰中包括氧组分，因样品中氧对响应值有效应，在此，采用除烃净化空气求出空白值，从总烃峰中扣除氧组分值。

# 2 定义

总烃：氢火焰检测器所测的碳氢化合物 $(\mathrm{C}_1\sim \mathrm{C}_8)$ 为总烃，以甲烷计。

# 3 试剂和材料

3.1 载气和辅助气体
3.1.1 氮气： $99.9\%$ 的纯氮。用装有5A分子筛净化管净化。
3.1.2 氢气： $99.9\%$ 。用装有硅胶和活性炭净化管净化。
3.1.3 空气：由高压钢瓶或用无油空气压缩机供气，用装有硅胶和活性炭净化管净化。
3.1.4 净化空气：经除烃的净化空气装置处理（详见附录A）。
3.2 制备色谱柱使用的试剂和材料
3.2.1 制备色谱柱和填充物参考4.3条的有关内容。
3.2.2 色谱标准物：以氮气为底气 $7.1\mathrm{mg / m^3}$ (10 ppm)左右的甲烷标准气体。

# 4 仪器

4.1 仪器：配有氢火焰离子化检测器的气相色谱仪。
4.1.1 仪器主要部件：带有氢火焰离子化检测器，附带柱加热炉，温度调节器，注入口加热器，检测器加热器及相应的温度调节器，放大器等。
4.1.2 控制载气的压力表、流量计和辅助气体的指示流量计。
4.2 进样器：仪器自带不锈钢进样口或六通阀， $1\mathrm{ml}$ 定量管。
4.2.1 注射器： $1\mathrm{ml},5\mathrm{ml},50\mathrm{ml},100\mathrm{ml}$ 注射器若干个。
4.3 色谱柱：

4.3.1 色谱柱类型：空柱或填充柱。
4.3.2 色谱柱数量： $2\sim 3$ 根。
4.3.3 色谱柱的特征：
4.3.3.1 材料：不锈钢。
4.3.3.2 长度： $1\sim 2\mathrm{m}$
4.3.3.3 内径: $\phi 5 \mathrm{~mm}$ 。
4.3.3.4 形状：螺旋形。
4.3.4 填充物：
4.3.4.1 装硅烷化玻璃微珠或不装任何填料（空柱子）。
4.3.4.2 粒度： $60\sim 80$ 目。
4.3.5 填充方法：

不锈钢柱的一端用玻璃棉塞住，接真空泵；柱的另一端通过软管接漏斗，将担体慢慢通过漏斗装入色谱柱内。在装担体的同时，开启真空泵抽气，并轻轻敲击色谱柱使担体在色谱柱内填充紧密均匀，填充完毕后用玻璃棉塞住。为防止玻璃棉及担体抽入真空泵，在真空泵和色谱柱之间联结一毛细管和缓冲瓶。

4.3.6 色谱柱的老化：

将填好的色谱柱一端接到仪器进样口上，另一端不接检测器，用低流速（约 $10\mathrm{ml / min}$ )的载气通入，柱温升至 $100\mathrm{^\circ C}$ 老化约 $24\mathrm{h}$ ，然后将色谱柱接入色谱系统，待基线走平直为止。

4.4 检测器：氢火焰离子化检测器。
4.5 记录仪：量程 $5\sim 10\mathrm{mV}$

# 5样品

5.1 样品性质
5.1.1 样品名称：环境空气样品。
5.1.2 样品状况：气体。
5.2 采样和贮存方法

用 $100\mathrm{ml}$ 注射器在人的呼吸带高度抽取待测空气样品，反复置换三次后，抽取 $100\mathrm{ml}$ 样品用橡皮帽封住针头送往实验室待测；样品当天分析完毕。

# 6 操作步骤

6.1 调整仪器
6.1.1 汽化室： $70\sim 100^{\circ}\mathrm{C}$
6.1.2 柱温： $70^{\circ}\mathrm{C}$
6.1.3 载气流速氢气及空气的流速根据仪器的具体情况选择。
6.1.4 检测器温度： $150^{\circ}\mathrm{C}$
6.1.5 记录仪：
6.1.5.1 衰减：根据样品含被测组分浓度调节记录仪衰减。
6.1.5.2 纸速： $5\mathrm{mm / min}$
6.2 校准
6.2.1 外标法
6.2.1.1 校准次数：根据仪器的稳定性，一般在一批样品中，可在开始、中间和最后各校准一次。
6.2.1.2 标准样品的制备：以标准甲烷气，在仪器的线性范围内用注射器以氮气为底气配制一系列浓度的标准气体。

6.2.2 标准样品的使用条件
6-2-2-1 标准样品进样体积与试样进样体积相同，标准样品的响应值应接近试样的响应值。
6.2.2.2 使用的标准样品，要求只出单一峰，没有其他干扰物质。
6.3 试验
6.3.1 进样
6.3.1.1 进样方式：采用注射器或六通阀进样。
6.3.1.2 进样量： $1\mathrm{ml}$
6.3.1.3 操作：用注射器抽取待测样品，反复置换三次后，准确抽取 $1\mathrm{ml}$ ，迅速注入色谱系统。

6.5 定量分析

6.5.1 色谱峰的测量

以峰的起点和终点的联线作为峰底，从峰高极大值对时间轴作垂线，对应的时间即为保留时间，此线从峰顶至峰底间的线段即为峰高。

6.5.2 计算

总烃按下式计算：

$$
C _ {\text {总}} = E \times \frac {H _ {\mathrm {i}} - H _ {\mathrm {a}}}{H _ {\mathrm {s}}}
$$

式中： $C_{\text{总}}$ ——气样中总烃浓度（以甲烷计）， $\mathrm{mg / m^3}$

$E$ ——甲烷标准气浓度， $\mathrm{mg / m^3}$ ，即 $\mathrm{ppm}\times \frac{16}{22.4},\frac{16}{22.4}$ 为换算因子；

$H_{\mathrm{i}}$ ——样品中总烃峰高（包括氧的响应）， $\mathrm{cm}$ ；

$H_{\mathrm{a}}$ ——除烃净化空气峰高， $\mathrm{cm}$

$H_{\mathrm{s}}$ ——甲烷标准气体经总烃柱的峰高， $\mathrm{cm}$ 。

# 7 结果的表示

7.1 定性结果

根据甲烷标气在柱上的保留时间，确定被测试样的总烃。

7.2 定量结果

根据计算公式计算出样品中总烃的含量，结果以三位有效数字表示。

8 精密度和准确度

5个实验室分析含 $1.42\mathrm{mg / m^3}$ 及 $3.55\mathrm{mg / m^3}$ 的统一甲烷标准样品。

# 8.1 精密度

# 8.1.1 重复性

重复性相对标准偏差分别为 $3.28\%$ 及 $2.23\%$

# 8.1.2 再现性

再现性相对标准偏差分别为 $3.64\%$ 及 $2.26\%$

# 8.2 准确度

相对误差分别为 $+3.48\%$ 及 $-3.8\%$ ，环境空气样品加标回收率为 $96.3\% \pm 14.6\%$ 。

# 9 仪器测定下限

色谱仪在开机基线稳定的情况下，以测试样品时基线噪声2倍作为仪器的测定下限。本方法要求仪器的敏感度不低于 $10^{-10}\mathrm{g / s}$ 。

# A2 操作步骤

A2.1 除烃催化管的制备

U型管为内径 $4\mathrm{mm}$ 的不锈钢管，内装 $10\mathrm{g}$ 催化剂，床层高约 $7\sim 8\mathrm{cm}$ ，在U型管前接 $1\mathrm{m}$ 长、内径 $4\mathrm{mm}$ 的不锈钢预热管。

除烃净化空气装置通入室内空气或空气钢瓶，炉温升至 $450\sim 500^{\circ}\mathrm{C}$ ，温度恒定 $2\mathrm{h}$ 后，取除烃净化空气至GDX—502柱色谱测定无峰，即认为除烃完全。
