# 中华人民共和国国家标准

# 环境空气氟化物的测定石灰滤纸·氟离子选择电极法

GB/T 15433—1995

Ambient air-Determination of the fluoride-- Method by lime-paper sampling and fluorine ionselective electrode analysis

# 1 主题内容与适用范围

# 1.1 主题内容

本标准规定了测定环境空气中氟化物的石灰滤纸采集、氟离子选择电极法(简称LTP法)。

# 1.2 适用范围

# 1.2.1 应用领域

本方法适用于环境空气中氟化物长期平均浓度的测定。

# 1.2.2 测定限

当采样天数为一个月时，方法的测定下限为 $0.18\mu \mathrm{g} / (\mathrm{dm}^2\bullet \mathrm{d})$ 。

# 2 原理

空气中的氟化物(氟化氢、四氟化硅等)与浸渍在滤纸上的氢氧化钙反应而被固定，用总离子强度调节缓冲液提取后，以氟离子选择电极法测定，求得石灰滤纸上氟化物的含量，反映在放置期间空气中氟化物的平均污染水平。

# 3 试剂

本标准所用试剂除另有说明外，均为分析纯试剂，所用水为去离子水。

# 3.1 高氯酸： $72\%$ 。

# 3.2 氢氧化钠溶液： $c(\mathrm{NaOH})\approx 2.5\mathrm{mol / L}$

# 3.3 氢氧化钠溶液： $c(\mathrm{NaOH})\approx 5\mathrm{mol / L}$

3.4 石灰悬浊液的制备：称取 $56\mathrm{g}$ 氧化钙，加 $250\mathrm{mL}$ 水消化，在搅拌下缓慢加入高氯酸(3.1) $250\mathrm{mL}$ ，加热至产生白烟。冷却后再加水 $200\mathrm{mL}$ ，加热蒸发至产生白烟，重复三次，如有沉淀物，用玻璃砂芯漏斗(G3)过滤，在搅拌下向所得透明滤液中加入氢氧化钠溶液(3.2) $1000\mathrm{mL}$ ，得到氢氧化钙悬浊液，静置沉降后，倾出上清液，再用水重复洗涤 $5\sim 6$ 次，最后加水至 $5000\mathrm{mL}$ ，浓度约为 $1\%$ ，贮备待用，用时摇匀。
3.5 总离子强度调节缓冲液(TISAB)：称取 $58.0\mathrm{g}$ 氯化钠， $10.0\mathrm{g}$ 柠檬酸钠，量取冰乙酸 $50\mathrm{mL}$ ，加入约 $500\mathrm{mL}$ 水，溶解后，加氢氧化钠溶液(3.3)约 $135\mathrm{mL}$ ，调溶液 $\mathrm{pH}$ 值为5.2，加水定容至 $1000\mathrm{mL}$ 。
3.6 氟化钠标准溶液: $1000 \mu \mathrm{g} / \mathrm{mL}$ 。称取 $0.2210 \mathrm{~g}$ 氟化钠(优级纯试剂, 于 $110^{\circ} \mathrm{C}$ 烘干 $2 \mathrm{~h}$ , 放在干燥器中冷却至室温), 溶解于水中, 移入 $100 \mathrm{~mL}$ 容量瓶中, 用水定容至标线。贮于聚乙烯瓶中。此溶液为氟标准贮备液, 在冰箱中保存, 用时取出放至室温再用。

3.6.1 氟化钠标准溶液 $\mathrm{I}:100.0\mu \mathrm{g / mL}$ 。准确吸取氟化钠标准溶液(3.6) $10.00\mathrm{mL}$ ，移入 $100\mathrm{mL}$ 容量瓶中，用水定容至标线。
3.6.2 氟化钠标准溶液 $\mathbb{I}:50.0\mu \mathrm{g / mL}$ 。准确吸取氟化钠标准溶液I(3.6.1)50.00mL，移入 $100~\mathrm{mL}$ 容量瓶中，用水定容至标线。
3.6.3 氟化钠标准溶液 $\mathbb{I}$ : $25.0\mu \mathrm{g / mL}$ 。准确吸取氟化钠标准溶液 $\mathbb{I}$ (3.6.2) $50.00\mathrm{mL}$ ，移入 $100\mathrm{mL}$ 容量瓶中，用水定容至标线。
3.6.4 氟化钠标准溶液 $\mathrm{IV}:10.0\mu \mathrm{g / mL}$ 。准确吸取氟化钠标准溶液I(3.6.1) $10.00\mathrm{mL}$ ，移入 $100\mathrm{mL}$ 容量瓶中，用水定容至标线。
3.6.5 氟化钠标准溶液 $\mathrm{V}:5.0\mu \mathrm{g / mL}$ 。准确吸取氟化钠标准溶液 $\mathrm{N}$ (3.6.4) $50.00\mathrm{mL}$ ，移入 $100~\mathrm{mL}$ 容量瓶中，用水定容至标线。
3.6.6 氟化钠标准溶液 $\mathrm{VI}:2.5\mu \mathrm{g / mL}$ 。准确吸取氟化钠标准溶液V(3.6.5)50.00mL，移入 $100~\mathrm{mL}$ 容量瓶中，用水定容至标线。

注：上述溶液 $(3.6.1\sim 3.6.6)$ 均贮于聚乙烯塑料瓶中，临用现配。

# 4 仪器与材料

# 4.1 LTP标准采样装置：

a. 采样盒: 外径 $13 \mathrm{~cm}$ , 内径 $12.6 \mathrm{~cm}$ , 高 $2.5 \mathrm{~cm}$ (不包括盖) 的平底塑料盒, 具盖。盒内具有塑料环状垫圈 (外径 $12.5 \mathrm{~cm}$ , 内径 $11.0 \mathrm{~cm}$ ) 和固定滤纸用的塑料焊条 (或弹簧圈)。
b. 防雨罩：采用盆口直径 $30 \mathrm{~cm}$ ，盆高 $9 \mathrm{~cm}$ 的搪瓷盆，盆底用铁皮焊一个直径 $13 \mathrm{~cm}$ ，高 $3 \mathrm{~cm}$ 的圈，用于安装采样盒（见采样装置图）。

# 4.2 氟离子选择电极：

a. 测量氟离子浓度范围: $10^{-1} \sim 10^{-5} \mathrm{~mol}$ 。
b. 测定曲线斜率：在 $t^{\circ}\mathrm{C}$ 下，为 $(54 + 0.2t)\mathrm{mV}$ 。

# 4.3 甘汞电极：盐桥溶液为饱和氯化钾。

4.4 小型超声波清洗器。
4.5 磁力搅拌器：具聚乙烯包裹的搅拌子。
4.6 离子活度计或精密酸度计：分度值小于 $0.1\mathrm{mV}$ 。
4.7 聚乙烯塑料杯： $100~\mathrm{mL}$

4.8 聚乙烯塑料瓶： $100\mathrm{mL},1000\mathrm{mL}$
4.9 石灰滤纸：用两个大培养皿（直径约 $15\mathrm{cm}$ ）各放入少量石灰悬浊液，将直径 $12.5\mathrm{cm}$ 定性滤纸放入第一个培养皿中浸透、沥干，再放在第二个培养皿中浸透、沥干（浸渍 $5\sim 6$ 张滤纸后，换新的石灰悬浊液），之后摊放在大张定性滤纸上（干净、无氟），于 $60\sim 70^{\circ}\mathrm{C}$ 烘干，装入塑料盒（袋）中，密封好放入干燥器中备用（干燥器中不加干燥剂）。

4.10 定性滤纸： $\phi 12.5\mathrm{cm}$ 。

# 5 采样

取一张石灰滤纸，平铺在平底塑料采样盒底部，用环状塑料卡圈压好滤纸边，再用具有弹性的塑料焊条或卡簧沿盒边压紧（盒上可安装铆钉卡住焊条），将滤纸牢牢地固定，盖好盖，携至采样点。

采样点间距离一般为 $1\mathrm{km}$ 左右，距污染源近时，采样点间距离可缩小；远离污染源的采样点间距离可加大。采样点应设在较空旷的地方，避开局部小污染源(如烟囱等)，采样装置可固定在离地面3.5 $\sim 4\mathrm{m}$ 的电线杆或采样架上；在建筑物密集的地方，可安装在楼顶，与基础面相对高度应大于 $1.5\mathrm{m}$ 。

采样时，将装好石灰滤纸的采样盒的盒盖取下，装入采样防雨罩的盆底铁圈内，固定好，使石灰滤纸面向下，暴露在空气中，采样时间为7天到一个月。做好采样记录[记录放样品地点、样品编号及放样、取样时间(月、日、时)]。收样品时，取下采样盒，加盖密封，带回实验室，贮存在空干燥器内，在六个星期内完成分析。

# 6 分析步骤

# 6.1 校准曲线的绘制

取6个 $100\mathrm{mL}$ 聚乙烯塑料杯，按表1配制标准系列，也可根据待测样品浓度配制，不得少于6个点（分别取等体积的六种标准稀释液）。

将离子活度计接通，并按要求将清洗好的氟离子选择电极及甘汞电极插入制备好的标准系列溶液中，测定从低浓度到高浓度逐个进行，在磁力搅拌器上搅拌数分钟，待读数稳定后（即每分钟电极电位变化小于 $0.2\mathrm{mV}$ ），停止搅拌，静置后读取毫伏值，同时记录测定时的温度。

在半对数坐标纸上，以对数坐标表示氟浓度，以等距坐标表示毫伏值，绘制校准曲线。

或作直线回归，取得回归方程，要求 $\gamma > 0.999$ 以上，斜率符合 $(54 + 0.2t)\mathrm{mV}$ 。

校准曲线应在测定样品的同时绘制。

# 6.2 样品测定

取出石灰滤纸样品，剪成小碎块（约为 $5 \times 5 \mathrm{~mm}^2$ ），放入 $100 \mathrm{~mL}$ 聚乙烯塑料杯中，加 $25.00 \mathrm{~mL}$ TISAB缓冲液及 $25.00 \mathrm{~mL}$ 水，总体积 $50.00 \mathrm{~mL}$ ，在超声波清洗器中提取 $30 \mathrm{~min}$ ，取出放置过夜，（加盖，防止放置时污染）按校准曲线测定方法，读取毫伏值后，从校准曲线上查得氟含量或根据回归方程计算氟含量。测定样品时温度与绘制校准曲线时温度之差不应超过 $\pm 2^{\circ} \mathrm{C}$ 。

# 6.3 空白值测定

$\clubsuit$ 抽取 $4\sim 5$ 张未采样的石灰滤纸，分别用标准加入法进行测定，即在剪碎的空白石灰滤纸中加入 $0.5\mathrm{mL}$ 的氟化钠标准稀释液 $(10~\mu \mathrm{g / mL})$ ，然后按样品测定方法测定其氟含量，测定溶液总体积为 $50.00\mathrm{mL}$ 。取其平均值为空白石灰滤纸的氟含量(空白石灰滤纸的氟含量每张不应超过 $1\mu \mathrm{g}$ )；空白石灰滤纸氟含量为测定值 $(\mu \mathrm{g})$ 减去加入标准氟含量 $(5\mu \mathrm{g})$ 。

# 7干扰的消除

当测定体系中有 $\mathrm{Si}^{4+}$ 、 $\mathrm{Fe}^{2+}$ 、 $\mathrm{Al}^{3+}$ 存在时，(其浓度上限为 $20 \mathrm{ppm}$ )产生的干扰可采用加入总离子强度调节缓冲液来消除。

# 8 分析结果的计算

$$
C (\mathrm {F} \mu \mathrm {g} / \mathrm {d m} ^ {3} \cdot \mathrm {d}) = \frac {W - W _ {0}}{S \cdot n}
$$

式中： $C$ ——环境空气中氟化物的浓度， $\mu \mathrm{g} / \mathrm{dm}^3 \cdot \mathrm{d}$

$W$ ——石灰滤纸样品中氟含量， $\mu \mathrm{g}$ ；

$W_{0}$ ——空白石灰滤纸平均氟含量， $\mu \mathrm{g}$ ；

$S$ ——样品滤纸暴露在空气中的面积， $\mathrm{dm}^3$ ；

$n$ ——样品滤纸在空气中放置天数，准确至 $0.1\mathrm{d}$ 。

# 9 精密度与准确度

# 9.1 精密度

五个实验室在不同地点采平行样，每组5个样品，共取得31组样品，测定均值范围为 $2.81\sim 192\mu \mathrm{g} / (\mathrm{dm}^3\cdot \mathrm{d})$ ，平均变异系数为 $5.5\%$ ，最大为 $16.6\%$ 。

# 9.2 准确度

五个实验室测定统一制备的含氟 $50.0\mu \mathrm{g}$ 石灰滤纸标准样品，相对误差为 $0.23\%$ 。
