# 销毁日本遗弃在华化学武器

# 大气污染物排放标准（试行）

Emission standards of air pollutants for the destruction of

chemical weapons abandoned by Japan in China

# 1 范围

本标准规定了销毁日本遗弃在华化学武器过程中主要大气污染物的最高允许排放限值，同时规定了标准执行中的相关要求。

本标准适用于销毁日本遗弃在华化学武器全过程的大气污染物排放管理以及销毁设施建设的环境影响评价、设计、竣工验收及其运行后的大气污染物排放管理。

# 3 术语和定义

下列术语和定义适用于本标准。

# 3.1 最高允许排放浓度 maximum allowable emission concentration

指处理设施排气筒中污染物任何 $1\mathrm{h}$ 浓度平均值不得超过的限值。

# 3.2 标准状态 normal state

指温度为 $273\mathrm{K}$ 、压力为 $101325\mathrm{Pa}$ 时的气体状态。本标准规定的标准值，均指标准状态下的干气体中的数值。

# 3.3 排气筒高度 height of stack

指自排气筒（或其主体建筑构造）所在地平面至排气筒出口处的高度。

# 3.4 二噁英类 dioxins

多氯代二苯并-对-二噁英和多氯代二苯并呋喃的总称。

# 3.5 二噁英类毒性当量 dioxins toxicity equivalent quality (TEQ)

二噁英毒性当量因子（TEF）是二噁英毒性同类物与2，3，7，8-四氯代二苯并-对-二噁英对Ah受体的亲和性能之比。二噁英毒性当量可以通过下式计算：

$$
\mathrm {T E Q} = \Sigma \text {(二噬英毒性同类物浓度} \times \mathrm {T E F})
$$

# 4 大气污染物排放限值

4.1 日本遗弃在华化学武器销毁设施的排气筒高度一般不得小于 $30\mathrm{m}$ 。高度大于或等于 $30\mathrm{m}$ 的排气筒，其大气污染物排放浓度不得超过表1中的标准值。

4.2 排气筒高度小于 $30\mathrm{m}$ ，但大于或等于 $15\mathrm{m}$ 的，其最高允许排放浓度应按表1中标准值的0.5倍执行。
4.3 排气筒高度低于 $15\mathrm{m}$ 的，其最高允许排放浓度应按表1中标准值的0.1倍执行。
4.4 销毁处理设施排气筒周围半径 $200\mathrm{m}$ 内有建筑物时，排气筒高度必须高出最高建筑物 $5\mathrm{m}$ 以上，不能达到该要求的排气筒，其最高允许排放浓度应按表1中标准值的0.5倍执行。

# 5 其他规定

5.1 所有销毁处理日本遗弃在华化学武器的作业都必须在密闭、负压的环境中进行。排放的废气必须经过净化处理，严禁无组织排放。
5.2 有可能引起日本遗弃在华化学武器爆炸的作业必须在防爆设施内进行。防爆设施的抗爆强度应能足够耐受该设施内所有化学武器装药总量的爆炸力。

# 6 排放废气监测

# 6.1 非毒剂污染物的监测

# 6.1.1 采样点设置与采样装置

排气筒中颗粒物或气态污染物监测的采样点数目、采样点位置的设置及采样装置的配置，按GB/T16157—1996执行。

# 6.1.2 采样方法

污染物的采样方法按GB/T16157—1996和表2所列的分析方法执行。

# 6.1.3 采样时间和频次

在排放时段内以连续方式采样 $1\mathrm{h}$ ，或在 $1\mathrm{h}$ 内以等时间间隔采集四个样品，取其算术平均值。

当进行污染事故排放监测时，应按需要设定采样时间和采样频次，不受上述要求的限制。

# 6.2 毒剂污染物的监测

毒剂芥子气、路易氏剂、二苯氰肿、二苯氯肿、苯氯乙酮、氰化氢、光气的日常监督性监测，除执行6.1的规定外，还应另外设置带有报警功能的毒剂监测器。在日本遗弃在华化学武器销毁工厂，设置毒剂网络监测系统，进行在线连续运行，并将监测数据实时传送到工厂主控室，对整个销毁作业的运行进行全方位监控。

# 6.3 监测分析报告

监测分析结束，按GB/T20001.4—2001中A.18的要求写出监测报告。
