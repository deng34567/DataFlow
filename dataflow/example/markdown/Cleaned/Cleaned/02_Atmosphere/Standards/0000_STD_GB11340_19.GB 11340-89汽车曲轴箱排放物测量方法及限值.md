# 汽车曲轴箱排放物测量方法及限值

Motor vehicles—Crankcase emissions—Measurement methods and limit

# 1 主题内容与适用范围

本标准规定了发动机曲轴箱排放物的测量仪器设备、测量装置、测量方法及测量结果的修正方法，以及曲轴箱排放物的限值。

本标准适用于汽车用发动机。不适用于曲轴箱为进气系统一部分的二冲程发动机。

# 2 引用标准

JB3743 发动机性能试验方法

# 3 术语

3.1曲轴箱入口：指曲轴箱和大气之间的任何通气口，其作用是使曲轴箱能与外部交换气体。
3.2曲轴箱出口：指曲轴箱所有不通向大气而只通向进气系统某规定地点的通气口。
3.3 交换气体：指通过曲轴箱入口和外部交换的气体。

由外部流入曲轴箱的气体流量定义为正值，或由曲轴箱向外部流出的气体流量定义为负值（即漏气的一部分）。

# 4 仪器、设备

# 4.1 组成

由PCV阀（曲轴箱强制通风装置）、真空泵、流量调节阀、稳压筒、U型压力计、流量计及其附属的温度计和压力计、压缩空气源、环境大气压力计和温度计、测功机、转速表等组成。

# 4.2 要求

4.2.1 测量仪器应定期由国家认定的部门标定，在合格证有效期内使用。
4.2.2 仪器上应有防止被测气体冷凝的装置和油气分离装置。
4.2.3 仪器的测量范围及精度应符合表1的规定。

表1


| 测量项目 | 测量范围         | 仪器精度          |
| ---- | ------------ | ------------- |
| 流量   | 0~3m3/h      | ±0.15m3/h     |
| 流量   | 大于3m3/h      | ±5%           |
| 压力   | 0~0.05kPa(水) | ±0.0025kPa(水) |
| 压力   | 0~0.66kPa(汞) | ±0.033kPa(汞)  |
| 压力   | 大于0.05kPa(水) | ±5%           |
| 压力   | 大于0.66kPa(汞) | ±5%           |
| 温度   | 按测量对象选择      | ±2K           |

# 8 测量结果修正

# 8.1 标准大气状态

a. 大气压力： $100\mathrm{kPa}$
b. 大气温度： $298\mathrm{K}$ 。

# 8.2 修正

本试验在测量完各项参数后进行下述两项修正。

# 8.2.1 流量计流量修正

流量计均有标定的压力和温度，应根据实测时的压力和温度将实测流量换算到标定的压力和温度状态下的流量。

# 8.2.2 标准大气状态流量修正

将流量计标定流量修正到标准大气状态的流量，用下式：

$$
Q _ {\mathrm {p}} = Q _ {\mathrm {a}} \frac {P _ {\mathrm {a}}}{T _ {\mathrm {a}}} \cdot \frac {T _ {\mathrm {p}}}{P _ {\mathrm {p}}}
$$

式中： $P_{\mathrm{a}}$ ——流量计标定压力， $\mathrm{kPa}$

$T_{\mathrm{a}}$ ——流量计标定温度，K；

$Q_{\mathrm{a}}$ ——流量计标定后流量， $\mathrm{m}^3 /\mathrm{h}$

$P_{\mathrm{p}}$ ——标准状态的压力， $\mathrm{kPa}$ ；

$T_{\mathrm{p}}$ ——标准状态的温度，K；

$Q_{\mathrm{p}}$ ——标准状态的流量， $\mathrm{m}^3 /\mathrm{h}$ 。

# 9 限值

按7.2测得的交换气体量或按7.4计算出的交换气体量不得为负值。
