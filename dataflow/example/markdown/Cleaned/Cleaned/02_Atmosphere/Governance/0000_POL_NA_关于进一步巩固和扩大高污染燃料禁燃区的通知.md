# 关进巩扩于一步固和大高污染

# 燃料禁燃的通知区

# 渝 〔环 2018〕262 号

各（自治）、保局，江新保分局：区县县经开区环两区环

为实庆坚战实落《重市污染防治攻施方案（2018—2020年）》（渝委 〔发 2018〕28号庆贯彻国务赢蓝）《重市院打天保三年行划施方案》（渝府 〔卫战动计实办发 2018〕134号）有要求，持固和大高污染燃料禁燃，少高污染燃关续巩扩区减料燃造成的大气污染，助推打好污染防治攻、打天烧坚战赢蓝保，根据《中人民共和大气污染防治法》《重市大卫战华国庆气污染防治例》有定，就一步固和大高污染燃条关规现进巩扩料禁燃有要求通知如下。区关

一、一步明确并格行《高污染燃料目》进严执录

各（自治）、（以下各）要格按照区县县经开区简称区县严原境保部布的《高污染燃料目》（大气环护发录国环规〔2017〕2 号规执辖区环质）定行，根据大气境量改善要求、

能源消构、承受能力，三种燃料管控合（费结经济从组类别见表 1）其中一行。于全市煤建和高污染燃料选择类执鉴无区创禁燃建已有好的工作基，鼓有件的第区设经较础励条区县选择Ⅲ 行，或者类执 $\overset  { \boldsymbol { \pi } }$ 域合理划定不同的高污染燃料禁燃区类别区。

对质严环护办厅于直接燃用的生物燃料，格按照原境保部公《于高污染燃料禁燃管理中直接燃用生物等的复关区对质问题函》（大气函〔 环办 2017〕1886 号关释执）有解行。

# 二、持固和大高污染燃料禁燃范续巩扩区围

（一）展法查，固建成果。开执检巩设各要定期区县组织对本地已划定的高污染燃料禁燃展法查，原上每年区区开执检则不少于 2 次，重在秋冬季气象不利件展。查中点条时开对检发现违区规为时的反高污染燃料禁燃管理定的行，要及按照《中华国庆条人民共和大气污染防治法》《重市大气污染防治例》有定查，避免出污染反。关规严厉处现弹

（二）加大工作力度，大禁燃范。扩围北碚、渝北、巴区区南、涪陵、、江津、永川、南川、昌、区区长寿区区区区荣区万州、梁平、江、巫山、黔江、石柱及其他有区区垫县县区县条区县坚战标责书达件的要按照《污染防治攻目任》下的任务扩区过实，制定大高污染燃料禁燃工作方案，通施淘汰落后产过产业环迁乱能、化解剩能、燃煤企保搬、“散污”整治、清洁能源推等工程措施，少高污染燃料散象，确保到广减烧现2020 年，全市划定的高污染燃料禁燃面到区总积达 3000 平方公里以上。

（三）加工作度，化通考核。强调强报各要通全市大区县过气污染防治管理信息系每月送高污染燃料禁燃固和统报区巩扩大工作展情，并通工作信息的方式送工作中好的进况过报经验态环将报区县进将、做法。市生境局每月通各工作度，并目标务况纳任完成情入年度考核。

# 三、其他事项

各要本地布的高污染燃料禁燃划定通告、工作区县对区发区方案等范性文件行清理，其中仍在引用原家境保规进对国环护总关规环发局《于划分高污染燃料的定》（ 〔2001〕37 号）中的高污染燃料种的公文，及行修或者止，确保合法类时进订废有效。
