特此通知。

北京市海淀区人民政

2023年2月

北京市海淀区声环境功能区划实施细则

（2022年修订）

为加强区域环境噪声治理，改善城市声环境质量，提高人民群众生活水平，认真贯彻落实《中华人民共和国噪声污染防治法》相关要求依据《声环境质量标准》（GB3096-2008）、《声环境功能区划分技术规范》（GB/T15190-2014）、原环境保护部办公厅《关于加强和规声环境功能区划分管理工作的通知》（环办大气函〔2017〕1709号）、原市环保局《关于印发<北京市声环境功能区调整划分技术细则>的通知》（京环发〔2012〕255号），制定本细则。

# 一、适用范围

本细则适用于海淀区行政区域内的声环境管理。

机场周围区域受飞机通过（起飞、降落、低空飞行）噪声的影响不适用于本细则。

# 二、声环境功能区划分情况

本细则适用的昼间、夜间时段分别为：昼间6:00～22:00，夜间22:00～次日6:00。

海淀区声环境功能区依据《海淀分区规划（国土空间规划）（2017年—2035年）》划分为1、2、4类。

# （一）1类声环境功能区

海淀区辖区范围内除2类和4类声环境功能区外的区域，面积378.97平方公里。

# （二）2类声环境功能区

2类声环境功能区共约50.38平方公里，分为4个区域。

1.中关村永丰科技园：区域边界由北侧顺时针为玉河路-玉河南路-区界-唐家岭路-邓庄南路-六里屯社区路-永丰西路-永澄南路-永澄北路-丰渠东路，面积约16.10平方公里。
2.中关村上地科技园：区域边界由北侧顺时针为后厂村路-京包铁路-上地南路-信息路-农大南路-树村郊野公园东侧边界-北大生物城西二马连洼南路-北大生物城西一路-马连洼北路-上地西路-东北旺南路-西北旺南路-西北旺东路，面积约5.91平方公里。
3.清河、西三旗地区：区域边界由北侧顺时针为海淀昌平区界-西小口路-黑泉路-永泰庄北路-后屯路-后屯中街-后屯东路-西小口路-安宁庄街-安宁庄东路-毛纺路-朱房路-京包铁路。面积约13.67平方公里。
4.中关村翠湖科技园：区域边界由北侧顺时针为北清路-凝翠路-三星庄路-翠湖北环路-稻香湖路-翠湖南路-翠湖东路-京密引水渠-西六环路面积14.70平方公里。

# （三）4类声环境功能区

3.地面段公路和城市道路以最外侧非机动车道路或机非混行道路外沿为边界；高路基公路和城市道路以最外侧的边沟或路基边缘为边界；无辅路的高架公路和城市道路以高架段地面垂直投影的最外侧为边界；高速公路以护网处为边界，无护网的按一般公路和城市道路相关情况理。
4.4b类声环境功能区与4a类声环境功能区重叠的部分纳入4b类声环境功能区。
5.4类声环境功能区具体情况详见附件。

# 三、补充规定

（一）禁止在1类声环境功能区、严格限制在2类声环境功能区新建设产生噪声污染的工业项目。
（二）交通干线建设规划未实施前，应按照当前声环境功能区类别管理；规划实施后，交通干线两侧及时调整为4类声环境功能区。
（三）根据辖区内规划建设用地变化情况，适时开展声环境功能区划评估。

# 四、区划说明

本细则自2023年3月9日起生效，区政府《关于印发本区声环境功能区划实施细则的通知》（海行规发〔2013〕9号）同时废止。

本细则的具体适用问题由区生态环境局负责解释说明。
